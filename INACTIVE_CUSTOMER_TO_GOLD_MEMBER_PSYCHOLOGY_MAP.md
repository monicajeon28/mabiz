```

```

## 🧠 2부: 심리학 원리 × 영업 기법 완전 맵

### 2.1 손실회피 편향 (Loss Aversion Bias) — 가장 강력한 심리

**Kahneman & Tversky (1979) 원본 발견**: 손실의 고통이 동일한 이득의 기쁨보다 **약 2배** 강함

#### 부재중 고객에게 왜 강력한가?

```
이미 경험한 크루즈 = "이미 얻었던 기쁨"
1년 부재 = "그 기쁨을 상실한 상태"

따라서:
- "새로운 크루즈"를 파는 것이 아니라
- "잃었던 것을 되찾기"라는 프레임 사용
→ 손실회피 편향이 가장 강하게 작동
```

#### 콜 스크립트 적용

```
❌ 약한 버전:
"크루즈는 정말 좋아요. 다시 가보세요"

✅ 강한 버전 (L6 렌즈 + 손실회피):
"지난 2년간 배 위에서의 그 느낌, 기억하세요?
그걸 놓쳤죠?

지금 예약 안 하면:
- 체력은 계속 떨어지고
- 다음 기회는 1-2년 뒤
- 그 사이 인생이 바뀔 수 있어요

지금 준비하면 안전해요"
```

### 2.2 PASONA 6단계 × L0-L10 렌즈 통합

**PASONA**: 간다 마사노리의 카피라이팅 프레임 (문제→공감→해결책→제안→한정→행동)

#### 부재중 고객용 PASONA 맞춤 설정

| PASONA | L0-L10 렌즈 | 부재중 고객 특화 메시지 | 심리학 원리 |
|--------|-----------|---------------------|----------|
| **P (Problem)** | L1: 가격 오해 | "여행을 못 가는 이유가 뭘까요?" (상황 파악) | Status Quo 깨뜨리기 |
| **A (Affinity)** | L5: 적합성 의심 | "당신도 우리를 찾았어요. 처음엔 다들 의심했어요" (자기투영) | Narrative Transportation |
| **S (Solution)** | L2-L4: 준비/자유도 | "우리가 다 챙겨드려요. 당신은 쉬기만 하세요" (구체성) | Priming |
| **O (Offer)** | L3: 차별성 | "월 33K로 배 위의 리조트를 소유" (올인클루시브) | Scarcity |
| **N (Narrowing)** | L6: 타이밍 | "7월까지만, 자리 3개 남음, 지금이 체력 최고" (3중 한정) | Loss Aversion + FOMO |
| **A (Action)** | L10: 클로징 | "3가지 선택 중 어디로?" (거부 옵션 제거) | Commitment |

### 2.3 SPIN 판매 기법 × 부재중 고객 맞춤

**SPIN**: Neil Rackham의 대규모 거래 영업 기법 (상황→문제→함축→해결책)

#### 부재중 고객 재활성화에서 SPIN의 역할

**S (상황 파악)**
```
"최근 몇 년 해외여행은 어떻게 지내셨어요?"
→ 부재의 원인 파악 (일, 아이, 부모 봉양 등)
→ 현재 상황의 변화 포착 (아이 독립? 직책 상승?)
```

**P (문제 탐색)**
```
"이전에 크루즈 다녀오신 거 기억하세요?
그런데 지난 1년 동안 
가족과 그런 시간을 못 가졌어요?"
→ 부재의 대가를 고객 스스로 인정하게 유도
```

**I (함축/심각성)**
```
"계속 미루다가 5년 뒤에
아이가 중학생이 되면, 
부모와 여행을 안 가려고 할 텐데요.
그럼 추억을 만들 기회를 놓치는 건 아닐까요?"

→ 손실 회피 심리 극대화
→ 부재의 심각성을 고객이 스스로 깨달음
```

**N (해결책의 가치)**
```
"만약 배 위에서 가족과 진정한 쉼을 취할 수 있다면,
그게 얼마나 소중할까요?"
→ 자기설득 (고객이 가치를 정의함)
```

## 🎯 3부: 신민형 5STEP 클로징 × 부재중 고객 완전 시나리오

### 3.1 비교 모드 → 클로징 모드 전환의 비결

부재중 고객은 "신뢰 회복" 후 **"비교 모드"**에 들어갑니다.
```
"크루즈 괜찮긴 한데, 다른 여행사는?"
"정말 월 33K가 최저가?"
"내 상황에 맞을까?"
```

이 순간이 **클로징의 분기점**입니다.

### 3.2 신민형 5STEP 완전 스크립트

#### STEP 1: 주도권 탈취 + 시간 희소성

```
상담사: "선생님, 좋아요. 비교하는 거 맞아요.
        근데 제가 지금 2분이면 명확하게 
        비교 드릴게요. 괜찮으세요?"

→ 5초 침묵

심리학:
- Scarcity (희소성): "지금 2분" = 제한된 기회
- Reciprocity (상호성): 우리가 먼저 베풂 → 호의 반환
```

#### STEP 2: 비교 기준 선점 + 신뢰도 확보

```
상담사: "일반 패키지 여행사로 홍콩 5박 가면,
        항공 + 호텔 + 식사 + 가이드 다 포함해서
        1인당 120~150만원?

        근데 여기서 중요한 게 있어요.
        항만세, 팁, 기항지 택시비가 빠져있잖아요.
        보통 10~15만원 더 들어요.
        실제로 130~165만원이에요.
        
        우리 골드멤버는 그 여행이 50만원대.
        항만세까지 다 포함이고요."

심리학:
- Anchoring (앵커링): 130~165만원 vs 50만원 (극명한 대비)
- Transparency (투명성): 숨은 비용 먼저 노출 = 신뢰도 ↑
- Self-Persuasion (자기설득): 고객이 "50만원이면 나도..." 생각
```

#### STEP 3: 손실 앵커 극대화 (L6 타이밍 렌즈)

```
상담사: "그리고 한 가지 더 있어요.
        지금 바로 신청하시면
        선사 이벤트 상품 30% 할인이 7월까지인데,
        다음 달부턴 일반가로 올라가요.
        
        그럼 70~80만원 더 비싸지는 거죠.
        
        그리고 이제 체력이 제일 좋을 때 가셔야,
        5박 6일을 제대로 즐기실 수 있어요."

심리학:
- Loss Aversion (손실회피): "70~80만원 더 비싼다" = 손실 감정 극대화
- Scarcity (희소성): "7월까지만" = 시간 한정
- Ikigai (인생의 의미): "지금이 최고의 체력" = 마지막 기회
```

#### STEP 4: 삼중선택 제시 (L10 클로징 렌즈)

```
상담사: "그래서 선생님께 3가지 방법이 있어요.

        방법1) 지금 바로 결제하고 여름에 가기
              (완납 189만원)
        
        방법2) 월 33,000원으로 골드멤버 되신 후,
              선사 할인받아서 가기
              (월납, 상조+렌탈+웰니스 가입 후 계속 할인)
        
        방법3) 새미패키지로 20명이상 모으면
              스탭 무료지원으로 가기
              (단체 할인)
        
        이 3가지 중 선생님 상황에 제일 맞는 게 뭘까요?"

심리학:
- Choice Deletion (거부 옵션 제거): "안 하기"는 없고 "방법 선택"만
- Frame Shift (프레임 전환): "갈까 말까" → "어떻게 갈까"
- Triple Bind (삼중선택): 3개 중 하나는 반드시 선택
```

#### STEP 5: 즉시 액션 (모멘텀 유지)

```
상담사: "좋은 결정이세요.
        그럼 지금 카톡으로 신청 링크 보내드릴게요.
        5분이면 완료되니까..."

→ 냉각 기간 차단
→ 모멘텀 유지 (결정 → 행동의 즉시 전환)

심리학:
- Default Bias (초동편향): 일단 신청 → 취소 확률 ↓
- Commitment & Consistency (확약-일관성): 
  신청 완료 = 심리적으로 이미 멤버
```

### 3.3 부재중 고객 세그먼트별 STEP 4 변형

방법3) 새미패키지 (아이 데리고 가기)

→ L0 감정적 재연결 강화 필요

#### C: 1년 이상 부재 고객 (깊은 감정 필요)

방법2) 월 33,000원 (가족 추억 만들기)
방법3) 새미패키지 (세대간 경험 공유)

→ L5 자기투영 + L10 클로징 강화

### 4.1 5단계 신경화학적 전환

콜 25분 동안 고객의 뇌에서 일어나는 신경화학적 변화

#### Stage 1 (0:00-5:00): Dopamine ↑ + Cortisol ↓
```
L0 재연결 메시지:
"당신의 자리는 항상 예약돼 있었어요"

효과:
- Dopamine (보상감): "나를 기다렸다" = 특별함
- Cortisol (스트레스): 감소 (신뢰도 ↑)
```

#### Stage 2 (5:00-10:00): Amygdala 활성화
```
PASONA P + SPIN I (함축) 단계:
"계속 미루면 체력도 떨어진다"

효과:
- Amygdala (위협 감지): 손실의 심각성 인식
- Anxiety: "지금 해야 한다" 긴급성 유발
```

#### Stage 3 (10:00-15:00): Oxytocin 급증
```
자녀/배우자 사례:
"아이의 얼굴, 배우자의 웃음"

효과:
- Oxytocin (신뢰 호르몬): 감정적 안정감 + 신뢰
- Mirror Neurons: 자동 공감 회로 활성화
```

#### Stage 4 (15:00-20:00): Prefrontal Cortex 억제
```
신민형 STEP 4 (삼중선택):
"3가지 중 어디로?"

효과:
- Prefrontal Suppression: 비판적 사고 ↓
- Limbic Activation: 감정 결정 모드
```

#### Stage 5 (20:00-25:11): Commitment Lock-in
```
"월 33,000원" (낮은 가격 프레이밍) +
즉시 카톡 링크

효과:
- Commitment Lock-in: 심리적 약정 완성
- Default Bias: 일단 신청 = 취소 확률 ↓
```

## 🗺️ 5부: 완전한 콜 플로우 & 메시지 지도

### 5.1 부재 1년+ 고객 완전 콜 시나리오 (25분)

**상황**: 3년 전 크루즈 다녀온 50대 남성, 지난 1년 연락 없음

#### Opening (0:00-1:00) — L0 감정적 재연결

```
상담사: "선생님, 안녕하세요. 저 모니카예요.
        3년 전 홍콩 크루즈 다녀오셨잖아요.
        그 때 정말 좋으셨어요.
        
        요즘 어떻게 지내세요?
        (경청)"

심리학: Affiliation + 개인화된 기억 제시
```

#### Situation & Problem (1:00-7:00) — SPIN S+P + PASONA P

```
상담사: "최근 몇 년 해외여행은 어떻게 지내셨어요?"
고객: "음... 일이 많아서 거의 못 다니는 중이에요"

상담사: "그렇구나요. 그럼 그 때 크루즈 다녀온 이후로
        아내분, 자녀분과 특별한 시간을 못 가졌네요?"
고객: "맞아요. 요즘 많이 바빴어요"

심리학: 문제의식 자연스럽게 유도 (거부감 X)
```

#### Implication (7:00-12:00) — SPIN I + L6 손실회피

```
상담사: "계속 이렇게 일만 하다가 5년 뒤에
        자녀가 대학생이 되면,
        부모와 여행을 안 가려고 할 텐데요.
        
        그럼 당신이 지금 놓친 것을 
        영원히 되찾을 수 없잖아요?"

고객: "(침묵 5초) 음... 맞네요"

심리학: 손실회피 심리 극대화 + 시간의 절박성
```

#### Need-Payoff (12:00-18:00) — SPIN N + L5 자기투영

```
상담사: "만약 배 위에서 아무 생각 없이 
        아내분, 자녀분과 진정한 쉼을 취할 수 있다면,
        당신의 인생이 얼마나 달라질까요?"

고객: "(눈이 반짝임) 그런데... 지금 가능할까요?"

상담사: "좋은 질문이세요. 
        선생님처럼 체력이 좋으신 50대 분들,
        요즘 많이 와요.
        특히 지난해 크루즈 다녀오신 분들이
        '다시 가고 싶다'고 하더라고요."

고객: "정말요?"

심리학: 자기투영 + 사회증명 (다른 사람들도 함)
```

#### Offer & Narrowing (18:00-23:00) — 신민형 5STEP

```
상담사: "그래서 선생님께 3가지가 있어요.

        방법1) 완납 189만원 (빠르게 가기)
        방법2) 월 33,000원 + 골드멤버십 (계속 할인)
        방법3) 새미패키지 (친구분들과)
        
        이 중 선생님 상황에 맞는 게 뭘까요?"

고객: "음... 월 33,000원? 그게 뭐예요?"

상담사: "네, 매달 33,000원씩 내시면,
        크루즈 예약할 때마다 30% 할인,
        여행 중 스파도 무료,
        선실도 업그레이드해드려요.
        
        그래서 이번 여행은 50만원대,
        다음 해 여행은 50만원대,
        3년에 걸쳐서 여러 번 다니시는 거예요."

심리학: 거부 옵션 제거 + 프레임 전환
```

#### Action (23:00-25:00) — 즉시 확약

```
상담사: "좋은 선택이세요.
        그럼 지금 카톡으로 신청 링크 보낼게요.
        5분이면 완료되니까."

고객: "네, 좋아요"

상담사: (링크 전송 후) "완료되셨나요?
        그럼 이번 달 내로 첫 번째 여행 일정 잡아드릴게요"

심리학: 모멘텀 유지 + 초동편향
```

## 💡 7부: 실전 체크리스트

### 7.1 부재중 고객 (1년+) 콜 전 준비

```
[ ] 고객 이전 크루즈 경험 조사
    - 언제? (3년 전)
    - 어디? (홍콩)
    - 누구와? (가족)
    - 만족도? (매우 만족)

[ ] 현재 생활 상황 조사
    - 직업 변화? (임원진으로 승진)
    - 자녀 상황? (고등학생 → 대학 준비)
    - 건강 상태? (문제 없음)

[ ] 부재 원인 분석
    - "일이 바빴어요" → L2 준비 부담 공감 필요
    - "아이 때문에" → L5 자기투영 강화 필요
    - "비용이" → L1 가격 재설정 필요

[ ] 콜 시나리오 준비
    - L0: 감정적 재연결 메시지 2-3개 준비
    - SPIN: 상황/문제/함축 질문 3개씩 준비
    - 신민형: STEP 1-5 스크립트 숙지
```

### 7.2 콜 중 체크리스트

```
[ ] L0 재연결 (0:00-2:00)
    - 이전 경험 구체적으로 언급했는가?
    - "당신을 기다렸다" 메시지 전달했는가?

[ ] SPIN S+P (2:00-7:00)
    - 상황 파악 질문 3개 이상 했는가?
    - 문제를 "고객이" 인정했는가?

[ ] SPIN I (7:00-12:00)
    - 함축 질문 3개 이상 했는가?
    - 손실회피 심리가 활성화됐는가? (침묵 있었나?)

[ ] SPIN N (12:00-18:00)
    - Payoff 질문 3개 이상 했는가?
    - 고객이 자신이 말한 이익을 믿는가?

[ ] 신민형 5STEP (18:00-23:00)
    - STEP 1: 2분 동의 받았는가?
    - STEP 2: 비교 기준 선점했는가?
    - STEP 3: 손실 앵커 3개 제시했는가?
    - STEP 4: 삼중선택 명확하게 제시했는가?
    - STEP 5: 카톡 링크 즉시 전송했는가?

[ ] 영업자 말 비율
    - 고객 말 70% 이상인가?
    - 내 설명이 30% 이하인가?
```

## 🏆 최종 결론

### 부재 1년+ 고객 → 골드회원 전환의 완전 경로

```
부재 심리 상태
("아, 맞다. 그 회사...")

↓ L0 (감정적 재연결)

신뢰 회복
("당신을 기다렸어요")

↓ PASONA P + SPIN S/P/I

문제 인식 + 손실 각인
("지금 안 하면 안 되겠다")

↓ SPIN N + L5-L10

자기설득 + 불안감 해소
("월 33K이면 나도 할 수 있다")

↓ 신민형 5STEP

클로징 모드
("3가지 중 어디로?")

↓ 즉시 액션

골드회원 전환 (최고 충성도)
("당신을 신뢰했습니다")

예상 전환율: 75-85%
재신청율: 62-97% (부재 기간별)
```

### 핵심 3가지

1. **L0가 가장 중요**: "당신을 기다렸다"는 한 문장이 모든 심리 장벽을 녹임
2. **PASONA + SPIN의 순서**: 문제→함축→해결책 순서를 절대 뒤바꾸지 말 것
3. **신민형 5STEP 필수**: "어떻게 갈까"로 프레임을 전환하는 순간 클로징 성공
