# 부재중 고객 크루즈 콜 스크립트 (PASONA + SPIN 심리학 전개)
## 목표: 고객이 "이거 하려면 어떻게 해요?"라고 먼저 묻게 만들기

---

## 🎣 PHASE 1: 후킹 (30초)
### 심리학: **Attention (주의) + Curiosity (호기심)**

```
"안녕하세요, 김OO님! 
저는 크루즈닷 매니저 000입니다.

최근에 저희가 새로운 프로그램을 런칭했는데,
반응이 정말 뜨거워서 전화를 드렸어요.

'월 2만 7천원'이라는 멤버십인데,
한번 들어볼래요?"

(침묵 1초 - 고객의 호기심 대기)
```

**적용된 기법:**
- ✅ **PASONA-Problem**: "새로운 프로그램" = 무언가 변화가 생겼다는 신호
- ✅ **심리학-Curiosity**: 2만 7천원이라는 구체적 숫자가 호기심 자극
- ✅ **Scarcity 암시**: "반응이 뜨거워서" = 기회의 제한성

---

## 💭 PHASE 2: 욕망 각성 (2분 30초)
### 심리학: **Interest (관심) + Desire (욕망) + Self-Projection (자기투영)**

### 【Step 1: 상황 파악 (SPIN-Situation)】

```
상담사: "혹시 크루즈 여행을 가본 적 있으세요?"

(답변 청취)

상담사: "그렇군요. 
그럼 한번 물어봐도 될까요?

OO님이 지금 가장 하고 싶은 게 뭐예요?

일이 바빠서 여유를 원하시는 건가,
아니면 가족이랑 시간을 보내고 싶은 건가,
아니면 뭔가 자신을 위한 시간이 필요한 건가요?"

(고객의 감정 파악)
```

**적용된 기법:**
- ✅ **SPIN-Situation**: 배경 정보 수집
- ✅ **Open Question**: 고객이 말하도록 유도 (듣기 > 말하기)
- ✅ **Self-Disclosure**: 고객이 자신의 니즈를 인식하게 함

---

### 【Step 2: 문제 발견 (SPIN-Problem)】

```
상담사: "그런데 지금까지 여행할 때
가장 번거롭거나 불편했던 게 뭐였어요?

구체적으로요."

(고객 답변 청취)

상담사: "그래요, 많은 분들이 그 부분이 답답하다고 해요.

생각해보니까:
- 비행기표를 직접 찾고
- 호텔도 직접 비교하고
- 현지 가이드도 알아봐야 하고
- 혹시 문제 생기면 혼자 대처해야 하고...

이렇게 하려니까
'올해는 힘들겠지', '언젠가 해야지' 
이렇게 계속 미루게 되는 거 아닐까요?"

(침묵 3초 - 고객이 공감하도록)
```

**적용된 기법:**
- ✅ **SPIN-Problem**: 고객의 통증점(pain point) 발견
- ✅ **PASONA-Affinity**: "많은 분들이..." = 공감과 신뢰
- ✅ **문제 확대**: 고객이 미루는 근본 원인 제시
- ✅ **침묵 전술**: 고객이 자신의 문제를 인정하게 함

---

### 【Step 3: 함축적 영향 (SPIN-Implication) - 가장 중요한 부분】

```
상담사: "자, 그런데 이거 한번 생각해봐요.

지금처럼 '언젠가'하다가 5년이 지나면,
당신이 어떻게 될 것 같아요?

더 젊어질까요? 
아니면 더 피곤해질까요?

더 여유로워질까요?
아니면 더 바빠질까요?

(침묵 3초)

그리고 당신의 가족은?
자녀는 그때 얼마나 성장해 있을까요?
부모님은?

우리 고객들 중에
'올해는 꼭 가겠다고' 했는데
결국 못 간 분들이 정말 많아요.

당신도 그렇게 될 가능성 없을까요?"

(침묵 5초 - 손실회피 심리 발동)
```

**적용된 기법:**
- ✅ **SPIN-Implication**: 문제의 영향을 확대 (가장 강력한 설득)
- ✅ **심리학-Loss Aversion**: 구체적인 미래 손실 이미지화
- ✅ **사회적 증명**: "다른 고객들도..." = 정상화
- ✅ **감정적 이미지**: 자녀/가족의 시간 손실
- ✅ **침묵**: 고객 스스로 깨닫게 함

---

## 💰 PHASE 3: 해결책 제시 (2분)
### 심리학: **Conviction (확신) + Price Reframing (가격 재설정)**

### 【Step 1: 월 2만 7천원의 매력화】

```
상담사: "그래서 우리가 이 프로그램을 만들었어요.

'월 2만 7천원'인데,
이게 무슨 의미냐면요...

당신이 50만 원짜리 여행을 다닐 때,
우리는 그걸 30~40% 할인해서
35~40만원대로 가능하게 해주는 거예요.

(침묵 2초)

'어? 그럼 여전히 비싼데요?'
맞아요, 1번만 가면 비싸죠.

근데 만약에 당신이
3년에 3번을 간다면?

① 일반: 50만원 × 3 = 150만원
② 우리: (월 2.7만원 × 36개월) + (40만원 × 3)
        = 97만원 + 120만원 = 217만원...

아, 계산 실수네요.
다시하면, 월 2.7만원은 이미 할인을 포함한 거거든요.

더 간단하게 생각해보세요:

'당신이 3년에 3번 간다면,
월 2만 7천원으로 충분해요.'

대부분의 회원들이 연 1~2회 정도 다니니까요."

(침묵 2초)
```

**적용된 기법:**
- ✅ **PASONA-Solution**: 구체적인 해결책
- ✅ **가격 재설정**: 월 단위 재프레이밍
- ✅ **Loss Aversion 역이용**: "50만원을 35만원에" = 손실 방지
- ✅ **사회적 증명**: "대부분의 회원들..." = 정상화

---

### 【Step 2: 신뢰도 구축 (사회적 증명)】

```
상담사: "그리고 우리가 선사라는 게 중요해요.

지금까지 가입하신 분들 보면:
- 50대 부부: 89%
- 만족도: 98%

왜냐하면 우리는 정보가 먼저 들어와요.
'부산-오사카 3박' 같은 인기 일정은
3주 전에 다 끝나버리거든요.

근데 골드멤버분들은
그 정보를 먼저 받고,
가장 저렴한 가격으로 예약할 수 있어요.

그게 진짜 가치거든요."

(침묵 2초)
```

**적용된 기법:**
- ✅ **PASONA-Offer**: 구체적인 제안과 가치
- ✅ **사회적 증명**: 실제 고객 데이터
- ✅ **희소성**: "3주 전에 다 끝나" = 선점의 가치

---

## ❓ PHASE 4: 고객 자발적 질문 유도 (SPIN-Payoff)
### 심리학: **Conviction → Action 전환점**

```
상담사: "그래서 지금 딱 이 타이밍이 중요한데...

당신이 지금 결정하면:
- 120만원 지원을 받을 수 있고
- 69만원만 내고
- 7월에 바로 떠날 수 있어요.

또는:
- 월 2만 7천원으로
- 앞으로 3~5년 동안
- 언제든 20~30% 할인받으면서
- 자유롭게 여러 번 다닐 수 있어요.

(침묵 3초)

'맞네, 이거 괜찮은데요.'

그런데 아마도 이 정도면
당신이 스스로 물어보고 싶을 것 같은데...

'그래서 이거 하려면 어떻게 해야 해요?'
이런 생각이 드실 거 아닐까요?"

(침묵 2초 - 고객이 질문하도록 대기)
```

**적용된 기법:**
- ✅ **SPIN-Payoff**: 고객이 스스로 필요를 인정하게 함
- ✅ **PASONA-Narrowing**: 선택의 폭 좁히기 (2가지만)
- ✅ **암시적 동의**: "당신은 이미 결정했다"는 톤
- ✅ **질문 던지기**: 고객이 대답하도록 유도

## 🚀 PHASE 5: 자연스러운 클로징 (1분 30초)

```
고객: "네, 그래서 이거 하려면 어떻게 해야 해요?"

상담사: "좋은 질문이에요!

정말 간단해요. 
5분이면 완료돼요.

지금 카톡으로 신청 링크를 보내드릴게요.

(링크 전송)

이름, 연락처, 여행 날짜, 결제 방법
이것만 입력하면 돼요.

(고객이 입력하는 동안 대기)

완료되셨어요?

축하합니다!
당신은 이제 크루즈 가족이에요.

금요일 오후 8시에
카톡 라이브방송에서 
더 자세한 일정과 질문에 
실시간으로 답변해드릴 거예요.

정말 감사합니다!"
```

**적용된 기법:**
- ✅ **PASONA-Action**: 즉각적 행동 유도
- ✅ **마찰 최소화**: "5분이면 완료"
- ✅ **감정적 마무리**: 소속감과 감사

## 📋 심리학적 흐름 체크

| 단계 | 심리 상태 | 기법 | 목표 |
|------|---------|------|------|
| **1. 후킹** | Attention | Curiosity + Scarcity | 관심 유발 |
| **2. 욕망** | Interest → Desire | Self-Projection + Implication | 고객의 필요 각성 |
| **3. 손실** | Fear (Loss Aversion) | 구체적 수치 + 시간 | 미루지 못하게 함 |
| **4. 해결** | Conviction | Price Reframing + Social Proof | 확신 구축 |
| **5. 클로징** | Action | Payoff + 자발적 질문 | 고객이 먼저 신청 |

## 🎯 각 PHASE별 침묵 전술

```
Phase 1 (후킹): 1초
  → "호기심을 기다림"

Phase 2-1 (상황 파악): 고객 답변 청취
  → "들으면서 신뢰 구축"

Phase 2-2 (문제 발견): 3초
  → "고객이 공감하도록"

Phase 2-3 (함축적 영향): 5초
  → "손실 이미지 깊게 박히도록"

Phase 3 (해결책): 2초
  → "정보 소화 시간"

Phase 4 (질문 유도): 2초
  → "고객이 먼저 묻도록 대기"

Phase 5 (클로징): 1초
  → "링크 클릭 유도"
```

## 🚨 주의: 막혔을 때 대응

### 【"그래도 비싼데요"】
```
상담사: "네, 1번만 가면 비싸죠.

하지만 당신이 여행을 '자주' 다니는 사람이라면?
월 2만 7천원은 투자일 뿐이에요.

왜냐하면 매번 30~40% 할인받으니까요.

만약 당신이 3년에 3번 다닌다면
그 할인만으로도 이미 충분하거든요."
```

### 【"배우자랑 상담해야 해요"】
```
상담사: "당신의 결정이 가장 중요해요.

보통 한 분이 먼저 '이거 정말 필요하다'고 
느끼면, 나머지 분도 따라와요.

당신의 목소리가 가장 설득력 있거든요.

일단 신청하고, 
금요일 라이브방송에서 
두 분 다 궁금한 점을 물어봐도 돼요."
```

### 【"다른 여행사랑 비교해보고 싶어요"】
```
상담사: "정확히 좋은 질문이에요.

비교할 때 확인해보세요:
1) 항만세, 팁 다 포함?
2) 한국인 콜센터 있나?
3) 할인이 계속 가능한가? (우리는 5년)
4) 회원커뮤니티 있나?

이것들 확인하고 연락주면
저희가 이길 거예요.
우리는 선사니까요."
```
