# LeadGen PPTX η-3: 랜딩페이지 (한약복용 성공률)

## 주제
한약복용 다이어트 랜딩페이지 (실패율 문제 해결)

## 핵심 메시지
8가지 체중감량법 중 성공률 1위 = 한약복용 26.0% (원푸드 8.9% vs 한약 26.0%), 과학적 근거로 신뢰도 극대화

## 강의 목적 및 타겟 청중
- 다이어트 실패 경험 있는 40-60대 중년층
- 한약/건강기능식품 관심층
- 빠른 흡수/편리한 복용 중시 고객

## 마비즈 적용 가능 부분
1. **신뢰 설계**: 랜딩페이지에 통계자료(서울홍익병원 연구) 활용
2. **제품강점 강조**: 체험단 1위 만족도 (편리성 1위/빠른흡수 2위/효과 3위)
3. **CRM 리뷰 이벤트**: 후기 작성 시 문화상품권 증정 자동화
4. **SMS 가이드**: 복용법/주의사항 Day 0-7 시퀀스

## 메모리 (150단어)
1주차 과제 랜딩페이지. 엑스정(고농축 동결건조 타블렛)으로 의약품 수준의 신뢰도 확보. 메인 메시지는 '8가지 성공률 비교': 원푸드8.9%→건강기능식품17.5%→한약복용26.0%, 3배 차이 시각화. 2,161명 표본 연구결과로 객관성 강화. 제품 특장점 3가지(빠른흡수+편리한복용+부드러운 목넘김)를 체험단 조사결과로 증명. 이미지로 타블렛 형태 강조. CRM 구매고객에게 '이 통계를 알고 있었나요?' 메시지로 재구매 유도 가능. 가격대 100만원 상품군의 신뢰기반 마케팅 핵심 사례.
