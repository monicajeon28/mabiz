# LeadGen PPTX η-11: 린치핀_해설판 (러셀 브런슨)

## 주제
러셀 브런슨 린치핀 + 클릭퍼널 연속성 전략 (스타트업→9자리 매출)

## 핵심 메시지
린치핀=광고비 상쇄 장치, 연속성(월간 구독) 통한 무한 수익, 7단계 MIFGE 설계 완성

## 강의 목적 및 타겟 청중
- 온라인 마케팅 초급자→중급 진급자
- 기존 지식 버리고 배움 의지 있는 학생
- 스타트업부터 9자리 매출 추구 창업자

## 마비즈 적용 가능 부분
1. **린치핀 개념**: SLO(자체청산)→연속성상품 자동연결 아키텍처
2. **멤버십 단계**: CRM 구독 레벨 설계 (97/297/997/2500$/월)
3. **연속성 콘텐츠**: 7가지 필수요소(새로운기회/소비/월간개선/수리/라이브러리/커뮤니티/승격)
4. **상향판매**: 이너서클(5만$/년) → 카테고리킹(15만$) → 아틀라스(25만$)
5. **매력적캐릭터**: 브랜드 플라이휠로 고객 충성도↑

## 메모리 (180단어)
리드젠 자료. 러셀 브런슨 클릭퍼널 철학서. 린치핀은 광고비 상쇄 장치+연속성 진입점. 사람들이 놓치는 이유: 광고경험 부족/이해부족/무료광고만경험. 린치핀은 감사페이지에 간단한 버튼 하나만 필요. 연속성이 없으면 사업 없음(50% 수익이 연속성). 멤버십 4단계: 레벨1(MIFGE 97$/월 혼자하기), 레벨2(대량연속성297$ 그룹), 레벨3(고가997$ 코칭), 2500$ 단독마스터마인드. 연속성 7가지요소: 새로운기회/소비/월간개선/월별수리/도서관/커뮤니티/승격. 매력적캐릭터플라이휠로 MIFGE→멤버십→코칭→마스터마인드 상향판매. CRM 멤버십 구조 설계에 정확히 적용 가능.
