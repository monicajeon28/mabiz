# LeadGen PPTX η-5: 랜딩페이지 (엑스정 제품)

## 주제
다이어트 한약 '엑스정' 제품 발표 랜딩페이지

## 핵심 메시지
고농축 동결건조 타블렛 형태, 빠른 흡수+편리한 복용 (체험단 1위 만족도)

## 강의 목적 및 타겟 청중
- 다이어트 약 복용 경험자 (기존 고객)
- 편의성 중시하는 중장년층
- 빠른 효과를 원하는 고객

## 마비즈 적용 가능 부분
1. **제품 편리성 강조**: 목넘김 부드러움 + 빠른 흡수 강점
2. **CRM 사용후기**: 체험단 100명 조사결과 자동집계 및 배포
3. **SMS 복용 가이드**: 복용방법 Day 0/1/3 자동화
4. **재구매 유도**: "편리성 1위 만족도" 메시지로 기존고객 재신청
5. **이벤트 설계**: 구매후 리뷰 작성 시 인센티브 자동 지급

## 메모리 (160단어)
이평재 1주차 참고자료. 엑스정 중심 랜딩페이지는 짧고 명확. 좌상단 메인 메시지: "8가지 체중감량법 중 성공률 1위는 한약복용" 통계. 우상단 제품 소개: 고농축 동결건조 타블렛, 4가지 강점(빠른흡수/빠른효과/편리한복용/부드러운목넘김). 아래 체험단 만족도: 편리성1위→빠른흡수2위→효과3위. 100명 조사근거로 신뢰도 확보. 의약품급 신뢰도 전달. CRM에서 체험단 구성원들의 후기를 자동수집하여 신규고객 설득. 완성도 높은 1페이지 랜딩으로 고객 여정 최소화. 구매결정 가속화 가능.
