### 1.3 그룹 카드 컴포넌트

#### 구조
```
┌──────────────────────────────────────────────────┐
│ ┌───┐                                             │
│ │ Z │  그룹명 + 인라인 (234명)                   │
│ │   │                                             │
│ └───┘  설명: "..."                               │
│                                                  │
│ ✓ 퍼널 연결됨: 지중해 준비 퍼널                 │
│                                                  │
│ [복제] [내보내기] [⚡ 즉시발송] [⚙️]           │
│                                                  │
│ /* 블래스트 패널 (선택적) */                     │
│ <textarea>메시지...</textarea>                   │
│ [대상 확인] [발송]                              │
└──────────────────────────────────────────────────┘
```

#### 색상 아바타
- 너비: 40px
- 높이: 40px
- 모양: 원형 (border-radius: 50%)
- 배경색: group.color
- 텍스트: 그룹명 첫 글자 (흰색, 굵음)
- 폰트 크기: 14px

#### 그룹명 라인
```
[아바타] <그룹명> (234명)
```
- 그룹명: 16px, 굵음, 회색-900
- 인라인 (234명): 12px, 회색-400
- 수직 정렬: center

#### 설명 라인
- 색상: 회색-400
- 폰트: 12px
- 최대 1줄, 텍스트 오버플로우 처리 (...)

#### 퍼널 표시
```
✓ 퍼널 연결됨: 지중해 준비 퍼널
```
- 있을 때: 아이콘(GitBranch, 초록색) + 초록색 텍스트
- 없을 때: "퍼널 없음" + 회색 텍스트

#### 액션 버튼
```
[복제] [내보내기] [⚡ 즉시발송] [⚙️]
```

| 버튼 | 스타일 | 동작 |
|------|--------|------|
| 복제 | 회색 보더 | cloneGroup(id) |
| 내보내기 | 회색 보더 | exportGroup(id), 클립보드 복사 |
| 즉시발송 | 골드 보더 + 골드 텍스트 | openBlast(id) |
| ⚙️ | 아이콘만 | groupSettings(id) [미구현] |

## 2. 새 그룹 생성 폼 (이미지 #5 참고)

### 2.1 폼 레이아웃

```
┌────────────────────────────────────────┐
│ <h3>새 그룹 만들기</h3>                │
├────────────────────────────────────────┤
│                                        │
│ [필드 1] 그룹 이름 *                 │
│ <input placeholder="예: 지중해..." /> │
│                                        │
│ [필드 2] 설명                          │
│ <input placeholder="..." />            │
│                                        │
│ [필드 3] 색상                          │
│ [●] [●] [●] [●] [●] [●] [●] [●]     │
│                                        │
│ [필드 4] 연결할 퍼널                  │
│ <select>                               │
│   <option>퍼널 없음</option>          │
│   <option>지중해 준비 퍼널</option>  │
│ </select>                              │
│ ✅ 이 그룹에 고객 배정 시 즉시 퍼널 │
│                                        │
│ (오류 메시지)                          │
│                                        │
│ [그룹 만들기]  [취소]                 │
└────────────────────────────────────────┘
```

### 2.2 필드 상세

#### 필드 1: 그룹 이름
- Label: "그룹 이름 *"
- Type: text
- Placeholder: "예: 지중해 관심 고객"
- 검증: 1-100자, 필수
- 에러: "그룹 이름은 필수입니다."

#### 필드 2: 설명
- Label: "설명"
- Type: text
- Placeholder: "이 그룹에 대한 간단한 설명"
- 검증: 선택, 최대 500자
- 에러: 없음 (선택 필드)

#### 필드 3: 색상
```
[●●●●●●●●]
 1 2 3 4 5 6 7 8
```

**색상 옵션**:
```javascript
const COLOR_OPTIONS = [
  "#1E2D4E",  // 진한 네이비
  "#C9A84C",  // 골드
  "#10B981",  // 에메랄드
  "#3B82F6",  // 블루
  "#8B5CF6",  // 보라
  "#EF4444",  // 빨강
  "#F59E0B",  // 앰버
  "#6B7280",  // 그레이 (기본값)
];
```

- 크기: 각 원 28px
- 선택: 이 원 주위에 ring (테두리 강조)
- 스케일: 선택 시 125% 확대

#### 필드 4: 퍼널
- Label: "연결할 퍼널 (그룹 배정 시 자동 시작)"
- Type: select
- Options:
  - "퍼널 없음 (수동 발송만)" [기본값]
  - [동적] 조직의 모든 퍼널 목록
- 선택 시: 초록색 체크 메시지 표시
- 메시지: "✅ 이 그룹에 고객 배정 시 즉시 퍼널 시작"

### 2.3 폼 상태

| 상태 | 버튼 상태 |
|------|----------|
| 이름 없음 | disabled |
| 검증 오류 | disabled |
| 저장 중 | disabled + "저장 중..." |
| 유효함 | enabled |

## 3.1 탭: 고객 (Contact List)

### 구조 (이미지 #4 참고)
```
┌──────────────────────────────────────────┐
│ "그룹에 포함된 고객 (234명)"             │
│                                          │
│ [검색 입력]  [CSV 다운로드]  [고객 추가]│
├──────────────────────────────────────────┤
│ 고객명          │ 전화        │ 추가일   │
├─────────────────┼─────────────┼──────────┤
│ 홍길동 (체크)   │ 010-1234-5 │ 5월 10일 │
│ 김영희 (체크)   │ 010-5678-9 │ 5월 12일 │
│ 이순신           │ 010-9876-5 │ 5월 15일 │
├─────────────────┼─────────────┼──────────┤
│ 체크된 항목 2개  │ [일괄 제거] │          │
└──────────────────────────────────────────┘
```

### 테이블 열
| 항목 | 너비 | 내용 |
|------|------|------|
| 체크박스 | 40px | 다중 선택 |
| 고객명 | 200px | Contact.name |
| 전화 | 180px | Contact.phone (마스킹) |
| 추가일 | 120px | ContactGroupMember.addedAt |
| 액션 | 80px | 제거 버튼 (체크 시) |

### 우측 버튼
- [CSV 다운로드]: 고객명, 전화, 이메일, 추가일 3개 열
- [고객 추가]: 모달 팝업 → 고객 검색 → 일괄 선택 → 추가

### 페이지네이션
- 1페이지: 20명
- 하단: "1 2 3 ... 12" (페이지 번호)

```

├──────────────────────────────────────────┤
│                                          │

│ 색상                                     │

│                                          │
│ 연결할 퍼널                              │
│ <select>                                 │
│   <option>퍼널 없음</option>            │
│   <option selected>지중해 준비</option>│
│ </select>                                │
│                                          │
│ [저장]  [취소]                          │
│                                          │
├──────────────────────────────────────────┤
│ 위험 영역                                │
│                                          │
│ 이 그룹을 삭제하면 멤버도 함께 제거됩니다│
│ (Contact는 유지)                        │

│ [그룹 삭제] (빨강)                      │

```

### 저장 로직
```
[저장] 클릭
  → 검증 (이름 필수, 색상 형식)
  → PATCH /api/groups/[id]
  → 성공 시 토스트 "저장되었습니다"
  → 실패 시 에러 메시지
```

### 삭제 로직
```
[그룹 삭제] 클릭
  → 확인 모달 "이 그룹을 삭제하시겠습니까?"
  → 내용: "234명의 멤버도 함께 제거됩니다 (Contact는 유지)"
  → [삭제] [취소]
  → 삭제 시 DELETE /api/groups/[id]
  → 성공 시 목록으로 돌아가기
```
