# Funnel Wizard Schema Migration Summary (2026-06-24)

## Migration ID
**20260624_add_funnel_wizard_fields**

## Purpose
Extend Funnel, FunnelSmsMessage, and FunnelEmailMessage models with psychology lens, PASONA framework fields, and versioning for the Partner Funnel Wizard Phase 1.

## Changes Overview

### 1. Funnel Model (+5 fields)
| Field | Type | Default | Purpose |
|-------|------|---------|---------|
| `createdByUserId` | String? | NULL | Track who created the funnel (for sharing) |
| `visibility` | String | "PUBLIC" | Access control: PERSONAL / TEAM / PUBLIC |
| `autoGenerated` | Boolean | false | Flag for AI-generated funnels |
| `psychologyLens` | String? | NULL | L0-L10 psychology classification |
| `templateVersion` | Int | 1 | Version tracking for edit history |

**New Indexes (3):**
- `idx_funnel_org_creator_time` — "My Funnels" list + sorting
- `idx_funnel_org_psychology_visibility` — Lens & visibility filtering
- `idx_funnel_org_template_version` — Version history queries

### 2. FunnelSmsMessage Model (+6 fields)
| Field | Type | Default | Purpose |
|-------|------|---------|---------|
| `pasonaStage` | String? | NULL | PASONA framework stage (P/A/S/O/N/A) |
| `psychologyLens` | String? | NULL | L0-L10 lens for this message |
| `psychology` | String? | NULL | Psychological technique (LOSS_AVERSION, SCARCITY, etc.) |
| `segmentCode` | String? | NULL | Target segment (L0_INACTIVE, L1_PRICE, etc.) |
| `templateVariant` | String? | NULL | A/B test variant ID |
| `dynamicVariables` | Json? | NULL | Personalization variables map |

**New Index (1):**
- `idx_funnel_sms_message_pasona` — PASONA stage filtering

### 3. FunnelEmailMessage Model (+7 fields)
Same as FunnelSmsMessage, plus:
| Field | Type | Default | Purpose |
|-------|------|---------|---------|
| `preserveHtmlFormat` | Boolean | true | Backward compatibility for HTML emails |

**New Index (1):**
- `idx_funnel_email_message_pasona` — PASONA stage filtering

## Indexes Added (5 Total)

### Funnel Indexes
```sql
-- Query: "My Funnels" sorted by date
SELECT * FROM "Funnel" 
WHERE "organizationId" = $1 AND "createdByUserId" = $2 
ORDER BY "createdAt" DESC;
-- Index: idx_funnel_org_creator_time ✅

-- Query: Filter by lens + visibility
SELECT * FROM "Funnel" 
WHERE "organizationId" = $1 AND "psychologyLens" = $2 AND "visibility" = $3;
-- Index: idx_funnel_org_psychology_visibility ✅

-- Query: Version history
SELECT * FROM "Funnel" 
WHERE "organizationId" = $1 AND "isActive" = true 
ORDER BY "templateVersion" DESC;
-- Index: idx_funnel_org_template_version ✅
```
