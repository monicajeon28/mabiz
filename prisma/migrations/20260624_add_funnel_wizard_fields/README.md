## 📊 What's Being Added

### Funnel Model: 5 New Fields
```sql
ALTER TABLE "Funnel" ADD COLUMN "createdByUserId" TEXT;                    -- Funnel creator
ALTER TABLE "Funnel" ADD COLUMN "visibility" VARCHAR(10) DEFAULT 'PUBLIC'; -- Access level
ALTER TABLE "Funnel" ADD COLUMN "autoGenerated" BOOLEAN DEFAULT false;     -- AI flag
ALTER TABLE "Funnel" ADD COLUMN "psychologyLens" VARCHAR(3);               -- L0-L10
ALTER TABLE "Funnel" ADD COLUMN "templateVersion" INTEGER DEFAULT 1;      -- Version tracking
```

### FunnelSmsMessage Model: 6 New Fields
```sql
ALTER TABLE "FunnelSmsMessage" ADD COLUMN "pasonaStage" VARCHAR(20);       -- PASONA stage
ALTER TABLE "FunnelSmsMessage" ADD COLUMN "psychologyLens" VARCHAR(3);     -- Psychology lens
ALTER TABLE "FunnelSmsMessage" ADD COLUMN "psychology" VARCHAR(50);        -- Technique
ALTER TABLE "FunnelSmsMessage" ADD COLUMN "segmentCode" VARCHAR(30);       -- Segment
ALTER TABLE "FunnelSmsMessage" ADD COLUMN "templateVariant" VARCHAR(50);   -- A/B variant
ALTER TABLE "FunnelSmsMessage" ADD COLUMN "dynamicVariables" JSONB;        -- Personalization
```

### FunnelEmailMessage Model: 7 New Fields
(Same 6 as above, plus:)
```sql
ALTER TABLE "FunnelEmailMessage" ADD COLUMN "preserveHtmlFormat" BOOLEAN DEFAULT true; -- HTML compat
```

### Indexes Added: 5 Total
1. `idx_funnel_org_creator_time` — Creator + date sorting
2. `idx_funnel_org_psychology_visibility` — Lens + visibility filtering
3. `idx_funnel_org_template_version` — Version history
4. `idx_funnel_sms_message_pasona` — SMS message filtering
5. `idx_funnel_email_message_pasona` — Email message filtering
