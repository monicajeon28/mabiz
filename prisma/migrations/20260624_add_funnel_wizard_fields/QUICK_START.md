## ЁЯЪи If Something Goes Wrong

### Issue: "duplicate key value violates unique constraint"
тЖТ Run migration again: `npx prisma migrate deploy`

### Issue: "column already exists"
тЖТ Migration already applied. Skip step 1, go to step 2 (update schema)

### Issue: TypeScript errors (tsc != 0)
тЖТ Check schema.prisma for syntax errors in copied model definitions
тЖТ Compare with `schema_changes.prisma` line-by-line

### Issue: "Prisma Client not updated"
тЖТ Run: `npx prisma generate`
тЖТ Then: `npm run dev` (restart dev server)
