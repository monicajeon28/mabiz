**작성 일시**: 2026-05-19

## 개요

키워드 검색은 고객의 **무의식적 욕망의 신호**입니다. 검색 볼륨, PC/모바일 비율, 검색 조합(의도)를 분석하면 고객의 심리 단계를 정확히 파악할 수 있고, PASONA Solution 7가지와 매핑하여 콜 스크립트를 최적화할 수 있습니다.

#### 고객 심리 단계
| PASONA | 심리 상태 | 키워드 신호 | 최적 Solution |
|--------|---------|-----------|--------------|
| **P** (인식) | "휴가가 필요해" | 모바일 37K검색 (감정이 논리의 4.4배) | Solution 5 (기대감) |
| **A** (공감) | "가족과의 추억을 만들고 싶어" | "가족" 키워드 함께 검색 | Solution 5 (기대감) |
| **S** (탐색) | "어디를 가지?" | "동남아", "홍콩", "지중해" 등 지역 검색 | Solution 2 (방법) |
| **O** (비교) | "렌탈이나 호텔과 어떻게 다른가?" | "크루즈비용", "가격" 검색 증가 | Solution 4 (효율성) |
| **N** (결정) | "이 타이밍이 맞나?" | "할인", "이벤트" 키워드 | Solution 1 (중요포인트) |
| **A** (행동) | "지금 예약할까?" | 즉시 클릭, 카톡 요청 | Solution 3 (간단함) |

#### 주요 심리학 원리
- **Narrative Transportation** (이야기 몰입): 모바일 검색 ≫ PC 검색 → 감정이 논리 선행
- **Anticipatory Joy** (기대의 즐거움): 예약 전 기대감이 이미 50% 경험
- **Loss Aversion** (손실 회피): 가격 인상/자리 한정 정보에 민감

#### 콜 스크립트 최적 조합
```
Step 1 (신뢰):    Solution 6 (비밀) → "여행사가 안 말하는 크루즈의 진짜 가치"
Step 2 (욕구):    Solution 5 (기대감) → "가족이 함께 매일 밤 야경 감상하는 경험"
Step 3 (해결책):  Solution 2 (방법) → "7박 일정의 각 day별로 어떻게 되는지..."
Step 4 (가격):    Solution 4 (효율성) → "호텔 예산으로 4개 도시 경험"
Step 5 (클로징):  Solution 7 (배움) → "자녀가 살아가며 이런 경험이..."
```

#### 세그먼트별 맞춤 전략
- **30대 커플** → Solution 5 강화 (로맨틱 기대감)

## 카테고리 6: 신규 DB vs 부재중 DB 심리 전략

### 6-1. 신규 DB (콜드 콜) - 신뢰 구축 우선

#### 심리 장벽
```
"이 회사 누군데?" 
→ "크루즈? 비쌀 것 같은데?" 
→ "정말 괜찮을까?" 
→ "사기 아니겠지?"
```

#### 해제 순서 (PASONA 순)
1. **Solution 6 (비밀)** 1순위
   - "여행사 입장에서 말하지 않는..." (신뢰 기초)

2. **Solution 5 (기대감)** 2순위
   - "기대감부터 시작되는 경험..." (감정 활성화)

3. **Solution 1 (중요포인트)** 3순위
   - "지금 예약할 때와 6월의 차이..." (손실회피)

4. **Solution 4 (효율성)** 4순위
   - "동일 예산으로 2배 경험..." (합리성)

#### 콜 흐름 (15분)
```
[1-2분] 신뢰 구축
"저희 고객분들이 제일 놀라는 게... 
 호텔 예산으로 크루즈를 한다는 거예요. 
 어떻게 가능할까요?"

[2-4분] 기대감 활성화
"선생님 상상해보세요. 
 매일 밤 다른 야경을 보면서 식사하고, 
 아침에 새로운 도시에 도착해 있어요."

[4-6분] 해결책 제시
"이게 가능한 이유는... 
 선박 구조 때문인데요. 
 24시간을 배 위에서 활용하는 거죠."

[6-10분] 가격 정당화
"호텔 5박은 하루 8시간 자고 나가는 거고, 
 크루즈 5박은 24시간이 여행이에요. 
 생각해보니 크루즈가 저렴하지 않나요?"

[10-15분] 클로징
"지금 이 일정이 마지막이 될 수 있어요. 
 다음 달부터는 가격이 올라가거든요. 
 선택하실래요?"
```

### 6-2. 부재중 DB (웜 콜) - 업그레이드 제안

#### 심리 상태
```
"아, 이 회사 알아" 
→ "뭔가 달라졌나?" 
→ "좋은 게 있으면 들어보지" 
→ "오 이건 괜찮은데" 
→ "바로 해야겠는데?"
```

#### 해제 순서 (상향식)
1. **Solution 5 (기대감)** 1순위
   - "지난번보다 2배 더 특별한 경험..." (재활성화)

2. **Solution 7 (배움)** 2순위
   - "이번엔 자녀들이 배우는 여행이..." (고도화)

3. **Solution 2 (방법)** 3순위
   - "지난번 피드백 반영해서..." (구체화)

4. **Solution 1 (중요포인트)** 4순위
   - "지난번 대비 가격은..." (비교 제시)

#### 콜 흐름 (10분)
```
[1-2분] 따뜻한 인정
"선생님 지난번 여행 정말 좋으셨대요! 
 가족분들 다 행복해하셨다고 들었어요."

[2-4분] 업그레이드 제안
"그래서 이번엔 지난번과 좀 다르게 
 준비해봤는데..."

[4-6분] 구체적 이점
"이번 일정은 2개 도시가 추가되고, 
 혼잡 없는 VIP 객실이라서 
 지난번보다 훨씬 여유로우실 거예요."

[6-10분] 클로징
"어때요? 이번 여름이 딱 좋은 일정이 
 남아 있어요."
```
