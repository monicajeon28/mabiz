## 개요

### SEO 메타데이터의 역할
1. **검색 엔진**: 페이지 주제 파악 → 검색 결과 표시
2. **SNS**: Open Graph로 링크 공유 시 리치 미디어 표시
3. **사용자**: 검색 결과에서 클릭 의도 증대 (CTR +35-50%)
4. **심리학**: 권위성, 사회증명, 긴급성 심리 로직 반영

### 마비즈 도메인 정보
- **Primary Domain**: https://mabizcruisedot.com (파트너 CRM)
- **Organization**: 마비즈 크루즈닷파트너스
- **Locale**: ko_KR (한국)

### 1️⃣ Title 태그 (50-60자)

- **패턴**: [주요 키워드] — [부가 설명] (대시로 구분)
- **길이**: 50-60자 (Google SERP 기준)
- **심리학**: 권위성 + 즉시성 강조
- **키워드**: 좌측부터 중요도 순서

#### 페이지별 타이틀

| 페이지 | 타이틀 | 길이 | 심리학 |
|--------|--------|------|--------|
| **Landing** | 마비즈 크루즈닷파트너스 — 파트너 CRM | 28자 | 권위성(브랜드명) + 명확성 |
| **Join** | 파트너 가입 — 마비즈 크루즈닷파트너스 | 27자 | 행동 유도(가입) |
| **Register** | 회원가입 — 마비즈 크루즈닷파트너스 | 26자 | 행동 유도(회원가입) |
| **Dashboard** | 대시보드 — 마비즈 크루즈닷파트너스 | 25자 | 권위성 + 기능성 |
| **Settings** | 설정 — 마비즈 크루즈닷파트너스 | 22자 | 명확성 |

- **톤**: 전문적 + 친근 (한국형 마케팅)
- **심리학**: 사회증명 + 긴급성 + 즉시성

```
PASONA 프레임워크 적용:
P: 문제 제시 (현재 상황의 불편함)
A: 자극 (감정적 호소)
S: 솔루션 제시 (우리 서비스)
O: 오퍼 (구체적 혜택)
N: 좁혀진 범위 (긴급성)
A: 행동 유도 (CTA)

예시:
"고객관리 + 수당추적 [S] 을 한 곳에서 [O] | 무료 + 5분 시작 [N] | 지금 가입하세요 [A]"
```

### 4️⃣ Open Graph (OG) 메타 태그

#### 원칙
- **용도**: 페이스북, 인스타그램, 링크드인, 카톡 공유 시 리치 미디어 표시
- **핵심**: og:title, og:description, og:image, og:url, og:type
- **이미지**: 1200x630px (가로:세로 = 1.91:1)
- **심리학**: 시각적 인상 3초 내 결정

#### OG 메타 태그 구조
```html
<!-- og:title: 공유 시 제목 (55자 이내) -->
<meta property="og:title" content="마비즈 크루즈닷파트너스 — 파트너 CRM" />

<!-- og:description: 공유 시 설명 (125자 이내) -->
<meta property="og:description" content="크루즈닷 파트너 전용 CRM. 고객관리, 수당확인, 영업도구를 한 곳에서. 즉시 가능, 무료 사용." />

<!-- og:image: 공유 시 표시 이미지 (1200x630px) -->
<meta property="og:image" content="https://mabizcruisedot.com/og-image.png" />
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />
<meta property="og:image:alt" content="마비즈 크루즈닷파트너스 — 파트너 CRM 솔루션" />

<!-- og:url: 공유 페이지 정확한 URL -->
<meta property="og:url" content="https://mabizcruisedot.com/landing" />

<!-- og:type: 페이지 유형 -->
<meta property="og:type" content="website" />

<!-- og:site_name: 사이트 이름 -->
<meta property="og:site_name" content="마비즈 크루즈닷파트너스" />

<!-- og:locale: 언어 -->
<meta property="og:locale" content="ko_KR" />
```

#### 페이지별 OG 이미지 사양

| 페이지 | 이미지 파일 | 주요 요소 |
|--------|-----------|---------|

| **Join** | `/og-image-join.png` | 로고 + "무료 가입" + 색상(초록색) |

### PASONA 프레임워크 × 메타 태그

```
PASONA 단계별 메타 적용:

P (Problem): Description 상단에 "문제" 제시
  → "고객관리와 수당 확인을 따로 관리하고 있나요?"

  → "수동 수당 계산", "분산된 데이터"

S (Solution): Title에 "솔루션" 제시
  → "파트너 CRM — 한 곳에서 관리"

O (Offer): Description에 "구체적 혜택" 제시
  → "무료 + 5분 시작 + 자동 계산"

N (Narrow): Description에 "긴급성" 강조
  → "지금 가입하면" / "한정 시간"

A (Action): Title에 "행동" 암시
  → "가입", "시작", "신청"
```
