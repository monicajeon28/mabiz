**목표**: 병렬 6에이전트 학습으로 L2 고객의 심리, 저항 원인, 전환 전략 파악

## 📋 L2 렌즈 정의

- **렌즈명**: L2 준비 부담형
- **핵심 오해/Pain**: "준비 복잡할 것 같다"
- **고객 발화**: 
  - "비자 어떻게 해요?"
  - "여권 유효기간은 맞나?"
  - "짐은 뭘 가져가요?"
  - "예방접종 필요한가요?"
  - "준비할 게 너무 많을 것 같아..."
- **경험 수준**: 없음 (처음 크루즈)
- **전환 전략**: **준비 0시간 증명** (자동화, 체크리스트, 간소화)

- 고객의 불안감 → 자신감으로 전환하는 질문은?
- L2 고객에게 가장 효과적인 ROI 재정의 방식은?

**학습 자료**: `D:\mabiz-crm\docs\한영인`

### **Agent Beta: 모니카 10단계 이의 대응**
**질문**:
- L2 고객의 주요 이의 3가지는?
  1. "비자/여권 문제"
  2. "준비할 게 너무 많아"
  3. "시간이 부족해"
- 각 이의별 10단계 대응 스크립트는?
- "자동화" 개념을 고객에게 어떻게 설명할까?

**학습 자료**: `D:\mabiz-crm\docs\세일즈`

### **Agent Gamma: 조던벨포트 + 아비테 (통합)**
**질문**:
- L2 고객의 "준비 불안감"을 Pain으로 증폭하는 방법은?
- "혼자 준비하면 뭐가 빠질까?" → "우리와 함께하면 완벽해" 로 인지 전환하려면?
- Cognitive Dissonance (준비 복잡 vs 우리의 자동화)를 어떻게 활용할까?

**학습 자료**: 웹서치 + 이전 학습 내용

---

### **Agent Delta: SPIN 판매 + 심리학**
**질문**:
- SPIN의 "Implication" 단계에서 L2에 적용할 질문은?
  - "만약 준비하는데 실수가 생기면 어떻게 될까요?"
  - "시간을 못 내서 준비를 못하면?"
- 심리학: **Cognitive Load (인지부하)** 이론
  - 크루즈는 이미 복잡한데 + 준비까지? → 과부하 인지
  - 우리의 "자동화"가 이를 어떻게 해결하나?

**학습 자료**: SPIN 판매 기법 + Cognitive Load 이론

---

### **Agent Epsilon: 통합 톤/프레임워크**
**질문**:
- L1(가격)과 L2(준비)의 톤 차이는?
  - L1: "가격 > 가치" 전환 (PASONA 중심)
  - L2: "복잡 > 간소" 전환 (효율성 중심)
- L2 콜 스크립트의 PASONA+SPIN 통합 순서는?
- L2 SMS 시퀀스의 심리학 구조는?

**학습 자료**: L1 완성된 문서 + 이전 학습

---

### **Agent Zeta: L2 심리학 깊이**
**질문**:
- **1. 특징별 심리 메커니즘**
  - "준비 복잡해 보여요" → 시간부족? 능력부족? 정보부족?
  - 해당 고객들의 MBTI/인지 특성은?

- **2. 특징별 깨뜨리는 방법**
  - "복잡하다"는 인식을 "체계적"으로 변환하려면?

- **3. 감정 변화 맵** (불안 → 신뢰까지)
  - 각 단계에서 필요한 감정 요소는?

- **4. 우리가 피해야 할 말**
  - "쉬워요" (오히려 과소평가 인상)
  - "복잡하지 않아요" (부인이 아닌 해결책 필요)
  - 진정한 금지 표현은?

**학습 자료**: 심리학 이론 웹서치 + L1 템플릿 활용

## 📊 L2 vs L1 비교표

| 항목 | L1 (가격 오해) | L2 (준비 부담) |
|-----|-------------|-------------|
| **핵심 장벽** | 가격이 비싸 보임 | 준비가 복잡해 보임 |
| **심리 근원** | 지식부족 → 가격만 판단 | 시간부족 → 준비 불안 |
| **감정 색** | 의심 → 신뢰 | 불안 → 안도 |
| **콜 포인트** | 가치 재정의 | 자동화 증명 |
| **PASONA 비중** | 100% | 70% (A 빠짐) |
| **SPIN 비중** | 30% | 100% (전략 중심) |
| **메시지** | "가격 아니라 가치" | "준비 다 해드릴게" |
| **행동 심리** | 손실회피 (가격) | 불안 회피 (복잡도) |

## 🎬 L2 콜 스크립트 예상 구조 (6단계)

1. **Opening**: 공감 + 준비 자동화 안내
2. **Problem Discovery**: 고객의 실제 준비 불안감 파악
3. **Amplification**: 준비 실수의 결과 (VISA 거절? 맞지 않는 짐?)
4. **Solution Intro**: "우리가 준비 다 챙겨드려요"
5. **Automation Proof**: 체크리스트 + 자동 알림 시스템
6. **Close**: "준비는 우리가 담당, 당신은 휴식만"

---

## 💬 L2 주요 이의 예상 (3가지)

1. **"비자 처리 어려울 것 같아요"**
   - 대응: "저희가 비자 서류 다 준비해드려요"
   - 증명: 자동화 프로세스 + 성공 사례

2. **"짐 뭘 가져가야 할지 모르겠어요"**
   - 대응: "선박별 짐 체크리스트 있어요"
   - 증명: 카테고리별 자동화 리스트

3. **"시간 없는데 준비 가능해요?"**
   - 대응: "온라인으로 10분만 해도 돼요"
   - 증명: 자동화 + 모바일 앱 데모
