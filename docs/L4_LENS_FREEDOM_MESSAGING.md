## 📄 "약정 없음" 서식 (고객에게 제시)

### 서식 1: 예약 확인 문서

```markdown
═══════════════════════════════════════════════════════════
                  크루즈 예약 확인서
═══════════════════════════════════════════════════════════

예약자: OOO님
배: Royal Caribbean - 지중해 7박
출발일: 2026-07-15
귀항일: 2026-07-22
객실: 오션뷰 선실

═══════════════════════════════════════════════════════════
                    🎯 당신의 자유
═══════════════════════════════════════════════════════════

✅ 약정 기간: 없음 (0일)
   → 당신의 자유로운 여행입니다.

✅ 위약금: 0원
   → 여행 7일 전까지 무료 취소 가능합니다.

✅ 다음 예약: 당신의 선택
   → "또 가고 싶으면" 그때 다시 예약하세요.
   → "이번이 마지막" 그냥 끝나요.

✅ 구독 강제: 없음
   → 우리는 당신을 절대 묶지 않아요.

═══════════════════════════════════════════════════════════

이 예약은 당신의 자유로운 선택입니다.
우리는 당신의 자유를 100% 보장합니다.

                    마비즈 크루즈팀
```

### 서식 2: 취소 정책 명시 (고객 앱에 노출)

```
┌─────────────────────────────────────┐
│     예약 취소 정책 (마비즈 크루즈)    │
└─────────────────────────────────────┘

📅 여행 90일 이상 전:
   → 100% 환불 (수수료 0원)

📅 여행 90-60일 전:
   → 95% 환불 (수수료 5%)

📅 여행 60-30일 전:
   → 80% 환불 (수수료 20%)

📅 여행 30-7일 전:
   → 50% 환불 (수수료 50%)

📅 여행 7일 이내:
   → 환불 불가 (변경 가능)

─────────────────────────────────────
⭐ 우리의 약속:
   "당신의 자유를 보장합니다.
    언제든 취소, 언제든 선택."
```

## 💬 CallScript L4 (자유도 강조 버전)

### PHASE 1-6 핵심 문구 (자유도 반복)

```
PHASE 1: 오프닝
"당신이 걱정하는 '뭔가 묶이는 건 아닐까?'
그건 절대 없어요.
우리는 '선택지'만 드리고,
당신의 자유를 100% 존중합니다."

[포즈: 2초]

PHASE 2-3: 문제 + 증폭
"스포티파이나 피트니스처럼
'빠져나올 수 없는' 구독은 없어요.

당신은:
- 언제든 그만둘 수 있고
- 위약금도 없어요
- 다음에 또 하고 싶으면, 그때 다시 예약하면 돼요"

PHASE 4: 해결책
"우리의 시스템은 간단해요.

1) 당신이 크루즈 예약해요 (여행을 예약하는 거)
2) 여행을 즐겨요
3) 끝

그 다음에는?
- '또 가고 싶다' → 다시 예약해요
- '이번이 마지막' → 그냥 끝나요

우리는 당신을 '묶어두지 않아요'.
당신은 100% 자유예요."

PHASE 5: 신뢰 형성
"우리 고객들이 돌아와서 하는 말:
'아, 이 정도면 약정이 아니네'
'정말 자유롭네'
'다음에 또 할 때 편하겠다'"

[포즈: 2초]

PHASE 6: 클로징
"그래서 당신이 걱정하는
'뭔가 묶이는 건 아닐까?'
그건 절대 없어요.

이번 크루즈를 예약하면,
당신은 완전히 자유예요.

[포즈: 3초]

혹시 이제 마음이 조금 편해졌어요?
그럼 당신 가족에 딱 맞는 배,
오늘 정해보시겠어요?"
```
