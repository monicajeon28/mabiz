## 📊 L5 렌즈 고객 프로필

### 타겟 고객 정의

```
Primary Target:
├─ 비교 가능한 인구통계 특성 고객 (나이, 직업, 가족, 예산, 건강)
├─ 자신감 부족 / 사회적 증명에 약한 고객
└─ 첫 크루즈이면서 "혼자", "가족", "친구"의 의사결정 단계

Specific Personas from 20Lens:
├─ #1 신혼맞벌이 ("직장인 같은 사람들 후기 필요")
├─ #4 신혼저소득 ("월급 350만원 사람들은 어떻게 가?")
├─ #6 40대자녀초등 ("3자녀 가족은 가능해?")
├─ #9 40대한부모 ("혼자 가는 게 괜찮나?")
├─ #16 50대은퇴예정 ("내 나이에 맞는 분들?")
├─ #17 60대은퇴왕성 ("70대 분들도 이렇게 활발하나?")
└─ 모든 첫 크루즈 고객 (불안감이 있는 경우)
```

### 핵심 Pain Point

| 요소 | 설명 | 증거 |
|------|------|------|
| **자신감 부족** | "나도 할 수 있을까?" | "다들 크루즈 경험이 있으신가요?" |
| **혼자 불안감** | "혼자 가는 게 어색하지 않을까?" | "보통 몇 명이 가요?" |
| **가족 적합성** | "우리 가족에 맞을까?" | "애기 있는 가족도 많이 가나요?" |
| **나이대 적합성** | "내 나이에는 맞지 않을까?" | "우리 나이대에 맞는 사람들 많아요?" |
| **예산 적합성** | "우리 예산으로 괜찮을까?" | "월급 350만원 직장인도 가나요?" |
| **건강 우려** | "건강이 안 좋아도 괜찮을까?" | "당뇨병 있어도 가능한가요?" |
| **사교 불안** | "단체 활동이 어색하지 않을까?" | "아무도 모르는 사람들과 함께인데..." |

### 저항 패턴 분석

```
고객 발언 → 심리 상태 → 현행 대응 → 문제점 → L5 대응

"근데 보통 이거 경험 있는 사람들만 가는 거 아니에요?"
→ 자신감 부족, 선택지 배제
→ "아니요, 첫 크루즈 고객도 많습니다" (통계 제시)
→ 추상적, 구체적 사례 없음
→ "당신 같은 직장인 5,000명이 올해 처음 가요" (숫자+공감)

"혼자 가도 괜찮을까요?"
→ 사교 불안, 외로움 우려
→ "네, 혼자 가시는 분들도 있어요" (가능성)
→ 심리적 안정감 부족
→ "혼자 가신 30대 여성분들, 3박 후에 친구 3명 사귀셨어요" (구체 사례)

"우리 아이들도 즐거워할까요?"
→ 부모 책임감, 자녀 맞춤 불안
→ "어린이도 즐길 수 있는 프로그램이 있어요" (프로그램)
→ 감정 설득 부족
→ "10대 자녀 있는 부모 200명, 아이들이 제일 신나해요" (사례+증거)

"당뇨병 있는데도 괜찮을까요?"
→ 건강 우려, 신체 제약
→ "배에 의료진이 있으니 괜찮습니다" (정보)
→ 신뢰감 부족
→ "70대 당뇨/고혈압 있는 분들, 매년 오세요" (사례+반복)
```

## 🎯 L5 렌즈 완전 콜 스크립트 (Phase 0-4)

### 전략 프레임워크

```
심리학 원리: Social Proof by Similarity (상황별 유사인 증명)
           + Narrative Transportation (구체적 고객 사례)
           + Self-Projection (당신의 미래 그리기)
           + Comparison Effect (당신과 같은 사람들의 선택)

PASONA 매핑:
├─ Problem: "나도 이거 할 수 있을까?" (불안감 인식)
├─ Affinity: "당신 같은 분들이 정말 많아요" (공감+증명)
├─ Solution: "당신 직업/나이/가족 같은 분들이 이미 다 했어요" (구체화)
├─ Offer: "실제 후기 + 통계 + 구체적 사례" (신뢰 구축)
├─ Narrowing: "당신이 혼자가 아니라는 것" (심리 안정)
└─ Action: "당신 같은 분들의 선택 3가지 (혼자/가족/친구)" (행동)

SPIN 적용:
├─ Situation: "혹시 크루즈를 간 적이 있으신가요?" (경험 확인)
├─ Problem: "처음이신데 '나도 할 수 있을까?'라는 걱정 있으신가요?" (심리 문제 명확화)
├─ Implication: "만약 당신 나이대/직업/가족 같은 분들의 후기를 보면?" (해결책 암시)
└─ Need-Payoff: "당신도 할 수 있다는 확신 들어요?" (신뢰 구축)

신민형 5STEP (클로징):
├─ Step 1: "당신 같은 분들의 선택이 있어요"
├─ Step 2: "혼자 가신 분도, 가족이랑 가신 분도, 친구들이랑 가신 분도" (옵션)
├─ Step 3: "다들 '우리도 할 수 있겠네'라고 생각했어요" (손실 앵커)
├─ Step 4: "당신은 혼자? 가족? 친구? 어떤 방식이 편하신가요?" (선택)
└─ Step 5: "그럼 지금 예약 시작할까요?" (즉시행동)
```

### Phase 0: 오프닝 (0-1분)

**목표**: 자신감 부족 인식 + 사회적증명 힌트 제시  
**심리 원리**: Low Barrier Opening + Social Validation  
**PASONA 맵**: Problem 인식

#### 스크립트 A: 자신감 부족 공감

```
"안녕하세요! 크루즈닷 신민형입니다.

혹시 처음 크루즈를 생각해보고 계세요?

많은 분들이
'나도 할 수 있을까?'
'나 혼자 가도 괜찮을까?'
이렇게 걱정하세요.

그런데 비결이 있거든요.
당신 같은 사람들이 이미 다 했다는 거예요.
맞나요?"

[고객 반응 대기 - 30초]

"네, 이게 핵심인데요.
당신 나이대, 당신 직업, 당신 가족구성 같은 분들
이미 5,000명이 올해만 다녀갔어요."
```

#### 스크립트 B: 직업별 공감

```
"혹시 직장인이세요?

직장인 중에 크루즈 경험 있는 분 많은가? 하시면
요즘 직장인 82%가 이미 크루즈를 경험하셨어요.

남은 18%가 당신이실 수도 있다는 거죠.
맞나요?"
```

#### 스크립트 C: 가족 구성별 공감

```
"혹시 아이들이 있으세요?

아이들이 처음 크루즈 가면
'엄마, 이게 최고의 휴가네'라고 말해요.

실제로 3자녀 있는 부모들 200명,
매년 오세요."
```

#### 스크립트 D: 나이대별 공감

```
"혹시 60대 전후이세요?

이 나이대가 크루즈를 가장 즐기는 나이대예요.
왜냐하면
정년 후 시간 여유도 있고,
건강도 좋으신 황금기이거든요.

60대 은퇴하신 분들,
올해 45,000명이 크루즈를 타고 계세요."
```

### Phase 1: 욕구 발굴 (1-3분)

**목표**: 구체적 자신감 부족 영역 파악 (혼자/가족/나이/예산/건강)  
**심리 원리**: Diagnostic Listening + Empathy  
**PASONA 맵**: Affinity (공감)  
**SPIN 맵**: Situation + Problem

#### Q1: 경험 수준 확인

```
"혹시 해외여행을 많이 다니셨어요?
혹은 처음이신가요?"

├─ 많이 다님: "그러면 크루즈는 더 쉬워요. 호텔 같으니까"
└─ 처음: "그래서 당신 같은 분들을 위해 준비했어요"
```

#### Q2: 혼자/함께 의사결정

```
"혹은 혼자 가실 생각이신가요?
아니면 가족이나 친구들이랑 가실 생각인가요?"

├─ 혼자: "혼자 가신 분들이 제일 신나해요. 자유니까. 후기 보여드릴까요?"
├─ 가족: "가족이랑 가실 때 중요한 게 뭘까요? 아이 나이?"
└─ 친구: "친구들이랑 가실 때 가장 좋은 게 뭘까요?"
```

#### Q3: 자신감 장벽 파악

```
"혹시 '내가 할 수 있을까?' 하는 걱정이 있으신가요?
구체적으로는 뭐가 가장 불안해요?

[고객 선택 - 하나 이상 복수 선택 가능]

□ 혼자 가는 게 어색할까?
□ 우리 가족에 맞을까?
□ 우리 나이대에 맞을까?
□ 우리 예산으로 가능할까?
□ 건강 때문에 힘들지 않을까?
□ 단체 활동이 어색할까?"

[각 선택별 후기 제시 준비]
```

#### Q4: 우려사항 세분화

```
[가장 큰 우려 1가지 선택 후]

"당신처럼 [우려사항]이 있던 분들은
실제로 어떻게 되셨는지 보고 싶으세요?"

예)
- "혼자 가도 괜찮을까?" → "혼자 간 30대 여성 후기"
- "애기가 있어도 될까?" → "2-5세 자녀 있는 부모 후기"
- "70대도 괜찮을까?" → "70대 고령층 후기"
- "월급 350만원인데?" → "중산층 가족 후기"
- "당뇨병이 있는데?" → "건강 우려 있던 분 후기"
```

### Phase 2: 욕망 증폭 (3-6분)

**목표**: 당신 같은 사람들의 실제 사례로 신뢰 구축  
**심리 원리**: Narrative Transportation + Similarity-Attraction  
**PASONA 맵**: Solution (당신 같은 사람들이 했으니 가능)

#### 2-1: "당신 직업 같은 사람들"

**케이스 A: 직장인 (일반)**

```
"직장인 중에 처음 크루즈 가신 분 많은데요.

실제 통계 보시면
직장인 82%가 크루즈를 경험했어요.

그 이유가 뭘까 아세요?

'일 때문에 피곤한데,
배에만 올라타면
아무것도 하지 않아도 돼요.'

예를 들면
아침에 눈 떠서 바다 풍경,
무한정 먹을 수 있는 빔,
저녁에 쇼도 보고,
친구들도 만나고.

일주일간 아무 결정도 안 해도 돼요.

그래서 직장인들이 가장 많이 가죠."

[구체 사례 1명 제시]
30대 마케팅 과장, 박미영님
- 나이: 35세
- 직업: 광고 회사 마케팅 과장
- 동반: 남편, 아이 1명
- 예산: 1인 250만원대
- 후기: "처음엔 걱정했는데, 3박 후엔 '왜 더 안 자주 왔지?'라고 생각했어요."
```

**케이스 B: 자영업**

```
"자영업 하시는 분들은 어떨까요?

자영업 분들은
'언제 한번 쉬어야지...' 이러다가
크루즈를 가면
'진짜 일 못 생각해요.'

왜냐하면
배 위에선 와이파이도 비싸고,
신호도 안 좋고,
그래서 정말 집에서 멀어지는 거죠.

자영업 부부, 특히 40대가
가장 선호해요."

[구체 사례 1명 제시]
40대 식당 사장, 이정환님
- 나이: 48세
- 직업: 치킨집 사장
- 동반: 아내
- 예산: 1인 280만원대
- 후기: "10년 만에 아내랑 둘이만 쉬었어요. 배는 최고의 도피처예요."
```

**케이스 C: 공무원**

```
"공무원 분들 많이 가세요.

왜냐하면
정년이 정해져 있으니까
'지금 안 가면 언제 가나?' 이러거든요.

퇴직금이 나오면 그때 가려고 하는데
'아니다, 지금 가자. 정년 전에.'

그래서 공무원들, 40-50대가
가장 후회 없이 예약하세요."

[구체 사례 1명 제시]
50대 세무공무원, 김영수님
- 나이: 52세
- 직업: 세무서 과장
- 동반: 아내, 아들(14세)
- 예산: 1인 220만원대
- 후기: "정년이 3년 남았는데, 지금 가기를 잘했어요."
```

#### 2-2: "당신 나이 같은 사람들"

**케이스 A: 30대 신혼**

```
"30대 신혼 분들은요.

신혼 때 여행을 못 가면
아이가 생기고,
일이 생기고,
나중에 '우리도 할 수 있었겠네'라고 해요.

그래서 30대 신혼 분들은
'지금 아니면 언제?'라고 생각해요.

특히 맞벌이하는 30대 신혼 부부
월급 350만원대도
'이 가격이면 우리 생활비 한 달보다 크다?'
하면서도 가요.

왜냐하면
배에선 숙식이 다 포함되고,
추가 비용이 거의 없거든요."

[구체 사례 2명 제시]

케이스 1: 30대 신혼 맞벌이, 김준호 & 이선희 부부
- 나이: 33세, 32세
- 직업: 은행원, 마케팅
- 예산: 월급 1인당 350만원대
- 예약: 3박 650만원 (1인당 250만원대)
- 후기: "신혼 때 갈 수 있어서 다행이에요. 아이 생기면 언제 또 둘이만 가죠?"

케이스 2: 30대 신혼 저소득, 박준석 & 최민지 부부
- 나이: 30세, 29세
- 직업: 편의점 점장, 병원 행정직
- 예산: 월급 1인당 280만원대
- 예약: 3박 550만원 (분할 가능)
- 후기: "우린 신혼 때 그게 최고의 투자였어요."
```

**케이스 B: 40대 가족**

```
"40대 가족 분들은요.

아이들이 초등학교 다니는데
'마지막으로 우리 가족 다 함께 여행을 가자'
이 생각이 들어요.

중학교 가면
아이들이 친구들 때문에 못 떠나고,
고등학교 가면
입시 때문에 못 가거든요.
```
