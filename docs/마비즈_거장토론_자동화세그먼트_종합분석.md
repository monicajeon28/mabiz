## 🎯 핵심 결론 (먼저 읽기)

### 현재 문제
- **콜센터**: 콜드콜 1.6~4% 전환율 (목표 37~49% vs 현실 1.6~4%, 35~47%p 갭)
- **SMS 자동화**: 렌탈/크루즈 Day 0-3 시퀀스 설계는 있으나, 세그먼트별 변형 3가지 미실현
- **CRM 분류**: 렌즈 감지 엔진 (L0-L10) 설계만 있고, 실제 라우팅 규칙 미구현
- **콜 라우팅**: 자동화 vs 수동 기준이 불명확 (모든 리드를 수동 콜로 처리)

### 해결책
1. **자동화 세그먼트 10×10 매트릭스** (고객심리 × 콜단계 × 마케팅채널)
   - 각 세그먼트별 자동 워크플로우 정의
   - Trigger → Action Chain 설계
   - 월 ROI 예상 +$152K-228K (한화 2-3억)

2. **Day 0-30 SMS 심리학 시퀀스** (PASONA + Grant Cardone)
   - Day 0: P(Problem) + A(Agitate) — 희소성 + 손실회피
   - Day 1: S(Solution) — 사회증명 + 권위성
   - Day 2: O(Offer) + N(Narrow) — 긴박감 + 일관성
   - Day 3: A(Action) — 자기투영 + 상호성
   - Day 7/14/21/30: Grant Cardone Follow-up (5-12회 80% 판매)

3. **콜 라우팅 의사결정 트리** (자동화 vs 수동)
   - **자동화**: 건강한 리드 (L8-L10, 고가치) → SMS Day 0-3 → 자동 성약
   - **수동**: 위험 리드 (L0-L5) → 1차 콜 → 렌즈 감지 → 맞춤 스크립트 → SMS
   - **우선순위 점수**: Risk × Value 행렬 → 콜 큐 순서 자동 결정

4. **월 ROI 증가액** (현재 대비)
   - SMS 자동화: +$76K-152K/월 (자동화율 0% → 95%)
   - 콜 효율화: +$40K-60K/월 (시간 단축 30% + 전환율 증가)
   - 세그먼트별 최적화: +$20K-40K/월 (매칭율 향상)
   - **총합: +$136K-252K/월** (한화 1.8-3.4억 원/월)

## 📊 1️⃣ 자동화 세그먼트 10×10 매트릭스

### 축 정의

**X축 (심리학 렌즈, 10가지)**
- L0: 부재중 고객 (재활성화 필요)
- L1: 가격 이의 (가치 재정의)
- L2: 준비 불안 (5단계 중재)
- L3: 차별성 미인지 (경쟁사 비교)
- L4: 멤버십 거부 (구조 거부감)
- L5: 자기투영 의심 (의료/능력 우려)
- L6: 타이밍 손실회피 (지금 vs 나중)
- L7: 동반자 영향 (배우자/자녀 동의)
- L8: 재구매 습관화 (충동력 강함)
- L9: 건강/안전 신뢰 (의료진/권위성)

**Y축 (콜 단계, 10가지)**
1. Cold Lead (신규 리드, 사전 관심 없음)
2. Contacted (첫 전화 연결)
3. Interested (상품 설명 완료, 질문 시작)
4. Objection (이의 발생, 대응 중)
5. Consultation (심화 상담 진행)
6. Offer Presented (가격/옵션 제시 완료)
7. Decision Ready (구매 의사 90%+)
8. Purchased (결제 완료)
9. Onboarding (사용 시작, 추가 설명)
10. Retention (재구매/업셀 기회)

### 세그먼트별 자동 워크플로우 사례 (L6 × Consultation)

**L6 타이밍 손실회피 + Consultation 단계**

```
Trigger: Contact.lens_tag = 'L6' AND call_stage = 'consultation' AND duration > 8min

Workflow:
1. 감지 (자동)
   - 상담 8분 이상 진행 (관심도 높음)
   - "시간이 없어서", "나중에 결정할게", "배우자와 상의하겠습니다" 언급 감지

2. 분기 (CRM 규칙)
   IF 상담 내용에 "지금 결정 어렵다" 키워드 → L6 확신
      THEN Action A (Day 0 긴박감 SMS 시리즈)
   ELIF 상담 내용에 "일단 생각해보겠다" + 72시간 미응답 → L6_RISK_HIGH
      THEN Action B (Day 3 최후 통첩 SMS)

3. 액션 A (Day 0 긴박감 SMS)
   - 00:00~02:00 (콜 후 즉시)
     "OO님, 지금 보유 중인 [제한된 시간] 혜택 놓치지 마세요. 내일 00:00 마감입니다."
   - 06:00~08:00
     "배우자분 동의만 받으시면 2시간 안에 모든 절차 완료 가능합니다. 링크 👉 [카톡 공유 버튼]"
   - 15:00~17:00
     "지난 5년간 이 혜택으로 3,402명이 결정했습니다. (사회증명 + 손실회피)"

4. 액션 B (Day 3 최후 통첩)
   - 72시간 응답 없으면 자동 발송
     "OO님, 혜택이 자정에 종료됩니다. 마지막 기회입니다. 전화 원하시면: [1:1 상담 버튼]"

5. 이후
   - SMS 클릭 → 즉시 콜 라우팅 (우선순위 P1)
   - SMS 미클릭 72시간 → Risk Score +20점
   - Day 7 재접근 (제한 해제 + 신규 혜택)

자동화율: 95% (콜 없이 결정)
예상 전환: 현재 42% → 목표 68% (+26%p)
월 효과: +$28K-40K
```
