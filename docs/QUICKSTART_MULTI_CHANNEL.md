## 🎯 5분 안에 시작하기

### 1단계: Unified Composer 열기

마비즈 CRM 대시보드 → **📨 문자 CRM** → **Unified Composer**

### 2단계: 채널 선택

```
☑️ SMS           (₩50/건)
☑️ Kakao         (₩30/건)
☐ Email          (무료)

또는 "모든 채널 선택" 클릭
```

### 3단계: 메시지 작성

```
입력: "고객님 안녕하세요! 렌탈 30% 할인 중입니다. [링크]"

자동 변환:
├─ SMS:    "고객님 안녕하세요! 렌탈 30% 할인 중입니다..." (90자)
├─ Kakao:  "고객님 안녕하세요! 렌탈 30% 할인 중입니다. [링크]" (1000자)
└─ Email:  제목 + 본문으로 자동 분리
```

### 4단계: 수신 그룹 선택

```
☑️ VIP 고객      (250명)
☑️ 부재 고객     (1,200명)
☐ 신규 고객      (500명)

총 수신자: 1,450명
```

### 5단계: 발송

```
예상 비용: ₩45,250
├─ SMS:    250×2 × ₩50  = ₩25,000
├─ Kakao:  250×2 × ₩30  = ₩15,000
└─ Email:  250×2 × ₩0   = ₩5,250

[지금 발송] 또는 [예약 발송]
```

**완료!** ✅ 캠페인이 즉시 또는 예약 시간에 발송됩니다.

## 🔄 Day 0-3 자동화 설정

### 렌탈 부재 고객 재활성화 시나리오

#### Day 0: 긴박감 유도 (L6 타이밍 렌즈)

```
채널: SMS 100%

메시지:
"안녕하세요! 특가가 오늘까지만 30% 할인입니다. 지금 예약하세요!"

심리학: 손실회피 (오늘까지만!) + 긴박감
```

#### Day 1: 이의 대응 (PASONA S)

```
채널: SMS 60% + Kakao 40%

메시지:
"너무 비싸시나요? 이미 1,500명 이상 예약했습니다. [상세보기]"

심리학: 사회증명 (1,500명) + 긴박감
```

#### Day 2: 가치 강조 (PASONA O)

```
채널: Kakao 50% + SMS 40% + Email 10%

메시지:
"렌탈 고객 평균 ⭐⭐⭐⭐ (4.8점)

✅ 무료 배송 + 설치
✅ 30일 환불 보장
✅ 24시간 고객지원

[지금 예약하기]"

심리학: 권위성 (평점) + 신뢰도 향상
```

#### Day 3: 최종 액션 (PASONA A)

```
채널: Kakao 60% + Email 20% + SMS 20%

메시지:
"마지막 기회! 오늘 예약하면 추가 15% 할인.

[지금 예약] 또는 상담원 연결"

심리학: 희소성 (마지막) + 선택지 제공
```

### Day 0-3 자동 설정 (API)

```bash
curl -X POST https://api.mabiz.com/api/campaigns/multi-channel \
  -H "Content-Type: application/json" \
  -d '{
    "name": "렌탈 부재 고객 재활성화 - Day 0-3",
    "channels": ["SMS", "KAKAO"],
    "message": "안녕하세요! 특가가 오늘까지만 30% 할인입니다.",
    "recipients": [{"contactId": "con_123", "phone": "010-1234-5678"}],
    "lensType": "L6",
    "segmentId": "seg_inactive"
  }'

응답:
{
  "campaignId": "cmp_abc123",
  "status": "DRAFT",
  "estimatedCost": 1450
}
```

Day별 자동 채널 배치가 적용됩니다:
- Day 0: SMS 100%
- Day 1: SMS 60% + Kakao 40%
- Day 2: Kakao 50% + SMS 40% + Email 10%
- Day 3: Kakao 60% + Email 20% + SMS 20%

## 🚀 고급 기능

### A/B 테스트 (채널별)

```
실험 설정:
└─ 채널: SMS vs Kakao
└─ 그룹: VIP 100명 (각 50명)
└─ 메시지 A: "30% 할인!"
└─ 메시지 B: "한 명만 추가할 수 있습니다!"
└─ 측정 지표: 개방율 → 클릭율 → 전환율

결과 (3일 후):
SMS:
├─ 개방율: 25%
├─ 클릭율: 8%
└─ 전환율: 2%

Kakao:
├─ 개방율: 45%  ← 승자!
├─ 클릭율: 13.5%
└─ 전환율: 4%

결론: Kakao + 메시지 B 조합이 최고 성과
→ 다음 캠페인에 자동 적용
```

### 크로스채널 어트리뷰션

```
고객 여정:
Day 0 → SMS 개방 → SMS 클릭 (CLICK)
Day 1 → Kakao 개방 (TOUCH)
Day 2 → Email 개방 (TOUCH)
Day 3 → Kakao 클릭 → 전환 (CONVERSION)

어트리뷰션 모델:

First-Touch (첫 채널):
└─ SMS 100% 기여도

Last-Touch (마지막 채널):
└─ Kakao 100% 기여도

Time-Decay (시간 반영):
├─ SMS: 25% (가장 먼저)
├─ Kakao (Day 1): 35%
├─ Email: 15%
└─ Kakao (Day 3): 25% (가장 가까움)

Data-Driven (ML 학습):
├─ SMS: 20%
├─ Kakao: 60%
└─ Email: 20%

결론: Kakao가 가장 큰 기여 (전환 직전)
```

## ❓ FAQ

### Q1. SMS 하루에 몇 건까지 보낼 수 있나요?

**A.** 무제한이지만 발송 시간 제약이 있습니다.
- ✅ 주간 09:00 - 21:00
- ❌ 야간 21:00 - 09:00 (차단)
- ❌ 수신거부 고객 (opt-out)

수신거부 해제: 고객이 "080 unsubscribe" 회신 시 자동 처리

### Q2. 카카오 발송은 언제부터 가능한가요?

**A.** 준비 완료되었습니다.
- Aligo 계정에서 카카오 알림톡 활성화 필요
- 템플릿 검수 승인 필요 (1-2일)
- 설정 → SMS → Kakao 설정에서 활성화

### Q3. Email 발송 한도가 있나요?

**A.** 제한이 없습니다 (무료).
- 다만 스팸으로 분류될 수 있으므로 이메일 제목/내용 주의
- HTML 형식 권장 (클릭율 +15-30%)

### Q4. 예약 발송은 얼마나 정확한가요?

**A.** 분 단위 정확도입니다.
- 예: 2026-05-28 10:00 → 10:00:00 - 10:00:59 사이 발송
- 시스템 부하가 많으면 몇 초 지연 가능

### Q5. 크로스채널 어트리뷰션은 정말 정확한가요?

**A.** 초기(Phase 1)는 단순 모델입니다.
- First-Touch / Last-Touch / Time-Decay 제공
- Data-Driven (ML)은 Phase 3에서 제공
- 정확도는 발송량 증가에 따라 개선

### Q6. 어떤 채널부터 시작해야 하나요?

**A.** 추천 우선순위:
1. **SMS**: 이미 보유한 전화번호 활용, 빠른 효과
2. **Kakao**: 높은 개방율 (45%), 낮은 비용 (₩30)
3. **Email**: 신뢰성 중시 고객, VIP용

## 🆘 문제 해결

### 문제 1: "채널 선택이 안 되어요"

**원인**: 설정 미완료

**해결**:
1. 설정 → SMS 설정에서 Aligo 정보 입력
2. 설정 → Email 설정에서 발신자 정보 입력
3. 설정 → Kakao 설정에서 템플릿 승인 대기

### 문제 2: "발송 후 상태가 'FAILED'로 표시되어요"

**원인**: 
- 수신거부 번호
- 야간 발송 차단
- 잘못된 폰 번호

**해결**:
1. Contact에서 opt-out 여부 확인
2. 발송 시간 확인 (09:00-21:00)
3. 폰 번호 형식 확인 (010-1234-5678)

### 문제 3: "Kakao 개방율이 0%인데요?"

**원인**: 메트릭 추적 미구현

**상태**: Phase 2에서 해결 예정
- Aligo 콜백 통합
- 실시간 메트릭 집계
