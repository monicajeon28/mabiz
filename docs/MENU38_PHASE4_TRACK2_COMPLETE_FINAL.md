### 렌즈 정의
```
L10: 즉시 구매형 (Immediate Purchase Decision Delay)

├─ 핵심오해: "이미 원하는데 마지막 고민"
```

### 신민형 5STEP 클로징
```
Step 1️⃣ 주도권 탈취: "그럼 가시기로 하셨어요?"
       └─ Commitment 원리 → "네"라고 한 번 말하면 일관성 유지

Step 2️⃣ 비교 명확화: "크루즈 vs 일반여행, 어느 게 낫나?"
       └─ Framing 효과 → 선택지 2개로 집중력 복원

Step 3️⃣ 손실 앵커: "지금 못 가면 ₩54,000 손실, 5년간 ₩324,000"
       └─ Loss Aversion → 손실액을 구체적으로 계산

Step 4️⃣ 삼중선택: Option 1(지금)/2(1주일)/3(자료)
       └─ Paradox of Choice 해결 → 3개만 놓고 선택

Step 5️⃣ 즉시행동: 서명→링크→24시간 내 확정→3일 내 탑승
       └─ Immediate Action → 심리적 약속을 행동으로 전환
```

## 💡 L10이 다른 렌즈와 다른 이유

### L1-L9: "정보 부족" 문제 해결
```
고객: "크루즈가 비싸다" → L1
고객: "준비가 복잡하다" → L2
고객: "배와 호텔이 같은가?" → L3
고객: "약정처럼 묶이나?" → L4
고객: "나는 탈 수 있나?" → L5
고객: "언제 가야 하나?" → L6
고객: "배우자를 설득하려면?" → L7
고객: "이미 갔는데 왜?" → L8
고객: "배멀미/지병/안전은?" → L9

대응: "정보 제공" → "설득" → "교육"
```

### L10: "결정 지연" 문제 해결
```
고객: "이미 모든 정보를 알아봤는데, 지금 할까? 다음 달? 내년?"

특징:
├─ 정보: 충분함 (이미 배도 정함, 객실도 정함)
├─ 고민: "지금이 타이밍인가?"
└─ 의사: "이미 원하지만 마지막 고민"

대응: "정보 제공 (X)" → "결정 강제 (O)" → "신민형 5STEP"
```

### 심리학의 차이

```
L1-L9: Information Gap 해결
└─ PASONA (설득) + SPIN (질문) → 정보 제공

L10: Choice Paralysis 해결
└─ 신민형 5STEP (결정 강화) + Loss Aversion (손실극대화)
└─ "정보는 충분하니, 이제 선택만 하세요"
└─ 선택지를 3개로 축소 → 판단 쉽게
```
