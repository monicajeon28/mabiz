# 세일즈봇 QnA PASONA 통합 — Executive Summary

**작성**: 2026-05-19  
**프로젝트명**: Menu #38 세일즈봇 콜 스크립트 통합 (Phase 4)  
**목표**: 콜 전환율 +7-10%, 담당자 생산성 +17% 달성

---

## 🎯 핵심 요약

### What (무엇을 하는가?)
기존의 세일즈봇 QnA를 **PASONA Solution 7가지**와 **신민형 5Step 콜 스크립트**와 통합하여, 담당자가 콜 중 실시간으로 참고할 수 있는 구조화된 카테고리 기반 QnA 시스템 구축.

### Why (왜 하는가?)
```
현재 문제:
- 세일즈봇 QnA = "상품정보 + 예약절차 + 일반 Q&A"로만 분류
- 콜 스크립트와 독립적
- 담당자가 "지금 내가 어떤 심리학 기법을 써야 하나?" 를 판단 어려움

솔루션:
- QnA를 "욕구발굴(손실회피), 상품스토리(효율성), 신뢰감(배타성), 배움경험(의미)" 로 재구성
- 콜 Step별 필요한 카테고리 자동 제시
- 담당자가 심리학 기반 제안을 실시간으로 참고 가능

기대 효과:
- 전환율: 23% → 30% (+7%p)
- 통화 시간: 18분 → 15분 (-3분, 생산성 +17%)
- 고객 만족도: 3.8점 → 4.3점 (+0.5점)
- 연간 매출: +5,880만원
```

### How (어떻게 하는가?)

#### Phase A: QnA 설계 & 작성 (현재 완료)
```
✅ 4가지 신규 카테고리 설계
   - 욕구 발굴 (Solution 1, 5) × 15개 Q&A
   - 상품 스토리 (Solution 2, 3, 4) × 20개 Q&A
   - 신뢰감 형성 (Solution 6) × 10개 Q&A
   - 배움과 경험 (Solution 7) × 15개 Q&A
   → 총 60개 Q&A + 메타데이터 (심리학, 효과, 사용 시점)

✅ 콜 Step과 매핑
   - STEP 1 (오프닝) → 신뢰감 카테고리 (Q11-20)
   - STEP 2 (욕구 발굴) → 욕구발굴 카테고리 (Q1-5)
   - STEP 3-4 (상품 설명 & 가격) → 상품스토리 카테고리 (Q6-15)
   - STEP 5 (클로징) → 배움경험 카테고리 (Q21-25)

✅ 자동 이메일 시나리오 설계
   - "생각해볼게요" → 24h 후 욕구발굴 Q&A 자동 발송
   - 2일 후 → 배움경험 Q&A 감정적 재접촉
   - 목표: 재접촉율 35% 이상
```

#### Phase B: 기술 구현 (1주, 개발팀)
```
예상 작업:
- Prisma 스키마 추가 (2일)
  * SalesbotQnACategory, SalesbotQnA, SalesbotQnALog 모델
  * 마이그레이션 SQL 작성
  
- API 엔드포인트 (3일)
  * GET /api/salesbot/qna/categories (카테고리 조회)
  * GET /api/salesbot/qna/category/[id] (카테고리별 QnA)
  * POST /api/salesbot/qna/log (사용 로그)
  * POST /api/salesbot/chat (자동 응답)
  
- NLU 의도 분류 (2일)
  * Claude API 기반 Intent Classification
  * Fallback: 키워드 기반 분류
  
- 담당자 실시간 참고 UI (React) (2일)
  * 우측 패널 (SalesbotQnAPanel)
  * Step별 카테고리 자동 전환
  * QnA 클릭 로그

예상 일정: 10일 (병렬 작업)
예상 코드량: 500줄 (API + UI + NLU)
```

#### Phase C: 테스트 실행 (2주, 담당자)
```
A/B 테스트:
- Group A (Control): 모니카, 기존 스크립트 v1.0, 세일즈봇 미활성화
- Group B (Test): 신민형, 신규 스크립트 v2.0 + 세일즈봇 활성화

일정:
- Week 1 (5/26-30): 준비 & 교육
- Week 2-3 (6/2-13): 실행 (30콜씩)
- Week 4 (6/16): 분석 & 보고

통과 기준:
- Group B 전환율 > Group A + 7% (30% 이상)
- Group B 통화 길이 < 15분
- Group B 만족도 > 4.3점
```

#### Phase D: 최적화 & 배포 (1주)
```
- 데이터 분석: 어느 QnA가 가장 효과적?
- v2.5 스크립트: 최적화된 버전 작성
- 전사 배포: 모든 담당자 대상 (신입 교육 포함)
- 월간 리뷰: QnA 개선 루프 설정
```

## 📊 기대 효과 (상세)

### 1. 전환율 향상
```
기초 데이터:
- 현재 전환율: 23% (100콜 당 23건 예약)
- 목표 전환율: 30%

추진 메커니즘:
STEP 1 (신뢰감 강화)
  → Q16: "모니카님 같은 특별한 고객분들께만" (+3% 신뢰도)

STEP 2 (욕구 자극)
  → Q3: 감정적 이미지 그리기 (+5% 구매욕)
  → Q5: 시간 손실회피 강조 (+4% 긴급성)

STEP 3 (이견 처리)
  → Q8: 가격 설명 (포함 항목 명시) (+3% 가치 인식)
  → Q9/Q10: 세그먼트별 맞춤 (-2% 이견)

STEP 5 (감정적 클로징)
  → Q22: "인생 최고의 경험" (+4% 의미 부여)

총 전환율 향상: 23% → 30% (+7%p)

재무 임팩트:
- 월 100콜 기준
- 기존: 100콜 × 23% × 70만원 = 1,610만원
- 목표: 100콜 × 30% × 70만원 = 2,100만원
- 증분: 월 490만원 = 연 5,880만원
```

### 2. 담당자 생산성 향상
```
통화 시간 단축:
- 현재: 18분/콜 × 100콜 = 1,800분/월 (30시간)
- 목표: 15분/콜 × 100콜 = 1,500분/월 (25시간)
- 절감: 5시간/월 = 60시간/년

원인:
- 세일즈봇이 제안하는 QnA로 불필요한 설명 반복 제거
- 담당자가 더 정확한 정보 제공 가능
- 고객이 빠르게 의사결정 가능

추가 효과:
- 절감된 시간으로 월 50콜 추가 가능
- 추가 수익: 월 490만원 × 1.5 = 735만원 (추가 콜 효과)
```

### 3. 고객 만족도 향상
```
점수 향상:
- 현재: 3.8/5점
- 목표: 4.3/5점 (+0.5점)

기여 요소:
- 구조화된 설명 (명확성 +0.3점)
- 더 자신감 있는 톤 (신뢰도 +0.2점)
- 감정적 연결 (만족도 +0.2점)
- 개인화된 제안 (맞춤성 +0.15점)

재방문율 향상:
- 현재: 60% (재예약율)
- 목표: 65% (+5%p)
- 추가 수익: 월 50콜 × 65% × 70만원 = 2,275만원/년
```
