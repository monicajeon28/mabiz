## 📈 10렌즈 네트워크 (매핑)

### 렌즈 간 연관성

```
L1 (가격) ◄─── 정보 부족 ───► L3 (차별성)
│                               │
│                               │
└─── 교육 필요 ───────────── L4 (멤버)
     │                        │
     │                        │
     ├─── L2 (준비) ◄─── 막리(Ready) ───► L7 (동반자)
     │
     └─── 결정 불확실성
          │
          ├─── L5 (적합성) ◄─── "나도?" ───► L9 (건강)
          │
          └─── L6 (타이밍) ◄─── "언제?" ───► L8 (재구매)
               │
               └─── 선택회피 ───► L10 (즉시구매) ✨
```

### L10이 다른 렌즈와의 관계

```
L1-L9: 정보/교육/확신 부족 → "정보 제공" 필요
L10: 정보 충분하되 결정 지연 → "결정 강화" 필요

L1-L9 고객 → SMS: 설득/교육
L10 고객 → SMS: 손실극대화/긴급성 + 5STEP

L1-L9 예상 SMS 클릭율: 20-40%
L10 예상 SMS 클릭율: 60-18% (누적)

L1-L9 전환율: 25-50%
L10 전환율: 75%
```

## 💡 최종 통찰

### "10렌즈"의 본질

10렌즈는 **"고객의 고통점별 정확한 처방"**입니다.

```
기존: 모든 고객에게 같은 스크립트 (비효율)
└─ "준비가 복잡한데" 고객에게 "가격비교" 설명? (맞지 않음)

신규: 렌즈별 정확한 처방 (효율)
└─ "준비가 복잡한" 고객 → L2로 "시간 0시간" 강조
└─ "마지막 고민"하는 고객 → L10으로 "5STEP 결정" 강요
```

### "L10의 혁신성"

L10은 **"정보 문제가 아니라 결정 문제"를 해결**합니다.

```
L1-L9: "정보를 모르니까 못 산다"
└─ 해결: 더 많은 정보 + 교육

L10: "정보는 충분한데 결정을 못 한다"
└─ 해결: 선택회피 극복 + 결정 강제 (신민형 5STEP)
```

### "SMS의 역할 변화"

```
L1-L9 SMS: 정보 전달 (설득)
└─ "가격이 이 정도예요"
└─ "준비는 짐만 싸면 돼요"

L10 SMS: 심리 자극 (손실극대화)
└─ "지금 못 가면 ₩54,000 손실"
└─ "선실 3개 → 1개 → 0개 추적"
└─ "자정까지 마지막"
```
