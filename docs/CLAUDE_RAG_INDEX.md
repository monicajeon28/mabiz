  - 심리학 기반 재설득 기법

#### 1.3 클로징 8단계 (심리학 원리)
- [[grant_cardone_closing]] - 5-8단계 클로징 정의

  - 6가지 거절 시나리오 대응
  - Day 0-3 자동화 시퀀스 (SMS+콜 통합)

#### 1.4 Follow-up 실패 패턴
- [[grant_cardone_followup_mistakes]] - 10가지 실패 패턴

  - +320% 전환율 기대 효과
