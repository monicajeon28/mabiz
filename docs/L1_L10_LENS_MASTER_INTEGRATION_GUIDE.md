## 🎯 10렌즈 신속 식별 플로우차트

```
고객이 전화했을 때 첫 5초:

"안녕하세요, 크루즈 여행을 생각하고 계세요?"

→ 고객이 뭘 말하는지 들어보기

  1️⃣ "가격이 너무 비싼데..."
     → L1 (가격 오해형) ✅
     
  2️⃣ "준비가 복잡할 것 같은데..."
     → L2 (준비 부담형) ✅
     
  3️⃣ "호텔이랑 뭐가 달라요?"
     → L3 (차별성 미인지형) ✅
     
  4️⃣ "멤버십이 묶이는 건 아닐까..."
     → L4 (멤버십 저항형) ✅
     
  5️⃣ "내 같은 사람도 가도 돼?"
     → L5 (적합성 의심형) ✅
     
  6️⃣ "언제 가는 게 좋을까요?"
     → L6 (타이밍 미결형) ✅
     
  7️⃣ "배우자/가족이 동의를 안 해요"
     → L7 (동반자 이슈형) ✅
     
  8️⃣ "이미 한 번 갔는데 또 가야 해?"
     → L8 (재구매 유보형) ✅
     
  9️⃣ "나이가 많으면 힘들 것 같은데"
     → L9 (건강/안전 불안형) ✅
     
  🔟 "좋아요. 근데 정확히 어떻게 진행돼요?"
     → L10 (즉시 구매형) ✅
```

## 🔄 렌즈별 확인 흐름 (콜센터 매니저용)

```
고객 렌즈 분류 후 체크리스트:

L1 가격 오해 고객:
□ 올인클루시브 가치 3회 반복
□ 호텔 vs 크루즈 비용 비교 제시
□ "당신 페르소나별 가치" 제시
□ 페르소나 선택: 신혼/40대/중년/고령
□ 해당 페르소나 가격 재프레이밍 적용
□ SMS Day 0 자동 발송 확인

L2 준비 부담 고객:
□ "30분이면 준비 끝" 강조
□ 준비 시간 비교 (호텔 vs 크루즈)
□ 우리의 지원 범위 명시
□ 짐 체크리스트 제공
□ 타임라인 이미지 보여주기
□ SMS Day 1 체크리스트 발송

L3 차별성 미인지 고객:
□ "배 = 리조트" 개념 설명
□ 기항지 vs 배 위 시간 비교
□ 공간 비교 이미지 제시
□ 감정 곡선 설명 (마지막까지 좋음)
□ 4가지 이미지 설명
□ SMS Day 2 차별성 강조 발송

L4 멤버십 저항 고객:
□ "선택지 제공"이 핵심 메시지
□ 약정 없음 명시 (3회)
□ 위약금 0원 증명 (서류)
□ 취소 정책 상세 설명
□ 예약 확인서 제공
□ SMS Day 0-2 자유도 발송

L5 적합성 의심 고객:
□ 페르소나 파악: 신혼/40대/중년/고령
□ 해당 페르소나 사례 3개 제시
□ "당신 같은 사람들이..." 강조
□ 성공 확률 수치 제시
□ 페르소나별 스크립트 사용
□ SMS Day 0 "당신들이 가장 만족" 발송

L6 타이밍 미결 고객:
□ 손실 앵커 강조: 자리 줄어듦 / 가격 오름
□ 실시간 자리 현황 보여주기
□ "3주 후엔 다 찬다" 공포 심화
□ 가격 변동 타임라인 제시
□ 선택지 축소 이미지 제시
□ SMS Day 0-1 FOMO 발송 (자리 업데이트)

L7 동반자 이슈 고객:
□ 동반자 설득 키트 3장 제시
□ 배우자용 / 부모님용 / 친구용 구분
□ 당신이 할 말 시뮬레이션
□ 배우자와 함께 전화 권유
□ 혹은 문서로 설득 후 재통화
□ SMS Day 0-1 "함께 설득 준비" 발송

L8 재구매 유보 고객:
□ 연간 절약액 계산 제시
□ 호텔 vs 크루즈 5년/10년 비교
□ 포인트 누적액 확인
□ 재구매 할인 자동 적용
□ "이번이 마지막 가격" 강조
□ SMS Day 0-1 ROI 강조 발송

L9 건강/안전 불안 고객:
□ 의료진 자격 구체화 (M.D., RN 등)
□ 배의 의료 시설 상세 설명
□ 응급 대응 절차 명시
□ 배멀미 해결책 제시
□ 고령/지병 고객 성공 사례 제시
□ SMS Day 0-1 의료 안심 발송

L10 즉시 구매 고객:
□ 마지막 5가지 확인 (일정/객실/식사/보험/절차)
□ 삼중선택 제시: 3가지 객실 옵션
□ 고객이 선택하도록 유도
□ 최종 가격 제시
□ 신용카드 결제 진행
□ SMS Day 0 "마지막 확인 완료" 발송
```

## 💬 긴급 상황별 대응 (콜센터용)

```
만약 고객이 2개 이상 렌즈를 보인다면?

예시 1: L1 (가격) + L2 (준비)
→ 먼저 L1 대응 (가격)
→ 그 다음 L2 대응 (준비)
→ "가격도 저렴하고, 준비도 간단해요"

예시 2: L5 (적합성) + L7 (동반자)
→ L5 먼저: 당신은 맞는 고객
→ L7 나중: 배우자 설득 키트로 해결
→ "당신도 맞고, 배우자도 동의할 거예요"

예시 3: L6 (타이밍) + L10 (구매)
→ L6 먼저: 지금이 마지막 기회
→ L10 바로: 삼중선택으로 즉시 구매
→ "지금 결정하세요. 내일 늦어요"

이의 대응이 어려운 경우:
→ 콜센터 매니저 에스컬레이션
→ Monica 10단계 재교육 필요
→ 관리자 심급 고객 직접 콜 시도
```

THEN SendSMS('L2_Day0_Preparation', 즉시)

THEN SendSMS('L5_Newlyweds_Day0', 즉시)

IF Contact.Lens = 'L6'
THEN SendSMS('L6_Day0_Scarcity', 즉시)
     + SendSMS('L6_Day1_UpdatedSeats', 1일 후)
