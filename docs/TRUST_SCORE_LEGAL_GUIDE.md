## 🎯 1️⃣ PHASE 1: 약관 개정안 작성

### 1-1. 신뢰도 점수 정의 (초등학생 수준)

**약관에 이렇게 써요:**

```
┌────────────────────────────────────────────┐
│ "신뢰도 점수"는 뭔가요?                   │
├────────────────────────────────────────────┤
│                                            │
│ "신뢰도 점수"는 우리 회사에서               │
│ 상품을 팔아준 파트너 분들이                 │
│ "얼마나 제대로 일을 하는가?"를             │
│ 점수로 나타낸 것입니다.                    │
│                                            │
│ 점수는 이 3가지로 정해져요:                │
│ ✓ 환불율 (고객이 반품한 비율)             │
│ ✓ 판매건수 (팔아준 상품 수)                │
│ ✓ 기타 위반사항 (약관 위반 3회+)           │
│                                            │
│ 점수가 낮으면:                            │
│ 1️⃣ 30점 이하: 경고 통보                 │
│ 2️⃣ 35점 이하: 활동 제한 (1개월)         │
│ 3️⃣ 40점 이하: 활동 정지 (영구)          │
│                                            │
└────────────────────────────────────────────┘
```

### 1-2. 환불율 계산 방식 (초등학생 수준)

**약관에 이렇게 써요:**

```
┌────────────────────────────────────────────┐
│ 환불율은 어떻게 계산하나요?               │
├────────────────────────────────────────────┤
│                                            │
│ 환불율 = 반품된 상품 / 총 판매상품 × 100 │
│                                            │
│ 예시:                                     │
│ • 1월에 10개 팔았어요 → 2개 반품 됨       │
│   환불율 = 2 / 10 × 100 = 20%           │
│                                            │
│ • 2월에 20개 팔았어요 → 8개 반품 됨       │
│   환불율 = 8 / 20 × 100 = 40%           │
│                                            │
│ • 평균 환불율 = (20% + 40%) / 2 = 30% │
│                                            │
│ ⚠️  30% 넘으면 경고를 받습니다!          │
│                                            │
└────────────────────────────────────────────┘
```

### 1-3. 신뢰도 점수 계산 방식 (초등학생 수준)

**약관에 이렇게 써요:**

```
┌────────────────────────────────────────────┐
│ 신뢰도 점수는 어떻게 계산하나요?          │
├────────────────────────────────────────────┤
│                                            │
│ 기본점수 (100점)에서 감점합니다:          │
│                                            │
│ ① 환불율에 따른 감점                     │
│   • 20% 이상: -5점                        │
│   • 30% 이상: -10점                       │
│   • 40% 이상: -20점                       │
│                                            │
│ ② 판매건수에 따른 가산                    │
│   • 월 10건 이상: +5점 (보너스)          │
│   • 월 50건 이상: +10점 (보너스)         │
│                                            │
│ ③ 약관 위반에 따른 감점                   │
│   • 위반 1회: -5점                        │
│   • 위반 2회: -10점                       │
│   • 위반 3회 이상: -20점                  │
│                                            │
│ 예시 계산:                                │
│ 100(기본) - 10(환불) + 5(판매) = 95점  │
│ → "녹색" 상태 ✅ (활동 계속 가능)        │
│                                            │
│ 100(기본) - 20(환불) - 5(위반) = 75점  │
│ → "노란색" 상태 ⚠️ (주의 필요)           │
│                                            │
│ 100(기본) - 20(환불) - 15(위반) = 65점 │
│ → "빨간색" 상태 🔴 (활동 제한/정지)     │
│                                            │
└────────────────────────────────────────────┘
```

## ⚖️ 3️⃣ PHASE 3: 변호사 검토

### 3-1. 변호사 체크리스트 (검토 항목)

변호사에게 검토를 요청할 때 이 4가지를 꼭 확인해달라고 하세요:

#### ✅ 1️⃣ 약관의 법적 타당성

```
□ 신뢰도 시스템이 법을 어기지 않나요?
  → 확인사항:
     • 파트너가 불합리하게 피해를 입지는 않나?
     • 점수 계산 방식이 공정한가?
     • 이의 제기 절차가 충분한가?

□ 조치가 과도하지 않나요?
  → 확인사항:
     • "활동 정지"는 너무 심하지 않나? (파트너 생계 문제)
     • 1개월 "제한"은 적절한가?
     • 복구 가능성이 있나?
```

#### ✅ 2️⃣ 소비자보호법 준수

```
□ 소비자 피해가 없나요?
  → 확인사항:
     • 고객이 "파트너 신뢰도"를 알 수 있나?
     • 신뢰도가 낮은 파트너 상품을 표시하나?
     • 환불 시에도 신뢰도 영향을 미치나?

□ 환불 규정에 문제는 없나요?
  → 확인사항:
     • 환불율이 높은 파트너를 제한하면,
       고객이 환불받기 어려워지지 않나?
     • 고객의 환불권을 침해하지는 않나?
```

#### ✅ 3️⃣ 통신판매법 준수

```
□ 개인정보 보호에 문제는 없나요?
  → 확인사항:
     • 파트너의 "신뢰도" 정보가 고객에게 노출되나?
     • 그렇다면 파트너 동의가 필요한가?
     • 점수 계산에 민감한 개인정보(환불율 등)가 사용되나?

□ 약관 공지가 충분한가?
  → 확인사항:
     • 이메일 공지 30일이 충분한가?
     • 동의 절차가 명확한가?
     • "거부" 선택지는 왜 없나?
       (거부 불가의 법적 근거가 있나?)
```

#### ✅ 4️⃣ 개인정보보호법 준수

```
□ 민감한 데이터 처리에 문제는 없나요?
  → 확인사항:
     • 환불율, 판매건수 등이 "민감정보"인가?
     • 장기 보관이 필요한가?
     • 파트너가 자기 정보 열람/수정/삭제 권리를 가지나?

□ 데이터 보안에 문제는 없나요?
  → 확인사항:
     • 점수 계산이 공개되나?
     • 해킹되면 어떻게 하나?
     • 백업은 안전한가?

□ 동의와 거부가 명확한가?
  → 확인사항:
     • "동의" 버튼과 "거부" 버튼이 같은 크기?
     • 아니면 "동의"가 크고 "거부"가 작나?
       (부당 유도 금지)
```

## 📊 첨부: 신뢰도 시스템 계산식 (변호사용)

### 공식

```
신뢰도 점수 = 100 - 환불감점 - 위반감점 + 실적보너스

환불감점:
  • 환불율 20%~29%: -5점
  • 환불율 30%~39%: -10점
  • 환불율 40%+: -20점

위반감점:
  • 약관 위반 1회: -5점
  • 약관 위반 2회: -10점
  • 약관 위반 3회 이상: -20점

실적보너스:
  • 월 판매 10~49건: +5점
  • 월 판매 50건+: +10점
  • 고객평가 4.5/5.0+: +5점

최종 점수 범위: 0~120점
```

### 계산 예시

```
파트너 A: 
- 환불율 25% → -5점
- 약관 위반 0회 → 0점
- 월 판매 30건 → +5점
= 100 - 5 + 5 = 100점 (우수 파트너)

파트너 B:
- 환불율 35% → -10점
- 약관 위반 1회 → -5점
- 월 판매 5건 → 0점
= 100 - 10 - 5 = 85점 (정상)

파트너 C:
- 환불율 45% → -20점
- 약관 위반 2회 → -10점
- 월 판매 0건 → 0점
= 100 - 20 - 10 = 70점 (경고 단계)

파트너 D:
- 환불율 50% → -20점
- 약관 위반 3회 이상 → -20점
- 월 판매 0건 → 0점
= 100 - 20 - 20 = 60점 (활동 제한 단계)
```
