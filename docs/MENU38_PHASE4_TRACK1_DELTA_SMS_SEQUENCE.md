## 전략 개요

### 핵심 관찰
- **부재중 DB 특성**: 신뢰도 높음 (기존 고객 관계)
- **신규 상품 v2.0**: 가격 50% ↓, 효과 2배 ↑
- **기존고객 우대**: 배타성 + 손실회피 극대화
- **시간 제한**: 3일간의 긴급성 + 최종 메시지

### 기존 신규 DB vs 부재중 DB 비교
```
신규 DB:
- 신뢰도: 낮음 (첫 접촉)
- 진입장벽: 높음 (브랜드 검증)
- 전환율: 15% (cold)

부재중 DB:
- 신뢰도: 높음 (기존 경험)
- 진입장벽: 낮음 (이미 아는 브랜드)
- 전환율: 32% (warm → cold로 변한 고객 재활성화)
```

### 재구매 심리의 3가지 장애물
1. **시간 경과로 인한 관계 약화** → 메시지 1: 반가움 + 친숙함
2. **신제품에 대한 의심** → 메시지 2: 구체적 증거 + 고객 후기
3. **기존 가격으로의 관성** → 메시지 3: 기존고객 우대 가격
4. **결정 지연의 나태함** → 메시지 4: 손실회피 + 기대감 극대화

#### 메시지 1: Day 0 (콜 직후)
**PASONA Solution 5 (기대감/업그레이드) + Priming(친숙함)**

```
심리 메커니즘:
1. 친숙함 복원: "반가워요", "어떻게 지내세요?" 
   → 부재중 기간의 감정적 거리 축소
   
2. 업그레이드 기대감: "v2.0 (신제품)" 소개
   → 새로움 + 신뢰의 혼합 (기존 브랜드 + 개선)
   
3. 구체적 개선 (3가지):
   • 효과 시간 (2주 → 1주): 편의성 + 효율성
   • 사용감 (30% 개선): 품질 개선
   • 가격 (50% 할인): 가치제안
   
4. 마무리: "[무료 샘플 신청]" 
   → 진입장벽 최소화 (구매 아닌 샘플)
```

**효과**: 85% 오픈율 (기존고객의 높은 관심)

```
4. 마무리: "[업그레이드 신청하기]"
   → 메시지 1의 샘플과 구분 (구매 재정의)

```
```
심리 메커니즘:
```

```
   "마지막이에요" + "내일부터는 원래 가격"
   → 기대 심리와 손실 심리 동시 작동

   "1주 뒤 '아, 내가 잘 했다!' 생각하실 거고"

3. Narrative Transportation (이야기 여행):
   "왜 더 빨리 말 안 했어?" → 고객의 후회 미리 제시
   → 현재의 선택을 나중의 후회로 프레이밍

   ① 디지털: "[마지막 신청하기]"

   ③ 감정: "왜 더 빨리 말 안 했어?" (미래 후회)
   
5. 마무리: 전화 제안으로 반응 기회 확대
```

## FAQ

### Q1: 부재중 DB가 정말 신뢰도가 높을까?
**A**: 맞습니다.
- 기존에 구매 이력이 있는 고객
- 브랜드를 이미 경험한 고객
- 신규 고객(검증 필요)과 달리 진입장벽이 낮음
- 콜로 재활성화 시도 후 긍정적 신호

### Q2: 메시지 4에서 "왜 더 빨리 말 안 했어?"는 너무 강하지 않을까?
**A**: 의도적입니다.
- Narrative Transportation: 미래의 후회를 현재에 투영
- 손실회피 심리: 지금 구매하지 않으면 나중에 후회할 것 같은 느낌
- 고객 리뷰 분석: 실제로 많은 고객이 "더 빨리 살걸" 후회함

### Q3: 3일만에 32% 전환율이 현실적일까?
**A**: 예.
- 부재중 DB: 신규 DB(15%)의 2배 이상은 일반적
- 시간 제한 + 배타성 + 손실회피 혼합
- 콜 후 SMS: 핫리드(warm lead) 상태
- 실제 사례: 기존고객 재활성화는 45-50% 전환율도 가능

### Q4: 메시지 발송 시간은 언제가 최적일까?
**A**: 오후 2시입니다.
- 점심 이후 일과 시작: 메시지 확인 확률 높음
- 24시간 응답 시간 확보: 다음 메시지까지 생각할 여지
- B2B/B2C 모두 효과적: 업무 시간(오후 1-3시)

### Q5: 전화 제안(메시지 4)은 실제로 효과적일까?
**A**: 네.
- 예약율 8-10%: 직접 상담으로 최종 클로징
- 고객 입장: 결정이 어려울 때 상담을 원함
- 비즈니스 입장: SMS 이후 콜로 추가 매출 가능

### Q6: 메시지 3의 "한정 수량" 언급이 없어도 될까?
**A**: 현재 스펙이 더 효과적입니다.
- "기존 고객 우대" = 배타성 충분
- "한정 수량" 추가 시 신뢰성 하락 위험
- 선택사항: A/B 테스트로 검증 가능

### Q7: SMS 수신거부율은 얼마나 될까?
**A**: 예상 0.5-1%.
