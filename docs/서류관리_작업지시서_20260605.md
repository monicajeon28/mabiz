# 서류관리 시스템 개편 작업지시서 (2026-06-05)

## 배경
참고자료 `_document-management-ui-ux` 의도 + 사용자 요구를 반영해 4종 서류를
"바로 만들고 PNG 다운로드"할 수 있는 수동+자동 통합 UI로 개편한다.

## 확정 결정사항
1. 비교견적/구매확인/환불 3종 모두 **화면 미리보기 → PNG 다운로드 + 이메일** 둘 다.
2. 주문선택(자동) + 직접입력(수동) 이중 모드. 자동완성 소스 = `/api/documents/search-sales?q=`(이름검색 지원).
3. 계약서: ① 샘플양식 미리보기 ② 이름 자동완성→클릭선택 ③ **특약사항 제거** ④ 복수선택→각 건 개별 발급(클라 순차 POST).
