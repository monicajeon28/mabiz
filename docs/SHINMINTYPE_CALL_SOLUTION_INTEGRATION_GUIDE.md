## Step별 상세 스크립트

### STEP 1: 인사 & 신뢰 형성 (Solution 6 - 비밀스런 분위기)

#### Goal
- 경계심 해제
- "이 사람은 다른 여행사와 다르다"는 인식 형성
- 호기심 자극으로 적극적 경청 유도

#### 심리학 원리
- **Reciprocity** (호혜성): "특별한 정보"를 먼저 제시 → 보답 욕구 생성
- **Insider Status** (내부자 지위): "몰래 알려주는" 프레임 → 우월감 제공

#### 스크립트

**변형 1: 신규 고객 대상**
```
"안녕하세요, ○○입니다. 
요즘 크루즈 시장에 큰 변화가 있는 거 아세요?

여행사에서는 잘 말 안 하는데, 
최근 크루즈 가격 구조가 완전히 달라졌어요.

제가 몰래 알려드릴게 있는데, 
이 정보가 당신의 여행 결정에 엄청 도움이 될 거 같아요.

혹시 지금 크루즈 여행 생각 중이세요?"
```

**변형 2: 다시 전화한 고객 대상**
```
"안녕하세요, 지난번 ○○님과 얘기했던 ○○입니다.

그때 얘기가 계속 맴도는 부분이 있어서 
한 가지 더 알려드리고 싶어서 연락했어요.

여행사 동료들도 몰래 말하는 건데, 
요즘 5월~6월 시즌에는 이런 게 큰 메리트가 있거든요.

혹시 괜찮으시면 2분만 들어보실래요?"
```

**변형 3: 추천 고객 대상**
```
"안녕하세요, ○○님 추천으로 전화드렸던 ○○입니다.

○○님이 저한테 했던 얘기가 있는데,
'너는 다른 여행사와 달리 뭔가 다르더라'고 하셨어요.

비결이 뭔지 아세요? 
고객 입장에서 먼저 생각하는 거예요.

그래서 ○○님 같은 분들이 좋아해 주시는 것 같아요."
```

#### 주의사항
- ❌ "많이 팔려서 추천한다" (신뢰도 ↓)
- ❌ "지금 특가라서 서둘러야 한다" (거짓 희소성)
- ✅ "시장 변화에 대한 정보" (사실 기반)
- ✅ "내부자 정보" (우월감 제공)

#### KPI
- **응답률**: 80% 이상 (계속 전화 받으려고 함)
- **전환 신호**: "그런데 구체적으로 뭔데요?" (호기심 자극 성공)

### STEP 2: 욕구 발굴 (Solution 1 - 중요포인트 지적 + Solution 5 - 기대감)

#### Goal
- 고객의 **현재 고통점** 명확화
- "이 사람이 나를 이해한다"는 신뢰 형성
- 감정적 공감으로 심리적 거리 축소

#### 심리학 원리
- **Loss Aversion** (손실 회피): 고통을 명시적으로 언급 → 해결 욕구 증가
- **Empathy** (공감): "많은 분들이 같은 거예요" → 정상화 + 안심

#### 스크립트

**Phase 2-1: 현 상황 파악 (이전 여행 경험 물어보기)**

```
"먼저, ○○님은 지금까지 여행을 어떻게 다녀오셨어요?
주로 항공사로요, 아니면 다른 방식도 해보셨어요?"

🔍 고객 응답 분석:
- "항공사로만 다녔어요" → [Type A: 항공사 피로 고객]
- "투어 다녔어요" → [Type B: 시간 제약 고객]
- "개별 자유여행" → [Type C: 비용 부담 고객]
```

**Phase 2-2: 고통점 명확화 (Solution 1 활용)**

| 고객 Type | Step | 카피 |
|----------|------|------|
| Type A | 질문 | "그럼 지금까지 다닐 때, 가장 힘들었던 부분이 뭐였어요?" |
| | 자극 | "항공기 시간, 피로도, 숙박/식사 준비... 이런 거 맞죠?" |
| | 중요포인트 | "**이런 부분들이 실은 여행의 50%를 차지한다는 거 아세요?**" |
| Type B | 질문 | "투어 다니실 때 가장 답답했던 게 뭐였어요?" |
| | 자극 | "정해진 일정, 자유로운 시간 부족, 쇼핑 시간 부족... 맞나요?" |
| | 중요포인트 | "**사실 여행이란 시간이 자유로울 때 빛난다는 거 아시죠?**" |
| Type C | 질문 | "개별 여행할 때 제일 번거로웠던 게 뭐였어요?" |
| | 자극 | "숙박 예약, 식당 찾기, 교통편 구성... 복잡하지 않았어요?" |
| | 중요포인트 | "**결국 여행 준비가 70% 이상이라는 거 아시죠?**" |

**Phase 2-3: 감정적 공감 (Solution 5 활용)**

```
"많은 분들이 같은 거 겪으세요. 
사실 저도 처음엔 항공사로만 다녔는데, 
마지막 여행에서는 정말 피곤했거든요.

그런데 이 문제가 크루즈로는 **완전히 달라진다**는 걸 경험하고,
'아, 이게 진짜 휴가구나'라는 생각했어요.

○○님도 충분히 그런 경험을 할 수 있거든요."

💡 핵심: 
- "많은 분들" = 정상화 (당신만 그런 게 아니다)
- "저도" = 공감 (나도 경험했다)
- "완전히 달라진다" = 기대감 (해결 가능하다)
```

#### 체크리스트
- [ ] 고객의 **구체적인 고통점** 3가지 이상 도출
- [ ] 고객이 "네, 맞아요" 3회 이상 공감 표현
- [ ] 고객의 감정 변화: 경계심 → 호기심 (관찰)

#### KPI
- **공감 신호**: "그런데 그게 어떻게 가능한데요?" (해결 욕구 생성)
- **감정 신호**: "아, 그렇군요..." (안도감 표현)

### STEP 3: 해결책 제시 (Solution 2 - 방법 + Solution 3 - 간단함)

#### Goal
- **구체적 일정**을 상상하게 만들기
- **진입장벽** 제거 (복잡해 보이는 것을 단순화)
- "내가 이걸 할 수 있겠다"는 자신감 형성

#### 심리학 원리
- **Narrative Transportation** (이야기 빠져들기): 일정을 스토리로 전개
- **Effort Reduction** (노력 최소화): 복잡해 보이는 것을 단순화

#### 스크립트

**Phase 3-1: 방법 구체화 (Solution 2 활용)**

```
"○○님이 7박으로 다녀오신다고 하면,
하루하루가 어떻게 되는지 제가 그려드려볼까요?"

[예시: 제주-오사카-나가사키-홍콩]

Day 1) 부산 출발 (저녁 6시)
  → 가족과 함께 선상 디너 (배 흔들림? 거의 없어요)
  → 한눈에 보이는 야경 (사진 찍기 좋음)
  
Day 2) 제주 도착 (아침 8시)
  → 가족 자유롭게 해변 또는 쇼핑 (3~4시간)
  → 오후 4시 배 출발

Day 3) 해상일 (선상에서의 하루)
  → 아침: 풀장, 스파, 운동 (편하게 쉬기)
  → 오후: 아이들 클래스, 어른들 카지노
  → 저녁: 가족 디너쇼 (매일 다름)

Day 4) 나가사키 도착
  → 글로버 정원, 평화공원 (역사 배우기)
  → 현지 음식 맛보기

Day 5) 오사카
  → 도톤보리 거리 자유 관광
  → 일본 쇼핑 문화 체험

Day 6~7) 홍콩
  → 스타페리, 빅토리아 피크
  → 야경 감상

마지막 Day) 돌아오기
  → 배 안에서 편하게 쉬다가 부산 도착

이런 식으로 흘러가는 거거든요.
어때 보이세요?"
```

**Phase 3-2: 진입장벽 제거 (Solution 3 활용)**

```
"혹시 이렇게 복잡하면 어떻게 다 준비하지? 
이렇게 생각하실까봐 미리 말씀드릴게요.

사실 크루즈 여행은 **예약만 하면 나머지는 다 준비된 거거든요**.

✓ 비자? 필요 없거나 배 위에서 처리
✓ 음식? 먹고 싶을 때 먹을 수 있게 준비돼 있음
✓ 이동? 배가 움직이니까 호텔 이사 필요 없음
✓ 일정? 미리 짜여 있지만, 자유롭게 수정 가능

결국 여권 + 예약 = 준비 완료인 거죠.
어려울 게 정말 없어요."
```

#### 스토리텔링 팁
- ✅ **Day별로 진행**: "아침에 ~하고, 오후에 ~하고" (시간선 명확)
- ✅ **감각적 표현**: "야경이 얼마나 아름다운지", "현지 음식 맛" (상상 자극)
- ✅ **가족 중심**: "아이들은 이때 이렇게 즐거워하고" (현장감)
- ❌ **추상적 표현**: "좋은 경험을 하실 거예요" (상상 못 함)
- ❌ **준비 복잡성 강조**: "비자 처리, 숙박 예약..." (진입장벽 상승)

### STEP 4: 가격 제시 & 이견 처리 (Solution 4 - 효율성 + Solution 3 - 간단함)

#### Goal
- **가격 거부감** 제거
- **비용 대비 효과** 극대화
- 이견 처리로 최종 결정 유도

#### 심리학 원리
- **Cost-Benefit Analysis** (비용 대비 이득): 같은 돈으로 훨씬 큰 경험
- **Effort Reduction** (노력 최소화): 추가 비용/복잡성 없음을 강조

#### 스크립트

**Phase 4-1: 가격 제시 + 비교 (Solution 4 활용)**

```
"○○님 가족 4명이 7박을 간다고 하면,
다른 방식과 비교해볼까요?

【항공사 + 호텔 일정】
- 항공권: 50만 × 4명 = 200만 원
- 호텔(3박): 30만 × 3박 = 90만 원
- 호텔(4박): 30만 × 4박 = 120만 원  
- 식사비: 일일 평균 15만 × 7일 = 105만 원
- 교통비 (현지): 일일 10만 × 5일 = 50만 원
─────────────────────────────
소계: 약 565만 원 (배고픔, 피로도는 덤)

【크루즈 패키지】
- 크루즈료: 1인 150만 × 4명 = 600만 원
- 추가비용: 거의 0 (숙박, 식사, 이동 다 포함)
─────────────────────────────
소계: 600만 원

💡 결과:
- 가격은 5.6% 더 내지만
- 받는 경험은 **2배 이상**
- 피로도는 **70% 감소**

역산하면 여행 준비에 드는 스트레스까지 없으니까,
사실 훨씬 저렴한 거죠."
```

**Phase 4-2: 이견 처리 (Solution 3 활용)**

| 고객 이견 | 처리 카피 |
|----------|---------|
| **"너무 비싼데?"** | "어? 그런데 항공사로 다니는 것도 이 정도인데, 크루즈는 숙박 이사 안 하고, 차 안 빌려도 돼요. 훨씬 저렴하다고 봐요" |
| **"5월은 비수기 아닌가?"** | "5월이 **베스트 시즌**이거든요. 6월부터는 가격이 30% 올라가요. 지금이 진짜 기회예요" |
| **"캐빈이 좁지 않나?"** | "생각보다 훨씬 넉넉해요. 어차피 배 위에서 시간 거의 안 보내니까요. 밖으로 나가면 수영장, 카지노, 레스토랑... 선택지가 어마어마해요" |
| **"배멀미로 못 잘까봐?"** | "대부분의 선상 수면이 육지 호텔 수면보다 더 깊어요. 선박의 안정성이 많이 좋아졌거든요" |
| **"아이들이 지루하면?"** | "아이들 프로그램이 정말 많아요. 오전에 클래스, 오후에 활동... 부모가 쉬는 동안 아이들은 친구 만들고 놀아요" |

```
Step 4: 확신
"결국 훨씬 가성비 좋은 거 같지 않아요?"
(재정의 강화)
```

## Exception Handling (이견 처리)

### 타입 1: 가격 거부감

**심리: Loss Aversion (손실 회피) + Price Anchoring (가격 기준 고착)**

#### 대응 전략
```
High: "너무 비싼데요"
         ↓
Step 1: 재앵커링 (가격 기준 변경)
  "어, 그런데 항공사로 같은 일정 하려면..."
  → 비교를 통해 새로운 기준 제시

Step 2: 비용 분해 (어디에 쓰이는지 명확화)
  "그 안에 숙박, 식사, 이동 다 들어 있거든요"
  → 투명성으로 신뢰 형성

Step 3: 가치 재정의 (가격을 경험으로 환산)
  "하루에 10만 원이면 5성급 호텔에서 자고, 음식도 좋고..."
  → 경험 가치가 높다는 것을 수치로 증명

Step 4: 선택지 제시 (기준 제시)
  "혹은 더 저렴한 카테고리 배도 있어요"
  → 통제감 제공 (가격 거부감 감소)
```

#### 카피 예시
```
고객: "600만 원이 좀 많네요"
당신: "많이 들리긴 하지만, 
      항공사로 7박을 가면 항공권에만 200만,
      호텔에 300만, 식사 100만... 
      결국 비슷하거든요.
      
      차이점은 크루즈는 여행 준비가 없다는 거예요.
      그걸 생각하면 오히려 더 저렴한 거 같지 않아요?"
```

### 타입 2: 배 거부감 / 불안감

**심리: Uncertainty Aversion (불확실성 회피) + Control Need (통제욕구)**

#### 대응 전략
```
High: "배에서 혼자 남겨지면 불안할 것 같은데"
         ↓
Step 1: 안심 제공 (구체적 안전 정보)
  "크루즈는 24시간 스태프가 있고..."
  → 통제감 제공

Step 2: 정상화 (많은 사람들이 하고 있음)
  "대부분의 사람들이 처음엔 불안해하는데..."
  → 불안감이 정상적이라는 메시지

Step 3: 경험담 제공 (친숙한 사람의 경험)
  "저도 처음엔 불안했는데, 가보니..."
  → 신뢰성 높은 정보

Step 4: 선택지 제공 (통제욕구 만족)
  "혼자 조용한 객실, 또는 사람 많은 객실..."
  → 선택 권한 제공
```

#### 카피 예시
```
고객: "배에서 혼자 있으면 너무 답답할 것 같은데..."
당신: "많은 분들이 처음엔 그렇게 생각하세요.
      근데 가보면, 밖으로 나가서 
      수영장도 가고, 카지노도 가고, 
      레스토랑도 돌아다니다 보면
      시간이 빨리 가요.
      
      실제로 '배 안에서 뭔가만 했다'는 분이 거의 없어요.
      오히려 '시간이 모자라다'고 하실 정도예요."
```

### 타입 3: 시간 부족 / 일정 고민

**심리: Loss Aversion (시간 손실) + Commitment Anxiety (약속 불안)**

#### 대응 전략
```
High: "5월에는 일이 많아서..."
         ↓
Step 1: 유연성 제공 (일정 변경 가능성)
  "5월이 어렵다면 다른 달도..."
  → 통제감 복원

Step 2: 우선순위 제시 (강한 이유 제공)
  "근데 지금 안 가시면, 8월은 가격이 올라가고..."
  → 객관적 이유로 결정 촉진

Step 3: 간소화 (결정 부담 감소)
  "일정은 우리가 도와드릴 테니..."
  → 준비 부담 제거

Step 4: 작은 약속부터 (결정 심리 극복)
  "일단 예약금만 하시고, 나머지는 나중에..."
  → 심리적 진입장벽 낮춤
```

#### 카피 예시
```
고객: "5월은 좀 바빠서..."
당신: "아, 이해합니다. 
      그럼 6월이나 7월은 어떠신가요?
      
      다만, 한 가지 말씀드릴 게,
      5월 가격으로 갈 수 있는 기간이
      한정되어 있거든요.
      
      6월부터는 30% 올라가는데,
      혹시 5월에라도 한두 나흘 빼서 
      다녀오시는 건 어떨까요?"
```

## 측정 지표 및 최적화

### KPI 정의

#### 1차 지표 (Call Level)
```
📞 응답율 (Answer Rate)
   = 전화 받은 사람 수 / 콜 수
   목표: 80% 이상

💬 Step 2 통과율 (Qualification Rate)  
   = "욕구 발굴" 완료 고객 수 / 응답자 수
   목표: 70% 이상
   의미: 고객이 자신의 고통점을 명시적으로 언급

💡 Step 3 통과율 (Interest Rate)
   = "해결책" 제시 후 관심 표시 / Step 2 통과자
   목표: 60% 이상
   신호: "그럼 정확히 어디로 가는 건데?" 등의 질문

💰 Step 4 통과율 (Intent Rate)
   = 가격 제시 후 예약 의사 표시 / Step 3 통과자
   목표: 50% 이상
   신호: "언제부터 가능한데?" 등의 일정 질문

✅ 최종 예약율 (Conversion Rate)
   = 실제 예약 고객 / 콜 수
   목표: 20% 이상 (산업 표준 10~15%)
```

#### 2차 지표 (Solution Level)
```
📊 Solution별 효과도
   = (Step X-1 통과율 - Step X 통과율) / 
     각 Solution 사용 건수
     
🎯 고객 세그먼트별 전환율
   - 자녀 있는 가정: Solution 7 적용 후 전환율
   - 직장인 부부: Solution 4 적용 후 전환율

```
