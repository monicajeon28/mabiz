## 🎯 판매톤(Sales Tone) 분류 체계

### 식별된 8가지 판매톤

#### 1. **Friendly** (친근함)
- **특징**: 따뜻한 인사, 감사 표현, 긍정적 감정
- **키워드**: 안녕, 감사, 예, 좋아, 함께, ~~, ^^, 반갑
- **예시**: "안녕하세요~! 함께 봐주세요^^"
- **대상 고객**: 신규 고객, 처음 이용자

#### 2. **Professional** (전문성)
- **특징**: 정책 설명, 법적 근거, 체계적 안내
- **키워드**: 정책, 규정, 법적, 증명, 따라, 그러므로, 이에
- **예시**: "운영정책을 위반한 경우 제한이 있을 수 있습니다"
- **대상 고객**: 계약/정책 확인 필요 고객

#### 3. **Urgent** (긴급성)
- **특징**: 시간 압박, 빠른 응답 필요, 당장의 행동 유도
- **키워드**: 지금, 오늘, 급, 빨리, 당장, 서둘러, 내일
- **예시**: "지금 바로 탑승하셔야 합니다"
- **대상 고객**: 시간 제약 있는 고객

#### 4. **Empathetic** (공감)
- **특징**: 감정 이해, 어려움 공감, 함께하려는 태도
- **키워드**: 이해, 공감, 걱정, 도움, 힘들, 어려
- **예시**: "그 어려움 충분히 이해합니다"
- **대상 고객**: 문제/불만 신고 고객

#### 5. **Factual** (사실 기반)
- **특징**: 데이터, 통계, 구체적 수치 제시
- **키워드**: 데이터, 결과, 통계, 비교, 분석, 구체
- **예시**: "2026년 5월 기준 통계에 따르면..."
- **대상 고객**: 근거 기반 설득 필요 고객

#### 6. **Solution-Oriented** (해결 중심)
- **특징**: 문제 해결, 개선안 제시, 추천
- **키워드**: 해결, 개선, 최적, 효과, 추천, 도움
- **예시**: "이렇게 하시면 문제가 해결됩니다"
- **대상 고객**: 구체적 해결책 필요 고객

#### 7. **Casual** (캐주얼)
- **특징**: 편안한 분위기, 반어법, 유행 용어
- **키워드**: ㅋ, 진짜, 완전, 너무
- **예시**: "진짜 완전 좋아요!"
- **대상 고객**: 젊은 세대, 친구 같은 관계

#### 8. **Formal** (정중함)
- **특징**: 공식적 태도, 존경심, 정중한 표현
- **키워드**: 존경, 정중, 인사, 경의
- **예시**: "존경합니다"
- **대상 고객**: 고급 고객, 규격화된 응답 필요

## 🔄 데이터 포맷 및 구조

### 1. questions_rag_memory_with_tone.json
기존 RAG 메모리에 **sales_tone** 필드 추가

**샘플 구조**:
```json
{
  "id": "q0002",
  "question": "2026년 5월 9일 토요일 ...",
  "answer": "모니카님이 들어왔습니다...",
  "category": "정책&수수료",
  "source": "MSC벨리시마.txt",
  "type": "상담기록",
  "sales_tone": {
    "primary": "friendly",
    "secondary": ["professional"],
    "confidence": 0.33
  }
}
```

**필드 설명**:
| 필드 | 타입 | 설명 |
|-----|-----|-----|
| `primary` | string | 주요 판매톤 (8가지 중 1개) |
| `secondary` | array | 보조 판매톤 (0~2개) |
| `confidence` | float | 신뢰도 (0.0~1.0) |

### 2. sales_tone_training.json
세일즈봇 학습용 정제된 데이터셋

**샘플 구조**:
```json
{
  "id": "q0008",
  "question": "미리 다운받아 가시는것 추천드립니다...",
  "answer": "크루즈닷 빨간 후드티를 입고 여러분을 맞이할꺼예요...",
  "category": "탑승&수속",
  "primary_tone": "friendly",
  "secondary_tones": ["solution"],
  "confidence": 0.33
}
```

**필터링 기준**:
- Neutral 톤 제외 (학습 노이즈 방지)
- 신뢰도 0.33 이상만 포함 (최소 1개 키워드 매칭)
- 최대 질문 300자, 답변 400자로 제한

## 🎓 활용 방법

### 1. 세일즈봇 기본 학습
```python
# sales_tone_training.json 로드
with open('sales_tone_training.json') as f:
    training_data = json.load(f)

# 톤별 샘플 분류
tone_samples = {}
for sample in training_data['samples']:
    tone = sample['primary_tone']
    if tone not in tone_samples:
        tone_samples[tone] = []
    tone_samples[tone].append(sample)

# 각 톤별 학습
for tone, samples in tone_samples.items():
    print(f"{tone}: {len(samples)}개 샘플")
```

### 2. 고객별 톤 선택 로직
```
고객 프로필 → 상황 분석 → 적절한 톤 선택

예시:
- 신규 고객 + 첫 상담 → Friendly (친근함)
- 시간 제약 있음 → Urgent (긴급성)
- 문제 해결 필요 → Solution (해결 중심)
- 고급 고객층 → Professional + Formal (전문성/정중함)
```

### 3. 톤 혼합 전략
학습 데이터는 **secondary_tones** 필드로 복합 톤 제시
- Primary: 메인 톤 (주제에 따라)
- Secondary: 부가 톤 (상황에 따라)
