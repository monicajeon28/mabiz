## 🎓 이론적 배경 (최소 필독)

### Grant Cardone L1-L7 렌즈
```
L1 (가격 민감도)
   → 손실회피: 손실 고통 = 이득 쾌락 × 2.25배
   → 효과: 최저가 보증 → 즉시 구매

L2 (준비 복잡도)
   → 인지부하: 규칙이 많을수록 → 불안 ↑ → 지연
   → 효과: 체크리스트 3가지 → 안심 → 예약

L3 (경쟁사 인식)
   → 앵커링: 높은 앵커 제시 → 우리 가격 저가 인식
   → 효과: 비교표 → 신뢰도 ↑

L6 (타이밍 손실회피)
   → 후회이론: 지금 안 가면 → 10년 후회
   → 효과: 계절성 강조 → 즉시 의사결정

L7 (동반자 설득)
   → 사회증명: 비슷한 가족들 다 가는데 우리는?
   → 효과: 가족 프로그램 소개 → 모두 만족

L5 (의료 신뢰)
   → 권위성: 전문가 인증 → 신뢰도 극대화
   → 효과: 의료 기준 증명 → 확신 → 예약
```

### PASONA 프레임워크
```
P(Problem)    → 고객의 불안 인정
A(Agitate)    → 불안 증폭 (손실회피 자극)
S(Solution)   → 우리의 해결책 제시
O(Offer)      → 구체적 제안 (숫자/증거)
N(Narrow)     → 선택지 좁히기 (양자택일)
A(Action)     → 지금 결정하게 하기

크루즈 적용:
Day 0:  P + A (희소성 + 타임프레싱)
Day 1:  S (해결책: 체크리스트, 가이드)
Day 3:  O + N (제안: 가격, 탑승료 포함 + 날짜 선택)
Day 3+: A (행동: 예약서 작성 시작)
```
