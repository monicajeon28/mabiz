5. 파일별 사용법 (요약)
================================================================================

[product_mapping.csv] → BigQuery 로드/Pandas 분석/Excel 피벗 테이블
  열: question_id, question_text, category, product_ids,
       max_relevance, keywords_matched

[product_mapping.json] → 세일즈봇 백엔드/Redis 캐시/REST API
  구조: { mappings: [ { question_id, products: [{ product_id, relevance }] } ] }

7. 주요 발견사항
================================================================================

1️⃣  선상팁(TIP) 관련 질문은 100% 정확도로 Product ID 320 매핑
    → 이미 충분히 구체적인 질문 데이터

2️⃣  "객실" 관련 질문이 가장 많음 (64건)
    → 고객이 객실 서비스에 가장 관심 많음

3️⃣  미매칭 질문 135개의 49.1%는 실질 질문이 아님
    - 인사말만 있는 경우 (8%)
    - 사진/동영상 첨부 공지 (15%)
    - 순수 잡담/감정 표현 (12%)
    - 시간/날짜 정보만 (8%)
    → 이들은 자동화로 처리 불가능, 맥락 학습 필요

4️⃣  카테고리별 매칭율 편차 크다
    - 탑승&수속: 81% (구체적 절차 질문)
    - 기타: 19% (잡담 섞여 있음)
