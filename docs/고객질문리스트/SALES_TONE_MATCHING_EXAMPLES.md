## 📊 톤별 응답 템플릿

### Friendly 톤 (56개 샘플)

**상황**: 일반 상담, 신규 고객, 기본 안내

**템플릿**:
```
[인사] + [감사 표현] + [정보] + [배려] + [격려]

예:
"안녕하세요! 좋은 질문 감사합니다. 
[구체적 정보 제공]
이렇게 하시면 편리하실 거에요. 
더 궁금하신 점 있으면 언제든 물어봐주세요^^"
```

**특징**:
- ~~, ^^, 이모지 사용
- "고객님", "안녕하세요", "감사합니다"
- 반복 표현 자제 (지루함 방지)

예:

객실 입구 → 카드 수령 → 15층 식당

**특징**:

- "지금", "당장", "바로" 사용

## 🔧 세일즈봇 개발자 가이드

### 톤 선택 로직 예제 (Python)

```python
def select_tone(customer_context):
    """고객 상황에 따라 적절한 톤 선택"""
    
    tones = []
    
    # 신규 고객 확인
    if customer_context.is_new_customer:
        tones.append(('friendly', 0.9))
    
    # 시간 제약 확인
    if customer_context.has_deadline:
        tones.append(('urgent', 0.8))
    
    # 문제/불만 확인
    if customer_context.has_issue:
        if customer_context.is_emotional:
            tones.append(('empathetic', 0.9))
        else:
            tones.append(('solution', 0.8))
    
    # 고급 고객 확인
    if customer_context.tier == 'VIP':
        tones.append(('professional', 0.7))
        tones.append(('formal', 0.6))
    
    # 가중치가 높은 순서로 정렬
    tones.sort(key=lambda x: x[1], reverse=True)
    
    return {
        'primary': tones[0][0] if tones else 'friendly',
        'secondary': [t[0] for t in tones[1:3]],
        'weights': {t[0]: t[1] for t in tones}
    }
```

## 📈 톤 혼합 스코어 계산

```
최종 응답 톤 스코어 = 
  (Primary Tone × 0.7) + 
  (Secondary[0] × 0.2) + 
  (Secondary[1] × 0.1)

예시:
Primary: friendly (0.9)
Secondary[0]: urgent (0.8)
Secondary[1]: solution (0.6)

점수 = (0.9 × 0.7) + (0.8 × 0.2) + (0.6 × 0.1)
     = 0.63 + 0.16 + 0.06
     = 0.85 (Friendly 중심, Urgent 고려)
```
