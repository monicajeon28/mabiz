### 🔴 Priority 1 (즉시 - 1주)

**목표**: 배포 전 필수 정리

```

   - 예: "객실 야경" → Card 대신 Room으로 수정

2. 판매톤 키워드 확대

   - Urgent: 지금, 오늘 → 8개 추가
   
3. 테스트 케이스 수정
   - 영문 카테고리명 → 한글로 변경
   - TIP → 정책&수수료
   - Room → 객실&카드
   - Boarding → 탑승&수속
```
