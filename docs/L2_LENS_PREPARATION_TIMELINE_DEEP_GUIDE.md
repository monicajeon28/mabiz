## 📅 준비 타임라인 (3개월 전~당일)

### ✅ 3개월 전 (예약 확정 직후)

**당신이 할 일:**
```
□ 휴가 날짜 회사에 신청
□ 여권 확인 (유효기간 6개월 이상)
□ 가족/배우자와 함께 스케줄 확인
□ 예산 재검토 (추가 비용 항목)
```

**우리가 해주는 것:**
```
✅ SMS: "축하합니다! 예약 완료"
✅ 여권 확인 여부 체크 (CRM)
✅ 필요한 서류 리스트 발송
✅ 여행 가이드 (상세 130페이지) 발송
✅ 커뮤니티 그룹 초대 (같은 배 탑승자들)
```

**SMS 샘플:**
```
📲 "축하합니다, OOO님!

당신의 크루즈 예약이 완료됐어요.

지금부터 해야 할 3가지:

1️⃣ 회사 휴가 신청 (지금 바로!)
   - 일정: 7월 15-22일
   - 사유: 가족 여행

2️⃣ 여권 확인 (지금 바로!)
   - 유효기간이 6개월 이상인가?
   - 비자 필요한가? (괜찮음, 우리가 안내)

3️⃣ 다음 메시지 기다리기 (2주 뒤)
   - 그때 뭘 할지 알려드릴게요

지금은 이것만 하면 돼.

- 마비즈 크루즈팀"
```

### ✅ 2개월 전 (준비 본격화)

**당신이 할 일:**
```
□ 여권 발급 또는 갱신 (필요시, 2주 소요)

□ 기본 짐 리스트 검토 (우리가 준 체크리스트)
□ 당신의 방문 국가 비자 확인
```

**우리가 해주는 것:**
```

✅ 비자 요구사항 상세 안내

✅ 준비 영상 가이드 (10분)
✅ FAQ 가이드 (52가지 질문)
```

**SMS 샘플:**
```
📲 "OOO님, 2개월 전입니다!

2주일마다 '뭘 할 시간'을 알려드릴게요.

   - 만료일이 9월 이후인가?
   - 비자 필요한 국가가 있나?

✅ 아이 여권 (있다면)
   - 신생아: 5년 유효
   - 아동(13세 미만): 5년 유효

✅ 음식 알레르기 신고 (필요시)
   - 배에서 미리 준비해야 함

2주 뒤 다시 메시지 드릴게요.

- 마비즈 크루즈팀"
```

### ✅ 1개월 전 (짐 준비 시작)

**당신이 할 일:**
```
□ 짐 준비 본격화 (세탁, 정리)
□ 의약품 준비 (평소 복용약 3개월분)
□ 해외 보험 가입 (선택)
□ 외화 환전 (선택, 배에서도 가능)
□ 카드 해외 사용 확인
```

**우리가 해주는 것:**
```
✅ SMS: "짐 준비 시작하셨나요?"
✅ 짐 체크리스트 (카테고리별 100항목)
✅ "뭘 안 가져가도 되는가" 가이드
✅ 선실 배정표 (정확한 호실번호)
✅ 배 지도 및 시설 가이드
✅ 기항지별 액티비티 추천
```

**SMS 샘플:**
```
📲 "OOO님, 1개월 전입니다!

짐 준비가 '정말' 간단하다는 걸 증명할게요.

당신이 꼭 가져가야 할 것 (5가지):
1️⃣ 여권 (출입국용)
2️⃣ 카드/현금 (선택사항, 배에서도 가능)
3️⃣ 평소 복용약 (당뇨약 등)
4️⃣ 기본 옷 (5일치 + 특별날 옷)
5️⃣ 세면도구 (배에도 있음)

당신이 안 가져가도 되는 것:
❌ 헤어드라이어 (객실에 있음)
❌ 수건 (객실에 있음)
❌ 슬리퍼 (객실에 있음)
❌ 기본 세제/샴푸 (배에 있음)

결과: 짐이 호텔의 70% 수준

배로 오는 길에 편해져요.

상세 체크리스트는 첨부했어요.
필요한 것만 체크하고,
나머지는 안 가져가세요.

- 마비즈 크루즈팀"
```

## 💡 당신 vs 우리 (책임 분담)

```
준비 과정을 보면:

당신의 책임:
├─ 휴가 신청 (1번)
├─ 여권 준비 (1번)
├─ 짐 준비 (반복, 총 3시간)
├─ 카드 확인 (10분)
└─ 당일 탑승 (4시간)

총 준비 시간: 약 8시간
(호텔은 16시간)

우리의 책임:
├─ SMS 자동 발송 (매 2주마다)
├─ 체크리스트 제공 (5장)
├─ 비자 안내
├─ 선실 배정
├─ 식사 배정
├─ 액티비티 예약
├─ 기항지 가이드
├─ 당일 탑승 가이드
├─ 24시간 배 관리
└─ 응급상황 대응

결론: 당신은 짐만 챙기면 돼.
     우리가 나머지 다 할 거야.
```

## 📊 준비 체감도 변화

```
3개월 전: "준비가 엄청 많겠네... 불안해"
  → 우리 메시지: "사실 거의 없어요"
  → 체감: 불안도 30% 감소

2개월 전: "구체적으로 뭘 준비해야 하지?"
  → 우리 메시지: "이 체크리스트 따라하세요"
  → 체감: 명확해짐, 불안도 60% 감소

1개월 전: "아, 정말 간단하네"
  → 우리 메시지: "당신이 할 게 거의 없어요"
  → 체감: 안심, 불안도 80% 감소

2주 전: "거의 다 했다"
  → 우리 메시지: "완벽해요, 기대하세요"
  → 체감: 행복도 증가, 불안도 95% 감소

당일: "정말 간단했네!"
  → 실제 경험: "호텔보다 훨씬 쉽다"
  → 체감: 만족도 100%
```
