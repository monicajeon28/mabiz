# Menu #38 Phase 4 Track 2: L3렌즈 (차별성 미인지) 완전 구현

**목표**: "일반 여행이랑 같잖아"는 오해를 극복하는 완전한 판매 스크립트 + SMS 자동화 + CRM 분류규칙

**작성일**: 2026-05-19  
**렌즈 정의**: 저항유형 차별성 미인지형  
**핵심 오해**: "배만 타는 거? 일반 여행이랑 뭐가 달라요?"  
**경험수준**: 크루즈 미경험  
**전환목표**: 32% → 50% (+18p, 56% 개선)  
**핵심 전략**: "배=리조트" 개념 전환 (배는 목적지가 아니라 숙소+엔터테인먼트+레스토랑+커뮤니티)

## 📊 L3 렌즈 고객 프로필

### 타겟 고객 정의

```
Primary Target:
├─ 첫 크루즈 모든 고객
├─ 20대 후반~50대 전체
└─ 특히 여행에 "경험" 중심 고객

Specific Personas from 20Lens:
├─ #2 프리랜서 자유업 ("새로운 경험을 찾는 사람")
├─ #6 40대 자녀초등 ("아이에게 다양한 경험 제공")
├─ #17 60대 은퇴왕성 ("은퇴 후 새로운 삶의 의미")
└─ 모든 첫 크루즈 고객
```

### 핵심 Pain Point

| 요소 | 설명 | 증거 |
|------|------|------|
| **오해의 근원** | 배=이동수단으로만 인식 | "배 타면서 뭐 하는데?" |
| **경험 부족** | 크루즈를 본 적이 없거나 다큐 영상만 봤음 | "호텔처럼 객실도 있어요?" 확인 |
| **선입견** | "배는 멀미한다", "비행기처럼 답답하지 않을까?" | "갑판에서 나올 수 있어요?" 질문 |
| **비교대상 오류** | 일반 여행(비행기+호텔)과 비교 | "그냥 배 타는 거 말고 호텔 묵는 게 낫지 않아?" |
| **목적지 중시** | "배는 목적지가 아니라 이동수단 아니어?" | "도쿄, 오사카, 홍콩이 목적지인데, 배는?" |

### 저항 패턴 분석

```
고객 발언 → 심리 상태 → 현행 대응 → 문제점 → L3 대응

"배만 타는 거 아니에요?" 
→ 배=수송수단 고착
→ "아니요, 기항지가 있습니다" (기항지 강조)
→ 배 자체의 경험이 설득 안 됨
→ "배 위에서의 2박이 실제 경험의 70%입니다" (경험 재정의)

"일반 여행이랑 뭐가 다르지?"
→ 비교대상 혼동
→ "식사, 관광, 숙박이 다릅니다" (구성 나열)
→ 추상적, 감정 전달 안 됨
→ "배 = 움직이는 리조트 + 매일 새로운 창밖" (Mental Model Shift)

"갑판에서 나올 수 있어요?"
→ 답답함 불안
→ "갑판이 여러 층 있습니다" (정보 제공)
→ 구체성 부족
→ "사진 보여드릴까요?" (시각적 증거)
```

## 🎯 L3 렌즈 완전 콜 스크립트 (Phase 0-4)

### 전략 프레임워크

```
심리학 원리: Narrative Transportation (이야기를 통한 경험 이동)
           + Mental Model Shift (배=리조트 개념 전환)
           + Visualization (머릿속 그리기)
           + Social Proof (고객 후기)

PASONA 매핑:
├─ Problem: "배 = 배만 타는 것?" (오해 인식)
├─ Affinity: "많은 첫 크루즈 고객이 그렇게 생각해요" (공감)
├─ Solution: "배 자체가 목적지입니다" (개념 전환)
├─ Offer: "3박 4일 중 2박을 배 위에서, 배 위에서 모든 게 다 있어요" (구체화)
├─ Narrowing: "배 = 리조트 + 엔터테인먼트 + 식당 + 커뮤니티" (명확화)
└─ Action: "첫 크루즈 비디오 투어 + 고객 후기 + 예약" (행동)

SPIN 적용:
├─ Situation: "혹시 크루즈를 배로만 아셨나요?" (상황 확인)
├─ Problem: "그럼 배 위에서 3박을 어떻게 보낼지 궁금하신가요?" (문제 명확화)
├─ Implication: "만약 배 위에서도 호텔처럼 식사·쇼·휴식을 할 수 있다면?" (함축성)
└─ Need-Payoff: "배를 리조트로 생각하면 이해가 되시죠?" (필요충족)
```

### Phase 0: 오프닝 (0-1분)

**목표**: 배에 대한 오해를 인식하게 하기 + 새로운 각도 제시  
**심리 원리**: Low Barrier Opening + Diagnostic Question  
**PASONA 맵**: Problem 인식

#### 스크립트 A: 기존 고객(비교 도움)

```
"안녕하세요! 크루즈닷 신민형입니다.

혹시 여행 가보셨어요?
호텔이랑 리조트는 다르잖아요.

크루즈도 비슷한데...
배를 리조트처럼 생각하신 적 있으세요?"

[고객 반응 대기 - 30초]

"네, 이게 핵심이거든요.
많은 사람들이 '배 = 배만 타는 것'으로 아는데,
실제로는 '배 = 움직이는 리조트'예요."
```

#### 스크립트 B: 호텔 경험 활용

```
"혹시 5성급 호텔에 묵어보셨어요?

크루즈는요,
'5성급 호텔이 계속 움직인다'고 생각하면 돼요.

그럼 맞나요?"
```

#### 스크립트 C: 일반 여행 비교

```
"일반 여행할 때
아침에 호텔에서 나와서
종일 관광을 다니고
저녁에 호텔로 돌아가잖아요?

크루즈는
'호텔과 관광이 동시에 일어나요.'
저녁엔 배 위에서 쇼를 보거나 친구들과 카지노를 하죠."
```

### Phase 1: 욕구 발굴 (1-3분)

**목표**: 배에 대한 구체적 오해 파악 + 해소할 포인트 찾기  
**심리 원리**: Diagnostic Listening + Empathy  
**PASONA 맵**: Affinity (공감)  
**SPIN 맵**: Situation + Problem (상황파악, 문제명확화)

#### 단계 1-1: 배에 대한 오해 명시

```
"혹시 배에 대해 어떤 걱정이 있으세요?

예를 들면:
- 배 위에서 3박을 어떻게 보내지?
- 밖에만 있으면 답답하지 않을까?
- 갑판에 나갈 수 없는 시간이 많지 않을까?
- 날씨가 안 좋으면 뭐 하지?
- 멀미할까봐 걱정

뭐가 제일 걱정되세요?"

[고객 답변 깊이 경청 - 최소 40초]
```

#### 단계 1-2: 구체적 욕구 도출

```
"아, 그렇군요.
그럼 좀 더 궁금한 게, 
'배 위에서 뭘 하면 재미있을까?'
이런 생각이 드세요?"

[고객: 보통 "네, 정확해요"]

"혹시 호텔에서 
짐 내려놓고 
로비에서 시간을 보낸 적 있으세요?

크루즈가 그런 거거든요.
호텔 같은 공간에서
할 일이 엄청 많아요."
```

### Phase 2: 욕망 증폭 (3-7분)

**목표**: "배 = 리조트" 정신 모델 완전히 전환  
**심리 원리**: Narrative Transportation + Vivid Visualization + Mental Model Shift  
**PASONA 맵**: Solution (해결책) + Offer (구체화)  
**SPIN 맵**: Problem Implication (문제의 의미)

#### 단계 2-1: 배 구조 시각화

```
"자, 그럼 상상해보세요.

배는요, 
아래로는 객실이 20층이에요.
사람 5,000명이 묵습니다.

[생각의 중심]
→ 매 층에 다른 것들이 있어요:
  - 레스토랑 (3개 + 뷔페)
  - 바/라운지 (8개)
  - 스파 (1층 전체)
  - 수영장 (3개 + 온천)
  - 극장 (2,000명 수용)
  - 카지노
  - 사진관
  - 피자 수업, 와인 강좌...

위에는 갑판이 12층입니다.
10만 평방미터 정도.
언제든 나올 수 있어요."

[시각적 이해 확인]

"이게 '배'라고 생각하기보다
'움직이는 호텔'이라고 생각하면
이해가 쉬워요. 맞나요?"
```

#### 단계 2-2: 일과 비교 (Narrative Transportation)

```
"자, 3박 4일 일정을 함께 보시죠.

[Day 1] 목요일 오후
→ 부산 탑승 (5시)
→ 배 오리엔테이션 (6시)
→ 저녁 식사 (8시)
→ 나이트클럽에서 신혼부부 환영파티 (10시)
→ 11시에 침대에 누우면, 바다 위를 항해 중

[Day 2] 금요일 종일
→ 아침 7시에 깨어보니 도쿄 항구 입항!
→ 아침식사 후 (배에서)
→ 도쿄 관광 (오후 2시까지)
→ 오후 4시 배로 돌아와
→ 스파에서 마사지 (1시간)
→ 저녁 음악회 (8시)
→ 침대에서 바다 소리 들으며 숙면

[Day 3] 토요일 종일
→ 아침에 오사카 입항!
→ 오사카 관광
→ 배로 돌아와 선셋 갑판에서 와인
→ 저녁 환영 가라 디너 (배의 VIP 식당)
→ 밤샘 카지노

[Day 4] 일요일 아침
→ 부산 귀항

보면요,
일반 여행과 다른 점이 뭔가요?"

[고객 반응]

"맞아요!
호텔을 매번 옮길 필요가 없다는 거예요.
침대는 그대로인데,
창 밖은 매일 다르고,
할 일도 매일 다르고."
```

#### 단계 2-3: 배 경험의 3가지 차별성

```
"크루즈의 3가지 특별한 점을 말씀드릴게요.

[Point 1] 이동 중 관광
- 일반여행: 호텔 → 택시 → 관광지 (6시간 소요)
- 크루즈: 배에서 깨어나니 이미 목적지 (0시간)
- 심리: 창을 통해 매시간 다른 경치

[Point 2] 올인클루시브 리조트
- 일반여행: 호텔 80만원 + 식사 40만원 + 관광가이드 20만원
- 크루즈: 식사·쇼·이동·기항지 모두 포함
- 심리: 하루 10번 뭘 할지 계획할 필요 없음

[Point 3] 사교 커뮤니티
- 일반여행: 호텔 → 관광 → 호텔 (혼자)
- 크루즈: 같은 테이블 사람들과 식사, 갑판 파티, 이벤트
- 심리: '우리'가 함께하는 여행 (가족 같은 경험)

이게 배라서 가능한 거거든요."
```

#### 단계 2-4: 감정적 클로징

```
"많은 첫 크루즈 고객들이
배에서 내릴 때 말해요.

'아, 배가 리조트였네요!'
'침대는 그대로인데 매일 새로운 나라네요!'
'이렇게 편할 줄 몰랐어요.'

이게 크루즈의 매력이에요."
```

### Phase 3: 상품 설명 (7-10분)

**목표**: "배 = 리조트"의 증명 자료 제시  
**심리 원리**: Authority + Specificity + Social Proof  
**PASONA 맵**: Narrowing (구체화) + Offer (오퍼 제시)  
**SPIN 맵**: Need-Payoff (필요충족)

#### 단계 3-1: 배 사진/동영상 투어

```
"제가 배 내부 사진을 보여드릴까요?

[객실 사진]
→ "호텔 객실하고 같잖아요?"

[레스토랑 사진]
→ "이것도 배 위에 있고..."

[갑판 사진]
→ "여기서 선셋을 봐요.
   날씨 좋으면 이렇게 맑고,
   밤에는 별이 쏟아져요."

[라운지/바 사진]
→ "여기서 친구들 만나고..."

[스파 사진]
→ "여긴 전용 스파예요.
   마사지 받으면서 창밖이 이래요."
```

#### 단계 3-2: 실제 일정 제시

```
"우리가 권하는 3박 4일 일정은
'일본 오사카-도쿄-오키나와' 루트예요.

[구체적 일정]
DAY 1: 부산 탑승 → 선내 적응
DAY 2: 도쿄 입항 → 도쿄 관광 → 배로 돌아와 쇼 관람
DAY 3: 오사카 입항 → 오사카 관광 → 배로 돌아와 스파
DAY 4: 부산 귀항

비용: 부부 합쳐 350만원대
     (항공기 왕복 안 함, 호텔·식사·쇼 모두 포함)

멤버십: 월 33,000원 (첫 달 50% 할인)

준비물: 여권 + 짐만 싸면 끝"
```

#### 단계 3-3: 가격 vs 일반여행 비교표

```
"일반 여행과 비교해보면:

### Phase 4: 클로징 (10-13분)

**목표**: 첫 크루즈 특별 혜택 3중 선택  
**심리 원리**: Choice Architecture + Social Proof + Limited Time  
**PASONA 맵**: Action (행동 유도)

#### 단계 4-1: 첫 크루즈 특별 혜택

```
"첫 크루즈 고객을 위해
세 가지 혜택을 준비했어요.

[Option A] 추천 (가장 인기)
- 9월 도쿄-오사카 크루즈
- 선실 업그레이드 (5만원 → 무료)
- 선내 신용카드 10만원 (스파·쇼·바)
- 첫 달 멤버십 50% 할인

[Option B] 저예산
- 8월 오키나와-홍콩 (저가형 선박)
- 멤버십만 가입하고 상품은 나중에
- 상품비 최대 30% 할인 (3개월 이내)

[Option C] 프리미엄
- 11월 크루즈 (큰 선박)
- VIP 라운지 무료 이용 (6개월)
- 온보드 크레딧 30만원
- 전문가 투어 가이드 (선내)

어느 것이 맞을 것 같아요?"
```

#### 단계 4-2: 신민형 5STEP 삼중선택

```
