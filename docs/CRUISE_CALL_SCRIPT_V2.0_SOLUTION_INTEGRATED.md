## 📌 핵심: Solution 7가지란?

각 Solution은 **심리학 원리를 기반으로 한 특정 단계의 설득 기법**입니다.

| Solution | 심리학 원리 | 목적 | 타겟 단계 |
|----------|-----------|------|---------|
| 1 | Loss Aversion (손실회피) | "지금 하지 않으면 손실" | 전체 (특히 N단계) |
| 2 | Cognitive Ease (인지적 용이성) | "구체적 방법 제시" | S단계 (Solution) |
| 3 | Simplicity (단순성) | "3단계면 끝" | S단계 (Solution) |
| 4 | Value Proposition (가치제안) | "비용 대비 가치" | O단계 (Offer) |
| 5 | Narrative Transportation (스토리텔링) | "감정적 몰입" | A단계 (Affinity) |
| 6 | Social Proof + Scarcity (배타성) | "특별함" 느낌 | P단계 + N단계 |
| 7 | Identity & Meaning (정체성) | "인생의 의미" | A단계 (Action) |

## 🎬 전체 콜 흐름 (15분 / 5단계)

```
0-2분: 오프닝 (출처 명확화 + Solution 6: 배타성)
2-5분: 욕구 발굴 (Solution 5: 기대감 + Solution 1: 손실회피)
5-10분: 상품 설명 (Solution 2/3/4: 방법/간단/효율)
10-13분: 가격 & 긴급성 (Solution 1/6 재활용: 기한 + 배타성)
13-15분: 클로징 (Solution 7: 정체성 + 의미)
```

## 📋 STEP 1: 오프닝 (0-2분)
### "신뢰감 형성 + 배타성 암시"

### 스크립트 A (신뢰 중심)
```
상담사: "안녕하세요, 모니카님! 
        저는 크루즈닷 영업팀의 신민형입니다.
        
        혹시 크루즈 여행을 준비 중이신가요?
        (고객 반응 대기)
        
        저희 고객분들이 많이 추천해주셔서 전화를 드렸어요.
        2분만 귀 기울여 주시면, 
        특별한 제안이 있어서요."
```

### 심리학 원리
- **Rapport 형성**: 성명 + 회사 명확화
- **Social Proof**: "고객분들이 추천"
- **Curiosity**: "특별한 제안"으로 호기심 자극

### 🎯 Solution 6 활용: 배타성 암시
```
상담사: "실제로 저희는 모든 분께 전화를 드리지 않아요.
        크루즈 여행 경험이 있고,
        진정한 가치를 느낄 수 있는 분들께만
        특별한 패키지를 추천하거든요.
        
        모니카님이 그런 분들 중 한 분이라고 생각해서
        직접 전화를 드렸습니다."
```

### 효과
- "나는 특별한 고객" 인식 ✓
- "선택받은 느낌" → 신뢰도 상승 ✓

상담사: "모니카님은 지금까지 어디로 여행을 많이 다니셨어요?"
(고객 답변)

"그럼 크루즈는 경험해보셨어요?"
(고객 답변)

### 🎯 Solution 5: 기대감 높이기
```
상담사: "아, 그렇군요. 그럼 이렇게 생각해보세요.

        배 위에서 해 지는 것을 보는데...
        (감정적 이미지)
        
        ✨ 옆에 남편분이 있고,
        ✨ 발 아래로 대양이 있고,
        ✨ 하늘은 주황색으로 물들고 있어요.
        
        이런 경험, 평생 잊을 수 없을 거예요.
        
        그런데 선생님도 아시다시피,
        우리 건강은 점점 약해지잖아요.
        
        👉 [기대감] 지금이 이런 경험을 할 수 있는 최고의 타이밍이에요."
```

### 🎯 Solution 1: 손실회피 활용
```
상담사: "만약 지금 예약 안 하고

        😟 가격이 더 비싸지고,
        😟 건강 상태가 어떻게 될지 모르고,
        😟 이 특가는 더 이상 없을 거예요.
        
        지금 이 기회를 놓치면...
        (침묵 3초)
        
        👉 [손실] 후회하실 수도 있어요."
```

## 🎯 STEP 3: 상품 설명 (5-10분)
### "복잡성 제거 + 가성비 강조"

### 🎯 Solution 2: 방법 알려주기 (구체적 스토리)
```
상담사: "크루즈가 복잡할 수 있으니까,
        제가 정확하게 설명해드릴게요.
        
        ➡️ Day 1 (승선일): 인천에서 배를 타요.
           - 저희 가이드가 탑승부터 객실 배정까지 다 해주고,
           - 저녁은 배 위에서 정찬 레스토랑 식사,
           - 야경을 보면서 크루즈 일정 설명
        
        ➡️ Day 2-3 (해상일): 바다만 있는 날이에요.
           - 수영장, 스파, 쇼 다 무료,
           - 정해진 프로그램 중 고르셔서 참가,
           - 의료 서비스도 배 위에서 제공
        
        ➡️ Day 4-5 (기항지): 일본 나가사키, 홍콩
           - 저희 가이드가 한국인 단독 투어 제공,
           - 선내 프로그램만 해도 시간 부족할 정도
        
        ➡️ Day 6-7 (귀국): 배 위에서 마지막 시간 즐기고,
           - 인천 도착 시 저희가 집까지 배웅"
```

### 🎯 Solution 3: 간단함 강조 (선택지 최소화)
```
상담사: "정말 간단해요. 3단계면 끝나요.

        1️⃣ 예약하기: 오늘 예약금 30만원만
        2️⃣ 탑승하기: 정해진 날짜에 배를 타요
        3️⃣ 즐기기: 나머지는 우리가 다 챙겨요
        
        영어 몰라도 돼요.
        복잡한 서류 없어요.
        일정 변경도 우리가 다 처리해요.
        
        👉 선생님은 그냥 즐기시기만 하면 돼요."
```

### 🎯 Solution 4: 효율성 강조 (가성비)
```
상담사: "자, 이제 가성비 얘기인데요.

        일반 호텔 + 비행기 + 관광지 티켓을 따로 예약하면?
        - 5박 6일: 최소 500만원대
        - 영어로 예약, 모든 게 분리됨
        - 예기치 않은 비용 계속 발생
        
        근데 우리 크루즈닷 패키지는?
        - 5박 6일: 월 33,000원 × 6개월 = 198만원
        - 모든 게 포함 (항만세, 식사, 프로그램)
        - 숨겨진 비용 없음
        
        👉 [효율성] 호텔보다 **비용 70% 저렴**하면서,
           경험은 **10배 더 풍부**해요."
```

### 효과
- 복잡성 제거 (3단계 강조) ✓
- 신뢰도 상승 (구체성) ✓
- 가성비 명확화 ✓

## 💰 STEP 4: 가격 & 긴급성 (10-13분)
### "기한 강조 + 배타성 재활용"

### 🎯 Solution 1 재활용: 기한 강조
```
상담사: "그런데 여기 중요한 게 있어요.

        이 특가는 5월까지만 유지돼요.
        6월부터는 월 42,000원이 되거든요.
        
        📊 5월에 예약: 월 33,000원 × 6 = 198만원
        📊 6월에 예약: 월 42,000원 × 6 = 252만원
        
        👉 [기한] 차이가 54만원이에요.
           지금 바로 하는 게 현명한 거죠."
```

### 🎯 Solution 6 재활용: 배타성 강화
```
상담사: "그리고 한 가지 더 있어요.

        이 5월 선착순 패키지는 딱 50팀만 가능해요.
        
        지금 제 손에 들어온 명단을 보니까
        (실제 또는 암시)
        
        ✨ 4팀이 더 예약하면 마감이 되는 상황이에요.
        
        👉 [배타성] 선착순이니까
           지금 예약하지 않으시면
           아무리 와도 "매진되었습니다"가 될 거예요."
```

### 🎯 선택지 제시 (신민형 5단계 활용)
```
상담사: "그래서 선생님께 3가지 방법이 있어요.

        ✅ 방법 1) 완납 189만원
           - 지금 다 내시고 여름에 가기
           - 특가 + 추가 할인 5% = 179만원
        
        ✅ 방법 2) 월납 월 33,000원
           - 편하게 나눠서 내시기
           - 계속 할인 받기
        
        ✅ 방법 3) 새미패키지 (가족·친구 단체)
           - 20명 모으면 스태프 무료
           - 더 큰 할인
        
        이 3가지 중에 선생님 상황에 제일 맞는 게 뭘까요?"
```

### 효과
- 기한 명확화 (54만원 손실 각인) ✓
- 배타성 강화 (50팀 한정) ✓
- 결정 강제 (3가지 선택만) ✓

## 🎬 STEP 5: 클로징 (13-15분)
### "정체성 강화 + 의미 부여"

### 🎯 Solution 7: 배움의 요소 (Identity & Meaning)
```
상담사: "선생님, 한 가지 생각해보세요.

        지금 선생님은 회사원(또는 역할)로 살고 계신데,
        이 크루즈 여행 후에는 어떻게 될까요?
        
        ✨ 단순한 관광객이 아니라,
        ✨ '크루즈 여행도 경험한 세련된 사람',
        ✨ '세계를 본 사람'이 되시는 거예요.
        
        👉 [정체성] 선생님의 인생에 새로운 챕터가 생기는 거죠.
        
        특히 나중에 자녀나 손주한테 자랑할 수 있어요.
        '할머니가 세계 일주를 했어'라고요.
        
        👉 [의미] 이건 돈으로 살 수 없는 가치가 있어요."
```

### 감정 정점 조성
```
상담사: "지금 이 기회를 통해,
        평생 기억할 경험을 얻을 수 있어요.
        
        50년, 60년을 살아오면서
        가장 행복했던 순간이 될 수도 있어요.
        
        그런데 이건 지금, 이 순간에만 가능해요.
        (침묵 2초)
        
        선생님, 결정 도와드릴까요?"
```
