## 🎯 Executive Summary

Russell Brunson의 **ClickFunnels 8가지 형식**은 **각각 검증된 심리학 원리 기반**으로 설계되었으며, 마비즈 CRM의 현재 랜딩페이지 시스템에서 **최소 코드 추가**로 즉시 구현 가능합니다. 

**각 형식마다 평균 전환율이 명확하고 (15-35%), 필수 UI 요소가 고정되어 있어서** 사용자가 콘텐츠만 채우면 **자동으로 심리학이 적용된 고전환 페이지**가 완성됩니다.

## 1️⃣ VSL (Video Sales Letter) — 비디오로 감정 판매

### 개요
**심리학**: 비디오 = 감정 + 신뢰 (텍스트는 논리만 전달)
**전환율**: 15-35% (고객 engagement 높음)
**추천 상황**: 고액 상품 (크루즈 여행 100만원+)

### 마비즈 CRM에서의 필수 블록

```
┌─────────────────────────────────────────┐
│ 1. Hero 섹션 (비디오 플레이어)           │
│    - 중앙에 큰 비디오 플레이어           │
│    - 재생시간 표시 (예: "7:45")         │
│    - 자동재생 + 음소거 상태              │
│    - 진행도 바 (빨강→초록 변화)         │
└─────────────────────────────────────────┘
          ↓
┌─────────────────────────────────────────┐
│ 2. 자막/전사본 (비디오 옆 또는 하단)    │
│    - 음소거자 위한 텍스트 제공           │
│    - 핵심 문구 강조 (굵게/색상)         │
└─────────────────────────────────────────┘
          ↓
┌─────────────────────────────────────────┐
│ 3. 신뢰감 마크                           │
│    - 보안/개인정보 보호 약속              │
│    - "◆◆명이 이미 본 영상"             │
└─────────────────────────────────────────┘
          ↓
┌─────────────────────────────────────────┐
│ 4. CTA 버튼 (비디오 재생 중 나타남)    │
│    - 명확한 행동 지시문                   │
│    - "지금 가입하고 비법 배우기"         │
└─────────────────────────────────────────┘
```

### 마비즈 CRM 구현 체크리스트

```typescript
// landing-pages/new/page.tsx 수정 항목

□ 에디터 모드에 "비디오" 탭 추가
  - 현재: "이미지형" + "HTML형"
  - 추가: "비디오형" (YouTube 임베드)
  
□ 비디오 재생 시간 입력 필드
  - formConfig.videoConfig = { 
      url: string, 
      duration: "MM:SS",
      thumbnailUrl?: string 
    }

□ 자막 HTML 에디터 (비디오 하단)
  - formConfig.subtitlesHtml: string

□ CTA 버튼 (비디오 중앙)
  - 버튼 텍스트 + 색상 + 위치(하단 고정)

□ 진행도 바 애니메이션
  - CSS: width 0% → 100% (비디오 재생시간 동기)
```

### 크루즈 여행 예시 구성

```
제목: "크루즈 여행 비용 50% 줄이는 CEO의 비법"
비디오: CEO 신민형 5분 30초 영상 (YouTube)
자막: "저도 처음엔 100만원대였어요...
      근데 이 기술을 알고부턴 50만원대로!"
신뢰: "✓ 개인정보 보호 약속 | ✓ 357명 이미 봤어요"
CTA: "지금 가입하고 비법 배우기" (초록색 버튼)
```

### 심리학 원리 (왜 15-35% 전환?)

| 심리학 원칙 | 효과 | 마비즈 적용 |
|-----------|------|-----------|
| **감정 이입** | 비디오 = 얼굴, 목소리, 표정 | CEO 출연 |
| **FOMO** | 진행도 바가 "지금 봐야 함" 신호 | 빨강→초록 변화 |
| **시간 압박** | "영상 끝나면 사라져?" 느낌 | 재생시간 표시 |
| **사회증명** | "357명이 봤다" = 신뢰 | 조회 수 실시간 업데이트 |
| **긴급성** | 음소거 자동재생 = 놓칠까 빨리 봄 | autoplay + muted |

## 2️⃣ Squeeze Page — 이메일 극도로 단순 수집

### 개요
**심리학**: 선택지 최소화 = 심리적 저항 제거
**전환율**: 20-50% ← **가장 높음!**
**추천 상황**: 리드생성 (무료 가이드 제공)

### 마비즈 CRM에서의 필수 블록

```
┌─────────────────────────────────────────┐
│ 1. 극도로 명확한 제목 (ONE LINER)      │
│    "크루즈 전문가가 알려주는              │
│     50% 할인 비법 (무료)"               │
│    - 글자 크기: 32-48px                 │
│    - 색상: 매우 어두운 네이비            │
└─────────────────────────────────────────┘
          ↓
┌─────────────────────────────────────────┐
│ 2. 신뢰감 이미지 1개만                  │
│    - CEO 얼굴 사진 OR 상품 이미지       │
│    - 400×400px 정사각형 추천           │
└─────────────────────────────────────────┘
          ↓
┌─────────────────────────────────────────┐
│ 3. 3-5줄 설명 (너무 길면 안 됨!)      │
│    "상담사들이 절대 말해주지 않는        │
│     예약 기술을 지금 배우세요."         │
│    - 한 줄 최대 50자                     │
└─────────────────────────────────────────┘
          ↓
┌─────────────────────────────────────────┐
│ 4. 이메일 입력 필드 ONLY                │
│    (이름은 선택사항)                    │
│    - placeholder: "you@example.com"     │
│    - "필수" 강조 금지                   │
└─────────────────────────────────────────┘
          ↓
┌─────────────────────────────────────────┐
│ 5. "지금 무료로 받기" 버튼               │
│    - 버튼 색상: 브랜드 원색              │
│    - 버튼 크기: 전체 너비 (모바일 기준) │
│    - 마우스 호버: 살짝 어두워짐         │
└─────────────────────────────────────────┘
          ↓
┌─────────────────────────────────────────┐
│ 6. 신뢰 마크 (footer)                   │
│    "✓ 개인정보 100% 보호                │
│     ✓ 언제든 수신거부                   │
│     ✓ 스팸 절대 안 보냅니다"            │
└─────────────────────────────────────────┘
          ↓
└─ 네비게이션: 없음! (다른 링크 금지) ──┘
```

### 마비즈 CRM 구현 체크리스트

```typescript
// landing-pages 폼 필드 설정

□ 폼 필드 최소화 옵션 추가
  formConfig = {
    squeezeMode: "email_only" | "name_email" | "full",
    // "email_only" = 이메일만 (20-50% 전환)
    // "name_email" = 이름+이메일 (15-30% 전환, 10-15% 저하)
    // "full" = 모든 필드 (5-10% 전환, 권장 안 함)
  }

□ "무료" 텍스트 자동 강조
  - CTA 버튼에 "무료로" 텍스트 자동 포함

□ 네비게이션 숨김 옵션
  formConfig = {
    hideNavigation: true,
    // → header/footer/sidebar 모두 숨김
    // → 뒤로가기도 안 보이게 (CSS display:none)
  }

□ 신뢰 마크 자동 표시
  formConfig.trustMarks = [
    "privacy",     // ✓ 개인정보 보호
    "unsubscribe", // ✓ 언제든 수신거부
    "no_spam"      // ✓ 스팸 절대 없음
  ]
```

### 크루즈 여행 예시

```
제목: "크루즈 전문가가 알려주는 50% 할인 비법 (무료)"
이미지: CEO 신민형 프로필 사진
설명: "상담사들이 절대 말해주지 않는
      예약 기술을 지금 배우세요.
      무료 가이드 다운로드."
필드: email@example.com (이메일만)
버튼: "지금 무료로 받기"
신뢰: ✓개인정보 보호 | ✓언제든 수신거부 | ✓스팸 없음
```

### 왜 20-50% 전환율인가?

| 심리학 | 효과 | 적용 |
|-------|------|------|
| **선택지 최소화** | 1개만 선택 = 결정 쉬움 | 이메일만 입력 |
| **"무료" 마법** | 리스크 0 = 심리적 저항 제거 | 명시적 "무료" |
| **교환의 공정성** | "메일 주면 가이드 주겠다" | 명확한 거래 |
| **시각적 단순함** | 다른 선택지 없음 = 포기 불가 | 네비게이션 숨김 |
| **신뢰 마크** | "정말 스팸 안 보내겠네" | Footer 신뢰 마크 |

## 3️⃣ Webinar LP — 웨비나 등록 페이지

### 개요
**심리학**: 시간 제한 + 구체적 교육 내용 = 높은 등록율
**전환율**: 10-25%
**추천 상황**: 고객 교육/상담 (웨비나 개최 시)

### 필수 블록 구조

```

          ↓

## 4️⃣ One-Page Funnel — 긴 스크롤 1페이지

### 개요
**심리학**: 긴 스크롤 = 심리적 투자 증가 → 구매 확률 상승
**전환율**: 3-8% (낮지만 **AOV 매우 높음**, 예: 50만원~)
**추천 상황**: 고액 상품 (1회 결제 100만원+)

### 필수 섹션 순서 (Copywriting 기반)

```
Section 1: HERO (상단 5초 임팩트)
├─ 큰 제목 (28px 이상)
├─ 서브헤드라인
└─ "스크롤해서 전체 보기" CTA

     ↓ (스크롤)

Section 2: PROBLEM (문제 정의, 1-2분 읽기)
├─ "크루즈 너무 비싸...."
├─ "예약 복잡해..."
└─ 고객 통증 3-5가지 나열

     ↓

Section 3: STORY (창립자 스토리, 감정 이입)
├─ "저도 처음엔..."
├─ 과거: "100만원대..."
└─ 현재: "이제 50만원대!"

     ↓

Section 4: SOLUTION (해결책 소개)
├─ "우리가 어떻게 했는가"
├─ 프로세스 3단계
└─ 이미지 2-3개

     ↓

Section 5: SOCIAL PROOF (고객 후기, 최대 5개)
├─ 이름 + 사진 + 평점 + 후기
├─ "257명이 이미 가입"
└─ 평점 평균 표시 (⭐⭐⭐⭐⭐)

     ↓

Section 6: PRODUCT DETAILS (상세 설명)
├─ 포함된 것 (3-5개)
├─ 선실 옵션 사진 2-3개
└─ 일정 상세

     ↓

Section 7: PRICING (가격 + 보너스)
├─ "$199/인 (원가: $399)"
├─ "특별 보너스:"
│  └─ 여행 가이드 3권
│  └─ 1:1 상담 2회
│  └─ 짐싸기 체크리스트
└─ 남은 할인 시간 (타이머)

     ↓

Section 8: FAQ (자주 묻는 질문, 5-8개)
├─ 아코디언 형태
└─ 각 질문별 답변 2-3문장

     ↓

Section 9: CTA (최종 구매 버튼)
├─ "지금 예약하기"

## 5️⃣ Tripwire — 저가 상품 → 즉시 업셀

1. **사용자는 형식만 선택** → 나머지 심리학은 자동
2. **전환율이 데이터 기반** → 명확한 목표 설정 가능
3. **A/B 테스트가 쉬움** → 형식별 성과 비교 가능
4. **경매체계 구축** → "Squeeze로 리드 → VSL로 판매" 자동화

**결과: 마비즈 CRM이 "고전환 랜딩페이지 빌더"로 포지셔닝 됨**
