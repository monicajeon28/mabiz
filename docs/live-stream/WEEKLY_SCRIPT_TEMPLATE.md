## 📺 구성 (60분 타임테이블)

### PHASE 1: 오픈 (2분)
**심리학**: 공감 | Grant Cardone L0 (부재중 고객 재활성화)

**스크립트**:
```
"안녕하세요, 모든 분들을 환영합니다. 
저는 마비즈의 [이름]입니다.

오늘 저는 정말 중요한 주제를 가져왔습니다.
여러분 중에, 이런 경험 있으신가요?

[고객 문제 공감 1]:
- 가족과 함께 휴가를 가고 싶은데 시간이 없어서 못 간다
- 여행 준비가 너무 복잡해서 미루고만 있다
- 좋은 가격인 것 같은데 정말 괜찮은지 확신이 안 선다

만약 이런 생각이 드신다면, 
오늘 방송 끝나기 전에 해결책을 찾게 될 겁니다."

[아이컨택 3초 공백]
"그럼 시작할게요."
```

### PHASE 2: 문제 인식 (3분)
**심리학**: 증거/손실회피 | Grant Cardone L6 (타이밍 손실회피)

**스크립트**:
```
"지금 이 순간, 한국인의 80%는 휴가를 계획하지 않습니다.
아시나요? 왜일까요?

[근거 3가지]:
1. 준비 비용 — 평균 가족당 500만 원 필요
2. 준비 시간 — 여권, 비자, 호텔, 항공... 최소 2주 소요
3. 결정 피로 — '어디가 나을까?' 고민만 1개월

그 사이, 3명 중 1명은 '다음에 하자'라고 미뤄버립니다.

그리고 미뤄진 사람들은?
- 5년 후에도 같은 곳에서 같은 일을 합니다
- 가족과 추억을 못 만듭니다
- 나이가 들어 더 힘들어집니다"

[청중 시선 훑기 3초]
```

### PHASE 3: 해결책 제시 (5분)
**심리학**: 해결책 신뢰 | PASONA Solution + SPIN 질문

**스크립트**:
```
"하지만, 우리 마비즈 고객들은 다릅니다.

[사례 1 — 저가 고객]:
'김미영님, 56세, 주부'
- 예산: 400만 원 이하
- 문제: 남편 설득 안 됨
→ 마비즈 솔루션: 가족 패키지 (부부 + 자녀 3명) 350만 원
→ 결과: 3개월 만에 예약 완료 ✓

[사례 2 — 효도 여행 고객]:
'이순신님, 62세, 회사원'
- 예산: 1000만 원
- 문제: 부모님 체력, 의료 지원 필요
→ 마비즈 솔루션: 의료진 동반 + 휠체어 지원 패키지
→ 결과: 부모님 만족도 9.5/10 ✓

[사례 3 — 신혼 고객]:
'박준호, 28세, 신입사원 + 신부'
- 예산: 600만 원
- 문제: 휴가 일정 너무 짧음 (3일)
→ 마비즈 솔루션: 3박 4일 '꿀 같은 신혼' 프로젝트
→ 결과: 부부 친밀도 score 90점 달성 ✓

**이게 어떻게 가능한가?**

우리는 다음을 제공합니다:
✓ 1:1 맞춤 여행사 담당자 (1년 지속)
✓ 선결제 없음 (신뢰 보증)
✓ 72시간 취소 환불 (안심 보장)
✓ 가족별 의료 상담 무료 제공

[청중과 눈 맞춤 5초]
```

[만족도 영상]:

## 🎥 라이브 기술 체크리스트

- [ ] 카메라 (1080p 최소, 30fps)
- [ ] 마이크 (잡음 제거, 게인 -6dB 설정)
- [ ] 배경 (깔끔한 사무실 또는 여행 배경)
- [ ] 조명 (얼굴 기준 밝기 50Lux 이상)

- [ ] 채팅 모니터링 (전용 관리자 1명)
- [ ] 녹음 (백업용, 클라우드 저장)
- [ ] 인터넷 (업로드 10Mbps 이상 확인)
