# 라이브방송 진행자 실전 가이드
**대상**: 방송 진행자, 채팅 관리자, 기술팀  
**용도**: 실시간 방송 중 즉시 참조

---

## 🎙️ 진행자용 준비물 체크리스트 (방송 30분 전)

### 물리적 준비 (10분)
- [ ] 카메라: 테스트 촬영 → 해상도 1080p 확인
- [ ] 마이크: 음성 레코더로 테스트 → 배경음 -40dB 이하
- [ ] 조명: 얼굴에 그림자 없는지 확인 (밝기 60Lux 이상)
- [ ] 배경: 깔끔한지 확인 (책장/여행사진/회사로고 배경)
- [ ] 옷: 진한 색상 + 단정한 모습 확인
- [ ] 화장: 윤기 제거 (방송용 파우더)

### 기술적 준비 (10분)
- [ ] 라이브 플랫폼 로그인 (YouTube/Facebook/Naver Live)
- [ ] 화면 공유 테스트 (PPT, 이미지 3장)
- [ ] 채팅 화면 옆에 띄우기 (관리자가 질문 전달할 테)
- [ ] 녹화 시작 버튼 위치 확인
- [ ] 음량 점검 (마이크: 게인 -6dB, 스피커: 0dB)

### 정신적 준비 (5분)
- [ ] 스크립트 마지막 읽기 (인사말 3회)
- [ ] 심호흡 5회 (들이마시기 4초 → 내쉬기 4초)
- [ ] 웃음 연습 (자연스러운 표정 2-3회)
- [ ] '시작하겠습니다' 인사 음성 녹음 후 재생 (톤 확인)

## 📱 실시간 진행 타이밍 (분 단위)

### 18:50 - 준비 시간 (입장 유도)
```
[채팅 메시지 - 5분마다 반복]

"🎬 라이브방송 시작까지 10분입니다!
- 지금 입장하세요
- 댓글을 남기면 추첨 경품이 있습니다
- 오늘 주제: '가족 여행, 이제는 미룰 수 없습니다'

⏰ 19:00 정확히 시작합니다!"

[목표 시청자 수]
18:50 → 30명 이상
18:55 → 50명 이상
19:00 → 100명 이상
```

### 19:00 - 오픈 (진행자)
```
[진행자 인사 — 45초]

"안녕하세요, 마비즈의 [이름]입니다!
오늘 특별히 우리 가족들을 위해 
중요한 주제를 가져왔어요.

모두 환영합니다. 👋

[화면 공유 시작 — 제목 슬라이드]"

[채팅 관리자 메시지]
"🎁 **입장 이벤트**
지금 입장하신 분 중 추첨으로 5명에게 
'여행 사진 에디팅권 (50만 원 상당)' 증정합니다!
댓글로 #신청 을 입력해주세요!"
```

### 19:02~19:25 - Phase 1-4 진행
```
[진행자 톤 포인트]

1. Phase 1 (공감): 천천히, 눈 맞춤 3초
2. Phase 2 (증거): 숫자 강조, 손가락 제스처
3. Phase 3 (해결): 사례 영상 재생 (30초 x 3)
4. Phase 4 (신뢰): 권위성 이미지 보여주기

[채팅 관리자 메시지 — 5분마다]
"💬 지금 댓글 남기신 분들:
@김미영님: '저도 이런 생각했어요!'
@박준호님: '비용이 정말 저렴하네요'
@이순신님: '부모님이랑 꼭 가고 싶었는데!'

[하이라이트 피드에 올려드리겠습니다]"

[진행 중 특이사항 처리]
- 음소거: 마이크 끄고 2초 쉬기 (자연스러움)
- 실수: 웃고 "다시 말씀드릴게요"라고 계속 (감정 전달)
- 채팅 폭주: 관리자가 중요 질문만 필터링
```

### 19:25~19:30 - Phase 5 (긴박감 + 클로징)
```
[진행자 표정: 진지함 → 희망]

"자, 이제 마지막 10분입니다.

[카운트다운 이미지 보여주기]
현재까지 신청: OO명
남은 자리: OO명

만약 지금 신청하지 않으면?
다음 달에는 30% 비싸져요.

[직접 지목]
'파란 셔츠 입으신 분! 지금 신청하실래요?
아니면 5년 뒤에 후회하실래요?'

[클로징 문구]
'자, 그럼 신청하는 3가지 방법을 다시 한 번
보여드릴게요.'"

[채팅 관리자 집중 모니터링]
- 신청 링크 고정 메시지 (상단)
- 성공 알림: "@김미영님 신청 완료! 축하합니다! 🎉"
- 긴박감 강화: "남은 자리 15명! 빠르세요!"
```

### 19:30~19:50 - Q&A 시간
```
[진행자 톤: 친절 + 전문성]

"자, 이제 여러분의 질문 시간입니다!

[채팅에서 질문 추출 — 관리자가 전달]

관리자: "화면에 '@김민지님 질문입니다.
'취소 환불이 정말 가능한가요?'라고 하셨어요."

진행자: "좋은 질문이네요!
네, 저희는 72시간 내 전액 환불을 보증합니다.
이건 우리 약관에 명시되어 있고,
지난 5년간 100% 이행한 기록이 있습니다.'"

[자주 묻는 질문 (FAQ) 준비]

Q1: "비자는 어떻게 처리하나요?"
→ "저희가 전부 해드립니다. 여권만 있으면 돼요."

Q2: "어린 아이도 갈 수 있나요?"
→ "네, 0세부터 가능합니다. 
   아기 전담 가이드도 있고, 
   기저귀 교체실도 준비되어 있어요."

Q3: "다른 여행사보다 비싼가요?"
→ "처음엔 좀 비쌀 수 있습니다.
   하지만 1년 담당자, 의료 지원, 24시간 카톡 상담을 생각하면
   2배 이상의 가치를 받습니다.
   
   실제로 우리 고객들은 2번, 3번 와요."

Q4: "혼자 가도 되나요?"
→ "물론입니다! 혼자의 시간 패키지도 있어요."

Q5: "결혼식을 계획 중인데, 신혼여행과 통합이 가능한가요?"
→ "가능합니다! 신혼여행 + 웨딩사진촬영 
   통합 패키지가 있어요."

[관리자 역할]
- 질문 화면 보여주기: "이건 정말 많이 나오는 질문이네요"
- 진행자 분위기 전달: "지금 채팅이 뜨거워요!"
- 신청 진행 상황 전달: "방금 2명이 더 신청했어요!"
```

## 🚨 실시간 문제 대응 매뉴얼

### 상황 1: 음성이 끊겼을 때
```
[진행자 대응 — 5초 이내]

1단계: 마이크 확인
   → 마이크 뽑았나? 다시 꽂기
   → 게인이 0dB? -6dB로 조정

2단계: 소프트웨어 재시작 (Audacity, OBS 등)
   → 라이브 중단 (최대 30초)
   → "죄송합니다! 기술적으로 잠깐 문제가 있었어요.
      다시 시작하겠습니다!"

3단계: 백업 마이크 사용
   → 메인 마이크 실패 시, 노트북 내장 마이크로 전환
   → 음질은 좋지 않지만 소통 가능

[채팅 관리자]
"기술팀이 대응 중입니다. 잠깐만 기다려주세요!"
```

### 상황 2: 인터넷 연결이 끊겼을 때
```
[진행자 대응 — 즉시]

1단계: 모바일 핫스팟으로 전환
   → 라이브 중단 (최대 10초)
   → "죄송합니다! 잠깐만요!"

2단계: 재연결 후 복구
   → "다시 시작합니다!"

[채팅 관리자]
"WiFi 문제가 있었어요. 모바일로 다시 연결했습니다.
계속 봐주세요!"

⚠️ 주의: 3분 이상 끊기면, 라이브 자동 종료
    → 녹화본 유튜브에 업로드 (12시간 이내)
```

### 상황 3: 채팅이 악플/스팸으로 폭주할 때
```
[채팅 관리자 대응]

1단계: 악플 즉시 삭제
   → '여행사' 관련 스팸은 신고 후 차단
   → 욕설/광고성 댓글: 숨김 처리

2단계: 관리 공지
   "안녕하세요! 
   우리 방송은 건전한 대화를 위해 
   스팸 댓글은 자동 삭제되니 양해 부탁드립니다."

3단계: 진행자에게 알리지 않기
   → 진행에 방해 X
   → 배경에서 관리자가 조용히 처리
```

### 상황 4: 시청자가 갑자기 많이 입장할 때 (1000명 이상)
```
[서버 부하 대응]

1단계: 화질 자동 조정
   → 1080p → 720p (자동)
   → 30fps → 24fps (필요 시)

2단계: 채팅 속도 조절
   → '1초에 1개 댓글만 표시' 필터링
   → 관리자: 중요 질문만 진행자에게 전달

[채팅 관리자]
"와! 시청자 분들이 정말 많이 입장하셨어요!
지금 1500명 동시 시청 중입니다!"
```

### 상황 5: 진행자가 스크립트를 깜빡했을 때
```
[진행자 대응]

1단계: 자연스럽게 넘어가기
   → "아, 중요한 건 빠뜨렸어요. 다시 말씀드릴게요."
   → 재미있는 톤으로 (어색함 X)

2단계: 옆에 스크립트 놓기 (다음부터)
   → 타블릿에 스크립트 띄우기
   → 카메라에 보이지 않게 배치

[채팅 관리자]
"진행자님이 중요한 부분 다시 말씀해주고 있어요! 
놓친 분들은 지금 들으세요!"
```
