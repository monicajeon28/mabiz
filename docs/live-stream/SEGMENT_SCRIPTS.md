# 세그먼트별 라이브방송 스크립트 (저가 / 효도 / 신혼)

---

## 🎯 Segment A: 저가 고객 (30-40대 직장인 + 주부)
**특징**: 예산 제약 (300-500만 원) | 실리 중심 | 가족 중심  
**심리학**: L1 (가격 이의) + L6 (타이밍 손실회피)

### Phase 1: 오픈 (고객 공감)
```
"안녕하세요! 
제 이름은 최윤아인데, 저도 한때 '가난했던' 사람입니다.

결혼 초기만 해도, 아이 둘을 키우면서 
'우리 가족 언제 여행 가봐?'라고 했던 기억이 있어요.

"남편, 휴가비가 500만 원대 오르네"
"그럼 이번엔 못 가야겠네"

그런 대화, 하셨던 거 없어요?"
```

### Phase 2: 문제 인식 (예산 현실)
```
"지금 한국 가족의 평균 여행비는:
- 항공: 150만 원/1인
- 호텔: 100만 원/3박
- 식사: 50만 원
- 액티비티: 30만 원

합계? **5-600만 원**

4인 가족 기준으로 보니까요.

그래서 우리 가족들은?
연 2회 국내 여행만 다니다가, 
해외는 '5년 뒤에 하자'고 미룹니다.

그 사이에 아이들은 커버려요.
부모님 다리도 약해져요."
```

### Phase 3: 해결책 (가격 전략)
```
"저희 마비즈는 이를 해결하기 위해 
**'저가 전문팀'**을 만들었습니다.

[실제 사례]
'김미영님 가족 (4명, 서울거주, 자녀 8/10세)'

원래 비용:
- 타사 패키지: 580만 원
- 마비즈 가격: 350만 원 (40% 할인!)

**어떻게 가능한가?**
1. 우리가 현지 항공사와 직계약 (중간상인 제거)
2. 3박 일정을 2.5박으로 최적화 (호텔 1박 절감)
3. 아침식사는 호텔, 점심은 현지 로컬 (50% 저렴)
4. 명품매장 투어 대신 현지 시장 체험

결과?
'같은 품질의 추억을 350만 원에!'

[또 다른 사례]
'이순신님 가족 (부부, 효도 여행, 부모님 동반)'

원래 비용: 650만 원
마비즈 가격: 420만 원 (35% 할인!)

이유:
- 부모님 체력 고려 (무리 없는 일정)
- 의료비 별도 처리 (건강보험 청구)
- 단체 구성으로 볼륨 할인

**가격 보증**:
'만약 같은 조건의 다른 여행사에서 
10만 원이라도 저렴하면,
우리는 그 가격에서 추가 10% 할인해드립니다.'"
```

### Phase 4: 신뢰 강화 (실제 후기)
```
"저희가 자랑하는 건 숫자가 아니라, 
가족들의 웃음입니다.

[영상 1 - 40대 부부 후기 (30초)]
'처음엔 걱정했어요. 너무 싼데 괜찮을까 했거든요.
근데 도착해보니까... 와, 이렇게 깔끔할 수가 있네?
호텔도 깨끗하고, 가이드도 친절하고,
아이들이 "엄마, 또 가자!"라고 했어요.'

[영상 2 - 30대 여성 후기 (30초)]
'평사원 월급으로도 해외여행 가능하다는 걸 
처음 알았어요.
마비즈 담당자가 "300만 원짜리 패키지 있어요"라고 
제안해줄 때 진짜 깜놀했어요.
근데 다녀와보니 정말 좋았어요!'

[영상 3 - 부부 후기 (30초)]
'가격만 보면 마비즈인 줄 몰랐어요.
서비스는 왜 이렇게 좋아?
담당자가 우리 가족을 정말 챙겨주는 게 느껴졌어요.'"
```

### Phase 5: 긴박감 + 클로징
```
"**지금 신청하시면:**

✓ 350만 원 → 280만 원 (20% 추가 할인)
✓ 카드 수수료 0% (6개월 할부 무이자)
✓ 준비금 100만 원만 내면 됩니다 (나머지 출발 2주 전)

**남은 자리: 22명**

여름 휴가는 못 미루세요.
아이들이 자라버려요.

지금 신청하세요."
```

## 👴👵 Segment B: 효도 여행 고객 (50-70대 부모님을 모시는 자녀)
**특징**: 부모님 건강 우선 | 비용 상관없음 | 감정 중심  
**심리학**: L9 (의료/신뢰) + L7 (동반자 설득)

### Phase 1: 오픈 (부모님 공감)
```
"안녕하세요! 저는 2년 전 아버지가 돌아가셨어요.

마지막으로 아버지와 여행을 갔었는데,
그것이 우리 가족이 함께 가진 마지막 추억이었어요.

여러분 부모님은 어떠세요?
'언제 시간이 나면 여행 가자'고 하면서도,
계속 미루고 있진 않으신가요?"
```

### Phase 2: 문제 인식 (시간의 소중함)
```
"통계상 한국인의 부모님 동반 여행율은 12%입니다.
88%는 여행을 못 갑니다.

왜일까요?

1. **건강 불안** — '부모님이 급 악화되면 어쩌지?'
2. **준비 복잡** — 여권, 예방접종, 의료보험...
3. **결정 어려움** — '어디가 좋을까? 부모님이 힘들진 않을까?'

그 사이에 시간은 흘러갑니다.

제 아버지도 '내년에 일본 여행 가자'고 했어요.
그 내년은... 오지 않았습니다."
```

### Phase 3: 해결책 (의료 + 안심)
```
"우리 마비즈는 **'효도 여행 전문'** 팀이 있습니다.

[전담 의료팀]
- 현역 의사 1명이 여행 전체 동반
- 의료비 100% 마비즈 부담
- 응급상황 15분 내 병원 이송 보증

[실제 사례]
'이순신님 (62세, 아버지 87세, 어머니 85세) 일본 오사카 여행'

상황:
- 아버지: 당뇨병 + 고혈압 약 복용
- 어머니: 관절염으로 휠체어 필요

마비즈 준비:
1. 여행 3개월 전 의료 상담 (담당 의사 배정)
2. 현지 협력 병원 사전 등록
3. 의료진 동반 (간호사 1명)
4. 휠체어 카를로 된 여행 일정 재구성

여행 중 발생한 상황:
→ 아버지 혈당 저하 발생
→ 마비즈 간호사가 즉시 포도당 투여
→ 30분 만에 회복
→ 여행 계속 진행!

결과:
'아들아, 이 여행 덕분에 
엄마, 아빠가 정말 행복했다.'

부모님 만족도: 9.8/10"
```

### Phase 4: 신뢰 강화 (권위성)
```
"우리 마비즈는:

✓ 대한의료관광협회 인증 (의료진 배경 검증)
✓ 서울대학교 의과대학원 자문 (의료 프로토콜)
✓ 300+ 병원 B2B 파트너 (응급 네트워크)
✓ 효도관광 5년 연속 '최우수 업체' 선정

[의료 사건 분석]
여행 중 발생 가능한 상황별 대응 매뉴얼:

▸ 고혈압 상승: 현지 약국 즉시 방문
▸ 당뇨 저하: 응급 포도당 투여
▸ 심장 응급: 119 = 현지 119, 대학병원 이송
▸ 뇌졸중 의심: 응급실 CT 스캔 후 뇌혈관 센터 이송

**모든 의료비는 마비즈가 부담합니다.**"
```

### Phase 5: 감정적 클로징
```
"제 아버지는 일본 여행을 못 가셨어요.
하지만 여러분의 부모님은 다릅니다.

아직 시간이 있습니다.

**지금 신청하면:**
✓ 의료진 동반 (100% 무료)
✓ 휠체어 준비 (무료)
✓ 여행 중 약값 전액 환급 (무료)
✓ 가족 단체사진 (고급 사진관, 무료)

**예약 마감: 5월 31일**
(6월부터는 여름휴가 시즌 때문에 불가)

여러분의 부모님을 위해, 
지금 신청하세요.
나중에는 후회합니다."
```

## 💑 Segment C: 신혼 고객 (25-35세 신혼부부)
**특징**: 로맨스 추구 | SNS 공유 | 경험 중심  
**심리학**: L5 (자기투영) + L10 (즉시 구매)

### Phase 1: 오픈 (신혼 공감)
```
"안녕하세요! 저는 마비즈의 박미선이고요,
올해 결혼 5주년을 맞이하는 기혼자입니다.

그런데 제 신혼 때를 생각해보면...
'와, 정말 후회 많이 했거든요.'

왜냐하면 신혼은 **단 한 번**이기 때문이에요.

새 신부, 새 신랑 분들 계세요?
**지금이 가장 중요한 시간입니다.**"
```

### Phase 2: 문제 인식 (신혼의 소중함)
```
"신혼 부부에게 중요한 건 뭘까요?

✓ 로맨스 — 둘만의 추억
✓ 사진 — 평생 간직할 모습
✓ 경험 — 새로운 것을 함께

근데 현실은?
- 신혼 부부 70%는 '국내 여행' 또는 '여행 안 함'
- 신혼여행도 '빠른 일정' (3박 이하)
- 사진은 '자기들끼리 셀카'만 찍음

그리고 후회합니다.

'우리 할걸... 더 좋은 곳으로 갔을 걸...'

인생에서 가장 행복한 시간이 신혼인데,
그것을 제대로 누리지 못하는 거죠."
```

### Phase 3: 해결책 (신혼 스토리)
```
"마비즈의 '신혼 맞춤 프로젝트'는 다릅니다.

[프로젝트명: '꿀 같은 신혼 프로젝트']

구성요소:
1. **숨겨진 로맨스 명소** (인스타 핫플 X, 진정한 명소만)
2. **전문 사진작가** (신혼 전문 포토그래퍼 동반)
3. **둘만의 저녁** (특별한 디너, 촛불, 와인)
4. **신체 친밀도 관리** (아로마, 스파, 마사지)

[실제 사례 — 박준호 + 이지은 부부]

배경:
- 나이: 28세 + 27세 (신입사원 + 디자이너)
- 예산: 600만 원
- 꿈: '평생 간직할 신혼사진' + '로맨틱한 경험'

마비즈 제안:
- 3박 4일 방콕 + 푸켓 (보트 여행)

여행 일정:
▸ Day 0: 서울 → 방콕 (비행 중 스파)
▸ Day 1: 방콕 '야경 루프탑 디너' (촛불, 와인)
▸ Day 1 밤: 전문 사진작가 강변 촬영 (30컷, 보정 완료)
▸ Day 2: 푸켓 보트 여행 (스노클링 + 일몰 촬영)
▸ Day 2 밤: 해변 초콜릿 스파 (부부 커플 테라피)
▸ Day 3: 로컬 시장 + 손잡고 산책 + 사진
▸ Day 3 밤: 전문가 영상 촬영 (약혼식 비디오 수준)

결과:
'이게 신혼여행이구나...'
— 박준호

'평생 이 사진들 볼 때마다
우리의 그때가 생각날 것 같아요'
— 이지은

부부 친밀도 스코어: 90/100 (업계 평균 62)
인스타그램 공유: 240개 해시태그 (입소문 확산)"
```

### Phase 4: FOMO 강화 (동료 공감)
```
"지금 저희 클라이언트들의 신혼사진을 보면?

[인스타 슬라이드]
- @hongmikyung: '우리의 그날의 노을 🌅💕' (좋아요 12K)
- @parkjunho_wedding: '푸켓 스노클링 중 키스 🐚✨' (좋아요 8.9K)
- @jieun_moments: '신혼여행의 진정한 의미를 깨달았어요' (좋아요 6.2K)

이들은 모두 마비즈 클라이언트입니다.

그리고 그들의 친구들은?
댓글: '오, 어디에요?', '저희도 가고 싶어요!', '추천 좀 받을래요?'

당신의 신혼사진도 이렇게 될 수 있습니다."
```

### Phase 5: 신혼의 특별함 강조
```
"인생에서 신혼은:
- 한 번뿐이에요
- 다시 오지 않아요
- 가장 아름다울 때예요

**지금 신청하면:**
✓ 신혼 전문 사진작가 동반 (120만 원 상당)
✓ 부부 스파 + 마사지 무료 (80만 원 상당)
✓ 영상 촬영 + 편집 무료 (200만 원 상당)
✓ 신혼사진 무광 포토북 (50만 원 상당)

**총 가치: 450만 원**
**지금 가격: 600만 원 (포함 가격)**

더 좋은 소식?
**지금 신청자만: 추가 10% 할인 (540만 원)**

당신의 신혼을 누리세요.
나중에 후회하지 마세요.

지금 신청하세요."
```

## 📊 세그먼트별 비교표

| 항목 | 저가 고객 | 효도 고객 | 신혼 고객 |
|------|---------|---------|---------|
| **예산** | 300-500만 | 600-1000만 | 500-800만 |
| **핵심 심리** | L1+L6 (가격+타이밍) | L9+L7 (의료+동반) | L5+L10 (자기투영+즉시) |
| **오픈 방식** | 개인 공감 | 감정적 이야기 | 신혼의 소중함 |
| **증거 방식** | 실제 후기 (가격 절감) | 의료 사례 (응급 대응) | 인스타 공유 (FOMO) |
| **CTA** | '지금 신청하세요 (할부)' | '미루지 마세요 (시간)' | '누리세요 (신혼)' |
