## 💰 연간 크루즈 비용 비교

### 호텔 vs 크루즈 (연간 기준)

```
연간 휴가 2회 (5박씩) 기준:

┌─────────────────────────────────────┐
│ 호텔 여행 연 2회 (5박)             │
├─────────────────────────────────────┤
│ 호텔료:     $1,000 × 2 = $2,000    │
│ 음식:       $500 × 2 = $1,000      │
│ 활동/관광:  $500 × 2 = $1,000      │
│ 이동비:     $300 × 2 = $600        │
│ 추가비용:   $200 × 2 = $400        │
│                                      │
│ 연간 총액:  $5,000                │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│ 크루즈 연 2회 (5박)                 │
├─────────────────────────────────────┤
│ 크루즈료:   $900 × 2 = $1,800      │
│ 음식:       포함 (올인클루시브)     │
│ 활동:       포함 (올인클루시브)     │
│ 이동비:     0 (배가 이동)          │
│ 추가비용:   $100 × 2 = $200        │
│                                      │
│ 연간 총액:  $2,000                │
└─────────────────────────────────────┘

당신의 절약: $3,000/년 (60% 절감!)
5년 누적:   $15,000
10년 누적:  $30,000

"5년을 다시 간다면?
당신은 $15,000을 절약해요."
```

### 첫 번째 vs 두 번째 구매

```
당신이 이미 한 번 갔으니까:

위험:         "괜찮았나?" 불안
의사결정:     30% 확률로 미룸

[두 번째 구매 - 다음에 하는 당신]
비용:         $950 (오션뷰) ← 15% 할인
위험:         "알지, 정말 좋았어" 확신
의사결정:     70% 확률로 즉시 예약

당신을 위한 우리의 계산:

첫 가격: 1,200달러
재가격: 950달러
차이:   -$250 (20% 할인)

왜냐면:
- 당신은 '검증된 고객'이에요
- 우리가 비용 절감했어요 (마케팅 불필요)
- 당신이 주변에 홍보해줬어요 (입소문)

그 결과? 당신이 가장 싼 거래를 받아요.
```

## 🎁 멤버십 혜택 (첫 구매 vs 재구매)

### 첫 구매 고객 혜택

```
당신 (첫 번째):
├─ 가격: 정가 $1,200
├─ 포인트: 0점 (시작)
├─ 추가 할인: 없음 (신규 프로모션 제외)
├─ 우선 예약: 없음
└─ 총합: $1,200 지출

이유: 당신을 신뢰할 이유가 아직 없어요.
```

### 재구매 고객 혜택 (당신!)

```
당신 (두 번째 = 지금 신청할 때):
├─ 가격: 정가 $1,200
│         → 재구매 할인 -$200 = $1,000
│         → 멤버십 포인트 추가 -$50 = $950
├─ 포인트: 1,200점 누적 (다음 여행에 사용)
├─ 추가 할인: "멤버 전용" 시즌 프로모션 (15% 추가)
├─ 우선 예약: 신규 배 출시 시 우선 신청권
├─ 선물: 배 탑승 시 $50 크레딧
└─ 총합: $950 지출 + 1,200 포인트 누적

이유: 당신은 이미 '신뢰할 만한 고객'이에요.
      우리가 당신을 더 우대해요.

다음 여행 (3번째):
- 포인트 1,200점 사용 → 가격 $800
- 재구매 할인 추가 -$100 → $700
- 멤버 프로모션 15% -$105 → $595

연간 진행:
1번 가기:  $1,200
2번 가기:  $1,200 + $950 = $2,150 (avg $1,075)
3번 가기:  $1,200 + $950 + $595 = $2,745 (avg $915)
```

## 📊 멤버십 포인트 시스템

```
적립:
- 크루즈료의 10% 자동 적립
- $1,200 크루즈 = 120점 적립
- 1점 = $1 가치

사용:
- 다음 크루즈 예약 시 차감
- 스파/음식/액티비티 구매 시 사용
- 현금 환불 불가 (크루즈만 사용)

예시:
┌─────────────────────────────────────┐
│ 1차 여행 (5박)                       │
│ ├─ 지불: $1,200                    │
│ ├─ 적립 포인트: 120점               │
│                                      │
│ 2차 여행 (5박)                       │
│ ├─ 원래 가격: $1,200               │
│ ├─ 포인트 차감: -120점 ($120 절감) │
│ ├─ 재구매 할인: -$200              │
│ ├─ 총 지불: $880                   │
│ ├─ 새로운 포인트: 88점 적립        │
│                                      │
│ 3차 여행 (5박)                       │
│ ├─ 원래 가격: $1,200               │
│ ├─ 포인트 차감: -88점 ($88 절감)  │
│ ├─ 재구매 할인: -$200              │
│ ├─ 총 지불: $912                   │
│ └─ 포인트 계속 누적                 │
└─────────────────────────────────────┘

5년 후 (연 2회 = 10회 누적):
- 총 지출: $9,500 (대신 호텔은 $25,000)
- 절약액: $15,500
- 누적 포인트: 950점 = 다음 여행 거의 무료
```

## 📊 CRM 자동화 규칙 (L8 고객)

```
IF Contact.previousTrips >= 1 AND Contact.lens = 'L8'
THEN:
  - ContactLensL8Data 테이블 자동 생성
  - "재구매 할인" 자동 계산 (15-20%)
  - SMS Day 0-1 ROI 중심 발송
  - "포인트" 누적액 실시간 표시
  
IF Contact.lastTripDate < 2026-01-01 
   (6개월 이상 미예약)
THEN:
  - "다시 올 시기" SMS 자동 발송
  - 재구매 할인 + 포인트 재계산
  - 멤버 전용 프로모션 우선 제시

IF Contact.accumulatedPoints > 500
THEN:
  - "포인트로 다음 여행 반값" 제안
  - 포인트 만료 전 사용 권유
  - 다른 서비스(스파/음식) 포인트 사용 옵션

IF Contact.membershipStatus = 'INACTIVE'
   (12개월 미활동)
THEN:
  - "재가입" 특별 프로모션 발송
  - "추억의 배" 다시 타기 제안
  - 재가입 시 보너스 포인트 500점
```

## ✅ 재구매 체크리스트

```
당신의 재구매 결정을 위해:

□ 첫 여행 만족도 확인 (점수 8/10 이상?)
□ 가족 만족도 확인 (아이/배우자?)
□ 다음 여행 시기 결정 (언제?)
□ 예산 확인 (월 저축액?)
□ 포인트 누적액 확인 (몇 점?)
□ 재구매 할인 확인 (몇 % 할인?)

신청 전 최종 확인:

□ 비수기 시즌? (봄/가을 = 더 저렴)
□ 목적지 결정? (지중해/카리브/동남아?)
□ 배 선택? (같은 배 vs 새로운 배?)
□ 포인트 사용? (다음 여행 더 저렴하게)
```
