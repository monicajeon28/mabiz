# 크루즈 콜드콜 Opening 빠른 참고 카드
## 상담사 데스크용 (실시간 참고용)

**출력 권장:** A3 포스터, 상담사 모니터 옆에 부착  
**용도:** 통화 중 침묵 타이밍, 다음 단계 확인용  
**관련 전체 가이드:** docs/COLD_CALL_5_OPENING_SCRIPTS_COMPLETE.md

---

## 🎯 Opening 선택 (첫 10초 청취)

```
고객이 말한 키워드 → Opening 선택

"신혼입니다" / "결혼 기념일" → OPENING B
"가격이 비싸다" / "얼마예요?" → OPENING C  
"시간이 없어요" / "바빠요" → OPENING D
"부모님이랑..." / "효도" → OPENING E
일반 50-60대 또는 명확한 언급 없음 → OPENING A' ⭐
```

---

## 🔴 OPENING A' — 정식 Opening (기본값, 50-60대)

### PHASE 1 (0:00-0:45) — 경계심 해제

```
📞 "안녕하세요! OO님, 저 크루즈닷 상담원 민지라고 합니다.
     오늘 날씨 정말 좋네요. 혹시 지금 괜찮으세요?"

[⏱ PAUSE 2초] ← 고객 응답 대기

     "크루즈 여행에 관심이 있으셔서 저희 연락처를 
      남겨주신 거 맞죠?"

[⏱ PAUSE 1초] ← "네" 확인

     "제일 먼저 묻는 게 '안전한가?', '저렴한가?'예요.
      정상이고 중요한 질문입니다.
      
      지금 3-4분만 제 말을 들어주세요.
      모든 의심이 풀릴 거예요."
      
체크: 고객이 "네, 괜찮습니다" 또는 "좋아요" ✅
```

### PHASE 2 (0:45-1:45) — 신뢰 구축

```
📞 "저희 프로그램이 특이한데요.
     당신이 낸 돈이 저희 통장에 안 들어가요.
     신한은행 신탁 계좌에 들어가고,
     저희는 못 쓰는 돈이에요."

[⏱ PAUSE 2초] ← 정보 소화 대기

     "지금 3,000명 넘게 이용 중이고,
      앱스토어 평점은 4.8점이에요."

[⏱ PAUSE 1초]

체크: 고객이 놀라거나 안심하는 반응 ✅
```

### PHASE 3 (1:45-2:45) ⭐ 가장 중요 — YouTube 후킹

```
📞 "한 가지 여쭤봐도 될까요?
     혹시... 부모님이랑 아직도 제대로 여행을 
     한 번 못 가셨어요?"

[⏱ PAUSE 3초] ← 매우 길게, 고객 감정 변화 기다리기

고객: "네, 맞아요" 또는 침묵

     "저도 많이 봤는데요...
      그 '언젠가'가... 어느새 지나가버리더라고요.
      
      당신은 지금 몇 살이세요? 부모님은?"

[⏱ PAUSE 2초]

고객: "저는 50살, 부모님은 75살이요"

     "그렇군요. OO님, 질문 하나만 해도 돼요?
      
      지금 부모님과 함께할 수 있는 시간이...
      앞으로 얼마나 남아 있을 것 같아요?
      
      혹시 모를 일이 있을 수도 있잖아요.
      부모님이 지금처럼 여행할 수 있는 체력이...
      5년 뒤에도 같을까요?"

[⏱ PAUSE 4초] ← 아주 길게 (고객 깊은 생각 시간)

체크: 고객이 한숨 쉬거나 "정말이에요" 또는 침묵 ✅
     (= YouTube 후킹 성공, Mortality Salience 작동)
```

### ✅ PHASE 3 이후 자동 진입

```
📞 "그래서 저희가 프로그램을 만들었어요.
     
     부모님과 함께 여행 가는데,
     준비하는 데 1시간도 안 걸려요.
     
     저 하나의 전화면... 모든 게 다 됩니다.
     
     관심 있으세요?"

[⏱ PAUSE 2초]

성공 신호: 고객이 "궁금해요" 또는 "한번 들어볼게요"
```

## 🔵 OPENING B — 부부 타겟 (신혼/결혼기념일)

### 핵심 구조 (2분 30초)
```
1. "혹시 신혼이세요?" ← 즉시 확인
2. 결혼 후 여행 준비 경험
3. "당신은 여행 준비했고, 아내는?" ← Implication
4. "이 시간이 얼마나 남을까요?" ← 로맨틱 FOMO
5. "크루즈 = 우리 둘만의 시간"
```

### 침묵 타이밍
- 신혼 여부 확인 후: 1초
- "아내분은 피곤해하지 않았어요?" 후: 3초
- "신혼의 이 감정이 5년 뒤에도?" 후: 3초

---

## 🟢 OPENING C — 가격민감층 (자영업/사업가)

### 핵심 구조 (1분 50초)
```
1. "가격이 비싼 것 같다" ← 고객 말 먼저 듣기
2. 공감: "맞아요, 많은 분들이 그래요"
3. 재프레이밍: 호텔 vs 크루즈 포함 항목 비교
4. 깨달음 유도: "어? 크루즈가 반값인데?"
5. "왜 진작 몰랐지?" ← 자기 발견
```

### 침묵 타이밍
- 가격 비교 설명 후: 2초
- "어? 크루즈가 반값인데?" 후: 2초

---

## 🟡 OPENING D — 빠른 결정자 (직장인)

### 핵심 구조 (1분 20초)
```
1. "1분 가능하세요?" ← 시간 약속
2. "가장 번거로운 게 뭐예요?" (SPIN-Problem)
3. "그 때문에 잃어버리는 게?" (SPIN-Implication)
4. "그래서 저희가 있어요" (Solution)
5. "관심 있어요?" ← 즉시 결론
```

### 침묵 타이밍
- 최소 (3초 총합)
- 질문 후 고객 응답 대기만

---

## 🟣 OPENING E — 효도 고객 (부모님 모시기)

### 핵심 구조 (3분 10초) ⭐ 가장 강력
```
1. "부모님과 여행을 생각해본 적 있으세요?"
2. "언제쯤 가고 싶으세요?" ← "내년에..." 유도
3. "부모님 나이가 몇 살이세요?" ← Mortality 준비
4. "부모님이 앞으로 몇 년을 더 건강하게..." ← Mortality Salience
5. 4초 침묵 (깊은 생각)
6. "지금의 건강이 계속될 보장은 없다"
7. "그때가 마지막일 수도..." ← YouTube 후킹
8. "크루즈 = 효도의 완성"
```

### 침묵 타이밍 (13초 총합 - 가장 길게)
- 부모님 나이 언급 후: 1초
- "부모님이 앞으로 몇 년을..." 후: 4초 ⭐
- "그때가 마지막일 수도" 후: 3초
- "크루즈 = 효도" 후: 2초


## 🚨 위험 신호 (즉시 대응)

### 고객이 끊으려고 할 때
```
"OO님, 정말 1분만 더 주세요.
제가 한 가지 빠트린 게 있어서요."

→ Opening A'/E의 Mortality Salience 부분으로 즉시 이동
```

### "가격이 얼마예요?" 질문 들어올 때
```
"좋은 질문이에요. 
하지만 그 전에 한 가지만 말씀드리고 싶어요..."

→ Opening C로 전환 (포함 항목 비교)
```

### "이미 다른 데서 들었어요" 할 때
```
"그렇군요. 그럼 한 가지만 달라요...
저희는 신한은행 신탁 계좌를 쓰는데..."

→ 신탁 + 사회적 증명 (3,000명, 4.8점)으로 차별화
```

### "시간이 없어요" 할 때
```
"정확하게요. 바쁘신 분이시겠네요.
그래서 더 빠르게 드리려고 전화했는데..."

→ Opening D로 전환 (1분 20초)
```

## 🎓 상담사 교육 (신입용)

### 1주차: Opening A' 마스터
- 월: PHASE 1 연습 (신뢰 신호, 공감)
- 화: PHASE 2 연습 (신탁, 사회적 증명)
- 수: PHASE 3 연습 (부모님 감정, Mortality)
- 목: 침묵 타이밍 (시계 옆에 놓고 초 세기)
- 금: 실제 콜 3건 + 피드백

### 2주차: Opening B-E 개요
- Opening B (신혼): 파트너 피드백
- Opening C (가격): 피어 트레이닝
- Opening D (시간): 피어 트레이닝
- Opening E (효도): 고급 기법 (침묵 13초)

**마지막 팁:**
- 침묵이 제일 어렵다 (상담사가 먼저 말하고 싶어함)
- 4초 침묵 = 고객이 깊게 생각하는 중
- 상담사가 말하면 고객의 깨달음이 50% 줄어듦
- 침묵 = 가장 강력한 설득 기법

**인쇄 권장:** A3 포스터 3개 + 이 카드 A4 2-3장
**위치:** 상담사 모니터 양옆 + 책상 앞 + 클립보드

**갱신 주기:** 월 1회 (고객 반응 데이터 반영)
