# 📞 Support & FAQ

## "PASONA는 크루즈/렌탈에만 쓰이나?"

```
아니다. 모든 영업에 쓸 수 있다.

적용 예:
• B2B SaaS: "Problem (비용)" → "Solution (효율성)"
• 부동산: "Problem (위치)" → "Affinity (공동체)" → "Offer (금리)"
• 자동차: "Problem (안전)" → "Solution (간단함)" → "Offer (할부)"

PASONA는 심리학 법칙이지, 상품에 국한된 게 아니다.
```

## "우리는 다른 상품을 파는데 가능할까?"

```
가능하다.

Step 1: 자신의 상품/서비스의 세그먼트 식별
        "누가 우리 상품을 쓸까?" (나이, 직업, 욕구)

Step 2: 각 세그먼트의 "Problem"부터 시작
        "그들의 현재 상황에서 뭐가 문제인가?"

Step 3: 4단계 경로 설계
        Problem → Affinity → Solution → Offer

Step 4: 기법 맞춤화
        "우리 상품이라면 이 기법을 이렇게 쓸 거다"

이것이 바로 "적용"이다.
```

## "상담사 교육에 얼마나 걸릴까?"

```

→ 코칭: 10시간 (1주일)

→ 즉시 적용 가능: 0시간
= 총 4시간 (1시간 강의)

결론: 신입도 1주일이면 경력급이 된다.
```
