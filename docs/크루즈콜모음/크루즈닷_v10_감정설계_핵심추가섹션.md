## SECTION 0: v10 감정 여정 설계도 (전체 흐름)

```
[인셀덤 감정 여정을 크루즈닷에 완전 대입]

감정 여정:
자존감(선정) → 경계심해제(갑의전문가) → 고통공감(세그먼트Pain) →
욕망자극(5감각) → 신뢰구축(신한+전문가) → 소속감(골드멤버) →
commitment(카톡확보) → 자발적골인("어떻게하면돼요?")

각 단계 전환 신호:
✅ 자존감: 고객이 "저 선정됐어요?"
✅ 경계심해제: 고객이 질문을 함
✅ 고통공감: 고객이 "맞아요, 그게 고민이에요"
✅ 욕망자극: 고객이 "좋겠다" "가보고 싶다"
✅ 신뢰구축: 고객이 "그건 어떻게 돼요?" 구체적 질문
✅ 소속감: 고객이 "회원이 어떤 혜택이 있어요?"
✅ commitment: 카톡 번호 수락
✅ 자발적골인: "그럼 어떻게 신청하면 돼요?"
```

## SECTION 1: 알파 테크닉 오프닝 (신규 — v9 대체)

### 1.1 기존 v9 vs v10 비교

```
[기존 v9 — 을의 포지션]
S: "안녕하세요, 실례가 많겠지만 크루즈닷..."
→ 문제: "실례가 많겠지만" = 이미 을의 위치 선언

[v10 알파 테크닉 — 갑의 포지션]
S: "안녕하세요, [고객명]님 맞으시죠?
   저는 크루즈닷 시니어 여행 매니저 [이름]입니다.
   현재 저는 골드 멤버십 [X]분을 전담 관리하고 있고,
   이번에 [고객명]님께서 우선 배정 대상으로 선정되셔서
   제가 직접 연락드리게 됐어요."
```

### 1.2 알파 테크닉 전체 오프닝 (공통 — 세그먼트 공통 사용)

```
S: "안녕하세요, [고객명]님 되시나요?
   저는 크루즈닷에서 골드 멤버십 전담을 맡고 있는
   시니어 여행 매니저 [이름]입니다.

   이번에 [고객명]님이 골드 배정 우선 대상으로 선정되셔서
   제가 직접 확인 연락을 드리는 거고요,

   *(단 1초 침묵)*

   지금 통화 2~3분 가능하세요?"

C: "네, 괜찮아요."

S: "감사합니다. [고객명]님 상황을 정확히 파악해야
   딱 맞는 혜택을 드릴 수 있거든요.
   제가 몇 가지 여쭤봐도 될까요?"
```

**알파 테크닉 핵심 요소:**
- 이름을 먼저 확인 (권위와 맞춤감 동시)
- "전담" + "시니어" = 전문성 표방
- "직접 연락" = 특별함 부여
- 1초 침묵 = 갑의 자신감
- 통화 가능 여부 물어보기 = 오히려 갑의 여유

## SECTION 2: 개입 질문 5단계 (세그먼트별 완전판)

### 2.1 세그먼트 A (30대 신혼/커플)

```
[1단계 - 상황파악 (최소화)]
S: "요즘 여행은 어떻게 다니세요?
   국내 위주이신가요, 해외도 자주 가시나요?"

[2단계 - 문제파악 (불편함 도출)]
S: "아 그러세요. 그러면 비행기 예약하고,
   호텔 찾고, 현지에서 이동하고...
   둘이서 챙길 게 꽤 많으시겠네요?"

C: "맞아요, 그게 좀 피곤하더라고요."

S: "그렇죠, 여행 준비가 반이라는 말이 있잖아요.
   특히 신혼이시면 더 완벽하게 하고 싶으실 텐데..."
   *(부가의문문으로 공감)*

[3단계 - 문제강조 (타임머신)]
S: "만약 이런 패턴으로 앞으로 5년이 지난다면...
   바쁜 직장생활에 아이도 생기고 나면,
   둘만의 진짜 여행은 언제 가능하실 것 같으세요?"

C: "...그러게요."

[4단계 - 해결파악]
S: "그렇다면, 짐 한 번만 풀고
   일본, 대만, 홍콩을 한 번에 다 가는 방법이 있다면
   한번 들어보실 의향이 있으세요?"

[5단계 - 해결강조 (이득 시각화)]
S: "만약 그게 가능하다면,
   [고객명]님 파트너분이랑 어떤 여행 하고 싶으세요?
   가장 해보고 싶으신 게 뭐예요?"
```

### 2.2 세그먼트 B (30-40대 가족)

```
[1단계 - 상황파악]
S: "아이들이랑 가족 여행은 자주 가세요?
   마지막 해외여행이 언제셨어요?"

[2단계 - 문제파악]
S: "아이들이랑 가시면 짐도 두 배고,
   현지에서 이동도 신경 쓰이시겠네요?
   음식도 아이 입맛에 맞아야 하고..."

C: "맞아요. 솔직히 가고 나서 더 피곤해요."

[3단계 - 문제강조]
S: "그렇죠. 사실 그게 가족 여행의 딜레마잖아요.
   만약 이 패턴으로 아이들이 크고 나면,
   '우리 가족 진짜 여행다운 여행' 한번 한 적 없다는 느낌이
   드실 수도 있는데...
   혹시 그런 생각 해보신 적 있으세요?"

[4단계 - 해결파악]
S: "그렇다면, 아이들은 키즈클럽에서 즐기고,
   어른들은 자쿠지에서 쉬고,
   저녁엔 가족이 함께 랍스터 먹고...
   이런 여행이 가능하다면 어떠실 것 같아요?"

[5단계 - 해결강조]
S: "만약 그게 다 가능한 여행이 있다면,
   아이들한테 어떤 기억을 만들어주고 싶으세요?"
```

### 2.3 세그먼트 C (40-50대 중년)

```
[1단계 - 상황파악]
S: "선생님, 해외여행은 얼마나 자주 가세요?
   부부끼리 여행은 자주 다니시는 편이에요?"

[2단계 - 문제파악]
S: "그렇군요. 바쁘시다 보면 사실
   정작 부부끼리 여행은 미뤄지기 쉽잖아요.
   지난번 부부 둘이서만 간 여행이 언제예요?"

C: "...생각해보니 한참 됐네요."

[3단계 - 문제강조]
S: "맞아요. 일상에 치이다 보면 그렇게 되는데,
   만약 지금 이 패턴으로 5년이 더 지난다면...
   부부 둘만의 시간이 언제쯤 생길 것 같으세요?"

[4단계 - 해결파악]
S: "그렇다면, 밥도 걱정 없고, 이동도 없고,
   매일 다른 나라 항구에서 눈 떠지는 여행이 있다면
   어떻게 생각하세요?"

[5단계 - 해결강조]
S: "만약 그런 여행을 파트너분께 선물하신다면,
   어떤 표정 지으실 것 같아요?"
```

### 2.4 세그먼트 D (5060 은퇴)

```
[1단계 - 상황파악]
S: "어머님/아버님, 그동안 여행은 좀 다니셨어요?
   해외 여행도 종종 가셨어요?"

[2단계 - 문제파악]
S: "그렇군요. 이제 은퇴하시고 나면
   오히려 여행 계획 세우는 게 더 어려우실 수 있어요.
   어디서부터 시작해야 할지 막막하시지 않으세요?"

[3단계 - 문제강조 — 타임머신 핵심 단계]
S: "어머님, 한 가지 여쭤봐도 될까요?
   지금 건강하실 때 가시는 거랑,
   2년 뒤에 가시는 거랑 많이 다르거든요.
   2년 뒤에도 지금처럼 건강하실 자신이 있으세요?
   *(3초 침묵)*
   부모님 모시고 알래스카 빙하를 보고 싶은데
   타이밍을 놓치는 분들이 너무 많아요."

[4단계 - 해결파악]
S: "그렇다면, 자녀분들도 같이 가고,
   어머님은 배에서 편하게 쉬시면서
   빙하도 보시고, 지중해도 보시는 여행이 있다면
   생각해 보실 의향이 있으세요?"

[5단계 - 해결강조]
S: "만약 그런 여행을 다녀오신다면,
   자녀분들한테 가장 하고 싶으신 말씀이 뭐예요?"
```

## SECTION 3: 마이 프렌드 존 (세그먼트별 3종씩)

### 공식: "저희 고객 중에... 그분도 처음엔... 그런데 지금은..."

```
[세그먼트 A — 30대 커플]
"저희 고객 중에 신혼여행으로 크루즈를 생각하셨다가
 '너무 비싸다'고 포기하셨던 분이 계셨어요.
 그분도 처음엔 반신반의하셨는데, 결국 3개월 체험권으로
 알아보시고 가셨거든요.
 다녀오시고 나서 '일반 패키지보다 3배 행복했다'며
 1주년 여행도 크루즈로 예약하셨어요."

[세그먼트 B — 가족]
"저희 고객 중에 아이 둘 데리고 처음 해외 가는 거라서
 너무 긴장하셨던 분이 계셨어요.
 그분도 걱정이 많으셨는데, 막상 배 타시니까
 아이들은 키즈클럽에서 논다고 따로 가겠다고 했고,
 부부는 오히려 처음으로 둘만의 시간을 가지셨대요.
 '이게 진짜 가족 여행이구나' 하셨대요."

[세그먼트 C — 중년 부부]
"저희 고객 중에 부부 둘이 처음으로 제대로 된 여행을
 가시는 거라서 설레하셨던 분이 계셨어요.
 그분이 배 위에서 지중해 일몰 보시면서 눈물 흘리셨대요.
 '30년 만에 처음으로 완전히 쉰 것 같다'고 하시면서요.
 지금은 매년 크루즈 가시는 단골이에요."

[세그먼트 D — 5060 부모님]
"저희 고객 중에 자녀분이 부모님 환갑 선물로
 예약해드린 분이 계셨어요.
 어머님이 알래스카 빙하 보시면서 우셨대요.
 평생 TV에서만 보던 걸 눈 앞에서 보시니까요.
 그 가족한테는 최고의 선물이었대요."

[세그먼트 E — 고령]
"저희 고객 중에 처음에 '나 같은 나이에 크루즈가 되겠냐'
 하셨던 분이 계셨어요.
 그분도 처음엔 많이 걱정하셨는데, 배 안에서
 모든 게 다 있으니까 오히려 더 편하셨대요.
 다녀오시고 나서 '이게 여행의 끝판왕이다'라고 하셨어요."
```

## SECTION 4: 부정 명령 (청개구리 심리) — 압박 없이 행동 유발

```
S: "꼭 지금 결정하셔야 한다는 건 아니에요.
   천천히 생각해보세요.
   다만, 이번 달 3개월 체험권 조건은
   다음 달엔 어떻게 될지 제가 장담은 못하겠어서요.
   오늘은 그냥 들어만 보시는 거예요."
```

```

   제가 억지로 설득드리려는 게 아니거든요.

   말씀이라도 드리는 게 도리인 것 같아서요.
   그냥 어떤 건지만 들어보실게요?"
```

### 4.3 정보만 탐색 허용형

```
S: "오늘 신청까지 안 하셔도 돼요.
```

## SECTION 5: 버튼업 — 클로징 후 취소 방지 (신규 섹션)

```
S: "[고객명]님, 오늘 정말 현명한 결정 하셨어요.
   다른 분들은 고민만 하다가 자리 날아가시거든요.
   [고객명]님은 결단력이 있으시네요.

   *(1초 침묵)*

   혹시 나중에 주변에서
   '크루즈닷이 뭐야, 왜 했어?' 하시면,

   이렇게 당당하게 말씀하세요.

   제가 탑승하시는 날까지 끝까지 책임지고
   케어해드릴게요. 카톡으로 연락드릴게요."
```

### 5.2 세그먼트별 강화 버튼업

```
[세그먼트 A - 커플]
```
