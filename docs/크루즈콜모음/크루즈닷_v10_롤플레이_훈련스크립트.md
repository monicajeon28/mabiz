## 2026-03-15 | v10 감정설계 완전판 기반

---

> **훈련 철학:**
> "연습은 실전보다 더 어렵게 해야 한다."
> 트레이너는 고객보다 더 까다롭게 저항하고, 더 예상 밖의 반응을 낸다.
> 이 훈련을 통과한 상담사는 어떤 실전 콜도 두렵지 않다.

## 📋 훈련 가이드

### 사용 방법
1. **트레이너 역할** (T): 고객 역할 — 지정된 세그먼트 특성 유지
2. **트레이니 역할** (S): 상담사 역할 — 스크립트 없이 본인의 말로
3. **녹음 필수**: 본인 목소리를 듣지 않으면 발전 없음
4. **평가 체크리스트**: 매 롤플레이 후 반드시 채점

### 훈련 단계
```
Step 1: 기법별 미니 훈련 (5분씩) → 기법 체화
Step 2: 세그먼트별 완전 롤플레이 (20~30분) → 전체 흐름 체화
Step 3: 실패 시나리오 복구 훈련 → 위기 대응 체화
Step 4: 종합 평가 채점 → 합격 기준 충족 확인
```

### 합격 기준
- 8단계 감정 여정 모두 통과: **필수**
- 알파 테크닉 유지 (을의 언어 0회): **필수**
- 개입 질문 5단계 완수: **필수**
- 마이프렌드존 자연스럽게 삽입: 필수
- 클로징률 시뮬레이션 점수: **80점 이상**

# PART 1: 기법별 미니 훈련 (Step 1)

## 기법 1: 알파 테크닉 훈련 (5분)

### 목적
갑의 포지션에서 시작하는 첫 인상 체화

### 훈련 A: 오프닝 변환 연습

**트레이너 지시**
> "나는 방금 전화를 받은 50대 여성이에요. 바쁜 척 하면서 '네, 뭐죠?' 하고 짧게 답할게요."

**트레이니 수행**
아래 두 버전을 각각 말한 후 차이를 느낀다.

```
[버전 1 — 을의 포지션 (금지)]
"안녕하세요, 바쁘신데 갑자기 전화드려서 죄송한데요..."
→ 평가: 이 오프닝은 즉시 실격

[버전 2 — 갑의 포지션 (정답)]
"안녕하세요, [고객명]님 되시나요?
 저는 크루즈닷 시니어 매니저 [이름]인데요,
 이번에 골드 멤버십 우선 배정 대상으로 선정되셔서
 제가 직접 연락드리게 됐어요."
*(1초 침묵)*
"지금 2~3분 괜찮으세요?"
```

**트레이너 피드백 기준**
- ✅ "직접 연락드리게 됐어요" — 을이 아닌 갑의 프레임
- ✅ 1초 침묵을 채우지 않음 — 갑의 여유
- ❌ "실례가 많겠지만" — 즉시 피드백

### 훈련 B: 알파 태도 체크 (음성 훈련)

트레이니는 아래 문장을 **3가지 다른 방식**으로 읽는다.

문장: `"지금 3자리만 남아있어서요."`

```
방식 1 (을): 빠르게, 불안하게, 높은 음조
  → 고객 반응: "부담스럽다"

방식 2 (갑): 천천히, 낮은 음조, 담담하게
  → 고객 반응: "진짜인가?" (신뢰)

방식 3 (과도한 갑): 너무 느리고 위압적
  → 고객 반응: "뭔가 이상하다"
```

**정답 속도**: 평소 말하기 속도의 0.8배 (느리게)

## 기법 2: 개입 질문 5단계 훈련 (5분)

### 목적
고객이 스스로 문제를 발견하고 욕구를 표출하게 만들기

### 훈련: 공통 5단계 압축 연습

**트레이너 지시**
> "나는 '특별히 여행을 원하는 건 아닌데 뭔가 끌리는' 60대 아저씨예요.
> 소극적으로 대답하면서 가끔 공감은 해요."

**트레이니 수행 (5단계 연속)**

```
[1단계 — 상황파악] 현재 어떤지 파악
S: "여행을 좀 즐기시는 편이에요?
   국내 위주이신지, 해외도 다니시는지요."

[2단계 — 문제파악] 불편함을 살짝 건드리기
S: "해외 나가실 때 준비하는 게 번거로우시지 않으세요?
   비행기, 숙소, 이동...?"

[3단계 — 문제강조] 미래로 확대
S: "만약 지금처럼 계속 그 방식이면,
   나중에 체력이 좀 더 떨어지셨을 때 여행 가기가
   더 어려워지실 것 같지 않으세요?"
*(3초 침묵)*

[4단계 — 해결파악] 대안 제시
S: "만약 이동 걱정 없이, 한 번 짐 풀면
   매일 다른 나라 구경하고, 식사도 다 해결되는
   방법이 있다면 한번 들어보실 의향 있으세요?"

[5단계 — 해결강조] 이득 시각화
S: "만약 그게 다 가능하다면,
   어떤 여행을 해보고 싶으세요?
   가장 한번 가고 싶었던 곳이 어디예요?"
```

**트레이너 피드백 포인트**
- 3단계 후 반드시 침묵 3초 확인
- 4단계는 YES/NO 질문이 아닌 "의향 있으세요?"로 유도
- 5단계에서 고객이 "알래스카요" "유럽이요" 말하면 성공

## 기법 3: 마이프렌드존 훈련 (5분)

### 목적
직접 설득 대신 제3자 스토리로 고객이 스스로 납득하게 만들기

### 마이프렌드존 4단계 공식
```
① 동일한 상황: "비슷한 상황이신 분이 계셨는데..."
② 동일한 저항: "처음에 그 분도 OO이라고 하셨거든요"
③ 어떻게 해결: "근데 저희가 이렇게 안내드렸더니..."
④ 현재 결과: "지금은 OO하고 계세요"
```

### 훈련 A: 가격 저항 마이프렌드존

**트레이너 (고객):**
> "월 33,000원이 좀 부담스럽네요."

**트레이니 수행:**
```
S: "그러시죠, 저도 충분히 이해해요.
   사실 저번에 비슷한 상황이신 분이 계셨는데,
   그 분도 처음에 그 말씀 하셨거든요.
   (① 동일상황 + ② 동일저항)

   근데 제가 이렇게 여쭤봤어요.
   '선생님, 그러면 한 달에 한 번 외식하실 때
   얼마 쓰세요?' 하고요.
   (③ 해결 질문)

   그랬더니 '5만원 정도는 쓰죠' 하시더라고요.
   그래서 제가 말씀드렸어요.
   '그 금액의 절반으로 1년치 크루즈 적립이 되는 거예요'
   하니까, 그 분이 '그렇게 생각하면 별거 아니네'
   하시고 바로 진행하셨어요.
   (④ 현재 결과)

   [고객명]님은 어떻게 생각하세요?"
```

**평가 기준**
- ✅ "그 분도..." 자연스러운 전환
- ✅ 마지막에 반드시 고객에게 되돌리기
- ❌ 마이프렌드존 없이 직접 "하루 천원도 안 해요" 직진

### 훈련 B: "남편이 싫어해요" 저항 마이프렌드존

**트레이너:**
> "남편이 그런 거 별로 안 좋아하거든요."

**트레이니 수행 (빈칸 채우기 — 스스로 작성):**
```
S: "아, 맞아요. 사실 저번에 ___________
   (① 동일상황)

   그 분도 처음에 ___________
   (② 동일저항)

   근데 이렇게 됐는데요. ___________
   (③ 해결 과정)

   지금은 ___________
   (④ 현재 결과)

   어머님은 어떻게 생각하세요?"
```

**트레이너 피드백**: 자연스럽게 흘러가는지 확인

## 기법 4: 부정 명령 훈련 (3분)

### 목적
"지금 결정하세요" 대신 "안 하셔도 된다"는 역설로 자발적 결정 유도

### 부정 명령 3종 세트

```
종류 1 — 소프트 클로징
"꼭 지금 결정하실 필요는 없어요.
 다만, 이번 달 배정이 남아있는 동안만 이 조건이거든요."

종류 2 — 선택권 부여형
"안 하셔도 전혀 상관없어요.
 대신 어떤 부분이 마음에 걸리시는지 말씀해주시면,
 제가 한 번 더 체크해드릴게요."

종류 3 — 마무리형
"급하게 결정하실 필요 없고요,
 오늘은 그냥 자리만 잡아두시는 거예요.
 혹시 취소하고 싶으시면 7일 이내에 취소도 되니까."
```

**훈련 방법**
트레이너가 "생각해볼게요"라고 했을 때 위 3종 중 가장 자연스러운 것을 즉흥적으로 선택하여 말한다.

## 기법 5: 버튼업 훈련 (3분)

### 목적
클로징 성공 직후 남편 반대 등으로 인한 취소 방지

### 버튼업 3단계

**[클로징 성공 즉시 실행]**

```
[1단계 — 칭찬]
S: "[고객명]님, 정말 좋은 결정 하셨어요.
   이렇게 빨리 결정하시는 분들이 나중에
   제일 만족하시더라고요."

[2단계 — 사전방어]
S: "혹시 가족분한테 말씀하셨을 때
   처음엔 '그게 뭐야' 하실 수도 있는데,
   그건 아직 정보가 없어서 그러신 거예요.
   제가 카톡으로 자료 보내드릴게요.
   그거 보여드리면 다들 '잘했다' 하세요."

[3단계 — 관계 약속]
S: "저도 담당 매니저로 계속 챙겨드릴 거예요.
   궁금하신 거 있으면 언제든 저한테 연락 주시고요.
   카톡으로 연결해드릴게요."
```

**훈련 포인트**
- 클로징 성공 후 3초 이내 시작
- 칭찬 → 방어 → 관계약속 순서 절대 지키기
- 목소리: 따뜻하게, 빠르게 하지 말 것

## 기법 6: 타임머신+돋보기 통합 훈련 (5분)

### 목적
D/E 세그먼트(50~70대)의 "나중에 생각해볼게요"를 즉각 해소

### 통합 공식
```
[타임머신] 미래 장면 → [돋보기] 금전적 손실 확대
```

**트레이너:**
> "지금 당장은 좀 그렇고, 나중에 생각해볼게요." (65세 여성)

**트레이니 수행:**
```
S: "어머님, 맞아요. 급하게 생각 안 하셔도 돼요.
   (부정명령 선행)

   근데 제가 여쭤볼게요.
   지금으로부터 딱 2년 뒤를 한번 상상해보세요.
   2년이 지나면 어머님이 딱 [나이+2]세가 되시잖아요.
   그때도 지금과 똑같이 건강하실 거라고 생각하세요?
   (타임머신 — 미래 상황 설정)

   *(2초 침묵)*

   사실 여행은 체력 있을 때가 전부거든요.
   50대 후반이 크루즈 황금기예요.
   60대 중반이 넘어가시면 이동 자체가 부담이 되시거든요.
   (돋보기 — 현실 강조)

   그렇다면 '나중에'가 오히려 더 늦어질 수도 있는데,
   어머님이 생각하시는 '나중에'가 언제예요?"
   (질문으로 되돌리기)
```

## 기법 7: 인과관계 화법 훈련 (3분)

### 공식
`"[A]이기 때문에 [B]가 될 것이다"` — 무의식이 자동으로 수용하는 구조

### 훈련: 문장 완성하기

아래 빈칸을 채워 5문장 완성 (즉흥으로 2분 안에)

```
1. "어머님이 지금 건강하시기 때문에, _____________."
   (예: 지금이 가장 좋은 타이밍이에요)

2. "이 크루즈가 3박4일이기 때문에, _____________."
   (예: 주말에 출발하시면 회사 지장 없어요)

3. "월 33,000원이기 때문에, _____________."
   (예: 커피 하루 한 잔 줄이시면 해결돼요)

4. "신한은행에서 관리하기 때문에, _____________."
   (예: 저희가 손댈 수 없어요)

5. "50명 중 3자리 남았기 때문에, _____________."
   (예: 지금 결정하시는 게 나중에 후회 없으세요)
```

# PART 2: 세그먼트별 완전 롤플레이 (Step 2)

## 🎭 롤플레이 1: 세그먼트 A — 30대 커플 (약 20분)

### 고객 프로필 (트레이너용)
```
이름: 김지수 (32세, 여성)
직업: 회사원
동행: 남자친구 (결혼 예정)
특성: 스마트하고 바쁨. 정보에 민감. 직접 검색해보는 타입.
저항 패턴: 처음엔 "먼저 생각해볼게요", 가격 비교하려 함
감정 버튼: 신혼여행처럼 특별한 둘만의 경험
```

### 트레이너 상황 설정
> "저는 남자친구랑 결혼 전에 여행 한번 제대로 가보고 싶었는데,
> 일정 맞추기가 힘들어서 못 가고 있어요. 뭔진 모르겠는데 전화가 왔어요."

---

**[SECTION A: 알파 테크닉 오프닝]**

```
S: "안녕하세요, 김지수님 되시죠?
   저는 크루즈닷 시니어 매니저 [이름]인데요,
   이번에 지수님께서 골드 우선 배정 대상으로
   선정되셔서 제가 직접 연락드렸어요."

*(1초 침묵)*

T: "아... 저요? 어떤 거예요?"

S: "네, 지금 2~3분 괜찮으세요?
   지수님 여행 스타일 먼저 여쭤보고,
   딱 맞는 혜택을 챙겨드리고 싶어서요."

T: "뭔지 일단 들어볼게요."
```

**[트레이너 평가 포인트]**
- ✅ 사과로 시작하지 않았는가?
- ✅ "선정되셔서"로 수혜자 프레임 잡았는가?
- ✅ 1초 침묵 있었는가?

---

**[SECTION B: 개입 질문 5단계]**


# PART 3: 실패 시나리오 복구 훈련 (Step 3)

---

## 실패 1: 을의 언어 사용 후 복구

**상황**: 트레이니가 "실례가 많겠지만"으로 시작함

**트레이너 반응**: 그냥 계속 진행 (상담사가 스스로 인식해야)

**복구 방법**:
```
[인식]
내가 "실례"로 시작했다 → 즉시 톤 변환

[복구 멘트]
"제가 다시 말씀드릴게요.
저는 [고객명]님께서 배정 대상으로 선정되셨기 때문에
직접 연락드린 거예요. 간략하게 여쭤볼게요."

→ 프레임을 갑으로 즉시 전환
```

---

## 실패 2: 상품 설명 먼저 시작 후 복구

**상황**: 고객이 묻기 전에 "월 33,000원에..." 설명 시작

**트레이너 반응**: "비싸네요" (즉각 저항)

**복구 방법**:
```
[인식]
순서가 틀렸다 → 개입질문으로 되돌아가기

[복구 멘트]
"잠깐요, 제가 너무 앞서 나갔네요.
먼저 [고객명]님 상황 먼저 여쭤봤어야 했는데.
여행을 좋아하세요?"

→ 니즈 파악부터 다시 시작
```

---

## 실패 3: 침묵 견디지 못해 먼저 채움

**상황**: 5초 침묵 후 트레이니가 먼저 말함

**트레이너 반응**: 침묵만 유지

**복구 방법**:
```
[인식]
침묵을 채우면 안 된다 → 다음부터 숫자 세기

[훈련 방법]
침묵 후 속으로 1, 2, 3, 4, 5 센다
5 다음에도 고객이 말 없으면 그때 부드럽게
"어떻게 생각하세요?" 한 마디만
```

---

## 실패 4: 마이프렌드존 없이 직접 설득

**상황**: 가격 저항에 "하루 천원도 안 해요" 직진

**트레이너 반응**: "그래도 부담스러워요" (계속 저항)

**복구 방법**:
```
[인식]
직접 설득은 저항을 강화한다 → 마이프렌드존으로 전환

[복구 멘트]
"맞아요, 충분히 그렇게 느끼실 수 있어요.
사실 저번에 비슷한 상황이신 분이 계셨는데..."

→ 마이프렌드존으로 즉시 전환
```
