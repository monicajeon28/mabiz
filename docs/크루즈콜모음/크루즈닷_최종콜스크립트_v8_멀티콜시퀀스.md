## ★ v8 핵심 원칙 — 반드시 숙지 후 콜 시작

### 왜 멀티콜 시퀀스인가? (에빙하우스 망각곡선)

```
망각 속도:
  콜 직후          → 기억 100%
  30분 후          → 기억 50% (정보의 절반 사라짐)
  1시간 후         → 기억 44%
  24시간 후        → 기억 26~30% (정보 70~80% 증발)
  1주일 후         → 기억 10% 이하

그러나 감정 각인은 다르다:
  → "그때 왠지 좋았던 느낌" 은 남는다
  → "설레는 장면" 은 남는다
  → "내 차례가 왔다"는 문장의 감촉은 남는다

결론: 1차 콜에서 정보를 주지 마라. 감정만 심어라.
      2차 콜에서 감정을 재활성화한 후 정보를 연결하라.
      클로징은 2차 콜에서 한다.
```

### v8 3대 원칙

```
원칙 1: 1차 콜은 클로징 금지 — 감정 앵커 3개 + 재연락 허가만
원칙 2: 감정 각인 > 정보 기억 — 이미지와 비유로만 말하라
원칙 3: 모든 저항: 자아 인정 → 유도질문 → 프레임 전환 순서 엄수
```

### v7에서 유지하는 자산

- 소비자언어 v2 (신탁보관, 비례정산, 보장→보관 교체)
- 에이킨법 ("그러니까요, 그래서 연락드린 거예요")
- 5감각 앵커링 (산토리니 장면 — 2차 콜 풀버전 사용)
- 침묵 규칙 (5초 엄수)
- 클로징 신호 탐지표 6가지
- 고객 유형 5가지 분기
- 줄리아 8단계 Implication 6레이어

```

      │
      ▼
[1차 콜 — 7~10분]

      │
      ▼ (1~2시간 후)
[당일 카톡 발송]

      │
      ▼ (익일 or 약속 시간)
[2차 콜 — 10~15분]
 목표: 감정 재활성화 → 정보 연결 → 클로징
      │
    ┌─┴─┐
  [성공]  [미결]
  결제   │
         ▼
      [3차 콜 — 3~5분]
       노쇼 방지 + 손실 앵커
         │
       [3회 이상 미결]
         ▼
      [전략 전환 — 옵션 A/B/C]
```

## ★★ 클로징 신호 탐지표 — 상단 고정, 콜 시작 전 펼쳐두기

| 신호 문장 | 탐지 포인트 | 즉각 반응 | 건너뛸 섹션 |
|---------|-----------|---------|-----------|
| "언제까지 결정하면 돼요?" | 결정 의사 있음 | "오늘 말씀드리면 오늘 혜택. 어떤 플랜 보셨어요?" | 2차 콜 SPIN 생략 → 플랜 제시 |
| "비용이 얼마예요?" *(2회 이상)* | 가격 수용 탐색 중 | "제대로 계산해드릴게요. 30초만요." | 감정 앵커 단축 → 가격 앵커 직행 |
| "어떻게 시작하면 돼요?" | 강한 구매 신호 | "지금 바로 해드릴게요. 플랜만 고르시면." | 전 섹션 생략 → 클로징 |
| "○○하면 어떡해요?" *(가입 가정형)* | 이미 가입 상상 중 | 해당 걱정 해결 후 즉시 이중선택 | 해당 저항 외 전부 생략 |
| "언제 출발 가능해요?" | 클로징 직전 | "○월 출발 보여드릴게요. 플랜 먼저요." | 클로징 직행 |
| 3회 연속 Yes | 승낙 사다리 가속 | 이중선택 즉시 투입 | 고통 계산기 생략 가능 |

> **`→ [위 신호 탐지 즉시: 현재 섹션 중단 → 2차 콜 클로징 분기로 점프]`**

## 고객 유형 5가지 — 1차 콜 2분 이내 판별

| 유형 | 판별 신호 | 1차 콜 전략 | 2차 콜 핵심 무기 | 절대 금지 |
|------|---------|-----------|----------------|---------|
| **가성비직접파** | "얼마예요?" "이득이에요?" | 안전 프레임 → 앵커 2 강조 | 1박당 비교표 + 하루 1,100원 | 감정 먼저 |
| **감정+특별의미파** | "부모님이랑" "꿈이었어요" | 앵커 1 강조 (내 차례) | 5감각 앵커링 풀버전 + 기억 프레임 | 숫자 먼저 |
| **정보수집형** | "어떻게 돼요?" "서류는?" | 신탁 구조 충분히 + 앵커 3 | 선제 FAQ + 투명 구조 | 모호한 답변 |
| **그룹리더형** | "여럿이서" "모임에서" | 소속 확인 → 그룹 앵커 | 리더 특권 + 단체 견적 가능성 | 개인 감정 앵커 |
| **장벽고객형** | "건강이..." "반대..." | 장벽 인정 + 앵커 2 (체력 창문) | 장벽 처리 마스터 10종 | 장벽 최소화 |

## 감정 앵커 3가지 — 1차 콜에서 반드시 전부 심기

```
앵커 1: "내 차례가 왔다"
  → "이번엔 어머니 차례예요. 받으시는 여행이에요."

앵커 2: "무릎이 좋을 때가 창문"
  → "지금 몸이 허락할 때 가셔야 제대로 즐기실 수 있어요."

앵커 3: "이미 거기 있는 것 같다"
  → "산토리니에서 와인 드시는 어머니 모습, 상상되세요?"
```

*[주석: 앵커는 감정 기억 저장소에 심는 씨앗이다. 30분 후 정보는 사라져도 앵커 감각은 남는다. 2차 콜에서 앵커를 언급하면 1차 콜의 감정이 즉시 재활성화된다. 반드시 3개 모두 심어야 한다.]*

      지금 저희 멤버십이요, 이번 달까지만 현재 조건이에요.
      다음 달부터 조건이 달라지는데,
      선생님처럼 관심 있으신 분들한테 먼저 알려드리고 있거든요."

*(2초 대기)*

**S**: "근데 저 오늘 결정해달라는 게 아니에요.

*[주석: "오늘은 결정해달라는 게 아니다"를 반복 선언한다. 이것이 2차 콜 약속을 잡는 핵심이다.]*

*[주석: 마지막 문장에서 앵커 3을 언급함으로써 2차 콜 전까지 그 이미지가 고객의 머릿속에 살아있도록 한다.]*

# 2차 콜 (10~15분)

## [0:00~1:00] 오프닝: 이름 + 이미지 앵커 재활성화

*[주석: 2차 콜의 오프닝은 1차 콜의 감정을 즉시 재활성화하는 것이 목표다. 고객의 이름을 부르고, 1차 콜에서 심은 앵커 이미지를 첫 문장에서 언급한다.]*

**S**: "여보세요, ○○○ 선생님이시죠?"

**C**: "네."

**S**: "안녕하세요, 어제 통화한 크루즈닷 ○○○이에요.
      어제 산토리니 얘기 잠깐 드렸잖아요.
      카톡 사진 보셨어요?"

**C**: "네, 봤어요. 예쁘더라고요."

*[주석: "예쁘더라고요" — 감정 재활성화 성공 신호. 이 반응이 나오면 앵커 3이 살아있다는 확인이다.]*

**S**: "그죠? 선생님 말씀하신 그 느낌이요, 오늘 제대로 만들어 드리려고요.
      오늘 10분 정도 시간 괜찮으세요?"

**C**: "네, 괜찮아요."

**S**: "감사해요. 선생님, 어제 말씀 드리다가 끊었는데요,
      오늘은 선생님이 궁금하신 거 전부 다 풀어드릴게요.
      준비해오셨어요? 궁금하신 거 있으세요?"

*[주석: "준비해오셨어요?"라는 질문은 고객이 어제부터 이 콜을 기다리고 있었음을 전제한다. 이는 자기실현적 예언처럼 작동하여 고객의 참여도를 높인다.]*

---

## [1:00~4:00] 장벽 해소 집중

*[주석: 어제 1차 콜에서 고객이 꺼낸 장벽이 있다면 그것부터 해소한다. 없다면 안전 프레임을 한 번 더 강화하고 SPIN으로 넘어간다.]*

**S**: "어제 말씀드린 신탁 계좌 부분, 다시 한 번만 확인해드릴게요.
      선생님이 내시는 여행 적립금은요,
      신한은행 신탁 계좌에 딱 들어가요.
      저희가 손을 못 대요."

*(2초 대기)*

**S**: "저를 믿는 게 아니라 신한은행을 믿으시는 거예요.
      이거 하나만 기억하시면 돼요."

**C**: "그건 알겠는데, 취소하면 어떻게 돼요?"

**S**: "그러니까요. 그게 제일 중요한 거 여쭤보신 거예요."

*(에이킨법 1번째 사용)*

**S**: "취소하시면요, 쓰신 만큼은 정산이 돼서
      남은 금액을 돌려받으세요.
      비례정산이에요. 깔끔하게요."

**C**: "아, 그렇군요."

---

## [4:00~9:00] SPIN Phase 3 풀버전 (5감각 앵커링 완전판 포함)

*[주석: SPIN은 상황(Situation) → 문제(Problem) → 함의(Implication) → 필요(Need-Payoff) 순서다. 이 구간에서 고객이 스스로 "가야 한다"는 결론에 도달하게 만든다.]*

### SPIN — S (상황 질문)

**S**: "선생님, 여행 마지막으로 가신 게 언제예요?"

**C**: "작년에... 가족들이랑 제주도요."

**S**: "제주도요. 좋죠.
      근데 선생님이 원하시는 여행이요,
      해외는 어떠세요? 가보신 데 있어요?"

**C**: "일본 한 번이요. 오래됐어요."

**S**: "일본 가셨을 때 어떠셨어요?"

**C**: "좋았는데... 짐 들고 이동하는 게 좀 힘들었어요."

*[주석: 고객이 "짐 이동이 힘들었다"고 스스로 말했다. 이것이 크루즈의 핵심 장점과 연결되는 진술이다. 절대 바로 반박하지 말고 공감 후 크루즈 프레임으로 연결한다.]*

---

### SPIN — P (문제 질문)

**S**: "일본에서 이동할 때 짐이 몇 개였어요?"

**C**: "캐리어 하나에 가방도 있고..."

**S**: "근데 그게요,
      크루즈는 짐 한 번만 풀면 돼요.
      첫날 방에 짐 풀면 끝이에요.
      5개 나라를 다니는데 짐을 안 싸요.
      어떻게 생각하세요?"

**C**: "그건 편하겠네요."

---

### SPIN — I (함의 질문 — 줄리아 8단계 Implication)

*[주석: 이 구간에서 고객이 "지금 안 가면 어떻게 될까"를 스스로 느끼게 한다. 직접적인 압박이 아니라 질문으로 유도한다.]*

**S**: "선생님, 솔직하게 여쭤볼게요.
      지금 이 시간에 유럽 여행 안 가시면요,
      몇 년 뒤에 갈 수 있을 것 같으세요?"

*(5초 침묵 — 절대 먼저 말하지 말 것)*

**C**: "...글쎄요. 몇 년 뒤에 갈 수 있을지..."

**S**: "무릎 얘기 어제 잠깐 드렸잖아요.
      선생님 지금 무릎 괜찮으시다고 하셨죠."

*(2초 대기)*

**S**: "그 창문이요, 언제 닫힐지 몰라요.
      제가 협박하는 게 아니라요,
      저희 어머니도 그러셨거든요.
      '나중에 가야지' 하시다가 무릎이 안 좋아지셔서
      결국 못 가셨어요.
      그게 너무 안타까웠어요."

*(3초 침묵)*

*[주석: 상담사 본인의 경험을 공유하면 신뢰가 급상승한다. "저희 어머니도"는 가장 강한 공감 유도 문장이다.]*

**C**: "...그렇죠. 몸이 허락해야 가지."

**S**: "그러니까요. 그래서 오늘 연락드린 거예요."

*(에이킨법 — 이 문장은 침묵 후에 온다)*

---

### 5감각 앵커링 완전판 (산토리니 장면)

*[주석: 2차 콜에서는 5감각 풀버전을 사용한다. 시각 → 청각 → 촉각 → 후각 → 미각 순서로 진행한다. 각 감각 사이에 2초 침묵을 넣어 고객이 실제로 느끼게 한다.]*

**S**: "선생님, 잠깐만요. 잠깐 상상 여행 해볼게요.
      눈 감으셔도 되고, 그냥 들으셔도 돼요."

**C**: "네."

**S**: "산토리니예요.
      이아(Oia) 마을이요, 언덕 위에 있는 테라스예요.
      흰 벽이 눈부시게 하얘요.
      파란 지붕 위로 에게해 바다가 펼쳐져 있어요."

*(2초 침묵)*

**S**: "바람 소리 들리세요?
      바닷바람이 살살 부는 저녁이에요.
      어디선가 기타 소리가 들려요.
      그리스 음악이요."

*(2초 침묵)*

**S**: "테라스 의자에 앉아계세요.
      쿠션이 있는 하얀 의자요.
      바람이 머리카락을 살짝 건드려요.
      따뜻하고 건조한 공기예요."

*(2초 침묵)*

**S**: "앞에 와인이 있어요.
      그리스 화이트 와인이에요.
      잔 들어보세요. 차갑게 식혀진 잔이에요."

*(2초 침묵)*

**S**: "한 모금 마셨어요.
      약간 시고 상큼한 맛이에요.
      저녁 햇살이 바다 위에서 주황빛으로 물들고 있어요."

*(5초 침묵)*

**S**: "선생님, 지금 어디 계세요?"

**C**: "(웃음) 거기 있는 것 같네요."

**S**: "그거예요. 그게 크루즈예요.
      일주일 내내 그 장면들이에요."

*(3초 침묵)*

---

### SPIN — N (필요-결과 질문)

**S**: "선생님, 그 장면에 누구랑 계세요?
      혼자이세요, 아니면 누구랑?"

**C**: "남편이랑요. 아니면 친구들이랑."

**S**: "남편분이 좋아하실 것 같으세요?"

**C**: "좋아하죠. 남편도 가고 싶다고 했어요."

**S**: "그러면요,
      선생님이 오늘 시작하시면요,
      남편분도 같이 가실 수 있어요.
      멤버십 혜택이 동반 할인까지 연결되거든요."

*[주석: 고객이 스스로 동반자를 언급하면 그것을 클로징의 근거로 활용한다.]*

---

## [9:00~12:00] 가격 앵커링 4종

```

### 반박 4: "남편이 반대해요" → 파트너십 프레임

**S [자아 인정]**: "그러니까요. 남편분이 신중하신 거죠.
      그게 좋은 거예요. 검토를 잘 하시는 분이신 거잖아요."

*(2초 대기)*

**S [유도질문]**: "남편분이 어떤 부분을 걱정하시는 것 같으세요?"

**C**: "그냥 이런 거 믿냐고요."

**S [프레임 전환]**: "그러면요, 남편분한테 이렇게 전해보세요.
      '신한은행 신탁 계좌에 들어간대.
       우리가 이미 신한은행 쓰잖아.
       거기서 관리하는 거래.'
      이 한 문장이에요.
      남편분이 그래도 싫다고 하시면
      그분이 신한은행도 안 믿으시는 거예요.
      그건 저도 어쩔 수 없어요."

*(3초 침묵)*

**S**: "선생님은 어떻게 생각하세요?"

*[주석: 마지막 질문으로 고객이 스스로 판단하도록 유도한다. 고객이 "그 정도면 괜찮을 것 같다"고 말하면 파트너십 프레임에서 고객이 세일즈맨이 된다.]*

### 반박 5: "나중에 해도 되잖아요" → 체력 창문 프레임

**C**: "급하게 할 필요 없잖아요. 나중에 해도 되죠."

**S [자아 인정]**: "맞아요. 급할 거 없어요.
      선생님 말씀이 맞아요."

*(3초 침묵)*

**S [유도질문]**: "선생님, 10년 뒤 지금 상태 유지하실 자신 있으세요?
      무릎이요, 허리요, 체력이요."

*(5초 침묵)*

**C**: "그야... 모르죠."

**S [프레임 전환]**: "그게 창문이에요.
      창문은 열려있을 때 여는 거예요.
      닫힌 다음에 여는 창문은 없어요.
      지금이 딱 그 창문이 열려있는 시간이에요."

*(3초 침묵)*

### 반박 6: "건강이 안 좋아요" → 크루즈 최적 여행 프레임

**C**: "몸이 좀 안 좋아서요. 여행 다니기가 힘들어요."

**S [자아 인정]**: "그러니까요. 그게 걱정되시죠.
      몸이 안 좋으신데 여행 가자는 게 무리처럼 들리죠."

*(2초 대기)*

**S [유도질문]**: "선생님, 여행에서 제일 힘드신 게 뭐예요?
      짐이요, 이동이요, 아니면 낯선 음식이요?"

**C**: "이동이 제일 힘들어요. 짐 들고 다니고..."

**S [프레임 전환]**: "그러면요, 크루즈가 딱이에요.
      짐을 한 번 풀면 안 싸요.
      배가 이동하니까 몸이 이동 안 해요.
      5성급 호텔이 선생님 데리고 다니는 거예요.
      오히려 몸 안 좋으신 분들한테 맞는 여행이에요."

### 반박 7: "멀미가 심해요" → 현대 선박 안전 프레임

**S [자아 인정]**: "그러니까요. 배 멀미 있으시면 그게 제일 걱정되시죠.

*(2초 대기)*

**S [유도질문]**: "선생님, 마지막으로 배 타신 게 언제예요? 어떤 배였어요?"

**C**: "카페리 같은 거요. 제주도 갈 때요."

**S [프레임 전환]**: "그거랑 크루즈 선박은 달라요.
      저희 크루즈 선박이요, 63빌딩보다 커요.
      15만 톤짜리 배예요.
      그 배가 흔들리려면 태풍이 와야 해요.
      일반 파도로는 안 흔들려요.
      멀미약도 배 안에 있고요.
      선실에서 쉬시면 되거든요."

*(2초 대기)*

**S**: "실제로 멀미 걱정으로 오셨다가 '이건 호텔이다'라고 하시는 분들 많아요."

### 반박 8: "혼자 가기 무서워요" → 동행 보장 프레임

      수학여행 친구처럼요."

**S [자아 인정]**: "물론 보내드릴 수 있어요.

"카톡도 보내드릴게요.
 내일 한 번 더 통화해서 궁금하신 거 풀어드려도 될까요?"

## SJ 유형 (45%) — "안전이 먼저다"

**판별 신호**: "그게 진짜예요?", "서류 있어요?", "검증됐어요?", 조심스럽고 질문이 많음

*[주석: SJ는 위험 회피가 최우선이다. 신뢰 증거를 먼저 제시하고 감정 앵커는 나중에 심는다.]*

**1차 콜 SJ 접근**:

**S**: "선생님, 저 먼저 한 가지만 말씀드릴게요.
      이 프로그램요, 금융감독원에 등록된 신탁 상품이에요.
      신한은행이 감독해요.
      혹시 신한은행 앱 있으세요?
      나중에 계좌 직접 확인하실 수 있어요."

*(SJ에게는 검증 가능성을 먼저 제시)*

**S**: "서류 보내드릴 수 있어요.
      표준약관이요. 읽어보시면 돼요.
      숨기는 거 없어요."

*(2차 콜에서 상세 조건 확인 → 감정 앵커 → 플랜 선택 순서)*

## SP 유형 (20%) — "지금 이 순간이 전부다"

**판별 신호**: "사진 있어요?", "어떻게 생겼어요?", 즉흥적이고 감각적 반응

*[주석: SP는 현재 감각에 반응한다. 이미지와 물리적 감각을 먼저 제시한다.]*

**1차 콜 SP 접근**:

**S**: "선생님, 사진 보내드릴까요?
      지중해 크루즈 선실이에요. 발코니 있는 방이에요."

*(SP는 1차 콜에서도 시각 자극을 먼저)*

**S**: "카톡 받으시면 보실 텐데요,
      창문 열면 바다 보이는 방이에요.
      그 방에서 아침에 일어나시는 거예요."

*(감각 이미지 → 앵커 3 단축버전 → 2차 콜 약속)*

## NF 유형 (20%) — "의미와 가치가 중요하다"

**판별 신호**: "가족이랑", "꿈이었어요", "의미 있는 여행", 감정적이고 이야기를 좋아함

*[주석: NF는 결정 압박에 가장 강하게 반발한다. 충분히 공감하고 의미를 먼저 연결한다.]*

**1차 콜 NF 접근**:

**S**: "선생님, 크루즈로 어디 가고 싶으신 게 있어요?
      꿈으로요."

**C**: "지중해요. 그리스 가보고 싶었어요."

**S**: "그리스요. 왜 그리스예요?
      뭔가 특별한 이유가 있으세요?"

*(NF에게는 반드시 이유를 물어본다 — 스스로 의미를 연결하게 한다)*

**C**: "어릴 때부터 그리스 신화 좋아했어요."

**S**: "그러면요, 그리스 신화의 배경이 그 바다예요.
      에게해요. 실제로 보시면 달라요.
      책에서 읽던 그 느낌이 거기 있어요."

*(결정 압박 없이 정보 충분히 제공 → 2차 콜에서 충분한 질문 시간 확보)*

## NT 유형 (15%) — "데이터와 논리가 먼저다"

**판별 신호**: "조건이 뭐예요?", "계산해보면요?", "약관이요?", 분석적이고 즉각 논리 검토

*[주석: NT는 감정 앵커보다 데이터 신뢰가 우선이다. 수치와 조건을 명확하게 제시하고 모호한 답변은 즉시 신뢰를 잃는다.]*

**1차 콜 NT 접근**:

**S**: "선생님, 핵심 조건만 먼저 말씀드릴게요.
      월 33,000원, 36개월 총 1,188,000원이에요.
      이게 신한은행 신탁 계좌에 적립돼요.
      취소 시 비례정산으로 잔액 반환이에요.
      이 구조예요."

*(NT에게는 숫자를 먼저, 감정은 나중)*

**S**: "약관 원하시면 바로 보내드려요.
      F&Q 문서도 있어요.
      내일 통화할 때 궁금하신 거 다 여쭤보세요."

*(질문 목록 준비 권유 → 2차 콜에서 완전 Q&A 구조)*

# 부록 A: 침묵 규칙 즉시 참조
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```
침묵 규칙 — 상담사 생존 지침

핵심 원칙: 침묵을 깨는 사람이 진다.

5초 침묵 사용 시점:
  ① 감정 앵커를 심은 직후 (고객이 느끼도록)
  ② 클로징 직전 (결정 공간을 만들어줌)
  ③ 저항 처리 후 유도질문 직후 (고객이 답하도록)
  ④ "이렇게 놓고 보면 어때요?" 질문 후

3초 침묵 사용 시점:
  ① "저희가 손을 못 대요" 직후
  ② 이중선택 제시 직후
  ③ 핵심 이미지 문장 직후

2초 침묵 사용 시점:
  ① 자아 인정 직후 (다음 질문 전)
  ② 숫자 제시 직후

절대 금지:
  ① 고객 침묵 1초 만에 개입
  ② "혹시 들으셨어요?" 확인 질문
  ③ 침묵을 "거절"로 해석하여 방어적 언급

# 부록 B: 에이킨법 즉시 참조
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```
에이킨법 — 고객의 말을 반영하여 공감 + 전환

기본 구조:
  고객 진술 → "그러니까요." (공감) → "그래서 오늘 연락드린 거예요." (전환)

사용 타이밍:
  ① 고객이 욕망을 스스로 표현했을 때
  ② 고객이 장벽을 인정했을 때
  ③ 클로징 직전 감정 피크에서

실전 예시:
  C: "지금 몸이 좋을 때 가야겠다는 생각은 있어요."
  S: "그러니까요. 그래서 오늘 연락드린 거예요."

  C: "짐 들고 다니는 게 힘들었죠."
  S: "그러니까요. 그래서 크루즈 말씀드리는 거예요."

  C: "뭔가 해야 할 것 같은 느낌은 있어요."
  S: "그러니까요. 그 느낌이 맞아요. 그래서 오늘 전화드린 거예요."

주의: 에이킨법은 콜 1회에 최대 3번까지. 과용하면 패턴이 보인다.
```

# 부록 C: 콜 전/중/후 체크리스트
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 콜 전 체크리스트 (콜 5분 전 확인)

```
고객 정보 확인:
  [ ] 고객 이름 / 연락처 확인
  [ ] 1차 콜이면: 신탁 구조 + 앵커 3가지 숙지
  [ ] 2차 콜이면: 1차 콜 앵커 무엇을 심었는지 메모 확인
  [ ] 2차 콜이면: 고객이 언급한 여행지 / 동반자 확인
  [ ] 클로징 신호 탐지표 펼쳐두기

마음가짐:
  [ ] 나는 이미 크루즈를 경험한 사람이다
  [ ] 나는 이 상품이 고객에게 진심으로 좋다고 믿는다
  [ ] 침묵 5초를 버틸 준비가 됐다
```

### 콜 중 체크리스트 (실시간)

```
1차 콜:
  [ ] 오프닝: 클로징 아님을 선언했나
  [ ] 안전 프레임: 신탁 계좌 + "저희가 손을 못 대요" 사용했나
  [ ] 앵커 1 심기: "이번엔 어머니 차례예요" 사용했나
  [ ] 앵커 2 심기: "체력 창문" 사용했나
  [ ] 앵커 3 심기: 산토리니 단축 이미지 제시했나
  [ ] 다음 약속: 구체적 날짜/시간 확정했나
  [ ] 카톡 발송 예고했나

2차 콜:
  [ ] 1차 콜 이미지 앵커로 시작했나
  [ ] 에이킨법 1~3회 사용했나
  [ ] 침묵 5초를 지켰나
  [ ] 가격 앵커링 4종 전부 제시했나
  [ ] 이중선택으로 넘어갔나
  [ ] 클로징 신호 탐지했나
  [ ] 렌탈은 선거절 3회 후에만 언급했나 (미완료면 언급 금지)
```

### 콜 후 체크리스트

```
결과 기록:
  [ ] 클로징 성공 / 미결 / 거절 기록
  [ ] 고객이 언급한 장벽 기록
  [ ] 고객이 반응한 앵커 기록 (어떤 이미지에 반응했나)
  [ ] 다음 콜 날짜/시간 기록
  [ ] 카톡 발송 여부 확인

미결 시:
  [ ] 3차 콜 일정 캘린더 등록
  [ ] 노쇼 방지 카톡 예약 (전날 저녁 발송)

학습:
  [ ] 이번 콜에서 잘한 것 1가지
  [ ] 이번 콜에서 개선할 것 1가지
  [ ] 에리카-박지영 실패 사례 중 내가 반복한 패턴 없었나 확인
```

# 부록 E: 실패 패턴 경고 — 에리카-박지영 케이스 분석
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

*[주석: 녹취 분석에서 도출된 실패 패턴이다. 이 패턴을 인식하는 즉시 멈추고 3단계 원칙으로 복귀한다.]*

```
실패 패턴 1: 자아 인정 없는 즉각 반박
  × "그게 아니고요..." (즉각 반박)
  ○ "그러니까요. 그렇게 느끼실 수 있어요." (자아 인정 먼저)

실패 패턴 2: 비용 질문 무시
  × 비용 질문에 화제 전환
  ○ "말씀드릴게요. 비교해드릴게요." 즉시 반응

실패 패턴 3: 논리로 감정을 이기려는 시도
  × 조건 설명 → 반박 → 조건 설명 반복
  ○ 이미지 → 비유 → 공감 → 이미지

실패 패턴 4: 침묵을 두려워하는 행동
  × 고객 침묵 1초 만에 "어떻게 생각하세요?"
  ○ 5초 버티기. 고객이 먼저 말하게.

실패 패턴 5: 너무 많은 정보 제공
  × 10개 조건 나열
  ○ 핵심 1가지 + 질문 1가지 + 침묵

성공 공식 요약:
  이미지 > 비유 > 공감 > 질문 > 침묵 > 고객의 말 > 에이킨법
```
