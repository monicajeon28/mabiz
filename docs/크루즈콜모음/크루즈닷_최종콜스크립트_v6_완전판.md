## ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 오프닝 (Phase 1~2)
- [ ] "이름이 ○○○님이시죠?" — DB 재확인 Yes 받았나?
- [ ] 신탁: "신한은행 신탁 계좌에 보관"이라고 말했나?
- [ ] 가격/상품을 먼저 꺼내지 않았나?
- [ ] 시간 확인 이중선택 했나?

### SPIN (Phase 3)
- [ ] S→P→I→N 순서 지켰나?
- [ ] I 질문 후 2초 침묵했나?
- [ ] 산토리니 장면 후 3초 침묵했나?
- [ ] 고객이 먼저 욕망을 말했나? (상담사가 말하면 안 됨)

### 가격 앵커링 (Phase 4)
- [ ] 역질문 3단계 순서 지켰나? (인정→유도질문→가치스택)
- [ ] 시장가 앵커 먼저 말했나?
- [ ] 가격 말한 후 2초 침묵했나?

### 클로징 (Phase 5)
- [ ] 선거절 TOP 5 중 해당 내용 먼저 꺼냈나?
- [ ] 고통 계산기 사용했나?
- [ ] 이중선택 (A/B) 으로 마무리했나?
- [ ] 침묵 3초 지켰나?
- [ ] Yes/No 질문으로 끝내지 않았나?

### 렌탈 (Phase 6)
- [ ] 결제 완료 후에만 렌탈 언급했나?
- [ ] 오프닝에서 가전 먼저 꺼내지 않았나?

### 자동 감점 항목
- [ ] "신한은행이 보장해요" — 안 썼나? (법적 허위)
- [ ] "믿어주세요" / "진짜예요" — 안 썼나?
- [ ] 가짜 희소성 — 근거 없이 "오늘까지만"이라고 하지 않았나?
- [ ] 고객 말 중간에 끊지 않았나?
