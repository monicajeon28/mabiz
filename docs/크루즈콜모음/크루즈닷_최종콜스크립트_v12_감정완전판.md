## ★ SECTION A: 알파 테크닉 오프닝 + 역설 2문장 (v12 신규)

### 핵심: 갑의 자리 선언 → 역설로 신뢰 2배

```
[알파 오프닝 — 갑의 포지션]
"안녕하세요, [고객명]님 맞으시죠?
 저 크루즈닷 시니어 여행 매니저 [이름]이에요.
 이번 달에 [고객명]님을 우선 배정 대상으로
 선정해서 제가 직접 연락드리는 거예요."

*(1초 여유 — 갑의 침묵)*

[★ NEW v12 — 역설 오프닝 2문장 (아비테 기법)]
"그리고 저는 오늘 설득하려는 게 아니에요.
 [고객명]님이 어떤 여행을 원하시는지,
 저희가 그게 가능한지 — 그것만 확인드릴게요."

*(3초 침묵 — 고객이 먼저 말하게)*

[라포 연결]
"저희 홈페이지에서 상담 신청해 주셨잖아요.
 신청하실 때 어떤 부분이 제일 궁금하셔서 남겨주셨어요?"
```

**역설 2문장의 심리학:**
```
기존 "아무것도 안 팔 거예요" ❌ → 판매원 프레임 강화
v12 "설득 안 합니다 + 진단할게요" ✅ → 전문가/진단사 프레임

결과: 고객이 "판매원과 통화 중"이 아닌
     "전문가에게 진단받는 중"으로 인식
→ 저항 자동 하락
```

**⚠️ 제거 금지 멘트:**
```
❌ "10분이면 됩니다" → 판매원 신호
❌ "아무것도 안 팔 거예요" → 과도한 면피 = 역효과
❌ "실례가 많겠지만" → 즉시 을의 포지션
❌ "바쁘신데" → 통화 거절 빌미 제공
```

## ★ NEW v12 — 구매의향 사전동의 (오프닝 후 2분 내 필수)

### 핵심: 30분 낭비 방지 + 조기 세그먼트 확정

```
[라포 후 2분 내 삽입]

"[고객명]님, 솔직하게 한 가지만 여쭤봐도 될까요?

 만약 오늘 제가 말씀드리는 내용이
 '맞아, 이게 내가 원했던 거야' 싶으시다면,
 이번 달 안에 진행해보실 의향이 있으세요?"
```

**3분기 대응:**
```
[YES] "좋아요, 그럼 맞는지 확인해드릴게요" → 전체 플로우 진행
[MAYBE] "일단 들어보시고 판단하세요" → 전체 플로우 + 클로징 강화
[NO] "솔직하게 말씀해주셔서 감사해요.
      이번 달은 아니시고 언제쯤 생각해보실 수 있어요?"
     → 타임라인 확인 → 재연락 날짜 확정 → 통화 단축 종결
```

**왜 이게 필요한가 (인셀덤 원칙):**
```
"무료체험만 원하는 사람은 빠르게 거른다"
→ NO 받은 고객에 30분 쓰는 것 = 가장 큰 낭비
→ 조기 NO = 다음 콜에 시간 쓸 수 있음
→ 오히려 조기 NO 받으면 상담사 자존감 유지 + 다음 콜 집중
```

## ★ NEW v12 — 0-10점 스케일 (SPIN 전 브릿지)

### 핵심: 저항 원인 사전 파악 + 동기부여 면접 기법

```
[개입질문 5단계 완료 직후, 마이프렌드존 전 삽입]

"[고객명]님, 지금까지 말씀 들어보니까
 여행에 대한 생각이 있으신 것 같은데,
 솔직하게 여쭤볼게요.

 오늘 제가 설명드린 내용 기준으로,
 0에서 10점이면 몇 점이세요?
 0은 전혀 관심 없다, 10은 당장 시작하고 싶다."
```

**3구간 분기 대응:**
```
[7~10점] "7점이세요? 왜 10점이 아니에요?
          어떤 부분이 아직 걸리세요?"
          → 답변 = 진짜 저항 파악 → 맞춤 해소

[4~6점]  "4점이세요, 알겠어요.
          어떤 부분이 제일 걸리세요?
          가격이에요, 아니면 신뢰 문제예요,
          아니면 지금 시기가 아닌 것 같아서요?"
          → 유형 파악 → 맞춤 처리

[1~3점]  "솔직하게 말씀해주셔서 감사해요.
          그럼 지금은 아니시고, 언제쯤 생각해보실 수 있어요?"
          → 재연락 날짜 확정 → 단축 종결
```

**왜 효과적인가 (동기부여 면접 기법):**
```
7점 고객에게 "왜 10점이 아닌가?" 질문
→ 고객 스스로 저항 이유를 말함
→ 상담사가 파악하고 해소 가능
→ 고객이 스스로 말했으므로 Commitment 발동
```
