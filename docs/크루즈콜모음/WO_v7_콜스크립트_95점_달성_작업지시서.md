## PHASE A 작업 — RED (54점 → 75점)

### A1. 오프닝 패턴 인터럽트 (완전 교체)

**삭제**: 날씨 스몰토크
**교체**: "15분 후 내가 먼저 끊겠습니다" 심리적 통제권 이전

핵심 문장:
> "15분 안에 '이건 나한테 필요 없다' 싶으시면 제가 먼저 전화 끊겠습니다.
> 괜찮으세요?"

심리 메커니즘: 압박감 선제 제거 → 방어막 해제 → 통제권 이전 → 역설적 개방

**추가 오프닝 분기 (실전 상황)**:
- "카톡으로 보내주세요" 고객 처리
- "비용이 얼마예요?" 첫마디 고객 처리
- "지금 좀 힘들어요" 고객 처리

### A2. 클로징 신호 탐지표

아래 신호 감지 시 → 즉시 클로징 분기 (스크립트 건너뜀)

| 신호 | 즉각 반응 |
|------|---------|
| "언제까지 결정하면 돼요?" | "지금 바로 가능하세요?" |
| "비용이 얼마예요?" (3회 이상) | 가격 + 소프트 클로즈 |
| "어떻게 시작하면 돼요?" | "제가 안내해드릴게요" |
| "○○하면 어떡해요?" (가입 가정) | 바로 클로즈 |
| "언제 출발 가능해요?" | "고객님 일정 먼저 여쭤볼게요" |
| 3회 연속 Yes | 이중선택 클로즈 즉시 |

### A3. 가격 먼저 묻는 고객 처리

**패턴**: 고객이 PHASE 1-2에서 가격 요청 시

스크립트:
> S: "바로 말씀드릴게요. 딱 30초만 설명드리고 정확한 금액 드려도 될까요?
> 그래야 어떤 플랜이 맞으실지 제가 정확하게 안내드릴 수 있거든요."
> (허락 시) → 신탁 구조 30초 압축 설명 → 정확한 가격
> (거절 시) → "네 알겠습니다. 골드 기준으로 월 ○○만원이에요." 바로 답변

### A4. 상담사 자기 개방 문장 (3곳 삽입)

**위치 1** — Phase 1 신뢰 구축 시:

**위치 3** — 저항 처리 시:

## PHASE B 작업 — YELLOW (75점 → 88점)

6단계 파장 체인:
1. **즉각 손실**: "올해도 못 가시면..."

3. **체력 창문**: "60대 중반이 체력 최적 창문이에요"
4. **관계 파장**: "가족이랑 같이 가실 시간이..."
5. **기억 손실**: "10년 후에 후회하실 것들..."
6. **정체성**: "나는 어떤 사람으로 기억되고 싶은가"

수치 삽입 예시:
> "지금 1년에 한 번도 여행 못 가시면, 10년이면 10번이에요.
> 60대에 10번이 없으면... 그 기억들이 다 없어지는 거잖아요."

### B2. 5감각 + 자녀 인정 욕구 앵커링 (산토리니 장면)

**Phase 3 → Phase 4 전환 직전 삽입**

5감각 순서: 시각 → 촉각 → 후각 → 청각 → 미각

스크립트:
> S: "산토리니 이야기 하셨잖아요. 잠깐 상상해보실래요?
> 파란 지붕 위에 서 계시는 거예요 — 눈에 들어오는 에게해.
> (시각) 석양이 지는데 바람이 딱 살짝 불어요. (촉각)
> 거기서 소금기 있는 바다 냄새가 나고요. (후각)
> 파도 소리랑 멀리서 그리스 음악이 들리고. (청각)
> 그날 저녁에 신선한 올리브랑 와인 한 잔 하시는 거예요. (미각)"
>
> (3초 침묵)
>
> S: "그때 옆에 누구랑 가고 싶으세요?"
> (고객 대답 유도)
>
> S: "아드님/따님이 '엄마 진짜 잘 다녀오셨다' 하실 것 같지 않으세요?"
> *(자녀 인정 욕구 — 5060 최강 트리거)*

### B3. 고객 유형 5가지 분기 스크립트

**Phase 1 (2분 이내) 유형 판별 → 즉각 분기**

| 유형 | 판별 신호 | 핵심 접근 |
|------|---------|---------|
| **가성비직접파** | "얼마예요?" "이득이에요?" | 숫자 비교 → 1박당 비용 |
| **감정+특별의미파** | "부모님이랑" "꿈이었어요" | 5감각 앵커링 → 기억 프레임 |
| **정보수집형** | "어떻게 돼요?" "서류는?" | 단계별 설명 → FAQ 선제 |
| **그룹리더형** | "여럿이서" "모임에서" | 그룹 할인 → 리더 특권 |
| **장벽고객형** | "건강이..." "가족이 반대..." | 장벽 → 대안 → 소프트 랜딩 |

### B4. "자리 확보 — 취소 가능" 소프트 랜딩 클로즈
