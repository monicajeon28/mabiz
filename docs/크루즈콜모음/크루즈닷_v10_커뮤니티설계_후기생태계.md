---

> **핵심 철학:**
> "크루즈닷 고객은 여행을 산 것이 아니라, '골드 멤버'가 된 것이다."
> 소속감이 해지를 막고, 후기가 새 고객을 만들고, 공동체가 신뢰를 증명한다.

## 1. 골드 등급 시스템 설계

### 1-1. 등급 체계

| 등급 | 조건 | 혜택 | 심리 기제 |
|------|------|------|-----------|
| **실버** | 가입 첫 6개월 | 기본 혜택 | 입문 |
| **골드** | 6개월 이상 | 우선 배정 + 업그레이드 권한 | 소속감 |
| **플래티넘** | 1탑승 완료 | VIP 라운지 + 추가 할인 | 자부심 |
| **다이아몬드** | 3탑승 이상 | 전담 매니저 + 무료 업그레이드 | 특권감 |

### 1-2. 등급 언급 타이밍 (콜 스크립트 삽입)

**오프닝에서 (알파 테크닉과 연결)**:
```
"이번에 [고객명]님을 골드 우선 배정 대상으로 선정해서..."
→ 처음부터 "골드 멤버가 될 사람"으로 프레임
```

**클로징 직전 (소속감 자극)**:
```
S: "저희 골드 멤버들만 들어오는 단톡방이 있어요.
   거기서 다른 멤버분들이 기항지 팁이나 선내 꿀팁들
   공유해주시거든요. 거기 들어오시면 여행이 두 배로 풍성해져요."
```

**버튼업에서 (신규 가입 직후)**:
```
S: "[고객명]님, 이제 골드 멤버세요.
   제가 골드 전용 단톡방 링크 카톡으로 보내드릴게요."
```

## 2. 단톡방 운영 규칙 (골드 전용)

### 2-1. 단톡방 종류

| 방 | 대상 | 인원 | 목적 |
|----|------|------|------|
| **크루즈닷 골드 라운지** | 전체 골드 멤버 | 300명 제한 | 정보 공유, 커뮤니티 |
| **[선박명] 탑승 준비방** | 같은 일정 예약자 | 20~50명 | 일정 조율, 친목 |
| **탑승 후기방** | 탑승 완료자 | 무제한 | 후기 공유, 재탑승 유도 |

### 2-2. 골드 라운지 운영 규칙

**운영 원칙:**
```
1. 매주 월요일 09:00: 주간 인기 기항지 팁 공유
2. 매주 수요일 12:00: 특가 정보 선공개 (골드만 24시간 우선 예약)
3. 매주 금요일 18:00: 탑승 후기 스토리 1개 공유
4. 탑승 10일 전: 탑승 준비 체크리스트 개인 발송
```

**금지 규칙:**
```
- 직접 세일즈 메시지 금지 (자연스러운 정보 제공만)
- 정치/종교/분쟁 금지
- 개인정보 요청 금지
- 상업 광고 금지
```

**매니저 운영 멘트 예시 (정기 발송):**
```
[월요일 09:00]
"안녕하세요 골드멤버 여러분! 🌊
이번 주 인기 기항지 꿀팁 공유해요.
일본 가고시마 기항 시 꼭 가셔야 할 곳: [정보]
이번 주도 좋은 한 주 되세요!"

[금요일 18:00]
"이번 주 후기 주인공은 박OO 어머님이에요.
3박4일 일본 코스 탑승하시고 사진 보내주셨어요.
[사진 공유] '생전 처음 크루즈인데 너무 좋았어요!'
다음 주엔 누가 주인공이 될까요? 😊"
```

### 2-3. 탑승 준비방 운영 (일정별)

**탑승 D-10 자동 생성:**
```
방 이름: "MSC 벨리시마 5/10 출발 탑승팀"
```

## 3. 후기 생태계 선순환 구조

### 3-1. 선순환 흐름도

```
가입 → 탑승 준비방 → 탑승 완료 → 후기 요청 → 후기 공유 → 신규 가입자 신뢰 형성
  ↑                                                              |
  └──────────────────────────────────────────────────────────────┘
                    신규 가입자 → 같은 사이클
```

### 3-2. 후기 수집 전략

**탑승 후 D+1 자동 카톡:**
```
"어머님, 귀국하셨죠? 🌊
여행 어떠셨어요? 제가 너무 궁금해서요.
사진 한 장이라도 보내주시면 너무 좋겠어요!
골드 라운지에 공유해드릴게요."
```

**후기 수집 포맷 (전송):**
```
간단하게 3가지만 알려주세요:
1. 가장 좋았던 순간 한 문장
2. 사진 1~3장
3. 별점 (⭐⭐⭐⭐⭐)

보내주시면 제가 예쁘게 만들어서 공유할게요 💛
```

### 3-3. 후기 등급별 활용처

| 후기 유형 | 활용처 | 목적 |
|-----------|--------|------|
| 텍스트 1줄 | 단톡방 공유 | 커뮤니티 활성화 |
| 사진 + 텍스트 | 골드 라운지 + SNS | 바이럴 |
| 영상 후기 (15~30초) | YouTube Shorts 소스 | 영상 콘텐츠 |
| 상세 리뷰 | 앱스토어 리뷰 유도 | 신뢰도 |

**영상 후기 요청 멘트 (탑승 완료 후):**
```
"어머님, 혹시 15초짜리 짧은 영상 하나만 찍어주실 수 있어요?
'크루즈 타보니 어땠다' 한 마디만요.
저희 다른 분들한테 진짜 큰 도움이 돼요.
영상 주시면 어머님 다음 탑승 때 특별 혜택 드릴게요."
```

### 3-4. 재탑승 유도 타이밍

```
탑승 완료 D+3: 후기 수집 + 감사 인사
탑승 완료 D+30: "다음 일정 미리 보기" 자료 발송
탑승 완료 D+90: "이번 시즌 인기 코스" 맞춤 추천
탑승 완료 D+180: 다이아몬드 등급 안내 + 재탑승 특가
```

## 4. 후기 → 신규 연결 실전 스크립트

### 4-1. 콜에서 후기 활용 (마이프렌드존 연계)

```
[신규 고객 통화 중]

S: "어머님, 저희 고객분들이 탑승 후 남겨주신 후기가 있는데요,
   카톡으로 보내드릴까요?
   실제 탑승하신 분들 사진이랑 후기거든요.
   보시면 더 실감 나실 것 같아요."

→ 후기 자료 발송 → 신뢰도 즉각 상승
→ "이거 진짜 탑승하신 분들 후기예요?" 반응 유도
```

### 4-2. 후기 자료 카톡 템플릿

```
[카톡 발송 내용]

"[고객명]님, 말씀드린 후기 보내드려요 😊

⭐ 60대 박OO 어머님 후기
"일본 크루즈 3박4일 탑승했어요.
 가고시마 라멘이 진짜 맛있었고,
 배 안에서 쉬는 것도 너무 편하고 좋았어요.
 나이 들어서도 여행할 수 있다는 게 감사했어요."
[사진 1장]

⭐ 40대 김OO 어머님 후기
"아이들이랑 같이 탔는데 키즈클럽이 최고였어요!
 아이들이 또 가고 싶다고 해서 재탑승 예약했어요."
[사진 1장]

궁금한 거 있으시면 편하게 연락 주세요 💛
[이름] 매니저 드림"
```

## 5. 해지 방어 시스템 (커뮤니티 기반)

### 5-1. 해지 위험 신호

```
1. 단톡방 2주 이상 미반응
2. 3개월 연속 납입 후 갑자기 연락
3. "해지하고 싶다" 명시적 언급
```

### 5-2. 해지 위기 대응 스크립트

```
[전화 또는 카톡]

S: "[고객명]님, 오랜만이에요.
   혹시 최근에 어떻게 지내고 계세요?"

C: "사실... 취소할까 생각 중이에요."

S: "아, 말씀해주셔서 감사해요.
   무슨 이유인지 여쭤봐도 될까요?"

C: "경제적으로 좀 힘들어서요."

S: "그렇군요. 충분히 이해해요.

   근데 한 가지만 여쭤봐도 될까요?
   지금 적립된 금액이 [X]만원이에요.
   지금 취소하시면 그게 정산되는데,
   이걸 [X]개월만 더 유지하시면
   실제 탑승에 사용하실 수 있거든요.

   지금 취소하시는 것보다
   [X]개월 후에 여행 한 번 다녀오시는 게
   어떨까요?"
```

### 5-3. 커뮤니티 해지 방어

```
해지 의향 감지 후 → 단톡방 '골드 전용 특가' 선공개 → 참여 유도

"OO님, 이번에 골드 멤버 특가로 일본 코스가 나왔어요.
 혹시 탑승 계획 있으세요?
 적립금 쓰실 수 있는 좋은 기회예요."
```

## 7. 즉시 실행 체크리스트

```
D+0 (오늘 할 것):
□ 카카오 오픈채팅 "크루즈닷 골드 라운지" 개설
□ 기존 고객 전체 초대 발송
□ 첫 번째 탑승 후기 1개 수집 → 공유

D+7:
□ 탑승 준비방 운영 프로세스 확립
□ 후기 수집 카톡 템플릿 완성 및 전송 자동화 계획

D+30:
□ 후기 3개 이상 수집 완료
□ 영상 후기 1개 확보 → YouTube Shorts 소스
□ 재탑승 문의 1건 이상 발생 목표
```

---

> **선순환 요약:**
> 1. 가입 → 즉시 골드 커뮤니티 소속감
> 2. 탑승 전 → 준비방에서 설렘 증폭 + 정보 공유
> 3. 탑승 후 → 후기 수집 + 공유 = 다음 신규 가입자 신뢰 형성
> 4. 재탑승 → 다이아몬드 등급 + 더 강한 소속감
> 5. 후기가 콜에서 마이프렌드존으로 활용 → 클로징률 상승
>
> **결론**: 커뮤니티가 살아있으면 세일즈는 자동으로 따라온다.
