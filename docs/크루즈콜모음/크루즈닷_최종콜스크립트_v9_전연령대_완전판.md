## ⚠️ v8→v9 핵심 변경사항

```
v8 결정적 오류:
  - "어머니" 호칭 고정 → DB 신청자의 30~50% 잘못된 호칭 사용
  - 5060 여성 심리만 반영 → 30/40대에 완전히 다른 저항 미처리
  - 단일 감정 앵커("내 차례") → 신혼부부/가족 고객에게 무효

v9 해결:
  - 5개 세그먼트 완전 분기 (30대~60대+ / 신혼~은퇴)
  - 세그먼트별 호칭 / 앵커 / 저항 처리 / CTA 전부 개별화
  - 공통 자산(신탁구조, 에이킨법, 침묵규칙, 멀티콜) 유지
```

## ★★ 세그먼트 판별 시스템 — 오프닝 3개 질문

```
판별 질문 1: "혹시 누구랑 가시려고 생각하세요?"
  → "남편이랑요" / "아내랑요" / "신혼여행으로요"
     → 세그먼트 A (신혼 언급 시) / B (자녀 언급 시) / C/D (중장년 부부)

판별 질문 2: "자녀분 계세요? 아직 학교 다니세요?"
  → "네, 초등학생이요" → 세그먼트 B
  → "다 컸어요" → 세그먼트 C/D

판별 질문 3: "혹시 연세가 어떻게 되세요?" (간접 버전)
  "어디서 저희를 알게 되셨어요?" → 어투/반응으로 나이 추정
```

## ⚡ 공통 원칙 (전 세그먼트 동일 적용)

```
원칙 1: 1차 콜 = 감정 앵커 3개 + 재연락 허가 (클로징 절대 금지)
원칙 2: 모든 저항 = 자아 인정 → 유도질문 → 프레임 전환 3단계
원칙 3: 침묵 규칙 — 클로징 후 5초, 감정 앵커 후 3초, 반드시 지키기
원칙 4: 에이킨법 — "그러니까요, 그래서 오늘 연락드린 거예요" (콜당 최대 3회)
원칙 5: 소비자언어 v2 — 계약→신청, 해지→취소, 가입비→여행적립금, 보장→보관
원칙 6: 멀티콜 시퀀스 — 1차(감정)→카톡→2차(클로징)→3차(노쇼방지)
원칙 7: 렌탈 언급 = 2차 콜 가격 거절 3회 이상 후에만
```

# 부록 E: 실패 패턴 경고 (에리카-박지영 케이스 교훈)
# ══════════════════════════════════════════════════════

```
실패 패턴 1: 세그먼트 판별 전 고정 호칭 사용
  × 30대 커플에게 "어머니" → 즉각 불쾌감
  ○ 2분 내 세그먼트 판별 → 호칭 전환

실패 패턴 2: 자아 인정 없는 즉각 반박
  × "그게 아니고요..."
  ○ "그러니까요. 그렇게 느끼실 수 있어요."

실패 패턴 3: 비용 질문 무시
  × 비용 질문에 화제 전환
  ○ "말씀드릴게요. 비교해드릴게요."

실패 패턴 4: 논리로 감정을 이기려는 시도
  × 조건 설명 → 반박 → 조건 설명 반복
  ○ 이미지 → 비유 → 공감 → 이미지

실패 패턴 5: 침묵 두려움
  × 고객 침묵 1초 만에 개입
  ○ 5초 버티기

실패 패턴 6: 세그먼트 맞지 않는 앵커
  × 30대 커플에게 "체력 창문 = 무릎"
  ○ 30대 = "평생 첫 번째 여행", "파트너 표정"

성공 공식:
  이미지 > 비유 > 공감 > 질문 > 침묵 > 고객의 말 > 에이킨법
```
