## 항목 1: 오프닝 / 경계심 해제 — 9/15점
## ━━━━━━━━━━━━━━━━━━━━━━━━━━━

### v6 현재 설계
```
"여보세요~ ○○○ 선생님이시죠?"
날씨 1~2 문장
"지금 10분 정도 통화 괜찮으세요?"
```

### 문제점 3가지

**문제 1: 날씨 스몰토크는 실전에서 쓰이지 않는다 (-2점)**

실제 클로징 완료 콜 5개 분석 결과:
- 사라(이혜영): "안녕하세요. 크루즈 팀장 김유미인데요." → 바로 핵심
- 사라(최혁준): "5분 정도 혜택 안내 도와드리려고..." → 바로 핵심
- 저스틴(소영수): "이벤트 혜택으로 거의 5분의 1 가격에..." → 오프닝에서 바로 앵커

날씨 소재는 고객이 협조적일 때만 작동한다. 실전에서 절반 이상의 첫 마디는:
- "카톡으로 보내주세요" (김원현)
- "지금 좀 힘들 것 같은데요" (박지영)
- "비용이 얼마예요?" (박지영)

v6는 순조로운 오프닝만 준비되어 있다.

**문제 2: 심리적 통제권 이전 장치 없음 (-3점)**

SPIN 18단계 마스터의 오프닝 핵심:
> "15분 안에 '이건 나한테 필요 없다' 싶으시면 제가 먼저 일어나겠습니다."

이 문장의 심리 메커니즘:
- 압박감 선제 제거 → 방어막 해제
- 통제권을 고객에게 이전 → 역설적으로 고객이 더 열림
- "이 사람은 팔려는 사람이 아니다" 인상 형성

v6의 "날씨+시간 확인"으로는 이 심리 해제가 일어나지 않는다.

**문제 3: 즉각 저항 상황 대응 분기 없음 (-1점)**

"카톡으로 보내주세요" → v6 대응: 없음
"비용이 얼마예요?" → v6 대응: Phase 4까지 기다리라고 설계 → 실전에서 콜 죽음

### v6가 잘하는 것
- DB 재확인 첫 Yes 구조는 올바름
- 이중선택으로 시간 확보하는 것은 유효

## 항목 2: 감정 설계 (4단계 흐름) — 8/20점
## ━━━━━━━━━━━━━━━━━━━━━━━━━━━

**이 항목이 가장 심각한 결함이다. 상위 0.1%와 하위 50%를 가르는 핵심.**

### 줄리아가 설계한 4단계 감정 흐름
```
1단계: 고통 인식 — 고객이 몰랐던 불안을 스스로 자각
2단계: 고통 심화 — 최악 시나리오 상상 (공포 극대화)
3단계: 희망 커리 — 해결 후 이상적 미래 상상 (도파민 분출)
4단계: 긴박감 — "지금 안 하면 기회 없다" (공포 재진입)
→ 5단계: 증거 + 가격 (감정이 최고조일 때만 제시)
```

### v6의 실제 구조
```
Phase 1: 경계심 해제
Phase 2: 신뢰 구축 (신탁 설명)
Phase 3: 여행 재점화 (SPIN)
Phase 4: 가격 앵커링
Phase 5: 클로징
→ 정보 전달 + 신뢰 구축 구조 (감정 흐름 설계 없음)
```

### 4가지 치명적 누락

**누락 1: 고통 심화 구간 없음 (-4점)**

줄리아 수진 씨 사례:
> 고객: "얘기하다 보니 지금 내 상태가 생각보다 더 불안하네"
> → 이 말이 나오기까지 상담사가 설계한 것

v6의 SPIN P 질문: "어떤 부분이 제일 걸리세요?"
→ 고통을 발견하는 수준. 심화 없음.

5060 대상 고통 심화 문장 예시 (v6에 없음):
```
"은퇴하시고 나서 막상 가려 했더니 무릎이..."
"매년 '내년엔 가야지' 하시다 보면..."
"60대에 갈 수 있는 크루즈가 70대에도 가능할까요?"
```

**누락 2: 희망 커리(도파민 분출) 구간 없음 (-3점)**

v6의 산토리니 장면은 5감각 중 시각 2개:
```
현재: "파란 돔 성당... 석양... 와인 한 잔... 남편분이랑"
```

18단계 마스터 기준 완전판 (5감각 완전 동원):
```
"지금 오후 6시예요.
 어머니는 이아 마을 전망대에 서 계세요.
 하늘이 주황빛으로 물들고 있고,
 저 멀리 에게해가 금색으로 반짝여요.
 옆에서 남편분이 '우리 진짜 잘 왔다' 하시거든요.
 그 사진 카톡으로 자녀분들한테 오면
 애들이 뭐라고 할 것 같아요?"
```

차이: v6는 2가지 감각, 마스터는 5가지 + 자녀 인정 욕구까지 동원.
5060의 가장 강한 구매 동기는 자녀에게 인정받고 싶은 욕구인데 v6는 이것을 아예 건드리지 않는다.

**누락 3: 가격 제시 타이밍 역전 (-2점)**

줄리아의 핵심 원칙:
> "고객이 먼저 가격을 물어볼 때까지 가격을 꺼내지 않는다"
> "수진 씨가 '천만 원 내도 아깝지 않을 것 같다'고 생각할 때 → '한 달에 70만 원이면 됩니다'"

v6는 Phase 4에서 상담사가 먼저 가격 앵커링 시작. 가치가 충분히 쌓이기 전에 가격 노출 → "비싸다" 반응 발생 구조.

**누락 4: 손실 수치화 없음 (-3점)**

줄리아: "지금 월 200 버는 사람이 10년 후 22억 손실"처럼 기회비용을 숫자로.

v6 고통 계산기:
```
"지금 안 하시면 산토리니는 언제 가실 것 같으세요?"
→ 감정적 질문만. 숫자 없음.
```

강화 버전 (v6에 없음):
```
"알래스카 빙하는 지금도 매년 녹고 있어요.
 지구온난화로 10년 후엔 지금의 70%만 남는다고 해요.
 3개월 후 이 출발일 지나면 다음 동일 코스는 내년까지 기다리셔야 해요."
```

## 항목 3: SPIN 질문 구조 — 10/15점
## ━━━━━━━━━━━━━━━━━━━━━━━━━━━

v6의 SPIN 설계는 방향이 맞다. 그러나 3가지 결함.

### 결함 1: Implication 깊이가 치명적으로 얕다 (-3점)

**v6 현재 I 구간:**
```
"올해도 결국 못 가실 것 같으세요?"
"나중에 가야지 하시다 보면 건강이나 체력 때문에 더 힘드실 수도..."
→ 끝 (2문장)
```

**18단계 마스터 I 구간 (6레이어):**
```
"10년이요. 그러면 앞으로 10년도 같은 방식으로 다니시면 어떻게 될 것 같으세요?"
[고객 답변 후]
"60대에 '다음에 하지' 하신 분들 중 70대에 실행한 분이 23%예요."
[2초 침묵]
"그게 단순한 여행 문제가 아닌 거잖아요."
[고객 표정 변화 확인] "눈이 반짝이셨다"
"이게 포기하게 되는 거거든요."
```

SPIN 연구 결과: Implication은 구매 확률과 가장 강한 상관관계. v6는 이것을 2문장으로 처리한다.
고객은 현상 유지의 고통을 체감하기 전에 해결책을 듣게 되어 "좋은 것 같긴 한데 굳이 지금?"이 된다.

### 결함 2: "들어볼 의향 있으세요?" 허가서 없음 (-2점)

18단계 마스터의 N 단계 전 필수 문장:
> "그게 되는 방법이 있으면 한번 들어볼 의향은 있으세요?"

이 문장의 심리 메커니즘: 해결책을 상담사가 밀어넣는 게 아니라 고객이 요청하게 만드는 심리적 허가서.
v6는 이 허가 없이 산토리니 장면으로 넘어감 → 상담사가 강요하는 설명으로 들릴 위험.

### 결함 3: 경험자 분기 없음 (-0점 — 별도 항목에서 차감)

김원현 콜처럼 크루즈 경험자가 전화 받으면 SPIN S단계 전체가 불필요해진다.

## 항목 4: 가격 대응 / 앵커링 — 8/15점
## ━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 결함 1: 가격 먼저 묻는 고객 = 구매 신호 인식 없음 (-4점)

**에리카 콜 녹취 — 실전 최대 실패 사례:**
```
고객: "비용이 어떻게 될까요?" (구매 의향 있는 신호!)
상담사: "비용만 들으시면 어떤 내용인지 전혀 알 수가 없으시잖아요" (막음)
고객: "비용이 어떻게 되세요?" (2번째 질문)
상담사: (또 막음)
→ 2분 39초에 콜 사망
```

v6 설계: Phase 4(15~20분)에서 역질문 3단계 후 가격 제시.
실전: 고객이 3분 이내에 가격을 물어보면 v6 전체 설계가 붕괴.

**올바른 대응 (v6에 없음):**
```
고객: "비용이 얼마예요?"
상담사: "바로 말씀드릴게요. 보통 패키지로 유럽 가시면 인당 350~400만원이에요.
        저희는 그게 어떻게 가능한지 30초만 설명드려도 될까요?"
→ 가격 정보 주되, 맥락으로 감싸는 기술
```

가격 먼저 묻는 고객은 구매 의향이 이미 있는 것. 막으면 죽는다.

### 결함 2: 일일 분할 가격화 없음 (-2점)

줄리아: "한 달에 70만 원 → 카드 12개월 할부 → 너무 다행이다"
18단계 마스터: "월 33,000원, 하루 1,100원이에요. 커피 한 잔 값이에요"

v6: "월 ○○만원"으로만 표현. 일 단위 분할이 없음.

### 결함 3: 재여행 복리 개념 없음 (-1점)

줄리아 월당백의 핵심 무기: 재구매율 80% = 시간이 지날수록 자동으로 쌓이는 구조.
크루즈 재탑승률도 업계 최고 수준인데, "한 번 가시면 또 가고 싶어지세요"라는
재구매 구조 언급이 v6에 전혀 없음.

## 항목 5: 클로징 / 저항 처리 — 9/15점
## ━━━━━━━━━━━━━━━━━━━━━━━━━━━

### v6가 잘하는 것
- 소프트/미디엄/하드 클로즈 3단계 구조: 올바름
- 이중선택(A/B) 구도: 유효
- 역질문("어떤 부분을 더 생각해보고 싶으세요?"): 정확함

### 결함 1: 선거절이 3개뿐 — 15개 중 3개 (-3점)

v6 선거절: 상조/신탁/가족반대 3개
18단계 마스터 선거절 15개 중 v6에 없는 것:

```
❌ "자식들이 뭐라 할 것 같아" 처리
❌ "나이 들어서 몸이 안 좋은데" 처리
❌ "주변에 가본 사람이 없어서 검증이 안 돼" 처리
❌ "지금 바로 결정 못 하겠어" → "자리 확보 취소 가능" 처리
❌ "인터넷에 좋은 후기만 있지 않아요?" 처리
❌ 결제 직전 멈춤 처리
```

### 결함 2: 가장 강력한 소프트 랜딩 없음 (-2점)

18단계 마스터의 클로징 최강 문장:
> "오늘은 결정 안 하셔도 돼요. 다만 오늘 자리만 하나 확보해두시는 건 어떨까요?
>  자리 확보는 취소 가능하거든요. 결정은 나중에 하시고."

이 문장의 구조:
- "결정 안 하셔도 돼요" → 결정 부담 0
- "자리만 확보" → 행동은 유지
- "취소 가능" → 손실 공포 제거
→ 세 가지 저항을 한 문장으로 처리

실전 콜에서 클로징 신호를 놓친 사례:

```

→ 상담사: 날짜와 일정 설명으로 넘어감 (기회 날림)

박지영: "비용이 어떻게 될까요?" ← 구매 의향 신호
→ 상담사: 두 번 막음 (콜 사망)

박혜영: "갑자기 아파서 못 가면 어떡해요?" ← "내가 간다"고 상상하는 신호
→ 상담사: 잘 처리함 (이 경우만 성공)
```

v6에는 이 클로징 신호를 인식하고 즉시 Phase 5로 점프하는 규칙이 없다.

## 진단: v6가 54점인 진짜 이유
## ━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 구조적 원인 3가지

**원인 1: "정보 전달 + 신뢰 구축" 설계 vs "감정 여정 역설계" 설계**

v6는 상담사가 정보를 전달하고 신뢰를 쌓는 구조.
상위 0.1% 콜은 고객이 스스로 욕망을 말하고, 고통을 체감하고, 가치를 계산하게 만드는 구조.

핵심 차이: 주어가 다르다.
- v6: 상담사가 → 설명하고 → 납득시킨다
- 상위 0.1%: 고객이 → 스스로 느끼고 → 결정한다

**원인 2: "이상적 고객" 가정의 함정**

30분, SPIN 4단계, 협조적 고객 → 실전의 10~20%에만 해당.
나머지 80%에서는 Phase 2가 끝나기도 전에 구조 붕괴.

**원인 3: 기법이 병렬 나열, 상황별 분기가 없다**

v6는 수십 개의 기법을 알려주지만 "언제, 어떤 상황에서, 어떤 순서로"가 명확하지 않다.
실전에서는 기법을 선택하는 판단 기준이 더 중요한데,
v6는 선형 순서(Phase 1→2→3→4→5→6)로만 설계되어 있다.
