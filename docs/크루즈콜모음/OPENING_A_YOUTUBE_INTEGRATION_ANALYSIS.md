# Opening A YouTube 데이터 통합 분석

## 1. 입력 데이터 정리

### 1-1. 기존 자료
- **기본 구조**: PHASE 1-6 (경계심 해제 → 신뢰 → SPIN → 가격 → 이의 → 클로징)
- **기존 Opening A**: "부모님이랑 아직도 제대로 여행 못 가셨어요?"
- **문제 신호**: 고객 반응 "뭐야?" (신분 미확인 신호)

### 1-2. YouTube 데이터 분석 결과
| 감정 주제 | 클로징율 범위 | 나이대 특성 | 주요 메커니즘 |
|----------|-----------|----------|-----------|
| **효도** | 75-85% | 5060+, 부모님 동반 | Implication 깊이, 후회회피 극대 |
| **시간 FOMO** | 65-75% | 40-50대 부부 | 체력창문, 비행동의 비용, 손실회피 |
| **꿈** | 70% | 30-40대 | 자기투영, 희소성, 미래자아 |

### 1-3. Opening A의 구조적 문제점

**현재 Opening A:**
```
"부모님이랑 아직도 제대로 여행 못 가셨어요?"
↓
문제: PHASE 1(신분확인) 스킵 → 즉시 감정 자극
결과: "뭐야?" 반응 = 경계심 해제 실패 신호
```

## 2. v13 공식 PHASE 1-3 구조 검증

### 2-1. v13 STAGE 0-2 (= PHASE 1-2)

**STAGE 0: 오프닝 — 출처 명확화 (핵심!)**
```
상담사: "[고객성함]님 맞으시죠? 저는 크루즈닷 여행 전문 매니저 [이름]이에요.
        [제휴사명] 고객 혜택 안내드린다고 동의해 주셔서 연락드리게 됐는데요.
        지금 통화 2~3분 괜찮으실까요?"
        
효과: 출처 명확화 → 어디서 번호 알았어요? 차단
      2~3분 시간 제한 → 심리적 부담 제거
      자율성 부여 → 수용률 상승
```

**STAGE 1: 선거절 카드 — 초반 저항 즉시 해소**
```
상담사: "아, 맞아요. 바쁘신 분한테 길게 잡으려는 게 아니에요.
        딱 한 가지만 여쭤볼게요 —
        올해 여행 계획 있으세요, 아직 없으세요?"

효과: 부담 제거 후 단 하나의 오픈질문으로 화제 전환
      3가지 유형(바빠요/관심없어요/생각해볼게요) 분기 대응
```

**STAGE 2: 구매의향 사전동의 + 세그먼트 판별**
```
상담사: "만약 오늘 제가 말씀드리는 내용이 '맞아, 이게 내가 원했던 거야' 싶으면,
        이번 달 안에 진행해보실 의향이 있으세요?"

효과: 30분 낭비 방지 (YES/MAYBE/NO 분기)
      세그먼트 판별 3질문으로 접근 방식 결정
```

## 3. Opening A가 끼워넣어야 할 올바른 위치

### 3-1. 현재 구조 (잘못됨)

```
[STAGE 0] 출처 명확화 (NOT DONE) ❌
         ↓
[Opening A] "부모님이랑 아직도 제대로 여행 못 가셨어요?" ← 너무 빨리 감정 자극
         ↓
[STAGE 1] 선거절 카드
         ↓
[STAGE 2] 세그먼트 판별
         ↓
[STAGE 3] MI 욕망 탐색
```

**문제: 고객이 "뭐야?" = 신분이 확인되지 않은 상태에서 감정 자극 시도**

### 3-2. 올바른 구조 (수정안)

```
[STAGE 0] 출처 명확화 — "저는 크루즈닷 여행 전문 매니저 [이름]이에요" ✓
         ↓
         신뢰 기반선 확보 (PHASE 1 = 경계심 해제)
         ↓
[STAGE 1] 선거절 카드 + 세그먼트 판별 (PHASE 2 = 신뢰 구축)
         ↓
         세그먼트 확인 완료: A(30대 커플) / B(가족) / D(5060) 등
         ↓
[STAGE 3] MI 욕망 탐색 (PHASE 3 = 욕망 활성화) ← 여기서부터 Opening A 활용
         ↓
         → Desire/Reason 발화 유도 → Change Talk 수집
         ↓
[Opening A 최적화 버전] YouTube 심리학 통합
```

## 4. YouTube 데이터를 Opening A에 통합하는 방법

### 4-1. 문제: Opening A가 단순 감정 자극이었음

**기존 Opening A:**
```
"부모님이랑 아직도 제대로 여행 못 가셨어요?"

분석:
- 세그먼트 구분 없음 (30대 커플한테도 "부모님"은 부적합)
- YouTube 데이터(효도율 75-85%) 활용했으나
- PHASE 1 신뢰 기반 없이 사용 → 고객이 방어
- "뭐야?" = 신분 미확인 + 갑작스런 감정 자극
```

### 4-2. 해결책: YouTube 데이터를 세그먼트별로 분기

**세그먼트 D (5060 은퇴) — 효도 FOMO 강함 (75-85%)**
```
[STAGE 0] 출처 명확화 완료 ✓
         ↓
[STAGE 2] 세그먼트 판별
         상담사: "보통 누구랑 여행 다니세요?"
         고객: "아이들이랑... 근데 부모님도 가시고 싶어 하시고..."
         판정: Segment D (5060 본인 또는 부모님 동반)
         ↓
[STAGE 3] MI 욕망 탐색 - Reason 발화 유도
         
         상담사: "[성함]님이 크루즈를 생각해본 이유가 뭐였어요?"
         고객: "..."
         
         상담사: "혹은 부모님이 평생 가고 싶으셨던 여행이 있으세요?"
         고객: "네, 엄마가 유럽을 가고 싶어 했거든요."
         
         상담사: "부모님이 가고 싶으셨던 거군요.
                그런데 지금까진 못 가셨던 이유가 뭐였어요?"
         고객: "바빴고... 비싸기도 하고..."
         
         상담사: "아, 그렇군요. 그런데 만약 [고객님]께서 부모님을 모시고
                평생 못 드렸던 그 여행을 드릴 수 있다면?
                부모님 얼굴이 어떨 것 같으세요?"
         
         ← 이 순간이 Opening A 최적화 버전
         ← YouTube 데이터 (효도 심리 75-85%) 활용
         ← 고객 본인 말에서 나온 욕망이므로 저항 없음
```

**세그먼트 A (30대 커플) — 희소성/꿈 (70% 클로징)**
```
[STAGE 2] 세그먼트 판별 완료 ✓
         고객: "저희 둘이서 가려고요. 신혼여행으로..."
         판정: Segment A
         ↓
[STAGE 3] MI 욕망 탐색 - Desire/Reason
         
         상담사: "아, 신혼여행이군요. 신혼여행으로 크루즈를 생각해본 이유가?"
         고객: "유튜브에서 봤는데... 여행이 너무 멋있더라고요."
         
         상담사: "어떤 장면이 제일 멋있어 보였어요?"
         고객: "배 위에서 하늘 보는 거... 그리고 발코니에서 나랑 남편이..."
         
         상담사: "아~! 그 장면이. 그런데 그걸 보면서 어떤 생각이 들었어요?"
         고객: "우리도 저렇게 추억 만들고 싶다는..."
         
         상담사: "[성함]님, 신혼부부라면 평생에 딱 한 번인 첫 크루즈잖아요.
                그리고 그 경험이 앞으로 몇십 년의 결혼생활에서
                가장 특별한 추억으로 남는다는 거, 알고 계세요?
                심리학으로도 증명됐어요."
         
         ← YouTube 데이터 (꿈/희소성 70%) 활용
         ← 고객 본인 욕망에서 나온 표현 재활용
```

**세그먼트 C (40-50대 중년 부부) — 시간 FOMO (65-75%)**
```
[STAGE 2] 세그먼트 판별
         고객: "우리 둘이... 아이들도 크니까 조용한 여행 하고 싶어서"
         판정: Segment C (황금기 버전)
         ↓
[STAGE 3] MI 욕망 탐색
         
         상담사: "아, 부부 시간이 필요하신 거네요.
                그런데 지금까진 그럴 기회가 없으셨어요?"
         고객: "맨날 일하고... 애들 챙기고... 잠만 자죠."
         
         상담사: "맞아요. 40대 부부라면 지금이 정말 특별한 시기잖아요.
                아이들도 독립하기 전이고, 직장도 안정적인...
                이 황금기가 지나면 어떻게 될 것 같으세요?"
         고객: "글쎄... 더 바빠질 수도..."
         
         상담사: "그래요. 창문이 열려있을 때 여는 거죠.
                만약 다음 5년 동안 이런 기회를 못 쓴다면?
                정말 후회될 것 같아요?"
         
         ← YouTube 데이터 (시간 FOMO 65-75%) 활용
         ← 비행동의 비용을 가시화 (손실회피 심리)
```

          ↓

## 6. 실제 콜 예시: Opening A를 올바른 위치에 배치

### 6-1. Segment D (5060 효도) — 콜 플로우

```
STAGE 0 (0분-1분):
상담사: "안녕하세요, [이순신]님 맞으시죠?
        저는 크루즈닷 여행 전문 매니저 [경아]예요.
        [신한은행 VIP] 고객 혜택 안내로 연락드리게 됐어요.
        지금 2~3분 괜찮으실까요?"

고객: "네, 괜찮아요."

STAGE 1-2 (1분-4분):
상담사: "감사합니다. 보통 누구랑 여행 다니세요?"

고객: "아, 저희는 부모님이랑 아이들 같이 가요."

상담사: "아, 그러면 3대가 함께하시는군요!
        그런데 평소에 부모님과 함께 여행 가실 기회가 자주 있으세요?"

고객: "아니에요. 부모님이 건강해도 바빠서... 
      엄마가 유럽은 가보고 싶다고 하는데..."

STAGE 3 (4분-10분):
상담사: "아, 어머님이 유럽을 꿈꾸고 계셨군요.
        그런데 지금까진 못 가셨던 이유가 뭐였어요?"

고객: "비용도 있고... 시간도 없고..."

상담사: "[성함]님, 혹시 이런 생각은 안 해보셨어요?
        '내가 평생 엄마한테 못 해준 것 중에
        한 가지라도 해줄 수 있다면 어떨까?'
        어머님이 그런 생각을 하실 때 얼굴이 어떨 것 같으세요?"

고객: (잠깐 침묵) "... 정말 행복할 것 같아요."

상담사: "그죠. 그런데 말이에요, 지금이 그 시간인 거예요.
        어머님의 체력이 좋으실 때, 
        [성함]님도 함께할 수 있을 때.
        이 창문이 닫히기 전에 선물로 드릴 수 있는 마지막 기회가
        지금이라는 거죠."

← 여기서 YouTube 심리학 (효도 75-85%) + PHASE 3 (욕망) 완성
← 고객이 "부모님 따뜻한 얼굴"을 상상하며 Change Talk 발화
← 클로징률 75-85% 기대
```

## 7. v13 구조에서 Opening A의 올바른 역할 재정의

### 7-1. v13에서 Opening A는 "기술"이 아니라 "결과"

```
잘못된 이해:
"Opening A = 상담사가 먼저 말하는 오프닝 멘트"

올바른 이해:
"Opening A = 고객이 자신의 욕망을 정의한 후,
            상담사가 그것을 반영하여 그려주는 최종 이미지"

v13 구조:
STAGE 0: 신뢰 기반선 (출처명확화)
         ↓
STAGE 1-2: 세그먼트 판별 (누가 이 고객인가?)
         ↓

```

## 11. 최종 요약

### 기존 문제
- **Opening A가 PHASE 1 (신뢰) 구축 전에 배치됨** → 고객 반응 "뭐야?"
- **YouTube 데이터가 세그먼트별로 분류되지 않음** → 일괄 효도 심리 적용 (30대 커플에겐 부적합)
- **v13 구조에서 Opening A의 위치가 명확하지 않음** → 상담사가 "기술"로 오해

### YouTube 데이터 발견
| Segment | YouTube 심리 | 클로징율 | 핵심 이미지 |
|---------|-----------|--------|---------|
| **효도 (D)** | 부모 행복 = 자녀 행복 | **75-85%** | 부모님 따뜻한 얼굴 |
| **시간 (C)** | 창문/비행동 비용 | **65-75%** | 창문이 열렸을 때 |
| **꿈 (A)** | 희소성/자기투영 | **70%** | 평생 한 번의 특별함 |

### v13과의 완전 통합
```
✓ STAGE 0: 출처명확화 (신뢰 기반선)
✓ STAGE 1-2: 세그먼트 판별 (누가 이 고객?)
✓ STAGE 3: MI 욕망탐색 + YouTube 이미지 (Change Talk 수집)
✓ STAGE 4-5: 상품소개 (YouTube 연결)
✓ STAGE 7-8: 클로징 (고객 주도)
```
