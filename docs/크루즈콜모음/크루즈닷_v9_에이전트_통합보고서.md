| 유형 | 판별 신호 | 핵심 전략 |
|------|---------|---------|
| **즉결단형** | "얼마예요? 바로 말해봐요" | 즉시 가격+이중선택+여권 요청으로 빠른 이행 |
| **정보수집형** | "약관 볼 수 있나요?", 메모 소리 | 투명 정보 → FAQ 선제 → 카톡으로 서류 전송 |
| **감성형** | "꿈이었어요", "특별한 기억" | 5감각 장면 → 감정 피크 → 관계 앵커 |
| **의심형** | "사기 아니에요?", "주변에서 당했대요" | "확인하세요" 역설 신뢰 구축 → 검증 경로 안내 |
| **미루기형** | "생각해볼게요", "나중에요" | 미루기 이유 발굴 → 재연락 약속 + 카톡 확보 |
| **경제적저항형** | "비싸요", "형편이", "하루 단위로?" | 하루 환산 + 교환 가치 + 소프트 클로징 |
| **흥정형** | "더 싸게 해주면", "할인 없어요?" | 가치 정당화 → 포함 항목 구체화 → 이중 선택 |

### 추가 3: 3800 퍼널에서 추출한 크루즈닷 즉시 적용 기법 TOP 5

**기법 1: 당첨/선발 프레임 오프닝**
```
현재 v9: "크루즈닷 여행 적립 프로그램 보시고 연락처 남겨주셨잖아요"
업그레이드: "크루즈닷 골드 멤버십 우선 배정 대상으로 선정되셔서 연락드렸어요"
효과: 수신자 → 수혜자 정체성 전환. 방어심 즉시 해제.
```

**기법 2: 마이크로 약정 (첫 YES 연속 받기)**
```
추가할 문장:
"여행 좋아하시죠? 지금 건강하시죠? 그럼 자격 충분하세요."
원리: 일관성 원칙. 작은 YES 3개 → 큰 YES 확률 3배 상승.
```

**기법 3: 원가 먼저 → 할인 나중 (역설 앵커링)**
```
현재: "월 33,000원이에요"
업그레이드: "원래 정가가 [원가]인데, 이번 달에 한해서 특별히 33,000원이에요"
효과: 기준점 효과. 33,000원이 "비싸다"가 아닌 "할인됐다"로 인식.
```

**기법 4: 자유여행 함정 프레임**
```
추가할 섹션 (D/E 세그먼트 비교 상황):
"패키지나 자유여행으로 유럽 가시면 항공+호텔+식사+이동+보험 다 따로예요.
 저희는 그게 전부 포함이에요.
 그리고 크루즈가 더 싸게 나오는 경우가 많아요."
```

## ★ 줄리아 18단계 미완료 보완 (RAG 기반 추론)

줄리아 스크립트 파일이 단일 줄 구조로 Read 도구 처리 불가. 그러나 v8 RAG 마스터에 이미 반영된 내용 + 에이전트 분석 결과를 통해 18단계를 크루즈닷에 매핑:

| 줄리아 단계 | 내용 | 크루즈닷 v9 위치 |
|-----------|------|--------------|
| 1~3단계 | 라포 + 욕구 탐색 | 세그먼트 판별 질문 3개 |
| 4~6단계 | 니즈 확인/확대/증폭 | SPIN S→P→I→N |
| 7단계 | 문제인식 3문 체인 | "지금 안 가면 몇 년 뒤?" |
| 8단계 | Implication 6레이어 | 체력 창문 앵커 |
| 9단계 | 비전 제시 | 5감각 앵커링 |
| 10단계 | 자녀인정욕구 | B-앵커1 (아이 기억) |
| 11단계 | 증거 제시 | 신한은행+앱스토어 |
| 12단계 | 가격 앵커링 | 4종 앵커링 |
| 13단계 | 선거절 | 렌탈 전 가격 거절 3회 |
| 14단계 | 에이킨법 | "그러니까요" |
| 15단계 | 이중선택 | "스탠다드 vs 프리미엄" |
| 16단계 | 소프트 클로징 | "들어만 보세요" |
| 17단계 | 미디엄 클로징 | "이번 달 조건" + 5초 침묵 |

> 4. 카톡 이미지로 24시간 망각 방지
> 5. 2차 콜 SPIN으로 고객 자발적 욕구 표현
> 6. 가격 앵커링 4종 → "비싸지 않네요"
> 7. 이중선택 + 5초 침묵
> 8. 자발적 골인
