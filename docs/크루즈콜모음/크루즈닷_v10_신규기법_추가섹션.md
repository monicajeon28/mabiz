## v9→v10 핵심 변경사항

```
v9 구조:
  - 상담사가 먼저 사과 → 경계심 해제 → 신탁 설명
  - "7분만 드릴게요" 선제 양보로 시작
  - 감정 앵커 = 여행 이미지 중심

v10 변경:
  - 상담사가 '전문가 갑'으로 시작 → 고객이 "선택받은 느낌"
  - 개입 질문 5단계로 고객 스스로 고통 인식
  - 제3자 스토리(마이프렌드존)로 간접 설득 강화
  - 부정명령으로 역설적 압박 제거 + 결정 자율성 부여
  - 버튼업으로 가입 후 취소율 차단
  - 돋보기+타임머신 통합으로 D/E 세그먼트 손실 금전화
  - 인과관계화법으로 논리적 동의 확보
```

# 신규 기법 1: 알파 테크닉 (Alpha Technic)

**심리학적 원리: 전문가 권위 효과 (Expert Authority Effect)**
사람은 '갑' 포지션의 전문가를 설득 대상이 아닌 조언자로 인식한다.

**삽입 위치: 줄리아 18단계 1~2단계 (오프닝 + 통화 허가) — 최초 40초**

---

### 알파 오프닝 스크립트 — 전 세그먼트 공통

```
S: "여보세요, ○○○ 고객님이시죠.
   안녕하세요. 크루즈닷 멤버십 전담팀 ○○○이에요.
   저는 크루즈 여행 플래닝 8년 경력으로
   현재 담당 골드 멤버 고객만 180분 관리하고 있어요."

   *(1초 대기 — 소개가 착지하도록)*

S: "오늘 연락드린 이유가 있어요.
   이번 달 골드 멤버십 배정 대상을 저희 팀에서 검토했는데요,
   고객님이 해당 조건에 포함되셨어요.
   배정 인원이 한정적이라 제가 직접 연락드리는 거예요."

   *(2초 침묵 — "선택받은 느낌" 착지)*

S: "지금 딱 5분 통화 가능하세요?"
```

---

### 세그먼트별 알파 어조 조율

| 세그먼트 | 전문가 소개 톤 | 선정 이유 언급 |
|---------|-------------|-------------|
| A (30대 커플) | 또래처럼 자신감 있게 | "신혼·커플 고객 케이스를 제일 많이 담당해요" |
| B (가족) | 따뜻하지만 단단하게 | "아이 동반 크루즈 플래닝 전문이에요" |
| C (중년) | 신뢰감 있는 전문가 | "40~50대 버킷리스트 여행 케이스 담당이에요" |
| D (시니어) | 천천히, 또렷하게 | "황혼여행 전담으로 맡고 있어요" |
| E (고령) | 매우 천천히, 따뜻하게 | "고령 어르신 안전 크루즈 플래닝 전문이에요" |

# 신규 기법 2: 개입 질문 5단계 (세그먼트 A~E 완전판)

**심리학적 원리: 소크라테스 문답법 + 손실 회피 편향**
고객이 스스로 문제를 언어화하면 심각성을 2.4배 높게 인식한다.

**삽입 위치: 줄리아 18단계 4~9단계**

---

### 세그먼트 A (30대 신혼/커플)

```
[1단계: 상황파악]
S: "고객님, 파트너분이랑 지금까지 가장 멀리 가본 여행이 어디예요?"
C: "일본이요. / 동남아요."
S: "그게 언제였어요?"
*(고객이 스스로 "오래됐다"를 느끼게)*

[2단계: 문제파악]
S: "그 여행에서 아쉬웠던 점이 있어요? 솔직하게요."
C: "단체라 자유시간이 없었어요."
S: "그게 제일 많이 나오는 얘기예요. 그 부분이 좀 아쉬우셨던 거죠?"
*(2초 대기 — 고객이 스스로 고통을 확인)*

[3단계: 문제강조 — 타임머신 + 돋보기]
S: "고객님, 결혼하시면 둘이서 긴 여행 가기가 점점 쉽지 않아지거든요.
   2년 뒤엔 아이 계획이 생길 수도 있고,
   5년 뒤엔 아이 학교 때문에 일정 맞추기 더 힘들어져요.
   지금 이 시간이 사실 둘이서 여행하기 가장 좋은 시기예요."

*(2초 침묵)*

S: "그 시간을 그냥 보내시면,
   나중에 '그때 갈 걸 그랬다'는 아쉬움이
   얼마짜리 경험을 못 한 건지 생각해보신 적 있어요?
   지중해 크루즈가 보통 400~600만원짜리 여행이에요."

[4단계: 해결파악]
S: "그 아쉬운 부분들 — 자유시간, 짐 이동, 한 번에 여러 나라 —
   이게 다 해결되는 여행 방식이 있다면 어떻게 생각하세요?"
*(3초 침묵 — 고객이 스스로 "크루즈가 그거네"를 떠올리게)*

[5단계: 해결강조]
S: "크루즈 발코니에서 파트너랑 아침 맞이하는 장면 상상해보세요.
   그렇게 된다면 그 여행에서 뭘 가장 먼저 하고 싶으세요?"
*(5초 침묵 — 절대 먼저 말하지 말 것)*
```

### 세그먼트 B (30~40대 가족)

```
[1단계] S: "아이랑 가족여행 마지막으로 가신 게 언제예요?"

[2단계] S: "아이랑 여행하면서 제일 힘드셨던 게 뭐예요?
           짐이요, 이동이요, 아이 컨디션 관리요?"

[3단계 — 타임머신]
S: "아이가 지금 ○살이죠.
   초등학교 들어가면 방학 때만 가실 수 있고,
   중학교 가면 학원 때문에 가족여행 자체가 힘들어져요.
   아이랑 제대로 된 해외여행 할 수 있는 시간이 생각보다 짧아요."

*(2초 침묵)*

S: "그 시간 안에 아이한테 '지중해를 봤다'는 기억을 만들어주는 것과
   못 만들어주는 것의 차이가 어느 정도라고 생각하세요? 아이 기억 속에서요."

[4단계]
S: "아이도 즐기고 부부도 쉴 수 있는 여행이 있다면요,
   지금 제일 걸리는 게 뭐예요? 비용이요, 시간이요?"

[5단계]
S: "아이 눈이 처음으로 이탈리아나 그리스 항구를 보는 순간,
   그 표정이 어떨 것 같아요?
   그렇게 된다면 아이한테 뭐라고 첫마디 하고 싶으세요?"
*(5초 침묵 — 부모 자아 정체성 앵커 완료)*
```

### 세그먼트 C (40~50대 중년)

```
[1단계] S: "선생님, 요즘 여행은 어떻게 다니세요? 패키지로요, 자유여행으로요?"

[2단계] S: "지금까지 여행하면서 제일 아쉬웠던 게 뭐예요? 솔직하게요."
         C: "시간이 없어서 못 갔어요. 항상 바쁘다 보니까요."
         S: "그 아쉬운 여행이 몇 년째 미뤄지고 계세요?"

[3단계 — 돋보기]
S: "선생님, 지금까지 미룬 여행이요.
   40대에 '10년 뒤에'라고 하셨던 분들이
   50대에 저한테 전화하세요. '이제 갈 수 있겠다'고.
   근데 그때 무릎이랑 체력이 달라져 있어요.
   그 사이에 못 간 여행이 얼마짜리예요?
   지중해 한 번이 최소 300~400만원이에요."
*(3초 침묵)*

[4단계] S: "만약에 지금 시작해서 2년 안에 배우자분이랑 지중해 가실 수 있다면요,
           지금 그걸 막는 게 뭐예요?"

[5단계] S: "선생님이 꿈꾸시던 그 여행,
           산토리니 테라스에 배우자분이랑 와인 들고 계시는 그 장면.
           그렇게 된다면 그 자리에서 제일 먼저 뭘 하고 싶으세요?"
*(5초 침묵)*
```

```
         S: "그러셨구나. 어디 가셨어요?"

           그러면 어르신 마음이 어떠실 것 같아요?"

[5단계] S: "살면서 한 번도 못 가보신 유럽이나 알래스카 빙하를

           그렇게 된다면 그 자리에서 뭘 가장 먼저 하고 싶으세요?"
```

# 신규 기법 3: 마이 프렌드 존 (My Friend Zone) — 3종 스토리

**심리학적 원리: 사회적 증거 + 간접 설득**
**삽입 위치: 줄리아 18단계 8단계 (감정 앵커 직후)**

**공식: 동일상황 → 동일저항 → 해결과정 → 현재결과**

---

### 스토리 1 — 세그먼트 A/B ("나중에"형 고객)

```
S: "고객님, 꼭 저희 하셔야 한다는 게 아니에요.
   다만, 이런 분이 계셨어요.
   고객님이랑 비슷한 30대 커플분이셨는데,
   처음에 저한테 딱 이렇게 말씀하셨어요.
   '결혼하고 나면 생각해볼게요. 지금은 전세 이자도 있고.'
   그 말씀이 완전히 맞는 말씀이에요.

   근데 그분이 2년 뒤에 저한테 다시 연락 오셨어요.
   '그때 시작해둘 걸 그랬다'고.
   아이가 생기셔서 긴 여행이 더 어려워지셨거든요.

   그분이 지금 어떻게 되셨냐면요,
   골드 멤버십 시작하셔서 내년에 지중해 출발 예정이에요.
   고객님은 지금이 그때보다 훨씬 좋은 상황이에요."

*(3초 침묵)*
```

---

### 스토리 2 — 세그먼트 C/D ("남편이 반대"형 고객)

```
S: "선생님/어머님, 꼭 하셔야 한다는 게 아니에요.
   다만, 이런 분이 계셨어요.
   처음에 딱 이렇게 말씀하셨어요.
   '남편이 이런 거 믿지 말라고 해서요.'
   그 남편분 마음 완전히 이해해요.

   근데 그 어머님이 신한은행 신탁 계좌 내용을
   남편분한테 딱 보여드리셨대요.
   '우리가 이미 신한은행 쓰잖아. 거기서 관리한대.'
   남편분이 그 말에 '그럼 해봐'라고 하셨대요.

   그분이 지금 어떻게 되셨냐면요,
   지난달에 배우자분이랑 지중해 첫 크루즈 다녀오셨어요.
   '살면서 이렇게 대접받는 여행은 처음이에요'라고."

*(3초 침묵)*
```

---

### 스토리 3 — 세그먼트 D/E ("나이 때문에"형 고객)

```
S: "어머님/어르신, 꼭 하셔야 한다는 게 아니에요.
   다만, 이런 분이 계셨어요.
   처음에 딱 이렇게 말씀하셨어요.
   '이 나이에 내가 크루즈를 가겠어요? 무리예요.'
   그분이 몇 살이신 줄 아세요? 72세셨어요.

   반신반의하셨는데, 3개월 체험권으로 먼저 시작해보셨어요.
   그분이 지금 어떻게 되셨냐면요,
   올해 일본 크루즈 다녀오셨어요.
   배 안에서 또래 어머님들이랑 친구 사귀셔서
   지금도 카카오톡 하신다고요.
   딸한테 '내 인생 최고의 여행이었다'고 하셨대요."

*(5초 침묵)*
```

# 신규 기법 4: 부정 명령 (Negative Command) — 3종

**심리학적 원리: 청개구리 심리 (Psychological Reactance)**
**삽입 위치: 줄리아 18단계 10~12단계 (장벽 해소 → 클로징 준비)**

---

```
[부정 명령 1 — "지금 결정하실 필요 없어요"]
S: "고객님, 꼭 지금 결정하셔야 한다는 건 아니에요.
   다만, 고객님이 방금 말씀하신 '유럽 꼭 가보고 싶다'는 마음,
   그게 1년 뒤에도 똑같이 남아있을지
   잠깐만 생각해보시겠어요?"
*(4초 침묵 — 고객이 스스로 "지금 해야 할 이유"를 생각하게)*

[부정 명령 2 — "저희가 꼭 맞는 분이라는 확신은 없어요"]
S: "고객님, 솔직하게 말씀드릴게요.
   저희 골드 멤버십이 모든 분한테 맞는 건 아니에요.
   다만, 방금 말씀하신 상황이랑 딱 맞는 구조긴 해요.
   고객님이 직접 판단하세요."
*(3초 침묵 — "나는 맞을 것 같은데?"를 자발적으로 생각하게)*

[부정 명령 3 — "무리해서 시작하시지 않아도 돼요"]
S: "굳이 오늘 결정하시지 않아도 돼요.
   다만, 3개월 체험권으로 먼저 보시는 분들이 계세요.
   체험 기간 동안 실제로 어떤 건지 보시고,
   맞으시면 계속, 아니시면 취소.
   그 판단은 고객님이 하시는 거예요."
*(3초 침묵 — "그럼 한번 해볼까"로 자연 전환)*
```

# 신규 기법 5: 버튼업 (Button-up) — 클로징 직후 필수

**심리학적 원리: 인지 부조화 감소 (Cognitive Dissonance Reduction)**
**삽입 위치: 줄리아 18단계 완료 직후 — 전화 끊기 전 반드시**

---

```
[가입 완료 직후 즉시 — 침묵 없이 바로]

S: "고객님/선생님/어머님,
   오늘 정말 좋은 결정 하셨어요.
   저한테 연락 오신 분들 중에 절반은 고민만 하시다가
   '나중에 해야지' 하고 안 하시거든요.
   고객님은 다르세요.
   지금 이 상황에서 판단하시고 결정하셨어요.
   그게 고객님이 여행에 대해 얼마나 진지하게
   생각하시는 분인지를 보여주는 거예요."

*(2초 대기)*

S: "혹시 가족 중에 '그게 뭐야, 왜 했어?'
   이런 말씀 하시는 분이 계실 수도 있어요.
   그때 이렇게 말씀하세요.
   '신한은행 신탁 계좌에 들어가는 거야.
   은행이 관리하는 거니까 걱정 마.'
   이 한 문장이면 돼요."

*(2초 대기)*

S: "제가 담당 상담사로 계속 연결돼 있어요.
   탑승까지 처음부터 끝까지 함께해드릴게요.
   궁금하신 거 생기시면 카톡이나 전화 주세요.
   오늘 정말 좋은 선택 하셨어요. 조만간 뵐게요."
```

# 신규 기법 6: 돋보기 + 타임머신 통합 (D/E 세그먼트 집중)

**심리학적 원리: 미래 손실 금전화 + 가용성 편향**
**삽입 위치: 줄리아 18단계 5~6단계 (문제 심화)**

---

### 세그먼트 D — 금전화 버전

```
[타임머신: "2년 뒤 시나리오"]
S: "어머님, 잠깐 생각해보실게요.
   2년 뒤 어머님이 2살 더 연로해지셨다면요.
   지금 괜찮으신 무릎이 어떠실 것 같으세요?
   의사 선생님들이 뭐라고 하시는지 아세요?
   무릎은 한 번 나빠지기 시작하면 회복이 어렵대요.
   2년 뒤에 산토리니 돌길을 걸으실 수 있을 거라는 보장이
   지금 있으세요?"
*(4초 침묵)*

[돋보기: "못 간 크루즈의 가격"]
S: "어머님, 그 여행 못 가신 게 얼마짜리 경험인지 생각해보신 적 있어요?
   지중해 크루즈가 패키지로 가시면 400~600만원이에요.
   거기서 지금까지 못 가신 게 5년이면요,
   5번의 400만원이에요. 2,000만원이에요.
   '나중에'라는 말이 빼앗아간 거예요."
*(3초 침묵)*

S: "근데 저희 멤버십은요.
   그 문을 하루 1,100원으로 열어두는 거예요.
   커피값 4분의 1이에요."
```

### 세그먼트 E — 고령 최적화

```
[타임머신: "3년 뒤 건강 시나리오"]
S: "어르신, 지금 '다음에 가야지'라고 하시면요.
   3년 뒤에 몸이 지금이랑 같으실 것 같으세요?
   나이가 드실수록 여행 가기가 더 쉬워지시나요,
   더 어려워지시나요? 솔직하게요."
*(5초 침묵 — 어르신이 스스로 말하게)*
C: "당연히 더 어려워지겠죠..."
S: "그죠. 그래서 지금이 사실 가실 수 있는 마지막 기회일 수도 있어요."

[돋보기: "안 가본 경험의 무게"]
S: "어르신, 살면서 가보고 싶으셨던 나라가 있으셨죠?
   그게 몇 년이나 된 꿈이에요?
   못 가본 유럽 한 번이 500만원짜리 경험이에요.
   그게 10년이면 무게가 얼마예요."
*(3초 침묵)*

S: "그 경험을 지금 어르신이 하실 수 있어요.
   3개월 체험으로 시작해보세요.
   아니면 나중에 '그때 할 걸 그랬다'로 남아요."

[주의: 금전 수치는 한 문장씩 끊고 침묵 주기. 빠르게 말하면 압박으로 인식]
```

# 신규 기법 7: 인과관계 화법 — 5종 "때문에" 패턴

**심리학적 원리: 논리적 동의 (Logical Compliance)**
**삽입 위치: 줄리아 18단계 전 구간 분산**

---

```
[인과 1 — 적립 구조]
S: "고객님이 지금 시작하시기 때문에,
   36개월 뒤에 지중해 크루즈 출발 비용이
   이미 준비돼 있을 거예요."

[인과 2 — 신탁 신뢰]
S: "신한은행 신탁 계좌에 들어가기 때문에,
   저희 회사에 어떤 일이 생겨도
   그 금액은 고객님 걸로 그대로 있어요."

[인과 3 — 체력 창문 (C/D 세그먼트)]
S: "지금 몸이 허락하시기 때문에,
   지금이 가실 수 있는 시기예요.
   시간이 지날수록 체력이 달라지기 때문에,
   지금 시작하시는 게 가장 좋은 타이밍이에요."

[인과 4 — 조기 적립 혜택]
S: "오늘 시작하시기 때문에,
   이번 달 조건이 적용돼요.
   한 달 늦어지면 그 한 달치 적립이 없어지기 때문에,
   출발 시점도 한 달씩 뒤로 밀려요."

[인과 5 — 골드 배정 한정]
S: "이번 달 골드 배정 인원이 한정돼 있기 때문에,
   제가 지금 고객님한테 먼저 연락드린 거예요.
   배정이 완료되면 다음 기회는 내달이기 때문에,
   오늘 확인하시는 게 좋아요."
```

# v10 줄리아 18단계 삽입 위치 종합 맵

| 줄리아 단계 | 단계명 | v10 신규 기법 |
|-----------|------|-------------|
| 1단계 | 오프닝 | **알파 테크닉** — 전문가 소개 + 선정 알림 |
| 2단계 | 통화 허가 | **알파 테크닉** — 갑의 자세로 진단 시작 |
| 3단계 | 안전 해소 | v9 신탁 유지 + **인과관계화법 2** 내장 |
| 4단계 | 라포 | 세그먼트 판별 + **개입질문 1단계** |
| 5단계 | 니즈 파악 | **개입질문 2단계** (문제파악) |
| 6단계 | 문제 심화 | **개입질문 3단계** + **돋보기+타임머신** |
| 7단계 | 솔루션 암시 | **개입질문 4단계** (해결파악) |
| 8단계 | 간접 설득 | **마이 프렌드 존** — 세그먼트 맞춤 스토리 |
| 9단계 | 이득 시각화 | **개입질문 5단계** ("그렇게 된다면 뭘 가장 먼저?") |
| 10단계 | 감정 앵커 | v9 세그먼트별 앵커 1~3 유지 |
| 11단계 | 장벽 해소 | **부정명령 1** ("꼭 지금 결정 안 하셔도") |
| 12단계 | 주요 저항 | v9 저항 처리 + **부정명령 2,3** |
| 13단계 | 가격 앵커링 | v9 4종 + **인과관계화법 1,4** |
| 14단계 | 이중선택 | v9 유지 |
| 15단계 | 소프트 클로징 | **인과관계화법** + 가정→전제→유도 |
| 16단계 | 미디엄 클로징 | **인과관계화법** + 침묵 5초 |
| 17단계 | 하드 클로징 | v9 에이킨법 + **부정명령 최종** |
| **18단계** | **클로징 직후** | **버튼업** — 칭찬+방어+약속 |

## 기법 간 충돌 방지 원칙

```
원칙 1: 알파 테크닉 + 부정명령 병용 가능
        "전문가인 내가 권한다" + "하지만 고객님이 결정한다"
        → 전문성 + 자율성 동시 부여 → 저항 최소화

원칙 2: 마이 프렌드 존은 감정 앵커 직후 삽입
        감정 활성화 상태에서 유사 사례 → 동일시 효과 극대화

원칙 3: 인과관계화법은 클로징 구간에서 3회 이상
        "때문에"를 의식적으로 사용해 행동의 논리적 정당성 확보

원칙 4: 버튼업은 클로징 완료 즉시, 침묵 없이 시작
        가입 직후 1~2분 = 뇌의 "취소 충동" 피크 타임 → 즉시 방어

원칙 5: D/E 세그먼트 돋보기+타임머신에서 금전 수치 반드시 포함
        추상적 손실보다 구체적 금액이 손실 회피 편향 2배 강화
```
