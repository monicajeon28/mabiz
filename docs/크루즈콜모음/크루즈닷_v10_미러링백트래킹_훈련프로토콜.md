## 1. 미러링 (Mirroring) 완전 가이드

### 1-1. 미러링이란?
고객의 말투, 속도, 단어, 감정 톤을 자연스럽게 따라하는 기법.
의식적으로 하되, **들키지 않을 정도**로 섬세하게.

### 1-2. 미러링 3가지 레벨

**레벨 1: 언어 미러링 (가장 쉬움)**
```
고객이 쓴 단어를 그대로 사용하기

고객: "크루즈가 재미있을 것 같긴 한데..."
❌ 상담사: "흥미로우시겠죠"
✅ 상담사: "재미있을 것 같으시죠?"

고객: "남편이 좀 까다로운 편이라서요"
❌ 상담사: "배우자분이 신중하신 거죠"
✅ 상담사: "남편분이 까다로우신 편이세요?"
```

**레벨 2: 속도/리듬 미러링 (중요)**
```
고객이 빠르게 말하면 → 조금 빠르게
고객이 느리게 말하면 → 조금 느리게
고객이 짧게 답하면 → 짧게 물어보기
고객이 길게 설명하면 → 길게 반응하기

예시:
고객 (빠르게): "어, 그러니까 지금 일본이 좋다는 거죠?"
상담사 (빠르게): "맞아요, 일본 코스가 제일 인기예요!"

고객 (느리게, 신중하게): "음... 그게... 잘 모르겠는데요..."
상담사 (느리게): "충분히... 이해해요. 처음엔 그렇게 느끼시죠..."
```

**레벨 3: 감정 톤 미러링 (최고 효과)**
```
고객의 감정 에너지를 맞추기

고객 (신나서): "일본 가보고 싶었어요! 맛집도 많고!"
상담사: "맞아요! 일본 크루즈는 그 맛집들이 기항지마다 나와요!"

고객 (조심스럽게): "음... 뭔가 좀 걱정이 되긴 해서요..."
상담사: "그렇죠, 처음이시면 당연히 그런 마음이 드실 수 있어요..."
```

### 1-3. 크루즈닷 실전 미러링 훈련

**훈련 1: 단어 미러링 반사 훈련 (즉흥)**

트레이너가 아래 문장을 말하면, 트레이니는 핵심 단어를 그대로 반사하는 질문을 즉흥으로 만든다.

```
트레이너: "여행이 피곤한 게 싫어요."
트레이니: "피곤한 게 싫으시구나. 보통 어떤 부분이 제일 피곤하세요?"

트레이너: "가족이 다 같이 즐길 수 있어야 해요."
트레이니: "다 같이 즐길 수 있어야 한다고 하셨는데, 보통 가족이 몇 명이에요?"

트레이너: "솔직히 확신이 없어요."
트레이니: "확신이 없으시군요. 어떤 부분에서 확신이 안 서세요?"

트레이너: "남편이 여행을 별로 좋아하지 않아서요."
트레이니: "남편분이 여행을 별로 안 좋아하시는군요. 이유가 있어요?"
```

**훈련 2: 속도 미러링 시뮬레이션**

트레이너가 3가지 속도로 말한다. 트레이니는 각각에 맞춰 속도를 조절한다.

```
[빠른 속도] 트레이너: "그래서 일본이 얼마나 걸려요? 부산 출발이에요?"
트레이니 목표: 같은 빠른 속도로 "네! 부산 출발이고요, 일본까지 하룻밤이에요!"

[중간 속도] 트레이너: "그렇군요. 제가 크루즈를 한 번도 안 가봐서요."
트레이니 목표: "처음이시군요. 그럼 제가 처음이신 분들 기준으로 설명드릴게요."

[느린 속도] 트레이너: "음... 잘... 모르겠는데요... 생각해봐야 할 것 같아서요..."
트레이니 목표: 천천히... "맞아요... 충분히 생각하셔야 해요... 어떤 부분이 제일 마음에 걸리세요?"
```

## 2. 백트래킹 (Backtracking) 완전 가이드

### 2-1. 백트래킹이란?
고객이 한 말의 핵심 단어나 감정을 그대로 되받아치기.
"그러셨군요" + [고객의 핵심 표현]으로 공감하고 신뢰를 쌓는다.

### 2-2. 백트래킹 3가지 유형

**유형 1: 단순 반복 (기본)**
```
고객: "아이들이랑 가보고 싶었어요."
상담사: "아이들이랑요!"
(같은 단어를 감탄으로 반복 → 공감 표현)
```

**유형 2: 재구성 반복 (심화)**
```
고객: "여행 준비가 귀찮아서 자꾸 미루게 되더라고요."
상담사: "아, 준비 과정이 번거로우셔서 계속 미루고 계셨군요."
(같은 내용을 상담사의 언어로 정리해서 반복)
```

**유형 3: 감정 백트래킹 (최고 효과)**
```
고객: "솔직히 한번쯤은 제대로 된 여행 가보고 싶은데..."
상담사: "제대로 된 여행... 그게 어떤 건지 말씀해주실 수 있어요?"
(감정/욕구의 핵심어를 반복 + 확장 질문)
```

### 2-3. 크루즈닷 실전 백트래킹 훈련

**훈련: 즉각 반응 훈련**

트레이너가 말하면, 트레이니는 2초 이내에 백트래킹 반응한다.

```
고객: "사실 남편이 여행을 싫어해서요."
✅ "남편분이 여행을 싫어하세요..."
❌ "아, 그러시군요. 근데 크루즈는..."

고객: "일본은 한번 가보고 싶었어요."
✅ "일본을 한번 가보고 싶으셨군요! 일본 어느 지역이에요?"
❌ "저희 일본 코스가 있어요."

고객: "애들이 가보고 싶다고 해서요."
✅ "아이들이 가고 싶다고 했군요. 지금 몇 살이에요?"
❌ "그렇군요. 저희 가족 상품이..."

고객: "체력이 예전 같지 않아서요."
✅ "체력이 예전 같지 않으시다고 느끼시는군요... 어떤 부분에서요?"
❌ "크루즈는 체력이 별로 안 필요해요."
```

## 3. 미러링 + 백트래킹 통합 실전 (완전 시나리오)

### 시나리오: 60대 여성 고객과의 첫 2분

**트레이너 지시**: 60대 여성으로, 목소리 느리고, 조심스럽게, 가끔 "그러니까요"라는 말버릇 사용

```
T: "여보세요?" (느리게, 조심스럽게)

S: "안녕하세요, 이순자님 맞으시죠?" (미러링: 같은 속도로)

T: "네, 그런데요..."

S: "저는 크루즈닷 시니어 매니저 [이름]인데요,
   어머님 배정 대상으로 선정돼서 연락드렸어요."
   (1초 침묵)

T: "아, 그게 뭔지 모르겠는데..." (느리게, 불확실)

S: "모르셔도 되요. 제가 3분만 설명드릴게요.
   지금 통화 괜찮으세요?" (미러링: 느리게, 부드럽게)

T: "그러니까요... 뭔지나 들어봐야죠."

S: "맞아요, 들어보셔야 알죠.
   어머님, 여행 좋아하세요?" (백트래킹: "들어봐야죠" → "들어보셔야 알죠")

T: "좋아하긴 하는데... 요즘은 잘 안 가게 되더라고요."

S: "요즘 잘 안 가게 되신군요..." (백트래킹: 핵심 반복)
   "이유가 있으세요?" (확장 질문)

T: "그러니까요... 나이도 있고, 남편이 여행을 별로..."

S: "남편분이 여행을 별로 안 좋아하시는군요." (백트래킹: "남편이 여행을 별로" 반복)
   "그럼 어머님 혼자 또는 친구분들이랑 가시고 싶으셨겠네요?"

T: "맞아요. 친구들이랑 한번 가보고 싶은데."

S: "친구분들이랑요!" (백트래킹: 감탄으로 반복)
   "지금 친구분들도 여행에 관심 있으세요?"
```

**분석:**
- S가 미러링으로 속도, 톤을 맞췄다 → 고객이 편안함 느낌
- 백트래킹으로 "요즘 잘 안 가시는군요", "남편이" 반복 → 고객 "맞아요" 확인
- 2분 이내에 친구 동행 정보 확보

## 4. 부정 감정 백트래킹 (고급 기법)

### 고객이 부정적일 때 사용법

```
고객: "사기 같아서요."
❌ 즉각 방어: "사기 아니에요! 신한은행이랑..."
✅ 백트래킹 먼저: "사기 같다고 느끼시는군요. 어떤 부분에서 그런 느낌이 드세요?"

→ 고객이 구체적인 우려를 말하면 → 그 부분만 정확하게 해결
```

```
고객: "시간 낭비인 것 같아서요."
❌ 즉각 설득: "5분만 들어보세요!"
✅ 백트래킹: "시간 낭비라고 느끼시는군요. 그럼 제가 딱 하나만 여쭤볼게요.
   해외여행 한번 가보고 싶으신 곳 있으세요?"
```

## 5. 미러링&백트래킹 금지 사항

```
❌ 과도한 미러링: 모든 단어를 앵무새처럼 반복 → 이상하다 느낌
   고객: "비싸요."
   상담사: "비싸다고요. 비싸다고 하셨군요." → X

✅ 자연스러운 1회: "비싸다고 느끼시는군요..." → O

❌ 감정 불일치: 고객이 슬픈데 상담사가 신남
   고객: "남편이 작년에 돌아가셔서..."
   상담사: (신나게) "그렇군요! 그럼 혼자 여행하기 더 좋으시겠어요!"  → X

✅ 톤 맞추기: 부드럽게, 낮게 "많이 힘드셨겠어요... 이제 어머님 스스로를 위한 시간이 필요하시겠네요." → O

❌ 백트래킹 후 바로 상품 이야기: 신뢰 쌓이기 전에 상품 들이밀기
   고객: "여행 좋아해요."
   상담사: "여행 좋아하시는군요! 저희 크루즈 상품이..." → X

✅ 한 단계 더 질문: "여행 좋아하시는군요! 주로 어디 많이 다니셨어요?" → O
```

## 6. 훈련 체크리스트

### 일일 훈련 10분 루틴

```
Day 1-3: 언어 미러링 (단어 반복)
  → 트레이너 문장 10개 → 즉흥 반응

Day 4-6: 속도 미러링
  → 빠름/중간/느림 × 5회 반복

Day 7-9: 감정 미러링
  → 신남/조심/슬픔 × 5회 반복

Day 10-12: 백트래킹 3유형
  → 단순/재구성/감정 × 10회

Day 13-15: 통합 시나리오
  → 완전 2분 롤플레이 × 3회
```

### 합격 기준 체크리스트

| 항목 | 기준 | 통과 |
|------|------|------|
| 언어 미러링 | 고객 단어 3개 이상 자연스럽게 반복 | □ |
| 속도 미러링 | 고객 속도 맞추기 (의식 없이) | □ |
| 감정 미러링 | 고객 감정 에너지 일치 | □ |
| 단순 백트래킹 | 핵심 단어 반복 + 질문 | □ |
| 감정 백트래킹 | 욕구/감정 반복 + 확장 질문 | □ |
| 부정 감정 처리 | 방어 전 백트래킹 먼저 | □ |
| 자연스러움 | 들키지 않을 정도로 | □ |

**7개 모두 통과 = 미러링&백트래킹 숙달 완료**

## 7. 실전 투입 전 최종 셀프 체크

```
통화 후 녹음을 들으면서 확인:

□ 내가 말하는 속도가 고객보다 빨랐는가?
  (빠르면: 을의 포지션 신호. 속도 줄이기)

□ 고객의 핵심 단어를 한 번도 반복하지 않았는가?
  (없으면: 백트래킹 부재. 다음 콜에서 최소 3회)

□ 고객이 "맞아요", "그렇죠", "네" 했는가?
  (없으면: 미러링 & 백트래킹 부족)

□ 내가 고객보다 더 많이 말했는가?
  (그렇다면: 개입질문 부족. 고객이 6:4로 더 말해야)
```
