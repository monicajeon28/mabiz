### 핵심 원인 1: 클로징 신호 3회 포착 실패 (최중요)

고객이 수용 신호를 콜 중 **3회 연속** 발화했으나 매니저가 모두 놓쳤다.

```
수용 신호 1: "아 그것도 괜찮고요" (요금제 수락 직후)
    → 매니저 반응: "혹시 연령층이 어떻게 되시는지..." (나이 확인으로 이탈)

수용 신호 2: "네 그 부분을 원하시는 것에요" (쪼인+납입 구조 동의)
    → 매니저 반응: "혹시 최근에 은퇴를 하시는 부분이..." (직업 확인으로 이탈)

수용 신호 3: "고맙습니다" (안마기 선물 수락)
    → 매니저 반응: "여기서 끝나는 건 아니고요... 소개해주실 분..." (소개 권유로 이탈)
```

**수용 신호 1 직후 올바른 대응**:
```
[상담사] "좋으세요! 그러면 지금 바로 진행해드릴게요.
          체크카드 가지고 계세요? 2분이면 바로 등록 가능해요."
```

> **핵심 교훈**: 고객이 "네", "괜찮고요", "원해요"를 말하는 순간 = 클로징 신호.
> 이 순간 다른 설명 추가 금지. 즉시 결제 단계 진입.
