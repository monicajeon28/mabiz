## 원콜 핵심 원인 4가지

### 핵심 원인 1: 황금 신호 "홍콩 너무 좋아하죠" 무시 (최중요)

```
[실제 발화]
고객: "저 홍콩 같은 거 너무 좋아하죠."

[상담사 반응]
"네네. 네 좋아하죠."
→ 그냥 넘어감
→ 욕망 확인 신호를 그대로 묻어버림
```

이 발화는 **모니카 1단계 욕망 극대화 진입 트리거**였다.

```
[올바른 대응 — 욕망 극대화 진입]

"홍콩 좋아하세요! 어디 가보셨어요?
 홍콩 어디가 제일 좋으셨어요?"

⏸ (고객이 스스로 경험 말하게)

고객이 경험 얘기 시작하면:
"맞아요! 홍콩이 그래서 계속 가게 되는 거잖아요.
 선생님, 홍콩을 다시 간다면 — 다음엔 어디 가고 싶으세요?
 또 비슷한 데, 아니면 마카오나 베트남 이어서?"

→ 욕망을 20도에서 80도까지 올린 다음 상품 얘기
```

**핵심 교훈**: "좋아한다"는 발화 = 욕망 극대화 초대장. 무시하면 이탈.

---

### 핵심 원인 2: 욕망 단계 0회 — 상품 스펙 직접 진입

```
[실패 순서]
① "3만 3천원, 260회" (납입 구조 선제 노출)
② "유류세 오르고 항만세 추가금 붙을 수 있어요" (리스크 먼저)
③ "B플랜은 동남아 크루즈" (상품 설명)

→ 욕망 극대화 단계 0회
→ 고객은 숫자 계산 모드 진입
→ "260회면 총 얼마지?" → 비용 비교 프레임 형성
```

**모니카 법칙 위반**: 욕망 부풀리기 완료 전 상품/납입 얘기 금지

```
[올바른 순서]
1. 욕망 극대화: "홍콩 좋아하세요? 어디 가보셨어요?"
2. 경험 공감: "맞아요, 그래서 홍콩이 중독이잖아요"
3. 꿈 확장: "다음엔 더 길게, 더 여유롭게 가고 싶으신 적 있으세요?"
4. 현실 장벽: "근데 막상 일정 잡으려면 뭐가 걸리세요?"
5. 막연함: "이대로면 언제쯤 가실 수 있을 것 같아요?"
6. 해결책: "그래서 저희 방법이 있어요" (여기서 상품 얘기)
```

### 핵심 원인 3: 항만세/유류세 리스크 먼저 언급 — 구매 심리 하락

```
[실패 발화]
"전쟁이 터질 줄 몰랐잖아요. 유류세도 엄청 오르고.
 항만세라고 있거든요 — 추가금이 붙을 수 있거든요."

→ 이것은 상품 소구점이 아님
→ 오히려 구매 저항 심리를 먼저 심어줌
→ "그러면 최종 가격이 더 올라가는 거잖아요" → 비교 심리 강화
```

**원칙**: 리스크/단서/추가금 언급은 클로징 직전 Safety Net 단계에서만.
오프닝~SPIN 구간에서 리스크 언급 = 자기 발등 찍기.

```
[올바른 접근]
추가금 언급은 클로징 직전:
"한 가지 미리 말씀드릴게요.
 항만세는 여행 출발 시점에 정산되는데
 보통 3~5만원 수준이에요.
 저희 회원분들은 이미 그거 포함해도 비회원 대비
 훨씬 저렴하게 다녀오세요."
```

### 핵심 원인 4: 패키지 단가 노출 → 하나투어 비교 모드 유도

```
[실패 발화]
고객: "홍콩 몇 박 며칠에 얼마예요?"
상담사: "5박 6일 기준으로 1인당 189만 원부터 시작하거든요"

→ 고객이 즉시 "은행에서 하나투어 모두투어가 조인돼 있잖아요"
→ 패키지 단가 비교 프레임으로 완전 전환
→ 이미 비교 채널 갖고 있음 → 크루즈닷 경쟁력 0
```

**문제**: 멤버십 상품을 패키지 여행 상품으로 설명하면 하나투어와 비교 대상이 됨.
크루즈닷 골드멤버십은 **월 납입 × 여행 접근권** 구조 — 1회 패키지 단가 비교 불가.

```
[올바른 대응]

고객: "홍콩 몇 박 며칠에 얼마예요?"

→ "선생님, 그 질문에 답드리기 전에 한 가지만요.
   하나투어로 홍콩 5박 가시면 얼마 드려요?
   지금 기준으로."

고객: "100~150만원 정도?"

→ "맞아요. 저희 골드 회원이면 그 여행이 50만원 안쪽이에요.
   1인당이요.
   어떻게 그게 가능하냐면 — 잠깐 말씀드려도 될까요?"

→ 비교 기준을 크루즈닷이 먼저 선점 (ROI 선점 원칙)
```

## 모니카 4단계 적용 시 이 콜이 어떻게 됐어야 하는가

```
[실제 발화 포인트: "저 홍콩 같은 거 너무 좋아하죠"]
→ 이 순간이 욕망 극대화 진입 트리거

[1단계: 욕망 극대화]
"홍콩 좋아하세요! 어디 가보셨어요?
 홍콩 어디가 제일 좋으셨어요?"

고객: "야경이 진짜 좋았어요 / 음식도 좋고요"

"맞아요. 홍콩 야경은 진짜 계속 생각나잖아요.
 선생님, 다시 간다면 — 이번엔 더 여유롭게 가고 싶으시죠?
 3박 4일 빡빡하게 아니라, 5박 6일 천천히."

고객: "그렇죠, 항상 짧게 가서 아쉬웠어요"

[2단계: 현실 장벽]
"맞아요. 5박 6일 홍콩 — 지금 바로 가시면 뭐가 걸려요?
 일정이에요, 비용이에요?"

고객: "비용이 좀 부담이죠"

[3단계: 막연함 심화]
"비용이 걸리시면 — 그럼 언제쯤 가실 수 있을 것 같으세요?
 올해 안에 되겠어요?"

고객: "글쎄요..."

⏸ (침묵 — 고객이 스스로 느끼게)

[4단계: 해결책 제시]
"선생님, 그래서 저희 방법이 있어요.
 하나투어로 홍콩 5박 가시면 1인 100~150만원이잖아요.
 저희 회원이면 그 여행이 50만원 안쪽으로 내려와요.
 그게 어떻게 가능한지 — 1분만 들어보시겠어요?"
```

**결과 예측**: 이 순서로 진행했다면 클로징 확률 50%+ (HOT DB × 홍콩 욕망 확인 × 비용 장벽 → 해결책 수용)

## 이 고객이 왜 클로징 가능했는가

```
클로징 5대 조건:
✅ 홍콩 여행 강한 욕망 ("너무 좋아하죠")
✅ 비용 이슈 있음 → 해결책 수용 가능
✅ 인바운드 신청 → 크루즈 관심 확인
✅ 8시 전 통화 가능 → 시간 여유 없으나 전화 받음
✅ 비교 채널(하나투어) 있음 → ROI 선점 시 전환 가능

→ 욕망 극대화 + ROI 선점 + 클로징 시도만 있었으면 성공 확률 높은 고객
→ 이 고객을 "비교 후 연락"으로 보낸 것 = 클로징 포기 선언
```

## 학습 포인트 (v27.2 반영)

```
1. 황금 신호 = 즉시 욕망 극대화 진입 트리거
   "홍콩 좋아하죠" → "어디 가보셨어요? 제일 좋았던 게 뭐예요?"

2. 납입 구조(260회)는 N 질문 이후까지 절대 언급 금지

3. 리스크(추가금/유류세)는 Safety Net 단계에서만
   오프닝~SPIN 구간에서 리스크 = 자기 발등 찍기

4. 패키지 단가 먼저 노출 금지 → 먼저 비교 기준 선점
   "하나투어로 가시면 얼마예요?" → ROI 역산

5. "비교하고" 발화 = 이탈 신호가 아닌 클로징 개입 신호
   → "제가 지금 비교해드릴게요" 즉시 차단

6. 시간 제약 고객(8시 전): 더욱 짧고 강하게
   → 오프닝 30초 이내 욕망 1개 확인 → 즉시 플랜 1개
```

> **종합 판정**: 이 고객은 홍콩 여행 욕망이 확인된 HOT 신호 고객이었다.
> 욕망 극대화 단계 없이 납입 구조+항만세+패키지 단가를 먼저 제시하면서
> 고객이 하나투어 비교 프레임으로 진입했고 이후 클로징 불가 상태가 됐다.
>
