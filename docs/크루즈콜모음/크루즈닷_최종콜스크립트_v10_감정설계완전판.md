# 크루즈닷 최종 콜 스크립트 v10 — 감정 설계 완전판
## 한영인 영업론 + 영업의 정석 + 인셀덤 감정여정 완전 통합
## 4개 에이전트 병렬 학습 결과 최종 통합 | 2026-03-15

---

> **v10 핵심 철학:**
> "상담사는 파는 것이 아니라, 고객의 감정이 자연스럽게 구매로 흘러가는 레일을 깔아놓는 건축가다."
> — 설명하지 않고 질문하여, 고객이 스스로 "어떻게 하면 돼요?"를 먼저 묻게 만들어라.

---

## v9 → v10 핵심 변화

| 관점 | v9 | v10 |
|------|----|----|
| 포지셔닝 | 을의 자리 ("실례가 많겠지만") | **갑의 자리** (알파 테크닉) |
| 질문 방식 | SPIN 4단계 | **개입 질문 5단계** 공식화 |
| 설득 방식 | 상담사 직접 설득 | **마이프렌드존** (제3자 스토리) |
| 압박 방식 | 희소성 직접 강조 | **부정명령** (청개구리 심리) |
| 클로징 후 | 없음 | **버튼업** (취소 방지) |
| 기대감 | 없음 | **플라시보** 탑승 전 설계 |
| 전략 방향 | 수비형 (저항 처리) | **공격형** (자발적 구매 유도) |

**예상 클로징률**: v9 20~25% → v10 **37~49%**

## 8단계 감정 여정 설계도

```
[1단계: 자존감]   선정 프레임         → "나는 선택받은 사람이다"
      ↓
[2단계: 기대]     크루즈 특별함 인식   → "뭔가 다를 것 같다"
      ↓
[3단계: 고통공감] Pain Point 확대      → "맞아, 나 지금 그게 없어"
      ↓
[4단계: 욕망]     5감각 시각화         → "내가 거기 있는 것 같다"
      ↓
[5단계: 신뢰]     전문가 케어          → "이 사람이 나를 챙겨준다"
      ↓
[6단계: 소속감]   골드 커뮤니티        → "나도 그 사람들 중 하나"
      ↓
[7단계: commitment] 카톡 교환         → "이미 간 것이나 다름없다"
      ↓
[8단계: 자발적 구매] "어떻게 하면 돼요?"  → 클로징은 고객이 한다
```

각 단계 성공 신호:
```
1단계: "저요? 어떤 프로그램이에요?"
2단계: "오~ 그래요?" / "발코니 룸이 따로 있어요?"
3단계: 고객이 자신 이야기를 길게 꺼냄
4단계: "오~" "우와" 감탄사 / "거기 가보고 싶었는데"
5단계: "신한은행이요?" / "멀미 걱정이 있긴 했는데..."
6단계: "단톡방은 어떻게 들어가요?"
7단계: 카톡 번호 제공 / 다음 통화 시간 동의
8단계: "그러면 어떻게 신청하면 돼요?"
```

## ★ SECTION A: 알파 테크닉 오프닝 (신규 — v9 대체)

### 핵심 원칙: 갑의 자리에서 시작하라

```
[v9 — 을의 포지션]
S: "안녕하세요, 바쁘신데 실례가 많겠지만..."
→ 문제: 사과로 시작 = 을의 포지션 선언

[v10 — 갑의 포지션]
S: "안녕하세요, [고객명]님 되시나요?
   저는 크루즈닷에서 골드 멤버십 전담을 맡고 있는
   시니어 매니저 [이름]입니다.
   이번에 [고객명]님께서 골드 배정 우선 대상으로
   선정되셔서 제가 직접 연락드리게 됐어요."
```

### 알파 테크닉 전체 오프닝 스크립트 (공통)

```
S: "안녕하세요, [고객명]님 맞으시죠?
   저는 크루즈닷 시니어 여행 매니저 [이름]이고요,
   저는 현재 골드 회원 분들을 전담하고 있는데,
   이번 달에 [고객명]님을 우선 배정 대상으로
   선정해서 제가 직접 연락드리는 거예요."

*(1초 여유 — 갑의 침묵)*

C: [보통 "아, 네" 또는 "어, 저를요?" 반응]

S: "지금 통화 2~3분 괜찮으세요?
   [고객명]님 여행 스타일이 어떤지 먼저 여쭤보고,
   딱 맞는 혜택을 챙겨드리고 싶어서요."

C: "네, 괜찮아요."

S: "감사합니다. 제가 맞게 안내드리려면
   몇 가지 여쭤봐야 하는데 괜찮을까요?"
```

**알파 태도 매뉴얼 (상담사 훈련용)**
```
✅ 목소리 속도: 평소보다 1.3배 느리게. 서두르지 말 것.
✅ "갑자기 전화드려서 실례가" → 절대 금지. "연락드렸어요"로.
✅ 1초 침묵을 두려워하지 말 것. 갑은 침묵을 채우지 않는다.
✅ 고객이 말하는 동안 끊지 말 것. 끝까지 듣고 3초 후 답.
✅ 앵커 심은 후 5초는 반드시 기다릴 것.
```

### ★ 세그먼트 A — 30대 신혼/커플 (호칭: 고객님)

**개입 질문 5단계:**
```
[1단계 — 상황파악]
S: "요즘 여행은 어떻게 다니세요?
   배낭여행 스타일인지, 아니면 호텔 패키지인지요."

[2단계 — 문제파악]
S: "그러시면 비행기 예약하고, 호텔 찾고,
   현지에서 이동까지... 챙길 게 꽤 많으시겠네요?"

C: "맞아요, 그게 좀 피곤하더라고요."

[3단계 — 문제강조 (타임머신)]
S: "만약 이 패턴으로 5년이 더 지나신다면...
   바쁜 직장생활에 아이도 생기고 나면,
   둘만의 진짜 여행은 언제 가능하실 것 같으세요?"

*(3초 침묵)*

[4단계 — 해결파악]
S: "그렇다면, 짐 한 번만 풀고
   일본, 대만, 홍콩을 한 번에 다 가는 방법이 있다면
   한번 들어보실 의향이 있으세요?"

[5단계 — 해결강조 (이득 시각화)]
S: "만약 그게 가능하다면,
   파트너분이랑 어떤 여행 하고 싶으세요?
   가장 해보고 싶으신 게 뭐예요?"
```

**세그먼트 A — 5감각 앵커 (오프닝 후 적용)**

[5단계 — 해결강조]
S: "만약 그게 다 가능하다면,
   아이들한테 어떤 기억을 만들어주고 싶으세요?"

## ★ SECTION C: 마이프렌드존 (세그먼트별 — v9 대체)

## 콜 중 미러링 체크
```
□ 고객이 빠르게 말하면 → 나도 속도 올리기
□ 고객이 차분하게 말하면 → 나도 낮추기
□ 고객이 웃으면 → 같이 웃기
□ 고객이 한숨 쉬면 → "맞아요, 그렇죠..."
```

## 백트래킹 적용
```
C: "요즘 너무 바빠서 여행을 못 가고 있어요."
S: "아, 요즘 너무 바쁘셔서 여행을 못 가고 계시군요."
   (키워드 그대로 반복 → 공감 표현)
```

## 알파 태도 점검
```
□ 시선: 당당하게 (전화라면 목소리 톤)
□ 자세: 어깨 펴고 앉기 (앉는 자세가 목소리에 영향)
□ 침묵: 어색해도 채우지 말 것
□ 속도: 평소보다 1.2~1.5배 느리게
```
