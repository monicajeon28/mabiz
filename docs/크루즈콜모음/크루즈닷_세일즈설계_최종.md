# 크루즈닷 멤버십 세일즈 설계서 — 최종판
## 랜딩페이지 DB 신청자 → 콜 상담 → 클로징 완전 설계

> **핵심 원칙**: 고객은 이미 꽂혀있다. 다시 꽂을 필요 없다. 신뢰만 주면 된다.
> 상담사의 임무: 고객의 감정을 지키면서 논리를 보완하고 결제 경로를 열어주는 것.

---

## 1. 고객 프로파일 분석 (랜딩페이지 DB 신청자)

### 1-1. 신청자의 심리 상태 (콜 받기 전)
```
행동 신호: 랜딩페이지를 보고 자발적으로 DB를 남겼다
           → "이미 1번 Yes를 했다" = 가장 중요한 전제

감정 상태:
├── 설렘 (크루즈 여행에 대한 욕망이 촉발됨)
├── 궁금함 (실제로 얼마인지, 어떻게 되는 건지)
├── 경계심 (전화 세일즈에 대한 본능적 방어)
└── 불안 (상조 아닌지, 사기 아닌지, 가족이 반대하면)
```

### 1-2. 저항의 4대 축 (상담 전 미리 알고 있어야 하는 것들)
| 저항 유형 | 실제 심리 | 처리 원칙 |
|---------|---------|---------|
| "상조 아니에요?" | 낯선 판매 구조 불신 | "여행계" 프레임, 3000명 증거 |
| "비싸요" | 비교 대상이 없음 | 랜딩 앵커링, 대안 비교 |
| "생각해볼게요" | 결정 회피/가족 눈치 | 역질문으로 진짜 이유 발굴 |
| "가족과 상의해야 해요" | 실질적 장벽 | 함께 설계하는 방향 전환 |

온도    시간     상태             상담사 목표
─────────────────────────────────────────────
 0°C   0:00    경계심 MAX        이름/날씨 → 긴장 해제

78°C  24:00    "지금 해야겠어"   ★2차 클로징 피크
결제  28:00    "잘했다"         불안 제거 → 결제 완료
```

## 3. 6-Phase 구조 설계

### PHASE 1: 경계심 해제 (0~3분)
**목표**: "이 사람은 팔려는 사람이 아니다"는 인상 심기

**핵심 전략**:
- 자기 소개: 이름 + 소속 + "~도와드리려고 전화드렸어요" (판매 아님 강조)
- 날씨/안부: 1~2문장으로 인간적 연결
- DB 확인: "혹시 크루즈 여행에 관심 있으셔서 남겨주신 거 맞죠?" → 첫 번째 Yes 재확인
- 시간 확인: "지금 10분 정도 통화 괜찮으세요?" → 심리적 안도감

**절대 금지**:
- ❌ "오늘 특별히 연락드린 이유는..."
- ❌ 가격/상품 먼저 언급
- ❌ "좋은 기회가 있어서요"

---

### PHASE 2: 신뢰 구축 (3~7분)
**목표**: "구조가 안전하다" + "3000명이 하고 있다" 납득

**핵심 전략**:
- 신탁 설명 (세탁소 비유): "고객님 돈을 신한은행 신탁 계좌에 넣어두는 거예요. 우리가 못 쓰고요, 고객님이 여행을 가셔야만 그때 정산이 됩니다."
- 숫자 증거: "지금 3,000명 넘게 이용하고 계시고요, 앱스토어 4.8점이에요"
- 반환 구조: "여행 안 가시면 여행비는 100% 돌아오고요, 만기 되시면 원금도 100% 환급돼요"
- 소속감: "저희 회원분들 중에 고객님 또래 분들이 제일 많이 가시는데..."

**핵심 표현**:
- ✅ "신한은행 신탁 계좌에 보관돼요"
- ❌ "신한은행이 보장해요" (법적 틀림)

---

### PHASE 3: 여행 재점화 — SPIN 기법 (7~15분)
**목표**: 고객 스스로 욕망을 말하게 만들기 (상담사가 말하는 게 아님)

**SPIN 4단계**:

**S (Situation)**: 현황 파악
```
"요즘 여행은 얼마나 자주 가세요?"
"지금까지 해외여행은 어디어디 가보셨어요?"
"보통 여행 비용은 어떻게 준비하시는 편이에요?"
```

**P (Problem)**: 문제 발견 (고객이 말하게)
```
"마음은 가고 싶은데 실제로는 잘 못 가시는 편이세요?"
"어떤 부분이 제일 망설여지세요? 비용? 아니면 다른 이유가 있으세요?"
"예약하다 보면 이것저것 챙기는 게 번거로우시죠?"
```

**I (Implication)**: 문제의 파장 (고통 확대)
```
"그럼 올해도 결국 못 가실 것 같으세요?"
"나중에 가야지 하시다 보면 건강이나 체력 때문에 더 힘드실 수도 있잖아요"
"지금 못 가면 언제 가실 것 같으세요?"
```

**N (Need-Payoff)**: 해결책 확인 (자발적 욕망)
```
"그런데 만약에 매달 조금씩 모아두면서 언제든 떠날 수 있으면 어떨까요?"
"크루즈 한 번 가보시면 어떤 곳 가보고 싶으세요?"
"산토리니? 지중해? 아니면 가까운 일본 크루즈부터?"
```

**★ 산토리니 30초 영화 장면 (1차 감정 피크 클라이맥스)**:
```
상담사: "산토리니 말씀하셨잖아요..."
        [낮은 목소리로, 천천히]
        "파란 돔 성당 앞에서 석양 지는 거 보면서
         와인 한 잔 하시는 거... 상상이 가세요?
         옆에 남편분이랑 같이 계시는 거요."
        [3초 침묵]
        ← 절대 설명 더하지 말 것. 고객이 먼저 말할 때까지.
```

---

### PHASE 4: 가격 앵커링 — 역질문 3단계 (15~20분)
**목표**: 가격에 대한 의문을 우리 프레임으로 전환

**역질문 3단계**:
```
1단계 — 인정: "가격이 제일 궁금하시죠?"
         → 고객: "네, 얼마예요?"

2단계 — 유도질문: "따져보고 싶으신 거죠, 그쵸? (웃음)
                  제대로 해보죠. 비교해드릴게요."

3단계 — 가치스택 비교:
         "지중해 크루즈를 일반 패키지로 가시면 인당 350~400만원 내셔야 해요.
          저희 멤버십은 같은 여행을 회원가로 가시거든요.
          월 몇만원씩 모으시는 건데, 어떻게 보면 여행계 드신다고 보시면 돼요."
```

**앵커링 순서**:
```
① 시장가 앵커: "일반 여행사 지중해 크루즈 → 인당 350만원"
② 기회비용: "매년 못 가시면 10년에 한 번도 못 가시는 거잖아요"
③ 멤버십 가치: "월 OO만원으로 이게 다 가능해요"
④ 구체 계산: "1년에 OO만원인데, 지중해 한 번 값이에요"
```

**플랜별 가전 언급 (가치 증폭)**:
```
A 플랜: 더함TV + 쿠쿠청소기 + 안마베드
B 플랜: 삼성 에어드레서 + 55인치 QLED + 로보락
C 플랜: 비스포크냉장고(852L) + 더세리프55인치 + 에어드레서
       → "이게 멤버십 드리면서 사용하시는 거예요. 반납 없어요."
```

### PHASE 5: 클로징 — 3단계 + 선거절 (20~27분)
**목표**: Yes를 받는 것이 아니라 결정의 길목을 지키는 것

**선거절 (선제 이의처리) — 자주 나오는 TOP 5**:
```
선거절 1: "많이들 처음에 상조 같다고 하시더라고요.
           근데 상조랑 완전 달라요. 상조는 사망 시 쓰는 거고,
           저희는 살아계실 때 여행 가시는 거거든요."

선거절 2: "돈 넣고 못 찾으면 어쩌나 걱정하시는 분들 많으신데요,
           신한은행 신탁에 묶여있어서 저희가 손 못 대요."

선거절 3: "남편분이 반대하실 수도 있잖아요.
           근데 대부분 가고 나서는 정말 좋아하세요."
           [웃음] "같이 오셔도 되고, 따로 설명드려도 되는데요."

선거절 4: "지금 당장 결정 안 하셔도 돼요.
           근데 오늘 이 자리에서 여쭤보고 싶은 게 있어요.
           가고 싶은 마음이 있으세요? 없으세요?"

선거절 5: "생각해보고 싶으시면 어떤 부분을 더 생각해보고 싶으세요?"
           → 역질문으로 진짜 이유 발굴
```

**클로징 3단계**:

**소프트 클로즈**:
```
"제가 보기엔 고객님이 이 여행 정말 원하시는 것 같아요.
 오늘 혜택으로 시작하시겠어요? 아니면 한 번 더 자료 보시겠어요?"
→ 이중선택 구도 (Yes/No 제거)
```

**미디엄 클로즈**:
```
"솔직하게 여쭤볼게요.
 오늘 결정 못 하시는 진짜 이유가 뭐예요?
 돈이에요? 가족이에요? 아니면 다른 이유가 있으세요?"
→ 3초 침묵 → 답 기다리기
```

**하드 클로즈**:
```
"이렇게 해보시죠.
 지금 바로 결정 안 하셔도 돼요.
 대신 오늘 플랜만 선택해두시면, 첫 달 납입은 다음 달이에요.
 그 사이에 남편분이랑 이야기 나눠보시고.
 어떠세요? A 플랜이랑 B 플랜 중에 어디가 더 맞으세요?"
→ 이중선택 재적용
```

**클로징 직전 고통 계산기**:
```
"고객님, 솔직하게 여쭤볼게요.
 지금 안 하시면 어떻게 되는 거예요?
 내년에 이 여행 어떻게 가실 거예요?
 지금이랑 1년 후랑 달라지는 게 있을까요?"
[3초 침묵]
"매달 OO만원 모으셔서 5년 후에 산토리니 가실 수 있어요.
 아니면 지금 시작하시면 올해 안에도 가능하거든요."
```

### PHASE 6: 렌탈 보너스 — 결제 후에만 (27~30분)
**목표**: 결제 확정 후 추가 혜택으로 만족도 극대화

**절대 원칙**: Phase 6 이전에는 렌탈 절대 언급 금지 (자동 -8점)

**언급 순서**:
```
결제 확인 → "한 가지 더 알려드릴게요. 좋은 거예요." [웃음]
→ "저희 회원분들 혜택으로 가전 렌탈도 되거든요.
   A 플랜 선택하셨으면 더함TV 쓰실 수 있고,
   청소기랑 안마베드도 같이 나가요."
→ "따로 비용 내시는 게 아니고, 멤버십 안에 포함이에요."
→ "이건 고객님 집으로 바로 배송해드리거든요."
```

## 4. 저항 처리 완전 매뉴얼

### "생각해볼게요"
```
❌ 틀린 응대: "네, 생각해보세요~"
✅ 맞는 응대: "어떤 부분을 더 생각해보고 싶으세요?"
              → 답 들은 후 → 그 부분만 해결 → 재클로징
```

### "비싸요"
```
1단계 - 인정: "맞아요, 솔직히 저렴하진 않아요."
2단계 - 분리: "근데 비싸다는 게 어떤 기준이세요?
               한 번에 내시는 금액이 부담이세요?
               아니면 매달 나가는 게 부담이세요?"
3단계 - 재구성: "지중해 크루즈 한 번 가시면 400만원이에요.
                 저희는 그걸 월OO만원으로 나눠서 모으시는 거거든요."
```

### "남편이 반대해요"
```
1단계 - 공감: "그럴 수 있죠. 처음에 남편분들 거의 다 그러세요."
2단계 - 전환: "남편분이 반대하시는 게 구체적으로 어떤 부분이에요?
               돈 때문이에요? 아니면 '이런 거 왜 하냐'는 거예요?"
3단계 - 제안: "저희가 남편분께 직접 설명드리는 자료 보내드릴 수 있어요.
               아니면 남편분이랑 같이 한번 더 통화하셔도 되고요."
```

### "상조 아니에요?"
```
1단계 - FEEL: "저도 처음에 그런 이야기 들었어요."
2단계 - FELT: "다른 분들도 처음엔 다 그렇게 생각하셨는데요."
3단계 - FOUND: "근데 상조는 사망 시 서비스예요.
                저희는 살아계실 때 여행 가시는 거거든요."
4단계 - 증거: "지금 3,000명이 이용 중이고요, 앱스토어에서 확인하실 수 있어요."
```

### "렌탈이잖아요"
```
★ 이 저항이 나오면 설명 설명 절대 금지
1단계: "어떤 부분이 마음에 걸리세요?"
2단계: 들은 후 → 신탁 구조로 귀결
3단계: "가전은 멤버십 혜택이고요, 여행비는 신한은행에 따로 보관돼요."
```

## 5. 승낙 사다리 8단계 설계

```
Yes 1: "크루즈 여행에 관심 있으셔서 남겨주신 거 맞죠?" (DB 재확인)
Yes 2: "지금까지 해외여행 가보고 싶으셨던 곳 있으세요?" (욕망 확인)
Yes 3: "매달 조금씩 모아서 여행 가는 게 나쁘지 않죠?" (구조 동의)
Yes 4: "신한은행에 보관된다고 하니까 안심되시죠?" (신뢰 확인)
Yes 5: "산토리니 한번 가보고 싶으신 거 맞죠?" (욕망 구체화)
Yes 6: "지금 안 하시면 언제 가실 것 같으세요?" (긴박감)
Yes 7: "A랑 B 중에 어느 게 더 맞으세요?" (이중선택)
Yes 8: [결제] "정말 잘 결정하셨어요." (긍정 강화)
```

## 6. 침묵 규칙 5가지

```
Rule 1: "I 질문(Implication)" 직후 → 2초 침묵
Rule 2: 산토리니 클로징 훅 → 3초 침묵
Rule 3: 양자택일 이후 → 3초 침묵 (가장 중요)
Rule 4: 가격 말한 직후 → 2초 침묵 (설명 충동 참기)
Rule 5: 고객이 말 끊겼을 때 → 1초 더 기다리기
```

## 7. 자동 감점 항목 7가지

```
❌ -10점: "신한은행이 보장해요" (법적 허위 표현)
