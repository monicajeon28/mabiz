## Part 1: 세그먼트별 완전 콜 플로우

### Segment D — 5060 은퇴 (효도 75-85%)

#### [STAGE 0] 오프닝 — 출처명확화 (0-1분)

**상담사:**
```
"안녕하세요, [이영희]님 맞으시죠?
저는 크루즈닷 여행 전문 매니저 [경아]라고 해요.
[신한은행] VIP 고객분들께 특별 혜택 안내드리는데요,
지금 통화 2~3분 괜찮으실까요?"
```

**분석:**
- ✓ 이름 명확히 (권위 + 맞춤감)
- ✓ 전문가 표명 (신뢰 기반선)
- ✓ 출처 명시 (신한은행 = 안전감)
- ✓ 시간 제한 (심리적 부담 제거)

**예상 반응:**
- **"네, 괜찮아요."** → Stage 1로
- **"어디서 번호 알았어요?"** → 재확인 멘트 사용
- **"지금 바빠요"** → Stage 1 카드 A 사용

#### [STAGE 1] 선거절 카드 — 초반 저항 처리

**상황 1: "지금 바빠요"**

**상담사:**
```
"아, 바쁘신 분한테 길게 잡으려는 게 아니에요.
올해 혹은 내년에 여행 계획이 있으세요, 아직 없으세요?"
```

**분석:** 부담 제거 후 단순한 이진 선택 → 화제 전환

---

**상황 2: "관심 없어요"**

**상담사:**
```
"그렇군요. 솔직히 말씀드리면,
처음부터 관심 있다고 하시는 분이 거의 없어요. (가볍게 웃음)
근데 신기하게도, 10분 뒤에 끊으실 때
'이거 생각해봐야겠는데요' 하시는 분들이 많거든요.
딱 3분만 드릴게요. 별로면 끊으셔도 돼요."
```

**분석:**
- ✓ 고객 감정 공감 (역설 오프닝 = 아비테 기법)
- ✓ "3분" 구체적 약속 (저항 낮춤)
- ✓ 선택권 부여 (자율성 = 역설적으로 수용률 상승)

#### [STAGE 2] 세그먼트 판별 + 구매의향 확인

**상담사:**
```
"네, 감사합니다. 
[이영희]님, 한 가지만 여쭤봐도 될까요?
만약 제가 말씀드리는 내용이 '맞아, 이게 내가 원했던 거야' 싶으면,
이번 달 안에 진행해보실 의향이 있으세요?"
```

**고객이 YES/MAYBE/NO 중 선택:**
- **YES** → "좋아요. 그러면 [이영희]님께 딱 맞는 걸 찾아드릴게요."
- **MAYBE** → "일단 들어보시고 판단하세요. 그렇게 하시는 분들이 더 잘 결정하시더라고요."
- **NO** → "솔직하게 말씀해주셔서 감사해요. 그러면 언제쯤은 생각해보실 수 있을까요?"

**세그먼트 판별 질문:**

**상담사:**
```
"그런데 [이영희]님은 보통 누구랑 여행을 다니세요?"
```

**고객:** "저희는 부모님이랑... 근데 부모님이 계속 유럽을 가고 싶다고 하세요."

**상담사:**
```
"아, 그래요? 부모님께서 유럽을 꿈꾸고 계세요.
그런데 지금까진 못 가신 이유가 뭐였어요?"
```

**판정:** ✓ Segment D 확정 (5060 본인 또는 부모 동반)

#### [STAGE 3] MI 욕망 탐색 (DARN-CAT) + YouTube 심리학

**상담사:**
```
"아, 그렇군요. 혹시 [이영희]님이 부모님을 위해 
어떤 선물을 해주고 싶으신 적이 있으세요?
특별한 계기가 있었던 건가요?"
```

**고객:** "글쎄요... 막 엄마가 자꾸 '어디 한 번 가보고 싶다'고 하는데..."

**상담사 (DESIRE 발화 유도):**
```
"어머님이 가보고 싶다는 게 정말 큰 소원이시네요.
그럼 혹시 어머님이 언제쯤이 가장 행복해 보이세요?
어떤 상황일 때요?"
```

**고객:** "음... 아이들이 보낸 선물 받을 때?"

**상담사 (IMPLICATION 발화 유도 — ★ 핵심):**
```
"아, 그래요. 아이분들의 선물이 그렇게 의미 있으시구나.
그런데 [이영희]님, 이런 생각은 안 해보셨어요?

'내가 어머니께 평생 못 해준 것 중에,
마지막으로 이 한 가지라도 해줄 수 있다면?'

어머님이 그 생각을 하실 때,
어머님의 얼굴이 어떨 것 같으세요?"
```

**고객:** (침묵... 목소리가 작아짐) "... 정말 행복할 것 같아요."

**상담사 (NEED 발화 유도 — YouTube 이미지 완성):**
```
"그죠. 그런데 말이에요, [이영희]님.
지금이 그 시간이에요.

어머님의 체력이 좋으실 때,
[이영희]님도 함께할 수 있을 때.
함께 유럽의 그 아름다운 하늘을 보고,
산토리니의 하얀 건물과 파란 바다를 함께 볼 수 있는 시간.

그런데 이 창문이... 언제까지 열려있을까요?

만약 5년을 미룬다면?
10년을 미룬다면?
'아, 그때 갔어야 했는데' 하는 후회만 남을 것 같아요."
```

**분석:**
- ✓ 고객의 Change Talk ("행복할 것 같아요") 확보
- ✓ YouTube 심리학 (효도 + Implication) 통합
- ✓ 체력 창문 + 비행동의 비용 (손실회피) 활성화
- ✓ 고객이 "어머님 얼굴" 장면을 자신의 머릿속에 그림

#### [STAGE 4] 상품 소개 — 5감각 + YouTube 장면 연결

**상담사:**
```
"네, 그래서 제가 전화드린 거예요.
지금 [이영희]님과 어머님께 딱 맞는 일정이 있거든요.

산토리니, 아테네, 크레타... 
이렇게 3주간 지중해를 도는 일정이 있는데요.

첫 날 배에 승선하실 때만 해도,
'어라, 이게 비행기인가?' 할 정도로 커요.
그런데 배 위에 나가면...

(여기서 멈춤 — 고객이 상상하게)

저도 안 가봤지만, 유튜브 영상만 봐도
'아, 이래서 사람들이 자꾸 크루즈를 간다고 하는구나' 싶어요.

어머님이 갑판에 나가시는 순간,
그 탁 트인 하늘을 처음 보실 때...
[이영희]님께 이렇게 말씀하실 것 같아요:
'우리 딸(아들)이가 이런 경험을 주다니...'"
```

**분석:**
- ✓ 5감각 앵커 (크레타, 산토리니, 하늘)
- ✓ YouTube 장면 (갑판에서 처음 보는 하늘)
- ✓ 고객 입장에서의 부모 반응 재구성
- ✓ 감정선이 상향곡선

```
그건... 비용이 아니라,
가장 비싼 선물이자 동시에 가장 저렴한 선물이 되는 거예요.
```

## Part 3: 절대 금지 사항 + 체크리스트

### ❌ 절대 금지

```
1. STAGE 0 (출처명확화) 스킵하고 Opening A 사용
   "부모님이랑 아직도 제대로 여행 못 가셨어요?"
   → "뭐야?" 반응 = 경계심 해제 실패

2. 세그먼트 판별 전에 Opening A 적용
   → 30대 커플한테 "부모님" 호칭 = 부적합

3. STAGE 3 (욕망 탐색) 없이 Opening A 배치
   → 고객의 Change Talk가 없으므로 설득력 0

4. Opening A를 "기술"로 암기해서 말하기
   → 고객은 상담사의 "스크립트" 감지 = 신뢰도 하락

5. YouTube 이미지를 일괄 적용
   → Segment별로 다른 효도/FOMO/꿈 활용해야 함
```

### ✓ 체크리스트

```
단계별 확인 사항:

□ STAGE 0 (0-1분)
  □ 이름 명확히 했는가
  □ 출처(신한, 신한카드 등) 명시했는가
  □ 전문가 표명 했는가
  □ 시간 제한 (2~3분) 제시했는가
  → 성공 신호: 고객이 출처를 인정하고 수용

□ STAGE 1 (0-3분)
  □ 선거절 카드 (A/B/C) 중 맞는 것 선택했는가
  □ 부담을 제거했는가
  □ 재진입 포인트를 확보했는가
  → 성공 신호: 고객이 대화 계속

□ STAGE 2 (1-4분)
  □ 구매의향 확인 (YES/MAYBE/NO) 했는가
  □ 세그먼트 판별 3질문 했는가
    ① "보통 누구랑 여행 다니세요?"
    ② "최근 해외 다녀오신 게 언제예요?"
    ③ "여행에서 뭘 제일 중요하게 생각하세요?"
  → 성공 신호: Segment A/B/C/D/E 중 하나로 판정됨

□ STAGE 3 (4-12분) ← Opening A가 들어가는 구간
  □ DARN-CAT (Desire/Ability/Reason/Need) 순서로 했는가
  □ 오픈 질문 (개방형) 사용했는가
  □ 고객의 Change Talk 수집했는가
    - "좋겠다" / "부럽다" / "가보고 싶다" 등
  □ 고객이 말한 단어를 반영했는가
  □ YouTube 이미지를 Segment별로 적용했는가
  → 성공 신호: 고객이 자기 입으로 욕망을 말함
                 (상담사가 설득하지 않음)

□ STAGE 4-5 (12-18분)
  □ 5감각으로 상품을 설명했는가
  □ YouTube 장면을 고객의 Change Talk와 연결했는가
  □ 마이프렌드존 (실제 고객 사례)를 구체적으로 말했는가
  → 성공 신호: 고객이 "그런 경험이 되면 좋겠다" 말함

□ STAGE 7-8 (18-22분)
  □ 클로징 신호 감지했는가
  □ 희소성 (가격 인상 / 시간 제한) 제시했는가
  □ 침묵으로 고객의 자발적 "어떻게 신청하죠?" 대기했는가
  → 성공 신호: 고객이 먼저 신청 방법을 묻음
```

```

→ ✓ 신뢰 기반선 (이미 신청한 고객)

→ ✓ 세그먼트 탐색 시작

"어떤 크루즈를 타고 싶다라고 하면서 좀 알아보신 건 있으신가요?"
→ ✓ STAGE 3 시작 (욕망 탐색)
```

**개선점:**
- YouTube 이미지 (발코니에서 보는 밤하늘 + 신혼부부) 추가 필요
