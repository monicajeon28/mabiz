## 🔄 PII 접근 제어 흐름

```
┌─────────────────────────────────────────────────────────────┐
│  API 요청 (예: GET /api/admin/organizations)               │
│  - 사용자: AGENT                                            │
│  - 조직: org123                                             │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  1️⃣ RBAC 권한 확인 (getAuthContext)                        │
│  ✓ GLOBAL_ADMIN만 조직 조회 가능                           │
│  ✗ AGENT는 불가 → 403 Forbidden                           │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  2️⃣ 기존 조직 데이터 조회                                   │
│  SELECT * FROM organizations WHERE id = org123             │
│  → { id, name, email, phone, ... }                         │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  3️⃣ PII 필드 필터링 (PiiAccessControl)                      │
│                                                              │
│  accessibleFields = await piiAccessControl.filterFields(   │
│    'OWNER',  // 역할                                       │
│    ['email', 'phone', 'bankAccount'],  // 요청 필드       │
│    'read'    // 읽기                                       │
│  )                                                          │
│                                                              │
│  → OWNER는 ['email', 'phone'] 읽기 가능                    │
│  → 'bankAccount' 제외 (필요 권한 없음)                     │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  4️⃣ 응답 데이터 마스킹 (UI 출력)                            │
│                                                              │
│  displayData = {                                           │
│    id: 'org123',                                           │
│    name: 'Partner A',                                      │
│    email: 'h***@example.com',  // 마스킹                  │
│    phone: '010-****-5678',     // 마스킹                  │
│    bankAccount: '[접근 불가]'  // 거부                    │
│  }                                                         │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  5️⃣ 감시 로그 기록 (AuditLogger)                            │
│                                                              │
│  await auditLogger.record({                               │
│    organizationId: 'org123',                               │
│    userId: 'user456',                                      │
│    action: 'READ',                                         │
│    resourceType: 'Organization',                           │
│    piiFieldsAccessed: ['email', 'phone'],  // 접근한 필드  │
│    status: 'SUCCESS'                                       │
│  })                                                        │
│                                                              │
│  INSERT INTO "AuditLog" (...)                              │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  6️⃣ 클라이언트에 응답                                       │
│  { ok: true, data: displayData }                           │
└─────────────────────────────────────────────────────────────┘
```

## 🔐 역할별 권한 매트릭스

```
┌─────────────────────────────────────────────────────────────────────┐
│                         PII 필드 접근 권한                            │
├─────────────────────────────────────────────────────────────────────┤
│ 필드                │ GLOBAL  │ OWNER   │ AGENT   │ ANALYST │ READONLY
│                     │ ADMIN   │         │         │         │
├─────────────────────────────────────────────────────────────────────┤
│ phone               │ R/W     │ R/W     │ R/W     │ R       │ ✗
│ email               │ R/W     │ R/W     │ R/W     │ R       │ ✗
│ name                │ R/W     │ R/W     │ R/W     │ ✗       │ ✗
│ bankAccount         │ R       │ ✗       │ ✗       │ ✗       │ ✗
│ idNumber            │ R       │ ✗       │ ✗       │ ✗       │ ✗
│ passport            │ R       │ ✗       │ ✗       │ ✗       │ ✗
│ creditCard          │ R       │ ✗       │ ✗       │ ✗       │ ✗
├─────────────────────────────────────────────────────────────────────┤
│ 최대 수출 행        │ 100K    │ 1K      │ 100     │ 100     │ 0
│ 최대 쿼리 결과      │ 100K    │ 10K     │ 1K      │ 1K      │ 100
│ 승인 필요           │ ✗       │ ✗       │ ✗       │ ✓       │ ✓
├─────────────────────────────────────────────────────────────────────┤

범례:
R   = 읽기 가능
W   = 쓰기 가능
R/W = 읽기/쓰기 가능
✗   = 접근 불가
```

┌──────────────────────────────────────────────────────────────┐
