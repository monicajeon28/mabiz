## 📊 시각화 #1: 호텔 vs 크루즈 준비 시간 (총량 비교)

### 호텔 여행 준비 (5박)

```
실제 호텔 준비 항목 & 소요 시간:

1. 여행지 선택하기              2시간
   └─ 가이드북 읽기, 후기 보기, 의견 정하기

2. 항공권 예약하기              1시간
   └─ 가격 비교, 시간대 선택

3. 호텔 선택하기                2시간
   └─ 위치 확인, 후기 보기, 예산 결정

4. 비자 신청 (해외라면)         3시간
   └─ 서류 모으기, 신청서 작성, 제출

5. 여행자보험 구매              0.5시간

6. 예방접종 받기               0.5시간 (예약 포함)

7. 짐 준비하기                  3시간
   └─ 옷 선택, 무게 확인, 쌀 쌀기

8. 환전 / 신용카드 준비         1시간

9. 현지 교통 & 관광지 정보      2시간
   └─ 구글맵, 가이드북, 버스 노선

10. 기타 (숙박 컨펌, 날씨 확인 등) 1시간

────────────────────────────────
총 준비 시간: 약 16시간
실제 준비 부담감: 20-30시간 (불안 때문에 반복 확인)

✅ 당신의 역할: 100% (다 직접)
❌ 전문가 도움: 0%
```

### 크루즈 여행 준비 (5박)

```
우리가 제공하는 크루즈 준비:

Day 1: 여행 정보 받기          5분
       └─ 우리가 이메일로 전체 일정 보내드림

Day 2: 여권 확인               5분
       └─ 여권 유효기간 확인 + 사진

Day 3: 비자 신청서 자동생성     5분
       └─ 우리가 서류 다 작성했으니 서명만

Day 4: 필요 서류 제출          5분
       └─ 우리가 준비한 체크리스트 따라 제출

Day 5: 짐 리스트 확인          5분
       └─ 우리가 제시한 체크리스트 확인

Day 10: 최종 안내 메시지       5분
        └─ 떠나기 3일 전, 최종 확인

Day 14: 준비 완료 알림         0분
        └─ "당신은 완벽하게 준비됐습니다"

────────────────────────────────
총 준비 시간: 약 30분 (우리 담당)
당신의 실제 시간: 약 30분 (10분씩 × 3일)

✅ 당신의 역할: 10% (우리가 시킨 것만)
✅ 전문가 도움: 90% (우리가 다 챙김)
```

## 📊 시각화 #2: 일정별 준비 타임라인 (호텔 vs 크루즈)

### 호텔 여행 준비 타임라인 (여행 1개월 전부터)

```
Month 1 (30-20일 전):
┌──────────────────────────────────────┐
│ 여행지 결정 (2주간 고민)              │
│ ├─ 가이드북 읽기 (3시간)              │
│ ├─ 인터넷 검색 (5시간)                │
│ ├─ 가족회의 (2시간)                   │
│ └─ 최종 결정 (불안감으로 2-3회 반복) │
└──────────────────────────────────────┘
   → 당신의 스트레스: 높음 (선택 부담)

Month 2 (20-10일 전):
┌──────────────────────────────────────┐
│ 항공권 + 호텔 예약 (1주간)           │
│ ├─ 항공권 비교 (3시간)                │
│ ├─ 호텔 비교 (4시간)                  │
│ ├─ 예약 (1시간)                       │
│ └─ 예약 확인 (2-3회)                  │
└──────────────────────────────────────┘
   → 당신의 스트레스: 높음 (가격 비교)

Month 3 (10-5일 전):
┌──────────────────────────────────────┐
│ 여행 준비 (5-10시간)                  │
│ ├─ 비자 신청 (3시간)                  │
│ ├─ 짐 준비 (3시간)                    │
│ ├─ 신발 구매/교환 (1-2회)             │
│ └─ 여행 가방 구매 등                   │
└──────────────────────────────────────┘
   → 당신의 스트레스: 높음 (시간 부족)

Month 3 (5-1일 전):
┌──────────────────────────────────────┐
│ 최종 확인 (불안감)                    │
│ ├─ 날씨 확인 (3회)                    │
│ ├─ 교통 경로 다시 보기 (2회)          │
│ ├─ 호텔 위치 다시 확인 (2회)          │
│ └─ 짐 다시 정리 (2회)                 │
└──────────────────────────────────────┘
   → 당신의 스트레스: 매우 높음 (출발 직전)

총 당신의 시간: 16-20시간
총 당신의 정신 소비: 매우 높음 (반복 확인의 불안)
```
