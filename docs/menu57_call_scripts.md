# Menu #57: 파트너 교육 - 5가지 콜 스크립트 (역할극/실전용)

**작성일**: 2026-05-25  
**목표**: 신입 파트너 Week 3 역할극 + 실전 콜 스크립트  
**형식**: Grant Cardone 4단계 + PASONA + SPIN 통합

---

## 📞 Script #1: 신규 인입 (자격심사)

### 난이도: ⭐ 초급
### 시간: 5-8분
### 목표: 고객 정보 수집 → 자격심사 → 약속 확보

---

### Phase 1: Opening (30초)
```
[전화 받기 - 명확하고 친절한 톤]

"안녕하세요! [회사명] 마비즈 크루즈입니다. 
혹시 지금 말씀 드릴 수 있는 시간이 있으신가요?"

[Wait for response]

"감사합니다! 저는 [이름]이라고 합니다. 
오늘은 크루즈 여행 상품에 대해 
특별한 정보를 드리려고 전화했습니다.

혹시 크루즈 여행에 관심이 계신가요?"
```

**체크포인트**:
- [ ] 자신감 있는 목소리
- [ ] 5초 내 자기 소개
- [ ] 고객 동의 확인

---

### Phase 2: Qualifying (3-4분)
#### SPIN 질문 구조 (Situation → Problem → Implication → Need)

```
S(상황) - "혹시 가족과 함께 여행을 생각하고 계신가요?"

[고객 응답 청취]

S(상황) - "몇 분이 함께 떠나실 예정이세요?"

[고객 응답 청취]

S(상황) - "주로 어느 계절에 여행을 좋아하세요?"

P(문제) - "혹시 일반 관광과 다른 특별한 경험을 원하시나요?"

[고객 응답 청취]

P(문제) - "그렇다면 가족 모두가 편하고 즐거워할 수 있는 
          옵션을 찾고 계신 거죠?"

I(함의) - "그럼 크루즈는 정말 완벽한 선택이 될 거예요.

          왜냐하면:
          • 이동하면서 계속 새로운 경험
          • 일정이 고정되어 편함
          • 가족 모두 함께할 수 있는 다양한 활동"
```

**정보 기록 항목**:
- 이름 / 나이 / 성별
- 가족 구성 (배우자, 자녀)
- 선호 여행 시즌
- 예상 예산 범위
- 크루즈 경험 여부
- 관심 상품 (골드/일반)

**체크포인트**:
- [ ] 모든 SPIN 질문 완료
- [ ] 5개 항목 이상 정보 수집
- [ ] 고객 긍정 신호 3개 이상 포착

---

### Phase 3: Presenting (1-2분)
#### PASONA O(Offer) 단계

```
"[이름]님이 말씀하신 조건에 완벽하게 맞는 상품이 있습니다.

저희 [크루즈 상품명]은:

1) 가족 친화적:
   → 키즈 클럽, 수영장, 극장 공연 포함
   → 부모님도 스파, 다이닝 즐기실 수 있음

2) 편안한 일정:
   → 항구별 1-2일 체류 (충분한 관광)
   → 배에서 충분한 휴식

3) 가성비:
   → 항공 + 숙박 + 식사 + 활동 모두 포함
   → 분할 결제 가능

지난 3개월간 저희 고객 분들의 만족도는 97%입니다.

특히 가족 여행으로는 다른 상품보다
3배 이상 선호하고 있어요."
```

**체크포인트**:
- [ ] 3가지 핵심 이점 설명
- [ ] 사회증명 (만족도/선호도) 언급
- [ ] 고객 반응 확인

---

### Phase 4: Closing (30-60초)

```
"이제 두 가지 옵션이 있습니다.

1) 지금 바로 예약
   → [예약 특가] 30% 할인 가능
   → 카드 결제 최대 12개월 분할
   → 5월 말까지만 유효

2) 2-3일 생각해 보신 후 다시 연락
   → 가족 의견 모으실 수 있음
   → 단, 할인은 오늘만 가능

[일시정지 - 고객 반응 기다리기]

어느 것이 더 나을까요?"

[만약 거절: Grant Cardone 재접근]
"[이름]님, 오늘 결정하신 이유가 있으신가요?
아니면 어떤 부분이 더 확인되어야 하나요?"
```

**체크포인트**:
- [ ] 명확한 선택지 제시 (2가지)
- [ ] 희소성 강조 (할인 유효 기간)
- [ ] 침묵 시간 줌 (고객이 답할 시간)
- [ ] 거절 시 재접근 질문

## 📞 Script #2: 가격 이의 처리 ("너무 비싸요")

### 난이도: ⭐⭐ 중급
### 시간: 8-12분
### 목표: 가치 재정의 → 이의 해소 → 약속 확보

---

### Phase 1: LISTEN (30초)
#### Grant Cardone 6단계 프레임워크 - 1단계: 경청

```
[고객이 "너무 비싸요"라고 말함]

"알겠습니다. 가격이 concern이 되시는군요.

[일시정지 - 5초 침묵]

혹시 다른 상품과 비교해서 비싼 건가요?
아니면 전체적으로 예산이 맞지 않는 건가요?"

[고객 응답 청취 - 절대 말 끊지 않기]
```

**핵심**: 고객이 완전히 다 말할 때까지 듣기

---

### Phase 2: ISOLATE (1분)
#### 정확한 이의 파악

```
고객 A: "다른 여행사가 더 싸요"
→ 대응: "그럼 정확히 몇 원 차이인지 말씀해주실래요?"

고객 B: "우리 예산이 1000만원인데..."
→ 대응: "그럼 이 금액이 최대치인가요?
         1000만원 안에서 할 수 있는 게 뭐가 있을까요?"

고객 C: "여행을 가기로 하지 않았어"
→ 대응: "아, 그럼 여행 계획 자체가 아직 미정이라는 거네요.
         그럼 어떤 상황이 되면 여행을 다시 생각할까요?"
```

---

### Phase 3: VALIDATE (1-2분)
#### Grant Cardone 10렌즈 #1: 손실회피 + #6: 타이밍

```
[고객의 이의를 인정하되, 가치 재정의]

"[이름]님 입장에서는 그렇게 보일 수 있죠.
저도 처음엔 그렇게 생각했거든요.

하지만 한번 봐봐요:

1) 가격 비교
   순수 숙박비:
   - 일반 호텔 4박: 80만원
   - 크루즈 4박: 150만원 (DIFFERENCE: 70만원)
   
   그런데 크루즈에는:
   + 모든 식사 포함 (4박 5끼 = 일반 식사 40만원 절약)
   + 이동 비용 포함 (항공료 60만원 절약)
   + 엔터테인먼트 포함 (공연/활동 20만원 절약)
   
   실제 차이: 150만원 - 120만원 = 30만원만 추가

2) 가족과의 시간
   - 일반 여행: 자동차/버스 이동으로 피로
   - 크루즈: 배 안에서 가족과 품질시간 (가치평가 불가)

3) 타이밍
   - 지금 예약: 가격 고정
   - 6개월 후: 성수기라 30% 더 비쌈
   
   즉, 지금 결정이 더 저렴해요."

[일시정지]

"이렇게 보니 어떤가요?"
```

**체크포인트**:
- [ ] 손실회피 메시지 (절약액 강조)
- [ ] 타이밍 긴박감 (가격 상승)
- [ ] 수치로 정확히 설명
- [ ] 고객 반응 확인

---

### Phase 4: EMPATHIZE (30초)

```
"제가 [이름]님 입장을 이해합니다.
돈은 소중하니까요.

그래서 저는 최대한 합리적인 옵션을 드리고 싶어요.

현재 가능한 것:
• 분할 결제: 최대 12개월 (월 약 25만원)
• Early Bird: 추가 5% 할인 (지금만)
• 동반자 추천: 친구분 부르면 10% 할인"
```

---

### Phase 5: PROBE (1분)
#### SPIN Implication

```
"그럼 이 옵션들 중에서,

'분할 결제 월 25만원'이 가능하다면?
'Early Bird 5% 할인'이 있다면?

이게 문제를 해결할까요?

아니면 다른 뭔가가 아직도 확실하지 않은 건가요?"

[고객 응답 청취]
```

---

### Phase 6: REPEAT / CLOSE (1분)

```
[만약 가격 이의 해소됨]
"그럼 [이름]님은 월 25만원으로 가족과 
최고의 추억을 만드실 수 있다는 뜻이네요.

지금 예약하시겠어요?
카드로 할까요, 계좌이체로 할까요?"

[만약 여전히 거절]
"[이름]님, 혹시 가격 말고 다른 concern이 
더 있는 건가요? 

예를 들어:
• 일정이 맞지 않는 건가요?
• 배멀미가 걱정되나요?
• 가족이 동의하지 않은 건가요?

어느 것이 진짜 이유인가요?"

[정확한 이의 파악 후 그에 맞는 처리]
```

## 📞 Script #3: 준비 복잡 불안 고객

### 난이도: ⭐⭐ 중급
### 시간: 8-10분
### 목표: 5단계 중재 질문 → 불안 해소 → 약속

---

### 시작:
```
[고객의 불안 신호 감지]
"음... 좋긴 한데... 준비할 게 많을 것 같은데?"
```

### Phase 1: 정의 (Define)
```
"[이름]님, 괜찮아요. 
많은 분들이 그런 걱정을 해요.

근데 정확히 뭐가 가장 복잡할 것 같으신가요?

예를 들어:
• 패스포트/여권 준비?
• 어디서 예약해야 하는지?
• 비용 결제?
• 짐 준비?

이 중에 뭐가 가장 걱정되나요?"

[고객이 구체적으로 말할 때까지 기다림]
```

### Phase 2: 복잡성 인정 (Complexity)
```
"아, 그 부분이 복잡하게 느껴지시는군요.

실은 많은 분들이 그렇게 느껴요.

하지만 저는 그 부분을 정확히 설명해드릴 수 있어요.

그럼 우리가 step by step 해볼까요?"
```

### Phase 3: 불안 이름 붙이기 (Naming the Anxiety)
```
"[이름]님 입장에선, 

'혼자 준비해야 한다'는 생각이 들어서
불안한 거겠네요?

맞나요?"

[고객 확인]

"알겠습니다. 그럼 
'혼자가 아니라 우리가 함께 준비한다'
고 생각하면 어떨까요?

실제로 저희가 제공하는 것들:
• 패스포트 확인 체크리스트
• 자동 여행보험 가입
• 탑승 전 오리엔테이션 영상
• 24시간 콜센터 지원

이렇게 하면 걱정 좀 덜어질까요?"
```

### Phase 4: 고객 발화 (Customer Articulation)
```
"그래도 이해가 안 되시는 게 있으면
지금 다 물어봐도 돼요.

예를 들어, 여권 제출은 언제까지인가요?
→ "45일 전까지 제출하면 됩니다"

배 타기 전에 특별히 준비할 게 뭐가 있나요?
→ "밤옷, 슬리퍼, 처방약 정도면 충분합니다"

패키지에 뭐가 포함되고 뭐가 추가인가요?
→ [정확히 설명]

이렇게 하면 좀 명확해지나요?"

[모든 질문에 명확하게 답변]
```

### Phase 5: 해결 방안 제시 (Resolution)
```
"자, 이제 정리해보니까 어떤가요?

생각보다 훨씬 간단하지 않나요?

그리고 저희는 이 과정을 
step by step 함께 할 거구요.

그럼 이제 예약하셔도 될 것 같은데,
어떻게 생각하세요?"

[고객이 "알겠습니다" 신호 → 바로 클로징]
```

## 📞 Script #4: 차별성 강조 (경쟁사 비교)

### 난이도: ⭐⭐⭐ 고급
### 시간: 10-12분
### 목표: 경쟁사 대비 차별화 → 불안 해소 → 선택 유도

---

### Opening
```
"[이름]님, 다른 여행사도 접촉하셨나요?"

[고객: "네, 몇 곳 봤어요"]

"아, 그럼 우리 상품이 왜 다른지 
정확히 비교해 드릴 수 있어요.

지금 비교 중인 회사가 몇 곳이세요?"

[고객 응답]

"좋아요. 그럼 저희와 비교해볼까요?"
```

### 경쟁사 vs 우리 (구체적 비교표)

| 항목 | 경쟁사 A | 경쟁사 B | 우리 마비즈 |
|------|---------|---------|----------|
| **이동** | 자유 없음 (고정) | 자유지만 피로 | 자유 + 휴식 |
| **숙박** | 3인실만 | 2인실/3인실 | 다양한 옵션 |
| **식사** | 제한적 | 일부만 | 전부 포함 |
| **가족 활동** | 부족 | 보통 | 최고 수준 |
| **A/S** | 한국사무소 없음 | 24시간 전화만 | 카톡/전화/웹 |
| **가격 (4박)** | 150만원 | 140만원 | 150만원 |

**핵심 메시지**:
```
"[이름]님, 표로 보니까 아시겠지만,

경쟁사는:
• 가격은 싸지만 서비스가 제한적
• 또는 서비스는 좋지만 비싸거나
• 또는 A/S가 약함

우리는:
• 가격은 시장 중간 수준
• 하지만 서비스/A/S는 최고 수준
• 특히 가족 중심 설계

즉, '가격 대비 최고의 가치'라고 할 수 있어요."
```

### 5가지 이의 처리

**이의 1: "경쟁사 A가 가격이 저렴한데?"**
```
"네, 맞아요. 그 회사는 15만원을 세이브합니다.

하지만 [이름]님, 생각해보세요.

3박 4일 동안 가족과 함께하는데,
15만원(하루 3만 7천원) 때문에
서비스 품질을 포기하실래요?

특히 아이들이 있다면요?

우리는 그 15만원으로:
• 어린이 클럽 전담 스태프
• 패밀리 다이닝 우선석
• 키즈 엔터테인먼트 매일 운영
• 비상시 아동 의료 서비스

을 제공해요.

15만원으로 이런 가치를 사신다고 보면,
굉장히 저렴한 거예요."
```

**이의 2: "경쟁사 B의 자유도가 더 높던데?"**
```
"맞습니다. 경쟁사 B는 자유도가 높아요.

다만 그건 '자유롭지만 피로로움'이라는 뜻이에요.

왜냐하면:
• 항구마다 1시간 소요 (탑승/하선)
• 관광지 자체 찾아가기 (길 헤매기)
• 식사 시간 관리 (일정 주의)
• 짐 나르기 (매일 피로)

vs

우리:
• 호텔처럼 여유있게 아침 늦게 나가기
• 짐은 배에서 모두 관리
• 식사 시간 자유 (24시간 카페)
• 피로도 최소 (배에서 쉬기)

즉, '자유롭지만 피로'보다는
'편하지만 자유'가 훨씬 낫지 않을까요?"
```

**이의 3: "그냥 기존 여행사에서 예약했어"**
```
"아, 그렇구나요.

혹시 아직 입금 안 하셨어요?"

[고객: "아직 안 했어"]

"다행이네요. 그럼 한번 더 비교해볼 가치가 있어요.

왜냐하면:
• 우리도 같은 크루즈를 판매합니다
• 하지만 우리는 추가 서비스가 있어요

예를 들어:
• 한국인 가이드 제공
• 한글 정보 제공
• 귀국 후 영상/앨범 서비스

그리고:
• 입금 기한이 동일하므로
• 지금 우리로 변경해도 됩니다

결과적으로 '같은 크루즈, 더 나은 서비스'
를 받을 수 있다는 거예요.

변경하시겠어요?"
```

• 예산 최종 확인?

내일 컨펌하시면 
15% 할인이 10% 할인으로 줄어듭니다.

그래도 내일 확인하실래요?"

"아니면 지금 예약하고,

## 📞 Script #5: 최종 클로징 (즉시 구매)

### 난이도: ⭐⭐⭐⭐ 최고급
### 시간: 5-8분
### 목표: 감정적 마무리 → 즉시 결정 → 성약 확보

---

### Context (앞의 스크립트 진행 후)
```
[모든 이의가 해소되고, 고객이 "좋아요"라고 함]
```

### Phase 1: 감정 확인 (Emotional Check)

```
"[이름]님, 지금 기분이 어떠세요?

혹시 설레나요?

아니면 아직 조금 불안해요?

솔직하게 말씀해주세요."

[고객 응답 청취]

"그 기분이 정상이에요.

큰 결정을 하려고 하니까요.

하지만 이 기분, 가족과 함께 
크루즈를 떠날 때 누백배가 돼요.

상상해보세요.

아이들이 배에 올라서서 
"우와!" 하는 소리. 

배우자가 당신 손을 잡고 
석양을 바라보는 순간.

그런 기억이 평생 남는 거예요.

금전적으로는 150만원이지만,
감정적으로는 무한대의 가치가 있어요."

[일시정지 - 고객이 감정적으로 몰입]
```

**체크포인트**:
- [ ] 고객의 감정 상태 확인
- [ ] 미래 이미지 제시 (가족과 함께)
- [ ] 금전 vs 감정 가치 비교

---

### Phase 2: 삼중 선택 (Triple Choice)

```
"자, 이제 결정의 순간이에요.

[이름]님이 선택할 수 있는 게 3가지가 있어요.

선택지 1: 지금 바로 예약 (8월 출발)
• Early Bird 30% 할인 (오늘만)
• 분할 결제 12개월 가능
• 패밀리 스위트 룸 배정
→ 월 약 12만원씩 부담

선택지 2: 다음 달 예약 (9월 출발)
• Early Bird 할인 없음 (10% 기본)
• 분할 결제 12개월 가능
• 스탠다드 룸만 가능
→ 월 약 13만원씩 부담

선택지 3: 지금은 하지 않음
• 나중에 비싸진 가격에 예약
• 좋은 시간대 배정 어려움
• 좋은 객실 선택 불가

[일시정지 - 3초간 침묵]

[이름]님, 이 중에 뭐가 
당신 가족에게 가장 좋을까요?

선택지 1이겠지, 맞죠?"

[만약 고객이 "아직도 생각해봐야"라고 함]

→ Grant Cardone Rebuttal:

"[이름]님, 혹시 뭔가 더 알고 싶으신 게 있으신가요?

가격이 아직도 비싼 건가요?
일정이 안 맞는 건가요?
가족이 반대하는 건가요?

또는 정말로 여행 자체를 포기하려는 건가요?"

[정확한 거절 이유 파악 후 다시 처리]
```

**체크포인트**:
- [ ] 3가지 선택지 명확히 제시
- [ ] 선택지 1의 이점 강조 (할인, 좋은 객실)
- [ ] 일시정지로 고객이 답할 시간 제공

---

### Phase 3: 경제적 강화 (Financial Reinforcement)

```
만약 고객이 여전히 가격을 언급:

"[이름]님, 다시 한번만 봐봐요.

월 12만원을 위해 이런 것들을 포기할래요?

• 아이들의 웃음소리
• 가족과의 품질시간
• 스트레스 해소
• 추억 만들기
• 가족 결속력 강화

이 중에 12만원으로 사실 수 없는 거잖아요?

그래서 이건 '지출'이 아니라 
'투자'라고 생각해야 해요.

10년 뒤에 아이가 성장했을 때,
'엄마, 아빠와 함께한 크루즈 여행'을 기억하고

그게 인생의 행복 지수에 
얼마나 큰 영향을 미칠지

생각해봤어요?"

[일시정지]

"자, 그럼 예약할래요?"
```

---

### Phase 4: 최종 성약 (Closing)

```
[고객이 "네, 예약할게요"라고 함]

"좋아요! 최고의 결정이에요!

[이름]님이 얼마나 행복할 결정을 했는지
앞으로 알게 될 거예요.

자, 그럼 필요한 정보들 정리할게요.

1) 여권번호 / 생년월일
2) 선호 객실 위치 (전망 / 고요함)
3) 특별 요청사항 (허니문, 기념일 등)
4) 결제 방법 (카드 분할 / 계좌이체)

이 정보들 지금 말씀해주실래요?"

[정보 수집]

"완벽해요. 그럼 오늘 계약서 보내드릴게요.
내일까지 사인해서 보내주시면
바로 예약 확정입니다.

그리고 가족과 함께
이 설레임을 나누세요.

다음주부터는 준비 체크리스트도
보내드릴 테니까요.

정말 감사합니다, [이름]님!"

[긍정적 톤으로 마무리]
```

**체크포인트**:
- [ ] 감정적 강화 (행복의 추억)
- [ ] 실질적 정보 수집
- [ ] 다음 단계 명확히 안내
- [ ] 긍정적 마무리

## 📋 역할극 평가 체크리스트

### 각 스크립트별로 평가:

```
Script #1: 신규 인입
□ Opening: 5초 내 자기 소개 (5점)
□ Qualifying: SPIN 모든 단계 (20점)
□ Presenting: 3가지 이점 (15점)
□ Closing: 선택지 제시 (10점)
총 50점

Script #2: 가격 이의
□ LISTEN: 완전히 경청 (10점)
□ ISOLATE: 정확한 이의 파악 (15점)
□ VALIDATE: 손실회피 + 타이밍 (20점)
□ EMPATHIZE: 고객 입장 인정 (10점)
□ PROBE: 추가 concern 확인 (10점)
□ REPEAT: 재접근 또는 클로징 (15점)
총 80점

Script #3: 준비 불안
□ Define: 정확한 불안 파악 (15점)
□ Complexity: 복잡성 인정 (10점)
□ Naming: 불안 이름 붙이기 (10점)
□ Customer Articulation: Q&A (20점)
□ Resolution: 해결 방안 + 클로징 (20점)
총 75점

Script #4: 차별성
□ Opening: 경쟁사 파악 (10점)
□ 비교표: 정확한 데이터 (20점)
□ 이의처리 5가지: 각 15점 (75점)
□ 최종 강조 (10점)
총 115점 → 정규화 100점

Script #5: 클로징
□ 감정확인: 공감 (15점)
□ 삼중선택: 명확한 제시 (25점)
□ 경제강화: 감정적 메시지 (20점)
□ 최종성약: 정보수집 + 마무리 (20점)
총 80점

최종 평점 기준:
90-100점: Excellent (✓ 실전 투입 가능)
75-89점: Good (△ 추가 연습 후 투입)
60-74점: Fair (✗ 재교육 필요)
60점 미만: Poor (✗ 1주일 추가 교육)
```
