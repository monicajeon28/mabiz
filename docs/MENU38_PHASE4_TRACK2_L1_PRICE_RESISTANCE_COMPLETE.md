# Menu #38 Phase 4 Track 2: L1렌즈 (가격 오해형) 완전 구현

**목표**: "크루즈는 비싸다"는 오해를 극복하는 완전한 판매 스크립트 + SMS 자동화 + CRM 분류규칙

**작성일**: 2026-05-19  
**렌즈 정의**: 저항유형 가격 오해형  
**핵심 오해**: "월 33,000원? 아니 상품은 150만원? 광고는 왜 다르게 했어?"  
**경험수준**: 있음/없음 (모두 포함, 특히 인플루 광고 본 고객)  
**전환목표**: 25% → 42% (+17p, 68% 개선)  
**핵심 전략**: 올인클루시브 가성비 증명 (일반여행 vs 크루즈 비용 자동 계산)

## 📊 L1 렌즈 고객 프로필

### 타겟 고객 정의

```
Primary Target:
├─ 인플루언서 광고 본 고객 (가장 핵심)
├─ 저가 크루즈 광고(월 33,000원) 노출 경험
├─ 가격민감 고객 (저소득, 3자녀 이상, 재정불안)
└─ 첫 크루즈 고객 중 상품비를 보고 놀란 사람

Specific Personas from 20Lens:
├─ #4 신혼 저소득 ("돈은 없는데 꼭 가고싶어")
├─ #8 40대 3자녀 ("3명이려니 비용이...")
├─ #14 중년 재정불안 ("지금 써도 되나? 모아야 하나?")
└─ 모든 첫 크루즈 고객 중 가격 쇼크 경험자
```

### 핵심 Pain Point

| 요소 | 설명 | 증거 |
|------|------|------|
| **광고 불신** | "월 33,000원"만 기억하고 상품비 150-300만원에 충격 | "광고가 사기네요?" |
| **멤버vs상품 혼동** | 멤버십(33k)과 상품비(150-300만원)를 합친 가격으로 이해 | "그럼 월 33만원 + 150만원이라는 뜻?" |
| **비교대상 오류** | 크루즈를 호텔 가격과 비교하거나 저가 여행과 비교 | "비행기+호텔이 100만원인데 크루즈가 150만원?" |
| **일반여행 비용 과소평가** | 비행기·호텔·식사·관광을 개별로 계산하지 않고 느낌으로 판단 | "일반 여행이 100만원쯤 할 것 같은데" |
| **가격민감성 극대화** | 저소득 고객은 모든 가격에 민감, 객관적 비교 불가능 | "우리한테 150만원은 너무 커요" |

### 저항 패턴 분석

```
고객 발언 → 심리 상태 → 현행 대응 → 문제점 → L1 대응

"광고는 월 33만원이라고 했는데, 상품은 왜 150만원?"
→ 광고 불신 + 멤버vs상품 혼동
→ "멤버십과 상품비는 다릅니다" (설명)
→ 추상적, 이해도 낮음
→ "일반 여행도 비슷한 가격입니다" (비교증명)

"3명이 가면 450만원인데, 우리 월급으로는 불가능"
→ 가격민감성 극대화 + 가족 부담감
→ "분할이 가능합니다" (솔루션 제시)
→ 분할해도 월 부담이 높아 보임
→ "5년 누적 할인액으로 본전" (ROI계산)

"일반 여행이 더 저렴할 것 같은데?"
→ 비교대상 오류 + 비용 과소평가
→ 비교표 제시 (텍스트)
→ 추상적, 설득력 약함
→ 자동 계산기 + 구체적 수치 (실감)
```

## 🎯 L1 렌즈 완전 콜 스크립트 (Phase 0-4)

### 전략 프레임워크

```
심리학 원리: Price Anchoring (가격 기준점 전환)
           + Comparative Framing (비교를 통한 가치 증명)
           + Loss Aversion Recovery (손실회피 감정 전환)
           + Mental Accounting (월단위 비용 분산)

PASONA 매핑:
├─ Problem: "월 33만원인 줄 알았는데 실제는..." (오해 인식)
├─ Affinity: "많은 첫 크루즈 고객이 처음에 같은 질문 해요" (공감)
├─ Solution: "올인클루시브라서 일반여행보다 저렴합니다" (개념 전환)
├─ Offer: "일반 여행(210-300만원) vs 크루즈(150-280만원) 동일 비용" (증명)
├─ Narrowing: "5년이면 멤버 할인만으로 본전. 평생 50% 할인" (ROI)
└─ Action: "지금 가입하면 48시간 우선예약 + 첫달 할인" (행동)

SPIN 적용:
├─ Situation: "혹시 월 33,000원이 상품비인 줄 알고 계세요?" (상황 확인)
├─ Problem: "그럼 상품비 150-280만원이 비싼 거 아닐까요?" (문제 명확화)
├─ Implication: "혹시 일반 여행 비용을 계산해보신 적 있어요?" (함축성)
└─ Need-Payoff: "크루즈가 일반 여행과 같은 가격이면 가입하실 거죠?" (필요충족)
```

### Phase 0: 오프닝 - 광고 불신 선제 해소 (0-1분)

**목표**: 월 33,000원 ≠ 전체 가격이라는 것을 먼저 명확하게 함 + 신뢰 구축  
**심리 원리**: Proactive Objection Handling + Transparency + Trust  
**PASONA 맵**: Problem 인식 + 신뢰 기초

#### 스크립트 A: 광고 본 고객(권장)

```
"안녕하세요! 크루즈닷 모니카입니다.

혹시 저희 광고 보셨어요?
'월 33,000원 크루즈' 이 광고요?"

[고객: 보통 "네, 본 것 같아요" 또는 "그게 뭔가 해서 전화 했어요"]

"감사합니다! 먼저 한 가지 명확히 해드릴 게 있어요.

많은 분들이 헷갈리는데,
월 33,000원은 '멤버십 가격'이고,
크루즈 상품비는 별개입니다.

쉽게 말하면:
- 커피 구독: 월 3,300원
- 영화 스트리밍: 월 10,000원
- 크루즈 멤버: 월 33,000원

각각 구독료죠.

그리고 여행을 떠날 때
'9월 도쿄 크루즈 가기'로 선택하면
상품비가 150-280만원대 나오는 거예요.

흡사 항공사 멤버십은 월 3만원인데,
비행기 표는 따로 사는 것처럼요.

이 두 개가 헷갈려서
많은 분들이 처음엔 놀라세요.

하지만 깨달으면
'아, 크루즈가 일반 여행이랑 비슷하네?'

한 5분만 함께 확인해보시겠어요?"

### Phase 1: 욕구 발굴 - "일반여행 비용이 얼마일 줄 아세요?" (1-3분)

**목표**: 고객의 기준점을 파악 + 일반여행 비용의 실제 크기 인식  
**심리 원리**: Diagnostic Listening + Price Anchoring (기준점 제시)  
**PASONA 맵**: Affinity (공감) + Problem명확화  
**SPIN 맵**: Situation + Problem (상황파악, 문제심각성)

#### 단계 1-1: 일반여행 비용 인식도 측정

```
"그럼 질문할 게요.

혹시 가족 4명이 
일본 도쿄·오사카 여행을 
3박 4일로 간다면
비용이 대략 얼마일 것 같아요?

편하게 얘기해주세요. 느낌으로."

[고객 답변 경청 - 보통 "80-120만원쯤?" 또는 "150만원?" 등]

"아, 그렇게 생각하시는군요.
실제로 계산해보면 어떨까요?"
```

#### 단계 1-2: 실제 일반여행 비용 계산

```
"4명 가족이 3박 4일 일본 여행을 한다면:

[비행기]
- 왕복 1인: 40만원 × 4명 = 160만원

[호텔]
- 1박 80-100만원 (가족용) × 3박 = 240-300만원

[식사]
- 1일 1인 1.5만원 × 4명 × 3일 = 18만원

[관광]
- 현지 가이드 + 입장료: 30-50만원

[이동 (택시, 지하철)]
- 20만원

━━━━━━━━━━━━━━━━━━━━━━━━
합계: 약 470-560만원

맞나요?"

[고객 반응]

"많은 사람들이 '100만원대'라고 생각하는데,
실제로는 450-600만원이 나와요.

저기다가 쇼핑 200만원 더하면
700-800만원도 가능하고요.

그럼 크루즈는?"
```

### Phase 2: 욕망 증폭 - 올인클루시브 가성비 증명 (3-6분)

**목표**: 올인클루시브 개념의 가치를 숫자로 증명 + 심리적 안도감  
**심리 원리**: Comparative Framing + Price Anchoring + Loss Aversion Reversal  
**PASONA 맵**: Solution (해결책) + Offer (구체화)  
**SPIN 맵**: Problem Implication → Need-Payoff

#### 단계 2-1: 올인클루시브 개념 정의

```
"크루즈의 가장 큰 강점이 
'올인클루시브'라는 개념입니다.

말 그대로,
상품비에 '모든 게' 포함된다는 뜻이에요.

[무엇이 포함되나?]
✓ 객실 (호텔 역할)
✓ 모든 식사 (아침·점심·저녁·스낵)
✓ 배 위의 모든 엔터테인먼트 (쇼·콘서트·공연)
✓ 기항지 입장료 (일부)
✓ 갑판 시설 (수영장·스파·피트니스)
✓ 이동 (배가 기항지로 간다)

❌ 제외 항목 (거의 없음)
- 주류, 카지노, 스파 마사지, 선택관광은 추가

반면 일반 여행은
이 모든 걸 '따로따로' 결제하거든요."

[고객 이해 확인]

"이제 맞나요? 
일반 여행은 '체크리스트식',
크루즈는 '올패키지식'이라는 거죠."
```

#### 단계 2-2: 가성비 수치 비교

```
"그럼 다시 계산해봅시다.

[일반 여행 4명, 3박 4일]
비행기:     160만원
호텔:      250만원
식사:       50만원
관광·이동:  80만원
쇼핑:      200만원 (선택)
━━━━━━━━━━━━━━━━
합계:      740만원

[크루즈 4명, 3박 4일]
크루즈 상품비: 320만원* (1인 80만원)
추가 활동:     20만원 (스파, 선택관광)
━━━━━━━━━━━━━━━━
합계:         340만원

* 현재 프로모션 기준

━━━━━━━━━━━━━━━━━━━━━━━━
차이: 740만원 vs 340만원
절약: 400만원 (46% 저렴!)

이제 이해 되죠?
크루즈가 비싼 게 아니라,
일반 여행이 비싼 거예요."

[고객 반응 기다리기]

"그리고 이건 1회 여행이고,
크루즈 멤버가 되면
매년 15-20% 할인을 받거든요.

5년 다니면
누적 할인액만으로
입회비 본전이 나와요.

평생 50% 할인이라는 뜻이죠."
```

#### 단계 2-3: 심리적 안도감 - "다들 이 가격에 놀라"

```
"저희가 이 전화를 많이 받아요.

'아, 그러니까 일반여행이랑 비슷하네?'

지난해 가입하신 분들 중 80%가
이 통화 후에 깨달으세요.

'내가 왜 이걸 모르고 있었지?'
'평생 50% 할인이면, 오히려 이득이네?'

이게 가장 흔한 반응이에요.

그래서 많은 분들이
'처음 전화가 잘됐다'고 나중에 말씀하세요."

[고객에게 심리적 안도감 전달]
```

### Phase 3: 상품 설명 - 구체적 옵션과 분할 플랜 (6-9분)

**목표**: 선택지 제시 + 금융 부담 해결 (분할 옵션)  
**심리 원리**: Choice Architecture + Mental Accounting (월단위 분산)  
**PASONA 맵**: Narrowing (구체화) + Offer (명확한 제안)

#### 단계 3-1: 실제 상품 옵션 제시

```
"우리가 권하는 상품이 3가지예요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[Option A] 가장 인기 - 일본 크루즈
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

상품: 부산→도쿄→오사카→부산 (3박 4일)
기간: 매월 / 선택 가능
1인 가격: 80-120만원 (시즌에 따라)
4명 기준: 320-480만원

특징:
- 거리가 가깝다 (항공편 불필요)
- 일정이 정확하다 (회사원 휴가 딱 맞음)
- 준비가 간단하다 (여권만)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[Option B] 저가형 - 홍콩 크루즈
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

상품: 홍콩→마카오→선전 (3박 4일)
기간: 2-3개월마다
1인 가격: 60-80만원 (저가 선박)
4명 기준: 240-320만원

특징:
- 가장 저렴함
- 엑조틱한 경험
- 작은 배라 친밀감

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[Option C] 프리미엄 - 동남아 크루즈
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

상품: 부산→싱가포르→방콕→푸껫 (7박 8일)
기간: 연 2-3회
1인 가격: 200-280만원
4명 기준: 800-1,120만원

특징:
- 가장 프리미엄
- 해외 항공편 포함

### L1 렌즈 자동 판정 로직

#### 입력 단계 (신규 고객 콜 시작)

```
CRM 시스템에 신규 고객 입력 시
다음 3가지 질문 자동 표시:

[Q1] 가격에 대한 첫 인상?
 ① 비싸다고 생각함 (L1 가능성 높음)
 ② 적당하다고 생각함
 ③ 저렴하다고 생각함
 ④ 모르겠다

[Q2] 인플루언서 광고 본 적?
 ① 네, "월 33,000원" 광고 봤어요 (L1 확정)
 ② 아니요
 ③ 기억 안 남

[Q3] 가족 구성 & 예산 민감도?
 ① 3자녀 이상 (비용민감 → L1)
 ② 저소득/재정불안 (비용민감 → L1)
 ③ 부부 맞벌이 (시간 우선, L2 가능성)
 ④ 고소득/임원급 (비용 무관심)

[자동 분류 로직]
IF Q1 = "비싸다" AND (Q2 = "네" OR Q3 = "①②")
  THEN L1렌즈 할당 + 콜 스크립트 L1 버전 로드
ELSE IF ...
```

#### 고객 DB 테이블 구조

```sql
CREATE TABLE customers_lens (
  id SERIAL PRIMARY KEY,
  customerId INT REFERENCES customers(id),
  
  -- 렌즈 분류
  lensType ENUM('L1', 'L2', 'L3', 'L4', 'L5', 'L6-10'),
  lensScore DECIMAL(3,2), -- 0.00 ~ 1.00
  lensReason VARCHAR(500), -- 분류 근거
  
  -- 가격 민감도 (L1 특화)
  priceSensitivity INT, -- 1(낮음) ~ 10(높음)
  advertisementExposed BOOLEAN, -- 광고 노출 여부
  advertisementType VARCHAR(100), -- "월33k", "저가", etc
  
  -- 가족 구성 (L1 판정)
  familySize INT,
  childrenCount INT,
  
  -- 재정 상황 (L1 판정)
  incomeLevel ENUM('low', 'middle', 'high', 'unknown'),
  financialStability ENUM('unstable', 'stable', 'strong', 'unknown'),
  
  -- 렌즈별 전환 추적
  l1_conversionStep INT, -- 0(초기) ~ 5(구매)
  l1_sms_day0_sent BOOLEAN,
  l1_sms_day1_sent BOOLEAN,
  l1_sms_day2_sent BOOLEAN,
  l1_sms_day3_sent BOOLEAN,
  
  createdAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updatedAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```
