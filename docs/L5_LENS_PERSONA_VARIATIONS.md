## 1️⃣ 신혼부부형 (20-35세, 경험 추구)

### 페르소나 언어 스타일
- 속도감 있고 열정적
- "신혼여행이 이 정도면 괜찮겠네"
- "우리 둘이만 있는 여행"이 중요
- 새로운 경험에 개방적

### 콜 스크립트 (PHASE 1-6)

```
PHASE 1: 오프닝
"신혼부부들이 많이 생각하는 거 있어요.
'우리가 아직 크루즈를 갈 나이일까?'

근데 제가 확신해서 말씀드리는 게,
신혼부부들이 가장 만족하는 고객이에요.
왜냐면..."

[포즈: 2초]

PHASE 2: 문제 + 공감 (Monica Step 2)
"신혼부부들은 보통
'호텔 vs 크루즈' 고민하시거든요.

그런데 당신 같은 신혼부부들이 말해요:
'호텔은 밤에만 머물고, 낮에는 뭘 해요?'
'배는? 아침부터 밤까지 우리 둘이 즐길 게 끝이 없네.'

당신도 이런 고민, 없어요?"

[고객이 3-5줄 설명]

PHASE 3: 문제 + Implication (SPIN)
"그래요. 맞아요.
신혼때 가는 여행이 중요한 이유가

평생 기억에 남는 추억이 결정되거든요.

호텔 여행은 어떻게 되냐면:
- 낮: 관광지 투어 (남 여행과 비슷)
- 밤: 호텔 방 (따로 지낼 수 있는 시간도 없어)

배 여행은 어떤가요?
- 아침: 스파 함께, 운동 함께
- 낮: 기항지 둘이 탐험
- 저녁: 나이트클럽 둘이 춤
- 밤: 침실 프라이빗

48시간을 당신 둘이만의 시간으로 만들 수 있어요.

이게 평생 추억의 차이가 되는 거, 느껴지죠?"

[포즈: 3초]

PHASE 4: 해결책 (제시)
"우리 신혼부부 고객들이
평균 85점을 주는 이유가 바로 이거예요.

'우리 둘만의 시간이 이렇게 많았던 적이 없다'
'매일 새로운 기항지, 매일 새로운 경험'
'호텔은 지겹지만 배는 계속 달라져'

그래서 1달, 3개월 후 다시 물어보면:
'또 가고 싶아요' 하는 거죠.

당신도 이런 경험, 원하지 않아요?"

PHASE 5: 신뢰 형성
"제가 볼 때, 당신 같은 신혼부부는
이미 '일반 여행지'는 안 맞아요.

왜냐면 당신 둘은
'새로운 경험'을 추구하거든요.

그런데 호텔은 매번 똑같아요.

배는? 매 여행이 다른 배, 다른 기항지, 다른 경험.

이게 당신 둘에게 진짜 맞는 여행이야."

[포즈: 2초]

PHASE 6: 클로징 (삼중 선택)
"그래서 이렇게 생각해봐요.

'내가 가도 되나?' 고민하셨는데,
애초에 당신은 '꼭 가야 하는' 고객이에요.

다만 어떤 배, 어떤 기항지를 고르냐의 문제일 뿐.

[포즈: 3초]

당신 둘이 '낭만적인 크루즈'를 원해?
아니면 '파티하는 배'를 원해?
아니면 '조용하고 럭셔리한 배'를 원해?

셋 중에 골라주면 돼. 그러면 끝."

[포즈: 3초]
```

### SMS Day 0 (신혼부부 변형)
```
📲 "안녕하세요, OOO님!

방금 통화 때 '우리도 크루즈 가도 돼?'
이렇게 물으셨어요.

YES. 당신들이 '제일' 가야 하는 고객들이에요.

왜냐면:
✅ 신혼부부들이 가장 만족 (평균 85점)
✅ 48시간을 둘이만 보낼 수 있어 (호텔과 완전 다름)
✅ 매일 새로운 기항지 (같은 경험 안 함)
✅ 일반 여행과 비교할 수 없는 추억

당신 둘이 지금 '새로운 경험'을 찾는 거 맞죠?

그럼 여기서 끝나지 않아.

- 마비즈 크루즈팀"

[효과]: "당신들이 맞는 고객" 확신 + 신혼 특수성 강조
```

### 이의 대응 (Monica 5가지, 신혼부부)

```
이의 1: "근데 우리 예산이 좀..."
→ "신혼부부들은 호텔+관광지 투어하면
   크루즈랑 비슷한 비용 나와요.
   대신 경험이 완전히 달라.
   호텔은 밤만 있고, 배는 48시간을 우리 둘이만."

이의 2: "시간이 없어서..."
→ "신혼부부들이 그래요.
   직장 때문에 길게 못 간다고.
   근데 5박이면?
   목요일 저녁 출발 → 화요일 오전 귀항.
   주말 연차 2일이면 가능해.
   그리고 배 위에선 '휴식'이라서 시간이 안 느껴져."

이의 3: "멀미가 걱정되는데..."
→ "신혼부부 300명 중 정말 멀미한 사람?
   딱 3-4명.
   그것도 '배멀미' 아니고 '속이 좀 안 좋다' 정도.
   약 먹으면 끝.
   신혼부부들은 이것 때문에 못 간다고 하는데,
   가본 사람들은 '아무것도 아니네'라고 해."

이의 4: "선택이 많아서 어떤 배를 고를지..."
→ "좋은 신호야.
   당신들이 이미 '기대'하고 있다는 뜻이니까.
   우리가 당신들한테 딱 맞는 배 3개만 골려드릴게.
   그럼 끝이야."

이의 5: "배우자가 아직 확신이 안 서..."
→ "당신이 '가면 좋겠다'고 생각했다는 게 이미 절반이야.
   배우자에게 보여줄 거 있어.
   당신 같은 신혼부부 후기 3개.
   이거 보면 '아, 이거 우리네'라고 해."
```

## 2️⃣ 40대 가족형 (직장인, 아이, 시간 부족)

### 페르소나 언어 스타일
- 현실적이고 실용적
- "일이 많아서"가 기본값
- "아이가 즐길 수 있을까"가 핵심 concern
- ROI(투자대비 효과) 민감

### 콜 스크립트 (PHASE 1-6)

```
PHASE 1: 오프닝
"40대 직장인 가족들 많이 생각하는 게 있어요.
'내 상황에 크루즈가 맞을까?'

직장도 많고, 아이 학원도 많고,
휴가도 길게 못 내잖아요.

근데 제가 확신해서 말씀드리는 게,
40대 직장인 가족들이
'가장 효율적으로 휴가를 보내는' 고객들이에요."

[포즈: 2초]

PHASE 2: 문제 + 공감 (Monica Step 2)
"40대 가족들은 보통 이렇게 말해요:
'호텔은 호텔인데, 짐 준비가 많고,
아이들 짐도 많고,
가면서 뭘 해야 할지 계획도 짜야 하고,
그 와중에 일 메일도 계속 와서...'

당신도 이런 고민, 있어요?"

[고객이 3-5줄 설명]

PHASE 3: 문제 + Implication (SPIN)
"맞아요. 맞아요.
40대라서 그런 거 있어요.

직장도 신경 쓰이고, 아이도 신경 쓰이고.
그런데 뭐가 문제냐면

휴가가 '휴가'가 아니라 '노동'이 되는 거죠.

호텔 여행:
- 짐 준비 (3시간)
- 운전 (4시간)
- 기항지 계획짜기
- 아이 핸들링 (음식, 이동)
- 밤: 숙소에서 도배워
결과: 휴가 동안 '당신'은 일해요

배 여행:
- 짐 준비 (30분, 룸서비스 있음)
- 운전 (0시간, 배가 움직임)
- 기항지 계획 (없음, 배가 정했음)
- 아이 핸들링 (키즈클럽이 해줌)
- 밤: 당신이 쉬어
결과: 휴가 동안 '당신'도 쉬어요

당신이 진짜 원하는 휴가가 뭐죠?
일하면서 노는 거? 아니면 쉬면서 노는 거?"

[포즈: 3초]

PHASE 4: 해결책
"40대 직장인 가족 고객들이
'크루즈 갈 걸 진작 알았어'라고 하는 이유가 이거예요.

아이들:
'학원 안 가도 되니까 좋아' (겨울방학/여름방학만 가면 됨)
'매일 다른 나라, 다른 바다' (경험 100배)

배우자:
'당신이 처음 웃는다' (일에서 벗어나니까)
'아이도 나도 함께 쉬어' (준비 0, 관리 0)

당신:
'아, 이게 휴가네' (일 메일 0, 일정관리 0)

이게 40대가 찾던 휴가 아닙니까?"

PHASE 5: 신뢰 형성
"제가 볼 때, 당신은
'호텔 휴가는 더 이상 못 해' 타입이에요.

왜냐면:
- 일이 많아서
- 시간이 짧아서
- 준비할 여유가 없어서

그럼 어떤 휴가를 원해요?
'무언가 준비 없이, 그냥 쉬는 휴가'

그게 바로 당신이 필요한 거야.

그리고 그 휴가? 
지금까지 당신은 못 본 거예요.

크루즈는 그걸 선물하는 거야."

[포즈: 2초]

PHASE 6: 클로징 (삼중 선택)
"당신이 '내 상황에 맞을까' 고민했는데,

애초에 당신 같은 직장인 가족이
'꼭 필요한' 고객이야.

다만, 어떤 배를 고르냐일 뿐.

아이가 많이 놀 배? (키즈클럽 200명)
아니면 조용하고 럭셔리한 배? (가족 10명)
아니면 그 중간?

셋 중에 고르면, 내가 나머지는 다 챙길게."

[포즈: 3초]
```

### SMS Day 0 (40대 가족 변형)
```
📲 "안녕하세요, OOO님!

방금 통화 때 '우리 일정에 크루즈가 맞을까?'
이렇게 물으셨어요.

YES. 당신 같은 40대 직장인 가족이
'제일' 필요한 고객들이에요.

왜냐면:
✅ 호텔은 준비/관리/일의 연속
✅ 배는 짐 30분 + 그 후 쉼
✅ 아이들도 자유 (키즈클럽)
✅ 당신도 자유 (일 메일 끄고)
✅ 5박만 해도 그동안 못 본 배우자 얼굴 봐요

당신이 진짜 원하는 게 '휴가'라면,
이게 유일한 방법이에요.

내일까지만 연락 주세요.

- 마비즈 크루즈팀"

[효과]: "당신 상황에 딱 맞음" + 효율성 강조
```

## 3️⃣ 중년부부형 (45-55세, 여유 시간)

### 페르소나 언어 스타일
- 차분하고 성숙한
- "이 나이대가 크루즈를 가야 할 이유"에 공감
- 건강/안전 관심 높음
- 럭셔리함 선호

### 핵심 메시지
```
"아이도 다 컸고, 직장도 안정됐으니까
이제 우리 시간이 있어."

당신 같은 중년부부들이
'크루즈의 진정한 고객'이에요.
왜냐면 당신들은 '품질'을 알고,
'여유'를 아는 시기니까요."
```

## 4️⃣ 고령층형 (60+, 건강 관심)

### 페르소나 언어 스타일
- 존중과 배려 강조
- "이 나이대에 가능한가"가 가장 큰 concern
- 건강/의료/안전 최우선
- 여유 있고 우아한 톤

### 핵심 메시지
```
"60대는 아직 '움직일 수 있는' 마지막 시기에요.
아직 건강하고, 아직 힘이 있고,
이제 '여유'도 생겼으니까요.

근데 5년 뒤면? 그럼 못 가요.
지금이 '가야 할' 시기예요."

당신 같은 60대 고객들이
'진정한 여행의 의미'를 아는 고객들이에요.
왜냐면 인생 경험이 많으니까."
```
