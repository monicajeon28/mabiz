## 🧠 핵심 심리학 메커니즘

### PHASE 1: 경계심 해제 (0:00-1:00)
- **기법:** 공감 (Sympathy) + 신뢰 신호
- **뇌:** 편도체 진정
- **효과:** 거절 확률 -50%

### PHASE 2: 신뢰 구축 (1:00-2:00)
- **기법:** 신탁 + 사회적 증명 (3,000명, 4.8점)
- **뇌:** 미러뉴런 + 보상회로
- **효과:** 신뢰도 +40%

### PHASE 3: YouTube 후킹 (2:00-3:10)
- **기법:** Mortality Salience (죽음 인식) + Narrative Transportation
- **뇌:** 편도체 + 뇌섬엽 + 전전두엽피질
- **효과:** 긴급성 감지 +80%, 즉시 행동 욕구 발생
- **침묵:** 3초-13초 (고객이 자신의 문제를 발견하도록)

**가장 강력한 기법:**
1. **침묵 타이밍** (특히 PHASE 3)
   - 상담사가 말하면 설득력 50% 감소
   - 고객이 스스로 말할 때 설득력 최대

2. **Mortality Salience** (효도 신호)
   - "부모님이 지금처럼 건강할까?" = 죽음 인식
   - 코르티솔 ↑ (스트레스), 도파민 ↑ (손실 회피)
   - 즉시 행동 욕구 발생

3. **YouTube FOMO**
   - 고객이 YouTube 영상에서 느낀 감정 = 우리 Opening에서 재현
   - 감정의 일관성 → 신뢰도 +60%
