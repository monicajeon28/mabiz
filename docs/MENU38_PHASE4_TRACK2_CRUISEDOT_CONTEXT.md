## 🚢 크루즈닷 실제 비즈니스 모델

### 1. 주요 판매 상품

| 상품군 | 상세 | 타겟 고객 | 가격대 |
|--------|------|----------|-------|
| **골드 플랜 A** | 아시아 크루즈 (일본·대만·홍콩) | 신혼부부, 가족 | 월 33,000원 + 크루즈비 |
| **골드 플랜 B** | 동남아 크루즈 (태국·베트남·싱가포르) | 중년 부부, 직장인 | 월 66,000원 + 크루즈비 |
| **골드 플랜 C** | 유럽·미국·알래스카 크루즈 | 고가 고객, 시니어 | 월 99,000원 + 크루즈비 |
| **KB 헬스케어** | 건강관리 서비스 (의료진 상담, 검진, 간병인) | 50세 이상 | 월 27,000원 |
| **스팟 크루즈** | 단일 크루즈 예약 (멤버십 미가입) | 즉흥적 고객 | 15~150만원 (상품별) |

### 2. 크루즈닷의 차별화 요소

✅ **평생 할인** (한 번 가입하면 계속 적용)  
✅ **유연성** (원하는 달만 결제, 언제든 중단 가능)  
✅ **무약정** (위약금 0원)  
✅ **우선 예약** (48시간 조기 오픈)  
✅ **매니저 동행** (단체 출발 시 무료)  
✅ **커뮤니티** (정보 공유, 함께 떠나는 느낌)

### 기존 구조 (Generic)
```
❌ Healthcare / Rental / Product (신규DB) / Product (부재중DB)
```

### 신규 구조 (크루즈닷)
```
✅ 크루즈 카테고리별:
   1. 아시아 크루즈 판매 (Plan A 중심)
   2. 동남아 크루즈 판매 (Plan B 중심)
   3. 유럽/미국 크루즈 판매 (Plan C 중심)
   4. 건강관리 애드온 (KB Healthcare)
   5. 추가 여행 서비스 (여행보험, 비자, 짐운반)

✅ 고객 세그먼트별:
   - 신규 DB (콜드 고객): 크루즈 경험 없는 사람
   - 부재중 DB (웜 고객): 크루즈 경험 있지만 1년 이상 미구매
   - 비회원 고객: 스팟 크루즈 구매자
```

## 📞 Track 2 v2.0: 크루즈 상품별 콜 스크립트

### Category 1: 아시아 크루즈 (Plan A) - 신규/웜 고객
**심리학**: 가까운 거리, 저렴한 가격 → **접근성 강조**  
**PASONA**: Simplicity (단순성) + Relativity (상대적 가치)

#### 신규 고객 (처음 크루즈하는 사람)
```
"안녕하세요! 크루즈닷 모니카입니다.

크루즈 한 번도 안 가봤는데,
일본·대만·홍콩은 가깝지 않나요?

우리 고객들 90%가
'아, 이렇게 편하다고?'
이렇게 말해요.

3박 4일, 비행기 4시간 + 크루즈로
일본 가는 게 얼마나 편한지
5분만 설명해드릴게요?"
```

#### 부재중 고객 (예전에 크루즈 다녀온 사람)
```
"안녕하세요 모니카님! 오래간만이에요 😊

작년 홍콩 크루즈 즐거웠나요?

올해는 우리가 '신규 선박' 가져왔거든요.
더 크고, 더 좋아진 시설,
더 저렴한 가격으로요.

작년에 쓰신 금액으로
올해는 더 좋은 선박 탈 수 있어요.

관심 있으신가요?"
```

---

### Category 2: 동남아 크루즈 (Plan B) - 신규/웜
**심리학**: 이국적인 매력, 프리미엄 경험 → **모험심 + 프리미엠 가치**  
**PASONA**: Narrative (내러티브) + Expectation (기대감)

#### 신규 고객
```
"안녕하세요! 크루즈닷 모니카입니다.

혹시 태국·베트남·싱가포르
다 한 번에 가본 적 있으세요?

매년 5,000명이 우리랑 함께
동남아 크루즈를 다녀가는데,

'평생 못 잊을 경험'이라고
다들 말해요.

그 이유가 뭘까요?
3분만 설명할게요."
```

#### 부재중 고객
```
"안녕하세요 모니카님! 

동남아 크루즈 다녀온 지도 2년이 됐네요.

올해는 '업그레이드 플랜'을 만들었는데,
지난해 티켓값으로
더 좋은 선실, 더 많은 기항지,
더 많은 경험을 드릴 수 있어요.

이번엔 가족 추천도 할 수 있을 정도로
만족할 거예요.

관심 있으신가요?"
```

---

### Category 3: 유럽/미국 크루즈 (Plan C) - 신규/웜
**심리학**: 프리미엄 경험, 버킷리스트 → **인생의 가치 + 손실회피**  
**PASONA**: Learning Value (배움) + Loss Aversion (손실회피)

#### 신규 고객
```
"안녕하세요! 크루즈닷 모니카입니다.

혹시 유럽이나 알래스카,
크루즈로 가본 적 있으신가요?

우리 고객 중에 '인생이 바뀌었다'
이렇게 말씀하시는 분들이 가장 많은 경험이
바로 유럽 크루즈예요.

왜 그럴까요?
그리고 생각보다 저렴할 수 있어요.

궁금하신 가요?"
```

#### 부재중 고객
```
"안녕하세요 모니카님!

작년에 알래스카 다녀오셨을 때,
'인생 최고의 경험'이라고 하셨잖아요?

올해는 '그 이상'의 경험을 준비했어요.
- 더 나은 선박
- 더 많은 기항지 (노르웨이, 핀란드까지!)
- 같은 가격 또는 더 저렴함

가족도 함께 가면 좋을 것 같은데,
관심 있으신가요?"
```

### Category 4: KB 헬스케어 애드온
**심리학**: 안심 + 미래 보장 → **Risk Removal**  
**PASONA**: Simplicity (간단함) + Authority (권위)

#### 신규 고객
```
"안녕하세요! 크루즈닷 모니카입니다.

올해는 크루즈 떠나기 전에
'건강체크'를 먼저 하는 게 트렌드예요.

KB생명 의료진과 함께
검진·상담·병원 예약까지
다 해드려요.

월 27,000원인데,
크루즈 다녀올 때 안심이 생겨요.

관심 있으신가요?"
```

#### 기존 고객에게 (크루즈 예약 후 제안)
```
"모니카님, 크루즈 티켓 확보했는데,

혹시 건강 체크하고 싶으신가요?
크루즈 떠나기 전에요.

KB생명 의료진이 맞춰줄게요.
예약·검진·상담 다 포함해서 월 27,000원.

여행 중 응급상황 대비도 되고요."
```

## 💬 SMS 시퀀스 재구성

### 기존 (Generic)
```
❌ Healthcare SMS (45% → 55%)
❌ Rental SMS (50% → 58%)
❌ Product New DB SMS (35% → 45%)
❌ Product Inactive DB SMS (65% → 75%)
```

### 신규 (크루즈 중심)
```
✅ 아시아 크루즈 신규 고객 SMS (Day 0-3)
   - Day 0: 가까운 거리, 간단한 크루즈 → "아, 이렇게 쉬워?"
   - Day 1: 가성비 (비행기X, 비용X) → "돈이 남네?"
   - Day 2: 우선예약 (48시간 조기) → "빨리 안 되나?"
   - Day 3: 할인 (10-20%) + 긴급성 → "이번 달 가자!"

✅ 동남아 크루즈 신규 고객 SMS (Day 0-3)
   - Day 0: 이국적 매력 (3개국) → "이렇게 다 갈 수 있어?"
   - Day 1: 프리미엄 경험 (5성 호텔) → "크루즈가 이 정도?"
   - Day 2: 후기 (매년 5,000명) → "이렇게 많이 가?"
   - Day 3: 한정 선실 + 할인 → "지금 아니면 언제?"

✅ 유럽/미국 크루즈 신규 고객 SMS (Day 0-3)
   - Day 0: 버킷리스트 (알래스카, 유럽) → "진짜 가능해?"
   - Day 1: 인생 경험 (고객 후기) → "이 정도면..."
   - Day 2: 다양한 기항지 → "세계 일주네?"
   - Day 3: 선실 한정 + 가격 기한 → "언제까지?"

✅ 헬스케어 애드온 SMS (Day 0-3)
   - Day 0: 출발 전 안심 (의료진 상담) → "미리 체크하자"
   - Day 1: 기존 고객 후기 (응급상황 대비) → "역시 있으면 좋네"
   - Day 2: KB신용도 (국내 신뢰사) → "믿을 수 있네"
   - Day 3: 즉시 등록 (크루즈 출발 전 필수) → "지금 등록하자"
```
