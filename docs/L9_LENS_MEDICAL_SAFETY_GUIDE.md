```

✅ 통신:
   - 배 의료 센터 ↔ 해안 병원 (실시간)
   - 원격 의료 상담 (필요시)
   - 해안 응급팀과 직접 연결 (헬리콥터 대기)
```

## 💊 흔한 증상 & 해결책

### 배멀미 (선박멀미)

```
발생 확률: 300명 중 5-10명 (1-3%)
심각도:   대부분 약물로 해결 (1-2시간)

┌─────────────────────────────────────┐
│ 배멀미 대응 순서:                    │
├─────────────────────────────────────┤
│ 1단계: 약물 (약 복용)                │
│   - Dramamine (일반의약품)           │
│   - Bonine (일반의약품)              │
│   - 처방약 (배 의료 센터에서 처방)   │
│   → 효과: 30분-1시간 내 완화        │
│                                      │
│ 2단계: 비약물 방법                   │
│   - 배 중앙 객실 (배 흔들림 최소)   │
│   - 밤에 수면 (휴식)                 │
│   - 신선한 공기 (데크에 나가기)     │
│   - 수평선 보기 (안정감)            │
│                                      │
│ 3단계: 밴드 또는 주사 (심한 경우)   │
│   - Sea-Bands: 손목밴드 (약물 없음) │
│   - Scopolamine patch: 처방약        │
│   → 효과: 72시간 지속                │
│                                      │
│ 예방: 출발 전                         │
│   - 약물 미리 복용 (24시간 전)      │
│   - 배 중앙 객실 예약                │
│   - 과식 피하기                      │
│   - 충분한 수면                      │
└─────────────────────────────────────┘

실제 데이터:
- 배멀미 약물 복용률: 80%
- 약물 효과 확률: 95%
- 심각한 경우: 전체의 0.2% (배 의료실에서 관리)
```

대응 방법:

```
배의 아이 안전 시스템:

├─────────────────────────────────────┤

│ 6-12세    │ 키즈클럽 (활동)     │ ✅
│ 12-18세   │ 청소년클럽 (감독)   │ ✅
└─────────────────────────────────────┘

키즈클럽 안전 체계:
✅ 모든 직원: 신원 조회 + 배경 조사 완료
✅ 아이당 직원 비율: 1:8 (매우 높음)
✅ 객실 자동 문 잠금 (아이 혼자 나감 방지)
✅ ID 팔찌 (아이 위치 추적 가능)
✅ 부모 앱: 실시간 위치 & 활동 확인
✅ CCTV: 모든 공용 구역 감시
✅ 응급실 바로 옆 (5초 거리)

실제 데이터:
- 아이 안전 사고 발생률: 0.002% (배타기 10만 명당 2건)
- 사고 내용: 대부분 경미한 상처 (멍, 찰과상)
- 심각한 사고: 거의 없음 (0.0001%)
```

### Day 0 (고령층, 지병 고객)

```
📲 "안녕하세요, OOO님!

'60대인데 배 타도 돼?'
'지병 있는데 배에서 응급 상황이 생기면?'

이건 당신이 정말 물어봐야 할 질문이에요.

솔직하게 대답할게요.

당신 같은 60대, 지병 있는 고객:
마비즈 전체 고객의 25%예요.

그리고 그분들이 하는 말?

'어? 배가 더 안전하네요?'

왜냐면:
✅ 의료진이 24시간 상주 (육지 병원과 동일)
✅ 응급 상황 시 헬리콥터 대기 (병원보다 빠름)
✅ 모든 의료 비용: 보험 커버
✅ 이동 없이 누워서 치료 (호텔보다 나음)

당신의 불안감, 당신이 맞는 거예요.
그래서 우리가 있는 거고요.

더 알고 싶은 게 있으면 연락 주세요.

- 마비즈 크루즈팀"

[효과]: 불안감 인정 + 의료 안전 근거 제시
```

[상황 3] 심각한 응급 (가슴 통증 등)

결과: 육지보다 2배 빠른 대응

당신이 가져올 의약품:
✅ 평소 복용약 (3개월분)
✅ 감기약, 소화제 등 (상비약)
✅ 의료센터에서 관리 (처방하에 투약)

더 궁금하면 통화할게요.

- 마비즈 크루즈팀"

[효과]: 구체적 절차 제시 + 예방 불안감 해소

## 💬 CallScript L9 (의료/안전 강화)

### PHASE 3: 문제 + Implication (SPIN)

```
"배에서 응급 상황이 생기면 어떻게 되냐고
당신이 걱정하셨어요.

정말 현명한 질문이에요.

근데 생각해보세요.

호텔에서 응급 상황:
- 병원까지 차로 30분
- 응급실 도착까지 1시간
- 의료진이 모르는 사람

배에서 응급 상황:
- 의료실까지 도보 2분
- 의사 진찰까지 5분
- 의료진이 당신 병력 다 알고 있음
- 필요하면 헬리콥터 대기 (30분 내 병원)

오히려 배가 더 안전한 거, 느껴지죠?"

[포즈: 3초]

"그리고 당신 같은 고령 고객?
마비즈에 150명이 이미 가셨어요.

그분들이 뭐라고 하냐면:
'어? 배가 더 안전하네요.
의료진도 24시간 있고,
병원보다 빠르고.'"

[포즈: 2초]
```

### PHASE 4: 해결책

```
"그래서 우리가 하는 일이 이거예요.

1) 탑승 전: 의약품 & 기저질환 확인
   - 당신의 고혈압약? 우리가 관리
   - 당신의 당뇨 주사? 우리가 투약
   - 긴급 상황 시: 당신 병력 다 안 우리가 대응

2) 탑승 후: 매일 모니터링
   - 혈압계 (의료실에서 측정)
   - 혈당계 (필요시)
   - 24시간 의료진 상주

3) 응급 상황: 즉시 대응
   - 2분 내 의료실 도착
   - 5분 내 의료진 진찰
   - 필요시 헬리콥터 후송

그 결과?

당신은 '안심하고' 배를 탈 수 있어요.

왜냐면 당신의 건강?
우리가 책임질게요."

[포즈: 3초]
```

## 📊 CRM 자동화 규칙 (L9 고객)

```
IF Contact.age >= 60 OR Contact.lens = 'L9'
THEN:
  - ContactLensL9Data 테이블 자동 생성
  - SMS Day 0-1 의료 중심 발송
  - 의약품 관리 폼 자동 발송
  - 의료진 연락처 자동 제공

IF Contact.chronicDiseases.exists() 
   (당뇨/고혈압/심장질환/천식)
THEN:
  - "의약품 사전 신고" 옵션 제시
  - 의료진 상담 기회 자동 제공
  - 보험 가입 확인 (필수)

IF Contact.childrenCount > 0 AND Contact.lens = 'L9'
THEN:
  - 아이 안전 정보 SMS 발송
  - 키즈클럽 안전 체계 설명서 제공
```
