## 📊 L8 렌즈 고객 프로필

### 타겟 고객 정의

```
Primary Target:
├─ 이미 크루즈 경험 있는 고객 (부재중 DB)
├─ 멤버십 휴지 또는 취소한 고객
├─ "스팟으로 필요할 때마다" 생각하는 고객
└─ 연간 1-2회 여행 고객

Key Insight:
- 크루즈 경험으로 신뢰도 높음
- 가격 거부감 없음 (이미 탔으니)
- 하지만 "매번 새로 돈을 들어야 한다"는 부담
- 멤버 재활성화 시 "평생 할인 혜택" 미인식
```

### 핵심 Pain Point

| 요소 | 설명 | L8 대응 |
|------|------|--------|
| **할인가치 미인식** | "멤버하나 안 하나 상품비는 같지 않나?" | "5년이면 누적 절약액 본전" 계산 |
| **반복 비용 부담** | 매번 새로 상품비를 내야 한다는 느낌 | "스팟 450만원 vs 멤버 200만원" 비교 |

```

혹시 저희를 기억하세요?

아! 정말요?

만족도가 아주 높으셨거든요.
```

```
"지난번 크루즈가 언제였어요?"

"그렇군요.

뭐가 제일 컸어요?"
```

### Phase 2: 욕망 증폭 - "평생 할인의 누적 가치" (2-6분)

**목표**: 단순 "월 33k"가 아니라 "평생 절약액 수백만원"임을 실감  
**심리 원리**: Lifetime Value Framing

```
"그럼 역산해봅시다.

만약 지난 2년 동안 
해마다 1-2회씩 다녔다면?

[시나리오 1: 연 1회]
- Year 1: 350만원 (멤버할인 50%) vs 스팟 450만원 → 절약 100만원
- Year 2: 350만원 vs 스팟 450만원 → 절약 100만원
━━━━━━━━━━━━━━━━
누적 절약: 200만원

[시나리오 2: 연 2회]
- Year 1: 700만원 (멤버할인 합산) vs 스팟 900만원 → 절약 200만원
- Year 2: 700만원 vs 스팟 900만원 → 절약 200만원
━━━━━━━━━━━━━━━━
누적 절약: 400만원

'아, 멤버를 왜 안 했지?'
이 정도 수준이 나와요.

그런데요, 앞으로는 다를 수 있어요.

○○님이 지금부터 5년 더 다닌다면
어떻게 될까요?

5년 연 2회 기준:
━━━━━━━━━━━━━━━━
Year 1-2: 200만원 × 2 = 400만원
Year 3: 240만원 (VIP 업그레이드)
Year 4-5: 240만원 × 2 = 480만원
━━━━━━━━━━━━━━━━
5년 누적 절약: 1,120만원

멤버 비용 (월 33k × 60개월): 198만원

결국 5년 누적하면
'멤버 비용을 절약액으로 회수'하고도 남아요.

이후론 평생 50% 할인.

○○님이 70세까지 다니면
30년 누적 절약: 6,000만원 이상

멤버비용 vs 절약액?
비교가 안 돼요."
```

### Phase 3: 상품 설명 - 신규 상품 & 업그레이드 경로 (6-9분)

**목표**: "새로운 것도 있고, 더 좋아진 게 많다"는 확신

```
"지난번 이후 새로운 게 많아요.

[새 선박 - 신박스]
- 싱가포르-방콕-푸껫 (7박 8일)
- 객실이 기존보다 20% 더 넓음
- 스파 시설이 3배
- 1인 기준: 200-280만원
- 멤버 기준: 100-140만원

[기존 인기 상품 가격 인하]
- 도쿄 크루즈: 기존 120만원 → 지금 80만원
- 멤버할인 50% 적용 시: 40만원

[VIP 멤버 제도 신설]
- 월 50,000원 (기본 33k + 추가 17k)
- 할인율: 50-60%
- VIP라운지 + 우선예약
- 연 3회 이상 다니면 기본 회원보다 저렴

어느 게 제일 끌려요?"
```

### Phase 4: 클로징 - 재활성화 + 신규상품 예약

**목표**: 멤버 재활성화 + 신규 상품 구체적 예약

```
"그럼 이렇게 할까요?

[Option A] 멤버만 먼저
- 이달부터 멤버 재활성화
- 상품은 준비되면 예약
- 복귀 보너스: 첫달 50% 할인

[Option B] 멤버 + 상품 동시
- 9월 도쿄 또는 신규 싱가포르
- 분할 3개월, 월 80-100만원

[Option C] VIP로 바로 시작
- 월 50,000원
- VIP라운지, 우선예약

어느 게 좋을 것 같아요?"

[고객 선택 유도 후]

"그럼 정확히 정리할게요.

1) 오늘: 멤버 재활성화 신청서 톡
2) 내일: 상품 정보 상세 안내
3) 모레: 우대 혜택 확인

이번 복귀 기념으로
세 가지를 드릴게요:

① 첫달 멤버: 26,000원 (기존 33,000원)
② 첫 상품 구매 시: 추가 10% 할인
③ VIP 업그레이드 할 때: 첫 3개월은 기본가

정말 오랫동안 자리를 비우셨는데,
다시 만나게 되어 기뻐요.

크루즈로 만나요!"
```
