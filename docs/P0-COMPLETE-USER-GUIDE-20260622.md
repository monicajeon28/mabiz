```
┌────────────────────────────────────────────────────────┐
│ 1️⃣ 누가 발송하나?                                     │
├────────────────────────────────────────────────────────┤
│ 관리자 ─────→ 누구한테나 발송 가능 ✅                │
│ 대리점 ─────→ 자신의 판매원 고객한테만 ✅            │
│ 판매원 ─────→ 자신의 고객한테만 ✅                 │

└────────────────────────────────────────────────────────┘
         ↓
┌────────────────────────────────────────────────────────┐
│ 2️⃣ 고객 목록 필터링                                   │
├────────────────────────────────────────────────────────┤
│ DB에서 쿼리:                                         │
│ "이 사람한테 발송 권한 있는 고객들 찾기"            │
│                                                     │
│ 조건:                                               │
│ - 같은 조직                                         │
│ - 만약 판매원이면: 자신이 담당한 고객               │
│ - 만약 대리점이면: 자신 조직의 고객들               │
└────────────────────────────────────────────────────────┘
         ↓
┌────────────────────────────────────────────────────────┐
│ 3️⃣ 고객 목록 표시                                     │
├────────────────────────────────────────────────────────┤
│ UI에 자동 표시:                                      │
│ ✅ 이순신 (담당: 나)                                │
│ ✅ 강감찬 (담당: 나)                                │

└────────────────────────────────────────────────────────┘
         ↓
┌────────────────────────────────────────────────────────┐
│ 4️⃣ 발송 + 기록 저장                                   │
├────────────────────────────────────────────────────────┤
│ SMS 발송 ──→ 고객한테 전달                          │
│                ↓                                     │
│ 기록 저장 ──→ SmsLog 테이블에 저장                 │
│ (발송자, 고객, 시간, 내용 등)                      │
└────────────────────────────────────────────────────────┘

## 🎯 초등학생 수준 설명 (아이도 이해할 수 있게)

### **예시 1: 수익계산기**

```
🧒 아이의 말: "수익계산기는 뭐예요?"

👨‍🏫 설명:
"학교의 돼지저금통이라고 생각해봐.

관리자 = 학교 교장 선생님
→ 모든 돼지저금통을 열어볼 수 있어!

대리점장 = 학급 담임 선생님
→ 우리 반 돼지저금통만 열어볼 수 있어! (다른 반은 못 봄)

판매원 = 학생
→ 자신의 돼지저금통만 열어볼 수 있어!

교장 선생님이 "이번엔 대리점장도 돈을 못 세게 할래"라고 하면
→ 대리점장도 돈을 못 봐! (비밀이야)"
```

### **예시 2: 문자 발송 권한**

```
🧒 아이의 말: "왜 판매원은 전체 학생한테 문자를 못 보내요?"

👨‍🏫 설명:
"좋은 질문이야!

판매원은 자기 고객들만 담당하는 사람이야.
만약 모든 학생한테 문자를 보낼 수 있다면?
→ 잘못된 메시지를 실수로 보낼 수도 있지!

그래서 규칙을 정한 거야:
- 관리자: 모든 학생한테 보낼 수 있어 (책임감 있으니까)
- 대리점: 자신의 판매원 학생들한테만 (감시 가능)
- 판매원: 자신의 고객한테만 (실수 방지)

이렇게 하면 안전해!"
```

### **예시 3: 고객 삭제 불가**

```
🧒 아이의 말: "왜 판매원은 고객을 지울 수 없어요?"

👨‍🏫 설명:
"학교의 학급부장이라고 생각해봐.

학급부장이 실수로 학생 정보를 다 지워버리면?
→ 선생님이 정말 화날 거야!

그래서:
- 교장 선생님(관리자)만 지울 수 있어
- 학급 담임(대리점)도 지울 수 있어
- 학급부장(판매원)은 못 지워!

이렇게 하면 중요한 정보가 지워지지 않아!"
