# 세일즈봇 QnA PASONA Solution 통합 설계

**작성**: 2026-05-19  
**목표**: 콜 스크립트의 7가지 Solution을 QnA 카테고리로 체계화하여 세일즈봇 자동화 및 담당자 참고 자료 제공

---

## 📊 핵심 개념

### 기존 세일즈봇 QnA (문제)
- 상품정보 + 예약절차 + 고객Q&A로만 분류
- 콜 스크립트와 독립적
- 담당자가 콜 중 참고할 수 있는 Solution 기반 제안 없음

### 신규 세일즈봇 QnA (솔루션)
- **4가지 신규 카테고리 추가**: 욕구발굴, 상품스토리, 신뢰감형성, 배움경험
- **콜 스크립트 5 Step과 매핑**: 각 Step에서 필요한 Solution 제시
- **자동화 + 수동 참고 병행**: 세일즈봇 자동 답변 + 담당자 실시간 제안

---

## 🎯 4가지 신규 QnA 카테고리

### 카테고리 1: 욕구 발굴 (Solution 1, 5)
**목적**: 고객이 여행을 망설일 때, 손실회피와 기대감을 자극하는 Q&A

**기본 설명**
- **Solution 1 (손실회피)**: "지금 하지 않으면 손실" → 가격 인상, 자리 부족 강조
- **Solution 5 (기대감)**: "이렇게 좋을 텐데..." → 감정적 이미지 그리기

**Q&A 예시** (총 15개, 여기서는 5개 샘플)

```
Q1: "지금 예약해야 하는 이유가 뭐에요?"
A: "5월까지만 특가이고, 6월부터 20% 인상됩니다. 
   + 이 크루즈는 평생 잊을 수 없는 경험이 될 거예요.
   (Solution 1: 손실회피 + Solution 5: 기대감)"

Q2: "언제까지 예약을 결정해야 하나요?"
A: "매월 남은 자리가 달라지는데, 현재 특가는 5월까지만입니다.
   지금 결정하시면 최저가로 즐길 수 있어요.
   (Solution 1: 희소성 강조)"

Q3: "크루즈 여행이 뭐가 좋아요?"
A: "배 위에서 해 지는 것을 보는데, 옆에 가족이 있고, 
   대양이 발 아래로 펼쳐지고... 이런 경험은 일상에서 불가능해요.
   (Solution 5: 감정적 스토리텔링)"

Q4: "비행기와 뭐가 다른가요?"
A: "비행기는 12시간 고통인데, 크루즈는 12시간 여행입니다.
   탈것이 아니라 목적지 자체가 크루즈예요.
   (Solution 1: 현재 고통 vs 미래 행복)"

Q5: "지금 이걸 못 가면 언제 갈 건가요?"
A: "음... 나이가 들수록 체력이 떨어지잖아요.
   지금이 최고의 시간입니다.
   (Solution 1: 시간 손실 회피)"
```

**자동 발송 시나리오**
- 고객이 "생각해볼게요" → 세일즈봇이 Q1-3 자동 발송 (이메일)
- 담당자가 "가격 물어봄" → 세일즈봇이 Q2, Q4 제안
- 담당자가 콜 Step 2 진입 → 세일즈봇이 Q3, Q5 실시간 제안

---

### 카테고리 2: 상품 스토리 (Solution 2, 3, 4)
**목적**: 상품을 구체적으로 설명하되, 간단함과 효율성을 강조하는 Q&A

**기본 설명**
- **Solution 2 (방법 알려주기)**: 일정을 단계별로 구체적 스토리로 전개
- **Solution 3 (간단함)**: "3가지면 끝" "일정은 배가 다 짜줌"
- **Solution 4 (가치 제안)**: 가격 대비 포함되는 모든 것 강조

**Q&A 예시** (총 20개, 여기서는 5개 샘플)

```
Q6: "3박 일정이 어떻게 되나요?"
A: "Day 1) 출항 (환영 칵테일) 
    Day 2) 해변 목적지 (수상스포츠)
    Day 3) 귀항 (갈라디너)
    정말 간단해요. 배 타면 모든 게 준비되어 있어요.
    (Solution 2: 구체적 스토리 + Solution 3: 간단함)"

Q7: "짐은 어떻게 가져가요?"
A: "수하물을 객실에 두고 내려요. 하루가 안 되니까 짐걱정 없어요.
    여행이 진짜 편한 이유가 이거예요.
    (Solution 3: 진입장벽 제거)"

Q8: "가격이 비싸지 않나요?"
A: "3박에 50-70만원대인데, 다음을 포함합니다:
    ✓ 숙박 3박
    ✓ 모든 식사 (조식/점심/저녁 + 24시간 간식)
    ✓ 선상 엔터테인먼트 (쇼, 야외극장, 무도회)
    ✓ 해변 활동 비용
    가성비로는 한국에서 최고입니다.
    (Solution 4: 가치 제안)"

Q9: "혼자 가도 괜찮아요?"
A: "크루즈는 그룹 여행이 아니라 개인 여행 위의 공유공간이에요.
    객실은 혼자, 식당/바/라운지는 함께.
    가장 편한 혼자 여행 방식입니다.
    (Solution 3: 불안감 제거 + Solution 2: 실제 경험 설명)"

Q10: "아이들이 있어도 괜찮아요?"
A: "아이 풀장, 아이 클럽, 아이 전용 프로그램이 있어요.
    부모는 스파/카지노, 아이는 안전한 환경에서 놀아요.
    가족 만족도가 가장 높은 여행입니다.
    (Solution 4: 모든 세그먼트에 가치 제안)"
```

**자동 발송 시나리오**
- 콜 Step 3 진입 (해결책 제시) → 세일즈봇이 Q6-10 실시간 제안
- 고객이 "가격 다시 묻기" → Q8 재발송
- 담당자가 "혼자 여행" 언급 → Q9 제안

### 카테고리 3: 신뢰감 형성 (Solution 6)
**목적**: 고객이 불안해할 때, 배타성과 특별함을 강조하는 Q&A

**기본 설명**
- **Solution 6**: "모든 사람에게 이 제안을 하지 않아요"
- "당신은 특별한 고객이라서 직접 연락드렸습니다"
- 신뢰감 + 우월감 제공

**Q&A 예시** (총 10개, 여기서는 4개 샘플)

```
Q11: "왜 저한테 직접 전화를 주셨어요?"
A: "모니카님 같은 특별한 고객분들께만 이 기획을 말씀드립니다.
    여행 경험이 있고, 정말 좋은 경험을 원하는 분들을 위해서요.
    (Solution 6: 배타성 + 우월감)"

Q12: "이 상품은 누구를 위한 거예요?"
A: "정말 특별한 경험을 원하는 분들만을 위해 기획했습니다.
    저가 투어와는 완전히 다릅니다.
    (Solution 6: 선별된 감각)"

Q13: "왜 이 회사여야 하나요?"
A: "크루즈닷은 고객을 선택합니다. 
    모든 예약을 받지 않아요. 함께할 고객을 선택해요.
    그렇기 때문에 서비스가 다릅니다.
    (Solution 6: 회사의 선별성)"

Q14: "다른 여행사도 같은 상품을 판는데?"
A: "같은 배를 타더라도, 어떤 담당자를 만나느냐가 전부입니다.
    크루즈닷의 담당자들은 스스로를 콘시에르주라고 생각합니다.
    (Solution 6: 사람의 가치)"
```

**자동 발송 시나리오**
- 콜 오프닝 (Step 1) → 세일즈봇이 Q11, Q12 제안
- 고객이 "왜 나한테?" → Q11 강조
- 가격 이견 → Q13, Q14로 차별성 강조

### 카테고리 4: 배움과 경험 (Solution 7)
**목적**: 고객의 감정적 연결을 강화, 경험의 가치 강조하는 Q&A

**기본 설명**
- **Solution 7**: "이 여행은 단순한 관광이 아니라 인생의 의미"
- 세상 보기, 자신감, 관계 재확인

**Q&A 예시** (총 15개, 여기서는 4개 샘플)

```
Q15: "이 여행에서 뭘 배울 수 있어요?"
A: "다른 문화, 새로운 사람들과의 만남, 자신감과 여유.
    그리고 가족/친구와의 관계가 새로워집니다.
    (Solution 7: 정체성 + 의미)"

Q16: "돈을 쓸 가치가 있나요?"
A: "인생에서 최고의 경험 중 하나가 될 겁니다.
    은퇴 후 후회하지 않으려면 지금 해야 해요.
    (Solution 7: 인생 의미)"

Q17: "왜 크루즈를 권하세요?"
A: "크루즈는 여행이 아니라 '삶의 질 향상'입니다.
    속도를 내려놓고, 대양과 사람들을 느껴봐요.
    (Solution 7: 의미 추구)"

Q18: "경험 많은 분들이 크루즈를 왜 가나요?"
A: "숙련된 여행자일수록 크루즈를 선택합니다.
    왜냐하면 평범한 여행에 물렸기 때문이에요.
    크루즈는 특별함 그자체입니다.
    (Solution 7: 지혜로운 선택)"
```

**자동 발송 시나리오**
- 고객이 "생각해볼게요" → Q15, Q16 자동 발송 (감정적 접근)
- 콜 클로징 (Step 5) → 세일즈봇이 Q17, Q18 실시간 제안
- 고객이 "나이가..." 언급 → Q16 강조

## 🔄 콜 스크립트 5 Step과 QnA 매핑

```
┌─────────────────────────────────────────────────────────────┐
│ STEP 1: 오프닝 (0-2분) → Solution 6: 신뢰감 카테고리 활성화    │
│ Q11-14: "왜 저한테?", "누구를 위한?", "왜 이 회사?"           │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ STEP 2: 욕구 발굴 (2-5분) → 욕구발굴 카테고리 활성화          │
│ Q1-5: "왜 지금?", "언제까지?", "뭐가 좋아?", "비행기와 뭐가?" │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ STEP 3: 해결책 제시 (5-10분) → 상품스토리 카테고리 활성화     │
│ Q6-10: "일정은?", "짐은?", "가격은?", "혼자는?", "아이는?"    │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ STEP 4: 가격 & 긴급성 (10-13분) → 욕구발굴+신뢰감 재활용       │
│ Q2, Q8 (가격), Q13, Q14 (차별성)                              │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ STEP 5: 클로징 (13-15분) → 배움경험 카테고리 활성화           │
│ Q15-18: "배울 점?", "가치 있나?", "왜 크루즈?", "경험자는?"   │
└─────────────────────────────────────────────────────────────┘
```

## 📱 세일즈봇 통합 아키텍처

### 옵션 1: 실시간 담당자 참고 (권장)
```
담당자가 콜 중 "욕구 발굴" 단계 진입
  ↓
세일즈봇 앱에서 "욕구 발굴" 탭 터치
  ↓
Q1-5가 카드 형태로 나타남
  ↓
담당자가 Q3 ("크루즈 여행이 뭐가 좋아요?")의 답변을 참고하며 고객에게 설명
  ↓
고객: "오, 그렇네요!"
  ↓
담당자: "Step 3로 이동" → 세일즈봇 "상품 스토리" 탭 활성화
```

**장점**: 담당자의 콜 스크립트 형식화 방지, 자연스러운 대화 유지, 실시간 가이드

### 옵션 2: 자동 이메일 발송
```
고객이 콜 후 "생각해볼게요"
  ↓
CRM에 "Follow-up Status: Thinking" 저장
  ↓
24시간 후 자동 이메일 발송 (Q1-3: 욕구발굴)
  ↓
2일 후 자동 이메일 발송 (Q15-16: 배움경험)
  ↓
고객이 이메일 링크 클릭 → CRM 재접촉 기회 생성
```

**장점**: 높은 재접촉율 (이메일 오픈율 35%), 자동화 효율

### 옵션 3: 세일즈봇 자동 답변 모드
```
고객이 세일즈봇 채팅에 "왜 지금 예약해야 하나요?" 입력
  ↓
NLU가 의도 분류: "욕구발굴" 카테고리
  ↓
세일즈봇이 Q1 답변 자동 출력
  ↓
고객: "다음 질문..." → Q2 자동 답변
  ↓
고객: "담당자랑 통화하고 싶어요" → 자동으로 담당자 연결
```

**장점**: 24/7 답변, 담당자 시간 절약, 고객 자율성 존중

## 🎲 A/B 테스트 계획

### 그룹 A: 기존 콜 스크립트 (Control)
- 담당자가 수동으로 설득
- 세일즈봇 QnA 없음
- 측정 지표: 전환율, 통화 길이, 고객 만족도

### 그룹 B: 콜 스크립트 v2.0 + 세일즈봇 QnA (Test)
- 담당자가 Step별로 세일즈봇 QnA 참고
- 자동 이메일 발송 활성화
- 측정 지표: 전환율, 통화 길이, 고객 만족도

### 기간
- 2주 (담당자 2명 × 콜 30건씩 = 120건)
- 데이터: 콜 길이, 전환율, 고객 피드백

### 예상 결과
- **전환율**: Group A 25% → Group B 32% (+7%)
- **통화 길이**: Group A 18분 → Group B 15분 (-3분, 효율 향상)
- **고객 만족도**: Group A 3.8/5 → Group B 4.3/5 (+0.5점)
