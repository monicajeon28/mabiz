## 🎯 L3 렌즈 3가지 특징 (최종 확정)

| # | 특징 | 고객 발화 | 심리 근원 | 우리의 대응 원리 |

## 📞 L3 콜 스크립트 (6단계, 15분)

### **기본 원칙**
✅ **호텔 경험 인정 → 구조적 차이 시각화 → 리조트 개념 → 이동의 가치 → 프리미엄 포지셔닝 → 욕망 자극**  
✅ **"호텔과 다르다" 주장 금지, 대신 "호텔+α 구조"로 설명**  
✅ **이미지/스토리 활용** (L1, L2보다 시각화 중요)  
✅ **감정적 호소** (배에서의 일몰, 새로운 나라 깨어남)  

---

### **PHASE 1: 오프닝 (0-30초)**
**목표**: 호텔 경험 인정 + 새로운 각도 제시

```
고객이 "호텔이랑 뭐가 달라요?"라고 말할 때:

"아, 정말 좋은 질문이에요!
당신은 호텔 여행을 많이 다니셨어요? (고객이 "네"라고 답할 확률 90%)

그럼 당신의 경험이 정확해요.
호텔 여행은 정말 좋은 여행이죠.

근데... 혹시 생각해본 적 있어요?
호텔의 모든 편안함을 유지하면서,
'매일 다른 나라를 여행'하는 방식이요?"

[심리학]: 호텔 경험 인정 + 새로운 가능성 제시
```

---

### **PHASE 2: 문제 발견 (30초-2분)**
**목표**: 호텔 여행의 한계 자각하게 유도

```
"당신이 호텔 여행할 때,
보통 한 도시에 며칠을 있어요?"

→ 고객이 "5박" 등 답함

"그럼 그 5박 동안, 
매일 같은 호텔로 돌아왔죠?"

→ 고객이 "응"

"그리고 다음 나라를 가려면,
짐을 싸고, 비행기 타고, 새 호텔에 체크인하는
번거로움이 있었죠?"

→ 고객이 "응, 진짜 피곤했어"

"그렇다면... 혹시 이런 생각을 해본 적 있어요?
'호텔의 편안함은 있으면서,
매일 자동으로 다른 나라로 옮겨가는 여행이 있다면?'"

[심리학]: SPIN의 Situation/Problem + 호텔 여행의 한계 자각
```

---

### **PHASE 3: 증폭 (2-5분)**
**목표**: 호텔 기준의 "한계"를 구체화

```
"생각해 보세요.
호텔 여행의 문제점:
- 한 도시에만 있어서, 5박 후엔 지겨워져요
- 다른 나라 가려면, 이동과 체크인 반복
- 이동 시간에 여행 시간이 낭비돼요
- 저녁이 되면 호텔 근처에만 다닐 수밖에 없어요

그래서 많은 분들이 "여행은 피곤해"라고 해요."

→ 고객의 아픔을 언어화

"만약 이 모든 번거로움이 없다면?
당신은 얼마나 더 휴식이 될까요?"

[심리학]: Loss Aversion + Opportunity Cost (기회 손실)
```

---

### **PHASE 4: 리조트 개념 소개 (5-8분)**
**목표**: "배 = 리조트" 시각화

```
"자, 이렇게 생각해 보세요.

호텔 여행은:
- 침실: 호텔 방 (1개)
- 거주지: 침실 + 로비 + 카페 (3개 공간)
- 저녁: 호텔 밖으로 나가야 함

크루즈는:
- 침실: 프리미엄 선실 (호텔 이상)
- 거주지: 침실 + 스파 + 극장 + 풀장 + 카지노 + 10개 레스토랑 + 10개 바 + 피트니스 + 라운지... (50개 이상 공간!)
- 저녁: 배 위의 극장에서 브로드웨이 쇼 봐요. 나갈 필요 없음.

당신의 실제 거주지 크기:
- 호텔: 20평 (침실)
- 크루즈: 배 전체 (침실은 자는 곳만)

보이세요? 
당신이 '사용하는 공간'이 완전 달라요."

[이미지 보여주기: 호텔방 vs 선실 vs 배 전체]

[심리학]: Spatial Value Recognition + Visualization
```

---

### **PHASE 5: 이동의 가치 (8-12분)**
**목표**: "배가 이동한다"는 경험의 가치

```
"그리고 가장 중요한 차이:

호텔은... 당신이 움직여야 해요.
'내가 짐을 싸서, 내가 이동해야 해'
그 과정이 피곤하죠.

크루즈는... 배가 당신을 움직여요.
당신은 자는 동안,
배가 당신을 다음 나라로 옮겨요.

아침에 눈을 떠봤어요.
어제까지 있던 이탈리아는 사라지고,
지금은 그리스 산토리니 항구.

'어? 벌써 그리스야?'

당신은 선실에서 그냥 일어나서,
갑판에 나가면 신선한 그리스 공기.

짐을 안 싸도, 체크인을 안 해도,
당신은 이미 새로운 나라에 있어요.

그게 크루즈예요."

[이미지/영상: 배에서의 일몰, 다른 항구의 아침, 갑판 풍경]

[심리학]: Narrative Transportation + Emotional Connection + Movement Value
```

---

### **PHASE 6: 클로징 (12-15분)**
**목표**: 프리미엄 포지셔닝 + 욕망 자극

```
"그래서 사실, 
호텔 여행과 크루즈는
'다른 등급의 여행'이에요.

호텔 여행: 깊이 있는 여행
크루즈: 넓고 편한 여행

당신이 지금까지 한 여행은
'한 도시의 깊이를 파는 여행'이었어요.

이제 새로운 경험을 해보고 싶지 않아요?
'5개 나라를 편안하게 경험하는 여행'

당신은 선실에서 깨어나고,
갑판에서 커피를 마시고,
스파에서 휴식하고,
저녁이 되면 극장에서 브로드웨이 쇼를 봐요.

그 모든 경험을 하면서,
배가 당신을 세계로 옮겨줘요.

당신이 그런 여행을 할 자격이 있다고 생각해요."

(구체적 다음 약속)

"혹시 관심 생겼어요?"

[심리학]:
- Prestige Positioning (다른 등급)
- Emotional Connection (자격감, 존중)
- Anticipation (기대감)
```

## 💬 L3 주요 이의 대응

### **이의 #1: "패키지가 더 저렴한데..."**

```
[Step 1-3] 인정+감사+공감

[Step 4-5] 원인 파악:
"가격 비교는 당연하죠. 
근데 뭘 비교하는지가 중요해요."

[Step 6-7] 재프레이밍:
"패키지 $3000: 침실 1개, 정해진 일정, 1개 레스토랑
크루즈 $7000: 침실 1개 + 배 전체, 자유로운 일정, 10개 레스토랑"

가격 차이 = 당신이 사용하는 '공간의 크기'와 '자유도'의 차이"

[Step 8-10] 사례:
"생각해 보세요. 호텔에서는 저녁에 뭐 해요?
호텔 근처 레스토랑에 가거나, 호텔 바에 앉아있죠.

크루즈는? 극장, 라이브음악, 카지노, 라운지...
밤새 선택할 게 너무 많아요.

그 자유도를 $4000에 사는 거라고 생각하면,
비싼 게 아니라, 굉장히 싼 거예요."
```

### **이의 #2: "호텔이 더 편하지 않을까?"**

```
[Step 1-3] 인정+감사+공감

[Step 4-5] 원인 파악:
"호텔이 편하다는 거, 맞아요.
근데 '편함의 정의'를 다시 생각해 볼까요?"

[Step 6-7] 재프레이밍:
"호텔의 편함: 한 침대, 안정감
크루즈의 편함: 짐 안 싸도 돼, 이동 안 해도 돼, 밤새 활동

호텔은 '정신적 안정감'
크루즈는 '신체적 편안함 + 정신적 새로움'"

[Step 8-10] 시각화:
"호텔에서는 매일 아침,
'오늘 또 이 도시구나'라는 마음에 안주해요.

크루즈에서는 매일 아침,
'어제는 몰디브, 오늘은 스리랑카?'
이런 기대감과 새로움으로 깨어나요.

어느 게 더 편하고 싶으신가요?
'같은 것의 반복' vs '다양함의 편안'?"
```

### **이의 #3: "배에서 멀미는 안 할까?"**

```
[Step 1-3] 인정+감사+공감

[Step 4-5] 원인 파악:
"당연한 우려죠.
배 = 흔들림 이라는 이미지가 있으니까요."

[Step 6-7] 재프레이밍:
"현대 크루즈는 안정화 기술(Stabilization)이 완벽해요.
비행기보다 승차감이 부드러워요."

[Step 8] 증명:
"우리 고객 중 70대 할머니, 5살 아이들도 다 타요.
올해 287명 중 멀미 호소? 거의 없어요."

[Step 9-10] 대안:
"만약 정말 민감하면,
선실을 배 중앙(가장 안정적인 부분)에 배치해 드려요.
그럼 거의 흔들림을 못 느껴요."
```
