# 영업 도구함 빠른 참조 (Quick Reference) 🚀

## 📍 5가지 메인 탭

| 탭 | 아이콘 | 설명 | 대상 사용자 | 평균 사용시간 |
|----|--------|------|-----------|-----------|
| **대시보드** | 📊 | AI 추천 도구 + 자주 쓰는 도구 | 모든 영업사 | 1-2분 |
| **상품교육** | 📚 | 5가지 크루즈 상품 기본정보 | 신입, 상품 약한 영업사 | 5-10분 |
| **콜스크립트** | 🎤 | 페르소나별 고객 대응 스크립트 | 통화 중, 고객 약한 부분 | 1-3분 |
| **플레이북** | 📖 | 8가지 상황별 대응 가이드 (기존) | 이의 대응, 성공사례 학습 | 3-5분 |
| **콜분석** | 🔊 | 통화 녹음 AI 피드백 (기존) | 자기계발, 성약 분석 | 2-5분 |

## 🗺️ 상품 교육 (5가지)

### 1️⃣ 부산출도착 🏴‍☠️
- **특징:** 국내 출발 (비행기 불필요)
- **대상:** 직장인, 단기 휴가 고객
- **추천 이의대응:** "시간이 없어요" → "부산에서 출발하니까 가까워요"

### 2️⃣ 일본크루즈 🗾
- **특징:** 4개 항구, 6박 7일
- **대상:** 문화 관광 고객, 신혼부부
- **추천 이의대응:** "너무 길어요" → "일본을 제대로 느끼려면 6박 정도 필요해요"

### 3️⃣ 동남아크루즈 🌴
- **특징:** 열대 리조트, 싱가포르/태국
- **대상:** 은퇴층, 휴가 많은 직장인
- **추천 이의대응:** "비수기 가능?** → "12월-2월이 최고예요"

### 4️⃣ 상하이크루즈 🏮
- **특징:** 중국 문화, 2박 3일
- **대상:** 중국 관심 고객, 신혼
- **추천 이의대응:** "위험한 건 아닌가요?" → "매년 30만명 이상 방문해요"

### 5️⃣ 알래스카크루즈 ❄️
- **특징:** 빙하 + 야생동물, 백야 경험
- **대상:** 자연 애호가, 프리미엄 고객
- **추천 이의대응:** "비싼데요?" → "평생 한 번 경험할 수 있는 기회예요"

## 👥 콜 스크립트 (5가지 페르소나)

### 1️⃣ 저가민감 고객 💰
```
Q: "가격이 너무 비싼데요?"
A: "이 가격은 왕복 항공료 + 4박 5성 호텔 + 모든 식사가 포함된 거예요.
   실제로는 하루 $150 정도로 계산되니까 엄청 저렴한 거죠."

👉 플레이북 "거절대응" 참고
```

### 2️⃣ 효도 여행 👨‍👩‍👧
```
Q: "부모님이 겁내세요"
A: "안심하세요. 배는 따뜻하고 음식 맛있고, 같은 또래 분들도 많아요.
   고급스러운 환경에서 편하게 여행할 수 있어요."

👉 "상품교육" → "부산출도착" 확인
```

### 3️⃣ 신혼부부 💑
```
Q: "신혼 여행으로 추천하는 상품?"
A: "일본 크루즈 추천해요. 로맨틱하고 짧은데 알찬 일정이거든요.
   신혼부부 특가 객실도 있어요."

👉 "상품교육" → "일본크루즈" 확인
```

### 4️⃣ 혼자 여행 🧳
```
Q: "혼자 가도 외로우면 어쩌죠?"
A: "배는 살아있는 도시예요. 매일 공연, 쇼, 게임, 식사 등이 있어서
   혼자 와도 따뜻한 분들을 만날 수 있어요."

👉 "플레이북" → "페르소나" 참고
```

### 5️⃣ 재구매 고객 🔄
```
Q: "지난번이랑 다른 상품?"
A: "이번엔 동남아 상품 새로 나왔어요. 지난번 부산과 완전 달라요.
   처음 가실 때와 다른 느낌을 원하신다면 추천드려요."

👉 "상품교육" → "동남아크루즈" 확인
```

## 🔥 꿀팁 TOP 5

### 1️⃣ 추천 도구 항상 먼저 확인
AI가 당신의 최근 활동을 분석해서 추천해줍니다.

### 2️⃣ 상품교육 북마크 하기
`Cmd+D` (Mac) 또는 `Ctrl+D` (Windows)로 북마크

### 3️⃣ 플레이북 새 창에서 열기
"전체 플레이북 ↗" 버튼으로 전체화면 모드 사용

### 4️⃣ 콜분석 활용하기
녹음 TXT 업로드 → AI가 점수 + 피드백 즉시 제공

### 5️⃣ 페르소나별 필터 설정
고객 유형 파악 후 "콜스크립트" 탭에서 필터링
