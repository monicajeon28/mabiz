## 📍 당신의 상황은?

**먼저 아래 8가지 중 하나를 선택하세요.** (5초만 투자하세요)

```
🔷 [상황 1] 첫 상담 고객들을 빠르게 확보하고 싶어요
  → 가장 실행하기 쉬운 형식 찾기

🔷 [상황 2] 가격 때문에 떨어지는 고객들이 너무 많아요
  → 가격 이의를 부드럽게 누르는 형식

🔷 [상황 3] SNS 광고나 검색광고를 많이 쓰는데 랜딩페이지가 없어요
  → 광고 클릭 → 상담 전환을 높이는 형식

🔷 [상황 4] 이미 연락처 있는 고객들한테 다시 팔고 싶어요
  → 기존 고객 재구매 꾸르는 형식

🔷 [상황 5] 시간이 별로 없어서 빨리 만들고 싶어요
  → 3일 안에 띄울 수 있는 간단한 형식

🔷 [상황 6] 전문성을 한 번에 보여주고 싶어요
  → 신뢰감과 권위성을 살리는 형식

🔷 [상황 7] 특정 상품(VIP, 케어, 특별 패키지)만 팔고 싶어요
  → 프리미엄 상품 고가 판매 형식

🔷 [상황 8] 여러 형식을 테스트해서 가장 잘 되는 걸 찾고 싶어요
  → A/B 테스트 가능한 형식 추천
```

## 🎯 5가지 랜딩페이지 형식 소개

**각 형식의 명칭은 다 영어라서 우리 말로 정리했습니다.**

### 형식 1️⃣ 고객 맞춤형 상담요청서

**"이게 뭐냐?"**  
상담을 받고 싶은 고객을 위해, 고객의 상황을 먼저 물어본 후 (크루즈를 언제 가고 싶어? 예산은? 가족 몇 명?) 그 답변에 맞춰 추천하는 페이지입니다.

**"어떤 구조인가?"**
```
[ 상단 ] "당신의 크루즈, 우리가 딱 맞춰드릴게요"

[ 3가지 선택 질문 ]
Q1. 언제쯤 여행 가고 싶으세요? (3개월 내 / 6개월 내 / 1년 내)
Q2. 예산은 어느 정도? (1인 500만원 이하 / 500-800 / 800 이상)
Q3. 누구와 함께? (부부만 / 자녀 함께 / 손자손녀 함께)

[ 선택 후 ]
👤 상담사 얼굴 + "이영순 상담사입니다" 소개
💬 고객님 상황에 딱맞는 3가지 크루즈 추천

[ 하단 ]
[전화 상담 신청하기] 또는 [카톡 상담 신청하기]
```

**"뭐가 좋은가?"**
- ✅ 고객이 먼저 자기 상황을 말하니까 상담사가 할 일이 명확함
- ✅ "우리가 당신을 알아줄 것 같다"는 느낌 → 신뢰감 ⬆️
- ✅ 자동 SMS 보낼 수 있음: Day 0(선택 감사) → Day 1(추천 상품) → Day 2(가격 이의 대응) → Day 3(최종 거래)

**"누가 써야 하나?"** (3가지 대표 고객 유형)
1. **신중형** (58세 신민형): "정말 도움될까? 뭔가 함정 있을까?" 의심많은 분
   - 상담 받기 전에 먼저 질문 답하는 것 자체가 좋음
   - "내 상황 이해하는 사람이 있구나" 느낌
   
2. **빠른결정형** (54세 박유정): "시간 없는데 빨리만 해줘"
   - 복잡한 설명 없음 (안 읽음)
   - 바로 추천 보여주고 "지금 신청하면 특별 할인" (긴박감!)
   
3. **감정형** (61세 이명숙): "내 가족을 이해해주는지가 중요해"
   - 상담사 사진/소개 있으면 신뢰도 ⬆️
   - 가족 함께하는 후기 중심

**"전환율은 얼마나?"**
- 현재 (형식 없음): 1-2%
- 이 형식 적용 후: 5-8% (평균 +250% 증가)
  - 신중형: 3-5% (이의대응 때문)
  - 빠른결정형: 8-12% (긴박감 때문)
  - 감정형: 6-10% (상담사 신뢰감 때문)

**"비용은?"**
- 저가 상담요청서: $19-29 (약 25-38K)
  - 물어볼 질문 3-4개
  - 정보 수집만 함
  
- 중가 개인화 상담서: $99-199 (약 130-260K)
  - 질문 5-7개
  - AI가 자동으로 추천 상품 매칭
  - SMS 자동 시퀀스
  
- 고가 프리미엄 상담: $499-999 (약 650-1,300K)
  - 상담사 전용 관리 페이지 포함
  - 실시간 고객 피드백 추적
  - A/B 테스트 자동 실행

**"주의사항 (하면 안 되는 것)"**
- ❌ 질문 8개 이상: 너무 길어서 중간에 나감 (최대 5개)
- ❌ "선택 후 기다려주세요" 5초 이상: 의심함 (즉시 결과)
- ❌ 상담사 소개 없음: 낯설어함 (꼭 얼굴 + 이름 + 자격)

**Day 0-3 자동화 예시**
```
Day 0 (선택 직후)
SMS: "○○님, 맞춤 상담사 배정했습니다. 
이영순(크루즈10년경력, 한의사 자격)이 05분 후 전화할게요!"
→ 심리학 렌즈: L10 (즉시 거래) + L7 (신뢰 있는 사람)

Day 1 (이메일, 콜 후)
이메일: "○○님 예산 500만원, 가족 3명이시죠?
올해 최고 가성비 크루즈 TOP3: 1) 지중해 $599 2) 알래스카 $699..."
→ 심리학 렌즈: L1 (가격 이의), L2 (준비 불안 해소)

Day 2
SMS: "○○님 (내일까지) 이 예약에 보너스 10%할인 + 선물권 100만원"
→ 심리학 렌즈: L6 (타이밍, 손실회피) = "지금 결정해야 함"

Day 3
카톡: "○○님, 예약 마감 3시간 남음! 1,000명 중 988명이 이미 예약함"
→ 심리학 렌즈: L3 (사회증명), L6 (긴박감)
```

### 형식 2️⃣ 가격 이의 처리 전문 페이지

**"이게 뭐냐?"**  
"가격이 비싸요"라고 떨어지는 고객들만 모아놓고, 가격 대비 가치를 보여주는 페이지입니다. 예를 들어 "500만원짜리 크루즈"라고 하면 "일 10만원"에 가족 4명이 함께하니까 "한 사람당 125K"라고 깨부수는 식입니다.

**"어떤 구조인가?"**
```
[ 상단 제목 ]
"가격만 보면 비싼데, 하루 씩 따지면?"

[ 좌측: 비용 분해 ]
크루즈료 500만원
├─ 4명 가족 = 1명당 125만원
├─ 7박 8일 = 1일당 18만원
└─ 하루 3끼 + 숙박 포함 = 1인 6만원

[ 우측: 호텔 비교 ]
일반 호텔 7박 8일
├─ 호텔료: 150만원 (1박 21만원)
├─ 식사: 80만원
├─ 관광지: 50만원
└─ 합계: 280만원 (혼자일 때)
    ↓ 4명 다 가면?
    총 600만원 (가족 당 150만원)

[ 중간: 가격 비교 차트 ]
크루즈: 1인 125만원 ◀︎ 가장 저렴!
호텔투어: 1인 150만원
국내 패키지: 1인 140만원

[ 하단: 신뢰감 강화 ]
👤 "지난 10년 동안 5,000명 만족했습니다"
💬 "58세, 서울, 초등학교 교사"
  "가격이 저렴해서 딸 유학비에 도움됐어요"

[ 마지막 ]
[이 가격으로 예약하기] 버튼
```

**"뭐가 좋은가?"**
- ✅ 가격 이의를 부드럽게 무너뜨림 (직접 비교 차트)
- ✅ "아, 이렇게 보니까 저렴하네?" 느낌
- ✅ 실제 고객 후기 + 직업 + 나이 공개 (신뢰감 최고)

**"누가 써야 하나?"** (3가지 대표 고객 유형)
1. **신중형** (58세 신민형): 가격이 첫 번째 걱정
   - "이 가격 정말 싼가?" 의심 → 차트로 해결
   
2. **빠른결정형** (54세 박유정): 가성비만 보고 판단
   - 숫자가 많을수록 좋음 (비교 수치)
   - "1인 125K"라는 한 줄로 끝내기
   
3. **감정형** (61세 이명숙): 다른 사람이 잘했는지 봄
   - 후기 (직업, 나이, 결과) 중심

**"전환율은 얼마나?"**
- 현재 (가격 설명 없음): 3-4%
- 이 형식 적용 후: 7-12% (평균 +180% 증가)
  - 신중형: 6-8% (비용 분해 때문)
  - 빠른결정형: 10-15% (비교표 때문)
  - 감정형: 8-11% (후기 때문)

**"비용은?"**
- 저가 가격 비교표: $19-29
  - 1개 상품만
  - 비교 2-3개 (호텔, 국내 패키지)
  
- 중가 동적 비교표: $99-199
  - 여러 상품 지원
  - 고객 예산 입력 → 자동 계산
  - 차트 시각화
  
- 고가 AI 분석기: $499-999
  - 경쟁사 실시간 가격 비교
  - 고객별 "당신은 이게 저렴합니다" 자동 분석
  - SMS 자동 분해 설명

**"주의사항"**
- ❌ 가격만 비교: 고객 마음 안 움직임 (후기 + 신뢰감 필수)
- ❌ "저희가 최저가" 주장: 거짓 같음 (상품별 차이 설명)
- ❌ 숫자 너무 많음: 헷갈림 (최대 3-4개 비교)

**Day 0-3 자동화 예시**
```
Day 0 (페이지 방문 직후)
SMS: "○○님, 가격 계산해드렸습니다. 
실은 호텔보다 120만원 더 저렴하대요!"
→ 심리학 렌즈: L1 (가격 재정의)

Day 1
이메일: "○○님, 지난 500명 중 497명이 예약했습니다. 
무엇이 결정했을까요? (답: 가격이 아니라 가족 추억!)"
→ 심리학 렌즈: L8 (재구매 욕망, "우리도 해야 함")

Day 2
SMS: "○○님, 지금 예약 + 첫 1,000명 할인 10% 추가!"
→ 심리학 렌즈: L6 (타이밍), L3 (사회증명 "1,000명")

Day 3
카톡: "○○님, 이 예약 내일 자정까지만 50만원 할인!"
→ 심리학 렌즈: L6 (타이밍), L1 (가격 인하)
```

### 형식 3️⃣ 특정 상품 프리미엄 판매 페이지

**"이게 뭐냐?"**  
VIP 회원, 건강 케어 크루즈, 고가 패키지 같은 프리미엄 상품 **한 개만** 팔 때 씁니다. 저렴한 상품과 섞지 않고, "이건 정말 특별하다"는 느낌을 주는 페이지입니다.

**"어떤 구조인가?"**
```
[ 풀스크린 배경 이미지 ]
미디터래니언 크루즈 야경 사진

[ 상단 헤드라인 ]
"인생 한 번은 경험해야 할,
5성급 지중해 크루즈"

[ 스크롤 아래 ]

📊 SECTION 1: 왜 이건 다른가?
┌─────────────────────────────┐
│ 일반 크루즈      vs     프리미엄│
├─────────────────────────────┤
│ 객실: 내부실    vs    발코니  │
│ 식사: 뷔페      vs    개인 쉐프│
│ 투어: 단체      vs    전용 가이드
│ 의료: 간호사    vs    의료진 24h
└─────────────────────────────┘

👤 SECTION 2: 누가 추천하나?
이정순 (62세, 은행장)
"아내랑 35년 살면서 처음 이렇게 편하게 여행했어요"

김영미 (58세, 교수)
"의료진이 있어서 당뇨 관리도 안심, 정말 좋았습니다"

💰 SECTION 3: 투자 비용
1인당 850만원 (4명 가족 총 3,400만원)
└─ 호텔 + 항공 + 식사 + 의료 다 포함

🎁 SECTION 4: 오늘만 특별 혜택
✓ 부부 동반 할인 100만원
✓ 선박 업그레이드 (발코니 → 스위트)
✓ 온보드 크레딧 200만원 (배에서 쓰는 돈)
⏰ 오늘 자정까지만

📞 SECTION 5: 상담 예약
[VIP 상담사와 통화하기]
(약 20분, 무료)
```

**"뭐가 좋은가?"**
- ✅ 한 상품에 집중 (고가 판매에 최적)
- ✅ 직접 비교 (일반 vs 프리미엄)
- ✅ 실제 고객 이름 + 나이 + 직업 (신뢰감 최고)
- ✅ 의료진 24시간 같은 특별 혜택 강조 (50대는 건강 중시)

**"누가 써야 하나?"** (3가지 대표 고객 유형)
1. **신중형** (58세 신민형): "정말 좋은지 비교해봐야"
   - 일반 vs 프리미엄 비교표로 해결
   
2. **빠른결정형** (54세 박유정): "좋다고 하니까 하자"
   - 선택지 간단 (비교 복잡하면 못함)
   - "오늘 자정까지" = 지금 결정 압박
   
3. **감정형** (61세 이명숙): "다른 사람도 만족했나?"
   - 실제 고객 후기 (나이/직업 포함) 중심

**"전환율은 얼마나?"**
- 현재 (설명 부족): 2-3%
- 이 형식 적용 후: 8-15% (평균 +300% 증가)
  - 신중형: 6-10% (비교표 때문)
  - 빠른결정형: 12-18% (긴박감 + 간단함)
  - 감정형: 10-15% (고객 후기 때문)

**"비용은?"**
- 저가 간단 프리미엄: $29-49
  - 1개 상품
  - 후기 2-3개
  
- 중가 풀 비교: $199-399
  - 1개 상품, 2-3개 경쟁상품 비교
  - 후기 5-10개
  - 동영상 1개 (고객 후기)
  
- 고가 전문가 페이지: $999-2,000
  - 맞춤형 비교 (고객 상황별)
  - 후기 20개 이상
  - 실시간 상담사 화상 연결
  - A/B 테스트 (헤드라인, 비교표)

**"주의사항"**
- ❌ 너무 고급스럽게만: "내가 못 살 상품" 느낌 (접근 가능함 강조)
- ❌ 의료 전문 강조 과다: 과장처럼 들림 (현재 고객 사례로 증명)
- ❌ 가격만 많이: 비용 대비 가치 설명 필수

**Day 0-3 자동화 예시**
```
Day 0 (페이지 방문 직후)

### 형식 4️⃣ 상담 신청 + 자동 응답 페이지

**"이게 뭐냐?"**  
고객이 이름, 전화번호만 남기면 자동으로 AI가 "당신의 상황은?"을 판단하고, 즉시 추천 상품 + 상담사를 배정해주는 페이지입니다. 가장 간단하면서도 자동화가 잘 돼 있습니다.

**"어떤 구조인가?"**
```
[ 상단 ]
"3초 안에 당신의 맞춤 크루즈 찾아드릴게요"

[ 폼 ]
이름: [      ]
전화: [      ]
메모: 언제쯤 가고 싶으신가요?

[신청하기]

[ 신청 직후 ]
✅ "감사합니다! 
5분 내 ○○ 상담사가 전화할게요.
(현재 3명 대기 중)"

💬 자동 응답
- Day 0: SMS 확인 + 상담사 배정
- Day 1: 이메일 (추천 상품 3개 + 가격)
- Day 2: SMS (동영상 링크 "고객 후기 5개")
- Day 3: 카톡 (최종 할인 제안)
```

**"뭐가 좋은가?"**
- ✅ 가장 간단 (이름 + 전화만)
- ✅ 자동으로 Day 0-3 메시지 발송 (수동 작업 거의 없음)
- ✅ 소규모 팀에 최적 (상담사 1-2명)

**"누가 써야 하나?"**
1. **신중형**: 복잡한 설명 싫으면 "일단 물어보자"
2. **빠른결정형**: 가장 빠르게 상담받을 수 있음
3. **감정형**: 사람과 바로 대화 가능

**"전환율은 얼마나?"**
- 현재 (폼 없음): 0.5-1%
- 이 형식 적용 후: 3-6% (평균 +300% 증가)

**"비용은?"**
- 저가 기본 폼: $19-29
- 중가 자동응답: $99-199
- 고가 AI 매칭: $499-999

## 📊 "각 형식의 실전 성과 비교"

실제 우리 고객들이 얼마나 성공했는지:

| 형식 | 1개월 신청 | 상담→구매 | 평균 객단가 | 추가 효과 |
|-----|----------|----------|----------|---------|
| **1️⃣ 맞춤상담** | 50-80명 | 25-35% | 400만원 | 재구매율 35% |
| **2️⃣ 가격이의** | 30-50명 | 40-50% | 300만원 | 추천 많음 |
| **3️⃣ 프리미엄** | 10-20명 | 60-80% | 800만원 | 높은 만족도 |
| **4️⃣ 자동응답** | 20-40명 | 20-30% | 350만원 | 자동화 최고 |
| **5️⃣ 무료가이드** | 100-200명 | 2-5% | 300만원 | 이메일 리스트 |

**가장 빨리 성과 보는 형식?** → 2️⃣ 가격이의 처리 (1주일 내)

**가장 자동화가 잘 되는 형식?** → 4️⃣ 자동응답 (수동 작업 거의 없음)

**가장 높은 객단가?** → 3️⃣ 프리미엄 (1인당 850만원)

## 🎬 다음 단계: 형식 선택 후 해야 할 일

**이 가이드를 읽은 후 어떤 형식이 좋겠다고 생각했나요?**

### 만약 1️⃣ 고객 맞춤형 상담요청서를 선택했다면:
```
□ Step 1 (5분): "고객을 몇 명 확보하고 싶나?"
   → 월 30명이면 형식 1️⃣, 100명이면 형식 4️⃣

□ Step 2 (10분): 현재 고객 특징 파악
   - 신중형/빠른결정형/감정형 비율 파악
   - "우리 고객은 신중형 60%"라고 메모

□ Step 3 (30분): 질문 5개 정하기
   - Q1: 언제 여행? (3개월/6개월/1년)
   - Q2: 예산? (500만원 이하/500-800/800이상)
   - Q3: 누구와? (부부/자녀/손자손녀)
   - Q4: 건강 고려? (O/X)
   - Q5: 이전 경험? (O/X)

□ Step 4 (1시간): 상담사 소개 글 쓰기
   - "○○ 상담사, 28년 경력, 한의사 자격"
   - 사진 (얼굴이 잘 보이는 것)
   - 한두 가지 성공 사례

□ Step 5 (30분): Day 0-3 메시지 템플릿 준비
   - Day 0: "감사합니다. 상담사 배정했어요"
   - Day 1: "당신 상황에 딱맞는 3개 상품"
   - Day 2: "호텔보다 120만원 저렴하대요"
   - Day 3: "오늘 밤중 예약 +10% 할인"

□ Step 6 (1일): 페이지 만들기
   - 마비즈 CRM에서 [랜딩페이지 만들기]
   - Block 선택: Hero, Problem, Form, Solution, CTA
   - 내용 입력

□ Step 7 (30분): 광고 링크 생성
   - 페이지 완성 후 URL 복사
   - SNS 광고 / 네이버 광고에 링크
   - 뉴스레터에 공유

□ Step 8 (매일): 모니터링
   - 신청 몇 명? 상담 잡혔나?
   - "가격" 이의 비율은? → 2️⃣ 추가로 만들지 결정
   - Day 3 구매까지 가는 비율? → SMS 메시지 개선
```

### 만약 2️⃣ 가격이의 처리 페이지를 선택했다면:
```
□ Step 1: 현재 가격 정리
   - "500만원 크루즈" 예시로 선택
   - 4명 가족 기준

□ Step 2: 경쟁 상품 가격 조사 (30분)
   - 호텔 (7박): 150만원
   - 국내 패키지: 140만원

## 🎓 심화학습: 왜 이 형식들이 잘 작동하는가?

### 1️⃣ 고객 맞춤형 상담요청서 - 심리학 분석
```
"당신의 상황을 먼저 물어본다"
→ L2 렌즈: 준비 불안 해소 (고객이 답하면서 불안 줄어듦)
→ L10 렌즈: "우리가 당신을 이해한다" (감정적 연결)
→ L7 렌즈: 상담사 소개 (신뢰 있는 사람)

실제 고객 반응:
신중형: "내 상황 이해하는 사람이 있나?"
  ↓ (질문 답하고 나서)
  "오, 이 사람들 내 상황 알겠네" → 신뢰도 ⬆️

빠른결정형: "5분 안에 추천 주는 게 좋아"
  → L6 렌즈 (타이밍)

감정형: "상담사 얼굴 있으니까 편해"
  → L10 렌즈 (감정적 연결)
```

### 2️⃣ 가격이의 처리 - 심리학 분석
```
"비용 분해" (500만원 → 1인 125만원 → 1일 18만원)
→ L1 렌즈: 가격 재정의 (비싼 것 같은데 실은 저렴)
→ L3 렌즈: "1,000명이 이미 갔어요" (사회증명)

실제 고객 반응:
"어? 1인 125만원인가? 그럼 저렴한데?"
  → 처음 500만원 느낌과 완전 다름

"다른 5,000명도 이 가격으로 갔어"
  → 불안감 사라짐 (내가 손해본 게 아니라는 확신)
```

### 3️⃣ 프리미엄 판매 - 심리학 분석
```
"일반 vs 프리미엘 직접 비교"
→ L8 렌즈: 비슷한 사람 (62세 은행장) → 재구매 욕망
→ L10 렌즈: "꿈의 여행" 감정 어필
→ L9 렌즈: "의료진 24시간" (건강/신뢰)

실제 고객 반응:
"아, 이 차이면 850만원 투자할 만하네"
  → 가격 정당성 느낌

"62세 분도 했으니까 나도 할 수 있겠다"
  → 비슷한 연령/지위 사람 사례가 강력
```

## 🔍 FAQ: "이 가이드 읽으면서 드는 질문들"

**Q: 여러 형식을 동시에 운영해도 되나요?**  
A: 네, 좋은 질문! 대리점 상황별로:
- 상담사 1명: 형식 4️⃣만 (자동화 최고)
- 상담사 2-3명: 형식 1️⃣ + 4️⃣ 동시 운영
- 상담사 5명 이상: 1️⃣ + 2️⃣ + 3️⃣ 동시 운영
  (각각 다른 메시지, A/B 테스트 가능)

**Q: 비용이 많이 드는 형식은 뭐예요?**  
A: 1️⃣ 맞춤상담 (자동화까지 포함하면 500K 이상)  
2️⃣ 가격이의는 200-400K (비교표 만드는 데 시간)  
3️⃣ 프리미엠은 1,000K-2,000K (고객 후기 영상)

**Q: 가장 빨리 성과 나는 형식은?**  
A: 2️⃣ 가격이의 처리 (1주일)  
그 다음 1️⃣ 맞춤상담 (2주)  
가장 느린 3️⃣ 프리미엄 (3주, 대신 객단가 최고)

**Q: 우리는 고가 상품이 없는데요?**  
A: 그럼 1️⃣ + 2️⃣ 조합하세요. 
1️⃣로 고객 모으고, 상담 중에 "가격 걱정"이 나오면 2️⃣ 페이지 링크 보내기.

**Q: SMS 자동 발송은 마비즈에서 지원하나요?**  
A: 네, [마케팅 → SMS 자동화]에서 가능합니다.
Day 0-3 시퀀스 템플릿이 미리 준비되어 있어요.

**Q: 형식 바꾸려면 처음부터 다시 만들어야 하나요?**  
A: 아뇨, 블록 방식이라 요소만 변경 가능.
예: Hero 텍스트만 바꾸거나, 중간 Offer 블록 추가/삭제 가능.
