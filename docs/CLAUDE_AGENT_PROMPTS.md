## 📌 Template 5: CRM 규칙 + 자동화 설계

📍 Contact 자동분류 (렌즈 기반)
IF 조건:
  - 부재 기간 (0-6개월, 6-12개월, 1년+)
  - 구매금액 (low/mid/high)
  - 이의 유형 (가격/준비/기항지/자유/의료)
  - 세그먼트 (신혼부부/가족/50대시니어/전문직)
THEN 액션:
  - 렌즈 라벨 자동 할당 (L0-L10)
  - SMS 시퀀스 자동 시작
  - 담당자 자동 배치
  - 콜 스크립트 자동 제시

📍 Risk Flag 자동 생성 (거래실패 패턴)
Pre-call Red Flag:
  ✗ 부재 1년 이상 (재활성화 어려움)
  ✗ 이의 5회 이상 (의사 변경 신호)
  ✗ 보험료 20% 이상 (능력 부족 신호)

During-call Red Flag:
  ✗ "생각해볼게요" 반복 (우유부단)
  ✗ "배우자 동의 필요" (의사결정권 없음)
  ✗ "지금은 여유 없어" (우선순위 낮음)

Post-call Red Flag:
  ✗ 24시간 내 Follow-up 응답 없음
  ✗ 48시간 내 SMS 클릭 없음
  ✗ Day 3 추가 제안 거절

액션: 
  - 위험도 스코어 자동 계산 (0-100점)
  - 60점 이상: RED (관리자 알림)
  - 40-60점: YELLOW (모니터링 강화)
  - 40점 이하: GREEN (정상 진행)

2️⃣ 부재중 재활성화
  부재 감지 → 감정적 재연결 메시지 → Day 0-7 시퀀스 → 재신청

3️⃣ 이의 자동대응
  이의 기록 → Grant Cardone 반박법 + SPIN 질문 → Day 1 Follow-up

4️⃣ 업셀 시스템
  구매 확인 → MIFGE 3단계 제시 → 추가 상품 제안

5️⃣ 재구매 습관화
  구매 후 60일 → Day 0 감정적 메시지 → 연간 2-4회 자동 제안

📍 성과 메트릭 자동 추적
