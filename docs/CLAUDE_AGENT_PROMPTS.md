📍 Contact 자동분류 (렌즈 기반)
IF 조건:
  - 부재 기간 (0-6개월, 6-12개월, 1년+)
  - 구매금액 (low/mid/high)
  - 이의 유형 (가격/준비/기항지/자유/의료)
  - 세그먼트 (신혼부부/가족/50대시니어/전문직)
THEN 액션:
  - 렌즈 라벨 자동 할당 (L0-L10)
  - SMS 시퀀스 자동 시작
  - 담당자 자동 배치
  - 콜 스크립트 자동 제시

📍 Risk Flag 자동 생성 (거래실패 패턴)
Pre-call Red Flag:
  ✗ 부재 1년 이상 (재활성화 어려움)
  ✗ 이의 5회 이상 (의사 변경 신호)
  ✗ 보험료 20% 이상 (능력 부족 신호)

During-call Red Flag:
  ✗ "생각해볼게요" 반복 (우유부단)
  ✗ "배우자 동의 필요" (의사결정권 없음)
  ✗ "지금은 여유 없어" (우선순위 낮음)

Post-call Red Flag:
  ✗ 24시간 내 Follow-up 응답 없음
  ✗ 48시간 내 SMS 클릭 없음
  ✗ Day 3 추가 제안 거절

액션: 
  - 위험도 스코어 자동 계산 (0-100점)
  - 60점 이상: RED (관리자 알림)

2️⃣ 부재중 재활성화
  부재 감지 → 감정적 재연결 메시지 → Day 0-7 시퀀스 → 재신청

3️⃣ 이의 자동대응
  이의 기록 → Grant Cardone 반박법 + SPIN 질문 → Day 1 Follow-up

4️⃣ 업셀 시스템
  구매 확인 → MIFGE 3단계 제시 → 추가 상품 제안

5️⃣ 재구매 습관화
  구매 후 60일 → Day 0 감정적 메시지 → 연간 2-4회 자동 제안

📍 성과 메트릭 자동 추적

    - 전략: 강점 렌즈 집중 개발

Phase 3: Scaling (6주-3개월)
  월간 1: 월 30-50명 판매 목표
    - 자동화 도입: SMS 자동화 시작 (Day 0-3)
    - 팀 구성: 콜센터 또는 VA 고용 검토
    - 수익: 월 200-300만원 (자동화 10-20%)
  
  월간 2: 월 50-100명 판매 목표
    - 시스템 구축: Lead Magnet + 웨비나 퍼널
    - 온라인강의: 첫 OTO (One-Time Offer) 상품 출시
    - 수익: 월 400-600만원 (자동화 30-40%)
  
  월간 3: 월 100명+ 판매 목표
    - 멤버십 고려: 월 구독 프로그램 출시
    - 제휴 확대: 3-5개 제휴사와 협력
    - 수익: 월 800-1200만원 (자동화 50-60%)

Phase 4: Retention (3개월+)
  분기별 1: Freedom Economy 진화
    - 현재: 직판 위주 (자동화 30-40%)
    - 목표: 온라인강의 + 멤버십 (자동화 60-70%)
    - 수익: 월 1500-2000만원
  
  분기별 2-3: Long-term Partnership
    - 주간: 성과 체크인 (30분)
    - 월간: 개선 전략 워크샵
    - 분기: 사업 발전 상담
    - 연간: 성과 보상 + 해외 여행 incentive

📍 파트너 페르소나별 맞춤 경로 (3가지)

페르소나 A: "도전형 영업사원" (신입/활동적)
  특성: 높은 에너지, 콜 많이, 이의 대응 약함
  전략:
    - Week 1: Grant Cardone 이의대응 집중교육 (8시간)
    - Week 2-3: 실전 역할연기 (관리자와 10회 이상)
    - Week 4+: 주 2회 코칭 (이의대응 피드백)
  
  기대 효과: 전환율 28% → 38% (10% 증대)

페르소나 B: "신중형 상담자" (경험많음/느슨함)
  특성: 신뢰를 중시, 콜 신중, 높은 전환율, 속도 느림
  전략:
    - Week 1: 심리학 렌즈 고급 교육
    - Week 2-3: 자동화 도입 (SMSDay0-3)
    - Week 4+: 월 1회 전략 회의 (새로운 렌즈 실험)
  
  기대 효과: 월 20명 → 40명 (자동화로 2배 생산성)

페르소나 C: "기술형 온라인마케터" (디지털네이티브)
  특성: 온라인 선호, 콜 싫어, 창의적, 혼자 일하려함
  전략:
    - Week 1: Russell Brunson 마케팅 + VSL 스크립트
    - Week 2-3: 웨비나 퍼널 설계 워크샵
    - Week 4+: 월 1회 온라인 마케팅 전략 회의
  
  기대 효과: 월 30명 (온라인만) → 60명 (콜+온라인 하이브리드)

📍 자동화 체크리스트 (파트너 진행 추적)

Onboarding 완료 기준:
  ✅ 환영 이메일 수신 (Day 1)
  ✅ 4시간 기초 교육 완료 + 인증서 (Day 7)
  ✅ 콜 스크립트 암기 녹화 제출 (Day 14)
  ✅ 첫 콜 리허설 완료 (Day 14)
  ✅ CRM 계정 활성화 (Day 14)

Activation 완료 기준:
  ✅ 첫 판매 달성 (Week 4 내)
  ✅ 월 10명 판매 (Week 6 내)
  ✅ 렌즈별 고객 분석 (강점 렌즈 발견)
  ✅ Day 0-3 SMS 자동화 시작
  ✅ 주 2회 코칭 완료율 80%+

Scaling 완료 기준:
  ✅ 월 30명+ 판매 (Month 2)
  ✅ 월 50명+ 판매 (Month 3)
  ✅ 자동화율 30-40% (SMS/Email 자동화)
  ✅ First OTO 상품 출시 (Month 3)
  ✅ Tier 2 (실버) 이상 도달

Retention 추적:
  ✅ 월 100명+ 판매 (Month 3+)
  ✅ 월 수익 800-1200만원 (Month 3+)
  ✅ 자동화율 50-60% (온라인강의/멤버십)
  ✅ 파트너 Churn Risk Score < 40

📍 주간/월간 성과 리포팅 자동화

주간 리포트 (매주 월요일 자동 생성):
  1. 목표 vs 실적 (판매자수)
  2. 평균 전환율 (개인 vs 평균)
  3. 렌즈별 성과 분해
  4. SMS 성과 (도달/클릭/전환)
  5. Risk Flag (미흡사항 3가지)
  6. 격려 메시지 + 다음주 목표

월간 리포트 (매월 1일 자동 생성):
  1. 월 목표 vs 실적 (진행율%)
  2. 월 매출 + Commission
  3. 렌즈별 전환율 순위 (강점 vs 약점)
  4. 채널별 성과 (콜 vs SMS vs 온라인)
  5. 파트너 Tier 및 Bonus
  6. 월간 혁신 항목 (렌즈 실험 결과)
  7. 다음달 전략 (맞춤 권장사항)

📍 파트너 Churn 방지 (목표: 85% Retention)

Churn 신호 감지 (자동 알림):
  - 월 판매 0명 (2주 이상)
  - 일주일 콜 기록 없음
  - SMS 자동화 설정 안 함 (4주)
  - 교육 불참 (2회)
  - Commission 미수령 (1회)

자동 개입 프로세스:
  Day 1: 관리자 개인 연락 (상황 확인)
  Day 7: 문제 해결 회의 (30분 코칭)
  Day 14: 추가 교육 자료 제공 + 동료 멘토 배치
  Day 21: 비즈니스 모델 재검토 (페르소나 변경?)
  Day 30: 최종 기회 제시 또는 프로그램 일시정지
