## 📌 Phase 1: 준비하기 (Week 1-2)

### Week 1 월요일 (오늘)

**아침 30분: 팀 회의**

- [ ] 이 문서 읽고 이해하기
- [ ] "6개월 목표" 모두 공유하기 (월 1억 2,000만원, 업무 -75%)
- [ ] 각자 역할 정하기:
  - 팀장: 진행 총괄
  - 상담사 1명: 메시지 작성
  - 상담사 1명: 고객 분류
  - 개발팀 1명: 시스템 설정

**오후 1시간: 현황 파악**

- [ ] 지난 3개월 고객 데이터 모으기 (200명 이상)
- [ ] 현재 월 매출 확인 (기준선)
- [ ] 현재 메시지 오픈율 확인 (기준선)

### Week 1 수요일

**오전 30분: 데이터 분석**

- [ ] 고객 200명을 5그룹으로 나누기:
  ```
  그룹 1 (초기): 처음 온 사람 → 몇 명?
  그룹 2 (기본): 기본정보 있음 → 몇 명?
  그룹 3 (활동): 자주 온 사람 → 몇 명?
  그룹 4 (VIP): 최고 고객 → 몇 명?
  그룹 5 (휴면): 안 온 지 3개월+ → 몇 명?
  ```
- [ ] 각 그룹별 특징 정리 (1줄씩)

**오후 1시간: 목표 가격표 만들기**

- [ ] 각 그룹별 5가지 상품 선택
- [ ] 각 상품별 가격, 기간, 특징 정리
- [ ] 상담 스크립트 3줄짜리 작성 (각 그룹별)

### Week 1 금요일

**오전 1시간: 시스템 준비**

- [ ] CRM에 고객 5그룹 카테고리 생성
- [ ] 각 그룹별 기본 메시지 템플릿 3개 만들기
  - Day 0: 상담 직후 인사 메시지
  - Day 1: 옵션 제시 메시지
  - Day 3: 긴급 할인 메시지

**오후 1시간: 첫 테스트**

- [ ] 그룹 1 (초기 고객) 30명 선택
- [ ] Day 0 메시지 보내보기
- [ ] 응답 기록하기

### Week 2 월요일-금요일

**매일 30분: 피드백 수집**

- [ ] Day 0 메시지 응답률 확인 (목표 30%)
- [ ] 잘된 메시지 vs 안 된 메시지 분석
- [ ] 내용 수정 (이미지, 문구, 시간)

**금요일 1시간: Week 1 정리**

- [ ] 5그룹 분류 완료 (데이터 확인)
- [ ] 메시지 3가지 (Day 0/1/3) 최종 확정
- [ ] 준비 완료 체크:
  ```
  ✓ 고객 5그룹 분류 완료
  ✓ 메시지 3개 만들고 테스트 완료
  ✓ 상담 스크립트 각 그룹별 준비
  ✓ 대시보드 기준선 데이터 수집
  ```

## 📌 Phase 2: 그룹 분류 & 메시지 만들기 (Week 3-4)

### Week 3 월요일-금요일

**매일 아침 15분: 고객 새로 들어온 거 분류**

- [ ] 신규 고객 10명 → 5그룹 중 어디? 배치
- [ ] 각 고객 특징 1줄 메모 (예: "가격 민감, 부모님 동반")
- [ ] 해당 그룹 기본 메시지 발송

**매일 오후 30분: 응답 기록**

- [ ] 고객 응답 확인
- [ ] "상담 신청"한 사람 → 상담사에게 배정
- [ ] "다시 물어본 사람" → 대답 메시지 보내기

**금요일 1시간: 주간 분석**

- [ ] 이번 주 상담 신청 몇 명? (목표: 10명)
- [ ] 메시지 오픈율? (목표: 40%)
- [ ] 어떤 메시지가 잘 됐나?
- [ ] 안 된 메시지 이유 분석:
  ```
  ❌ 이미지 문제?
  ❌ 텍스트 너무 길었나?
  ❌ 보내는 시간 문제?
  ❌ 고객 그룹이 잘못됐나?
  ```

### Week 4 월요일-수요일

**내용 개선하기**

- [ ] 안 된 메시지 이미지 3개 바꾸기
- [ ] 텍스트 짧게 수정 (한 줄 = 15글자 이내)
- [ ] 보내는 시간 변경 (오전/오후/저녁)
- [ ] 고객 재분류 (그룹이 맞나?)

**목요일-금요일: 재테스트**

- [ ] 수정된 메시지 30명에게 다시 보내기
- [ ] 응답 기록
- [ ] 비교: 전 버전 vs 새 버전 (어느 게 나았나?)

**금요일 1시간: Week 3-4 정리**

- [ ] Phase 2 완료 체크:
  ```
  ✓ 신규 고객 100명 분류 완료
  ✓ 메시지 응답률 30% → 40%+ 달성
  ✓ 상담 신청 20명 이상 (일주일)
  ✓ 메시지 3가지 (Day 0/1/3) 최적화 완료
  ```

## 📌 Phase 3: 자동화 설정 (Week 5-6)

> **"자동화"는 당신이 손 안 대도, 시스템이 알아서 하는 것**

### Week 5 월요일

**오전 1시간: 자동 분류 설정**

- [ ] CRM 설정 → "자동 분류" 활성화
- [ ] 고객 온 날부터 Day 0/1/3 메시지 자동 스케줄
  ```
  Day 0: 상담 직후 (즉시)
  Day 1: 다음날 오후 2시
  Day 3: 3일 뒤 오전 10시
  ```
- [ ] 각 그룹별 메시지 자동 배정

**오후 1시간: 자동 알림 설정**

- [ ] 상담사 A: 그룹 1-2 담당 → 상담 신청 오면 자동 알림
- [ ] 상담사 B: 그룹 3-5 담당 → 상담 신청 오면 자동 알림
- [ ] 예약 확정 → 자동으로 "다음 단계" 메시지 발송

### Week 5 수요일

**오전 1시간: Day 7, 14, 21 메시지 설정**

- [ ] 응답 없는 고객 → Day 7 메시지 자동 발송
  ```
  내용: "혹시 궁금한 점 있으신가요?"
  ```
- [ ] 여전히 응답 없음 → Day 14 메시지
  ```
  내용: "특별 할인 쿠폰 드려요"
  ```
- [ ] Day 21 최후 메시지
  ```
  내용: "정말 도움 필요하면 전화 주세요"
  ```

**오후 1시간: 자동 분기 설정**

- [ ] 상담 신청 → 상담사 자동 배정
- [ ] 예약 확정 → "예약자" 그룹으로 자동 이동
- [ ] 예약 취소 → "복구" 그룹으로 자동 이동 (재접근)

### Week 5 금요일

**오전 1시간: 테스트**

- [ ] 5명 고객 직접 추가해서 흐름 테스트
- [ ] Day 0 메시지 제때 왔나?
- [ ] 상담 신청 → 상담사 알림 왔나?
- [ ] 자동 분류 제대로 됐나?

**오후 1시간: 수정**

- [ ] 안 된 부분 고치기
- [ ] 메시지 타이밍 조정

### Week 6 월요일-금요일

**매일: 모니터링**

- [ ] 자동화가 제대로 도는가?
- [ ] 메시지 전송 실패는 없나?
- [ ] 고객 응답은 제때 기록되나?

**금요일 1시간: Phase 3 정리**

- [ ] 자동화 완료 체크:
  ```
  ✓ Day 0-3-7-14-21 자동 메시지 설정 완료
  ✓ 상담사 자동 배정 완료
  ✓ 그룹 자동 이동 완료
  ✓ 자동 알림 완료
  ```

## 📌 Phase 4: 테스트 & 개선 (Week 7-8)

### Week 7 월요일-목요일

**매일: 데이터 모으기**

- [ ] 아침: 어제 상담 신청 몇 명?
- [ ] 오후: 메시지 오픈율 몇 %?
- [ ] 저녁: 예약 몇 건?
- [ ] 기록: 노트나 엑셀에 적기

**금요일 1시간: 주간 분석**

- [ ] 이번 주 목표 달성했나?
  ```
  상담 신청: 목표 10명 → 실제 몇 명?
  오픈율: 목표 40% → 실제 몇 %?
  예약: 목표 3명 → 실제 몇 명?
  ```
- [ ] 못 한 부분 분석:
  ```
  왜 상담 신청이 적나?
  - 메시지 제목 문제?
  - 이미지 문제?
  - 보내는 시간 문제?
  - 대상 고객 선택 문제?
  ```

### Week 8 월요일-목요일

**A/B 테스트 (메시지 비교)**

- [ ] 메시지 A (기존): 30명에게
- [ ] 메시지 B (새로운): 30명에게
- [ ] 누가 더 많이 열어봤나?
- [ ] 누가 더 많이 상담 신청했나?
- [ ] 이긴 메시지를 계속 사용

**예시:**

```
메시지 A: "크루즈 예약하세요! 저렴한 가격 보장"
메시지 B: "OTA보다 비싼 이유, 5분 안에 알려드려요"

결과: B가 30% 더 많이 열어봤음
→ 앞으로 B 스타일로 메시지 만들기
```

**금요일 1시간: Phase 4 정리**

- [ ] 테스트 완료 체크:
  ```
  ✓ 주간 데이터 4주 분량 수집
  ✓ A/B 테스트 4가지 완료
  ✓ 이긴 메시지 패턴 파악 (이미지/텍스트/시간)
  ✓ 개선된 메시지 세트 확정
  ```

## 📌 Phase 5: 계속 개선하기 (Week 9+)

> **이제부터는 계속 개선하는 단계입니다**

### 매주 (계속)

**월요일 아침 30분: 주간 목표 설정**

- [ ] 이번 주 상담 신청 목표: __ 명
- [ ] 오픈율 목표: __ %
- [ ] 예약 목표: __ 건
- [ ] 지난주보다 5-10% 높은 목표

**매일 아침 15분: 대시보드 확인**

- [ ] 어제 상담 신청: __ 명 (목표 대비)
- [ ] 메시지 오픈율: __ % (목표 대비)
- [ ] 예약: __ 건 (목표 대비)

**금요일 1시간: 주간 분석**

- [ ] 목표 달성했나?
- [ ] 못 한 이유는?
- [ ] 다음 주 개선점은?

### 매월 (계속)

**월초 2시간: 월간 분석**

- [ ] 지난달 월 매출 (기준: 1,400만원)
- [ ] 목표 (1,200만원 이상)
- [ ] 달성했나?

**월말: 다음달 계획**

- [ ] 이번달 뭐가 잘 됐나?
- [ ] 뭐가 안 됐나?
- [ ] 다음달에 뭘 바꿀까?

### 매분기 (3개월마다)

**분기말: 대규모 분석**

- [ ] 지난 3개월 성과 검토
- [ ] 목표 대비 달성도 (몇 %?)
- [ ] 상담사 개인 성과 (누가 잘했나?)
- [ ] 다음 분기 전략 수립

## ❌ 실패 신호 (이렇게 되면 조정)

### 만약 상담 신청이 목표의 50% 이하면

```
1. 메시지 이미지 비교 (A/B 테스트)
2. 메시지 텍스트 비교 (짧게 다시 쓰기)
3. 보내는 시간 변경 (오전/오후/저녁)
4. 광고 클릭율 확인 (광고가 먼저 문제일 수도)
```

### 만약 메시지 오픈율이 20% 이하면

```
1. 제목이 너무 길었나? (10글자 이내로 수정)
2. 이미지가 매력적이지 않나? (새 이미지 5개 테스트)
3. 보내는 시간이 문제? (오전/오후/저녁 테스트)
4. 고객 그룹이 맞나? (잘못 분류된 사람 재분류)
```

### 만약 예약 성공률이 10% 이하면

```
1. 상담 스크립트 확인 (Grant Cardone 10가지 심리 적용했나?)
2. 상담사 톤/태도 (너무 딱딱했나? 따뜻하게)
3. 가격 제시 방법 (한 가지만? 3가지 옵션 제시)
4. 이의 대응 (고객 "혹시...?" → 당신은 어떻게 답했나?)
```

## 📞 진행 중 막히는 것들

### "메시지 시간 설정이 어떻게 하지?"

```
CRM → Tools → SMS 자동화
→ "Day 0" 클릭
→ "발송 시간" 설정 (예: 오후 2시)
→ "저장" 클릭
```

### "고객 분류를 다시 하고 싶어"

```
CRM → Contacts
→ 고객 선택
→ "그룹" 드롭다운 변경
→ "저장"
```

### "메시지 테스트하고 싶어"

```
CRM → Tools → SMS 자동화
→ "메시지 미리보기"
→ 테스트할 고객 1명 선택
→ "미리보기" 버튼
→ 내용 확인 후 "보내기"
```

### "대시보드는 어디서 봐?"

```
홈 화면 → "대시보드" 버튼
→ 5개 숫자 확인 (초록=잘함, 빨강=주의)
```
