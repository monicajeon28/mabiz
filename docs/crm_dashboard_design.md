## ⚡ 실시간 업데이트 구조

### WebSocket 연결
```typescript
// src/lib/analytics/dashboard-stream.ts
const ws = new WebSocket('wss://api.mabiz.com/ws/analytics');

ws.onmessage = (event) => {
  const { type, data } = JSON.parse(event.data);
  
  switch(type) {
    case 'CALL_COMPLETED':
      updateCallCount(data);
      recalculateConversionRate();
      break;
    case 'SMS_SENT':
      updateSMSMetrics(data);
      break;
    case 'CONVERSION':
      incrementRevenue(data.amount);
      triggerAnimation();
      break;
  }
};
```

### 업데이트 빈도
```
Real-time (< 5초):
- 콜 완료/전환
- SMS 발송 확인

Near Real-time (1분):
- 전환율 업데이트
- 상담사 성과

Batch (1시간):
- CAC/LTV 계산
- 코호트 분석

Daily (아침 9시):
- 일일 리포트 생성
```
