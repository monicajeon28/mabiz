```

│  🔴 즉시 신청! 마감까지  ← L10 클로징 메시지                   │
```

- 패딩: 32px (md:p-8)

- 항목 간격: 16px (mb-4, mb-6)

## 🎬 애니메이션 타이밍 (Animation Timing)

### Pulse Animation (Critical 상태만)
```
시간 → 불투명도
0s    100% ████████████████
0.5s   50% ████████
1.0s  100% ████████████████
1.5s   50% ████████
2.0s  100% ████████████████ (1주기 완료, 반복)
```

**Easing**: `cubic-bezier(0.4, 0, 0.6, 1)` (표준 ease-in-out)

**주기**: 2초 (너무 빠르면 거슬림, 너무 느리면 효과 없음)

### Color Transition (Safe → Warning → Alert → Critical)
```
시간 경과
├─ 24시간+ → 초록 (smooth transition 500ms)
├─ 6-24시간 → 황색 (smooth transition 500ms)
├─ 1-6시간 → 주황 (smooth transition 500ms)
└─ 0-1시간 → 빨강 + 펄스 (smooth transition 500ms)

className={`... transition-all duration-500`}
```

## 🌙 Dark Mode (자동 지원)

Tailwind CSS 기본 색상이므로 자동으로 다크모드 지원:

### Light Mode
```
text-green-700  →  #15803d (진한 초록)
bg-green-50     →  #f0fdf4 (밝은 초록)
```

### Dark Mode (자동)
```
text-green-700  →  #86efac (밝은 초록, 검정 배경에 보임)
bg-green-50     →  #064e3b (진한 초록, 검정 배경에 맞춤)
```

## ♿ 접근성 (Accessibility)

### 색상 대비 (Color Contrast)
| 조합 | 대비율 | WCAG | 상태 |
|-----|--------|------|------|
| 초록(#15803d) vs 흰색(#fff) | 5.2:1 | AA ✅ | 통과 |
| 황색(#ca8a04) vs 흰색(#fff) | 4.8:1 | AA ✅ | 통과 |
| 주황(#ea580c) vs 흰색(#fff) | 5.1:1 | AA ✅ | 통과 |
| 빨강(#7f1d1d) vs 흰색(#fff) | 6.1:1 | AAA ✅ | 통과 |

### 안내성 (Readability)
- [x] 흰색 배경 ❌ (컨테이너 배경색 필수)
- [x] 폰트 크기 ≥ 16px (모바일)
- [x] 폰트 무게 bold (눈에 띔)
- [x] 아이콘 + 텍스트 조합 (색상만으로 의존하지 않음)
  - "🔴 즉시 신청!" (아이콘 + 텍스트)
  - "⚠️ 1시간 내" (아이콘 + 텍스트)

## 📊 예상 시각적 효과 (Expected Visual Impact)

### Before (기존 CountdownTimer)
```
      00 : 02 : 47         ← 빨강만, 작은 글자, 효과 약함
```

### After (개선된 Psychology-Driven)
```
┌─────────────────────────────────────┐
│  🔴 즉시 신청! 마감까지              │  ← 강조 메시지
├─────────────────────────────────────┤
│        00 : 02 : 47                │  ← 큰 글자, 펄스
│        일  시간  분                  │
│                                     │
│  🔴 마감 직전! 지금 신청해야 합니다 │  ← 손실회피 강조
│                                     │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│        [신청하기 버튼]               │  ← 즉시 클릭 유도
└─────────────────────────────────────┘
```

**심리적 변화**:
- 안락감: "마감이 언제인데?" → 긴박감: "지금 바로 결정해야겠다!"
- 지연: 시간 남음 → 즉시 행동: 버튼 클릭

## 🎯 측정 기준 (Success Metrics)

### 시각적 강조 순서
1. 🟢 Safe: 차분한 톤 (정보성)
2. 🟡 Warning: 주의 신호 (인지)
3. 🟠 Alert: 경고 신호 (긴급)
4. 🔴 Critical: 신호 + 애니메이션 (즉시)

### 클릭 가능성 (Clickability)
- [ ] 터치 타깃: 44px 이상 (모바일)
- [ ] 콘트라스트: 4.5:1 이상
- [ ] 레이블: 명확한 행동 문구
