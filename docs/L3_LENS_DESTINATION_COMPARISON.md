## 🎯 핵심 메시지

**"기항지는 '관광지 1일 체험', 배는 '리조트 5일 거주"**

호텔 여행 고객은 이렇게 생각해요:
- "배가 움직인다?" → 불편하다 (시간 낭비)
- "한 배에 5일?" → 지겨울 것 같은데?
- "배의 시설은?" → 호텔 방처럼 좁을 거 (선실 = 침실만)

우리의 재정의:
- "배가 움직인다" = 당신을 매일 다른 세상으로 옮겨줌
- "한 배에 5일" = 50+ 개의 공간을 자유롭게 사용 (침실은 자는 곳만)
- "배의 시설" = 호텔 "객실" vs 배 "리조트 전체" (스케일이 다름)

## 📊 시각화 이미지 설계 (4가지)

### Image 1: "기항지 1일의 실제 시간 사용" (파이차트)

```
기항지 1일 (8시간) 분석:
└─ 08:00 배에서 하선 및 이동 (30분) → 30분 낭비
└─ 08:30 관광지 도착
   ├─ 박물관/쇼핑 (2시간)
   ├─ 점심식사 (1시간)
   ├─ 관광 (2시간)
└─ 17:30 배로 귀환 및 정착 (30분) → 30분 낭비
└─ 18:00 저녁 식사 (1시간)

📊 실제 "배 밖" 경험: 5시간
🚫 "이동 + 대기" 낭비: 1시간
🎭 "야외 관광" 의존도: 100% (기항지 기후에 따라 변함)

→ "기항지는 '제한된 8시간' 안에서 정해진 곳만"
```

**시각화 방법**:
- 파이 차트: "배 밖 경험"(파란색 60%) vs "이동/대기"(회색 20%) vs "배 복귀"(주황색 20%)
- 타임라인: 08:00-18:00 시간대에 따른 색상 변화
- 주석: "한 기항지만 즐길 수 있어요"

### Image 2: "배 위의 5일 시간 사용" (시간당 활동 맵)

```
배 위의 5일 (120시간) 자유도 분석:

Day 1-5 매일:
├─ 08:00 아침 (침실/테라스)
│  └─ 당신의 선택: 침실? 갑판? 레스토랑?
│     (어디든 갈 수 있음, 움직임 0)
│
├─ 09:00-12:00 오전 (배의 50+ 공간)
│  ├─ 스파: 마사지 받기
│  ├─ 풀장: 수영하기
│  ├─ 극장: 쇼 보기
│  ├─ 라운지: 책 읽기
│  └─ 또는 갑판: 바다 보며 카페 (선택은 당신의 것)
│
├─ 12:00-14:00 점심
│  └─ 10개 레스토랑 중 선택 (매일 다른 곳)
│
├─ 14:00-17:00 오후
│  └─ 스파? 풀장? 영화? 쇼핑? (모두 배에 있음)
│
├─ 18:00 저녁 준비
│  └─ 침실에서 휴식
│
└─ 19:00+ 저녁
   ├─ 브로드웨이 쇼 (극장)
   ├─ 라이브 음악 (바)
   ├─ 카지노 (오락)
   ├─ 고급 레스토랑 (식사)
   └─ 선택은 당신의 것 (이동 X)

📊 실제 "배 위" 경험: 115시간 (96%)
✨ "새로운 체험" 가능도: 매일 다른 활동 (50+ 가지)
🚫 "계획 짜기" 부담: 0 (원하면 즐기고, 아니면 쉴 수 있음)

→ "배는 '자유로운 120시간' 안에서 50+ 가지 경험"
```

**시각화 방법**:
- 좌측: "기항지 타임라인" (제약이 많은 이미지, 회색)
- 우측: "배 위 타임라인" (자유로운 이미지, 다채로운 색)
- 중간: "시간 사용 비율" 비교
- 하단: "체험 가능성" 비교 (기항지: 5가지 vs 배: 50가지)

### Image 3: "공간의 크기" (비포 & 애프터)

```
호텔 여행 1일의 "당신의 공간":
┌─────────────────┐
│    호텔 침실    │
│   (20평, 침대  │
│    + 욕실)      │
│                 │
│    → 저녁:     │
│   호텔 로비 or  │
│   근처 카페     │
│                 │
│ 총 활동 반경:   │
│  500m 이내      │
└─────────────────┘

크루즈 1일의 "당신의 공간":
┌──────────────────────────────┐
│        배 전체 (리조트)       │
│  침실 + 갑판 + 스파 + 풀장   │
│  극장 + 레스토랑 × 10 + 바   │
│  카지노 + 라운지 + 피트니스  │
│  쇼핑몰 + 도서관 + 게임룸    │
│                              │
│  → 저녁:                    │
│  브로드웨이 쇼 + 라이브 음악 │
│  + 고급 레스토랑 (배 위)     │
│                              │
│ 총 활동 반경:               │
│  배 전체 (반복 이동 0)       │
│                              │
│ + 기항지: 하루에 새로운 나라 │
│ (이동은 배가 해줌)          │
└──────────────────────────────┘

📊 공간 크기 비교:
- 호텔 침실: 20평 (침실만)
- 배의 공용 공간: 호텔 층 전체 × 3-5층 규모
- 호텔의 자신의 거주지: 20평
- 배의 자신의 거주지: 50+ 공간 자유 사용

→ "당신의 거주 공간: 20평 → 배 전체(1000평+)"
```

**시각화 방법**:
- 좌측 박스: 호텔 침실 사진 (작은 공간)
- 우측 박스: 배의 여러 시설 사진 모음 (큰 공간)
- 중앙: 크기 비교 화살표 ("→")
- 하단: "침실은 자는 곳, 낮은 배 전체가 당신의 거실"이라는 메시지

### Image 4: "감정 변화" (고객 여정 맵)

```
호텔 여행의 감정 곡선:
Day 1: 설렘 ↗️
Day 2: 기대 ↗️
Day 3: 지침 ↘️ ← "또 이 도시 인가" (3박째 지루함)
Day 4: 불만 ↘️ ← "짐 싸야 하는데..."
Day 5: 피로 ↘️ ← "귀가 후 회복 필요"

평균 만족도: 65/100

---

크루즈 여행의 감정 곡선:
Day 1: 설렘 ↗️ ("배 탑승!")
Day 2: 완전 새로운 나라 ↗️ ("어제와 다른 항구!")
Day 3: 계속 새로움 ↗️ ("또 다른 나라가?")
Day 4: 배운 체험 + 휴식 ↗️ ("배 위에서 쉴 수도, 활동할 수도")
Day 5: 추억 + 아쉬움 ↗️ ("다음엔 더 긴 크루즈!")

평균 만족도: 92/100

---

감정 곡선의 차이:
- 호텔: "지루함이 찾아옴" (특히 3박째)
- 크루즈: "계속 새로움" (매일 다른 배경)
- 호텔: "체크인/체크아웃의 부담"
- 크루즈: "배가 당신을 옮겨줌"
- 호텔: "사후 회복 필요"
- 크루즈: "배 위에서 회복"

→ "지루함 0, 휴식 100인 여행이 바로 크루즈"
```

**시각화 방법**:
- 위/아래 두 개의 라인 차트
- 호텔: 빨간색 (하강 곡선)
- 크루즈: 파란색 (상승 곡선)
- 각 날짜별 아이콘 (Day 1: 설렘 😍, Day 3: 지침 😑, Day 5: 피로 😫)
- 하단: "왜 크루즈는 계속 새로울까?"라는 설명
