## 🔧 에이전트 업그레이드 3단계

### Stage 1: 학습 내용 RAG 통합 (완료)
```
메모리 파일 195개 + 새 PDF → RAG 인덱스 자동화
├─ 판매 심리학 (10개 렌즈)
├─ 마케팅 자동화 (8가지 채널)
├─ 고급 전략 (MIFGE/VSL/퍼널)
└─ 경제적 자유 체계
```

### Stage 2: 에이전트 Prompt 업그레이드 (진행 중)
```
기존 에이전트 지시:
"이 기능을 만들어줘"

업그레이드된 에이전트 지시:
"[심리학 원칙] + [자동화 패턴] + [CRM 규칙]을 고려해서 
[고객 렌즈]를 타겟한 [기능]을 설계하고 구현해줘.
성과 목표는 [전환율/메트릭]이고, 
[메모리 파일]들을 참고해서 진행"
```

### Stage 3: MCP/Hook 구성 최적화 (설계 중)
```
settings.json 개선 전략:
1. 판매 기법 Hook: commit 전 심리학 검증
2. 마케팅 Hook: 광고 운영 전 RAG 참고
3. 파트너 교육 Hook: 템플릿 생성 시 자동 적용
4. SMS 자동화 Hook: Day 0-3 심리학 기반 시퀀스
```
