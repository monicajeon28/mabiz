## 🎓 심리학 기법별 활용 매뉴얼

### 기법 1: Loss Aversion (손실회피)
**메모리**: [[l6_timing_loss_aversion]] + [[psychology_theories_master]]

**활용 예시**:
```
현재: "크루즈 일정을 지금 확정하세요"
↓ (손실회피 적용)
개선: "지금 예약하면 월 15만원 절약! 
       1주일 뒤엔 배치 마감으로 예약 불가"

UI/SMS 변화:
- 저축액 시각화 (월 15만원 × 12개월 = 180만원)
- 가격 상승 타이머 ("3시간 후 일반가 복구")
- 남은 배치 수 실시간 표시 ("배치 2개만 남음!")
```

### 기법 2: Social Proof (사회증명)
**메모리**: [[psychology_theories_master]] + [[sns_facebook_advertising]]

**활용 예시**:
```
현재: "이 크루즈를 예약하세요"
↓ (사회증명 적용)
개선: "지난 달 3,200명이 이 크루즈로 떠났습니다.
       ⭐⭐⭐⭐⭐ 4.9/5점 (2,100개 리뷰)"

UI 추가:
- 월별 예약자 수 (그래프)
- 실제 고객 후기 (이름 + 지역 + 사진)
- 만족도 스코어 (4.9점, 증가 추이)
```

### 기법 3: Scarcity (희소성)
**메모리**: [[grant_cardone_closing]] + [[sns_performance_ads]]

**활용 예시**:
```
현재: "지중해 크루즈 예약 가능"
↓ (희소성 적용)
개선: "지중해 크루즈 - 남은 배치 2개 (원래 20개)
       선착순 마감까지 2일 12시간 남음"

자동화:
- 배치 감소 감지 → SMS Day 0 발송
- 마감 24시간 전 → SMS 긴급 알림
- 남은 배치 1개 → 전화 직접 연락 (콜센터)
```
