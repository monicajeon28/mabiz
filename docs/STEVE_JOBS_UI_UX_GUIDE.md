# Steve Jobs: 50대 친화적 UI/UX 설계 가이드
**마비즈 CRM 전용 | 2026-06-15**

---

## 📌 목표

모든 UI/UX는 **"초등학생도 쓸 수 있을 정도로 간단"**하게 설계. 특히 50대 사용자 경험 최적화.

### 핵심 수치 기준
```
50대 가독성 기준
├─ 동공축소 (노안): 글자 16px 이상 필수
├─ 손가락 굵음: 버튼 48px × 48px 최소
├─ 반응시간 증가: 피드백 0.2초 이내 (안 그럼 먹통으로 느낌)
└─ 정보 처리 느림: 화면당 4개 항목 이하 (혼동 방지)
```

## 🎨 Section 1: 타이포그래피 (글자)

### 글자 크기 기준표

| 목적 | 크기 | 라인높이 | 색상 | 예시 |
|------|------|---------|------|------|
| **페이지 제목** | 24-28px | 1.2 | #1A1A1A | "자동 메시지 설정" |
| **섹션 헤드** | 20px | 1.3 | #1A1A1A | "💰 가격이 비싼 고객" |
| **본문 (읽을 글)** | 16px | 1.6 | #333333 | "이런 고객에게..." |
| **설명/도움말** | 14px | 1.5 | #666666 | "📝 선택 후 미리보기 확인" |
| **라벨** | 14px | 1.4 | #1A1A1A | "이름:" "전화번호:" |
| **폼 입력창** | 16px | 1.5 | #333333 | 입력 텍스트 |

### 색상별 대비도 (WCAG AA 기준)

```
배경: #FFFFFF (흰색)

텍스트 색상별 명암비:
✅ #1A1A1A (진검정) + 흰색 배경 = 대비도 21:1 (최상)
✅ #333333 (검정) + 흰색 배경 = 대비도 12:1 (우수)
✅ #666666 (진회색) + 흰색 배경 = 대비도 7:1 (우수)
⚠️ #888888 (중회색) + 흰색 배경 = 대비도 5.4:1 (경계)
❌ #AAAAAA (연회색) + 흰색 배경 = 대비도 3.3:1 (불합격)
```

### 금지 사항

```
❌ "FunnelSmsConfig Selector 설정"
   → 기술용어 (50대가 모름)

✅ "자동 메시지 어떤 고객한테?"
   → 일상 한글 (모두 이해)

❌ 11px 폰트
   → 50대 가독성 불가능

✅ 16px 폰트 (최소)
   → 모두 읽을 수 있음

❌ 회색 배경 + 검정 글자 (명도 부족)
   → 밝은 배경 + 진검정 (명도 충분)
```

## 🖱️ Section 2: 버튼 & 터치 (클릭)

### 버튼 크기 (모든 화면)

```
┌─────────────────────────────────────┐
│ 권장: 48px × 48px                  │
│ (성인 손가락폭 8-10mm = 48-56px)  │
│                                     │
│ [응! 해주기]  48px 높이            │ ← 기본 버튼
│                                     │
│ [선택하기 ▼]  48px 높이            │ ← 드롭다운
│                                     │
│ 💾 [저장하기]  48px 높이, 아이콘  │ ← 아이콘 버튼
│                                     │
│ ☐ 안함  ⭕ 응!                     │ ← 라디오 48px
│                                     │
└─────────────────────────────────────┘

❌ 작은 버튼 (40px):
┌─────────────┐
│ [버튼]      │ ← 50대가 놓치기 쉬움
└─────────────┘

✅ 큰 버튼 (48px):
┌──────────────────┐
│ [버튼]           │ ← 확실히 클릭
└──────────────────┘
```

### 버튼 상태 (피드백)

```
기본 상태
┌──────────────────────────────────────┐
│ [확인하기] (파란 배경, 흰 글자)      │
└──────────────────────────────────────┘

호버 상태 (마우스 올림)
┌──────────────────────────────────────┐
│ [확인하기] (더 진한 파란색 배경)     │ ← 색상 변경
│ 커서 손가락 모양 👆                 │
└──────────────────────────────────────┘

포커스 상태 (Tab 키 누름)
┌──────────────────────────────────────┐
│ ┌─ [확인하기] ─┐ (2px 테두리)       │ ← 키보드 표시
│ └──────────────┘                   │
└──────────────────────────────────────┘

활성화 상태 (클릭함)
┌──────────────────────────────────────┐
│ [확인하기] (더 진한 색, 눌린 느낌)  │
│ ⏳ 로딩 중... (0.5초)              │
└──────────────────────────────────────┘

완료 상태 (작업 완료)
┌──────────────────────────────────────┐
│ [✅ 저장 완료!] (초록색, 체크 아이콘) │ ← 2초 후 원래대로
└──────────────────────────────────────┘

비활성화 상태 (클릭 불가)
┌──────────────────────────────────────┐
│ [다음] (연한 회색, 커서 불가능)      │ ← 클릭 못함
└──────────────────────────────────────┘
```

```

## 🎨 Section 3: 색상 (시각 계층)

### 렌즈별 색상 시스템 (심리학 통합)

각 렌즈(L0-L10)는 고유 색상으로 시각적 구분:

```
L0: 부재중 고객 (재활성화)
├─ 주색: #9B59B6 (보라색)
├─ 배경: #F3E5F5 (연보라색)
├─ 사용처: "휴면 고객 복구" 섹션
└─ 심리학: 수치심 (잊혀진 느낌) → 재연결 제안

L1: 가격 이의 (할부)
├─ 주색: #FFD700 (황금색)
├─ 배경: #FFFACD (연황색)
├─ 사용처: "월 60만원 할부" 카드
└─ 심리학: 욕망 (저렴함) → 선택 유도

L3: 경쟁사 비교 (차별성)
├─ 주색: #4A90E2 (파란색)
├─ 배경: #EBF4FF (연파란색)
├─ 사용처: "우리 vs Royal 비교표"
└─ 심리학: 이성 (비교) → 합리적 선택

L6: 타이밍/손실 (긴박감)
├─ 주색: #E74C3C (빨간색)
├─ 배경: #FADBD8 (연분홍색)
├─ 사용처: "⏰ 남은 자리 2석만!" 배너
└─ 심리학: 손실회피 (빠질까봐) → 지금 신청

L10: 즉시 구매 (클로징)
├─ 주색: #27AE60 (초록색)
├─ 배경: #D5F4E6 (연초록색)
├─ 사용처: "[지금 신청하기]" 메인 버튼
└─ 심리학: 행동 (확신) → 결제 진행
```

### 다크모드 색상 변환 (밤 10시 이후)

```
라이트 모드 (낮):
배경: #FFFFFF (흰색)
텍스트: #1A1A1A (진검정)
카드: #F5F5F5 (연회색)

다크 모드 (밤):
배경: #1A1A1A (진검정)
텍스트: #FFFFFF (흰색)
카드: #2D2D2D (진회색)

색상 변환 공식:
다크모드 = 주색 - 20% 명도 + 10% 채도
예: 황금색(#FFD700) → 진황색(#E6C200)
```

### 여백 규칙 (Spacing Scale)

```
전체 화면 구조:

┌────────────────────────────────────┐
│ 페이지 제목 "자동 메시지 설정"     │ ← 상단 24px 여백
├────────────────────────────────────┤
│                                    │
│ ┌──────────────────────────────┐  │ ← 섹션 간격 24px
│ │ 섹션 1: 💰 가격이 비싼 고객  │  │
│ │ 패딩 16px                    │  │
│ │                              │  │ ← 요소 간격 16px
│ │ [라디오 버튼] 48px           │  │
│ │                              │  │
│ │ 설명: "이런 고객이..."        │  │
│ │ (14px 텍스트)                │  │
│ │                              │  │
│ │ [드롭다운] 48px               │  │
│ └──────────────────────────────┘  │ ← 섹션 간격 24px
│                                    │
│ ┌──────────────────────────────┐  │
│ │ 섹션 2: ⚖️ 다른 회사 비교    │  │
│ │ (동일 구조...)               │  │

- 한 화면 최대 높이: 2000px (스크롤 권장)

- 요소 간: 16px
- 카드 내부 패딩: 16px

## ⚡ Section 6: 피드백 & 미리보기 (반응속도)

### 반응 시간 기준

```
0ms ────── 0.2초 ────── 1초 ────── 3초 ────── 5초+

즉시 반응   빠른 반응   조금 느림   느림      매우 느림
(최고)     (좋음)     (낮음)     (나쁨)    (최악)

구체 사례:

✅ 미리보기 선택 → 즉시 메시지 표시 (0ms)
✅ 버튼 클릭 → 색상 변경 (0.2초)
⚠️ 폼 제출 → 서버 저장 (1-2초 - 로딩 스피너 필수)
❌ 페이지 로드 → 3초 이상 (사용자 이탈 시작)
❌ 응답 없음 (5초+) → 먹통으로 느낌
```

### 미리보기 UI 패턴

```
선택하기 → 실시간 미리보기

┌─────────────────────────────────────┐
│ 어떤 메시지? [▼ 할부제안 ▼]        │ ← 선택
│                                     │
│ 📱 실제로 이렇게 보내요: (미리보기) │
│ ┌─────────────────────────────────┐│
│ │ Day 1 (내일)                   ││
│ │ "좋은 소식입니다!"              ││
│ │                               ││
│ │ 월 60만원씩                    ││
│ │ 5개월 할부 가능                ││
│ │                               ││
│ │ [자세히 보기]                  ││
│ └─────────────────────────────────┘│ ← 즉시 반영 (0ms)
│                                     │
│ 📧 이메일 버전도 미리보기 (탭)      │
│ ┌─────────────────────────────────┐│
│ │ [이메일 미리보기]               ││
│ └─────────────────────────────────┘│
│                                     │
└─────────────────────────────────────┘
```

## ✅ 배포 전 검증 체크리스트

### 타이포그래피
- [ ] 제목: 20px 이상 (#1A1A1A)

- [ ] 설명: 14px 이상 (#666666)
- [ ] 모든 텍스트 명확한 한글 (기술용어 0)
- [ ] 색상 대비 4.5:1 이상

- [ ] 모든 버튼: 48px × 48px 최소
- [ ] 라디오/체크박스: 48px × 48px
- [ ] 드롭다운/입력창: 48px 높이 이상
- [ ] 버튼 간격: 16px 이상
- [ ] 호버 피드백 (색상 변경) 명확
- [ ] 포커스 피드백 (테두리) 명확
- [ ] 키보드 탭 네비게이션 가능

## 💡 자주 틀리는 실수 (Do's & Don'ts)

### ❌ 절대 금지

```
❌ "FunnelSmsTemplate Selector Configuration"
✅ "자동 메시지 어떤 고객한테?"

❌ 11px 폰트 (50대 불가능)
✅ 16px 폰트 (최소)

❌ 40px 버튼 (놓치기 쉬움)
✅ 48px 버튼 (명확)

❌ 회색 배경 + 검정 글자 (명도 부족)
✅ 흰색 배경 + 진검정 글자 (명도 충분)

❌ 선택 후 3초 딜레이 (응답 없음 같음)
✅ 즉시 반영 (0.2초 이내)

❌ 한 화면 10개 항목 (혼동)
✅ 한 화면 4개 항목 (명확)

❌ "또는 이동" (모호함)
✅ "[다음으로] / [이전으로]" (명확)

❌ 기술용어: "Query Parameter, Hook, Render"
✅ 일상 한글: "검색 조건, 시작, 화면 새로고침"
```

### ✅ 필수 포함

```
✅ 모든 입력창: 라벨 + 도움말 + 예시
✅ 모든 액션 버튼: 확인 필수 (삭제 시)
✅ 모든 오류: 빨간색 + 상세 설명
✅ 모든 성공: 초록색 + 체크마크
✅ 모든 로딩: 스피너 + "저장 중..." 텍스트
✅ 모든 선택: 미리보기 즉시 반영
```
