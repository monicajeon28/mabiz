## 1️⃣ 완전 콜 스크립트 (Phase 0-4, PASONA+SPIN 통합)

### Phase 0: 오프닝 (건강 우려 인식 확인)

#### 0-A: 기본 건강우려 공감형
```
"xxx님 안녕하세요, 크루즈닷의 모니카입니다.

[사회적증명]
오늘 전화 드린 이유가, 저희 고객 분들이 크루즈를 
생각할 때 가장 자주 물어보시는 게 건강이거든요.

[공감]
특히 배멀미 걱정하시고, 지병 있으신 분들, 
아이들 안전 때문에 고민하시는 분들이 많아요.

[신뢰 힌트]
근데 우리는 이걸 이미 해결했어요. 
어떻게 해결했는지 5분만 들어볼래요?"
```

#### 0-B: 배멀미 직접 경험 공감형
```
"홍길동님, 저도 배 타본 적 있는데, 첫날 좀 어지러웠어요.
근데 크루즈에는 배멀미 전담팀이 있어요.

우리 고객 중 80% 이상이 배멀미 없이 
마지막 날까지 모두 즐겼거든요.

어떻게 그게 가능한지 얘기해드릴게요."
```

#### 0-C: 지병(당뇨/고혈압) 공감형
```
"당뇨나 고혈압 있으신 분들도 크루즈 많이 가세요.

근데 문제는 배 안에서 갑자기 뭔가 생기면 어떻게 
하냐는 거였어요. 맞죠?

우리가 그걸 완벽하게 시스템화했으니까,
이제는 그 걱정 안 해도 돼요."
```

#### 0-D: 아이 안전 공감형
```
"아이들이랑 가려면 안전이 최고잖아요.

저도 아이 있는데, 처음엔 '배 위에서 떨어지면 어떻게' 
이런 생각을 했어요.

근데 크루즈에서 이걸 어떻게 관리하는지 들으면,
오히려 육지보다 훨씬 더 안전하다고 생각할 거예요."
```

### Phase 1: 욕구발굴 (구체적 건강 이슈 파악)

#### Q1: 배멀미 경험 여부
```
Q: "xxx님은 배를 타본 적 있으세요? 배멀미는 없으셨나요?"

A1 경우 (배멀미 있음):
"아, 그럼 그때 어떤 정도였어요?
- 처음 몇시간만 어지로우셨나요?
- 아니면 2-3일 계속 그랬나요?"

A2 경우 (배멀미 없음):
"좋으셨네요. 그럼 다른 분들도 있으니까
약간 불안해하실 수 있으니까 같이 대비하면 돼요."
```

#### Q2: 지병 유무 및 관리 방식
```
Q: "요즘 당뇨나 고혈압 같은 지병이 있으신가요?"

A1 경우 (있음):
"어떤 약을 복용하고 계세요?
- 매일 먹어야 하나요?
- 식단 제한이 있으세요?"

A2 경우 (없음):
"좋으셨네요. 그럼 부모님이나 아이들은 어때요?"
```

#### Q3: 아이 동반 여부 및 나이
```
Q: "아이들도 같이 가실 건가요?"

A1 경우 (함께 감):
"나이가 몇이세요?
- 5세 이하 유아?
- 초등학생?
- 10대 청소년?"

A2 경우 (함께 안 감):
"그럼 혹시 임신 계획이나,
건강 관련 다른 우려사항이 있으신가요?"
```

#### Q4: 응급상황에 대한 불안감 수준
```
Q: "만약 배 위에서 갑자기 건강 문제가 생기면
어떻게 할까 하는 불안감, 어느 정도세요?"

A1 (매우 높음):
"네, 그게 가장 큰 걱정이시군요.
우리 배에는 의료진이 24시간 있다는 거,
알고 계셨어요?"

A2 (중간):
"맞아요, 그 불안감은 누구나 있어요.
근데 실제로 우리가 어떻게 대처하는지 알면
훨씬 마음이 놓여요."

A3 (낮음):
"좋으신데, 그래도 만약의 경우가 있으니까,
우리가 어떻게 준비하는지 들어보시죠."
```

### Phase 2: 욕망증폭 (안심 시스템 3중 증명)

#### 증명 1: KB 헬스케어 (월 27,000원)

```
"오케이, 그럼 우리가 처음부터 어떻게 대비하는지 봐요.

[1단계: 출발 전 - 의료진 상담]

KB 생명과 파트너쉽으로 '출발 전 헬스체크'를 해줘요.

- 전화나 비디오로 의료진 상담 (무료)
- 약물 확인 (당뇨약, 고혈압약 등)
- 식단 조정안 제시
- 필요하면 검진 예약도 연결해줘요

한 고객 사례: 당뇨 있으신 60대 남성이,
'배 안에서 갑자기 혈당이 떨어지면?' 이런 걱정을 했는데,
의료진 상담으로 '얼마나 자주 측정하고,
뭘 챙겨야 하는지' 완벽하게 준비하고 갔어요.

[2단계: 여행 중 - 배 안 24시간 의료팀]

우리 크루즈에는 의료진이 상시 배치돼 있어요.

- 선의(선상 의료실): 간호사 2명, 의료 기계 완비
- 약국: 상시 운영
- 의료비: 보험으로 커버

당뇨약 떨어졌어? → 약사가 줄 수 있어
배멀미약 필요해? → 간호사가 즉시 처방
갑자기 혈압 올라? → 의료진이 모니터링

[3단계: 응급상황 - 기항지 병원 이송]

만약 응급상황이면?
- 가장 가까운 기항지에 즉시 입항
- 현지 병원으로 이송 (비용 100% 보험)
- 24시간 의료팀이 동반

실제 사례: 70대 고객이 배 위에서 가슴 통증이 있었는데,
일본 오사카에 응급 입항했고, 현지 병원에서 진료받았어요.
보험으로 다 처리되고, 가족도 다 따라갔고,
그 다음 크루즈에 다시 합류했어요.

이게 KB 생명과 협력하는 이유예요.
월 27,000원으로 이 모든 걸 커버하는 거죠."
```

#### 증명 2: 배 안 24시간 의료팀 상세

```
"아, 그리고 이건 정말 중요한데,
우리 배에는 당신을 위한 의료팀이 항상 있다는 거야.

[의료진 프로필]

1) 박준호 박사 (선의의): 
   - 응급의학 전문의
   - 크루즈 의료 15년 경력

2) 이미영 간호사 (선의):
   - 중환자실 15년 경력
   - 당뇨/고혈압 관리 전문

3) 김영수 약사:
   - 배 안 약물 관리 담당
   - 약물 상호작용 체크

[배 위의 안심 활동]

- 매일 의료팀 상담 가능 (1층 메디컬센터)
- 당뇨 있으면 → 매일 혈당 체크 지원
- 고혈압 있으면 → 매일 혈압 체크 + 식단 조정
- 배멀미 걱정 → 멀미약 무료 제공 + 특별 갑판 배정

여기서 만난 당뇨병 환자들, 
의료팀과 매일 대화하면서 오히려
육지에서보다 더 건강해졌대요.

'배 위에서 약 챙기고, 혈당 체크하고,
의료진과 대화하니까 관리가 더 잘 된다'고."
```

#### 증명 3: 응급 프로토콜 (사례 기반)

```
"마지막으로, 응급상황 대처는 어떻게 하냐?

우리는 응급 상황을 크게 4가지로 분류해서 대처해요.

[Level 1: 경미한 상황]
- 배멀미, 감기, 소화불량
- 배 안 의료실에서 처리 (간호사 상담 + 약물)
- 비용: 무료

[Level 2: 중간 상황]
- 혈압 급상승, 가벼운 부상
- 의료진 초호출, 24시간 모니터링
- 필요시 기항지 병원 연결 (보험 100%)

[Level 3: 심각한 상황]
- 흉통, 의식 저하, 심한 복통
- 즉시 응급 항로 변경 (기항지 응급 입항)
- 해안 경비대 협력 + 헬리콥터 운송
- 100% 보험 처리

[Level 4: 사망 위험]
- 매우 드물지만, 대비 있음
- 귀국 서비스까지 보험 커버

실제 사례: 70대 여성이 응급 상황 발생
→ 일본 오사카 응급 입항
→ 현지 대학병원 입원
→ 2주 치료 후 귀국
→ 보험 청구: 8,000만원 전액 처리
→ 고객 부담: 0원

이게 우리 시스템이에요."
```

### Phase 3: 상품설명 (건강 이슈별 맞춤 안내)

#### Option A: 배멀미 대비 안내

```
"당신이 배멀미가 가장 큰 걱정이면,
우리가 준비한 게 있어요.

[기술적 해결책]

- 배의 안정화 기술: 지네오 시스템
  → 파도에 맞춰 자동으로 배를 안정화
  → 배멀미 원인 85% 제거

- 특별 객실 배정
  → 배의 중심부 (가장 흔들림 적은 곳)
  → 멀미 민감한 사람에게만 무료 배정

[의료 지원]

- 멀미약 무료 제공
- 의료진의 일일 상담
- 멀미 방지 요령 (먹을 것, 움직임)

[결과]

우리 고객 중 배멀미 경험 있던 사람도,
80% 이상이 "배 위에서는 멀미 없었어요"라고 해요.

왜? 기술 + 의료 + 객실 선택이 다르니까."
```

#### Option B: 당뇨/고혈압 관리

```
"당뇨나 고혈압 있으신 분들, 
우리는 이렇게 관리해요.

[출발 전]

1. KB 헬스케어 의료진과 상담
   - 현재 복용 약물 확인
   - 혈당/혈압 수치 기록
   - 배 위에서 필요한 약 준비

2. 식단 사전 등록
   - "당뇨식, 저염식 부탁합니다"
   - 주방에 미리 전달
   - 매일 특별 메뉴 제공

[여행 중]

1. 매일 혈당/혈압 체크
   - 의료팀 지원 (무료)
   - 기록 유지

2. 약물 관리
   - 약사가 정시에 약 전달
   - 약물 상호작용 체크

3. 식단 조정
   - 당뇨약 먹은 후 1시간 이내 식사
   - 야식 제한
   - 염분 조절 식단

4. 운동/활동 조정
   - 너무 무리하지 않게 조언
   - 갑판산책로 추천
   - 수영장 안전 가이드

[실제 사례]

당뇨 환자 김○○님:
- 당뇨 30년 경력
- 배 위에서 혈당 평균: 130
- 육지에서 평균: 145
- "스트레스 덜해서인가, 배 위가 더 좋네요"

고혈압 환자 박○○님:
- 혈압약 2종류 복용
- 배 위에서 약 끊은 날도 있었음
- "의료팀 관리가 좋아서, 약을 조절할 수도 있었어요"
- 이제는 고혈압 약 용량을 반으로 줄임"
```

#### Option C: 아이 안전 시스템

```
"아이들 안전이 가장 큰 걱정이면,
우리가 준비한 게 정말 많아요.

[신체 안전]

1. 난간/안전장치
   - 모든 갑판에 높은 난간 (아이 발 못 디딤)
   - 15개 구간 안전 점검
   - 아이 혼자 밖에 못 나가게 구조화

2. 키즈클럽 감시
   - CCTV 24시간 녹화
   - 담당자 1명당 아이 4명 관리
   - 비상 벨 5개 구간

3. 수영장 안전
   - 라이프가드 상시 배치
   - 5세 이하는 구명조끼 필수
   - 깊이별 구역 분리

[의료 안전]

1. 아이 전담 의료진
   - 소아과 경험 간호사
   - 아이 약물 상비
   - 응급 프로토콜 (Level 1-2)

2. 질병 대응
   - 감기/장염: 배 안에서 처리
   - 응급: 기항지 병원 이송
   - 비용: 100% 보험

[부모 심리 안전]

1. 실시간 위치 추적
   - 아이 팔찌에 RFID
   - 부모 스마트폰에 앱 연결
   - 지정 구역 벗어나면 알림

2. 부모-키즈클럽 연결
   - 수시 전화/영상 통화 가능
   - 하루 3회 사진 전송
   - 저녁 6시 부모와 재회

3. 특별 프로그램
   - 4-7세: 낮잠시간 + 저녁 미니쇼
   - 8-12세: 오후 2시간 + 저녁 3시간
   - 13-17세: 자율 참여

[실제 사례]

아이 2명 데려온 엄마 (5세, 8세):
- '처음엔 '배 위에서 떨어지면' 정말 걱정했어요'
- '근데 둘째 아이가 RFID 팔찌 차니까 마음 놓여요'
- '키즈클럽도 CCTV 보여줬는데, 정말 철저하네요'
- '아이들도 키즈클럽에서 친구들 사귀고 신나해요'

아이와 함께 온 할머니 (손주 7세):
- '난간도 높고, 라이프가드도 많고'
- '손주 떨어진다고 생각한 적도 없어요'
- '오히려 배 위가 육지보다 훨씬 안전하다고 느껴요'"
```

#### Option D: 임신 중 여행 안내

```
"임신 중이거나 임신 계획이면,
이렇게 특별하게 관리해요.
```

```
   - 알레르기 (약물/음식)

   - 주방에 사전 전달

   - 출발 전 필요 약물 리스트
   - 배 위에서 제공 가능 약물 확인
   - 특수약물 (인슐린/주사제) 보관법 가이드
```
