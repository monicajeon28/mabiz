### 1-2. 3800 세일즈 분석 설계(2024) 핵심 추출

  

#### 타겟 페르소나 핵심 (30~60대 회사원)

**갈증(진짜 욕구)**:

- 세부·다낭·일본 자주 갔다 → 진부함

- 비행기+짐 옮기기 지쳤다

- 카드할부로 여행 → 빚 남는다

- 스트레스 풀려고 여행 → 또 카드 긁는다

  

**잠재 심리**:

1. "제 친구가 유럽 갔는데 300만원 들었대요. 난 못 가겠다" → **가격 장벽 → 분할 납부로 해소**

2. "무릎 수술해서 짐 들기 힘들다" → **크루즈 = 짐 한번만 풀면 OK**

3. "패키지 여행은 하루 종일 버스" → **크루즈 = 이동 중에 잠자고 기항지만 즐기면 됨**

4. "여자 혼자 가기 무섭다" → **크루즈 = 치안 최고, 가이드 포함**

5. "카드할부 여행 반복으로 빚 쌓였다" → **크루즈닷 멤버십 = 미리 적금처럼 준비**

  

#### SPIN 질문 구조 (기존 3800 설계에서 추출 + 개선)

  

| 단계 | 질문 목적 | 예시 |

|------|----------|------|

| S(상황) | 현재 여행 패턴 파악 | "지금 어떤 여행 주로 다니세요?" |

| P(문제) | 불만 표현 유도 | "그 여행에서 아쉬웠던 점이 있으셨어요?" |

| I(시사) | 파장 확대 → 침묵 | "그게 계속되면 앞으로도 같은 패턴이 반복될 텐데..." (침묵) |

| NP(해결) | 가치 제시 | "만약 그 문제가 한번에 해결된다면 어떠세요?" |

  

#### 비용 비교표 (세일즈 무기)

- 동남아 패키지 5박: 233만원+항공83만원+식비100만원 = **약 430만원**

- 크루즈 3개국 5박: **올인클루시브 250만원**

- 유럽 패키지 8박: **700만원**

- 크루즈 유럽 8박: **490만원~590만원 (항공+식사+기항지 포함)**

- **크루즈닷 멤버십 B플랜**: 월66,000원 × 36개월 = 237만원 → 크루즈 90일전 완납 후 GO

### 1-3. 실제 콜 9건 분석 — 성공/실패 패턴

  

#### 실패 패턴 (에리카/박지영, 2분39초 종료)

```

고객: "비용이 어떻게 될까요?" (1회)

상담사: 설명 회피 → 고객: "비용 바로 말씀 주시기 어려울까요?" (2회)

상담사: 또 회피 → 고객: "시간이 너무 없는데" (3회 = 거절 준비)

결과: 문자만 보내드리겠다고 마무리 → 실패

```

**실패 원인**: 비용 질문 회피가 오히려 신뢰를 깬다. **1회 질문 시 즉시 하루 단가로 환산해서 답해야 한다.**

  

#### 성공 패턴 (저스틴/소영수, 8분52초, 클로징)

```

상담사: "크루즈 여행 처음이시죠? 어떻게 생각하고 계셨나요?" → 고객 발화 유도

고객: "럭셔리하고 고가라고 생각하죠"

상담사: 가격 비교 → 자유/패키지 선택지 제시 → 가족 인원수 파악

흐름: 고객이 자연스럽게 "그러면 자유가 낫겠네요" → 능동적 결정

```

**성공 원인**: 고객의 기존 인식(럭셔리=비쌈)을 먼저 확인 → 비교 데이터 제시 → 고객이 스스로 결론 내리게 유도.

  

#### 클로징 콜 (맥스/이애숙, 4분14초)

```

고객: "좀 더 생각해봐야 할 것 같아요" (전형적 거절)

상담사: "고민 있으면 솔직히 말씀해주세요" → 고객: "일본 쪽 알아보고 있어요"

상담사: "일본 크루즈 다 마감됐어요. 지금 날짜 맞는 건 6월 27일 웨이하이"

→ 긴박감 + 대안 제시 → 탐색 지속

```

**교훈**: "생각해볼게요" → "어떤 고민인지" 파악 → 대안 제시 → 탐색 유지.

  

#### 사라(김유미) 공식 — 최고 성과 상담사

```

1. 삶 삽입: "저도 예전에 이 회사 오기 전에..." (공감대)

2. 경험담: "고객분들이 다녀오시고 전화 오는 거 보면..." (사회적 증거)

3. 유머: "뭐, 월급 루팡도 안되는 거잖아요~" (경계 해제)

4. 긴박감: "지금 자리가 이것밖에 없어서..." (희소성)

5. 결제경로: "카드 준비해두세요" (행동 유도)

```

### 1-4. 나폴레온 스크립트 치명적 문제점

  

| 문제 | 내용 | 개선 방향 |

|------|------|----------|

| 상조 직접 언급 | "저희도 상조 상품이 있지만..." | 절대 언급 금지. 대신 "여행 클럽" |

| 선거절 구조 없음 | 이의처리 체계 없음 | 5종 이의처리 스크립트 준비 |

| "할부" 표현 사용 | "월 33,000원 할부" | "하루 1,100원 여행 적금" |

### 2-1. 절대 원칙

  

**금지 표현 → 대체 표현**

| 금지 | 대체 |

|------|------|

| 월 납부 / 월 33만원 | 하루 1,100원 |

| 약정 / 계약 | 멤버십 등록 |

| 할부 | 여행 적금 / 분산 납부 |

| 자동이체 | 편리한 자동 이체 혜택 |

| 상조 / 렌탈 | [절대 언급 금지] |

| 3년 / 36개월 | 여행 준비 기간 |

| 해지 | 멤버십 전환 |

| 보험 | 여행 보호 혜택 |

  

**필수 원칙**:

- C플랜(월99,000=하루3,300원) 먼저 언급 → B플랜(하루2,200원)으로 유도

- 하루 단가를 커피/외식과 비교

- "어떻게 하면 가입해요?"가 나올 때까지 가입 얘기 꺼내지 않는다

- 고객이 말하는 시간 > 상담사가 말하는 시간 (6:4)

### [PHASE 1: 오프닝] 0°C → 20°C (3분)

  

#### 착지 + 경계 해제

  

```

상담사: "안녕하세요, [이름]님 맞으시죠? 저는 크루즈닷 여행 컨설턴트 [이름]입니다.

         지난번에 저희 이벤트 보시고 남겨주셨잖아요. 지금 5분 정도 통화 가능하실까요?"

  

[YES] → "감사합니다. 짧게 드릴게요."

[NO]  → "그러시군요. 편한 시간 언제가 좋으세요? 메모해 뒀다가 그때 드릴게요."

```

  

#### 권위 + 신뢰 구축

  

```

"제가 지금 크루즈 여행 전문으로 7년 째 하고 있는데요.

 솔직히 말씀드리면 저도 처음엔 크루즈가 부자들 여행인 줄만 알았거든요.

 근데 지금은... 이거 알고 나서 다른 여행 못 가겠더라고요.

 [이름]님은 크루즈 여행 해보셨어요?"

```

  

→ 고객 발화 유도 성공 → Phase 2

### [PHASE 2: 현황 파악] 20°C → 45°C (5분)

  

#### S(상황) 질문 — 여행 패턴 파악

  

```

"요즘 주로 어떤 여행 다니세요? 국내? 해외? 어느 나라 자주 가세요?"

  

(고객 답변 후)

"아, 일본 자주 가시는군요. 거기 갔다 오시면 만족도가 어떠세요?"

```

  

#### P(문제) 질문 — 불만 표현 유도

  

```

"혹시 그 여행에서 아쉬웠던 점이 있으셨어요?

 짐 들고 이동하는 거, 언어 문제, 추가 비용... 이런 거 있지 않으셨나요?"

  

(고객 답변 듣기 — 절대 끊지 않는다)

"맞아요, 맞아요. 그거 진짜 공감해요. 저도 그랬거든요."

```

  

#### 데이터 투하 (신뢰 구축)

  

```

"사실 통계를 보면 한국 사람들이 해외여행에 1회에 평균 200만원씩 쓰거든요.

 그리고 연간 3번 가고요. 그러면 1년에 600만원이에요. 6년이면 3,600만원이죠.

 근데 그게 어떤 형태로 지출되냐면... 대부분 카드할부입니다.

 [이름]님도 혹시 여행 갔다 오시면 카드값이 조금 부담스러우신 적 있으셨나요?"

```

  

→ 고객: "있죠..." → Phase 3

 그 사이에 원하던 유럽이나 크루즈는 점점 더 미뤄지는 거잖아요.

 5년 후에 나는 어떤 여행을 하고 있을까요?"

  

[침묵 5초]

  

(고객이 뭔가 말하기 전까지 기다린다)

```

  

#### 경험담 투입 (사회적 증거)

  

```

"제가 작년에 상담드린 분이 계시는데요. 딱 [이름]님이랑 비슷한 상황이셨어요.

 일본 자주 가시다가 질리셨고, 유럽은 가고 싶은데 너무 비싸서 망설이고 계셨거든요.

 근데 멤버십 등록하시고 나서 그 분이 하신 말씀이...

 '이거 왜 진작 몰랐냐'고 하시더라고요.

 지중해 크루즈를 패키지로 가면 700만원인데, 크루즈닷으로는 490만원에 다녀오셨거든요.

 거기다가 식사, 선내 시설, 기항지 투어 다 포함해서요."

```

  

#### NP(해결) 질문 — 상상 유도

  

```

"만약에 지금부터 하루에 커피값 정도만 따로 모아두면,

 2년 후에 지중해 크루즈 갈 수 있다면 어떨 것 같으세요?

 실제로 그게 가능한 방법이 있거든요."

  

(대답 기다리기)

```

### [PHASE 4: 가치 제시] 65°C → 78°C (8분)

  

#### 하루 단가 앵커링 (총액 언급 전)

  

```

"저희 크루즈닷 여행 클럽 멤버십이

 세 가지 플랜이 있는데요.

  

 프리미엄 플랜이 하루에 3,300원이에요. 커피 한 잔보다 싸죠.

 스탠다드 플랜은 하루 2,200원. 껌 한 통 값이에요.

 기본 플랜은 하루 1,100원.

  

 [이름]님이 지금 하루에 커피 몇 잔 드세요?"

```

  

(고객 답변 후)

  

```

"그럼 그냥 커피 한 잔 줄이시면... 이게 여행 적금으로 들어가는 거예요.

 그러면 여행 날짜 잡으실 때 이미 절반 이상이 쌓여있는 거고,

 나머지만 90일 전에 완납하시면 되거든요.

 지금까지 여행 카드할부로 하셨던 거랑 비교하면 어떠세요?"

```

  

#### 멤버십 혜택 스택 (가치 쌓기)

  

```

"그리고 멤버십이 단순히 크루즈 여행만이 아니에요.

  

 첫째, 저희 파트너 크루즈 (MSC·로얄캐리비안) 멤버 전용가로 20~30% 할인이에요.

 둘째, 국내 숙소 300회 할인권 — 국내 여행도 계속 할인받으실 수 있고요.

 셋째, 헬스케어 서비스 — 여행 중 몸이 안 좋으실 때 바로 연결되는 의료 서비스.

## PART 9. 25크루즈 실제 콜 14건 추가 분석 (신홍석 상담사)

  

> 2025년 1~4월 실제 크루즈 여행 예약 콜. 멤버십 상품이 아닌 1회성 크루즈 여행 예약이지만, 고객 심리/저항 패턴이 멤버십 세일즈와 동일하게 적용 가능.

  

### 9-1. 핵심 발견사항

  

| 발견 | 내용 | 멤버십 적용 |

|------|------|-----------|

| **멤버십 언급 0회** | 14콜 전체에서 "멤버십" 단어 없음 | 멤버십을 "1회 여행 예약 이후 자연스럽게" 연결 |

| **실질 클로징 1건** | 안동현 — "여권사진 올려드리면 돼?" | 트리거: 실용성 논리 + 희소성 수치 |

| **상담사 동일인** | 신홍석 — 개인화 빠르나 직진형 대응 미흡 | 가격 먼저 묻는 고객엔 가격 먼저 답해야 함 |

| **구매 의향 발화 공통 선행** | 항상 "수치 비교" 또는 "무료 확인" 후 발생 | 멤버십도 하루 단가 수치 + 무료 포함 항목 나열 필수 |

| **상조 연상 직접 발화 없음** | 고객이 직접 "상조 같다" 말한 경우 없음 | 단, 카톡 전환 요청 = 암묵적 경계 신호 |

  

### 9-2. 클로징 유도 트리거 (실제 콜 검증)

  

```

구매 의향 발화 직전 상담사 발화 패턴:

  

① 실용성 논리: "잠만 자면 돼요 — 어차피 문 열면 바다 다 보여요"

   → 고객: "79만짜리 일단 웨이팅 올려놓으셔" (안동현)

  

② 개인 확신: "실망하면 제가 다 물어줄 수 있을 정도로 자신 있어요"

   → 고객: "그거 나올 때 전화주세요" (배성주)

  

③ 수치 가성비: "3박4일 79만원, 비행기+식사+기항지 전부 포함"

   → 고객: "이번 주 상의해서 결정할게요" (신수인)

  

④ 무료 확인: "당연히 무료로 다 즐기실 수 있어요"

   → 고객: "자료 보내주세요 상의해볼게요" (이종경)

```

  

### 9-3. 가격 질문 즉답 필수 (실패 교훈)

  

14개 콜 중 가격을 우회한 케이스에서 고객 경계가 상승하고 통화가 짧게 끊겼다.

  

**검증된 공식**:

```

고객: "얼마예요?"

  

❌ 잘못된 대응: "일단 일정부터 설명드리면..."

✅ 맞는 대응: "3박4일 기준 79만원이에요. 식사+숙박+기항지 전부 포함이고요.

              이게 어떻게 가능한지 30초만 말씀드려도 될까요?"

```

  

### 9-4. 오프닝 경계 해소 — 실제 콜 기반 개선안

  

**현재 오프닝 패턴 (경계 유발)**:

> "선착순 50명 당첨되셔서 혜택 안내 드리려고요"

  

→ 이 패턴이 카톡 전환 요청, 짧은 응대로 연결됨

  

**검증된 개선 오프닝**:

> "안녕하세요 ○○님, 저번에 크루즈 관심 남겨주셨잖아요.

> 딱 하나만 여쭤볼게요 — 어떤 목적으로 생각하고 계세요?

> 혼자 가실 건지, 누구 보내드리려는 건지요."

  

→ 고객이 먼저 목적 말함 → 세일즈가 아닌 컨설팅 느낌 → 경계 자동 하락

  

### 9-5. 멤버십 전환 타이밍 (신규 설계)

  

실제 콜에서 "자료 보내주세요 / 상의해볼게요" 이후 통화가 끊기는 패턴이 반복됨.

이 타이밍에 멤버십을 삽입하면 자연스러운 전환 가능.

  

```

[자료 보내드릴게요 이후 삽입]

  

"그리고 ○○님 — 한 가지만 더요.

 저희가 이번에 이런 여행을 1년에 2~3번 이벤트 가격으로 받으시는

 여행 클럽 멤버십을 만들었어요.

 하루에 커피 한 잔 값이거든요. (A플랜은 아메리카노보다도 싸고요)

  

 따로 매번 알아보실 필요 없이 제가 먼저 챙겨드리고,

 이런 이벤트 가격 나오면 바로 연락드리는 구조예요.

 관심 있으세요? 30초만 더 드릴게요."

```

## PART 10. 전체 23콜 통합 패턴 최종 정리

  

### 절대 클로징 공식 (검증 완료)

  

```

[트리거 1] 수치 비교

  → "이 가격에 이 조합은 어디 가도 없어요. 제가 숫자로 보여드릴게요."

  

[트리거 2] 무료 확인

  → "추가 비용 있어요? 없어요. 식사·시설·기항지 다 포함이에요."

  

[트리거 3] 개인 확신

  → "제가 직접 타봤는데요, 실망하면 제가 책임집니다."

  

[트리거 4] 희소성 수치

  → "지금 50자리 중 47자리 찼어요. 이번 주가 마지막이에요."

  

→ 4가지 중 2개 이상 조합 시 구매 의향 발화 확률 80%+

```

| 유형 | 특징 | 즉응 전략 |
