# Offer 패턴 1: 고급상품 비유 (럭셔리 이미지)

## 정의
럭셔리, 5성급, 왕실 수준의 이미지로 상품을 포지셔닝

## v13에서의 위치

### [Stage 4-1] 5감각 시각화 (세그먼트 D/E — 5060)
```
상담사:
"아침에 알람 없이 눈이 떠져요.
창문 열면 오늘은 알래스카 빙하가 앞에 있어요.
새파랗고 거대한 빙하가 천천히... 움직이는 게 보여요.
차갑고 신선한 공기, 아무 소리도 없고.
옆에 남편분이랑 같이 서 계신 거예요."
```

**패턴 분석:**
- 럭셔리 이미지 = "알래스카 빙하" (자연의 위대함) + "남편과 함께" (정서적 럭셔리)
- 감각화: 시각(빙하의 푸름) + 온도(차갑고 신선한) + 청각(아무 소리 없음)
- 심리 원리: **Narrative Transportation** — 고객이 그 순간에 몰입 → 가치감 자동 상승

---

### [Stage 4-1] 세그먼트 A (커플)
```
상담사:
"아침에 커튼 열면 창밖에 산토리니 파란 돔이 보여요.
발코니에서 커피 한 잔 마시면서
오늘 어디 갈까 둘이 이야기하는 거예요."
```

**패턴:**
- 럭셔리 기항지 명시 = "산토리니" (그리스 럭셔리 여행의 대명사)
- 발코니 + 커피 = 고급 숙박시설의 표준
- 효과: "우리 같은 커플도 가능한가?" 느낌으로 자신감 상승

---

### [Stage 4-2] 앵커링 1 — 호텔 비교
```
상담사:
"제주도 괜찮은 호텔 1박이 20~30만원이잖아요.
크루즈는 5성급 호텔이 매일 새 항구로 이동해줘요.
4박5일 두 분에 99만원인데,
숙박+3식+이동+엔터테인먼트 다 포함이에요."
```

**패턴:**
- 명시적 럭셔리 프레임 = "5성급 호텔"
- 움직이는 럭셔리 = "매일 새 항구로 이동" (호텔 업그레이드 효과)
- 앵커링 전략: 일반 호텔(20~30만원) vs 크루즈 5성급(같은 가격대)

---

### [Stage 4-2] 앵커링 3 — 올인클루시브 럭셔리
```
상담사:
"배 안에 레스토랑이 12개예요.
랍스터가 맛있으면 두 번 더 드셔도 추가 비용 없어요.
뮤지컬 공연, 피트니스, 수영장, 스파... 다 포함이에요."
```

**패턴:**
- 럭셔리 체험 구체화: "12개 레스토랑" + "랍스터" (고급 식재료 명시)
- 반복 가능한 럭셔리 = "두 번 더" (무제한 누림)
- 다중 감각 럭셔리: 미각(식사) + 시각(공연) + 촉각(스파)

# Offer 패턴 2: 두 가지 가격대 (골드 vs 일반)

## 정의
프리미엄/골드 vs 스탠다드 두 선택지로 가격 선택의 폭 제공

## v13에서의 위치

### [Stage 8-3] 이중선택 클로징
```
상담사:
"[고객성함]님, 그러면 두 분 성함으로 해드릴까요,
아니면 일단 [고객성함]님 한 분 먼저 시작하시겠어요?"

또는

"스탠다드 선실로 시작하시겠어요,
아니면 발코니 선실로 가시겠어요?"
```

**패턴:**
- Yes/No 회피 → "어떤 종류로 할지" 선택지만 제시
- 명시적 2가지: 스탠다드(저가) vs 발코니(프리미엄)
- 심리: **이중선택** — 구매를 전제하고 "어떻게"만 선택

---

### [Stage 4-2] 앵커링 기준표 (암시적 가격대 분화)
```
| 상품 | 코스 | 가격 |
|------|------|------|
| #18 | MSC 벨리시마 일본 4박5일 | 109만원~ |
| #22 | MSC 벨리시마 일본 3박4일 | 99만원~ |
| #79 | 코스타 세레나 일본·대만 5박6일 | 159만원~ |
| #91 | 보이저 알래스카 7박8일 | 260만원~ |
| #101 | 익스플로러 지중해 7박8일 | 270만원~ |
```

**패턴:**
- 진입형(#22 99만원) → 중급(#79 159만원) → 럭셔리(#101 270만원)
- 같은 배(벨리시마) 내에서도 박수로 가격 분화: 3박 vs 4박
- 기항지로도 가격대 분화: 일본(저가) vs 지중해(최고가)

| 가격민감 | 스탠다드 + 99만원 | 같은 가격 다른 박수 | 3박→4박 upgrade |
| 럭셔리 지향 | 발코니 제시 | 같은 발코니 다른 기항지 | 지중해 선택 |

# Offer 패턴 3: 특별 혜택 (한정, 시간제한)

## 정의
"오늘만", "이번 주까지", "마지막 좌석" 등 희소성/긴급성 강조

## v13에서의 위치

### [Stage 9-2] 멀티콜 시퀀스 — Day 7 (카톡 #3)
```
상담사:
"[고객성함]님, 솔직히 말씀드릴게요.
[상품명] 이번 주까지예요.
잔여 좌석 [N]석이고요.
제가 잡아드릴 수 있는 마지막 타이밍이에요.
3분만 통화 가능하세요? 📞"
```

**패턴:**
- 시간 제한: "이번 주까지" (구체적 마감일)
- 재고 제한: "잔여 좌석 [N]석" (한정성)
- 개인적 기회: "제가 잡아드릴 수 있는 마지막 타이밍" (특별 대우)
- 심리: **Scarcity** (Cialdini) — "지금 안 하면 영원히 못 한다"

---

### [Stage 8-2] 선거절 카드 — 지금 당장 아님
```
상담사:
"**지금 당장 아님**은 — 출발 30일 전까지 취소 가능하거든요.
지금은 '갈 수도 있다' 정도로만 결정하시면 돼요."
```

**패턴:**
- 시간 제한을 역발상: "지금은 작은 결정만 하면 된다" (진입 장벽 낮춤)
- 후속: "출발 30일 전까지 취소 가능" (리스크 제거)
- 효과: 결정 심리 대폭 완화 → 일단 "예약" 유도

---

### [Stage 9-3] 카톡 템플릿 — 마감 임박
```
[KT-03] 마감 임박:
"[고객성함]님, 솔직히 말씀드릴게요.
[상품명] 이번 주 [요일]까지예요.
잔여 좌석 [N]석이고요.
제가 잡아드릴 수 있는 마지막 타이밍이에요."
```

**패턴:**
- 명시적 마감일: "이번 주 [요일]"
- 숫자로 하강감 유도: "N석 남았다"
- 개인화: "제가 잡아드릴 수" (상담사-고객 관계 강화 후 긴급성 활용)

# Offer 패턴 4: 보장 (환불, 변경자유, 만족보장)

## 정의
위험 제거 보장: 100% 만족, 날씨 환불, 일정 변경 자유

## v13에서의 위치

### [Stage 8-2] 선거절 카드 3종 — 지금 당장 아님
```
상담사:
"**지금 당장 아님**은 — 출발 30일 전까지 취소 가능하거든요.
지금은 '갈 수도 있다' 정도로만 결정하시면 돼요."
```

**패턴:**
- 명시적 보장: "출발 30일 전까지 취소 가능"
- 심리적 안전장치: "갈 수도 있다 정도로만 결정"
- 고객 심리: "예약해도 30일 전까지는 책임 없다" → 진입 장벽 제거

---

### [Stage 8-4] 소프트 클로징
```
상담사:
"일단 자리만 먼저 잡아드릴게요.
계약금은 [금액]만원인데, 출발 [N]일 전까지 전액 환불돼요.
지금은 '갈 수도 있다' 정도로만 결정하시면 돼요. 어떠세요?"
```

**패턴:**
- 단계별 보장: "계약금 [금액]만" → "전액 환불 가능"
- 시간 보장: "출발 [N]일 전까지"
- 심리적 완화: "갈 수도, 안 갈 수도" (결정의 유연성)

---

### [Stage 6] 저항처리 — MO-08 "지금 돈이 없어요"
```
상담사:
"지금 당장 결제하시는 게 아니에요.
자리만 먼저 잡아두시는 거고,
잔금은 출발 60일 전이에요.
지금 계약금이 [금액]만원이에요."
```

**패턴:**
- 분리 보장: "계약금" vs "잔금" (시간 차이 = 자금 확보 기회)
- 구체적 기한: "잔금은 출발 60일 전"
- 심리: "지금 작은 결정 → 나중에 큰 결정" (의사결정 부담 경감)

# Offer 패턴 5: 상품 네이밍 (감정적 포지셔닝)

## 정의
상품명 자체로 욕망/감정 자극: "럭셔리 카리브해", "꿈의 지중해"

## v13에서의 위치

### [Stage 4-1] 5감각 시각화 — 직접적 명명
```
상담사:
"아침에 커튼 열면 창밖에 산토리니 파란 돔이 보여요."

"아침에 알람 없이 눈이 떠져요.
창문 열면 오늘은 알래스카 빙하가 앞에 있어요."
```

**패턴:**
- 기항지명으로 자동 포지셔닝
- "산토리니" = 럭셔리 신혼여행의 대명사
- "알래스카 빙하" = 평생 못 볼 자연의 위대함

---

### [Stage 5] 마이프렌드존 사례 — 감정적 네이밍
```
상담사 (세그먼트 D-1):
"저희 고객 중에 자녀분이 부모님 환갑 선물로 예약해드린 분이 계셨어요.
어머님이 알래스카 빙하 보시면서 우셨대요.
평생 TV에서만 보던 걸 눈앞에서 보시니까요."
```

**패턴:**
- "환갑 선물" = 인생 중요 이벤트와 연결
- "알래스카 빙하" = 평생 꿈
- 감정: 눈물(감동)로 상품가치 극대화

---

### [Stage 7] 5060 심리 — 미래 자아 시각화
```
상담사:
"[기항지] 노을 앞에 [배우자/자녀]랑 서 계신 거예요.
그 순간 어떤 느낌일 것 같아요?"
```

**패턴:**
- "[기항지] 노을" = 감정적 로맨스
- 배우자/자녀 동반 = 사랑하는 사람과의 추억
- 심리: 상품이 아니라 "꿈의 순간" 판매

# Offer 패턴 6: 제안 내용 전달 (포장)

## 정의
"(3박 4일)이면 당신도 즐길 수 있습니다" — 구체적 상품명과 조건 묶음

## v13에서의 위치

### [Stage 4-2] 앵커링 2 — 패키지 비교
```
상담사:
"일본 자유여행으로 4박5일 가시면
항공+호텔+식사+이동 따로따로 계산하면 얼마 될 것 같으세요?
(고객이 말하게 유도)
맞아요, 그 정도 나오죠.
저희는 그게 전부 포함인데 [가격]원이에요."
```

**패턴:**
- 구체적 조건: "4박5일" + "항공+호텔+식사+이동" (5가지 항목 명시)
- 비교값: 일반 패키지 예상가 vs 올인원 가격
- 심리: "따로 사면 X, 우리는 Y" (가치 극대화)

---

### [Stage 8-2] 선거절 카드 — 버튼업 멘트

# Offer 패턴이 가장 강력한 구간

## 최고 효율 구간: Stage 4 + Stage 8

### Stage 4 (상품소개 12~16분)
**특징:** 고객의 욕망이 최고조일 때 / 저항 전에 Offer 제시

1. 고급상품 비유 (5감각 앵커링)

3. 상품 네이밍 (기항지 감정화)
4. 제안 내용 전달 (포함 항목 구체화)
5. 쓸모 정보 (올인클루시브 장점)
6. 독창성 (12개 레스토랑, 이동 없음)

**심리 작용:** 욕망(Stage 3) + 구체적 이미지화(Stage 4) = 구매 결심 완성

**고객 반응:** "얼마예요?" (구매 신호 1호)

**특징:** 마지막 저항 해소 + 구매 결정 순간

**등장하는 Offer 패턴:**
1. 두 가지 가격대 (이중선택 클로징)
2. 특별혜택 (마감 임박)
3. 보장 (환불자유, 취소가능)
4. 판매조건 (계약금만, 분할 결제)

**심리 작용:** 미안감(고민이 길어질 때) + 확신(Offer 강조) = 결정

**고객 반응:** "그래, 한번 해볼까" (구매 신호 7호)

# 신민형 5단계와의 Offer 패턴 매핑

v13은 신민형 5단계의 상담 구조를 따르고 있습니다:

```
신민형 Step 1: 오프닝 + 공감          → v13 Stage 0-1
신민형 Step 2: 상황 파악               → v13 Stage 2-3 (구매의향 + MI)

신민형 Step 5: 결정 + 버튼업            → v13 Stage 8-9 (클로징 + 멀티콜)
```

**Offer 기법의 집중도:**
- Step 3(피크 감정) = Stage 4 = Offer 6개 (고객 감정이 최고조일 때)
- Step 5(결정) = Stage 8-9 = Offer 4개 (마지막 저항 제거)

# Offer 패턴의 심리학 원리

## v13에 숨어 있는 심리학 기법

### 패턴 1 (고급상품 비유) → **Narrative Transportation**
"이야기에 몰입되면 논리적 저항이 사라진다"
- 5감각 시각화 → 고객이 그 순간에 있다고 느낌
- 결과: 가격 저항 자동 소멸

### 패턴 3 (특별혜택) → **Scarcity (Cialdini)**
"한정된 것은 더 가치 있다"
- "이번 주까지", "잔여 좌석 2석"
- 결과: "지금 안 하면 영원히 못 한다" 느낌

### 패턴 4 (보장) → **Loss Aversion (Kahneman)**
"얻는 것보다 잃는 것을 더 두렵다"
- "취소 가능", "전액 환불"
- 결과: 리스크 제거 → 진입 장벽 대폭 낮춤

### 패턴 7 (새로움강조) → **Temporal Motivation Theory**
"시간이 가까워질수록 동기 상승"
- "지금이 마지막", "2년 뒤는 힘들다"
- 결과: 지연 행동 제거 → 즉시 결정

### 패턴 9 (재미정보) → **Affective Forecasting**
"감정을 구체적으로 그리면 욕망이 강해진다"
- "눈물 흘릴 정도의 감동", "완전히 쉬는 느낌"
- 결과: 경험의 가치 극대화

### 패턴 10 (독창성) → **Differentiation**
"유일한 것은 가격 비교를 받지 않는다"
- "12개 레스토랑", "키즈클럽", "짐 한 번만"
- 결과: 자유여행과 비교 불가능 → 가격 저항 완화

### 패턴 11 (판매조건) → **Decoupling**
"큰 결정을 작은 결정들로 나누면 심리적 부담 ↓"
- 계약금 150만원 + 60일 뒤 잔금
- 결과: 지금-나중 선택 완화 → 현재 행동 용이

# 클로징 성공 vs 실패의 Offer 패턴 차이

## 성공한 콜의 특징

**v13 기반 분석:**

| 구분 | 성공 패턴 |
|------|---------|
| **Stage 4 활용도** | 패턴 1, 2, 6, 8, 10 모두 사용 |
| **Stage 8 진입** | 7개 이상 Offer 패턴 축적 후 진입 |
| **클로징 신호** | "얼마예요?" / "언제 출발?" = 구체적 조건 문의 |
| **Offer 조합** | 럭셔리(1) + 포함(6) + 독창성(10) + 이중선택(2) |
| **심리 상태** | 욕망 최고조 → 이성적 검증(가격) → 이중선택으로 종료 |

**예:** A세그먼트 "산토리니 + 12개 레스토랑 + 발코니 선택" = 3개 Offer 조합

## 실패한 콜의 특징

| 구분 | 실패 패턴 |
|------|---------|
| **Stage 4 활용도** | 패턴 1 또는 6만 사용 (복합적이지 않음) |
| **저항 등장** | "비싸요" / "생각해봐야 해요" |
| **원인** | Offer 패턴이 고객 불안과 맞지 않음 |
| **예** | D세그먼트 고객에게 럭셔리만(1) 강조 → 건강 불안 미해소 |
| **결과** | "좋긴 한데... 나중에" → 재콜 필요 |

**실패 사례:** D세그먼트 "알래스카 빙하 아름답다" (패턴 1만) → "건강이 안 좋으면?" (불안 미해소) → 클로징 실패

# 최종 정리: v13의 Offer 철학

## Offer 기법은 "설득"이 아니라 "확인"

v13 스크립트의 핵심:
```
상담사 역할 변화:
v11: "이래서 좋아요" (설득)
v12: "우리 고객들은..." (사회적 증거)
v13: "당신도 할 수 있어요" (확인)

Offer의 역할:
v11: "구매하게 만들기" (공격)
v12: "비교하게 만들기" (논리)
v13: "이미 결심한 거 확인하기" (수용)
```

---

## Offer의 최종 목표

```
고객이 말하게 하는 것:

"그렇게 되면 정말 좋을 것 같은데요." (욕망 확인)
  ↓
"비싸지는 않네요?" (가격 저항 완화)
  ↓
"그럼 할 수 있을 것 같은데?" (능력 확인)
  ↓
"그래, 한번 해볼까요." (결정)
  ↓
"어떻게 해요?" (실행)
```

---

## v13 Offer 기법의 진짜 효과

| 효과 | 증거 |
|------|------|
| 클로징률 상승 | 단콜 15~25% → 멀티콜 40~55% |
| 취소율 감소 | Offer 4,11(보장) 사용 시 |
| 업셀 확률 | 스탠다드 → 발코니 (패턴 2) = 30% |
| 재구매율 | 마이프렌드존 (패턴 5,9) = 단골 유도 |
