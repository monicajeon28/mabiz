**작성일**: 2026-05-19  
**목표 렌즈**: L1 가격 오해형  
**사용처**: 
1. 영업팀 교육 (Monica 10단계 표준화)
2. CRM 자동응답 (이의별 추천 답변)
3. 역할극 연습 (Role-play 스크립트)
4. 콜 녹음 검증 (Monica Step 정확도 체크)

## 🎯 Monica 10단계 구조 (모든 이의에 적용)

```
[Step 1] 인정: 고객의 의견을 먼저 인정
[Step 2] 감사: 솔직하게 말씀해줘서 감사
[Step 3] 공감: 당신도 처음엔 그렇게 생각했어요
[Step 4] 원인파악: 혹시 어디서 그렇게 생각하셨어요?
[Step 5] 증폭: (고객의 입으로) 문제를 더 깊게 인식하게 함
[Step 6-7] 재프레이밍: "그게 아니라..."로 새로운 각도 제시
[Step 8] 논리적증명: 구체적 수치/사례/비교
[Step 9] 시각화: 고객이 미래상을 그리도록
[Step 10] 확인: 이게 당신한테 도움 됐어요?
```

## 💬 L1 이의 대응 10가지

### 이의 #1: "가격이 너무 비싼데요"

**고객 심리**: 호텔 기준으로 크루즈 가격을 평가

```
[Step 1] 인정
"네, 처음엔 그렇게 보이죠."

[Step 2] 감사
"솔직하게 말씀해줘서 고마워요."

[Step 3] 공감
"저도 처음 들었을 때 깜짝 놀랐어요."

[Step 4] 원인파악
"혹시 다른 여행이랑 비교하셨어요?"
→ "네, 호텔 여행이랑 비교했어요"

[Step 5] 증폭 (고객 자기발화)
"호텔 여행할 때 
밥, 숙박, 엔터, 교통을 따로 사셨죠?"
→ 고객이 "네, 맞아요"라고 인정

[Step 6-7] 재프레이밍
"크루즈는 그걸 다 포함이거든요.
호텔은 각각 따로 사는 거고."

[Step 8] 논리적 증명
"호텔 5박: 비행기 $500 + 숙박 $750 + 밥 $250 + 엔터 $100 = $1,600
크루즈 5박: 올인클루시브 $1,200

오히려 크루즈가 싸죠?"

[Step 9] 시각화
"당신이 호텔에선 매일 밥값, 엔터값을 
따로 계산하잖아요.
크루즈론 그 모든 걱정이 없어요.
그냥 즐기면 돼요."

[Step 10] 확인
"이 정도면 가격이 정말 싼 거네요, 맞죠?"
→ 고객이 "응, 맞네"라고 동의
```

### 이의 #2: "호텔이랑 뭐가 달라요?"

**고객 심리**: 준거틀 오류 (호텔 기준 적용)

```
[Step 1] 인정
"많은 분들이 그렇게 물어봐요."

[Step 2] 감사
"좋은 질문이에요."

[Step 3] 공감
"처음엔 같다고 생각할 수 있죠."

[Step 4] 원인파악
"혹시 호텔 여행을 많이 다니셨어요?"
→ 고객이 "네"라고 답함

[Step 5] 증폭
"호텔에서 5박을 있으면서,
매일 같은 침대, 같은 호텔로 돌아왔죠?"
→ 고객이 "응, 맞아"

[Step 6-7] 재프레이밍
"크루즈는 호텔이 아니라 '리조트'예요.
호텔 = 침실 (20평)
크루즈 = 배 전체 (리조트 느낌, 50+ 공간)

당신의 거주지:
호텔: 침실 20평
크루즈: 배 전체 (침실은 자는 곳만)"

[Step 8] 논리적 증명
"호텔은 저녁이면 호텔 근처만 다닐 수 있죠?
크루즈는 배 위의 극장, 바, 스파, 라운지...
10개 이상의 활동이 다 배에 있어요."

[Step 9] 시각화
"호텔: 같은 곳에서 5박
크루즈: 매일 다른 나라에서 깨어나고, 
       배 위의 50+ 공간을 자유롭게 사용

보이세요? 완전 달라요."

[Step 10] 확인
"이제 호텔과는 '다른 등급의 여행'이라는 거 알겠죠?"
```

### 이의 #3: "진짜 괜찮나요? 멀미는?"

**고객 심리**: 미지의 경험에 대한 두려움

```
[Step 1] 인정
"당연한 걱정이에요."

[Step 2] 감사
"안전이 중요하니까 좋은 질문이에요."

[Step 3] 공감
"모르는 경험이니까 불안하시죠."

[Step 4] 원인파악
"혹시 멀미 경험이 있으세요?"
또는 "혹시 배에 타본 적 있어요?"
→ 고객 대답 경청

[Step 5] 증폭
"만약 멀미한다면, 
배 탈 때가 힘들겠네요."
→ 고객이 두려움을 더 표현

[Step 6-7] 재프레이밍
"크루즈는 거대한 배라 흔들림이 거의 없어요.
비행기 승차감이 더 나빠요."

[Step 8] 논리적 증명
"올해 우리 고객 300명 중:
- 70대 할머니도 왔어요
- 5살 아이들도 많이 탔어요
- 멀미 호소? 0명

우리 고객 중 천식 있는 아이,
지병 있는 분들도 많아요.
배에 의료진이 있거든요."

[Step 9] 시각화
"당신은 선실에서 깨어나고,
갑판에 나가면 신선한 공기.
배의 가운데 선실을 드릴 테니,
거의 흔들림을 못 느껴요."

[Step 10] 확인
"이 정도면 안전하다는 거 믿어지죠?"
```

### 이의 #4: "다른 여행사는 더 싼데요"

**고객 심리**: 가격 비교 / 불법 판매 우려

```
[Step 1] 인정
"가격을 비교하신 거, 당연해요."

[Step 2] 감사
"여러 곳을 알아보신 거, 현명하세요."

[Step 3] 공감
"저도 처음엔 가격만 봤어요."

[Step 4] 원인파악
"다른 여행사는 정확히 뭘 제공했어요?"
→ 고객이 설명

[Step 5] 증폭
"더 싼 가격도 중요하지만,
혹시 그 가격에 숨은 비용은 없을까요?"
→ 고객이 의심하기 시작

[Step 6-7] 재프레이밍
"가격만으로 비교하면 위험해요.
왜냐하면 '불법 판매하는 사람들'이 
싼 가격으로 후킹하거든요.
그런 곳을 통해 샀다가 
배를 못 탄 경우도 있어요."

[Step 8] 논리적 증명
"우리는:
- 국제 크루즈 공식 파트너
- 300명 고객, 비자 거절 0명
- 배 탑승 성공률 100%

더 싼 곳은:
- 불명확한 업체
- 배 탑승 실패 위험
- 환불이 안 될 수도"

[Step 9] 시각화
"싼 가격 때문에 
배를 못 탄다면?
그건 진짜 비싼 거예요.
당신의 시간, 가족의 실망...
돈으로 다 환산 안 돼요."

[Step 10] 확인
"그래서 우리는 
가격이 아니라 '신뢰'로 경쟁해요. 맞죠?"
```

### 이의 #5: "할인받을 수 없나요?"

**고객 심리**: 가격 협상 / 최저가 탐색

```
[Step 1] 인정
"할인을 원하신 거, 당연해요."

[Step 2] 감사
"직접 물어봐주셔서 감사해요."

[Step 3] 공감
"모두 좋은 가격을 원하죠."

[Step 4] 원인파악
"혹시 어느 정도 할인을 원하세요?"
→ 고객이 기대치 제시

[Step 5] 증폭
"만약 우리가 할인해준다면,
그 차이는 어디서 나올까요?"
→ 고객이 "음..." 생각하기 시작

[Step 6-7] 재프레이밍
"우리가 싸게 팔면,
서비스 품질이 떨어져요.
한 번 배를 못 타면,
그 손실은 돈으로 환산 안 돼요."

[Step 8] 논리적 증명
"우리의 가격정책:
- 고정가 (할인 정책 없음)
- 모든 고객에게 동일
- 품질 보장 (300명, 성공률 100%)

할인해주는 곳:
- 원래 가격이 거짓
- 숨은 비용 있을 수 있음"

[Step 9] 시각화
"당신이 받은 할인이 
배를 못 타는 비용보다 싸다면?
그건 할인이 아니라 '손해'예요."

[Step 10] 확인
"우리의 가격은 공정한 가격이에요. 맞죠?"
```

### 이의 #6: "그냥 호텔로 가면 안 되나?"

**고객 심리**: 익숙한 것 vs 새로운 경험

```
[Step 1] 인정
"호텔도 좋은 여행이죠."

[Step 2] 감사
"다양한 선택지를 생각해보신 거, 좋아요."

[Step 3] 공감
"호텔은 익숙하니까 편하죠."

[Step 4] 원인파악
"혹시 호텔은 몇 번 다니셨어요?"
→ 고객: "5번 정도"

[Step 5] 증폭
"5번이나 호텔을 다니셨는데,
매번 똑같은 경험은 아니셨나요?"
→ 고객이 "응, 항상 비슷해"

[Step 6-7] 재프레이밍
"그래서 많은 분들이 
호텔에서 피로를 느껴요.
크루즈는 그 피로를 '자동으로' 해결해요."

[Step 8] 논리적 증명
"호텔 5박: 짐 싸고, 체크인하고, 길 찾고, 식당 찾고...
당신의 노력 = 80%

크루즈 5박: 배가 당신을 옮겨주고, 
모든 게 배에 있고...
당신의 노력 = 0%"

[Step 9] 시각화
"당신이 지금까지 호텔로 했던 
"여행의 피로" 다 없애는 경험.
그게 크루즈예요."

[Step 10] 확인
"이제 호텔과는 
'차원이 다른 여행'이라는 거 알겠죠?"
```

### 이의 #7: "시간이 부족한데요"

**고객 심리**: 준비 시간 부족 / 휴가 부족

```
[Step 1] 인정
"요즘 바쁘신 분들 많으시죠."

[Step 2] 감사
"솔직하게 말씀해주셔서 감사해요."

[Step 3] 공감
"저도 시간 부족해요."

[Step 4] 원인파악
"정확히 어느 부분이 시간이 부족하세요?
- 준비 시간이요?
- 여행 가는 시간이요?"
→ 고객이 구체화

[Step 5] 증폭
"만약 준비에 시간이 부족하면,
호텔 여행은 거의 못 가시겠네요?"
→ 고객이 깊게 공감

[Step 6-7] 재프레이밍
"우리는 '준비를 0으로' 해드려요.
당신은 일정표만 따라가면 돼요."

[Step 8] 논리적 증명
"크루즈 준비: 매일 10분씩 × 6일 = 총 60분
호텔 준비: 매일 30분씩 × 5일 = 총 150분

크루즈가 더 빨아요."

[Step 9] 시각화
"당신이 바쁜 와중에도,
우리가 일정표를 메시지로 보내드려요.
당신은 그냥 따라오면 돼요."

[Step 10] 확인
"준비 시간 문제, 이제 해결됐죠?"
```

### 이의 #8: "나이가 많으면 힘들 것 같은데"

**고객 심리**: 신체능력 불안 / 연령차별 우려

```
[Step 1] 인정
"나이가 들면서 여행이 힘들어질 수 있죠."

[Step 2] 감사
"현실적으로 생각하신 거, 좋아요."

[Step 3] 공감
"저희도 고령 고객분들을 많이 봐요."

[Step 4] 원인파악
"정확히 뭐가 가장 힘들까 걱정되세요?"
→ 고객: "체력?, 배멀미?, 의료진?"

[Step 5] 증폭
"(고객이 걱정한 부분을 깊게)
예를 들어, 만약 배멀미가 생기면?"
→ 고객이 구체적 우려 표현

[Step 6-7] 재프레이밍
"크루즈는 호텔 여행보다 
훨씬 덜 힘들어요.
짐을 안 싸도, 이동을 안 해도 되거든요."

[Step 8] 논리적 증명
"우리 고객 중:
- 70대 할머니 (연간 5회)
- 80대 부부 (배멀미 0)
- 의료 지원 완벽
- 24시간 의료진 배치"

[Step 9] 시각화
"당신이 선실에서 편안하게
바다를 보면서 쉼.
배가 당신을 옮겨주니까
이동 스트레스 0.
호텔 여행이 훨씬 힘들어요."

[Step 10] 확인
"나이가 많을수록 크루즈가 
더 편하다는 거, 알겠죠?"
```

"여름은 비수기라 비싼 건 맞아요."

"만약 크루즈도 계절에 따라
