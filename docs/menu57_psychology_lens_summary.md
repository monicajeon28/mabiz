## 🎯 심리학 10렌즈 개요

크루즈 판매에서 고객을 결정하게 만드는 심리학적 원리 10가지

| 렌즈 | 개념 | 적용 시기 | 기대 효과 |
|------|------|---------|---------|
| **L0** | 부재중 고객 재활성화 | 1년+ 미구매 고객 | +62-97% |
| **L1** | 가격 이의 대응 | "비싸요" 고객 | +42-48% |

| **L10** | 즉시 구매 클로징 | 최종 결정 단계 | +70-95% |

## 📌 L0: 부재중 고객 재활성화

### 개념
**1년 이상 구매 이력이 없는 고객을 감정적으로 재연결하여 재신청 유도**

### 심리학 원리
- **손실회피**: "다시는 못 올 기회다"
- **향수**: "예전 크루즈 경험 재현"
- **사회증명**: "많은 사람들이 다시 왔다"

### 실전 콜 흐름

```
"아, 그렇군요!

```

## 📌 L1: 가격 이의 대응

### 개념
**고객이 "비싸요"라고 할 때, 가치 재정의를 통해 구매 결정 유도**

### 심리학 원리
- **손실회피**: "지금 예약 vs 나중 예약 비용 비교"
- **앵커링**: "일반 호텔과 크루즈의 실제 비용"
- **사회증명**: "97% 만족도 + 동료 추천"

### 가격 비교 테이블

#### 선택지 1: 일반 호텔 여행 vs 크루즈
```
일반 호텔 4박 여행 (대인 1명 기준)
─────────────────────────────
숙박:    80만원 (호텔 4박 × 20만원)
항공:    60만원
식사:    40만원 (1일 10만원 × 4일)
관광:    30만원 (투어, 입장료)
이동:    15만원 (렌터, 택시)
────────────────
합계:   225만원

크루즈 4박
─────────────────────────────
숙박 + 식사 + 엔터테인먼트:
         150만원 (모든 것 포함)
────────────────
합계:   150만원

차이:   -75만원 (33% 절약) ✓
추가 가치: 스트레스 없음 + 품질시간
```

#### 선택지 2: 분할 결제
```
크루즈 150만원 ÷ 12개월 = 월 12.5만원

=

스타벅스 한 잔 * 3개월 가치
=
가족 영화관람 * 3회 가치
=
일반 마사지 * 2회 가치

"월 12만원으로 평생 추억을 산다?"
→ ROI 관점에서 최고의 투자
```

### 실전 PASONA 구조
```
P(Problem): "분명히 좋지만 가격이 신경 쓰이네요"
A(Agitate): "계속 미루면 가격은 더 올라갑니다"
S(Solution): "실제 비용 계산하면 호텔보다 저렴해요"
O(Offer): "분할 결제로 월 12만원만 가능"
N(Narrow): "이 할인은 오늘까지만"
A(Action): "지금 예약하면 30% 할인"
```

### 체크리스트
- [ ] LISTEN: 고객의 가격 이의 완전히 경청
- [ ] 손실회피 메시지 (타이밍 강조)
- [ ] 정확한 비용 비교표 제시
- [ ] 분할 결제 옵션 제안
- [ ] 희소성 강조 (할인 마감일)

## 📌 L2: 준비 복잡 불안

### 개념
**크루즈 초보자의 "준비 과정이 복잡할 것 같다"는 불안을 5단계 중재 질문으로 해소**

### 심리학 원리
- **불확실성 회피**: "step by step 명확히 설명"
- **권위성**: "저희가 준비 과정을 다 도와드립니다"
- **상호성**: "무료 체크리스트 + 가이드 제공"

### 5단계 중재 질문 (Mediation Framework)

#### 1단계: Define (정의하기)
```
불안한 고객의 신호:
"음, 좋긴 한데... 준비할 게 많을 것 같은데?"

대응:
"[이름]님, 괜찮아요. 
많은 분들이 그런 걱정을 해요.

근데 정확히 뭐가 가장 복잡할 것 같으신가요?

• 여권/비자 준비?
• 예약 과정?
• 짐 준비?
• 탑승 절차?

이 중에 뭐가 제일 걱정되세요?"
```

#### 2단계: Complexity (복잡성 인정)
```
고객:
"그냥... 혼자서 할 수 있을지 모르겠어요"

대응:
"아, 그 부분이 복잡하게 느껴지시는군요.

실은 많은 분들이 그렇게 느껴요.

근데 좋은 소식은...
당신이 혼자가 아니라는 거예요.

우리가 step by step 함께 갈 거니까요."
```

#### 3단계: Naming (불안에 이름 붙이기)
```
"[이름]님의 불안은 실은 
'혼자라는 생각'에서 오는 거네요.

맞나요?

그렇다면, 
'혼자가 아니라 함께한다'고 생각하면?

우리가 제공하는 것들:
• 여권 확인 체크리스트
• 여행보험 자동 가입
• 탑승 전 오리엔테이션
• 24시간 콜센터"
```

#### 4단계: Customer Articulation (고객이 직접 말하게)
```
"그래도 확인할 게 있으면 지금 다 물어봐도 돼요.

예: Q1. 여권은 언제까지 제출해야 하나요?
    A1. 탑승 45일 전까지 제출하면 됩니다.

예: Q2. 배 안에서 뭐가 제일 불편할까요?
    A2. 대부분 불편하지 않습니다. 
        마치 호텔처럼 편리합니다.

예: Q3. 짐은 몇 개까지 가져가도 돼요?
    A3. 2개 캐리어 + 기내용 1개
        (대부분 배에 보관해서 괜찮음)

이렇게 하면 좀 명확해지나요?"
```

#### 5단계: Resolution (해결 및 클로징)
```
"자, 이제 정리해보니까 어떤가요?

생각보다 훨씬 간단하지 않나요?

그리고 우리는 이 과정을 
'완벽하게' 함께할 거구요.

정말로 혼자가 아니라는 거죠.

그럼 이제 예약하셔도 될 것 같은데,
어떻게 생각하세요?"
```

### 체크리스트
- [ ] 정확한 불안 파악 (Define)
- [ ] 복잡성 인정 + 공감 (Complexity)
- [ ] 불안에 구체적 이름 (Naming)
- [ ] Q&A로 해결 (Customer Articulation)
- [ ] 명확한 해결 약속 (Resolution)

## 📌 L3: 차별성 강조 (신규 렌즈)

### 개념
**경쟁사 고민 고객을 대상으로, 우리 상품의 차별화 포인트를 명확히 강조**

### 심리학 원리
- **비교 효과**: "다른 곳과는 이렇게 다릅니다"
- **권위성**: "업계 최고 평가 + 한국인 가이드"
- **손실회피**: "다른 곳 선택 시 놓칠 것들"

### 차별화 포인트 5가지

#### 1. 한국인 가이드 (유일)
```
경쟁사: 영어 가이드만 (의사소통 어려움)
우리: 한국인 가이드 + 한글 안내

고객 심리:
"아, 한국 분이 설명해주면 편하겠네"

판매 메시지:
"저희만 한국인 가이드를 제공합니다.
항구 관광 때 언어 걱정 없이 
마음 편하게 즐기실 수 있어요."
```

#### 2. 패밀리 친화적 (최고 수준)
```
경쟁사 A: 기본 키즈 클럽
경쟁사 B: 키즈 클럽 + 수영장
우리: 키즈 클럽 + 수영장 + 전담 스태프 + 나이별 프로그램

고객 심리:
"우리 아이들이 정말 즐거워할 수 있겠네"

판매 메시지:
"일반 여행사들은 '기본 키즈 클럽'만 제공하는데,
저희는 나이별 맞춤 프로그램을 운영합니다.

부모님은 스파/다이닝 즐기고,
아이들은 안전하게 재미있게 노는 거죠."
```

#### 3. A/S 시스템 (최고)
```
경쟁사 A: 한국 사무소 없음 (이메일만)
경쟁사 B: 전화만 가능 (24시간)
우리: 카톡 + 전화 + 웹 + 메일 (24시간)

고객 심리:
"혹시 모를 상황에 한국에서 빠르게 도움받을 수 있겠네"

판매 메시지:
"문제가 생겼을 때 영문 이메일 기다리나요?

저희는 한국 직원이 항상 대기 중이라서
카톡으로 즉시 해결됩니다."
```

#### 4. 사전 준비 지원 (포괄적)
```
경쟁사: 예약서만 제공
우리: 
• 여권 체크리스트
• 수하물 가이드
• 탑승 절차 안내
• 선상 이용 가이드 (한글)
• 항구 추천 정보

고객 심리:
"이렇게까지 챙겨주는 곳이 있네?"

판매 메시지:
"크루즈 처음이신 분들이 많은데,
저희는 단계별로 '어디서 무엇을 하는지'
다 설명해드립니다."
```

#### 5. 사후 서비스 (추가 가치)
```
경쟁사: 여행 끝 (끝)
우리: 
• 여행 영상 편집 제공
• 항구 사진 모음집
• 다음 여행 10% 할인쿠폰
• 동반자 추천 15% 수수료

고객 심리:
"음, 여행 후에도 기억이 남을 수 있겠네"

판매 메시지:
"여행은 끝나지만 추억은 계속됩니다.
저희는 그 추억을 '영상'으로 만들어드리고,
다음 여행도 함께 합니다."
```

### 체크리스트
- [ ] 경쟁사 파악 (정확한 상품명 / 가격)
- [ ] 5가지 차별점 정확히 설명
- [ ] 고객의 거절 이의 5가지 준비
- [ ] 비교표 (정량적 데이터)
- [ ] 경쟁사 고민 고객 → 우리로 전환

## 📌 L6: 타이밍 손실회피

### 개념
**"지금 구매"와 "나중에 구매"의 가격/이득 차이를 강조하여 즉시 결정 유도**

### 심리학 원리
- **손실회피**: "지금 안 하면 더 비싸짐"
- **FOMO** (Fear Of Missing Out): "자리가 떨어지면 못 탐"
- **긴박감**: "오늘이 마지막 기회"

### 타이밍 메시지 3가지

#### 메시지 1: 가격 변동
```
Day 0: "지금 예약하면 Early Bird 30% 할인"

[만약 고객이 미루면]

Day 1: "어제 E-mail: Early Bird 할인 마감"
Day 2: "지금은 기본 10% 할인만 가능 (20% 손실)"
Day 7: "신청 마감 3일 전" 
Day 14: "예약 마감, 다음 기회는 3개월 뒤"
```

**고객 심리**:
"아, 지금 결정하지 않으면 20%를 잃는다!"

#### 메시지 2: 자리 부족
```
"현재 예약 상황:
• 보유 객실: 50개
• 예약 완료: 43개
• 남은 객실: 7개 (13%)

특히 패밀리 스위트는 2개만 남았습니다."

[시각적으로 보여주기]
▓▓▓▓▓▓▓▓▓▓▓▓▓ 예약됨
░░ 남음

"이 2개 중 하나를 예약하시겠어요?"
```

#### 메시지 3: 나이/상황 변화
```
"[이름]님, 아이분들이 지금 몇 살이세요?"

[고객: "8살, 10살이요"]

"아, 딱 좋은 나이네요!

5년 뒤에는 이 아이들이 13살, 15살이 될 텐데,
그럼 '아이와 함께하는 가족여행'이 어렵겠죠.

지금이 마지막 기회예요."

[고객 심리]: "맞다, 지금이 베스트 타이밍이다!"
```

### Day 0-7 FOMO 시퀀스
```
Day 0 (구매 직후):
"지금 예약하신 [이름]님은 
최고의 선택을 하셨어요!

Early Bird 30% 할인으로 
150만원 → 105만원 절약!"

[감정적 강화]

Day 1:
"[이름]님, 가족분들께 
이 기쁜 소식 나누셨나요?

지금이 감정이 가장 높을 때니까요!"

[동반자 설득 유도]

Day 3:
"출발까지 2개월 남았어요.

준비 체크리스트를 보내드릴게요.
여권, 짐, 복장 등..."

[실행 중심화]

Day 7:
"혹시 마음이 바뀌었거나 
취소하고 싶으신 건가요?

→ 환불 규정 안내 (취소료 발생)
→ 심리적 확정 (취소 비용 > 이득)"

[손실회피로 재확정]
```

## 📌 L10: 즉시 구매 클로징

### 개념
**모든 이의가 해소된 후, 감정적 강화 + 삼중 선택으로 즉시 성약 확보**

### 심리학 원리
- **감정 동화**: "가족과의 행복한 미래 이미지"
- **선택의 역설 해결**: "3가지 중 고르면 결정이 쉬움"
- **일관성**: "지금까지의 결정 흐름 강화"

### 클로징 4단계

#### 1단계: 감정 확인 (Emotional Temperature)
```
"[이름]님, 지금 기분이 어떠세요?

혹시 설레나요?

아니면 아직도 조금 불안해요?"

[고객의 진짜 감정 파악]

→ 설렐 경우:
"맞아요! 그 감정이 정상이에요.
큰 결정을 하려고 하니까요.

하지만 이 기분, 가족과 함께
크루즈를 떠날 때 백배가 돼요."

→ 불안할 경우:
"그 불안도 정상이에요.
하지만 우리가 함께 해결할 거예요.
당신은 혼자가 아니거든요."
```

#### 2단계: 미래 이미지 제시
```
"자, 상상해봐요.

아이들이 배에 올라서서
'우와!' 하는 목소리.

배우자가 당신 손을 잡고
석양을 바라보는 순간.

함께 웃고 즐기는 시간들.

그런 기억들이 평생 남는 거예요.

150만원으로 평생의 추억을 산다고 생각하면,
엄청 저렴한 투자 아닌가요?"

[고객이 감정적으로 몰입]
```

#### 3단계: 삼중 선택 (Triple Choice)
```
"이제 선택의 순간이에요.

[이름]님이 고를 수 있는 게 3가지가 있어요:

선택지 1️⃣: 지금 바로 예약 (8월)
✓ Early Bird 30% 할인 (오늘만!)
✓ 최고의 객실 배정
✓ 월 12만원 분할 결제
→ '최고의 선택'

선택지 2️⃣: 다음 달 예약 (9월)
△ Early Bird 할인 없음 (10% 기본)
△ 좋은 객실 선택 어려움
△ 월 13만원 분할 결제
→ '괜찮은 선택'

선택지 3️⃣: 지금은 예약 안 함
✗ 6개월 뒤 30% 더 비쌈
✗ 자리 보장 안 됨
✗ 좋은 시즌 놓침
→ '후회할 선택'

[일시정지 - 3초간 침묵]

[이름]님, 이 중에서 
당신 가족에게 가장 좋은 게 뭘까요?

선택지 1이 맞죠?"

[일시정지 - 고객이 대답할 시간]
```

#### 4단계: 최종 성약 (Closing)
```
[고객이 "네, 예약할게요" 답변]

"최고의 결정이에요!

[이름]님이 앞으로 얼마나 행복할 결정을 했는지
나중에 알게 될 거예요.

자, 그럼 필요한 정보들 정리할게요.

1) 여권번호 / 생년월일
2) 선호 객실 위치
3) 특별 요청사항 (허니문, 기념일 등)
4) 결제 방법 (카드 분할 / 계좌이체)

이 정보들 지금 말씀해주실래요?"

[정보 수집]

"완벽해요!

오늘 계약서 보내드릴 테니까
내일까지 사인해서 보내주세요.

그럼 바로 예약 확정입니다.

정말 감사합니다, [이름]님!

가족과 함께 이 설렘을 나누세요.
다음주부터는 준비 체크리스트도 
보내드릴 테니까요!"

[긍정적, 따뜻한 톤으로 마무리]
```

### 체크리스트
- [ ] 감정 온도 확인 (설렘 vs 불안)
- [ ] 미래 이미지 생생하게 제시
- [ ] 3가지 선택지 명확히 제시
- [ ] 침묵의 힘 (고객이 답하도록 기다림)
- [ ] 정보 수집 + 다음 단계 안내

## 📊 실전 연습 시뮬레이션

### 상황 1: 신규 인입 + 가격 이의
```
고객: "음, 좋긴 한데 비싸네요..."

신입 파트너:
✓ L1 (가격 이의) + L6 (타이밍 손실회피)
✓ PASONA 구조로 설득
✓ 분할 결제 + 할인 제시
✓ L10 (삼중 선택) 클로징

→ 성약 확률: 약 45% 상향

신입 파트너:
✗ 그냥 "괜찮아요" (공감 없음)
✗ 할인안 제시하지 않음

→ 성약 확률: 약 15% 하향
```

### 상황 2: 재활성화 + 준비 불안
```
고객: "1년 전 구매했는데... 준비할 게 많을 것 같아서..."

신입 파트너:
✓ L0 (재활성화) - 과거 추억 언급
✓ L2 (준비 불안) - 5단계 중재 질문
✓ 체크리스트 + 24시간 지원 강조

→ 성약 확률: 약 60% 상향

신입 파트너:
✗ 재활성화 언급 안 함
✗ 불안을 단순히 "괜찮습니다" 대응

→ 성약 확률: 약 20% 하향
```

### 상황 3: 경쟁사 고민 + 차별성
```
고객: "다른 여행사도 보고 있는데..."

신입 파트너:
✓ L3 (차별성) - 5가지 포인트 명확히
✓ 비교표로 정량적 설득
✓ 한국인 가이드 + A/S 강조

→ 성약 확률: 약 50% 상향

신입 파트너:
✗ "우리가 더 좋습니다" (추상적)
✗ 경쟁사 수치 모르고 대응

→ 성약 확률: 약 15% 하향
```
