## ✅ L10 마지막 5가지 체크리스트

### 1️⃣ 일정 확인

```
당신이 고민하는 것:
"정확히 언제 출발하고, 언제 돌아와요?"

확인해드릴 것:

크루즈 일정:
│ 출발: 2026-07-15 (화요일) 오후 4시
│ 탑승: 오후 4시~7시 (당신은 5시에 와도 됨)
│
│ Day 1: 항해 (배 위에서 적응)
│ Day 2: 기항지 A (반나절, 오전)
│ Day 3: 기항지 B (종일, 아침~저녁)
│ Day 4: 기항지 C (반나절, 오후)
│ Day 5: 항해 (마지막 밤)
│
│ 귀항: 2026-07-22 (화요일) 오전 7시
│ 하선: 8시~10시

당신의 일정:
│ 목요일 저녁 출발 (일 안 빠짐)
│ 화요일 아침 귀항 (다음날 출근 가능)
│ 5박 6일 전체 휴가 = 연차 2-3일만 내면 됨

정확해요. 당신 일정 맞아요.
```

### 2️⃣ 객실 선택

```
당신이 고민하는 것:
"객실이 뭐가 다른가요?
어떤 객실을 고르면 좋아요?"

우리의 추천:

┌─────────────────────────────────────┐
│ 1. 오션뷰 (가장 인기)              │
│ - 가격: $1,200                      │
│ - 창문: 있음 (바다 경관)            │
│ - 공간: 중간 (200 sq ft)           │
│ - 추천: "첫 번째는 이걸로"         │
├─────────────────────────────────────┤
│ 2. 발코니 (로맨틱)                  │
│ - 가격: $1,500                      │
│ - 발코니: 자신의 프라이빗 공간      │
│ - 공간: 넓음 (250 sq ft)           │
│ - 추천: "부부/커플이라면 이걸로"   │
├─────────────────────────────────────┤
│ 3. 인테리어 (경제적)                │
│ - 가격: $900                        │
│ - 창문: 없음 (배 중앙)              │
│ - 공간: 작음 (150 sq ft)           │
│ - 추천: "예산이 중요하면 이걸로"   │
└─────────────────────────────────────┘

당신의 상황에 맞는 건?

(일반적으로) 오션뷰가 최고 추천입니다.
왜냐면:
- 가격이 합리적 ($1,200)
- 경험이 완벽함 (창문, 신선한 공기)
- 만족도가 높음 (95%)
```

### 3️⃣ 그룹/식사 설정

```
당신이 고민하는 것:
"혼자 앉을까, 그룹으로 앉을까?"
"식사 시간은 정해졌나요?"

우리가 할 일:

기본 설정:
│ 식사: 저녁 7:30pm (고정)
│ 테이블: 8명 테이블 (당신 가족 + 낯선 사람 3명)
│ 그룹: "첫 크루즈 고객" 그룹 모임 자동 연결
│ 선택사항: 카주얼 다이닝 (시간자유) 가능

당신의 선택:

□ 고정 테이블 (7:30pm, 같은 사람들)
  → 새로운 친구 사귀기 좋음
  → 7박 동안 같은 사람들과 밥 먹음

□ 카주얼 다이닝 (시간 자유, 사람 다양)
  → 배우자와 둘이만 시간 조정 가능
  → 매 저녁 다른 식당 경험

(일반적으로) 고정 테이블이 낫습니다.
이유: 배 위에서 '진정한 친구' 만들 수 있어요.
```

### 4️⃣ 여행 보험

```
당신이 고민하는 것:
"보험을 들어야 하나요?"

당신이 보험이 필요한 이유:

만약 이런 상황이 생긴다면?
│ 가는 길에 사고 (차 문제)
│ 비행기 취소 (날씨)
│ 질병 발생 (입원)
│ 가족 응급 (어머니 입원)
│ 여행 중 의료 (배멀미 아님, 진짜 응급)

보험료: $200 (5박 여행)
보험 범위: 
│ 취소: 100% 환불 (사유 불문)
│ 의료: 배 위의 응급실 비용 전액
│ 이동: 항공 취소 시 배 비용 재충당
│ 응급 귀국: 필요시 헬리콥터

당신의 의사:

☑️ 보험 들기 (강력 추천)
   - $200이 큰 금액이 아님
   - 만약의 상황이 생기면 $1,200 손실 방지

□ 보험 안 들기
   - 문제 없으면 저축 $200
   - 문제 생기면 손실 $1,200
```

### 5️⃣ 예약 확정 후 받는 것들

```
당신이 고민하는 것:
"예약하면 그 다음엔?"

당신이 받게 될 것들:

당일:
│ 예약 확인 메이
│ 크루즈 여행증 (프린트 필수)
│ CRM 대시보드 접근권

1주일 후:
│ 여행 가이드 (상세 일정)
│ 배 지도 & 규칙
│ 짐 준비 체크리스트
│ 출발 공항 가이드

2주일 후:
│ 탑승 확인서
│ 선실 배정표 (정확한 호실번호)
│ 좌석 배정표 (식당)
│ 활동 스케줄

3주일 후:
│ 앞 기항지 가이드 (뭘 할 수 있는가)
│ 비상 연락처
│ 여행 보험 증권

1일 전:
│ 탑승 시간 알림
│ 공항 이동 가이드
│ 최종 확인 메일

당신은 뭘 하면 되나요?
- 메일 확인만 하면 됨
- 우리가 모든 걸 챙길 거예요
```

## 💬 CallScript L10 (삼중선택 클로징)

### 최종 클로징 (마지막 3분)

```
"당신, 이미 결정했어요.

왜냐면:
- '가면 좋겠다'고 생각했고
- 가족도 동의했고
- 가격도 적당하고
- 시간도 나고
- 위험도 없는 거 확인했으니까.

이제 남은 건 '어떻게 갈 거냐' 뿐이에요.

[포즈: 2초]

그래서 이렇게 할게요.

당신 상황에 맞는 3가지 옵션을 줄 거고,
당신이 고르면 끝.

준비됐어요?

[포즈: 1초 (기다림)]

옵션 1: '가족 여행 패키지'
- 오션뷰 객실
- 고정 테이블 (가족들이랑)
- 스파 크레딧 $100
- 보험 포함
- 가격: $1,200/명

옵션 2: '로맨틱 발코니'
- 발코니 객실
- 카주얼 다이닝 (당신들끼리)
- 와인 크레딧 $50
- 보험 포함
- 가격: $1,500/명

옵션 3: '경제 여행 패키지'
- 인테리어 객실
- 고정 테이블
- 보험 포함
- 가격: $900/명

[포즈: 3초]

셋 중에 뭐 고를래요?

[고객이 선택]

'옵션 1이요?'

정확해요. 대부분의 가족들이 옵션 1을 고르거든요.

그럼 당신 가족은 몇 명이에요?
- 당신?
- 배우자?
- 아이들?

[고객이 답함: 4명]

좋아. 그럼:
- 4명 × $1,200 = $4,800
- 할인 (신규): -$400
- 최종: $4,400

이 가격, OK?

[고객이 OK]

좋아. 그럼 끝이에요.

이제 뭐를 하냐면:
1. 당신이 우리한테 $4,400 지불
2. 우리가 당신 배 선실 배정 (내일 오후)
3. 당신이 배에 탑승 (7월 15일 오후 4시)
4. 당신이 평생의 추억 만들기

이 4단계가 전부예요.

준비됐어요?"

[포즈: 2초]

"그럼 신용카드 있어요?"
```
