## 🎯 L1 핵심 심리 (페르소나별)

```
신혼부부: "신혼 여행은 비싸야 맞지, 하지만 이 정도면?"
40대 가족: "아이들 교육에 쓸 돈을 휴가에? 근데 맞은 투자일까?"
중년부부: "남은 시간도 적은데, 돈을 써야 하나?"
고령층: "이 나이에 이 비용? 건강이 몇 년이나 남을지도 모르는데"

공통점: 모두 '가격 정당성'을 원해요.
해결책: 페르소나별 가치를 보여주면 된다.
```

## 1️⃣ 신혼부부형 (20-35세, 경험 추구)

### 가격 재프레이밍

```
신혼부부가 생각하는 것:
"$1,200? 비싼데..."

우리가 보여줄 것:

신혼 여행의 가치 계산:

┌─────────────────────────────────────┐
│ 신혼 여행은 "투자"다                │
├─────────────────────────────────────┤
│ 첫 추억 가치: $5,000
│ 둘만의 시간 가치: $3,000
│ 호텔과의 차이 가치: $2,000
│ 신혼 특별함 프리미엄: $1,000
│                                      │
│ 총 가치: $11,000
│ 실제 지불: $1,200
│ ROI: 9배 (900%)
└─────────────────────────────────────┘

당신이 $1,200을 쓰면:
- 당신은 $11,000의 가치를 얻어요
- 호텔에서는 $1,200을 쓰면 $1,200의 가치
- 차이: 우리가 9배 더 좋아

왜냐면:
- 호텔: 낮에는 관광, 밤에는 침실
- 크루즈: 아침부터 밤까지 '당신들만의 시간'

"당신이 신혼때 못 만드는 추억?
평생 못 만들어요.

그래서 지금이 가장 싼 투자야."
```

### SMS Day 0 (신혼부부)

```
📲 "안녕하세요, OOO님!

'가격이 비싼데...' 하셨어요.

맞아요. $1,200은 호텔 비용으로는 비싸요.

근데 생각해보세요.

신혼 여행은 '비용'이 아니라 '투자'예요.

호텔 $1,200 = 경험 $1,200 (같음)
크루즈 $1,200 = 경험 $11,000 (9배!)

왜?

호텔은:
- 낮: 뭐할 건지 고민
- 밤: 피로로 따로 자게 됨

크루즈는:
- 아침부터 밤까지 둘이만의 시간
- 배 위 어디든 당신들의 무대

48시간이 당신들 인생을 바꿔요.

이 가격, 비싼가요?

- 마비즈 크루즈팀"

[효과]: "투자"로 재프레이밍 + 9배 가치 강조
```

## 2️⃣ 40대 가족형 (직장인, 아이, 시간 부족)

### 가격 재프레이밍 (교육 투자 관점)

```
40대 가족이 생각하는 것:
"아이 학원에 월 500만원 쓰는데,
휴가에 1,200달러? 비싼데..."

우리가 보여줄 것:

당신의 연간 교육/활동 지출:
│ 수학학원: $500/월 × 12 = $6,000
│ 영어학원: $400/월 × 12 = $4,800
│ 음악: $300/월 × 12 = $3,600
│ 체육: $200/월 × 12 = $2,400
│ 기타: $600/월 × 12 = $7,200
├─────────────────────────────────────┤
│ 연간 교육 지출: $24,000
└─────────────────────────────────────┘

당신이 아이를 위해 하는 것:
"공부를 잘하게"

크루즈는 다른 투자:
"아이에게 '세상을 보게' 하기"

경험의 가치:
│ 학원: 시험 점수 ↑ (측정 가능)
│ 크루즈: 인생관 변화 (측정 불가)
│
│ 학원: 한국 경쟁 (불안감)
│ 크루즈: 세상 경험 (자신감)

당신이 사실 두려워하는 것:
"아이가 뒤떨어질까?"

크루즈가 주는 것:
"아이가 '다양한 세상'을 알고
자신감있게 성장하게 해요"

가격: $1,200 × 4명 = $4,800
이 비용으로 당신의 아이는:
- 5개국 경험
- 새로운 음식 경험
- 배 위의 200개 활동
- 부모와의 진정한 시간

학원 비용의 20%, 경험의 가치는 10배.

"이게 비싼 교육일까요?
아니면 가장 싼 교육일까요?"
```

### SMS Day 0 (40대 가족)

```
📲 "안녕하세요, OOO님!

'비용이...' 하셨어요.

당신, 아이 교육에 연간 $24,000 쓰고 있어요.

학원, 학습지, 각종 활동.

근데 크루즈는 $4,800.

비용만 봐도 20% 저렴해요.

근데 가치는?

학원이 주는 것: 
→ 한국에서의 경쟁력

크루즈가 주는 것:
→ 세상을 보는 눈 (경쟁력의 기본)

당신이 교육에 투자하는 이유?
"아이가 잘되길"

그 아이가 크루즈에서 뭘 배울까요?
- 다양한 문화
- 독립성
- 가족의 소중함
- 세상은 크고 아름답다는 것

이런 경험이 결국 아이를 만들어요.

비싼 교육일까요?
가장 싼 교육일까요?

- 마비즈 크루즈팀"

[효과]: "아이 교육 투자" 재프레이밍
```

## 3️⃣ 중년부부형 (45-55세, 여유 시간 생김)

### 가격 재프레이밍 (시간 가치 관점)

```
중년부부가 생각하는 것:
"나이가 나이인데,
아직도 $1,200짜리 여행을 해?"

우리가 보여줄 것:

당신의 남은 '건강한 시간':

┌─────────────────────────────────────┐
│ 당신의 평균 여명: 40년 (80세까지)  │
│ 건강하게 활동 가능: 30년 (75세까지)│
│ 진정히 여유 있는 때: 55-65세 (10년)│
└─────────────────────────────────────┘

'진정히 여유 있는 10년'에
당신은 뭘 할 거예요?

시나리오 1: 그냥 일만 더 하기
→ 결국 은퇴 후 여행하자는 생각
→ 은퇴하면 체력이 떨어져있음
→ 밥 먹고 자고 끝나는 여행

시나리오 2: 지금부터 시작하기
→ 아직 건강할 때 경험 쌓기
→ 65세: 스스로 움직일 수 있는 여행
→ 70세: 추억과 함께 여유 있는 삶

당신의 '가장 귀한 10년':
- 아이 독립했음 (시간 자유)
- 경제력 있음 (돈 자유)
- 건강함 (체력 자유)

이 시간을 버릴 거예요?

$1,200의 가격 vs 10년의 시간 자유

"이 금액이 비싼가요?
아니면 당신의 가장 귀한 시간을 
버리는 게 더 비싼가요?"
```

### SMS Day 0 (중년부부)

```
📲 "안녕하세요, OOO님!

'이 나이에...' 하셨어요.

정확해요. 이 나이기 때문이에요.

당신:
- 아이들 다 컸어
- 직장에서 인정받아
- 경제적으로 안정됐어
- 아직 건강해

이런 조건, 언제까지 유지될까요?

5년 후: 체력 떨어짐
10년 후: 은퇴 / 건강 악화
15년 후: 여행 자체가 힘듦

그럼 당신의 '진정한 여행 적기'는?

지금이에요.

$1,200이라는 비용?

그건 당신의 '가장 귀한 10년'을
의미있게 보내는 투자예요.

나중에 미루지 마세요.

당신의 시간은 이미 한정돼 있어요.

- 마비즈 크루즈팀"

[효과]: "지금이 마지막 기회" + 시간 긴급성
```

## 4️⃣ 고령층형 (60+, 건강 관심)

### 가격 재프레이밍 (건강 투자 관점)

```
고령층이 생각하는 것:
"이 나이에 $1,200을 쓸까?
남은 시간도 모르는데..."

우리가 보여줄 것:

당신의 선택:

선택 1: 안 가기
→ 남은 돈: $1,200 추가 축적
→ 건강: 변화 없음
→ 남은 추억: 병원 기억만 쌓임
→ 5년 후: "그때 가볼 걸"

선택 2: 가기
→ 남은 돈: $1,200 감소 (하지만 충분함)
→ 건강: 5박 휴식 + 의료진 관리
→ 남은 추억: 당신 가족의 공동 추억
→ 5년 후: "그 때 갔어서 다행이야"

의료 관점에서:
│ 당신의 의료비: 월 $500 × 60개월 = $30,000
│ 크루즈: $1,200 (의료 포함, 한 번)

당신이 아프면:
│ 병원: 혼자 누워있음 (우울증 위험)
│ 크루즈: 가족이 옆에 있음 (회복 빠름)

심리학적으로:
- 당신이 활동할수록 수명 연장
- 당신이 행복할수록 건강 개선
- 당신이 가족과 함께할수록 치료 효과 증대

크루즈라는 활동이:
"당신의 건강투자"

"$1,200이 비싼가요?
아니면 당신의 건강과 행복을 
버리는 게 더 비싼가요?"
```

### SMS Day 0 (고령층)

```
📲 "안녕하세요, 할머니/할아버지께!

'이 나이에 여행을...' 하셨어요.

정확해요. 이 나이기 때문이에요.

당신은:
✅ 아직 건강해요
✅ 아직 활동할 수 있어요
✅ 아직 가족과 함께할 수 있어요

"아직"이 얼마나 남을지는 모르겠지만,
'지금'은 당신이 가지고 있어요.

$1,200이라는 비용은?

당신의 건강, 당신의 행복, 
당신의 마지막 여행을 위한 투자예요.

아이들과의 추억, 손주와의 추억,
당신 부부만의 추억.

그게 $1,200의 가치예요.

지금, 하지 마세요?

나중에 하고 싶으세요?

나중에 할 수 있을까요?

- 마비즈 크루즈팀"

[효과]: "지금이 유일한 기회" + 공포→행동
```
