# CruiseDot Gold + Healthcare ABC 통합 스크립트 (SPIN 기반)
## Repositioning from Rental+Healthcare → Lifetime Travel Privilege + Health Integration

**작성일**: 2026-06-08
**버전**: 1.0 (SPIN-based 크루즈닷골드 + 헬스케어 ABC 완성판)
**상태**: 코드 통합 준비 (scripts-data.ts에 병합 예정)

## 핵심 리프레이밍 (Key Repositioning)

### 기존 프레이밍 (Old)
- "렌탈 + 헬스케어 ABC 패키지"
- 비용 초점: 월 렌탈비 + 검진비 따로 내는 낭비 회피

### 새 프레이밍 (New)
- "크루즈닷 골드 회원 + 헬스케어 통합"
- 라이프스타일 초점: "평생 여행 특권" + "건강 책임 관리" 동시 달성
- 새미패키지 인솔자: 배 위에서 24시간 건강 관리하는 개인 의료진

## SPIN 프레임 5단계 (5-Phase Script Structure)

### Phase 1: SITUATION (0-3분) - "삶의 질 연결"
**목표**: 고객의 현재 여행 생활 방식 파악

**도입 질문**:
> "배우자분이나 가족분이랑 함께 여행을 자주 다니시는 편이세요? 어디 정도 자주 다니세요?"

**심리학 요소**:
- 첫 질문이 "렌탈/검진"이 아닌 "여행"으로 시작 → "삶의 질" 프레이밍
- 고객이 자신의 여행 빈도를 구체적으로 언급하도록 유도
- 자연스러운 대화 분위기 형성

---

### Phase 2: PROBLEM + IMPLICATION (3-8분) - "현황파악 + 의미화"
**목표**: 고객이 현재 시스템의 문제를 스스로 인식

**SPIN 질문 구조**:
1. **Situation** (현황): "월 여행비 + 렌탈비 + 검진비가 각각 얼마나 나가세요?"
2. **Problem** (문제): "따로따로 내다 보니 여행을 자주 못 가거나 건강 관리를 건너뛰신 적 있으세요?"
3. **Implication** (의미화): "따로 비용 내다 보니 선택의 폭이 줄어드는군요"

**고객 반응 유도**:
- "월 80만원 × 36개월 = 2,880만원" (손실 가시화)
- 실제 경험한 불편함 공감 유도
- **목표 달성**: 고객이 현재 상황의 문제를 인식

---

### Phase 3: NEED/PAYOFF (8-15분) - "손실회피 심리학"
**목표**: 고객에게 "지금 결정하지 않으면 손실"이라는 긴박감 생성

**3가지 손실 차원 강조**:

1. **돈의 손실**
   - "월 80만원을 따로 내다 보니 1년에 960만원, 3년이면 2,880만원"
   - 따로 구매 vs 패키지 가격 비교표로 절감액 명확화

2. **시간의 손실**
   - "건강 관리를 미루다 보니 조기 발견을 놓친다"
   - "조기 발견 80만원 vs 늦은 발견 800만원" (의료비 손실)
   - 나이 들수록 여행의 가치가 높아지는데 건강이 없으면 즐길 수 없음

3. **경험의 손실**
   - "건강이 안 좋으면 여행을 제대로 즐길 수 없다"
   - 나이가 들 수록 여행 욕구와 건강 관리 필요성이 동시에 증가

**심리학 메커니즘**:
- Loss Aversion (손실회피): 이득 강조보다 손실 강조가 훨씬 효과적
- 구체적 수치 (2,880만원, 80만원 vs 800만원)로 임팩트 증대
- "지금 이 순간도 몸 안에서 변화가 생기고 있을 수 있어요"

---

### Phase 4: PAYOFF (15-25분) - "솔루션 제시 + ABC 패키지 분석"
**목표**: 세 가지 손실을 한 번에 해결하는 솔루션으로 포지셔닝

#### 4.1: CruiseDot Gold 회원의 개념 재정의

**기존**: "렌탈 서비스"
**새로**: "평생 여행 특권" = 건강 상태에 맞춰서 맞춤형 크루즈 여행 계획

#### 4.2: ABC 패키지 3단계 구조

| 패키지 | 월비용 | 크루즈 여행 | 헬스케어 | 따로 구매 시 | 절감율 |
|--------|--------|-----------|----------|------------|--------|
| **A** | 59,000원 | 분기 1회 (연 2회) | 연 1회 검진 + 월 1회 BP 체크 | 150,000원 | 60% |
| **B** | 129,000원 | 월 1회 (연 4회) | 연 3회 정밀검진(200항목) + 분기별 혈액검사 + 맞춤형 식단 | 280,000원 | 54% |
| **C** | 199,000원 | 무제한 | 월 1회 정밀검진 + **새미패키지 인솔자** 24시간 동반 | 400,000원+ | 50%+ |

#### 4.3: 새미패키지 인솔자의 구체적 역할

**배 탑승 중 24시간 건강 관리**:
- **아침**: 건강 상태 체크, 운동 루틴 제시
- **점심/저녁**: 맞춤형 식단 추천
  - "오늘은 혈당 관리를 위해 이 음식을"
  - 고혈압? 저염식 제공
  - 당뇨? 당지수 낮은 식단 추천
- **저녁**: 운동 후 스트레칭, 혈압 재측정

**구체적 고객 시나리오**:
고객이 고혈압 있다면:
- 일반: 약만 복용
- 패키지 C: 배 위에서 인솔자가 식단 + 운동 + 스트레스 관리 
- 결과: 6개월 후 혈압약 용량 감소 또는 중단 가능 (몇천만원대 가치)

---

### Phase 5: NEED + ACTION (25-40분) - "범위 좁히기 + 결정"
**목표**: 고객의 우려사항을 해결하고 구체적 패키지 선택 유도

#### 5.1: 3가지 고객 세그먼트별 추천 (Segmentation)

**Type 1: 신중형** (우선 경험해 보고 싶은 분)
→ Package A 추천 (월 59,000원)
- "분기별 1회 여행으로 시작. 6개월 후 더 자주 다니고 싶으면 B로 업그레이드"
- 처음부터 C 계약했다가 안 맞으면 아까우니까

**Type 2: 활동형** (매달 여행 다니고 싶은 분)
→ Package B 추천 (월 129,000원) - **가장 인기**
- "매달 1회 여행은 충분히 현실적. 직장 생활하면서 월 1회 휴가 잘 쓸 수 있음"
- 새미패키지 아직 필요 없지만, 정밀 건강검진으로 기초 단단히

**Type 3: 프리미엄형** (평생을 준비하는 분)
→ Package C 추천 (월 199,000원)
- "[고객명]님이 올해 [나이]세실 것 같은데, 이때부터 준비하면 평생 건강하고 행복"
- 새미패키지 인솔자 = 배 위에서 개인 의사 + 영양사 + 트레이너 역할
- "이게 몇천만원대 가치"

#### 5.2: 주요 이의 대응 시나리오

| 이의 | 대응 |
|------|------|
| "월 비용이 부담돼요" | "이미 월 80만원 이상 따로 내고 계세요. 오히려 절약. 3-6개월 해보세요." |
| "새미패키지가 정말 도움이 될까?" | "고혈압 있으시면 지금은 약만 먹는데, 6개월 후 약량 감소/중단 가능. 몇천만원 가치." |
| "여행을 그렇게 자주 못 가는데" | "A부터 시작. 분기별 1회 경험해 보면 3개월 후 자연스럽게 B로 업그레이드." |
| "건강 상태가 좋은데" | "'침묵의 살인자' - 증상 없이 진행. 조기 발견이 핵심. 지금이 미리 준비할 시점." |

## 가격 및 혜택 전략

### 기본 가격 (Base)
- A: 59,000원/월 (따로 구매 150,000원)
- B: 129,000원/월 (따로 구매 280,000원)
- C: 199,000원/월 (따로 구매 400,000원+)

### 오늘의 프로모션 (Today Only)
1. **첫 달 50% 할인**
   - A: 29,500원
   - B: 64,500원
   - C: 99,500원

2. **첫 여행 비용 지원**
   - 항공료 50% 쿠폰 (최대 50만원)
   - 선내 식사비 20% 할인 쿠폰

3. **초기 건강검진 무료 업그레이드**
   - A: 기본 검진 → 150항목 정밀검진
   - B: 200항목 → 250항목
   - C: 월 1회 → 월 2회 정밀검진

4. **가족/지인 소개 보너스**
   - 1명 추천: 다음 달 구독료 50% 할인
   - 3명 추천: 3개월 무료
   - 5명 추천: 6개월 무료

5. **평생 회원 특전** (1년 구독 후)
   - 패키지 가격 20% 영구 할인
   - 예: B 패키지 129,000원 → 평생 103,200원

### 혜택 합산 메시지
```
A: 첫 3개월 비용만 8만8천원 (원가로는 17만7천원)
B: 첫 3개월 비용만 19만3천원 (원가로는 38만7천원)
C: 첫 3개월 비용만 29만8천원 (원가로는 59만7천원)

"이거 정말 오늘만이에요. 내일 가입하면 첫 달 50% 할인이 없어요."
```

## 손실강조 3단계 (Closing Power Move)

### 1번 손실: 돈
- 오늘 혜택 총가: 약 80만원대
- 내일 가입: 같은 서비스를 80만원 더 비싸게 구매하는 거

### 2번 손실: 시간
- 크루즈 운항 일정: 계절별, 월별 고정
- 한 달 늦으면: 한 달 늦게 여행 시작, 한 달 늦게 건강검진
- 조기 발견 기회 상실

### 3번 손실: 기회
- 이번 달 모집 정원: [N]명
- 오늘 기준 남은 자리: [M]개
- 내일이나 모레: 자리가 찼을 수 있음
- 다음 달 신청: 대기 리스트 진입 가능성

**수사학적 클로징**:
> "[고객명]님, 이 세 가지 손실을 다 감수할 수 있으세요? 아니라면, 지금 바로 결정하셔야 해요."

## 심리학 프레임 5가지 (5 Psychology Lenses)

### 1. Loss Aversion (손실회피)
- Phase 3-4에서 강조: 2,880만원 손실, 의료비 손실, 경험 손실
- 이득보다 손실 언급이 2배 이상 효과적

### 2. Self-Projection (자기투영)
- "고혈압 있으신 분들도 6개월 후 약량 감소" → 고객이 자신과 매칭
- 새미패키지 인솔자의 구체적 행동 시나리오로 개인화

### 3. Time Value (시간 가치)
- "나이 들수록 여행의 가치가 높아진다"
- "건강 관리를 빨리 시작할수록 조기 발견 가능성 높아진다"

### 4. Scarcity (희소성)
- "이번 달 모집 정원 [N]명, 아직 [M]자리 남음"
- "다음 달 신청하면 대기 리스트 진입 가능"
- 특히 Package C의 새미패키지 인솔자 수 제한 강조

### 5. Reciprocity (상호성)
- "오늘 혜택: 첫 달 무료, 검진 업그레이드, 항공료 할인"
- 고객이 받은 것이 많을 때 보답하려는 심리 작동

- **평균 패키지**: 기존 A위주 → 목표 B 비중 50% 이상

## 참고: SPIN 판매법 개요

SPIN은 고객의 요구사항을 이끌어내는 질문 기법:
1. **Situation**: 현재 상황 파악 ("여행 자주 가세요?")
2. **Problem**: 문제 인식 ("비용 부담 있으세요?")
3. **Implication**: 문제의 의미화 ("그러면 여행을 못 가는군요?")
4. **Need/Payoff**: 필요성 도출 ("이 문제를 해결하면 어떨 것 같으세요?")

이 스크립트는 SPIN의 4단계를 모두 포함하며, 추가로 Grant Cardone의 Loss Aversion 심리학을 결합.
