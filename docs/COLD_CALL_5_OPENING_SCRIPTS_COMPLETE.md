**📍 심리학 원리 (PHASE 1):**
- **공감 (Sympathy):** "당신의 의심은 정상" → 경계심 50% 감소
- **신뢰 신호:** "날씨도 괜찮고, 당신의 의견도 존중" → 편도체 진정
- **명확한 약속:** "3-4분" → 시간 불안감 제거
- **뇌 활성화:** 편도체 진정 → 전전두엽피질 열림

당신이 낸 돈이... 저희 통장에 들어가지 않아요.

"정확히 맞아요. 당신의 돈은 100% 안전합니다.

[사회적 증명 - 규모감과 신뢰]
지금 3,000명 넘게 이용 중이고요,

앱스토어 평점은 4.8점이에요.

거의 모든 고객분들이 '아, 이렇게 안전할 줄 알았으면

**📍 심리학 원리 (PHASE 2):**
- **신탁 계좌:** Loss Aversion 역이용 → "내 돈이 안전하구나" → 신뢰 +40%
- **사회적 증명 (Social Proof):** 3,000명 + 4.8점 → "나도 이 그룹에 속할 수 있다"
- **Third-party Validation:** "우리 말" 아닌 "고객 말" → 객관성
- **뇌 활성화:** 미러뉴런 + 보상회로 → 옥시토신 ↑ (신뢰 호르몬)

#### **PHASE 3: YouTube 후킹 시작 (1:45-2:45) ⭐ 가장 중요**

```
📞 상담사:
"자, 그런데 한 가지 여쭤봐도 될까요?

(고객이 답할 때까지 기다리기)

고객이 "네, 맞아요"라고 하면:

"저도 많이 봤는데요...

[침묵의 시간성 - 미래 후회 프레이밍]
그 '언젠가'가... 어느새 지나가버리더라고요.

당신은 지금 몇 살이세요? 부모님은?"

[포즈: 2초]

고객: "아, 저는 50살이고 부모님은 75살이요"

"그렇군요. OO님, 질문 하나만 해도 돼요?

혹시 모를 일이 있을 수도 있잖아요.

[포즈: 4초] ← 아주 길게, 고객이 깊게 생각하도록

(고객이 한숨을 쉬거나 침묵으로 답할 때까지 기다리기)

고객: (침묵) 또는 "아... 정말이에요"

"네, 그거예요.

그래서 크루즈가 필요한 거예요."

[포즈: 1초]
```

**📍 심리학 원리 (PHASE 3) - 가장 강력:**

1. **YouTube FOMO 신호 탐지:**
   - "부모님과 제대로 여행" = 영상에서 나오는 대표적 감정 신호
   - "언젠가" = YouTube에서 "이 시간을 놓치면..." 느낌
   - 호르몬 변화: 코르티솔 ↑ (스트레스), 도파민 ↑ (손실 회피)

2. **Narrative Transportation (이야기를 통한 감정 전달):**
   - 고객이 자신의 부모님을 자동으로 상상
   - 나이/건강을 계산하게 함
   - 미래의 후회를 현재로 가져옴

3. **Loss Aversion - Mortality Salience:**
   - "앞으로 얼마나 남아 있을까" = Mortality Salience (죽음 인식)
   - "혹시 모를 일" = 불확실성 극대화
   - 뇌 활성화: 편도체 + 뇌섬엽 → 긴급성 신호

4. **침묵 타이밍의 위력:**
   - 3초 + 4초 = 고객이 스스로 문제를 발견
   - 상담사 말 아닌 "자신의 깨달음"이 가장 강력

5. **뇌 화학:**
   - 코르티솔: 스트레스 호르몬 (긴급성)
   - 아드레날린: 각성 호르몬 (지금 행동)
   - 옥시토신: 신뢰 호르몬 (상담사 말 믿음)
