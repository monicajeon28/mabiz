```

    → Landing, Dashboard, Settings용
    
[ ] og-image-join.png (1200×630px)
    → Join 페이지용
    
[ ] og-image-register.png (1200×630px)
```

### 이미지 생성 명령어 (예)
```bash
# ImageMagick 사용
convert -size 1200x630 xc:"#003D99" \
  -pointsize 48 -fill white \
  -gravity center \
  -annotate 0 "마비즈 크루즈닷파트너스\n파트너 CRM" \
  og-image.png

# 또는 Figma/Photoshop에서 수동 생성
```

## 빠른 수정 (관리자용)

### Description 개선 필요시
```typescript
// 변경 전
description: '크루즈닷 파트너 CRM입니다.',

// 변경 후
description: '크루즈닷 파트너 전용 CRM. 고객관리, 수당확인, 영업도구를 한 곳에서. 즉시 가능, 무료 사용.',
```

### Keywords 추가 필요시
```typescript
// 배열에 요소 추가
keywords: [
  '크루즈 판매',
  '파트너 CRM',
  // ... 기존 키워드
  '새 키워드 추가', // 여기 추가
],
```

### Title 변경 필요시
```typescript
// 패턴 유지: [키워드] — [브랜드]
title: '새로운 키워드 — 마비즈 크루즈닷파트너스',
```
