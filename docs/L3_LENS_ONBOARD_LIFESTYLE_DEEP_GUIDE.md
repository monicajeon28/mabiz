## 📊 배 위 vs 호텔 (시간 비교)

### 호텔 5박 시간 분배

```
┌─────────────────────────────────────┐
│ 호텔 5박 120시간                    │
├─────────────────────────────────────┤
│ 이동 (짐, 숙소 찾기): 16시간        │
│ 준비 (아침/저녁): 12시간            │
│ 밥 (찾기/먹기): 12시간              │
│ 실제 경험 (활동): 80시간            │
│                                      │
│ 당신의 역할: 80% (계획, 이동, 선택) │
│ 자동 제공: 20%                      │
│                                      │
│ 재미 지수: ⭐⭐⭐⭐☆ (4/5)        │
│ 피로도: ⭐⭐⭐⭐⭐ (5/5)            │
└─────────────────────────────────────┘
```

### 배 위 5박 시간 분배

```
┌─────────────────────────────────────┐
│ 배 5박 120시간                      │
├─────────────────────────────────────┤
│ 탑승 (1시간)                        │
│ 준비 (6시간) = 아침/저녁만          │
│ 밥 (12시간) = 자동 제공, 선택만    │
│ 실제 경험 (활동): 101시간           │
│                                      │
│ 당신의 역할: 10% (선택만)           │
│ 자동 제공: 90% (모든 것)            │
│                                      │
│ 재미 지수: ⭐⭐⭐⭐⭐ (5/5)        │
│ 피로도: ⭐☆☆☆☆ (1/5)              │
└─────────────────────────────────────┘
```

**결론:**
```
호텔: 경험 80시간 + 일 40시간 = 피로한 휴가
배: 경험 101시간 + 쉼 19시간 = 진정한 휴가

당신이 '하는' 일이 줄어들수록
당신이 '즐기는' 시간이 늘어난다.
```

## 💡 차별성 재정의 (L3의 핵심)

```
고객이 생각하는 크루즈:
"배 위에만 있으니까 지루할 것 같은데"

우리가 보여주는 현실:
"배 자체가 '여행'이에요"

호텔 모델:
숙소(기항지) + 활동 = 여행

크루즈 모델:
배 = 숙소 + 활동 + 엔터테인먼트 + 음식 + 휴식

배가 '움직이는 리조트'라는 이유:
1) 아침에 일어나면 새로운 나라 (기항지 경험)
2) 낮에 배 위에서 200가지 활동 (리조트 경험)
3) 저녁에 새로운 친구들과 식사 (사교 경험)
4) 밤에 공연을 본다 (엔터테인먼트 경험)

호텔과 다른 점:
호텔: "이 도시에서 뭘 할까"
배: "이 배와 기항지에서 뭘 할까" (선택이 2배)

결과: 당신은 '선택의 자유'를 얻어요.
```
