## 🧮 감점/보너스 기준표

### 감점 (마이너스)

| 항목 | 기준 | 감점 |
|------|------|------|
| 환불율 | 20~29% | -5점 |
| 환불율 | 30~39% | -10점 |
| 환불율 | 40% 이상 | -20점 |
| 약관 위반 | 1회 | -5점 |
| 약관 위반 | 2회 | -10점 |
| 약관 위반 | 3회 이상 | -20점 |

### 보너스 (플러스)

| 항목 | 기준 | 보너스 |
|------|------|--------|
| 판매 실적 | 월 10~49건 | +5점 |
| 판매 실적 | 월 50건 이상 | +10점 |
| 고객평가 | 4.5/5.0 이상 | +5점 |

### 계산 예시

```
📌 우수 파트너 (점수: 100점)
   100(기본) - 5(환불율 25%) + 5(판매 30건) = 100점 ✅

📌 경고 파트너 (점수: 65점)
   100(기본) - 10(환불율 35%) - 5(위반 1회) = 85점 → 현재 65점
   (이전 달 마이너스 때문에 누적)

📌 제한 파트너 (점수: 45점)
   100(기본) - 20(환불율 45%) - 10(위반 2회) = 70점 → 현재 45점

📌 정지 파트너 (점수: 25점)
   100(기본) - 20(환불율 50%) - 20(위반 3회+) = 60점 → 현재 25점
   (생각보다 높은 이유: 시간이 지나면서 기록 초기화)
```

## ❓ 자주 묻는 질문 (Quick FAQ)

### Q1: 내 파트너 점수가 70점 이상이면 뭔가 바뀌나요?
**A**: 아니요! 70점 이상이면 지금과 똑같이 모든 활동을 할 수 있습니다. ✅

### Q2: 환불율이 왜 중요한가요?
**A**: 환불이 많다는 것은:
- 고객이 상품에 만족하지 못했다는 뜻
- 파트너의 설명이 정확하지 않았을 수도
- 우리 신뢰도가 떨어진다는 뜻

그래서 환불율을 낮게 유지해달라고 하는 거예요.

### Q3: 점수가 낮으면 정말 활동이 멈춰나요?
**A**: 
- 35~50점: 새로운 광고/오퍼만 금지. 기존 고객 관리는 OK
- 40점 이하: 모든 활동 정지 (협의 필요)

### Q4: 이의를 제기하면 점수가 올라가나요?
**A**: 아니요. 이의 검토는 점수 계산이 맞는지만 확인합니다.
- 이의가 인정되면: 점수 재조정 + 사과
- 이의가 거부되면: 왜 거부했는지 설명

### Q5: 왜 갑자기 이런 시스템을 도입하나요?
**A**: 더 공정한 플랫폼을 만들기 위해서입니다.
- 좋은 파트너는 더 많은 기회를
- 문제있는 파트너는 개선 기회를

모두가 WIN-WIN하는 구조입니다.
