**작성일**: 2026-05-19  
**렌즈 정의**: 저항유형 준비 부담형  
**핵심 오해**: "여행 준비 복잡할 것 같다" (호텔예약, 비자, 짐, 일정계획 등)  
**경험수준**: 없음 (크루즈 미경험)

## 📊 L2 렌즈 고객 프로필

### 타겟 고객 정의

```
Primary Target:
├─ 첫 크루즈 고객 (경험 없음)
├─ 여행 준비 경험이 적은 사람
├─ 업무 바쁨 또는 준비 능력 낮음
├─ 30-60대 (주로 직장인, 50대 자녀 있는 고령)
└─ "준비 어려울까봐" 걱정하는 사람

Specific Personas from 20Lens:
├─ #2 업무 바쁜 신혼 ("준비 시간이 없어요")
├─ #6 50대 직장인 ("여행 준비 처음이라")
├─ #9 60대 은퇴 부부 ("호텔 예약도 못 해봤어")
├─ #15 글로벌 기업직원 ("여권 갱신도...")
└─ 모든 페르소나 중 "준비" 키워드 언급한 사람
```

### 핵심 Pain Point

| 요소 | 설명 | 심리학 원리 |
|------|------|---------|
| **호텔 예약 부담** | "선실 예약을 내가 해야 하나?" | Activation Energy 높음 |
| **비자/여권 불안** | "비자 신청은? 여권 갱신은?" | Unknown Risk (미지의 위험) |
| **짐 준비 복잡함** | "뭘 챙겨야 하나? 얼마나?" | Overwhelm (과부하 느낌) |
| **일정 계획 어려움** | "기항지에서 뭐 할까? 미리 예약?" | Planning Paralysis |
| **식사 예약 번거움** | "식사 좋은 자리 미리 예약해야?" | Coordination Burden |
| **첫 경험 불안감** | "혼자 떨어져 있으면?" | First-Time Anxiety |

### 저항 패턴 분석

```
고객 발언 → 심리 상태 → 현행 대응 → 문제점 → L2 대응

"준비가 복잡할 것 같은데?"
→ Activation Energy 높음 + 미지의 위험
→ "저희가 안내해드립니다" (추상적 약속)
→ 구체성 낮음, 신뢰도 낮음
→ "3가지만 하면 됩니다" (구체적 체크리스트)

"여권 갱신도 못 해봤는데?"
→ 첫 경험 불안 + 자신감 부족
→ "보통은 간단합니다" (일반화)
→ 고객의 불안감 해소 안 됨
→ "6개월 전부터 단계별 가이드" (구체적 일정)

"기항지에서 뭐 할까?"
→ Planning Paralysis + 결정 피로
→ "자유롭게 돌아다니시면 돼요" (모호함)
→ 고객은 여전히 불안
→ "5가지 추천 일정 + 사전 예약 옵션" (선택지 제시)
```

## 🎯 L2 핵심 전환 로직

### 준비 0시간 프레임

**고객의 신념**:
```
"여행 준비 = 복잡하고 오래 걸린다"
```

**L2 강력한 대응**:
```
"네, 일반 여행은 준비가 복잡해요.

하지만 크루즈는?

모든 게 이미 준비돼 있어요.

호텔? → 선실이 이미 예약됨
식사? → 메뉴가 이미 정해짐
이동? → 배가 알아서 움직임
일정? → 매일 다른 나라 자동 방문

당신이 할 일은?
- 여권만 챙기세요 (1초)
- 짐만 싸세요 (30분)
- 출발지에서 배로 가세요 (5분)

끝입니다.

나머지는? 우리가 다 했어요.

6개월 전부터 1주일 전까지
단계별 가이드도 보내드리니까
그것만 따라하면 돼요."
```

### 3가지 준비 체크리스트

**3개월 전 (준비 Phase 1 - 개인문서)**
```
✓ 여권 갱신 (기간 초과 확인)
✓ 예방접종 (국가별 필수/권장)
✓ 여행보험 (선택사항)
```

**2개월 전 (준비 Phase 2 - 서류)**
```
✓ 신용카드 신청 (환전 수수료 낮춤)
✓ 은행 출국신고 (한도 증액)
✓ 여행가이드앱 다운 (오프라인 지도)
```

**1주일 전 (준비 Phase 3 - 짐싸기)**
```
✓ 의류 선택 (계절, 선실 짐장 제한 확인)
✓ 세면도구 (배에도 있지만, 원하는 것 챙김)
✓ 의약품 (감기약, 밴드, 소화제 등)
```

```
"안녕하세요! 크루즈닷 모니카입니다.

혹시 지금 여행 준비로 고민이 많으신가요?

제가 들은 얘기를 보면,
많은 분들이 '준비가 복잡할까봐' 걱정하세요.

비자 신청, 여권 갱신, 호텔 예약, 짐 싸기...
'이 모든 걸 내가 다 해야 하나?' 이런 걱정 말이에요.

하지만 여기 좋은 소식이 있어요.

크루즈는 일반 여행과 달라요.

당신이 할 일이 딱 3가지예요.

1) 여권 갱신 (또는 갱신 확인)
2) 짐 싸기
3) 배 타러 가기

끝입니다.

호텔은? 이미 선실 예약됨
식사 예약? 이미 메뉴 정함
일정 계획? 배가 자동으로 나라 이동

그래서 '준비 0시간 여행'이라고 말하는 거죠.

이 부분 한번 자세히 설명해볼까요?"
```

### 스크립트 B: 첫 여행 고객

```
"안녕하세요! 크루즈닷 모니카입니다.

처음 해외 여행이신가요?

그럼 많은 분들이 비슷한 걱정을 하세요.

'준비가 얼마나 복잡할까?'
'혼자 해낼 수 있을까?'

하지만 크루즈는 이런 걱정이 없어요.

왜냐하면 준비할 게 거의 없기 때문이죠.

예를 들어:

일반 해외여행:
- 비행기표 예약

- 식사 예약
- 현지 가이드 섭외
- 일정 계획
- 환전
- 짐 예약

(정말 많죠?)

크루즈는:
- 배 타기 (우리가 예약함)
- 짐 싸기
- 여권만 챙기기

딱 이 3개예요.

이게 어떻게 가능한지, 한번 설명해볼까요?"

### Activation Energy 최소화

```
"준비 부담을 극도로 낮춘 방법이 이거예요.

일반 여행:
'나는 다 해야 한다' → 시작 전 피로 (Activation Energy ↑↑↑)

크루즈:
'우리가 다 준비했으니까 따라만 해' → 부담 없음 (Activation Energy ↓↓↓)

심지어:
- 선실? 이미 예약됨
- 식사? 이미 배정됨
- 짐? 용량 내로 싸기만 하면 됨
- 일정? 배가 알아서 나라 이동

'당신이 준비할 게 없다'

이게 실제로 경험하신 분들의 후기예요.

'어? 벌써 가요? 준비 다 된 거라고 했더니 정말 그네요'

이런 말씀을 많이 들어요."

## 🎁 Phase 3: 상품 설명 (7-9분)

**목표**: 구체적 준비 가이드 + 단계별 지원 방식

### 준비 가이드 패키지

```
"우리가 제공하는 준비 지원이 3가지예요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

6개월 전: 여권 갱신 안내
3개월 전: 예방접종 가이드
1개월 전: 기항지별 추천 일정
1주일 전: 짐 싸기 가이드

포함 내용:
- PDF 가이드 4개
- 동영상 튜토리얼

[Q2] 여행 준비에 대한 느낌?
 ① 복잡할까봐 걱정돼요 (L2 확정)
 ② 조금 번거롭긴 한데 (L2 중간)
 ③ 크게 어렵지 않아요 (L2 낮음)

[Q3] 여행 준비 주도권?
 ① 내가 다 알아서 하는 편 (L2 낮음)
 ② 여행사의 도움을 받는 편 (L2 높음)
 ③ 부부/가족이 함께 상담 (L2 중간)

[Q4] 시간 여유?
 ① 매우 바쁜 편 (L2 높음)
 ② 중간 정도 (L2 중간)
 ③ 여유가 있는 편 (L2 낮음)

[자동 분류 로직]
IF (Q1 = "①" OR Q2 = "①") AND Q4 IN ("①","②")
  THEN L2렌즈 할당 + 콜 스크립트 L2 버전 로드
ELSE IF ...
```

#### 고객 DB 테이블 구조

```sql
CREATE TABLE customers_lens (
  id SERIAL PRIMARY KEY,
  customerId INT REFERENCES customers(id),
  
  -- 렌즈 분류
  lensType ENUM('L1', 'L2', 'L3', 'L4', 'L5', 'L6-10'),
  lensScore DECIMAL(3,2), -- 0.00 ~ 1.00
  lensReason VARCHAR(500), -- 분류 근거
  
  -- 준비 부담 (L2 특화)
  preparationBurdenLevel INT, -- 1(낮음) ~ 10(높음)
  travelExperienceCount INT, -- 해외 여행 횟수
  firstTimeTravel BOOLEAN, -- 첫 여행 여부
  
  -- 시간 여유 (L2 판정)
  timeAvailability ENUM('busy', 'moderate', 'leisure', 'unknown'),
  workIntensity INT, -- 1(여유) ~ 10(바쁨)
  
  -- 준비 성향 (L2 판정)
  preparationStyle ENUM('self_driven', 'assistance_seeking', 'collaborative', 'unknown'),
  
  -- 특정 불안점
  preparationAnxieties JSON, -- ["여권", "비자", "짐준비", "일정계획"] 등
  
  -- 렌즈별 전환 추적
  l2_conversionStep INT, -- 0(초기) ~ 5(구매)
  l2_sms_day0_sent BOOLEAN,
  l2_sms_day1_sent BOOLEAN,
  l2_sms_day2_sent BOOLEAN,
  l2_sms_day3_sent BOOLEAN,
  l2_mentoring_subscribed BOOLEAN,
  
  -- 가이드 수신 통계
  l2_guide_day0_opened BOOLEAN,
  l2_guide_day1_opened BOOLEAN,
  l2_guide_day2_opened BOOLEAN,
  
  createdAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updatedAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```
