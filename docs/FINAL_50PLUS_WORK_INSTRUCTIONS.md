## 2️⃣ 5가지 그룹 운영

### 고객을 어떻게 나눠서 관리할까?

```
┌─────────────────────────────────┐
│ 모든 고객                        │
└──────────┬──────────────────────┘
           │
    ┌──────┴──────┬──────────┬─────────────┬──────────┐
    │             │          │             │          │
    ▼             ▼          ▼             ▼          ▼
┌────────┐   ┌────────┐ ┌────────┐   ┌────────┐  ┌────────┐
│ 그룹 1 │   │ 그룹 2 │ │ 그룹 3 │   │ 그룹 4 │  │ 그룹 5 │
│ 초기   │   │ 기본   │ │ 활동적 │   │ VIP    │  │ 휴면   │
│ 고객   │   │ 고객   │ │ 고객   │   │ 고객   │  │ 고객   │
└────────┘   └────────┘ └────────┘   └────────┘  └────────┘
```

### 각 그룹별 역할

| 그룹 | 특징 | 연락 방법 | 목표 |
|------|------|---------|------|
| **초기** | 처음 만난 사람 | 문자 + 전화 | 상담 신청 |
| **기본** | 기본 정보 있음 | 문자 + 카톡 | 예약 받기 |
| **활동적** | 자주 와서 구매 | 특별 제안 | 많이 구매 |
| **VIP** | 최고 고객 | 직접 전화 | 계속 돌아오게 |
| **휴면** | 오래 안 옴 | 깜짝 선물 | 다시 불러오기 |

### 자동 분류하기

**매일 자동으로 정리됨** (손 안 대도 됨)

```
신규 문의 들어옴
  ↓
가격, 일정, 건강상태 확인
  ↓
자동으로 해당 그룹에 배치
  ↓
그룹에 맞는 메시지 자동 발송
```

## 3️⃣ 자동 처리 4단계

### "자동 처리"는 뭐예요?

**당신이 하지 않아도, 시스템이 알아서 하는 것**

예:
- 고객 문의 들어옴 → 자동으로 상담사에게 배정
- 상담 완료 → 자동으로 메시지 발송
- 예약 확정 → 자동으로 준비 단계로 이동

### 4단계 자동 처리

```
1️⃣ 감지
   ↓
   고객 상태 자동 인식
   예: "가격 때문에 고민하나?" 자동 감지

2️⃣ 분류
   ↓
   그룹에 자동 배치
   예: "이 사람은 가격 민감 그룹" 자동 정렬

3️⃣ 연락
   ↓
   메시지 자동 발송
   예: "가격 싼 대안 제시" 메시지 자동 전송

4️⃣ 추적
   ↓
   응답 자동 기록
   예: 고객 응답 → 자동으로 다음 단계 진행
```

### 10가지 자동 감지 신호

시스템이 이 신호들을 감지하면 자동으로 움직입니다.

```
1. 초기 고객 → "안녕하세요 상담 도와드릴게요" 메시지
2. 가격 고민 → "가격 비교표 보여드릴게요" 메시지
3. 준비 복잡 → "쉬운 준비 가이드" 제시
4. 경쟁사 비교중 → "우리만의 장점 3가지" 설명
5. 가입 거부 → "특별 할인 쿠폰" 자동 발송
6. 자신감 부족 → "비슷한 고객 후기" 자동 전송
7. 시간 부족 → "빠른 진행 옵션" 제안
8. 가족 동반 필요 → "부모님/부부 맞춤 가이드" 제시
9. 재구매 신호 → "다음 여행 미리 계획" 제안
10. 뜸했음 → "특별히 그리워합니다" 메시지

## 4️⃣ 판매 성공 6단계

### "6단계 판매"는 뭐예요?

**고객의 마음을 사는 6가지 단계**

```
1단계: 관심 끌기 (Hook)
   ↓ 고객이 "이게 뭐지?" 생각하게 만들기
   
2단계: 감정 연결 (Story)
   ↓ "나도 이런 경험 있어!" 느끼게 만들기
   
3단계: 가치 설명 (Offer)
   ↓ "이게 정말 좋구나" 느끼게 만들기
   
4단계: 불안 해결 (Objection)
   ↓ "이거 정말 괜찮은가?" 의심 해결하기
   
5단계: 서두르게 (Urgency)
   ↓ "빨리 해야겠다" 생각하게 만들기
   
6단계: 결정 받기 (Close)
   ↓ "네, 해주세요" 말하게 만들기
```

### 실제 예시

**저가 고객 (가격 민감한 사람)**

```
1단계: 관심
   광고: "OTA와 여행사의 차이, 알고 계세요?"
   
2단계: 감정
   "실제 고객 이야기: OTA에서 환불 못 받은 사연"
   
3단계: 가치
   "우리는 변경 손실금 100% 환급해드립니다"
   
4단계: 불안 해결
   "정말 환급 가능해요? (진짜 사례 보여주기)"
   
5단계: 서두르기
   "이번 주 예약하면 추가 할인 + 무료 상담"
   
6단계: 결정
   "그럼 오늘 상담 예약해주실래요?"
```

**효도 여행 (부모님 건강 걱정)**

```
1단계: 관심
   광고: "70세 부모님, 크루즈 여행 가도 괜찮을까?"
   
2단계: 감정
   "부모님이 처음 떠난 해외여행, 달라진 것들"
   
3단계: 가치
   "선박 의료진이 매일 건강 체크해드립니다"
   
4단계: 불안 해결
   "비슷한 부모님들, 모두 안전하게 다녀갔습니다"
   
5단계: 서두르기
   "5월 마지막 출발, 자리 3개만 남음"
   
6단계: 결정
   "부모님 건강 확인 후 예약하실래요?"
```

```

   "지난 고객들이 가장 많이 선택한 크루즈 3가지"

## 6️⃣ 성과 확인하기

### "대시보드"는 뭐예요?

**우리 팀이 잘하고 있는지 매일 보는 판**

### 매일 아침 확인할 5가지

```
1️⃣ 어제 몇 명이 상담 신청했나?
   목표: 10명 이상
   
2️⃣ 어제 메시지 열어본 사람이 몇 명?
   목표: 40% 이상
   
3️⃣ 어제 예약 몇 건 들어왔나?
   목표: 3명 이상
   
4️⃣ 지난달 매출이 얼마나 됐나?
   목표: 지난달보다 20% 증가
   
5️⃣ 어제 고객 만족도는?
   목표: 70점 이상
```

### 성과 대시보드 보는 방법

```
1. 홈 화면 → "대시보드" 버튼
2. 5개 숫자 확인 (초록색이면 잘하고 있는 것)
3. 빨간색이면 "왜 떨어졌나?" 확인
4. 주간/월간 추이 그래프로 보기
```

### 이렇게 개선하세요

**만약 상담 신청이 목표보다 적으면:**
```
1. Facebook 광고 클릭 확인
2. 광고 이미지 바꾸기 (다른 사진 3개 테스트)
3. 메시지 제목 바꾸기
4. 3일 후 다시 확인
```

**만약 예약 성공률이 떨어지면:**
```
1. 어떤 단계에서 떨어지나 확인 (상담 → 예약?)
2. 그 단계 스크립트 개선
3. 상담사와 함께 역할 연습
4. 1주일 후 다시 확인
```

## ❓ 자주 묻는 질문

### Q. 고객이 메시지를 안 읽으면?

**A. 자동으로 Day 7, 14, 21에 다시 보냅니다**

처음 3일 안 읽으면, 일주일 뒤 또 보냄.  
그래도 안 읽으면, 2주일 뒤 한 번 더 보냄.  
(자동화니까 당신이 손 안 대도 됨)

### Q. 예약 취소하면?

**A. 자동으로 "다시 해주세요" 메시지 발송**

일주일 뒤: "혹시 마음 바뀌셨나요?"  
2주일 뒤: "특별 할인 쿠폰 드려요"  
1개월 뒤: "정말 도움 필요하시면 전화 주세요"

### Q. 성과가 안 나오면?

**A. 3가지를 확인하세요**

```
1. 대시보드에서 어느 단계가 떨어졌나?
   (상담 신청? 예약? 성공?)
   
2. 그 단계에 해당하는 메시지/스크립트 확인
   
3. 3가지 테스트 해보기:
   - 메시지 이미지 바꾸기
   - 메시지 제목 바꾸기
   - 상담사 톤/태도 개선
```

### Q. 고객 정보는 안전한가?

**A. 네, 안전합니다**

모든 고객 정보는 암호화되어 저장.  
당신과 상담사만 볼 수 있음.  
정부 기준(개인정보보호법) 만족.
