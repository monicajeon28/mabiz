## 📌 L1 렌즈 한 줄 정의

**"크루즈를 모르니까 가격만 보고, 호텔 여행과 비교하면서 '비싼 것 같은데'라고 거절하는 고객"**

---

## 🎯 L1 렌즈 4가지 특징 (최종 확정)

| # | 특징 | 고객 발화 | 심리 근원 | 우리의 대응 원리 |
|---|-----|---------|---------|------------|
| 1 | **가격만 봐요** | "$ 얼마예요?" | 인지부하 + 시간부족 | 가치 재정의 |
| 2 | **다른 여행과 비교** | "호텔이랑 뭐가 달라요?" | 지식격차 + 준거틀 오류 | 카테고리 재구성 |
| 3 | **불안감 크다** | "진짜 괜찮나요? 멀미는?" | 미지의 경험 + 리스크 회피 | 제3자 증명 |
| 4 | **결정이 느리다** | "잠깐... 생각해 볼게요" | 합의 필요 + 신뢰 구축 시간 | 감정 연결 |

## 📞 L1 콜 스크립트 (6단계, 15분)

### **기본 원칙**
✅ **감정 → 논리** (순서 중요)  
✅ **공감 → 질문 → 교육 → 신뢰 → 제안 → 약속**  
✅ **절대 가격 먼저 꺼내지 말기** (고객이 물어올 때까지)  
✅ **호텔 여행을 기준으로 비교하지 않게 유도**  

---

### **PHASE 1: 오프닝 (0-30초)**
**목표**: 공감 + 신뢰 씨앗 심기

```
고객이 "크루즈 관심 있습니다"라고 연락했을 때:

"안녕하세요! 크루즈에 관심 주셔서 감사해요.
혹시 처음이세요? (고객이 "네"라고 답할 확률 70%)

저희가 본 거, 처음 크루즈 찾으시는 분들은 다 똑같아요.
제일 먼저 '가격이 얼마예요?'라고 물어보세요.
그건 아주 정상이고, 중요한 질문이에요.

다만, 가격만으로 결정하면 나중에 후회하는 경우가 많아서...
오늘 15분만 시간 주시면,
당신에게 딱 맞는 크루즈를 찾아드릴 거예요."

[심리학]: 공감(Sympathy) + 시간 약속(설정 명확화)
```

---

### **PHASE 2: 문제 발견 (30초-2분)**
**목표**: 고객의 실제 니즈/아픔 파악 (SPIN의 Situation & Problem) + 고객 자기발화 극대화

```
"지난 여름에 어디 여행 다녀오셨어요?"
→ 고객 답변 경청

"그 여행은 준비하면서 어떤 게 제일 번거로웠어요?"
→ 답변 경청 (대부분: 짐 정리, 호텔 선택, 식당 찾기 등)

[Monica Step 2 - 고객 자기발화 강화]
"그걸 구체적으로 말씀해 주실래요? 
예를 들어, 호텔을 고를 때 뭐가 힘들었어요?"
→ 고객이 3-5줄 자세히 설명 (예: "가이드북을 봐도 어떤 호텔이 좋을지 모르고, 
  아내랑 싸우고, 결국 '그냥 이것'으로 정했는데... 그 호텔은 실망이었어요")

"매번 여행마다 그런 일 반복되니까... 피곤하겠네요?"
→ 고객이 자신의 입으로 문제를 인정: "응, 정말 피곤해. 마지막 여행도 준비 때문에 싸웠어"

[심리학]: SPIN의 Situation(상황) & Problem(문제) + 고객 자기발화 (31배 더 강력)
```

**체크포인트**: 고객이 자신의 입으로 최소 3-5줄 문제를 설명할 때까지 → 신뢰 구축 + 자각 강화

---

### **PHASE 3: 문제 증폭 (2-5분)**
**목표**: 고객의 아픔을 뚜렷하게 인식 (SPIN의 Implication) + 50개 함축 질문 활용

```
[SPIN Implication 질문 - 고객이 자신의 손실을 깨닫게 함]

"그렇다면... 질문 하나만 해도 돼요?

당신이 올해도 또 다른 나라 여행을 다니신다면,
또다시 같은 일이 반복될 거잖아요.
짐 정리, 호텔 찾기, 식당 고르는 일들 말이에요."

→ 여기서 PAUSE [포즈: 3초]

"그게 당신한테 얼마나 부담될까요?
(고객 답변)

혹시 이번엔 아내분이랑 또 싸울 준비가 되어 있어요?
(고객 답변)

정말 호텔을 고르는 게 맞나 싶으면서,
온종일 인터넷 검색 하는 시간... 낭비라고 생각 안 하세요?
(고객 답변)

그렇다면, 매년 '준비 때문에 스트레스'가 반복된다면,
당신이 인생에서 얃마나 많은 여행 기회를 포기하게 될까요?
(고객 깊은 한숨)"

→ 고객이 자신의 니즈를 강하게 느낌: Loss Aversion 극대화

[심리학]: SPIN의 Implication(함축) + 손실회피(시간/체력/관계) + 4초 침묵 규칙
```

**체크포인트**: 고객이 한숨 쉬거나 "정말 그렇네..."라고 무거워할 때 → PHASE 4로 전환

---

### **PHASE 4: 해결책 제시 (5-8분)**
**목표**: 크루즈라는 완전히 다른 카테고리 소개 (새로운 세상 열기)

```
"근데 혹시... 그런 모든 번거로움이 완전히 없는 여행이 있다면?

예를 들어, 여행 가는 날 아침,
당신이 해야 할 일이 '집 문 잠그고 공항 가는 것'뿐이라면?

배가 당신을 밤새 이동시키니까,
아침에 눈을 떠보니 새로운 나라가 되어 있고...

준비도 필요 없고, 짐도 안 옮기고, 식당도 안 찾고...
그냥 휴식만 하면 되는 여행이 있어요.

그걸 크루즈라고 불러요.
당신 얘기 들어보니... 딱 당신 같은 분들이 사랑하는 여행이야."

[심리학]: 
- Narrative Transportation (이야기를 통한 자기투영)
- 카테고리 재정의 (호텔 기반 여행 → 배 기반 여행)
- Cognitive Dissonance 해결 ("번거로움이 없으면?" → 새로운 희망)
```

**체크포인트**: 고객의 눈이 반짝일 때 또는 "어... 그런 게 있어?"라고 물어올 때

---

### **PHASE 5: 사회적 증명 (8-12분)**
**목표**: 제3자 증명으로 신뢰 고도화 (한영인 기법 + 모니카 긍정 확인)

```
"네, 정말 있어요.
그리고 올해 우리 고객 300명이 이미 다녀왔어요.

그 분들이 돌아와서 뭐라고 했을까요?

'처음엔 가격이 비싼 줄 알았는데,
밥, 숙박, 엔터, 간식, 짐 안 옮김... 모두 포함이더니?
이 정도면 호텔 여행보다 오히려 싸네요.'라고 깨달았대요.

특히 처음 크루즈 다니는 분들이 그래요.
'왜 진작 몰랐지?'라고 하더라고요.

당신도 그런 경험을 할 수 있어요."

[심리학]:
- Social Proof (직접 증거 제시)
- Third-party Validation (우리 말이 아닌 고객 말)
- Reframing (비싼 → 싼 것으로 변환)
```

**체크포인트**: 고객이 "아, 그럼 가격도 괜찮겠네"라고 중얼거릴 때

---

### **PHASE 6: 클로징 (12-15분) - 침묵 타이밍 포함**
**목표**: 행동 약속 이끌어내기 (PASONA의 Narrow & Action) + 4초 침묵으로 Loss Aversion 극대화

```
"혹시... 관심 생겼어요?"

[포즈: 3초] ← 고객 답변 기다리기

(고객이 "응"이라고 답할 때까지 기다리기)

"그럼 이렇게 해 볼까요?
당신 가족한테 딱 맞는 배 3개 옵션을 찾아 드릴게요.
가족 구성, 여행 시즌, 예산 정도만 알려주시면...

내일 오전까지 메시지로 보내 드릴게요."

[포즈: 2초] ← 고객이 정보를 말할 준비 시간

(구체적인 다음 약속 = 신뢰 강화)

"이렇게 정보 받아보시고,
편한 시간에 다시 통화해서 어떤 배가 좋을지 이야기해요.

[포즈: 3초] ← 고객의 동의를 기다리기

무조건 사라는 건 아니고, 정보만 드리는 거니까.

[포즈: 2초]

당신 가족 구성부터 알려주시겠어요?"

[심리학]:
- Narrow (선택지 축소 = 결정 부담 완화)
- Action (구체적 다음 약속)
- Affirmation (무조건은 아니라는 심리 안정)
- [포즈] (침묵의 힘 - 고객의 손실회피 감정 강화, 주도권 이전)
```

**체크포인트**: 고객이 가족 구성/시즌/예산 등 구체적 정보 제공 → 성공

**목표**: 선택지 제공 + 손실회피 (놓치면 안 되는 배) 강조

```
📲 "OOO님, 당신 딱 맞는 배 3개 찾았어요!

✈️ 옵션 1: 지중해 7박 (로얄 캐리비안)

- 아이들 액티비티 최고

- 가격대: 900만원

✈️ 옵션 3: 동남아 7박 (로얄 캐리비안)
- 가성비 최고 (당신이 원하는 '편함 + 저예산')

+) 주의: 이 3개 배, 
여름 성수기라 자리 많이 안 남았어요.
당신 같은 분들한테 인기거든요.

편한 시간에 통화해서

[효과]: 
- 구체적 선택지 (선택 부담 ↓)

### **Day 2: 재접촉 (오후 2시)**
**목표**: 질문 유도 + 신뢰 강화

```
📲 "OOO님, 3개 배 중에
혹시 눈에 띄는 게 있어요?

아니면 다른 질문이 있으면
언제든 연락주세요.

예를 들어:
- '가족끼리 가기에 어떤 배가 좋아?'
- '비용이 좀 더 싸면 뭘 추천해?'
