## 2️⃣ Day 0-3 시각적 진화 전략

### 전체 시각적 구조 (모든 세그먼트 공통 Framework)

#### Day 0: 인식 단계 (Problem Agitation)
**심리학**: Loss Aversion (L6) + Scarcity (L1 가격) + Social Proof (L0)

**메시지 구조**:
```
현재 상황 (회색톤) ← VS → 크루즈 꿈 (컬러풀)
지금 vs 30년 후 후회
"지금 가면 건강하게, 못 가면 후회"
```

**비주얼 구성**:
```
[상단 절반]: 회색톤 일상 (사무실 책상, 피곤한 표정, 빽빽한 도시)
[하단 절반]: 컬러풀 크루즈 (선데크, 밝은 미소, 파란 바다)
[오버레이]: 텍스트 "vs" + 시계 아이콘
```

**컬러 처리**:
- 일상 이미지: grayscale 100% (채도 0%)
- 크루즈 이미지: 원본 컬러 100% (채도 +20%)
- 대비: contrast +30%

**CTA 버튼**:
- 크기: 16px × 48px (padding)
- 텍스트: "무료 상담 예약" (마음 설레게 하기)
- 색상: segment별 주색상

**예상 효과**:
- 시각적 충격도: 높음
- CTR: +45% (대비의 영향)

#### Day 1: 관심 단계 (Solution)
**심리학**: Authority (L9) + Social Proof (사례) + Differentiation (L3)

**메시지 구조**:
```
"이 경험을 빼앗길래요?"
크루즈의 구체적 경험 콜라주
"6,000명+ 고객이 경험한 인생 최고의 여행"
```

**비주얼 구성**:
```
[메인 이미지]: 크루즈의 아름다움 (선실 입구 또는 풀)
[4칸 그리드]: 경험 다중 콜라주
  ├─ 선데크 일몰
  ├─ 미쉐린급 다이닝
  ├─ 액티비티 (카약, 스노클링)
  └─ 저녁 쇼 또는 나이트라이프
[하단]: 고객 리뷰 3개 (이름, 별점, 한줄평)
```

**컬러 처리**:
- 메인: segment별 보조색 필터 (opacity 15%)
- 그리드: 각각 밝은 경계선 (2px, 주색상)

**CTA 버튼**:
- 크기: 16px × 52px
- 텍스트: "지금 신청 (할인 30% 마감까지 3일)"
- 배경: segment별 강조색
- 애니메이션: 호버 시 우측 화살표 슬라이드인

**예상 효과**:
- 시각적 풍부도: 높음
- CTR: +38% (다양한 콘텐츠 + 사례)

#### Day 2: 고려 단계 (Offer narrowing)
**심리학**: Price Reduction (L1) + Scarcity (L6 타이밍) + Self-Projection (L5)

**메시지 구조**:
```
"당신을 위한 럭셀리. 이제는 감당할 수 있어요"
구체적 상품 (객실/가격 대조)
"48시간 후 가격이 오릅니다"
```

**비주얼 구성**:
```
[좌측]: 객실 이미지 (내부 투어 스타일, 큰 사이즈)
[우측]: 가격표 + 혜택
  ├─ 원래 가격: $8,999 ✘ (빨강)
  ├─ 지금 가격: $4,999 ✓ (초록, 크게)
  ├─ 혜택 5개 (체크마크)
  └─ 타이머: "48시간 남음"
```

**컬러 처리**:
- 객실 이미지: 자연색 유지, 밝음 +15%
- 가격 비교: 원가 (회색/빨강), 할인가 (초록)
- 배경: segment별 밝은 배경색

**CTA 버튼**:
- 크기: 18px × 56px (크기 증대)
- 텍스트: "이 가격에 예약하기" (FOMO 유도)
- 색상: 밝은 초록 또는 segment별 강조색
- 애니메이션: 깜빡이는 타이머 효과

**예상 효과**:
- 시각적 긴박감: 높음
- CTR: +52% (가격 비교 + 타이머)
- 전환율: +41% (결정 유도)

**비주얼 구성**:
```

  ├─ 59분
  └─ 47초
[오버레이 우하단]: "Last seat" 또는 "2/3 sold"
[하단]: 추가 혜택 3개
  ├─ 🎁 온보드 선물 (와인/향초)
  ├─ 🏨 업그레이드 가능성 30%
  └─ 💳 60개월 무이자 할부
```

**컬러 처리**:
- 메인 이미지: segment별 컬러 필터 (opacity 20%), 밝음 +20%
- 타이머: 빨강 (#FF4444) 또는 segment별 강조색
- "Last seat": 빨강, 깜빡임 애니메이션
- 혜택 아이콘: segment별 강조색

**CTA 버튼**:
- 크기: 20px × 60px (최대 크기)
- 텍스트: "지금 예약하고 $500 보너스 받기"
- 배경: segment별 주색상 (가장 진한 톤)
- 애니메이션: 깜빡이기 (2초 간격) + 윤곽선 펄싱
- 위치: 화면 중앙 + 스티키 하단 (모바일)

**예상 효과**:
- 시각적 강압성: 매우 높음

- 전환율: +65% (최종 결정 유도)
