## 1️⃣ 완전 콜 스크립트 (Phase 0-4, 신민형 5STEP 핵심)

### 📍 L10 고객 심리 프로필

```
상태:
├─ 이미 크루즈를 "구매하기로 결정"한 상태
├─ 하지만 마지막에 "선택회피" 증상 나타남
├─ "지금 할까? 다음 달? 아니 내년?"
├─ 정보는 충분한데, 행동만 지연

특징:
├─ 높은 구매 의도 (이미 상품 선택함)
├─ 낮은 결정 확신도 (마지막 순간에 흔들림)
├─ 손실에 대한 과도한 민감성
└─ "지금 안 하면 나중에 후회하지 않을까?" 불안감

기대되는 전환율: 50% → 75% (+50%)
```

### 🎯 L10 핵심 전환 메커니즘

**선택회피 이론 (Overchoice Paradox)**:
- 고객이 정보를 "너무" 많이 알아서 오히려 선택 못 함
- 해결책: "정보 제거" 아니라 "선택지 축소" (삼중선택 도입)
- SPIN의 "Need-Payoff" + 손실회피 극대화

**신민형 5STEP의 심리학 기초**:
```
Step 1 (주도권 탈취): Commitment 이론
  └─ 고객이 "네"라고 한 번 말하면, 
     그 다음은 일관성 유지 본능으로 움직임

Step 2 (비교 명확화): Framing 효과
  └─ "크루즈 vs 일반여행" 선택으로 집중력 복원
  
Step 3 (손실 앵커): Loss Aversion
  └─ "지금 못 가면 내년 더 비싼 가격" 
     → 손실액을 구체적으로 계산

Step 4 (삼중선택): Paradox of Choice 해결
  └─ Option 1 (최고): 지금 예약
  └─ Option 2 (차선): 1주일 고민 후
  └─ Option 3 (최악): 자료만 받기
  → 3개만 놓고 선택하면 판단 쉬움

Step 5 (즉시행동): Immediate Action
  └─ "지금 예약" → "24시간 내 링크" → "3일 내 확정"
```

### 📞 Phase 0: 오프닝 (준비 완료 상태 감지)

#### 0-A: 기본형 (높은 관심도)
```
"xxx님 안녕하세요, 크루즈닷의 모니카입니다.

[공감]
오늘 전화 드린 이유가, 이미 우리 상품을 알아보신 분들은
대부분 여기서 한 가지 같은 고민을 하거든요.

[질문으로 리드]
이미 많이 알아보신 것 같은데,
지금 바로 예약하지 못하게 하는 마지막 고민이 뭔가요?"
```

#### 0-B: 이미 상품 선택한 경우
```
"아, 그럼 이미 배도 정했고, 객실도 정했다는 뜻이네요?

[확인]
그럼 이제 '언제 예약할까'만 남은 거 맞죠?

[심리적 거리 단축]
저희 고객들이 이 단계에서 대부분
'지금 할까? 아니면 다음 달?'로 고민하는데,
제가 그 고민을 3분 만에 확실하게 정리해드릴 수 있어요."
```

#### 0-C: 기존 고객 재활성화
```
"이전에 크루즈를 한 번 경험하셨다니까,
이번엔 준비도 훨씬 간단하실 거예요.

[신뢰 활용]
우리 경험 있는 고객분들은
'이번엔 왜 못 가나' 이 정도만 고민해요.

그 고민, 2분 안에 풀어드릴게요."
```

### 🔍 Phase 1: 욕구 발굴 (마지막 의심점 파악)

L10의 핵심은 **"마지막 의심점이 정확히 뭔가 찾기"**

#### Q1-A: 주요 거부 요인 4가지 (선택형)
```
"제가 여쭤볼 게 있는데,
지금 예약을 망설이게 하는 건
다음 중 어디가 가장 커요?

1) 객실 선택이 아직도 헷갈린다
2) 일정이 안 정해졌다
3) 함께할 사람을 아직 설득 못 했다
4) 그냥... 지금이 맞는 타이밍인지 불안하다"

→ 고객이 선택한 항목으로 집중
```

#### Q1-B: 객실 불확실성
```
Q: "객실 선택이 문제라면, 
   어떤 객실이 고민이 되세요?

A: 바다뷰 vs 발코니? 가격 vs 뷰?

→ 즉시 해결: 
'발코니 객실은 사실 대부분의 고객이 선택하고,
가격은 월 7만원만 더 내면 되는데,
배에서 밤하늘 경험이 완전 달라져요.
많은 분들이 나중에 업그레이드 하는데,
지금 하면 할인이 3% 더 줄여요.'"
```

#### Q1-C: 일정 불확실성
```
Q: "일정이 안 정해지셨다면, 
   대략 언제쯤이 가능해요?

A: 3개월 내 / 6개월 내 / 연말? / 내년?

→ 즉시 해결:
'아, 그럼 지금이 정말 마지막이에요.
3개월 뒤 크루즈는 가격이 이미 +5% 올라가 있거든요.
선실도 한정되고.

근데 "일정은 미정"이라면, 
이렇게 하시는 분들 많아요:
먼저 예약해두고, 2주 전에 일정 확정하는 거예요.
변경 수수료도 거의 없고요.'"
```

#### Q1-D: 타이밍 불안
```
Q: "그럼 '지금이 맞는 타이밍인가?'
    이 생각이 크신가요?

→ Loss Aversion 극대화:
'아, 그 생각 정말 중요한데요.
제가 역으로 여쭤볼게요:

지금 못 가면, 언제쯤 가실 것 같아요?
- 내년? (그럼 가격이 또 +3-5% 올라갈 거고)
- 아니면 그 다음년? (그때까진 선실 못 본 채로)

그리고 지금 가격은 이번 달까지만 유지돼요.
내일부터는 올려야 할 것 같아요.

실제로 선실이 3개 남음.
2시간 뒤쯤 또 팔릴 확률이 높아요.'"
```

### 💪 Phase 2: 욕망 증폭 (손실 극대화 + 긴급성)

**핵심 전략**: Loss Aversion을 이용해 "지금 안 하면 손실"을 구체적으로 계산

#### 손실 1️⃣: 시간 손실
```
"알았어, 그럼 제가 이런 질문을 할게.

지금부터 48시간이 정말 마지막이에요.
이유가:

첫 번째, 선실 손실:
- 현재 남은 발코니 객실: 3개 (₩700만 상당)
- 1개 (₩500만 상당)  
- 1개 (₩300만 상당)

선실 하나가 팔릴 때마다,
다음 등급으로 내려가야 돼요.

예를 들어, 오늘 안에 안 사면
발코니는 없고 내일부터 기본실만 남아요.
그럼 경험이 아예 달라져요."
```

#### 손실 2️⃣: 기회 손실 (라이프스테이지)
```
"두 번째, 기회 손실:

당신이 말한 '지금이 타이밍인가?'
실제로 그건 지금이 마지막 타이밍이라는 뜻이에요.

왜냐면:
- 자녀가 대학 가기 전 → 이번이 정말 마지막
- 부모님이 건강할 때 → 내년엔 못 갈 수도
- 당신의 에너지 있을 때 → 계속 미루면 나중에 못 가

많은 고객들이 '내년에'라고 했다가,
1년 뒤 '역시 올해 가니까 실패했어'라고 해요."
```

#### 손실 3️⃣: 가격 손실
```
"세 번째, 가격 손실:

이 가격(예: ₩1,800,000)은 오늘까지만이에요.

내일부터:
- 가격: +3% (₩1,854,000)
- 선실: 기본실만 남음
- 할인: 지금 주는 신규 10% 할인 안 됨

그러니까 지금 안 하면:
5일 뒤엔 +54,000원 더 내야 하고,
경험도 2등급 떨어지는 거예요.

평생 할인 개념으로 보면,
5년 후 6크루즈 탈 때마다
매번 이 차이가 생긴다는 뜻이에요.
총 324,000원 손실."
```

### 🎯 Phase 3: 상품 설명 (고객의 의심점별 해결책)

이미 Phase 1에서 파악한 의심점별로 명확한 해결책 제시

#### 만약 객실이 의심점이면:
```
"그래서 제가 추천하는 객실은 발코니예요.

이유:
1) 가격 - 기본실보다 월 7만원만 더 내면 됨
2) 경험 - 배 위에서 밤하늘을 혼자만의 프라이빗한 공간에서 즐김
3) 재활용 - 배에서의 시간을 가장 즐길 수 있음

그리고 우리 고객들 100명 중 78명이
발코니 객실 선택해요.

만약 지금 발코니 선택하면
'초급자 할인' 3% 추가로 드려요.
내일부터는 못 줍니다."
```

#### 만약 일정이 의심점이면:
```
"일정이 미정이라고 하셨는데,
대부분의 고객분들이 이렇게 해요:

1단계) 지금 예약 확정
2단계) 2주 전에 구체적 날짜 결정
3단계) 변경 수수료 거의 없음

왜냐면, 우리 배는 항상 있고,
예약 변경도 쉽거든요.

그래서 '언제 가야 할지 모르겠으면
먼저 예약해두고 나중에 정한다'가
현명한 고객들의 선택이에요."
```

#### 만약 비용이 의심점이면:
```
"비용이 크신가요?

그럼 이렇게 생각해봐요.

일반 여행 (3박 4일, 가족 4명):
- 비행기: 320만원
- 호텔: 120만원
- 식사: 50만원
- 입장료: 30만원
총 520만원

우리 크루즈 (3박 4일, 가족 4명):
- 크루즈: 180만원 (모든 식사/침대/쇼 포함)
총 180만원

차이: 340만원 더 저렴해요.

분할로 하면?
월 30만원 × 6개월이면 끝.
커피 한 잔 가격으로 세계여행이에요."
```

### 🎬 Phase 4: 신민형 5STEP 클로징 (절대 순서 지키기)

**이것이 L10의 핵심입니다. 이 5가지 Step을 정확하게 따라야 합니다.**

#### Step 1️⃣: 주도권 탈취 (Commitment 원리)

```
모니카: "그럼 정리가 되셨어요. 
         이번 달에 가시기로 하셨어요? 
         네, 가시기로?"

고객: (침묵 또는 "음...")

모니카: (2초 침묵 유지)
        "말씀 안 하신 거 보면 아직 고민이시네요.
         근데 고민할 필요 없어요.
         
         여기서 일단 '네, 가기로 하겠습니다'
         라고 한 번만 말씀해 보세요."

심리학 원리:
- 고객이 "네"라고 한 번 말하면,
  뇌는 그 약속을 지키려고 움직임
- 심리학자 Cialdini의 "일관성 원리"
- "말로 한 번 약속하면 실행할 확률 90%"
```

#### Step 2️⃣: 비교 명확화 (Framing 효과)

```
모니카: "그럼 마지막으로 제가 물어볼 게 있어요.

         일반 여행이랑 크루즈, 
         이중 어느 게 더 나으신 것 같아요?"

고객: "음... 크루즈가..."

모니카: "맞죠? 
         크루즈가 더 경제적이고, 
         시간도 효율적이고, 
         경험도 풍부하잖아요.
         
         그럼 크루즈를 선택한 거 확실하신 거네요."

심리학 원리:
- 고객이 "크루즈가 낫다"고 말한 순간,
  그 선택에 심리적으로 더 집중
- 정보 과부하 상태에서 벗어남
- "내가 이미 선택했다" → 행동할 차례
```

#### Step 3️⃣: 손실 앵커 (Loss Aversion)

```
모니카: "그 다음 한 가지만 더.
         
         지금 못 가면, 
         언제쯤 생각하고 계세요?"

고객: "음... 내년쯤?"

모니카: "내년이면, 
         가격이 또 올라가 있을 거고,
         '올해 그때 예약했으면...' 
         하실 거예요.
         
         실제로 지금 가격은:
         - 이번 달까지만 ₩1,800,000
         - 내달부터 ₩1,854,000
         
         그러니까 미루면 54,000원 손실이에요.
         그 돈으로 배 위에서 와인 한 잔 못 마신다는 뜻이죠.
         
         지금 이 선실(발코니)도,
         내년엔 못 본다고 봐야 해요.
         팔려버리니까."

심리학 원리:
- 손실 1,000원 > 이익 1,000원 (심리적 무게)
- 구체적 숫자로 현실화 (₩54,000)
- 미래의 후회를 지금 느끼게 함
```

#### Step 4️⃣: 삼중선택 (Overchoice 해결)

```
모니카: "좋아요. 그럼 이제 선택을 정해봐요.
         세 가지만 있으니까 쉬울 거예요.

         ─────────────────────────────
         Option 1: 지금 바로 예약
         ─────────────────────────────
         
         선실 3개 남음 (발코니 ₩1,800,000)
         지금 서명하시면:
         ✓ 발코니 확정
         ✓ 초급자 할인 3% 추가
         ✓ 내일 예약 링크 보내드림
         ✓ 3일 내 탑승 확정
         
         저희 강력 권장 (70% 고객이 이 선택)
         
         ─────────────────────────────
         Option 2: 1주일 고민 후 콜백
         ─────────────────────────────
         
         저희가 1주일 뒤에 다시 전화해요.
         그때까진 생각할 시간을 드려요.
         
         근데 (조용히) 1주일 뒤엔
```

## 5️⃣ L10 손실 프레임 극대화 데이터 셋

L10 고객과 대화 시 사용할 구체적 손실 데이터

### 💰 가격 손실 계산기

```
현재 가격: ₩1,800,000
내일 가격: ₩1,854,000
차이: ₩54,000

1주일 뒤: ₩1,908,000 (차이 ₩108,000)
1달 뒤: ₩1,962,000 (차이 ₩162,000)

평생 손실 계산 (5년, 6크루즈):
매 크루즈마다 ₩54,000 손실
총 ₩324,000 손실

월급 대비:
- 월급 400만원 기준, 일당: 13만원
- ₩324,000 = 약 2.5일분 월급
- "2.5일 일한 돈을 그냥 잃는다"
```

### ⏰ 시간 손실 (선실)

```
현재 (지금): 발코니 3개 남음
12시간 뒤: 발코니 1개 남음
24시간 뒤: 발코니 0개 (기본실만)
48시간 뒤: 기본실도 3개 남음

실제 영향:
- 발코니: 바다뷰 + 프라이빗 발코니 + 야외 시간 최고
- 기본실: 창문 작음 + 공공 시설만 이용

고객의 배 위 2박을 위해:
- 발코니: 프라이빗한 시간 가능
- 기본실: 공공 시설에만 머물러야 함

경험의 질 차이: 50% 이상
```

### 🎯 라이프스테이지 손실

```
대상: 40대 가족

현재 상황:
- 자녀: 초등 4학년 (10세)
- 부모: 70대 초

2-3년 뒤:
- 자녀: 중학생 (13세) → 부모와 여행 거부 확률 높음
- 부모: 건강 악화 위험 (당뇨/관절)

"지금 안 가면?":
- 자녀는 "너무 어렸다/너무 자랐다"
- 부모는 "다리 아파서/건강이"
- 결과: 3세대 함께 크루즈 탑승은 "마지막 기회"

실제 후회:
- "그때 가니까 얼마나 좋았는데"
- "엄마가 건강할 때 갔길 다행"
- "아이가 기억할 나이에 갔네"
```

### 🏥 건강 손실

```
연령별 여행 권장 한계:

40-50대: 여행 최고의 시기
60대: 여전히 활동적 (관절 주의)
70대: 건강 악화 급속 진행
80대: 해외 여행 위험

현실:
- 올해 70세 → 내년 71세 → 그 다음 72세
- "매 해마다 건강은 더 나빠짐"
- 회복 속도도 느려짐
- 배 위 계단 오르내리 어려워짐

결론: "부모를 데려가는 거라면 지금이 정말 마지막"
```
