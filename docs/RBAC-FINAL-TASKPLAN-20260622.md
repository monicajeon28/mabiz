## 🎯 역할별 권한 정의 (최종 확정)

### 👔 **관리자** (최고 권한)
```
✅ 전체 조직 고객 + 개인 모두 문자/이메일 발송 가능
✅ 원하는 대로 커스터마이징해서 발송
✅ 그룹 관리 (개인/관리용도) - 공유 안 함
✅ 전체 통솔 및 감시
✅ 모든 데이터 조회/수정/삭제 가능
```

### 🎯 **대리점장** (관리자 못할 때 대신 하는 사람)
```
✅ 판매원A의 고객한테도 문자 발송 가능 ← 핵심!
✅ 개인 커스터마이징 발송 가능
✅ 자신의 소속 판매원/프리세일즈 관리
✅ 구매관리: 자신 + 자신의 판매원 + 자신의 프리세일즈
✅ 문자/이메일 기록 조회 (자신의 조직만)

❌ 못하는 것:
  - 다른 대리점 데이터 볼 수 없음
  - 관리자 설정은 변경 불가
```

### 👤 **판매원** (고객 관리하는 사람)
```
✅ 자신의 고객 DB 관리 볼 수 있음 (보기만 OK)
✅ 자신의 고객한테만 문자/이메일 발송 가능
✅ 상담한 고객이 구매하면 수당인정 ✅
✅ 자신의 판매관리만 봄 (구매내역)
✅ 문자/이메일 기록 조회 (자신이 보낸 것만)

❌ 못하는 것:
  - 고객 삭제 불가 (대리점장/관리자만 가능)
  - 다른 판매원 고객 못 봄
  - 조직 전체 발송 불가
```

### 🚫 **프리세일즈** (상담 담당, CRM 없음)
```
❌ 로그인 자체 불가 (CRM 진입 완전 금지)
✅ 상담 신청 접수 → 대리점장 권한으로 자동 들어옴
✅ 자신의 판매관리만 봄 (이미 구현됨)
✅ 구매인정만 표시 (수당)

❌ 못하는 것:
  - 고객 DB 조회 불가
  - 상담 직접 입력 불가
  - 고객 접근 완전 불가
```
