# 기존 고객 재활성화 콜 스크립트 (15분) — 10렌즈 개선 최종판

**상담사가 보면서 그대로 읽을 수 있는 실전 스크립트**  
**대상**: 크루즈/렌탈/호텔 기존 경험 고객 (product_inactive_db)  
**목표**: 기존 경험 회상 → V2 업그레이드 가치 → 즉시 신청 (70% 이상 클로징)  
**시간**: 15분  
**핵심**: "오래간만이에요" + "당신은 이미 경험했으니 더 쉬워요" + "개인 맞춤"

## 🎯 CRM Integration Guide (product_inactive_db)

### 고객 데이터 조회 (Phase 1)

```
SELECT 
  c.id,
  c.name,
  pi.product_name,        -- "카리브해 7일 크루즈"
  pi.trip_date,           -- "2024-03-15"
  pi.cabin_type,          -- "스탠다드"
  pi.dining_type,         -- "뷔페식"
  pi.ports_count,         -- 4
  pi.price_paid,          -- 3500000
  c.spouse_name,          -- "배우자명"
  c.travelers_count       -- 총 4명
FROM contacts c
JOIN product_inactive c.id = pi.contact_id
WHERE pi.last_trip_date < DATE_SUB(NOW(), INTERVAL 1 YEAR)
  AND c.status != 'cancelled'
ORDER BY pi.trip_date DESC;
```

### 콜 후 상태 업데이트 (Phase 8)

UPDATE product_inactive

-- Day 0 이메일 발송 트리거
