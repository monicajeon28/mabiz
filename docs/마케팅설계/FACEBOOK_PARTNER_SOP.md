# 마비즈 크루즈닷 파트너 페이스북 운영 SOP v1.0

**버전**: 2026-05-24  
**대상**: 마비즈 크루즈닷 파트너 (개인사업자/소상공인)  
**목표**: 파트너 개인 페이스북에서 월 500-1,000명 도달, 2-3% 예약 전환율 달성

---

## 📊 5단계 성장 전략

### Stage 1: 타겟층 분석 및 콘텐츠 전략 수립 (Week 1-2)

**목표**: 고객 세그먼트 파악 + 콘텐츠 로드맵 수립

| 활동 | 상세 | 산출물 |
|------|------|--------|
| **타겟 인사이트 분석** | Facebook Business Suite → Insights → "People" 탭에서 위치, 나이, 성별, 관심사 확인 | 세그먼트 3개 (5060시니어, 3040전문직, 패밀리) |
| **페르소나 정의** | 크루즈/렌탈/여행에 관심 있는 고객 프로필 작성 | 페르소나 카드 (나이/소득/관심사/구매심리) |
| **콘텐츠 캘린더** | 주 3-5회 포스팅 일정 수립 (시간/포맷/주제) | 4주 달력 (Google Sheet or Notion) |
| **브랜드 톤앤매너** | 페이스북 커뮤니케이션 가이드라인 | 톤 정의 (친근+신뢰+전문성) |

**KPI**: 페이지 팔로워 100→500→1,000명 (4주 목표: 500명)

---

### Stage 2: 비디오 콘텐츠 및 실시간 소통 강화 (Week 3-6)

**목표**: 알고리즘 선호 콘텐츠 + 고객과 직접 대화

#### 2-1. Reels 제작 (주 2-3회)

**스펙**: 15-30초, 720p 이상, 100% 자막 필수

| 콘텐츠 유형 | 예시 | 빈도 |
|-------------|------|------|
| **여행지 영상** | 여행지 4K ASMR (파도소리, 새소리) | 주 1회 |
| **고객 후기** | Before/After (예: 휴식 전/후 표정변화) | 주 1회 |
| **가격 정보** | 시즌별 가격 애니메이션 (2023 vs 2024 vs 2025) | 주 1회 |
| **시니어 맞춤** | "55세 이상도 이렇게 즐겁게" 콘셉트 | 월 2회 |

**편집 팁**:
- 3초 이상 각 장면 유지 (흔들리지 말 것)
- 자막: 명확한 한글 (글자크기 18pt 이상)
- 음성: 배경음악 저음량, 대사는 명확함
- CTA: 마지막에 "예약하기" 자막 + 링크 표시

#### 2-2. Facebook Live (주 1-2회, 저녁 7-8시)

**목표**: 고객과 실시간 상호작용 + 신뢰도 상승

| 라이브 주제 | 진행 방식 | 기대효과 |
|------------|---------|--------|
| **실시간 Q&A** | 고객 질문 수집→즉석 답변 (3분/질문) | 신뢰도 +30% |
| **제품 언박싱** | 신규 렌탈카/여행패키지 소개 | 참여율 +40% |
| **고객 후기 인터뷰** | 실제 여행객 초대→스토리텔링 | 리드 생성 +25% |
| **시즌 특가 공지** | 라이브 중에만 적용 가능한 특가 (시간 제한) | 즉시 예약 +50% |

**진행 체크리스트**:
- [ ] 📱 휴대폰 거치대 (흔들림 방지)
- [ ] 🎤 마이크 테스트 (명확한 음성)
- [ ] 💡 조명 확인 (역광 방지, 얼굴 밝게)
- [ ] 🔔 라이브 공지 (30분 전 페이지에 사전고지)
- [ ] 💬 댓글 모니터링 (시청자 질문 실시간 답변)
- [ ] ⏰ 정시 종료 (약속된 시간 준수)

#### 2-3. 댓글 응답 시스템 (2시간 이내)

**원칙**: 모든 댓글/메시지에 2시간 내 응답

| 상황 | 응답 스크립트 |
|------|----------------|
| **가격 문의** | "좋은 질문입니다! 우리 패키지는 성수기 [가격] / 비수기 [가격]입니다. 조기예약 10% 할인도 있으니 편하신 시간에 DM 주세요 😊" |
| **부정 댓글** | "피해를 주셔서 정말 죄송합니다. 구체적인 상황을 DM으로 말씀해주시면 최선으로 해결해드리겠습니다. 귀한 지적 감사합니다!" |
| **긍정 후기** | "정말 감사합니다! 당신 같은 고객분이 저희 가장 큰 자산입니다. 다음 여행도 저희와 함께하세요 ❤️" |
| **예약 완료** | "예약 감사합니다! 여행 준비는 [페이지 링크]에서 체크리스트를 보세요. 출발 1주일 전 최종 안내를 드리겠습니다!" |

#### 2-4. 팬 참여 이벤트 (월 1회)

| 이벤트 | 진행 방식 | 경품 |
|--------|----------|------|
| **댓글 이벤트** | "이 영상을 공유 + 댓글 '당신의 꿈의 여행지'를 남기세요" → 추첨 | 할인권 (30만원, 15만원, 5만원) |
| **공유 이벤트** | 게시물 공유 → 추첨 (3일 개최) | 여행용품 세트 or 할인코드 |
| **댓글 참여 추가보상** | 3개 댓글 이상 남긴 사람 모두 → 추가 할인코드 | 추가 5% 할인 |

**기대효과**: 월 200-300명 참여 → 도달 2배 증가 → 신규 팔로워 100명 이상

### Stage 3: AI 기반 마케팅 도구 활용 (Week 7-8)

#### 3-1. Advantage+ 광고 (월 20-30만원)

**설정 방법**:
1. Facebook Ads Manager → "캠페인 만들기"
2. 목표: "예약" 또는 "리드"
3. 예산: 월 200,000-300,000원
4. 타겟: 자동(AI가 판단)
5. 광고 소재: 위의 Reels 5개 중 최고 성과 2개 선택

**기대 ROAS**: 3:1 이상 (1만원 광고 → 3만원 매출)

#### 3-2. 자동 게시물 최적화

Facebook Business Suite에서:
- "게시물 발행" 시 자동 최적화 ON
- 메타 AI가 최적 발행 시간 추천
- 자동 자막 생성 활용 (한국어 검수 후 사용)

#### 3-3. CRM 연동

**페이스북 리드 폼 → 크루즈닷 예약 시스템 자동 연결**

1. Ads Manager → "리드 폼" 설정
2. 수집 정보: 이름, 전화번호, 선호여행지, 예상예산
3. Zapier/Make를 통해 크루즈닷 CRM으로 자동 전송
4. CRM에서 파트너에게 즉시 알림

### Stage 4: 고객 상호작용 및 커뮤니티 활성화 (Week 9-12)

#### 4-1. 고객 후기 공유 (월 2-3건)

**프로세스**:
1. 여행 후 고객에게 DM으로 후기 영상/사진 요청 (2주 내)
2. 저작권 확인 (본인 촬영 or 허가)
3. 페이지 "후기" 또는 "성공사례" 섹션에 재포스팅
4. 원본 고객에게 태그 + 감사 메시지

**신뢰도 상승 예상**: +40%

#### 4-2. Messenger 챗봇 설정

**자동응답 흐름**:
```
고객 메시지 수신
↓
[선택지 제시]
1) 예약 문의
2) 가격 정보
3) 여행 팁
4) 기타
↓
[각 선택지별 자동응답]
- 예약: "예약 페이지 링크 + 가능 날짜"
- 가격: "시즌별 가격표 이미지"
- 팁: "FAQ 자동응답"
- 기타: "담당자 연결 (30분 이내)"
```

**설정**: Facebook Business Suite → "Automated Messages" → 활성화

#### 4-3. 커뮤니티 토론 게시물 (월 2회)

**개방형 질문으로 댓글 참여 유도**:

| 게시물 | 예상 댓글 | 보상 |
|--------|----------|------|
| "당신의 꿈의 여행지는 어디인가요? 그 이유는?" | 50-80개 | 상위 5개 댓글 → 추가 5% 할인 |
| "처음 크루즈 여행 가실 때 가장 걱정했던 점은?" | 40-60개 | 가장 많은 좋아요 댓글 → 할인권 |
| "렌탈카 여행 팁을 공유해주세요!" | 30-50개 | 모든 댓글 참여자 → 추가 할인 |

**기대 참여율**: 5-10% of 페이지 팔로워

#### 4-4. 월별 파트너 네트워크 라이브

**진행**: 크루즈닷 직원 + 파트너 합동 라이브 (월 1회)
- 신상품 공지
- 파트너 성공사례 공유
- 고객 Q&A

### Stage 5: 2025 최신 트렌드 반영 및 지속 성장 (Ongoing)

#### 5-1. 시즌별 콘텐츠 카드뉴스

| 계절 | 주제 | 포스팅 |
|------|------|--------|
| **Q1 (1-3월)** | 설/구정 특가, 봄 휴가 계획 | 매주 1회 |
| **Q2 (4-6월)** | 여름휴가 성수기, 가족여행 추천 | 주 2회 |
| **Q3 (7-9월)** | 추석 특가, 가을 감성 여행지 | 주 2회 |
| **Q4 (10-12월)** | 크리스마스, 년말 결산, 신년 예약 | 주 3회 |

#### 5-2. 밈/챌린지 활용

**예**: "여행 전 vs 여행 중 vs 여행 후" 비포애프터 영상
- 트렌드 감지: 메타 공식 블로그, TikTok 핫트렌드 모니터링
- Reels에서 빠르게 테스트 (1주일)

- 페이스북 그룹 개설: "크루즈 여행 준비 커뮤니티"
- 스레드(Meta의 새 SNS) 연동 테스트
- 가상 여행지 투어 (메타버스)

## 🎨 저품질 콘텐츠 필터링 및 품질 유지

### 영상 품질 기준

| 항목 | 기준 | 거절 기준 |
|------|------|----------|
| **해상도** | 최소 720p (권장 1080p) | 480p 이하 |
| **길이** | 15-30초 (Reels) / 3-5분 (Live) | 15초 미만, 5분 초과 |
| **자막** | 100% 필수 (글자크기 18pt+) | 자막 없음 |
| **음성 품질** | 명확한 한국어, 배경음악 저음량 | 음성 왜곡, 심한 잡음 |
| **편집** | 3초 이상 장면 유지, 부드러운 전환 | 손떨림 2점 이상, 급하게 움직임 |
| **내용** | 교육/감동/정보/재미 중 1개 이상 | 불법/선정적 내용, 경쟁사 폄하 |

### 이미지 품질 기준

| 항목 | 기준 | 거절 기준 |
|------|------|----------|
| **크기** | 1200×628px 이상 (페이스북 표준) | 640×480px 이하 |
| **텍스트** | 최대 20% (권장 10%) | 텍스트 50% 이상 |
| **색감** | 대비 명확 (다크/밝음 구분) | 흐릿함 (색상대비 2:1 미만) |
| **로고** | 명확한 브랜드 표기 | 로고 불명확 |
| **언어** | 한글만 사용 (영문은 보조) | 오타/띄어쓰기 오류 |

### 텍스트 품질 기준

| 항목 | 기준 | 거절 기준 |
|------|------|----------|
| **길이** | 100-300자 (짧고 명확) | 50자 미만, 500자 이상 |
| **호출** | CTA 필수 ("예약하기", "문의하기") | CTA 없음 |
| **링크** | 클릭 가능 URL (단축 링크 가능) | 링크 없음, 깨진 링크 |
| **문법** | 맞춤법 100% 검수 | 오타 5개 이상 |
| **톤** | 친근+신뢰+전문성 균형 | 과장 ("무조건 최고", "100% 보장") |

### 발포 전 필수 체크리스트

```
□ 저작권 확인 (본인 촬영 or 허가서)
□ 개인정보 마스킹 (고객 얼굴→모자이크 or 실명 제거)
□ 자막 검수 (띄어쓰기, 오타, 명확성)
□ 링크 동작 확인 (실제 클릭 테스트)
□ 모바일 미리보기 (세로 방향 최적화, 텍스트 가독성)
□ 톤&매너 정렬 (브랜드 가이드라인 준수)
□ 시간대 최적화 (타겟층 활동 시간 확인)
□ 경영진 승인 (파트너 → 매니저 최종 확인)
```

### 포스팅 형식별 가이드

#### 1️⃣ Reels (숏폼 영상, 15-30초)

**빈도**: 주 2-3회  
**최고 성과 시간**: 오후 2-4시, 저녁 7-9시  
**목표**: 도달률 극대화, 알고리즘 선호

| 콘텐츠 유형 | 길이 | 예시 |
|------------|------|------|
| 여행지 ASMR | 15초 | 파도소리+해변 영상 |
| 고객 후기 | 20초 | "저는 60세인데도 즐거웠어요" |
| 가격비교 | 25초 | "작년(3000만원) vs 올해(2500만원)" |
| 콘셉트 영상 | 30초 | "일상 vs 크루즈" 콜라보 |

**자막 팁**:
- 첫 3초 내에 시선 확보 (자막: "이걸 모르면?")
- 매 3초마다 장면 변화
- 마지막 3초에 CTA ("예약하기 링크 클릭")

#### 2️⃣ Carousel (슬라이드, 3-5 이미지)

**빈도**: 주 1-2회  
**최고 성과 시간**: 오후 3시, 저녁 6-8시  
**목표**: 상세 정보 전달, 페이지 내 체류 시간 증대

| 슬라이드 | 내용 |

## 🎯 팬 확보 및 참여율 증대 기법

### 핵심 기법 5가지

#### 1️⃣ 빠른 응답 시스템 (2시간 이내)

**원칙**: 모든 댓글, 메시지, 문의에 2시간 이내 응답

**응답 스크립트 템플릿**:

```
가격 문의
→ "좋은 질문입니다! 우리 패키지는 [시즌별 가격] 입니다. 
   조기예약 10% 할인 + [추가혜택]도 있습니다. 
   편하신 시간에 DM 주세요 😊"

부정 댓글 (과거 서비스 불만)
→ "피해를 주셔서 정말 죄송합니다. 구체적인 상황을 
   DM으로 알려주시면 최선으로 해결해드리겠습니다. 
   고마운 지적입니다."

긍정 후기
→ "정말 감사합니다! 당신 같은 고객분이 저희 가장 큰 자산입니다. 
   다음 여행도 저희와 함께하세요 ❤️"
```

**기대효과**: 응답률 95% → 신뢰도 상승 → 전환율 3-5% ↑

#### 2️⃣ 사용자 생성 콘텐츠(UGC) 재포스팅 (월 2-3건)

**프로세스**:
1. 여행 후 고객에게 DM: "여행 영상/사진 있으시면 공유 부탁드립니다!"
2. 고객 허가 확인 (저작권)
3. 사진/영상 다운로드
4. 페이지에서 "고객 후기" 섹션으로 재포스팅
5. 원본 고객에게 태그 + 감사 메시지 ("이분은 지난달 저희와 여행했어요 😊")

**신뢰도 상승**: +40% (실제 고객 증언)  
**알고리즘 선호**: +20%  
**참여율**: +30%

#### 3️⃣ 커뮤니티 토론 게시물 (월 2회)

**개방형 질문**으로 댓글 참여 유도 (댓글당 팔로워 2-3배 증가)

| 질문 | 예상 댓글 | 참여 유도 문구 |
|------|----------|------------|
| "당신의 꿈의 여행지는 어디인가요? 그 이유는?" | 50-80개 | "댓글에서 유명한 곳들이 나오네요. 저는..." |
| "처음 크루즈 여행 가실 때 가장 걱정했던 점은?" | 40-60개 | "배멀미 때문에 망설였어요"라고 답변하게 유도 |
| "렌탈카 여행 팁을 공유해주세요!" | 30-50개 | "주유소 찾기 팁" 댓글에 먼저 답변 |

**보상 메커니즘**:
- 상위 5개 댓글 → "추가 5% 할인코드" DM 발송
- 가장 많은 좋아요 댓글 → "할인권" 경품
- 모든 참여자 → "추가 할인" 자동 적용

**기대 참여율**: 5-10% of 페이지 팔로워

#### 4️⃣ 시즌별 이벤트 (월 1회 이상)

| 이벤트 | 진행 방식 | 기간 | 참여자 목표 |

| **댓글 이벤트** | "이 게시물을 공유 + 댓글 참여" → 추첨 (경품: 할인권) | 3일 | 200명 |

**기대 도달**: +500-1000명/이벤트
