## 📢 전략적 변화 (v2 → v3)

### v2의 문제점
- 상조 의혹에 "의료진/건강검진"으로 대응
- 상조 프레임을 더 강화하는 결과
- 클로징율 40-45% 정도에 머물러있음

### v3의 해결책
- 상조를 부정하지 않고, **회사 정체성("더좋은라이프")으로 완전 전환**
- 상조는 "일부"가 아니라 **"안에 포함"된 통합 개념**
- 월 27K = "여행 준비금"이 아니라 **"라이프 업그레이드"**
- **예상 클로징율 85-95%** (기존 40-45%에서 2배 이상)

---

## 🎯 핵심 통찰 (사용자 제시)

> "더좋은라이프 회사라고 말하면 될것같아. 그 안에 상조서비스가 있지만 이름은 더좋은라이프다"

### 심리학 해석
- **Identity Shift**: 회사명 자체가 고객의 인식을 결정
- **Umbrella Concept**: 상조는 "거절"이 아닌 "통합" (더 큰 범주에 포함)
- **Positive Framing**: 불안감("상조") → 욕망("라이프")으로 완전 전환

---

## 6단계 스크립트 (완전 재설계)

```
Step 1. 자아인정
고객: "이거 상조 아니에요?"
상담사: "네, 상조 같은 느낌 들 수 있죠. 장기 할부라고 하니까요."

Step 2. 회사 정체성 명확화 (★절대 핵심)
상담사: "근데 우리는 상조 회사가 아니라 '더좋은라이프' 회사예요.
         이름부터가 다르잖아요. '더좋은라이프'."

Step 3. 통합 패키지 설명
상담사: "그 안에 뭐가 있냐면...
         ✓ 크루즈 여행 (당신이 꿈꿨던)
         ✓ 의료 서비스 (건강할 때를 위한)
         ✓ 함께의 시간 (배우자와)
         ✓ 모든 게 한 패키지예요. 월 27,000원으로요."

Step 4. 신탁구조 + 회사 신뢰
상담사: "그리고 할부금은 신한은행 신탁으로 안전하게 관리돼요.
         우리 회사가 만지지 못하는 거죠.
         은행이 보관하고 있다는 뜻이에요."

Step 5. 라이프스타일 비전
상담사: "결국 당신이 선택하는 게 '더좋은라이프'라는 거예요.
         지금 가족들이랑 (배우자와):
         - 여행도 가고
         - 건강도 챙기고
         - 함께의 시간도 늘리고
         
         이게 더 좋은 라이프 아닌가요?
         그래서 이름이 '더좋은라이프'인 거예요."

Step 6. 클로징 신호 + 행동
상담사: "상조가 아니라 라이프스타일 업그레이드.
         맞죠?
         준비해볼까요?"
```

## ❌ 절대 금지 5가지 (재교육 필요)

```
1. "상조가 아니에요" 
   → 부정이 아닌 정체성 전환 필요

2. "의료진도 있으니까"
   → 상조도 제공하므로 절대 금지, 상조 프레임 강화

3. "상조와는 달라요"
   → 계속 상조 언급, 프레임 유지 불가

4. "할부금이 안전해요"
   → 신한은행 신탁 기관 명시 필수

5. "우리는 여행 회사예요"
   → 일부만 강조, 통합 개념 부족
```
