## 핵심 심리 설정

```
신제품 = "새로 출시" + "크루즈닷 신제품팀" + "만족도 92%" + "선발 고객 혜택"
         ↓
         Authority(크루즈닷 신뢰) + Social Proof(만족도) + Trust(테스트 검증)
         ↓
         "뭐예요?" → "정말 도움 될까?" → "지금이 최고 기회인가?"
         ↓
         선발 고객 혜택 + 한정가 + 환불 보장 조합

신제품 구매 장벽:
  ├─ 인지 부족 ("이게 뭔데?")
  ├─ 신뢰 부족 ("정말 좋을까? 사기 아니니?")
  ├─ 시간압박 부족 ("언제든 사면 되지")
  └─ 비용부담 ("새로운 제품에 돈 쓸 가치가 있나?")

돌파 순서:
  1. Authority (크루즈닷 신제품팀 = 전문성)
  2. Social Proof (만족도 92% = 검증됨)
  3. Trust (환불 보장 = 리스크 제거)
  4. FOMO (선발 고객 특가 = 지금만)
  5. Implication (안 하면 손해 = 나중에 후회)
  6. Payoff (하면 이득 = 먼저 누린 고객처럼)

⚠️ 주의:
  - "LG 협력" 등 기밀 정보는 언급 금지 (경쟁사 우려)
  - 검증 안 된 수치(시간절약, 재구매율)는 추정치로 표현
  - 첫 출시 가격은 "6개월 제한" 명시
```

## 💡 고급 팁

### Tip 1: 침묵의 마법
- 중요한 메시지 후 **1.5~2초 침묵**
- 고객이 "맞다"고 뇌에 각인되는 시간
- 서두르지 말 것

### Tip 2: 고객 언어 복제
- 고객: "시간이 부족해요"
- 상담사: "시간이 없으신 분들이..."
- → 공감과 신뢰 300% 상승

### Tip 3: 선택 강제하기 (이진 선택)
- "지금 3만 원" vs "6개월 후 4만 원대"
- 이진 선택만 제시 (3번째 옵션 없음)
- 뇌는 2가지 중에 선택할 때 가장 빠름

### Tip 4: SMS 링크의 힘
- **전화 내 결제 입력 제거** = 콜 시간 3분 단축
- SMS 링크 3단계 = 고객이 편하게 가입
- 환불 보장이 신뢰 강화

### Tip 5: 자동화가 관계를 만든다
- Day 1/7/14 메시지 = 계획된 터치포인트
- "혼자가 아니다"는 느낌 강화
- 환불율 감소 + 재구매율 증가

### Tip 6: 감정적 피크
- 9:00~12:00 마무리에서
- 고객이 가장 긍정적일 때
- "당신은 현명한 선택을 했어요" 강화
- → 재구매율 + 추천율 상승
