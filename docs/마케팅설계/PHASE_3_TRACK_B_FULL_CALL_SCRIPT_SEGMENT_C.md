# Phase 3 Track B: 25분 전체 콜 스크립트 — Segment C (중년부부)

**작성일**: 2026-05-21  
**타겟**: 중년부부 (45-55세, 자녀 독립/중고등, 황금기 의식 + 정체성 위기)  
**목표 클로징율**: 32% → 52% (+63% 개선)  
**통합 프레임워크**: Phase 2 Opening + Phase 3 Track A 이의처리 + PASONA + SPIN + MI + 모니카 4단계 + 자아 재발견  
**구조**: 8단계, 25분, 정체성 + 황금기 앵커

---

## 📋 Segment C 심리 프로필

```
심리상태: 정체성 위기 + 황금기 의식 + "이게 인생이 다야?" 허무감
신경화학: 도파민(새로운 경험 갈망) + 코르티솔(스트레스) + 세로토닌(의미 추구)
주요 장벽:
  ├─ "지금 써도 되나? 모아야 하나?" 경제불안
  ├─ "이 나이에 여행? 너무 늦지 않았나?" 자책
  ├─ 배우자와의 의견 차이 (한 명은 보수적, 한 명은 진취적)
  ├─ "뭐하다가 인생이 이렇게 지나갔지?" 후회
  └─ 건강 불안 ("이제 건강이 안 좋아지면?")

강점:
  ├─ 경제력 충분함 (주택담보→완제/자산 축적)
  ├─ 시간 여유 증가 (자녀 독립/정년 불안에 대한 각성)
  ├─ 부부 시간의 가치 새로 발견
  └─ "지금이 마지막"이라는 절박함 (시간 희소성 극대)
```
