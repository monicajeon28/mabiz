# 신혼부부(30-35세) 헬스케어 크루즈 여행 콜 스크립트 (15분) 최종 실전판

**상담사가 보면서 그대로 읽을 수 있는 실전 스크립트**  
**대상**: 신혼부부 (30-35세) / 건강·예방의료·웰빙 세그먼트  
**목표**: 신혼부부의 '함께'라는 욕망 → 건강 불안 → 의료적 신뢰 → 여행동의 (70% 클로징)  
**시간**: 15분  
**심리학**: PASONA(문제→자극→고통→해결→신청) + SPIN(상황→문제→암시→선택)

---

### [PHASE 5] 이의처리 (12:00~14:00) — 신혼부부 특화 대응 (5가지)

**고객:**
```
"음... 솔직히 월 28,000원이 좀 부담되는데요..."
```

**상담사:**
```
"아, 정확히 알겠습니다.

하지만 한 가지 다시 생각해보면 좋을 것 같은데요.

당신이 '월 28,000원'이라고 생각하는 게 맞아요.
근데 실제로 뭔가요?

1. 건강검진 (보통 30-50만원): 포함
2. 의사 상담 (1회 10만원): 무제한
3. 웰빙 프로그램 (1일 가치): 5만원
4. 가족 여행 (1박 가치): 30만원
5. 배우자와 함께 시간 (가치무한)

이걸 다 합치면...
실제로는 월 150만원 이상의 가치예요.

그런데 당신은 월 28,000원만 내는 거거든요.

오히려... '이 정도 투자로 이 정도 가치를 얻을 수 있나?'
하는 게 정상 반응이에요.

당신도 그렇게 생각하지 않으세요?"
```

[침묵 2초]

#### [대응 2] 의료 신뢰 거절 — "크루즈의 의료진이 정말 자격이 있나요?"

**고객:**
```
"근데 크루즈 의료진이 정말 자격이 있나요?
일반 병원처럼 믿을 수 있어요?"
```

**상담사:**
```
"정말 현명한 질문이에요.

우리 헬스케어 크루즈의 의료진은:

1. **자격 기준**
   - 의사: 미국/유럽 의학박사 (MD/DO)
   - 간호사: 국제 정간호사 자격 (RN)
   - 영양사: 임상영양사 자격 (RD/LD)

2. **인증 시스템**
   - ISO 13485 (의료기기 관리 국제표준)
   - JCAHO (미국 의료기관 인증)
   - 한국 해양안전청 인증

3. **안전 시스템**
   - 선박 내 응급실 (24시간 운영)
   - 해안에서 30마일 이내 항행 (1시간 내 병원 도착)
   - 응급 헬기 이송 시스템

4. **당신의 데이터 보호**
   - 의사-환자 비밀유지권 (의료법 준수)
   - 건강정보는 크루즈 종료 후 안전 폐기
   - GDPR 개인정보보호법 준수

육지 병원과 거의 동일한 수준이에요.

오히려 스트레스 없는 환경이라
회복이 더 빨라요.

당신은 충분히 안심하셔도 됩니다.

우리가 당신과 배우자의 건강을
책임진다는 뜻이니까요."
```

[침묵 2초]

#### [대응 4] 정보 보안 거절 — "개인 건강정보가 누출되지 않나요?"

**고객:**
```
"개인 건강정보가 새어 나가면 어쩌죠?
직장에 들통 나면..."
```

**상담사:**
```
"아, 정말 중요하고 현명한 우려예요.

우리가 당신의 건강정보를 보호하는 방식:

1. **의료 법적 보호**
   - 의사-환자 비밀유지권 (의료법 적용)
   - 당신의 동의 없이 제3자 공개 불가
   - 위반 시 법적 처벌 대상

2. **데이터 관리**
   - 암호화된 데이터베이스 (군사급 보안)
   - 크루즈 종료 후 안전 폐기
   - 개인 접근은 의료진만 허용

3. **당신의 통제**
   - 어떤 정보 공유할지 당신이 선택
   - 배우자와도 '공유 범위' 조절 가능
   - 언제든 데이터 삭제 요청 가능

4. **국제 표준**
   - GDPR (유럽연합 개인정보보호)
   - HIPAA (미국 의료정보 보호법)
   - 한국 개인정보보호법 준수

당신은 완벽하게 보호받습니다.

우리는 당신의 건강정보를 '신성'하게 대합니다.

직장이나 가족에 알려질 일은 절대 없어요.

당신의 프라이버시를 존중한다는 뜻입니다."
```

## 🎯 신혼부부 세그먼트 핵심 심리

### PASONA 매핑
- **P (Problem)**: 바쁜 신혼생활, 건강 체크 못함
- **A (Agitate)**: 배우자와 시간 줄어들 것에 대한 불안
- **S (Solution)**: 헬스케어 크루즈 (의료 + 여행 + 함께)
- **O (Offer)**: 월 28,000원 / 6개월
- **N (Narrow/Now)**: 오늘 신청 (신혼시절은 지금!)
- **A (Action)**: Day 0/1/3 약속

### SPIN 기법 적용
- **Situation**: "신혼생활이 어떤가요?" → 공감
- **Problem**: "건강은? 배우자와의 시간은?" → 발굴
- **Implication**: "5년 뒤는 어떻게 될까요?" → 손실회피
- **Payoff**: "헬스케어 크루즈면 해결되지 않을까요?" → 동의 유도

### Loss Aversion (손실회피)
- "지금 안 하면" → 건강 체크 영구 미루기
- "신혼시절은 다시 안 온다" → 감정적 공감
- "배우자와의 연결이 약해진다" → 관계 불안감

### Mental Accounting (정신 회계)
- 월 28,000원 = 하루 933원 = 커피보다 싼 가격
- 비용이 아니라 "투자" 재프레이밍
- "건강 + 여행 + 함께" 삼중 가치 강조

- 의사·간호사·영양사 언급 (전문가 신뢰)

### 📊 통계 분석 프레임

#### **1. 클로징율 비교**

```
A안 최종 클로징율 = (A안 클로징 수 / A안 총 콜 수) × 100
B안 최종 클로징율 = (B안 클로징 수 / B안 총 콜 수) × 100

신뢰도 검증:
- 표본 오류: √(P(1-P)/N) < 5% 기준
- 유의도: Chi-Square 검정 (p < 0.05)
```

**기대 결과**:
