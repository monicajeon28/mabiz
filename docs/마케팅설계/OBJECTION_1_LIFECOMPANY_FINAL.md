## 🎯 핵심 인사이트 (사용자 요청 기반)

> "더좋은라이프 회사라고 말하면 될것같아. 그 안에 상조서비스가 있지만 이름은 더좋은라이프다"

### 심리학 원리
- **Identity Shift**: 회사명 = 정체성 (상조 ❌ → 라이프 ✅)
- **Umbrella Concept**: 상조는 일부가 아닌 "안에 포함" (더 큰 범주)
- **Frame Reframing**: 부정이 아닌 "통합" (상조+의료+여행+함께의시간)

## 6단계 최종 스크립트 (완전 재설계)

### **Step 1. 자아인정** (공감)
```
고객:    "이거 상조 아니에요?"
상담사:  "네, 상조 같은 느낌 들 수 있죠.
          장기 할부라고 하니까요."
```

**심리학**: Validation
**목표**: 고객 불안을 "정상적 의심"으로 정상화
**핵심**: 부정하지 않고 인정

---

### **Step 2. 회사 정체성 명확화** (프레임 전환 — 절대 핵심)
```
상담사:  "근데 우리는 상조 회사가 아니라
         '더좋은라이프' 회사예요.
         
         이름부터가 다르잖아요.
         '더좋은라이프'."
```

**심리학**: Identity Shift + Complete Frame Reframing
**목표**: 상조 개념에서 완전 벗어나기
**핵심 메시지**: 회사명 자체가 정체성 (상조 vs 라이프)

---

### **Step 3. 통합 패키지 설명** (욕망 + 안전성 = 라이프의 완성)
```
상담사:  "그 안에 뭐가 있냐면...
         
         ✓ 크루즈 여행 (당신이 꿈꿨던)
         ✓ 의료 서비스 (건강할 때를 위한)
         ✓ 함께의 시간 (배우자와)
         ✓ 모든 게 한 패키지예요.
         
         월 27,000원으로요."
```

**심리학**: Umbrella Concept + Narrative Transportation
**목표**: 상조는 "일부"가 아닌 "안에 포함"으로 인식 전환
**핵심**: 월 27K = "라이프 전체의 확장"

---

### **Step 4. 신탁구조 + 회사 신뢰** (안전성의 신화)
```
상담사:  "그리고 할부금은 신한은행 신탁으로
         안전하게 관리돼요.
         
         우리 회사가 만지지 못하는 거죠.
         은행이 보관하고 있다는 뜻이에요."
```

**심리학**: Reassurance + Authority + Third-Party Trust
**목표**: 상조 사기 의심 완전 차단
**핵심**: 은행 신용 ≠ 회사 신용 (상조와의 근본적 차이)

---

### **Step 5. 라이프스타일 비전** (투자, 아니 선택)
```
상담사:  "결국 당신이 선택하는 게
         '더좋은라이프'라는 거예요.
         
         지금 가족들이랑 (배우자와):
         - 여행도 가고
         - 건강도 챙기고
         - 함께의 시간도 늘리고
         
         이게 더 좋은 라이프 아닌가요?
         그래서 이름이 '더좋은라이프'인 거예요."
```

**심리학**: Self-Generated Content + Value Repositioning
**목표**: 고객이 스스로 "이건 선택이다"로 결론
**핵심**: 라이프 = 투자 (상조 = 불안감과의 완전 차별화)

---

### **Step 6. 클로징 신호 + 행동** (결정)
```
상담사:  "상조가 아니라 라이프스타일 업그레이드.
         
         맞죠?
         준비해볼까요?"
```

**심리학**: Commitment + Action Trigger
**목표**: 확신 있는 행동 유도
**핵심**: "준비" ≠ "계약" (심리적 저항 최소화)

## 추가 심리학 해제

### 왜 "더좋은라이프"인가?
1. **회사명 = 약속**: "더좋은라이프를 만드는 회사"라는 명확한 정체성
2. **포괄성**: 상조/여행/의료/함께의 시간 모두를 통합하는 개념
3. **긍정 감정**: "더좋은" = 상향 추구 (vs 상조 = 하향 방어)
4. **장기 관계**: 라이프는 일회성이 아닌 지속적 관계

### 왜 "그 안에"인가?
- **상조를 부정하지 않음**: 고객의 의심을 인정
- **더 큰 범주로 포함**: 상조는 "일부"가 아닌 "안에 포함"
- **통합 메시지**: "여행만 아니라 의료도, 의료만 아니라 여행도"
- **의혹 해소**: 상조의 위험(자금 유용)을 은행 신탁으로 차단
