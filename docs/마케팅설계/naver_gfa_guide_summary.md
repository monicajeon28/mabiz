# 네이버 성과형 광고 + GFA 플랫폼 가이드 (크루즈닷 최적화)

## 1. 구글/네이버 성과형 광고 기본 설정

### 네이버 성과형 광고 (Performance DA)
- **실시간 입찰 자동화**: AI 기반 자동 입찰으로 ROI 극대화
- **타겟팅**: 검색 키워드 + 웹사이트 방문자 + 맞춤 세그먼트
- **광고 형식**: 검색 광고, 디스플레이(배너), 동영상(유튜브 채널), 네이티브 광고
- **예산 배분**: 일일 예산 설정 → 월 자동 최적화 분배

### 구글 성과형 광고 (Performance Max)
- **멀티채널**: 검색/디스플레이/유튜브/쇼핑/지메일 동시 배포
- **자동 크리에이티브**: 이미지 + 헤드라인 조합 자동 생성
- **전환 추적**: GA4 연동으로 고객 전체 여정 추적
- **예산**: 주당/월간 기반 자동 배분

---

## 2. GFA (Google For Ads) 플랫폼 활용

### GFA 핵심 기능
1. **AI 기반 캠페인 최적화**
   - 자동 입찰 전략 추천
   - 성과 예측 모델링
   - 예산 배분 최적화

2. **실시간 대시보드**
   - 일자별/채널별 ROI 추적
   - 전환 경로 분석
   - 광고 소비 vs 매출 비교

3. **세그먼트별 성과 분석**
   - 신규고객 vs 재구매고객
   - 기기별(모바일/데스크톱)
   - 지역별 성과

4. **자동화된 보고서**
   - 주간 성과 리포트
   - 이상 감지 알림
   - 개선안 추천

## 5. 효율적인 광고 예산 배분 전략

### 월 예산 배분 모델 (월 1,500만원 기준)

```
1. 크루즈 골드 (고객생명가치 높음)
   - 예산: 60% (900만원)
   - 채널: Google 성과형(70%) + 네이버 DA(30%)
   - 목표 CPA: 80,000원 → 월 120명 확보

2. 크루즈 일반 (부피 확대)
   - 예산: 25% (375만원)
   - 채널: 네이버 DA(60%) + 구글 검색(40%)
   - 목표 CPA: 50,000원 → 월 75명 확보

3. 렌탈/헬스케어 (보조상품)
   - 예산: 15% (225만원)
   - 채널: 네이버 DA(80%)
   - 목표 CPA: 25,000원 → 월 90명 확보
```

### 일일 예산 배분 알고리즘
- **주중(평일)**: 60% 배분 (검색 수요 높음)
- **주말**: 40% 배분 (비교검색 단계)
- **프로모션 기간**: 즉시 +30% 추가 예산

### 최적화 규칙
1. **CPA 상한선 초과** → 입찰 -15% 조정
2. **ROAS 3.0배 미만** → 크리에이티브 A/B 테스트
3. **전환율 하락** → 랜딩페이지 UX 개선
4. **월중 누적 ROAS > 3.5배** → 예산 +10% 확대
