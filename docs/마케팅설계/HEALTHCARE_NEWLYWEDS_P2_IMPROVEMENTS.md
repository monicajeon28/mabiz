## 🎯 P2-1: 상담사 접근성 — 톤&매너 가이드 추가

### 현재 문제
- 스크립트 읽기는 쉽지만 **어떤 톤으로 읽을지** 미정의
- 상담사가 감정 뉘앙스 해석 필요
- 역할 경험 부족한 상담사는 기계적으로 읽을 가능성

### [PHASE 1] 톤&매너 가이드

```markdown
### [PHASE 1] 오프닝 (0:00~2:00) — 신혼 축하 + 웰빙 동감

**🎤 톤 (Voice Tone)**
- 에너지 레벨: 60-70% (밝지만 지나치게 흥분하지 않음)
- 속도: 천천히 (1문장당 2-3초)
- 높이: 자연스러운 음역대 (높지도 낮지도 않음)
- 포인트: "축하한다"는 감정을 보여주기

**예시 음성 표현**:
```
"안녕하세요! 더좋은라이프 고객 상담팀의 [이름]입니다."
↓
밝고 따뜻하게, 공식적이지만 차갑지 않음
(컨설턴트라기보다 "친구의 친구"처럼)

"축하드립니다!" ↓
진심 어린 목소리, 잠깐 pause
(고객이 "고맙습니다"라고 느낄 정도)
```

**😊 매너 (Manner)**
- 자세: 편안하지만 전문적
- 미소: 전화 너머 들려야 함 (microphone로 잡힘)
- 눈맞춤: 고객 카드 또는 사진을 보며 말하기
- 리듬: 문장 끝에서 아주 약간의 pause

**❌ 피해야 할 것**
- 너무 빠른 속도 (고객이 따라가 못함)
- 지나친 enthusiasm (비즈니스같이 들림)
- 무감정한 읽기 (로봇 같음)
- 강한 사투리 (친근하지만 거리감 생김)

**✅ 체크리스트**
□ 고객이 "네"라고 자연스럽게 응답했는가?
□ 고객이 웃음 또는 따뜻한 톤으로 답했는가?
□ "축하" 감정이 고객에게 전달되었는가?

**💡 팁: 전화 거는 전에**
- 거울을 보며 미소 짓기 (목소리에 반영됨)
- 깊게 숨 쉬기 (안정감 전달)
- "이 사람 신혼 축하해야지"라고 마음먹기
```

### [PHASE 2] 톤&매너 가이드

```markdown
### [PHASE 2] 욕망 발굴 (2:00~5:00) — 신혼부부만의 '함께' 경험

**🎤 톤 (Voice Tone)**
- 에너지 레벨: 50-60% (낮춰서 고객 말을 잘 듣게)
- 속도: 더 느리게 (질문마다 pause)
- 높이: 약간 낮게 (고객에게 집중하는 느낌)
- 포인트: "당신의 진정한 욕망을 알고 싶다"는 태도

**예시 음성 표현**:
```
"그래서 제가 여쭤보고 싶은 게...

당신들이 정말 꿈꾸던 신혼 경험이 뭐였어요?"

↓
절반은 질문, 절반은 "내가 진짜 관심 있다"는 표현
(마지막 부분을 약간 내려서 읽기)
```

**😊 매너 (Manner)**
- 자세: 약간 앞으로 숙인 느낌 (경청의 자세)
- 침묵: 고객 답변 후 1-2초 더 기다리기
- 리액션: 고객 말 중간에 "맞아요", "그렇군요" 끼워넣기
- 눈맞춤: 고객이 말할 때는 고객 이름이나 사진을 봤다고 상상하기

**❌ 피해야 할 것**
- 고객 말이 끝나기 전에 끼어들기
- 너무 빨리 다음 질문하기 (고객이 더 말할 여지 없음)
- 자기 이야기 하기 (이 phase는 고객 about)
- 무감정한 질문 (로봇 같음)

**✅ 체크리스트**
□ 고객이 3문장 이상으로 답했는가? (단답X)
□ 고객이 자신의 진정한 욕망을 표현했는가?
□ 당신은 최소 3번 이상 "그렇군요", "맞네요" 이런 말 했는가?
□ 침묵 시간이 3초 이상 있었는가?

**💡 팁: 고객 단답형 답변 시**
- "그렇군요. 그런데 조금 더 구체적으로...?"
- 고객이 무언가 더 말할 여지를 남기기
```

### [PHASE 3] 톤&매너 가이드

```markdown
### [PHASE 3] 고통 증폭 (5:00~9:00) — 건강 + 관계의 손실회피

**🎤 톤 (Voice Tone)**
- 에너지 레벨: 40-50% (차분하고 깊게)
- 속도: 매우 천천히 (각 문장이 "심각하다"는 뉘앙스)
- 높이: 음역대 낮게 (진지함)
- 포인트: "이건 정말 중요한 얘기"라는 톤

**예시 음성 표현**:
```
"한 번 생각해보세요.

지금처럼 '나중에, 나중에' 미루다 보면..."

↓
마지막 "미루다 보면"에서 톤을 내려서 읽기
(슬프거나 무겁게, 하지만 극적이지 않게)
```

**😊 매너 (Manner)**
- 자세: 차분한 느낌 (흥분하지 않음)
- 침묵: 길게 (3-4초) 고객이 생각하게
- 리액션: 고객 말에 "음..." "맞죠..." 공감 표현
- 눈맞춤: 고객 이름이나 가족 사진을 상상하며 말하기

**❌ 피해야 할 것**
- 너무 무섭게 ("협박"처럼 들림)
- 지나친 감정 (연기하는 것 같음)
- 자극적인 표현 ("죽어간다" 같은)
- 빠른 속도 (긴급해 보임)

**✅ 체크리스트**
□ 고객이 "그럴 수도 있겠네..."라고 수긍했는가?
□ 고객의 목소리 톤이 더 차분해졌는가?
□ 당신이 너무 극적이지 않으면서도 진지한가?

**💡 팁: 고통 증폭의 정도 조절**
- 고객이 이미 불안해 보이면 (목소리 떨림) → 톤을 75% 수준으로 유지
- 고객이 무관심해 보이면 → 톤을 더 깊게 (50% → 30%)
```

### [PHASE 4] 톤&매너 가이드

```markdown
### [PHASE 4] 솔루션 (9:00~12:00) — 헬스케어 크루즈 + 의료적 신뢰

**🎤 톤 (Voice Tone)**
- 에너지 레벨: 60-70% (다시 밝혀짐)
- 속도: 정상 속도로 복구 (차분하지만 지루하지 않음)
- 높이: 자연스러운 음역대
- 포인트: "이건 좋은 소식"이라는 톤

**예시 음성 표현**:
```
"그래서 우리가 소개하는 게 '헬스케어 크루즈'인데요.

이건 일반 여행이 아니라...

'의료 전문가가 함께하는 특별한 경험'이에요."

↓
"특별한 경험"을 조금 강조해서 읽기
(희망감이 들어야 함)
```

**😊 매너 (Manner)**
- 자세: 약간 활기 있게 (솔루션 제시)
- 미소: 다시 들어갈 수 있음 (희망의 뉘앙스)
- 침묵: 적당하게 (2초)
- 리액션: 고객이 질문하면 반갑게 "좋은 질문이에요!"

**❌ 피해야 할 것**
- 너무 세일즈스러움 (신뢰 손상)
- 무분별한 낙관 (현실성 잃음)
- 빠른 속도 (고객이 따라가 못함)

**✅ 체크리스트**
□ 고객이 "오, 그런 게 있네?"라고 반응했는가?
□ 당신의 톤이 "판매"가 아니라 "소개"처럼 들렸는가?

**💡 팁: 의료진 설명 시**
- "의사, 간호사, 영양사"를 말할 때 조금 더 느리게
- 신뢰감을 주기 위해 각 직군을 명확히 구분해서 읽기
```

### [PHASE 5] 톤&매너 가이드

```markdown
### [PHASE 5] 이의처리 (12:00~14:00) — 신혼부부 특화 대응

**🎤 톤 (Voice Tone)**
- 에너지 레벨: 50-60% (고객 이의에 따라 조절)
- 속도: 정상에서 약간 느리게
- 높이: 자연스럽되 신뢰감 있게
- 포인트: "당신 말이 맞아, 그런데..." 톤

**예시 음성 표현** (비용 이의):
```
"아, 정확해요.

특히 신혼생활이 시작되면서 돈이 많이 들어가는 시기죠.

당신의 그 마음, 충분히 이해합니다."

↓
"이해합니다"를 진심 어린 톤으로
(고객이 "이 사람이 날 이해하는구나"라고 느껴야 함)
```

**😊 매너 (Manner)**
- 자세: 고객과 같은 편이라는 느낌
- 침묵: 고객 말 끝나고 1초 기다리기
- 리액션: "그렇군요", "정확해요", "맞아요" 자주 사용
- 눈맞춤: 고객이 중요하다는 태도

**❌ 피해야 할 것**
- "그게 문제 아니에요"라고 하지 않기 (고객 감정 무시)
- 너무 빨리 대응책 제시 (고객 불안 증폭)
- 방어적인 톤 (싸움 같이 들림)

**✅ 체크리스트**
□ 고객이 당신을 "같은 편"이라고 느꼈는가?
□ 고객의 거절이 "약해졌는가" (재거절이 적은가)?

**💡 팁: 이의 종류별 톤 조절**
- 비용 이의: 공감 강화 (고객이 정말 걱정함)
- 배우자 이의: 희망적 톤 (배우자도 좋아할 거라고 암시)
- 의료 이의: 전문적 톤 (신뢰감 강화)
```

### [PHASE 6] 톤&매너 가이드

```markdown
### [PHASE 6] 클로징 (14:00~15:00) — 삼중 선택 + 신청

**🎤 톤 (Voice Tone)**
- 에너지 레벨: 70-80% (절정)
- 속도: 정상 속도 (명확함)
- 높이: 자신감 있게
- 포인트: "당신이 올바른 선택을 하려고 한다"는 톤

```
"그럼 지금 당신은 3가지 선택지가 있어요."

↓
약간 위로 올리면서 읽기 (중요한 순간임을 알림)

"당신이라면 어떤 선택을 하시겠어요?"

↓
고객에게 주도권을 주는 톤
(명령이 아니라 진정한 질문)
```

**😊 매너 (Manner)**
- 자세: 전문적이고 자신감 있음
- 미소: 고객을 축하하려는 느낌
- 침묵: 길게 (2-3초) 고객에게 선택권 주기
- 리액션: 고객이 "신청하겠다"고 하면 진심 어린 "잘하신 결정이에요!"

**❌ 피해야 할 것**
- 3번 선택지를 당신이 강요하기 (고객 선택처럼)
- 너무 성급하게 마무리 (고객 확신 부족)
- 지루한 톤 (중요한 순간 살리지 못함)

**✅ 체크리스트**
□ 고객이 "신청하겠다"고 명확히 말했는가?
□ 당신의 톤이 고객에게 "당신 선택을 축하한다"는 느낌을 줬는가?

**💡 팁: 클로징 직전 깊게 숨 쉬기**
- 내용에 집중하되 목소리는 자신감 있게
- 마지막 "고마워요, 행운을 빕니다"는 다시 따뜻하게
```

## 📋 P2-2: 고객 시나리오 확대 — 무응답/단답 대응

### 현재 문제
- 모든 고객이 "happy path" 답변을 하지 않음
- 고객 무응답 (침묵 5초+), 단답형 ("네", "아뇨") 시 대응책 미흡
- 상담사가 즉흥적으로 대응하면 멘탈붕괴 또는 클로징 실패

### 개선: 분기별 대응 시나리오

---

```
상담사: "당신들이 정말 꿈꾸던 신혼 경험이 뭐였어요?"

[침묵 3초]

[침묵 5초]

고객: "음... 잘 모르겠는데요."
```

1. 고객이 너무 바쁘거나 피곤해서 집중력 부족
2. 질문이 너무 추상적
3. 고객이 진정한 욕망을 의식하지 못함

### 테스트 목표
현재 15분 스크립트 대비 P0-1 (이의처리 5개) 개선이 실제 클로징율 향상에 기여하는가?

### 실험 그룹

**A안 (기존)**: 이의처리 2개만
- 의료안전성 처리
- 배우자동의 처리

**B안 (개선)**: 이의처리 5개 완전판
- 의료안전성 처리 ✅
- 배우자동의 처리 ✅
- 비용 거절 처리 ✅ (신규)

- 시간 거절 처리 ✅ (신규)

### 측정 지표

| 지표 | A안 (기존) | B안 (개선) | 기대 개선 |
|------|----------|---------|---------|
| 클로징율 | 60-65% | 75-80% | +15-20% |
| 평균 통화시간 | 14:30 | 15:00 | +30초 |
| 고객 이의 발생율 | 45% | 45% | (동일) |
| 이의 해결율 | 55% | 85% | +30% |
| 고객 만족도 (NPS) | 7.2 | 8.5 | +1.3 |
| 재상담 요청율 | 8% | 3% | -5% |

### 테스트 실행 방법

1. **상담사 교육 (2026-05-27)**
   - A안 상담사 5명 (기존 유지)
   - B안 상담사 5명 (신규 이의처리 5개 학습)

2. **콜 배분**
   - 무작위 배분 (고객이 모를 것)
   - CRM에서 자동 배분 ("test_group_a" vs "test_group_b")

3. **데이터 수집**
   - 모든 통화 녹음 (감정 분석용)
   - Phase별 자동 로깅
   - 고객 만족도 설문 (통화 후 SMS)

4. **주간 분석**
   - 월요일 8AM: 주간 데이터 수집
   - 화요일: 이슈 분석 및 피드백
   - 수요일: 상담사 코칭 (미흡한 부분)

- 샘플: 각 그룹 50콜 (통계적으로 유의미한 샘플)
- 신뢰도: 95% (p-value < 0.05)
- 효과크기: 최소 15% 향상 필요

### 예상 결과

**성공 시나리오** (확률 70%):
- B안이 75% 이상 클로징율 달성
- 이의 해결율 85% 이상
- 전사 B안 도입 결정

**실패 시나리오** (확률 20%):
- B안도 A안 수준 (65% 미만)
- 이의처리 5개 중 일부만 효과 있음
- 효과 있는 이의처리만 추가

**재평가 시나리오** (확률 10%):
- 클로징율은 향상 (70%)
- 통화시간이 20분+ 초과
- 시간 최적화 필요
