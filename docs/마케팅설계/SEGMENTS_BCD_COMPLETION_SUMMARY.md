## 🎯 각 세그먼트별 핵심 특징

### Segment A: 신혼부부
- **타이밍**: 결혼 후 3년 이내, 첫 아이 전
- **심리**: 로맨스 + 미래 설계
- **감정 앵커**: "인생의 시간에 투자" + "신혼의 추억"
- **클로징율**: 40-58%
- **기대 전환율**: 신청 후 Day 3 55-65%

### Segment B: 40대 가족 ⭐ **신규**
- **타이밍**: 아이 10-15세, 황금기 부모-자녀 관계
- **심리**: 추억 남기기 + 타이밍 긴박 (1년 뒤 반항기)
- **감정 앵커**: 
  1. "아이의 영원한 기억" (Phase 2-3)
  2. "지금이 기회의 황금시간" (Phase 3-4)
  3. "5년 뒤 후회" (Phase 5-6)
- **세그먼트 특화 전략**:
  - Phase 3에서 "아이가 20대 되면 못 온다" SPIN I
  - Phase 4에서 교육비 재배치 (250K → 185K + 66K 여행)
  - Phase 5에서 배우자 설득 리더십 강화
- **클로징율**: 60-70%
- **기대 전환율**: 신청 후 Day 3 최종 60%+

### Segment C: 중년부부 ⭐ **신규**
- **타이밍**: 자녀 독립, 부부만의 시간 재발견
- **심리**: 황금기 (40-60세 전) + 배우자와의 재결속
- **감정 앵커**:
  1. "황금기는 지금" (Phase 2-3) — 60대 신체 변화
  2. "자기투영" (Phase 3-4) — "우리 부모님도 했으면..."
  3. "배우자 재결속" (Phase 4-5) — "신혼 때 그 느낌 다시"
- **세그먼트 특화 전략**:
  - Phase 3에서 나이 타이밍 강조 (60대 되기 전 활동 능력)
  - Phase 4에서 부부 상담 이론 (신혼 후 20년 위기) + 건강+여유시간
  - Phase 5에서 자녀 걱정 저항 분기 추가 (Intergenerational 선순환)
- **클로징율**: 65-75%
- **기대 전환율**: 신청 후 Day 3 최종 70%+

### Segment D: 시니어 ⭐ **신규**
- **타이밍**: 은퇴 전후, 건강의식 고조
- **심리**: 현실적 긴박 + 건강 의존 + 함께의 소중함
- **감정 앵커**:
  1. "함께의 소중함" (Phase 2-3) — 혼자가 아닌 배우자/친구
  2. "건강이 허락할 때" (Phase 3-4) — 의료적 긴박성
  3. "마지막 기회" (Phase 5-6) — 직설적이지만 윤리적
- **세그먼트 특화 전략**:
  - Phase 3에서 "건강이 허락할 때" 의료적 근거 (65-70 황금기)
  - Phase 4에서 "24시간 배 위 의료진" 강조 (심장/당뇨/고혈압 대응)
  - Phase 5에서 "건강 불안" vs "배우자 추억" 균형
  - **침묵 4회**: 노년층 생각 시간 충분히 제공
- **클로징율**: 70-80% (신중하지만 결정 빠름)
- **기대 전환율**: 신청 후 Day 3 최종 75%+
