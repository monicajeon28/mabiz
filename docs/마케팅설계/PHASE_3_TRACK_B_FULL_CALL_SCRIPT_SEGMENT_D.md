## 📋 Segment D 심리 프로필

```
심리상태: 은퇴 정체성 상실 + 손주 사랑 + "마지막 경험" 의식
신경화학: 옥시토신(가족) + 높은 코르티솔(건강불안) + 도파민(새로운 경험)
주요 장벽:
  ├─ "이 나이에 여행을? 너무 위험하지 않나?" 건강 불안
  ├─ "쉬고만 싶어" 정체성 상실로 인한 도피욕구
  ├─ 손주 동반의 실질적 어려움 (나이 차이)
  ├─ "아직 은퇴 안 했는데 쓸 수 있나?" 경제 불안
  └─ "이게 마지막이 될까?" 깊은 존재 불안

강점:
  ├─ 절대적 시간 여유 (더 이상 일 없음)
  ├─ 손주 사랑 (삶의 의미)
  ├─ 부부 유대감 강함 (함께 오래)
  ├─ 인생 경험이 풍부 (판단력 좋음)
  └─ "이게 마지막"이라는 절실함 (높은 전환율)
```
