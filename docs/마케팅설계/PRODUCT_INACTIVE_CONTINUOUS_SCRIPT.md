# 기존 고객 재활성화 콜 스크립트 (15분) 최종 실전판

**상담사가 보면서 그대로 읽을 수 있는 실전 스크립트**  
**대상**: 크루즈/렌탈/호텔 기존 경험 고객 (product_inactive_db)  
**목표**: 기존 경험 회상 → V2 업그레이드 가치 → 즉시 신청 (68% 이상 클로징)  
**시간**: 15분  
**핵심**: "오래간만이에요" + "V1은 좋았는데 V2는 더 나음" + "기존 고객 전용"

## 📋 상담사 체크리스트

□ Phase 1: "오래간만이에요" + 기존 경험 회상

□ Phase 5: 비용을 "지난번과 비슷 또는 저렴"으로 프레이밍

## 🎯 핵심 차별점 (부재중 고객과의 차이)

| 항목 | 부재중 고객 | 기존 고객 |
|------|-----------|---------|
| 신뢰 기반 | 0에서 시작 | 이미 구축됨 |
| 욕망 | 상기 필요 | 이미 존재 |
| 불안 | 새로운 경험 | 이미 경험했으니까 낮음 |
| 설득 포인트 | 첫 기회 | 업그레이드 + 기존 고객 할인 |
| 비교 기준 | 타사와 비교 | 지난번 경험과 비교 |
