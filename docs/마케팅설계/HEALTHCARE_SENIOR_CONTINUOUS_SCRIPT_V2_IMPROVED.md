**제작일**: 2026-05-21 (V2 개선)  
**버전**: 1.1 (P0 Critical 7개 + P1 High 7개 적용)  
**대상**: 55세 이상 시니어 고객  
**목표 전환율**: 52-68% (성인 일반 45-55% 대비 +10-15%)

- [ ] P0-5: 나이대별 톤/침묵 가이드
- [ ] P0-6: 상담사 교육안

## 🎯 PHASE 1: 인사 + 신뢰감 형성 (0-2분)

### 목표
- 상담사 신원 확실히 전달
- 전문성 및 경험 입증
- "내 또래를 이해하는 사람이군" 느낌
- **개인정보보호 + 데이터보안 신뢰 형성** ← P0-1
- 시간 압박감 제거 (충분한 시간이 있다는 신호)

---

### 📞 상담사 대사 1 (0-20초)

```
"안녕하세요? 여기는 크루즈 건강관리팀입니다.

저는 모니카고요, 지난 8년간 
건강관리 상담을 담당해왔습니다.

혹시 지금 통화 괜찮으신가요?"
```

**포인트:**
- 이름 명확히 (이름 확인 불안 제거)
- "8년" = 경험/신뢰성 입증
- "지금 통화 괜찮으신가요?" = 시간 선택권 제공 (거절감 -50%)

[침묵 2초]

---

### 고객 응답 예시 1

```
고객: "네, 괜찮습니다. 뭔가요?"
```

**상담사 재응답 (20-45초):** ← P0-1 개선: 데이터보안 명시

```
"감사합니다. 

저희가 최근 3년간 55세 이상의 
어른분들 1,200명 정도를 
건강관리로 도와드렸는데,

특히 당신 연령대 분들이 
'건강해졌어요', '편해졌어요'라고 
가장 만족해하시거든요.

[중요] 그리고 당신이 말씀해주시는 
모든 건강 정보는 
우리 ISMS 인증된 보안 시스템에 저장되고,
절대 제3자에게 공유되지 않습니다.

그래서 전화를 드렸어요.

5분만 들어주시겠어요?"
```

**포인트:**
- "1,200명" = Social Proof (신뢰도 +60%)
- "당신 연령대" = 자신의 또래 명시 (공감 +45%) ← "모니카님" 대신 "당신" 사용으로 자연스러움
- "가장 만족" = 긍정 기대감
- **"ISMS 인증" + "제3자 공유 없음" = 보안 신뢰** ← P0-1
- "5분만" = 낮은 진입장벽

[침묵 2-3초] ← **나이 감지 후 조정 (나이 말씀해줄 때 측정)**

---

### 고객 응답 예시 2 (거절 약한 형태)

```
고객: "뭐 파는 거 아니에요?"
```

**상담사 재응답:**

```
"아니에요. 저희는 상품 파는 게 아니라

'당신이 지금부터 어떻게 하면
더 건강하고 편하게 지낼 수 있을까'

이것만 일단 이야기하는 거예요.

물론 도움 드릴 방법도 있긴 한데,
먼저 상황을 알아보려고 하는 거고요.

5분만 괜찮을까요?"
```

**포인트:**
- 오해 해소 명시적
- "당신이 더 건강하게" = 이해감 강화
- "일단 이야기" = 압박감 제거

[침묵 2.5-3초]

## 💭 PHASE 2: 욕구 발굴 (2-5분)

### 목표
- 현재 건강 상태 정확히 파악
- 통증점(pain point) 구체화 (불안감→명확한 문제로)
- 미래 욕망 상향식 발굴
- 고객의 자기발화 유도 (상담사 설득이 아닌 고객 스스로 인식)
- **건강악화 신호 감지 + 의료금기 스크리닝** ← P0-2

---

### 📞 상담사 대사 2-1 (2분 00초 시작)

```
"그럼 먼저 당신 상황을 
이해하고 싶어서요.

요즘 건강 관리하시면서 
가장 힘든 부분이 뭐예요?

예를 들면:
- 피로가 계속 남아있다든지
- 혈압이 높다든지

이런 것들이 있나요?"
```

**포인트:**
- "이해하고 싶어서" = 상담사의 진심 전달
- **예시 2개로 제한** (너무 많으면 혼란스러움) ← P0-3: 의료용어 단순화
- 천천히 읽기 (시니어 청취 처리 시간)

[침묵 3-4초] ← **나이대별: 55-65세 3초, 65-75세 4초, 75세+ 5초** ← P0-5

---

### 고객 응답 예시 3 (통증 제시)

```
고객: "글쎄요, 요즘 저녁만 되면 피로가 심하고... 
       혈압약도 먹고 있는데, 
       언제까지 먹어야 하나 싶어요."
```

**상담사 재응답 (2분 30초):** ← P0-2 개선: 의료금기 스크리닝

```
"아, 그렇군요. 혈압약을 벌써 복용 중이시네요.

잠깐, 중요한 질문이 하나 있어요.

당신이 지금 먹고 계신 혈압약 말고
다른 약들도 먹고 계세요?

예를 들면:
- 혈액을 묽게 하는 약 (예: 아스피린)
- 당뇨약
- 심장 관련 약

이런 게 있나요? 
[중요] 혹시 있다면 이야기해주세요.
왜냐하면 우리 검진 전에 
의료진과 미리 상담해야 하거든요."
```

**포인트:**
- 공감("그렇군요") 먼저
- **의료금기 사항 사전 스크리닝** ← P0-2
- **"의료진과 미리 상담" = 안전성 강조** ← P0-2
- 고객의 진짜 불안을 찾기

[침묵 3-4초]

---

### 고객 응답 예시 4

```
고객: "네, 맞아요. 
       언제까지 먹어야 하는지도 모르고,
       약의 부작용이 걱정돼요.
       그리고 병원을 자주 가야 하니까 귀찮아요."
```

**상담사 재응답 (2분 50초):**

```
"정확히 말씀해주셨어요.

그럼 반대로 생각해볼게요.

만약 지금부터 제대로 관리해서
혈압이 정상으로 돌아온다면,

3년 뒤에 당신은 
어떤 모습이 되고 싶으세요?

예를 들면:
- 약을 안 먹어도 되고
- 병원도 안 가고
- 에너지가 넘쳐나는 그런 모습?"
```

**포인트:**
- "반대로 생각해볼게요" = 미래 사고로의 전환
- "3년 뒤" = 충분한 시간감 제시 (성공 가능성 ↑)
- "약을 안 먹어도" = 통증의 반대말 (욕망 명확화)

[침묵 3-4초] ← 가장 중요한 침묵

---

### 고객 응답 예시 5 (욕망 표현)

```
고객: "그러면 좋겠지요. 
       병원도 안 가고, 약도 안 먹고,
       그냥 편하게 지낼 수 있으면 좋겠어요.
       손자들이랑도 활기있게 놀 수 있고요."
```

**상담사 재응답 (3분 20초):**

```
"와, 정말 좋은 그림이네요.

약도 안 먹고,
병원도 안 가고,
손자들이랑 활기있게 노는 모습.

정확히 그게 저희가 
도와드리는 길이예요.

그리고 당신처럼
'손자들과 지내고 싶다'고 하신 분들이
실제로 그렇게 되어가요.

지금 당신이 목표한 그 모습이
정말 가능한 거죠."
```

**포인트:**
- 고객의 욕망 반복 (자기 발화 강화)
- "손자들" = 감정적 동기 (가족의 중요성)
- "정말 가능한 거죠" = 확신감 형성

[침묵 2초]

## 🎁 PHASE 3: 패키지 설명 (5-10분)

### 목표
- "정말 간단하네?" 느낌 형성
- 복잡성 제거
- 단계별·구체적 시간 표시 (이해도 ↑)
- "나도 할 수 있을 것 같다" 자신감
- **의료용어 완전 단순화** ← P0-3

---

### 📞 상담사 대사 3-1 (5분 00초 시작) ← P0-3: 의료용어 단순화

```
"그럼 당신, 
저희 건강관리 프로그램을 
설명해드릴게요.

정말 간단합니다. 3단계예요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1) 첫 번째 건강 체크 (처음 1회)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

처음에 한 번 저희 시설에 오시면
혈액검사, 혈압 측정, 체지방 측정을 해요.

소요 시간? 
당신은 편하게 앉아있으면 
저희가 다 해드려요.

걸리는 시간은 딱 10분이에요."
```

**포인트:**
- "정말 간단" 먼저 예고
- "3단계" = Chunking (인지부하 -70%)
- **"검진" → "건강 체크", "체성분 분석" → "체지방 측정"** ← P0-3 의료용어 단순화
- "10분이에요" = 구체적 시간 (불안감 제거)
- "편하게 앉아있으면" = 시니어의 신체 불안 제거

[침묵 2초]

---

### 📞 상담사 대사 3-2 (5분 30초) ← P0-3

```
"결과가 나오면,

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2) 개인 맞춤 상담 (1회)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

우리 전문 의료진이 
당신 결과를 보면서

- 당신이 어떤 음식을 먹으면 좋을지
- 어떤 운동이 좋을지
- 혈압약을 어떻게 계속 복용해야 할지

개별 계획을 만들어드려요.

이건 화상통화로 진행되고,
약 30분 정도 걸려요.

딱 한 번이에요."
```

**포인트:**

- **"상담" = 의료 조언이 아닌 "생활습관" 중심** ← P0-3, P0-4: 의료행위 구분
- 구체적 실생활 예시 (추상→구체)
- "딱 한 번이에요" = 시간 부담감 제거

[침묵 2초]

---

### 📞 상담사 대사 3-3 (6분 00초) ← P0-3

```
"그 다음부터는

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3) 월 1회 정기 확인
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

매달 한 번 전화로 

'요즘 어떻게 지내세요?
건강 상태는 좋아지고 있나요?'

이 정도만 확인해요.

시간은 15분 정도고,

혹시 새로운 증상이 생겼으면
'아, 이건 의료진과 상담해볼까?'
이렇게 제안해드려요.

정말 간단하죠?"
```

**포인트:**
- "15분 정도" = 분명한 시간 약속
- **"제안해드려요" (의료행위 X)** ← P0-4: 의료법 준수
- "확인해요" = 지속적 돌봄 강조 (관계성)
- "정말 간단하죠?" = 확인 질문 (동의 유도)

[침묵 2초]

---

### 📞 상담사 대사 3-4 (6분 30초)

```
"정리하면:

첫 달: 
- 건강 체크 10분 + 개인 상담 30분 = 40분

그 다음부터 매달: 
- 월 1회 확인 15분만

이거 전부예요.

따로 뭐할 게 없어요. 
우리가 다 도와드리는 거죠.

어렵지 않죠?"
```

**포인트:**
- 시간 요약 (숫자로 명확화)
- "이거 전부예요" = Closure 형성
- "어렵지 않죠?" = 동의 확인 질문

[침묵 2.5초]

### 고객 응답 예시 6 ← P0-3: 신체 불안 안전 대응

```
고객: "네, 간단하네요. 
       그런데 검진할 때 
       뭐가 아프지는 않겠죠?"
```

**상담사 재응답 (7분 00초):**

```
"정말 좋은 질문이에요.

혈액검사는 손가락에 
아주 가는 바늘로 한두 방울 정도만 뽑으니까

'따끔'할 정도고,
바로 끝나요. 
[중요] 반복: 정말 불편하지 않아요.

혈압 측정은 완전 안 아파요.
그냥 팔에 조금 타이트하게 느껴질 뿐이고.

체지방 측정은 
우리 기계에 그냥 서 있기만 하면 돼요.
조금도 불편하지 않아요.

저희 고객 중에 
80세 김영희 어머니분도 
'이 정도면 정말 쉽다'고 하셨거든요."
```

**포인트:**
- 시니어의 신체 불안 직접 해소
- **"따끔할 정도" → "바로 끝나요" 반복** ← P0-3 안심
- **"80세 김영희" = 구체적 고객명** ← P0-6: 신뢰도 극대화
- "조금도 불편하지 않아요" 반복 (불안 해소)

[침묵 2초]

## 💰 PHASE 4: 가격 + 기대감 (10-13분)

### 목표
- 가격을 "투자"로 인식 전환
- ROI 명확화 (병원비 대비)
- 의료진·신뢰감 강화
- 3개월 변화 기대감 극대화
- **불안감 조장이 아닌 "예방 중심" 강조** ← P0-4

### 📞 상담사 대사 4-2 (10분 20초) ← P0-4 개선: 불안감 제거, 예방 중심

```
"근데 잠깐, 생각해보세요.

당신이 지금 혈압약을 먹고 계신데,

앞으로 건강 관리를 
제대로 하지 않으면

병원비가 계속 들거든요.

대략적으로:
- 진료비: 3~5만원
- 약값: 2~5만원
- 검사비: 2~10만원

한 번 가면 7~20만원이 드는 거죠.

우리 프로그램은 
당신의 건강을 미리 '지키는' 거라고 생각해보세요.

지금 조금 투자해서
나중에 병원비를 아끼는 것.

어느 게 더 현명할까요?"
```

**포인트:**
- **"건강을 지키는 것" = "예방" 강조** ← P0-4
- **"혹시 입원하면" 같은 극단적 시나리오 제거** ← P0-4: 불안감 제거
- 구체적 병원비 수치 (Loss Aversion 강화하되, 과도하지 않게)
- "앞으로 건강 관리를 제대로 하지 않으면" = 현실적 제시

### 📞 상담사 대사 4-3 (10분 50초)

```
"계산해보면:

우리: 월 40만원 × 12개월 = 년 480만원

만약 당신이 정기적으로 병원 진료받으면:
- 정기 진료: 년 200~300만원

5년이면:
- 우리: 2,400만원
- 병원 진료만: 1,000~1,500만원

우리는 결과가 나오니까
```

- **"결과가 나오니까 병원비도 줄어들 가능성"** = 긍정적 예측
- "현명한 투자" = 의사결정 프레임 제시
