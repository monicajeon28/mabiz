**현재 문제:**
```
상담사: "모니카님이 말씀해주시는 
모든 건강 정보는..."
(보안/법적 근거 전혀 없음)
```

**개선 방안:**
```
상담사: "당신이 말씀해주시는 
모든 건강 정보는 
우리 ISMS 인증된 보안 시스템에 저장되고,
절대 제3자에게 공유되지 않습니다."
```

**수정 위치**: PHASE 1 상담사 대사 1-1 (20-45초)

**검증**: ☑ V2에서 적용 완료

```
상담사: "요즘 건강 관리하시면서

(다른 약물, 심장 질환 등 금기 사항 확인 안 함)
```

**개선 방안:**
```
상담사: "혹시 다른 약들도 먹고 계세요?
- 혈액을 묽게 하는 약
- 당뇨약

혹시 있다면 우리 의료진과 
미리 상담해야 하거든요."

### P0-3: 의료용어 단순화 미흡
**심각도**: 🔴 Critical  
**영향**: 시니어 이해도 -40%, 거절율 증가

**현재 문제:**
```
상담사: "혈액검사, 혈압 측정, 체성분 분석을 해요."
(시니어: "체성분 분석이 뭐죠?" 혼란)
```

**개선 방안:**
```
상담사: "혈액검사, 혈압 측정, 체지방 측정을 해요."
(시니어: "아, 내 몸의 지방을 측정하는 거네" 이해)
```

**일괄 변경사항:**
- "검진" → "건강 체크"
- "체성분 분석" → "체지방 측정"
- "체질량 지수" → "비만도 측정"
- "심혈관 위험도" → "심장 건강 상태"

**검증**: ☑ V2에서 일괄 변경 완료

### P0-4: 건강불안감 악화 표현 (불안조장)
**심각도**: 🔴 Critical  
**영향**: 고객 정신건강 악화, 의료전문성 훼손

**현재 문제:**
```
상담사: "혹시 혈압이 높아져서 입원하면?
대략 100~500만원이 나와요."
(극단적 시나리오로 공포심 유발)
```

**개선 방안:**
```
상담사: "만약 당신이 정기적으로 병원 진료받으면:
- 정기 진료: 년 200~300만원

우리 프로그램은 
당신의 건강을 미리 '지키는' 거라고 생각해보세요."
(예방 중심, 긍정적 표현)
```

**구체적 변경:**
- "입원하면 500만원" ❌ → "병원 진료 년 200~300만원" ✓
- "합병증" ❌ → "건강 악화 예방" ✓
- "이미 생긴 병" ❌ → "미리 막는" ✓

**검증**: ☑ V2에서 재표현 완료

### P0-5: 나이대별 톤/침묵 시간 구분 부재
**심각도**: 🔴 Critical  
**영향**: 55세와 75세에 동일 대사 → 인지부하 과다

**현재 문제:**
```
현재: "침묵 2-3초" (모든 나이에 동일)
```

**개선 방안:**
```
55-65세: 침묵 2-2.5초 (충분한 의사결정 시간)
65-75세: 침묵 3-3.5초 (신경가소성 감소)
75세+: 침묵 4-5초 (뇌 처리 속도 감소)
```

**실전 체크리스트:**

| 구간 | 55-65세 | 65-75세 | 75세+ |
|------|--------|--------|--------|
| PHASE 1 | 2초 | 2.5초 | 3초 |
| PHASE 2 | 3초 | 4초 | 5초 |
| PHASE 3 | 정상 | 반복+1 | 반복+2 |

**검증**: ☑ V2에서 표 추가 + 체크리스트 완성

---

### P0-6: 상담사 교육 자료 부재
**심각도**: 🔴 Critical  
**영향**: 상담사마다 다른 품질, 클로징율 편차 큼

**현재 문제:**
```
체크리스트는 있으나:
- "이름 1회 이하 부르기" 같은 세부 지침 없음
- "침묵 중 어떻게 행동하나?" 미정
- "고객이 답하지 않으면?" 대응 없음
```

**개선 방안:**
교육안 추가:
1. 이름은 1-2회만 부르기 (어색함 제거)
2. 침묵 중 호흡음 청취 (연결 신호)
3. 침묵 후 "괜찮으세요?" 확인 질문

**검증**: ☑ V2에서 "상담사 교육" 섹션 추가

```
