# Phase 3 Track B: 25분 전체 콜 스크립트 — Segment A (신혼부부)

**작성일**: 2026-05-21  
**타겟**: 신혼부부 (25-35세, 내년 3월 신혼여행 계획)  
**목표 클로징율**: 35% → 50% (+43% 개선)  
**통합 프레임워크**: Phase 2 Opening + Phase 3 Track A 이의처리 + PASONA + SPIN + MI + 모니카 4단계  
**구조**: 8단계, 25분, 세그먼트 맞춤형

---

## 📋 Segment A 심리 프로필

```
심리상태: 설렘 + 불확실성 + 시간압박
신경화학: 세로토닌(기대) + 노르에피네프린(불안) 혼재
주요 장벽: 
  ├─ 준비복잡성 (시간부족)
  ├─ 비용불확실성 (신혼금재
  ├─ 배우자 설득 필요성
  └─ "진짜 3월에 가능할까?" 의심

강점:
  ├─ 강한 욕망 (신혼여행은 인생에서 한 번)
  ├─ 결정속도 빠름 (둘 다 동의하면 OK)
  └─ 시간민감도 높음 (지금이 준비의 기회)
```

## 🎯 PHASE 3: 욕망 탐색 (4:30~8:30) — OARS + MI + 손실회피

### [3-1] DARN-CAT 욕망 형성 (1:30 소요)

**상담사:**
> "오늘 제가 보여드리고 싶은 게  
> 정확히 3가지거든요.  
>  
> 1) 유럽 크루즈로 신혼여행  
>    (3월은 안 되지만, 그 이유)  
> 2) 비용이 정말 '월 커피값'일 수 있는 이유  
> 3) 아내분이 안심할 수 있는 이유  
>  
> 이 3가지가 맞으면 이어갈 텐데...  
> 혹시 이거 말고 궁금한 게 더 있어요?"

**[침묵 1초]**

**고객**: "아뇨, 그것들이 제일 궁금한데요."

**상담사:** (DARN-CAT 시작)
> "좋아요.  
> 그럼 **욕망(Desire)** 부터 확인하자면요...  
>  
> 당신은 정말로 내년 초/중반에  
> 아내분과 유럽 크루즈를 타고 싶으세요?"

**[침묵 2초]** ← 고객이 먼저 욕망 언어화

**고객**: "네, 정말 가고 싶어요."

**상담사:** (MI — Affirmation)
> "정말 좋은 마음이에요.  
> 그걸 실현하고 싶은 진심이 느껴져요.  
>  
> 그럼 아내분도 비슷할 것 같은데...  
> **혹시 당신이 아내분께 '이번 여름은 크루즈 타자'라고  
> 제안하면 아내분이 뭐라고 할 것 같아요?**"

**[침묵 2초]** ← 배우자 설득 가능성 파악

**고객**: "아내가 좋아할 것 같아요."

**상담사:** (DARN-CAT: Ability 강화)
> "그렇죠. 왜냐하면 아내분이 이미 크루즈를 원하셨으니까.  
>  
> **그렇다면 '당신이 이걸 실현시킬 수 있다'고  
> 생각해본 적 있어요?**  
> 아직 정보 부족해서 못 생각했을 수도 있지만..."

**[침묵 1.5초]** ← 자기효능감 자극

---

**해설:**
- **DARN-CAT 단계**:
  - D (Desire): "정말 가고 싶으세요?" — 욕망 명시화
  - A (Ability): "당신이 할 수 있다고 생각해요?" — 자기효능감
  - R (Reason): 이유는 다음 단계에서
  - N (Need): 필요성은 Phase 4에서
  - C (Commitment): 약속은 Phase 7(클로징)에서
- MI (Motivational Interviewing): 고객의 자기발화에서 나온 근거를 존중

---

**[침묵 2초]** ← 고객이 자기 동기 언어화

**고객**: "신혼이고... 함께 추억을 남기고 싶어요."

**상담사:** (Affirmation + Need 강화)
> "정말 좋은 이유네요.  
> 당신이 그렇게 생각한다는 게.  
>  
> **그런데 생각해보세요.  
> 만약 지금 이 3개월 준비 기간에**

> **실제 크루즈 탈 때 추억이 얼마나 남을까요?**  
>  
> **아니면 미리 '이렇게 준비하면 돼'라고 알아두면**
