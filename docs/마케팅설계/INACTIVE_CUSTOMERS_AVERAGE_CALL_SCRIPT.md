# 부재중 고객 평균화 콜 스크립트 (25분) 최종 실전판

**상담사가 보면서 그대로 읽을 수 있는 실전 스크립트**  
**대상**: 부재중 고객(1년+) / 모든 세그먼트(신혼/가족/중년/시니어) 통용  
**목표**: 욕망 → 고통 → 솔루션 → 신청 (75% 평균 클로징)  
**시간**: 25분

---

## 📖 실전 콜 스크립트 — 도입부터 마무리까지

### [PHASE 1] 오프닝 (0:00~2:30) — 신뢰 + 기억 + 욕망

**상담사:**
```
"안녕하세요! 더좋은라이프 고객 상담팀의 [이름]입니다.

[고객성함]님 안녕하세요.

지난해 당신이 크루즈 여행을 신청했던 거 기억하세요?"
```

[침묵 1초 — 고객이 생각하게]

**고객:**
```
"아, 네. 기억하긴 하는데... 자리에 없어서..."
```

**상담사:**
```
"그렇죠. 바쁘신 거 알아요.

근데 제가 물어보고 싶은 게...
당시에 그 여행을 신청하려던 이유가 뭐였어요?

어떤 기대감이 있으셨나요?"
```

[침묵 2초]

**고객:**
```
"음... 가족이랑 함께 시간을 보내고 싶어서. 또는 배우자랑..."
```

**상담사:**
```
"정확해요. 그런 마음들이...
진짜 중요한 것들이잖아요.

그런데 왜 지금까지 못 가셨을까요?
그걸 저랑 얘기해보고 싶어요.

왜냐하면... 그걸 알아야
당신이 '지금' 해야 할 이유를 알 수 있거든요."
```

[침묵 1초]

### [PHASE 2] 욕망 탐색 (2:30~6:30) — 고객이 말하게

**상담사:**
```
"그래서 질문드릴게요.

당신이 떠나고 싶은 여행이
어떤 모습일까요?

어디로? 누구랑? 어떤 기분으로?

좀 구체적으로 얘기해주실래요?"
```

[침묵 3초]

**고객:**
```
"음... 배우자랑 한 번 편하게 가고 싶었어요. 또는 아이들이..."
```

**상담사:**
```
"아, 좋은 얘기네요.

그런데 여기가 중요한데...
지금까지 왜 그 여행을 못 가셨을까요?

시간? 비용? 준비 복잡?
뭐가 가장 큰 이유였어요?"
```

[침묵 2초]

**고객:**
```
"음... 제 일이 바빴던 것 같고. 또 준비가 복잡할 것 같고..."
```

**상담사:**
```
"아, 정상적인 생각이에요.

근데 한 가지...
'지금은 너무 바뻐서 나중에 해야지'
이렇게 미루다 보면...

당신이 지금 생각한 그 여행,
정말 현실이 될까요?

몇 년이 더 지나갈 것 같지 않나요?"
```

[침묵 1초]

5년 뒤, 10년 뒤는 어떻게 될까요?

생각해보세요.

'아, 그때 가볼걸...'

그 마음이 얼마나 클까요?"
```

[침묵 3초]

**고객:**
```
"음... 정말 아쉬울 것 같은데요..."
```

**상담사:**
```
"정확해요.

생각해보세요.

당신이 30대, 40대, 50대가 됐을 때...
가장 큰 후회가 뭘까요?

'돈을 못 벌어서?'
'일을 못 해서?'

아니거든요.

대부분 '그때 그 시간을 못 가져서'
이거예요.

당신도 그런 거 아닐까요?"
```

[침묵 2초]

### [PHASE 5] 이의처리 (13:30~18:30) — 분기별 대응

#### [대응 A] 상조 의혹

**고객:**
```
"근데... 혹시 이게 상조는 아니죠?"
```

**상담사:**
```
"아! 그런 불안 정상이에요.

할부라고 하니까 상조처럼 들릴 수도 있고요.

근데 우리는 상조가 아니라 여행회사예요.

차이가 뭐냐면...

상조는 '만약의 사태'에 대비하는 거고,
우리는 '지금 당신의 꿈'을 현실화하는 거거든요.

당신이 처음 신청했던 그 여행.
기억하시죠?

그걸 현실로 만드는 게 우리의 일이에요.

그리고 중요한 게...

할부금은 신한은행이 신탁으로 관리해요.

우리 회사 손에 못 닿는다고 봐도 돼요.
은행이 안전하게 보관하고 있는 거거든요.

결국 월 33,000원으로
더 좋은 경험을 누리는 거.

인생의 추억에 투자하는 거.

이게 상조가 아니라
여행이잖아요."
```

[침묵 1초]

지금까지 '여행을 안 가면서'
매달 얼마를 쓰고 계세요?

카페? 외식? 스트리밍?
인터넷 쇼핑?

월 33,000원이라는 건
그런 것들 중 하나를

'여행'으로 '재배치'하는 거거든요.

빼기가 아니라 옮기기예요.

그리고...

당신이 지금 느끼는 '부담'이 사실은
'후회가 더 클까봐' 하는 불안감 아닐까요?

왜냐하면...
당신은 이미 알고 있거든요.

'지금 안 하면 정말 후회하겠다.'

그걸 몸으로 느끼고 계신 거예요.

그런 생각이 맞으면...
월 1,100원은 비용이 아니라
'미래의 후회'를 사는 투자인 거죠."
```

[침묵 1초]

그리고 만약 배우자분이
뭘 궁금해하면...

당신이 다 설명 못 하는 부분은
저한테 물어봐주고 설명 받으시면 되고요.

다만...

당신이 먼저 신청해두고
배우자분께 보여드리면 어떨까요?

'자기, 보기만 해. 내가 이렇게 준비했어'

## 📋 상담사 체크리스트

□ Phase 1: 신뢰 회복 + 욕망 상기  
□ Phase 2: 미루기 이유 파악  
□ Phase 3: 후회 감정 극대화  
□ Phase 4: 월 1,100원으로 프레이밍  
□ Phase 5: 고객이 제기한 저항 완전 해소  
□ Phase 6: 삼중선택으로 결정 유도  
□ Phase 7: Day 0/1/3 약속 명확히

## 🎯 핵심 팁

**침묵의 위력**
- 질문 후 꼭 2~3초 침묵 → 고객이 생각하고 스스로 말함
- 침묵이 길수록 고객의 자기발화 확대 (MI 효과)

**고객이 말하게**
- "당신은 어떻게 생각해요?"로 끝낼 것
- 상담사가 너무 많이 말하면 X

**감정 앵커 (모든 세그먼트 통용)**
- "인생의 추억"
- "지금이 기회"
- "5년 뒤 후회"
- "평생 못 할 수도"

**Mental Accounting**
- 월 33,000원 = 하루 1,100원 = 아메리카노 한 잔
- 비용이 아니라 "투자"로 재프레이밍

**선택 아키텍처**
- 3가지 선택지 제시 (1번 선택지의 후회가 명확할수록 3번 선택)
- "당신이라면 어떤 선택?"으로 마무리
