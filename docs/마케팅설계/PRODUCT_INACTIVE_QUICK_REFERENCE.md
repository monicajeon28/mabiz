# 기존 고객 재활성화 콜 — 빠른 참조 가이드

**사용 대상**: 상담사 (콜 중 옆에 놓고 참고)  
**콜 타입**: 15분 연속 스크립트  
**목표**: 70% 클로징율  
**중요**: ⭐ 표시 부분은 필수 = 70% 달성의 핵심

---

## 🎯 8개 Phase 타이밍 맵

```
0:00 → PHASE 1 (2분) → 오프닝 + 기존 경험 회상
2:00 → PHASE 2 (1.5분) → V1 vs V2 간결 설명 ⭐
3:30 → PHASE 3 (2분) → 미루기 이유 파악
5:30 → PHASE 4 (2.5분) → 개인화된 업그레이드 ⭐
8:00 → PHASE 5 (1.5분) → 3가지 가격 플랜 ⭐
9:30 → PHASE 6 (2분) → 4가지 거절 대응 ⭐
11:30 → PHASE 7 (2분) → 삼중선택 + 개인주권 존중 ⭐
13:30 → PHASE 8 (1.5분) → Day 0/1 자동화 약속
        ↓
15:00 끝!
```

## ⚠️ 절대 금지 사항

```
❌ "아마도 또 가고 싶을 거예요" → 추측하지 말기
❌ "비슷한 수준" / "어떤 건 싸요" 모순 주장 → 3플랜으로 명확히
❌ 고객의 특정 지난 경험 모르고 일반화 → "당신은 스탠다드였죠?" 확인
❌ 배우자 의견을 대신 대변 → "배우자분이 뭐라고 하셨어요?"
❌ Phase 6 거절 중 새로운 대응 즉흥적 만들기 → 4가지 시나리오만 사용
❌ 배우자 동의를 필수 조건처럼 강요 → "당신이 결정하세요"
```

## 📊 클로징 성공의 3가지 핵심

### 1️⃣ 개인화 (Personalization)
```
"당신은 지난번에 스탠다드 객실을 다녀오셨죠?
이번엔 그 가격대에 발코니 객실까지 올려드릴 수 있어요."

← 모든 고객이 아닌 "당신만의" 업그레이드
```

### 2️⃣ 명확한 가격 (Clear Pricing)
```
"당신이 지난번에 3,500,000원 냈던 거 기억하죠?

플랜 A: 3,500,000원 (발코니)
플랜 B: 4,000,000원 (스위트+레스토랑)
플랜 C: 3,100,000원 (스탠다드 + 서비스업그레이드)"

← 구체 금액 = 신뢰도 +40%
```

### 3️⃣ 배우자 선택권 (Spouse Respect)
```
"당신이 먼저 결정하세요.
배우자분과 상의하고 싶으면 그렇게 하세요.
당신이 편한 대로 해주세요."

← 강요감 제거 = 거절 -20%
```

## 📋 콜 후 체크리스트

```
□ 신청 성공
  → product_inactive 테이블 업데이트 (call_status = 'closed_success')
  → Day 0 이메일 자동 발송 (상세 가이드 + 비교표)
  → Day 1 이메일/SMS 배우자 발송
  
□ 거절 (이의처리 진행)
  → product_inactive 테이블 업데이트 (call_status = 'closed_objection_[A/B/C/D]')
  → 어떤 대응 사용했는지 기록
  → 3일 후 재콜 예약
  
□ 고객 불응 (콜 받지 않음)
  → 다음주 재시도 (최대 2회)
```
