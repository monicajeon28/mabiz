   ```
   변경 전: "첫 출시 가격", "정가의 30% 할인"
   변경 후: "선발 고객 혜택", "6개월 한정 특가"

   ```
   상담사: "저희가 심사숙고해서 출시한 신제품이라서,
           초반에 구매하는 분들을 배려하는 거죠.
           
           이 할인가는 6개월 동안만 유효해요."
   ```

3. **공평성 강화: 모든 신제품에 동일 정책 적용**
   ```

**심리학 원리**:
- Reciprocity (감사): 배려받는 느낌 → 구매 의도 상승

### P0-7: 클로징율 현실화 (목표 관리)

**원본 문제**:
```
목표: 25% → 45% (+80% 개선)
문제: 신제품은 신뢰도 낮음
결과: 45% 달성 불가 → 팀 사기 저하
```

**해결 내용** (3가지):

1. **목표 수정**
   ```
   변경 전: 25% → 45% (비현실적)
   변경 후: 25% → 35% (현실적)
   
   근거:
   - 신제품 신뢰도: 낮음
   - 초기 신제품 기준: 20~30%
   - SMS + 환불보장: +5~10%
   - 결론: 35% 달성 가능
   ```

2. **단계화된 목표 수립**
   ```
   Phase 1 (0~1개월): 25% → 30%
   Phase 2 (1~3개월): 30% → 35%
   Phase 3 (3~6개월): 35% → 40%
   
   → 지속 가능한 성장 경로
   ```

3. **문서 전체에 35% 반영**
   - 스크립트 헤더: "클로징율 목표: 35%"
   - 체크리스트: "추천 클로징율 35% 설정"
   - 주의사항: "P0-7 비현실적 목표 수정"

**효과**:
- 팀 사기: 달성 가능 목표로 동기 부여
- 측정: 현실적 기준으로 성과 평가
- 지속성: 6개월 장기 성장 경로 제시
