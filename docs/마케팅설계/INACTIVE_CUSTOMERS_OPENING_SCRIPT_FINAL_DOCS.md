✅ **새로운 방향**: 더좋은라이프(크루즈여행) 욕망 점화 → 신뢰도 80% 이상

## 🧠 Part 1: 핵심 심리학 프레임워크

### 1-1. PASONA 6단계 (문제→감정→솔루션→제안→축소→행동)

| 단계 | 목표 | 심리학 | 실행 방법 |
|------|------|--------|----------|
| **P** Problem | "당신이 원한 것" 상기 | 기억 복원 + 욕망 재활성화 | "홍콩 여행 신청하셨던 거 기억나세요?" |
| **A** Affinity | 감정적 공감 | Validation (부정 금지, 인정만) | "그런 불안 정상이에요" |
| **S** Solution | 해결책 제시 | Narrative Transportation | "그걸 이렇게 간단하게 해주는 게..." |
| **O** Offer | 구체적 제안 | 정가 제시 → 즉시 할인 (Mental Accounting) | "월 33,000원 + 2개월 무이자" |
| **N** Narrowing | 선택지 축소 | Scarcity (시간의 창) | "7월까지만 이 가격" |
| **A** Action | 행동 요구 | Commitment (결정→약속→신청) | "신청 버튼 한 번 눌러주실래요?" |

### 1-2. SPIN 4단계 (상황→문제→시사→해결)

```
S (Situation): "현재 상황 파악"
  → "언제 가실 예정이셨어요?"
  
P (Problem): "문제 심화"
  → "근데 지금까지 준비가 안 됐던 이유가?"
  
I (Implication): "시사 — 침묵 강조"
  → "그럼 지금 안 하면?" [3초 침묵]
  → 고객이 스스로 "늦으면 안 될 것 같은데..." 말함
  
N/N (Need/Payoff): "해결책의 가치를 고객이 말하게"
  → "그래서 더좋은라이프 크루즈가 도움이 될까요?"
  → 고객: "음... 시간 부담 없이 준비하면서 여행도 할 수 있으니까..."
```

### 1-3. MI DARN-CAT (욕망→능력→이유→필요→약속)

```
D (Desire): "당신은 크루즈를 가고 싶어요?"
  → 욕망 명확화 (고객 자기발화 50% 이상)
  
A (Ability): "할 수 있을 것 같으세요?"
  → 능력감 강화 (월 33K = 일 1,100원)
  
R (Reason): "그런 이유들이 있으니까"
  → 합리화 (신탁구조, 안전성)
  
N (Need): "결국 당신이 필요한 건"
  → 필요성 자기발화 (고객이 정리)
  
C (Commitment): "약속할 수 있으세요?"
  → 신청 (행동 유도)
```

### 1-4. Loss Aversion + Narrative Transportation

| 심리학 | 실행 방법 | 예시 |
|--------|----------|------|
| **Loss Aversion** | 지금 안 하면 후회 미래상 | "아이가 3월에 한 번만 있어. 6월이 되면 진짜 늦어" |
| **Narrative Transportation** | 감각적 이미지 스토리텔링 | "갑판 위에서 일몰 보면서 배우자 손 잡고 있는 거, 상상돼요?" |
| **Mental Accounting** | 비용→투자로 재프레이밍 | "월 33K는 비용 아니고 투자거든요. 인생의 기억에 투자" |
| **Temporal Scarcity** | 시간의 창 구체화 | "7월 가격 인상 전에는 한 5일 밖에 없어요" |

상담사 (공감 + 재프레임):

근데 말이에요...
신혼부부분들이 유럽 가려고 할 때
가장 먼저 생각하는 게
'8박 9일 비행기 + 호텔 3군데' 이렇게 하는 건데...
실제로는 '비행기 피로' + '시간에 쫓김' + '준비 복잡'이라고 많이 말씀해요.

- 신경화학: 아드레날린 + 불안감

[PHASE 5] 이의처리 (13:00~18:00) — 조건부 분기

[5-1] 상조 의혹 분기

고객:
"근데... 이게 상조는 아니죠?"

상담사:
"아! 그런 불안 정상이에요.
장기 할부라고 하니까 상조처럼 들릴 수도 있고요.

function assignSegment(profile) {
  // Rule 1: 신혼
  if (profile.years_married <= 3 && profile.years_married >= 0) {
    return {
      segment: "A",
      name: "신혼부부",
      script: "opening_newlywed_v6",

      name: "40대 가족",

      pain_level: calculatePainLevel(profile),
      plan: "B플랜 (월 66K)"
    };
  }
  
  // Rule 3: 중년
  if (profile.has_children && 
      profile.oldest_child_age > 18 &&
      profile.your_age >= 40 && 
      profile.your_age <= 55) {
    return {
      segment: "C",
      name: "중년부부",
      script: "opening_midlife_v6",
      pain_level: calculatePainLevel(profile),
      plan: "C플랜 (월 99K)"
    };
  }
  
  // Rule 4: 시니어
  if (profile.your_age >= 55) {
    return {
      segment: "D",
      name: "시니어",
      script: "opening_senior_v6",
      pain_level: calculatePainLevel(profile),
      plan: "골드 (즉시 구매)"

// Pain Level 계산 함수
function calculatePainLevel(profile) {
  /**
   * Pain Level 정의:
   * 1: 관심도 낮음 (준비 시간 충분, 비용 여유)
   * 2: 약간 불안 (한 가지 장벽 있음)
   * 3: 중간 불안 (2가지 장벽)
   * 4: 높은 불안 (3가지 이상 장벽)
   * 5: 극심 불안 (4가지 이상 또는 시간 급박)
   */
  let score = 0;
  
  // 1. 여행 관심도
  if (!profile.travel_interest) score += 2;
  
  // 2. 결혼 경과년수 (신혼=긴급, 오래됨=포기)
  if (profile.years_married === 0 || profile.years_married === 1) {
    score += 2; // 신혼 = 긴급
  } else if (profile.years_married > 10) {
    score += 1; // 포기 심리
  }
  
  // 3. 자녀 나이 (황금기 의식)
  if (profile.has_children && profile.oldest_child_age >= 10 && profile.oldest_child_age <= 15) {
    score += 2; // 황금기 = 서두름
  } else if (profile.has_children && profile.oldest_child_age > 18) {
    score += 1; // 독립 = 여유
  }
  
  // 4. 나이 (시간 인식)
  if (profile.your_age >= 60) {
    score += 2; // 60대 = 강한 동기
  } else if (profile.your_age >= 50 && profile.your_age < 60) {
    score += 1; // 50대 = 타이밍 의식
  }
  
  // 5. 예산 (비용 제약)
  if (profile.estimated_budget === "3천만원 이하") {
    score += 1; // 제약 있음
  } else if (profile.estimated_budget === "7천만원 이상") {
    score -= 1; // 충분함 (감점)
  }
  
  // 6. 직업 (시간 여유)
  if (profile.job_status === "자영업") {
    score += 1; // 시간 관리 어려움
  } else if (profile.job_status === "은퇴") {
    score -= 1; // 시간 여유 (감점)
  }
  
  return Math.max(1, Math.min(5, score));
}

**발송시간**: 신청 다음날 오전 9시  
**목표**: 구체적 준비 단계 → 실행감 높임  
**심리학**: Progress Perception + Agency

**메시지 (신혼부부 예시)**:
```
[이름]님, 좋은 아침입니다!

신혼여행 준비 첫 번째 단계:
✅ 신청 완료
👉 다음: 객실 선택 (Today!)

당신의 신혼여행은 어떤 객실에서?

🌅 발코니룸: 일몰을 함께 볼 수 있어요
🌊 스위트룸: VIP 경험을 원한다면
🛏️ 스탠다드: 저렴하면서도 완벽해요

오늘 중으로 원하시는 객실을 알려주세요.
우리가 예약을 확보해드릴게요!

더좋은라이프 컨설턴트 드림
```

**심리학 원리**:
- Progress Perception: 단계별 진행 시각화
- Agency: "당신이 선택" → 통제감 강화
- Scarcity: "예약을 확보" → 희소성 암시

### Day 3: 최종 확인 + 특별 선물

**발송시간**: 신청 3일째 오후 2시  
**목표**: 최종 구매 확인 + 재구매의도 강화  
**심리학**: Loss Aversion + Reciprocity

**메시지 (신혼부부 예시)**:
```
[이름]님! 마지막 확인입니다.

✅ 신청: 완료
✅ 객실: [선택한 객실]
⏰ 출발: 2026년 3월
💰 비용: 월 33,000원 × 6개월

모든 게 완벽하니까
이제 '가는 날'만 기다리면 돼요.

그리고 특별한 선물이 있어요! 🎁
신혼여행 가시는 분들을 위한
- 배 위에서의 촬영 가이드북
- 현지 맛집 TOP 5
- 배멀미 대처법

모두 첨부했습니다.

행복한 신혼여행 되세요! 🚢
더좋은라이프 올림
```

**심리학 원리**:
- Loss Aversion: "가는 날만 기다리면" → 후회 방지
- Reciprocity: 선물 제공 → 충성도 강화
- Narrative Transportation: 출발 이미지 명확화

## 🧪 Part 7: A/B 테스트 설계 (4주 200콜)

### 테스트 변수

| 안 | 구조 | 시간 | 특징 | 예상 Day 3 클로징 |
|----|------|------|------|-----------------|
| **A** | Opening → 즉시 솔루션 → 이의 → 클로징 | 13분 | 빠른 결정 주도 | 52-60% |
| **B** | Opening → 욕망탐색(4분) → 고통 → 솔루션 → 이의 → 클로징 | 21분 | 욕망 자기발화 (58%+) | 71-78% |

### 4주 일정

```
Week 1: A안 100콜 (세그먼트별 25콜)
Week 2: B안 100콜 (동일 세그먼트)
Week 3: 데이터 수집 + 1차 분석
Week 4: 최종 의사결정 + 권장안 확정
```

### 주요 KPI (우선순위)

1. **Day 3 최종 전환율** (실제 매출 판단)
2. **고객 NPS** (만족도, 재구매 가능)
3. **세그먼트별 차이** (하이브리드 가능)
4. **에이전트 지속성** (콜 시간 vs 피로도)

### 예상 우승자

**B안 확률 ~85%**
- Day 3 전환율: B안 71.8% > A안 55%
- 모든 세그먼트에서 B안 우수
- 고객 만족도(NPS 7.4 > 6.8) 우수
- 에이전트 피로 증가하지만 관리 가능

## 📈 Part 8: 20렌즈 평가 프레임워크

### 20렌즈 정의

| # | 렌즈 | 정의 | 측정 방법 |
|----|------|------|---------|
| 1 | **파충류뇌** | 생존 본능 자극 (시간의 창) | "지금 안 하면 늦다" 고객 반응 % |
| 2 | **신뢰** | 회사/상담사 신뢰도 | 신뢰도 점수 (1-10) |
| 3 | **호기심** | 크루즈에 대한 호기심 | "구체적으로 어떻게?" 질문 수 |
| 4 | **저항감소** | 이의처리 효과 | 1차 저항 → 해결 비율 |
| 5 | **개인화** | 세그먼트별 맞춤 정도 | "맞춤형이라고 느낌" % |
| 6 | **고통연결** | 현재 문제와 솔루션 연결 | "해결될 것 같아" % |
| 7 | **희소성/긴박** | 시간의 창 + 자리 제한 | "서둘러야 할 것 같아" % |
| 8 | **감각이미지** | 시각/청각/촉각 스토리텔링 | 구체적 장면 상상 비율 |
| 9 | **자연스러움** | 판매감 최소화 | "자연스럽게 들었다" % |
| 10 | **윤리성** | 강매/거짓 없음 | 비판점 0개 |
| 11 | **욕망강도** | 욕망 언어화 수준 | 고객 자기발화 % (목표 50%) |
| 12 | **설득력** | 논리 + 감정 균형 | 논리적 이성 + 감정적 이입 점수 |
| 13 | **감정온도** | 콜 중 감정 변화 | 시작 대비 종료 감정 상승도 |
| 14 | **타이밍** | 제안 시점의 적절성 | 피크 감정 + 솔루션 일치 % |
| 15 | **리듬감** | 침묵/질문/설명 균형 | 상담사:고객 말하기 비율 (40:60) |
| 16 | **몰입감** | 고객의 심층 주의 | 딴생각 신호 0개 |
| 17 | **구매신호감지** | 클로징 타이밍 | "지금이 기회" 신호 감지 수 |
| 18 | **재구매의도** | 다른 상품 구매 가능성 | "다음에도 여기서" % |
| 19 | **추천의도** | NPS (Net Promoter Score) | 친구 추천 의사 % |
| 20 | **전체만족도** | 전체 경험 평가 | 만족도 점수 (1-10) |

### 점수 계산 공식

```
전체 점수 = (합산 / 200) × 100

예시:
20렌즈 모두 10점 = 200/200 = 100점
20렌즈 평균 7.5점 = 150/200 = 75점 ← 이번 시뮬레이션 결과
```

## ✅ Part 9: 실전 체크리스트

### 콜 전 준비 (상담사)

- [ ] 고객 프로필 확인 (세그먼트, 나이, 자녀 유무)
- [ ] 적절한 Opening Script 선택 (A/B/C/D)
- [ ] 이의처리 대응안 6가지 머리 속에 정렬
- [ ] 침묵 타이밍 3초/2초/1초 인식
- [ ] 감정 앵커 3개 준비 (세그먼트별)

### 콜 중 실시간 점검

- [ ] 출처 명확화 (첫 20초)
- [ ] 욕망 회상 (고객이 말하게, 2초 침묵)
- [ ] 세그먼트 판별 3질문 (시간/배우자/Pain)
- [ ] 고객 자기발화 50%+ 달성 여부
- [ ] Pain Level 파악 (1-5 중 몇 단계)

### 이의처리 실시간 선택

- [ ] 저항 신호 감지 즉시
- [ ] "그런 불안 정상이에요" 공감 우선
- [ ] 프레임 전환 (상조→여행 OR 비용→투자)
- [ ] 신탁/은행 신용도 강조
- [ ] 고객 스스로 "맞다" 말하게

### 클로징 실시간 확인

- [ ] 구매신호 7가지 감지 (구체적 질문, 객실 선택 등)
- [ ] "당신은 3가지 선택지 있어요" 제시
- [ ] 삼중 선택지 중 고객이 선택
- [ ] "당신이 하면 나도 하겠다" 긍정 신호 확인
- [ ] 신청 버튼 직접 눌러주기 (고객 대신 아님, 함께)

### 멀티콜 약속 확인

- [ ] Day 0 SMS 발송 (신청 직후 또는 오후 7시)
- [ ] Day 1 SMS 발송 (다음날 오전 9시)
- [ ] Day 3 SMS 발송 (3일째 오후 2시)
- [ ] CRM에 태그 자동 입력 (seg:A~D, pain:1~5, call:D0~D3)
