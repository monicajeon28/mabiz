**파일**: `INACTIVE_CUSTOMERS_OPENING_SCRIPT_FINAL_DOCS.md` Line 695-708

```javascript
function calculatePainLevel(profile) {
  /**
   * Pain Level 정의:
   * 1: 관심도 낮음 (준비 시간 충분, 비용 여유)
   * 2: 약간 불안 (한 가지 장벽 있음)
   * 3: 중간 불안 (2가지 장벽)
   * 4: 높은 불안 (3가지 이상 장벽)
   * 5: 극심 불안 (4가지 이상 또는 시간 급박)
   */
  
  let score = 0;
  
  // 1. 여행 관심도 (기본 점수)
  if (!profile.travel_interest) {
    score += 2; // 관심도 미기입 = 상담 필요
  }
  
  // 2. 결혼 경과년수 (신혼은 서두름, 오래 결혼은 이미 포기)
  if (profile.years_married === 0 || profile.years_married === 1) {
    score += 2; // 신혼 = 긴급성 높음
  } else if (profile.years_married > 10) {
    score += 1; // 오래 결혼 = 이미 포기 심리
  }
  
  // 3. 자녀 유무 + 나이 (가족 책임감)
  if (profile.has_children && profile.oldest_child_age >= 10 && profile.oldest_child_age <= 15) {
    score += 2; // 황금기 = 서두름
  } else if (profile.has_children && profile.oldest_child_age > 18) {
    score += 1; // 자녀 독립 = 준비 여유
  }
  
  // 4. 나이 (시간 인식)
  if (profile.your_age >= 60) {
    score += 2; // 60대 = "몸이 허락할 때" 강한 동기
  } else if (profile.your_age >= 50 && profile.your_age < 60) {
    score += 1; // 50대 = 타이밍 의식
  }
  
  // 5. 예상 예산 (비용 불안감)
  if (profile.estimated_budget === "3천만원 이하") {
    score += 1; // 비용 제한
  } else if (profile.estimated_budget === "7천만원 이상") {
    score -= 1; // 비용 충분 = 저점수 (한도 하향)
  }
  
  // 6. 직업 상태 (시간 여유)
  if (profile.job_status === "자영업") {
    score += 1; // 자영업 = 시간 관리 어려움
  } else if (profile.job_status === "은퇴") {
    score -= 1; // 은퇴 = 시간 여유 (저점수)
  }
  
  // 최종 계산: 1-5 범위로 클램핑
  return Math.max(1, Math.min(5, score));
}
```

```javascript
function getDay0SendTime(signupTime, signupDate) {
  const hour = signupTime.getHours();
  const dayOfWeek = signupDate.getDay(); // 0=일, 1=월, ..., 6=토
  const isHoliday = checkIsHoliday(signupDate);
  
  if (isHoliday) {
    return getNextBusinessDay(signupDate) + " 09:00";
  }
  
  if (dayOfWeek === 0 || dayOfWeek === 6) {
    // 주말
    if (dayOfWeek === 6 && hour < 15) {

    }
    return getNextMonday(signupDate) + " 09:00";
  }
  
  // 평일

    return signupDate + " 17:00";
  } else {
    return getNextDay(signupDate) + " 09:00";
  }
}
```

```markdown
## 📚 Part 11: 상담사 교육 프로토콜

### 침묵 타이밍 학습 가이드

#### 1. 침묵의 과학
```
WAIT 연구 (Mottet & Beebe, 1995):
- 1초 침묵 → 고객이 "통화 끊김" 오해 가능
- 2초 침묵 → 최소 효과 (약한 신호)
- 3초 침묵 → 최적 효과 (고객 깨달음 최대)
- 4초 침묵 → 고객 불편함 시작
- 5초+ 침묵 → 고객이 "전화 끊는" 행동 (탈주)

우리 콜의 침묵 정책:
- Opening 후: 1초 (고객 반응 확인)
- 욕망탐색 질문 후: 2초 (고객이 말하게)
- SPIN I (Implication) 후: 3초 (고객이 깨닫게)
- 클로징 전 선택지 제시 후: 2-3초 (고객이 결정하게)
```

#### 2. 침묵을 세는 방법

**방법 1: 내적 계수 (상담사용)**
```
"천천히 일" (1초 소요)
"천천히 이" (2초 소요)
"천천히 삼" (3초 소요)

주의: "1, 2, 3" 빠르게 세면 실제는 1초 미만 가능
→ 각 숫자마다 "천천히" 앞에 붙이기
```

**방법 2: 음성 신호 활용**
```
상담사가 침묵 직전에:
- 음성톤 ↑ "어떻게 될까요?" (물음표 톤)
- 시선 전환: 화면 보지 않기 (고객 말 듣기 모드)

침묵 3초 동안:
- 절대 말 금지
- 절대 "음..." "어..." 금지 (침묵 깨짐)
- 절대 키보드 소리 내기 금지

침묵 직후:
- 3.5초 후 고객이 말 안 하면
  "뭐라고 생각하세요?" (재확인)
```

#### 3. 침묵 중 고객이 말하는 경우

**시나리오 1: 침묵 1초 만에 고객 말함**
```
상담사 (3초 계획했지만):
"...?"
고객: "음, 그건..."

대응: 침묵 중단
상담사: [고객 말 끝까지 듣기 — 절대 중단 금지]
```

**시나리오 2: 침묵 중 고객이 웃음**
```
상담사 (계획한 침묵 진행):
[침묵 1초, 2초, 고객 웃음 소리 들림]

대응: 그대로 진행 (웃음 = 긍정 신호)
침묵 3초 완료 후:
상담사: "좋은 표정이네요. 뭐가 떠오르셨어요?"
```

**시나리오 3: 침묵 중 고객이 한숨**
```
상담사 (침묵 진행):
[침묵 2초, 고객 한숨]

대응: 다음 3초 완료 후:
상담사: "음, 그런 마음이 드시네요. 뭐가 제일 걱정이세요?"
(한숨 = 부정 신호 → 이의처리로 전환)
```

#### 4. 침묵 연습 (역할극)

**연습 1: 혼자 연습 (자가진단)**
```
녹음기 켜기 → 아래 질문 읽기 → 침묵 3초 유지 → 재생해서 확인

"당신이 신혼여행을 이번 해도 못 간다면, 5년 뒤는 어떻게 될까요?"
[침묵 세기: 천천히 1, 천천히 2, 천천히 3]
"뭐라고 생각하세요?"

→ 녹음 재생해서 침묵 길이 확인 (실제 3초인지 확인)
```

**연습 2: 페어 연습 (동료와)**
```
상담사 A: 스크립트 읽기 (침묵 포함)
상담사 B: 고객 역할

A: "지금 안 하면 늦을 것 같은데요?"
[침묵 — A가 3초 센다]
B: (고객처럼) 침묵 듣고 생각하기 → "음... 맞네요..."

→ 역할 바꿔서 반복
```

**연습 3: 관리자 코칭 (녹음 검토)**
```
실제 콜 녹음 → 관리자가 확인:
- 침묵이 진짜 3초였는가?
- 침묵 중 "음..." 같은 필러 소리는 없었나?
- 고객 반응이 긍정적이었나?

→ 1주일에 1회 코칭 (5콜 중 1콜 검토)
```

### 침묵 외 상담사 스킬

#### 음성톤 변화

| 상황 | 톤 | 예시 |
|------|-----|------|
| 신뢰 구축 | ⬇️ 낮고 느림 | "저는 크루즈닷 컨설턴트입니다" |
| 욕망 탐색 | ⬆️ 높고 호기심 | "가장 먼저 떠오르는 곳이 어디예요?" |
| 고통 증폭 | ⬇️ 진지함 | "5년 뒤를 생각해보세요" |
| 솔루션 제시 | ⬆️ 희망적 | "월 33,000원으로 가능해요!" |
| 클로징 | ➡️ 확신 있음 | "신청할래요?" |

#### 고객 반응별 음성톤 맞추기

**고객이 침울할 때**:
- 상담사 톤 ↑ (밝게, 희망 전달)

**고객이 흥미 있을 때**:
- 상담사 톤 유지 (같은 에너지)

**고객이 저항할 때**:
- 상담사 톤 ⬇️ (진지함, 공감)

### 고객 침묵 처리 가이드

| 상황 | 대응 | 타이밍 |
|------|------|--------|
| 고객이 말 안 함 | "뭐라고 생각하세요?" | 3초 후 |
| 고객이 웃음 | 그대로 진행 | 웃음 끝까지 |
| 고객이 한숨 | "걱정이 있으신가봐요" (공감) | 즉시 |
| 고객이 묻음 | 질문에 답하기 (판매 아님) | 즉시 |
| 고객이 방해음 | "지금 바쁘신가요?" (배려) | 즉시 |

```
