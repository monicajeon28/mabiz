# 헬스케어 시니어(55세+) 콜 스크립트
## 15분 실전 연속 대사 + 고객 응답 예시

**제작일**: 2026-05-21  
**대상**: 55세 이상 시니어 고객  
**목표 전환율**: 52-68% (성인 일반 45-55% 대비 +10-15%)  
**목표 통화 시간**: 정확히 15분  
**핵심 차별화**: 침묵 길이 증가(2-3초), 말 속도 저하, 신뢰감/권위감 강조, 건강 안전성 우선, 이해도 확인 반복

---

## 📋 시니어 심리학 기초

| 특성 | 심리학 원리 | 적용 전략 |
|------|-----------|---------|
| 의사결정 느림 | 신경가소성 감소 | 충분한 침묵(2-3초) + 이해도 확인 |
| 신뢰감 민감 | 권위편향↑ | 전문가 자격/경험 반복 언급 |
| 건강 불안 | Loss Aversion 강화 | 병원비·통증·의료비 구체화 |
| 속도감 거부 | 현상유지편향 | 천천히·명확히·단계별 설명 |
| 사회성 강함 | Social Proof↑ | "우리 고객들" "많은 분들" 강조 |
| 실제 증거 선호 | Concrete Thinking | 추상적→구체적 사례로 변환 |

## 🎯 PHASE 1: 인사 + 신뢰감 형성 (0-2분)

### 목표
- 상담사 신원 확실히 전달
- 전문성 및 경험 입증
- "내 또래를 이해하는 사람이군" 느낌
- 시간 압박감 제거 (충분한 시간이 있다는 신호)

---

### 📞 상담사 대사 1 (0-20초)

```
"안녕하세요? 여기는 크루즈 건강관리팀입니다.

저는 모니카고요, 지난 8년간 
건강관리 상담을 담당해왔습니다.

혹시 지금 통화 괜찮으신가요?"
```

**포인트:**
- 이름 명확히 (이름 확인 불안 제거)
- "8년" = 경험/신뢰성 입증
- "지금 통화 괜찮으신가요?" = 시간 선택권 제공 (거절감 -50%)

[침묵 2초]

---

### 고객 응답 예시 1

```
고객: "네, 괜찮습니다. 뭔가요?"
```

**상담사 재응답 (20-45초):**

```
"감사합니다. 

저희가 최근 3년간 55세 이상의 
어른분들 1,200명 정도를 
건강관리로 도와드렸는데,

특히 모니카님 연령대 분들이 
'건강해졌어요', '편해졌어요'라고 
가장 만족해하시거든요.

그래서 전화를 드렸어요.

5분만 들어주시겠어요?"
```

**포인트:**
- "1,200명" = Social Proof (신뢰도 +60%)
- "55세 이상" = 자신의 또래 명시 (공감 +45%)
- "가장 만족" = 긍정 기대감
- "5분만" = 낮은 진입장벽

[침묵 2초] ← 시니어는 인지 처리 시간 필요

---

### 고객 응답 예시 2 (거절 약한 형태)

```
고객: "뭐 파는 거 아니에요?"
```

**상담사 재응답:**

```
"아니에요. 저희는 상품 파는 게 아니라

'모니카님이 지금부터 어떻게 하면
더 건강하고 편하게 지낼 수 있을까'

이것만 일단 이야기하는 거예요.

물론 도움 드릴 방법도 있긴 한데,
먼저 상황을 알아보려고 하는 거고요.

5분만 괜찮을까요?"
```

**포인트:**
- 오해 해소 명시적
- "모니카님이 더 건강하게" = 이해감 강화
- "일단 이야기" = 압박감 제거

[침묵 2.5초]

## 💭 PHASE 2: 욕구 발굴 (2-5분)

### 목표
- 현재 건강 상태 정확히 파악
- 통증점(pain point) 구체화 (불안감→명확한 문제로)
- 미래 욕망 상향식 발굴
- 고객의 자기발화 유도 (상담사 설득이 아닌 고객 스스로 인식)

---

### 📞 상담사 대사 2-1 (2분 00초 시작)

```
"그럼 먼저 모니카님 상황을 
이해하고 싶어서요.

요즘 건강 관리하시면서 
가장 힘든 부분이 뭐예요?

예를 들면:
- 피로가 계속 남아있다든지
- 혈압이 높다든지
- 뭘 먹어야 할지 모르겠다든지
- 병원을 자주 간다든지

이런 것들이 있나요?"
```

**포인트:**
- "이해하고 싶어서" = 상담사의 진심 전달
- "구체적 예시" = 답변 용이도 +70%
- 천천히 읽기 (시니어 청취 처리 시간)

[침묵 3초] ← 가장 길게 (고객이 생각할 시간)

---

### 고객 응답 예시 3 (통증 제시)

```
고객: "글쎄요, 요즘 저녁만 되면 피로가 심하고... 
       혈압약도 먹고 있는데, 
       언제까지 먹어야 하나 싶어요."
```

**상담사 재응답 (2분 30초):**

```
"아, 그렇군요. 혈압약을 벌써 복용 중이시네요.

그럼 지금까지 어떤 영향이 있었어요?

혈압약 때문에 불편했던 게 있나요?

아니면 혈압약을 계속 먹어야 하는 게 
답답하신 거예요?"
```

**포인트:**
- 공감("그렇군요") 먼저
- 단계별 질문 (한 번에 하나씩)
- 고객의 진짜 불안을 찾기

[침묵 3초]

---

### 고객 응답 예시 4

```
고객: "네, 맞아요. 언제까지 먹어야 하는지도 모르고,
       약 때문에 다른 부작용이 있을까봐 걱정돼요.
       그리고 병원을 자주 가야 하니까 귀찮아요."
```

**상담사 재응답 (2분 50초):**

```
"정확히 말씀해주셨어요.

그럼 반대로 생각해볼게요.

만약 지금부터 제대로 관리해서
혈압이 정상으로 돌아온다면,

3년 뒤에 모니카님은 
어떤 모습이 되고 싶으세요?

예를 들면:
- 약을 안 먹어도 되고
- 병원도 안 가고
- 에너지가 넘쳐나는 그런 모습?"
```

**포인트:**
- "반대로 생각해볼게요" = 미래 사고로의 전환
- "3년 뒤" = 충분한 시간감 제시 (성공 가능성 ↑)
- "약을 안 먹어도" = 통증의 반대말 (욕망 명확화)

[침묵 3초] ← 가장 중요한 침묵

---

### 고객 응답 예시 5 (욕망 표현)

```
고객: "그러면 좋겠지요. 
       병원도 안 가고, 약도 안 먹고,
       그냥 편하게 지낼 수 있으면 좋겠어요.
       손자들이랑도 활기있게 놀 수 있고요."
```

**상담사 재응답 (3분 20초):**

```
"와, 정말 좋은 그림이네요.

약도 안 먹고,
병원도 안 가고,
손자들이랑 활기있게 노는 모습.

정확히 그게 저희가 
도와드리는 길이예요.

그리고 모니카님처럼
'손자들과 지내고 싶다'고 하신 분들이
실제로 그렇게 되어가요.

지금 모니카님이 목표한 그 모습이
정말 가능한 거죠."
```

**포인트:**
- 고객의 욕망 반복 (자기 발화 강화)
- "손자들" = 감정적 동기 (가족의 중요성)
- "정말 가능한 거죠" = 확신감 형성

[침묵 2초]

## 🎁 PHASE 3: 패키지 설명 (5-10분)

### 목표
- "정말 간단하네?" 느낌 형성
- 복잡성 제거
- 단계별·구체적 시간 표시 (이해도 ↑)
- "나도 할 수 있을 것 같다" 자신감

---

### 📞 상담사 대사 3-1 (5분 00초 시작)

```
"그럼 모니카님, 
저희 건강관리 프로그램을 
설명해드릴게요.

정말 간단합니다. 3단계예요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1) 초기 건강검진 (처음 1회)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

처음에 한 번 저희 시설에 오시면
혈액검사, 혈압 측정, 체성분 분석을 해요.

소요 시간? 
모니카님은 편하게 앉아있으면 
저희가 다 해드려요.

걸리는 시간은 딱 10분이에요."
```

**포인트:**
- "정말 간단" 먼저 예고
- "3단계" = Chunking (인지부하 -70%)
- "10분이에요" = 구체적 시간 (불안감 제거)
- "편하게 앉아있으면" = 시니어의 신체 불안 제거

[침묵 2초]

### 📞 상담사 대사 3-2 (5분 30초)

```
"검진 결과가 나오면,

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2) 맞춤 관리 계획 상담 (1회)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

우리 전문가가 
모니카님 검진 결과를 보면서

- 모니카님이 어떤 음식을 먹으면 좋을지

- 혈압약을 어떻게 복용해야 할지

개별 계획을 만들어드려요.

이건 화상통화로 진행되고,
약 30분 정도 걸려요.

딱 한 번이에요."
```

- "결과를 보면서" = 데이터 기반 (신뢰성 ↑)

- "딱 한 번이에요" = 시간 부담감 제거

[침묵 2초]

### 📞 상담사 대사 3-3 (6분 00초)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3) 월 1회 진행 상황 확인

이 정도만 확인해요.

시간은 15분 정도고,

혹시 새로운 증상이 생겼으면
'아, 이건 이렇게 해볼까?'
이렇게 조정해드려요.

정말 간단하죠?"
```

## 💰 PHASE 4: 가격 + 기대감 (10-13분)

### 목표
- 가격을 "투자"로 인식 전환
- ROI 명확화 (병원비 대비)
- 의료진·신뢰감 강화
- 3개월 변화 기대감 극대화

---

### 📞 상담사 대사 4-1 (10분 00초)

```
"그럼 비용 이야기를 할게요.

월 40만원이에요.

'어, 비싸네'라고 
생각하실 수도 있죠."
```

**포인트:**
- 직접 언급 (거절 선제 차단)
- "비싸네라고 생각하실 수도" = 공감

[침묵 2초]

---

### 📞 상담사 대사 4-2 (10분 20초)

```
"근데 잠깐, 생각해보세요.

모니카님이 지금 혈압약을 먹고 계신데,

앞으로 병원에서 처방받을 때마다
얼마나 드세요?

대략적으로:
- 진료비: 3~5만원
- 약값: 2~5만원
- 검사비: 2~10만원

한 번 가면 7~20만원이 들어요.

그리고 만약에 
혈압이 높아져서 입원하면?

대략 100~500만원이 나와요.

우리 프로그램은 
그런 일을 안 당하게 하는 거예요."
```

**포인트:**
- 구체적 병원비 수치 (Loss Aversion 강화)
- "앞으로 계속" = 지속 비용 인식 유도
- "입원하면?" = 최악의 시나리오 제시

[침묵 2.5초]

---

### 📞 상담사 대사 4-3 (10분 50초)

```
"계산해보면:

우리: 월 40만원 × 12개월 = 년 480만원

병원비: 
- 정기 진료: 년 200~300만원
- 혹시 합병증: 년 500만원 이상
- 입원까지 가면: 1,000만원 이상

5년이면:
- 우리: 2,400만원
- 병원: 5,000~10,000만원

어느 게 현명한 투자일까요?

우리는 병을 '방지'하는 거고,
병원은 이미 생긴 병을 '치료'하는 거예요."
```

**포인트:**
- 수치 대비 (Contrast Effect)

## 💡 시니어 소통 TIP 5가지

### 1. 침묵은 "생각 중"이다
- 시니어는 인지 처리에 시간이 더 필요
- 2-3초 침묵 후에도 답변 없으면 "괜찮으시죠?"라고 확인

### 2. 반복은 무례가 아니다
- 같은 내용을 3-4번 반복해도 OK
- "다시 한 번 정리하면..."이라는 표현으로 자연스럽게

### 3. 구체화는 필수다
- "많은 분들" X → "800명 이상" O
- "빨리 좋아져요" X → "3개월이면 달라져요" O
- "싸요" X → "월 40만원인데 병원비보다 적어요" O

### 4. 감정은 가족을 통한다
- "당신의 건강" < "손자들과 지낼 에너지"
- "약 안 먹기" < "자유로운 일상"

### 5. 신뢰는 권위에 있다
- "전 8년 경력입니다" ← 항상 언급
- "저희 의료진" ← 전문가 강조
- "1,200명 고객" ← Social Proof
