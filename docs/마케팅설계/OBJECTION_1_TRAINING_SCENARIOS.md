# 이의 1 상조 의혹 — 실전 트레이닝 시나리오 (2026-05-21)

## 트레이닝 목표
상담사가 상조 의혹에 대해 **6단계를 정확히 적용**하면서 **욕망을 재점화**할 수 있도록 훈련

---

## 시나리오 1: 신혼 부부 (A 세그먼트)

### 상황 설정
- **고객**: 김민영, 33세 (신혼 2년차)
- **배경**: 크루즈닷에서 홍콩 신혼여행 4박 5일을 검색했음
- **Opening 후**: "이거 상조 아니에요?"라고 물어봄
- **신뢰도**: 20% (급락)

### 대화 스크립트 (상담사 관점)

#### Step 1: 자아 인정 & 공감
```
상담사: "네, 그런 느낌 많으시죠. 할부라고 하니까 상조 같은 느낌도 드실 수 있고..."

체크포인트:
✓ 고객 불안감을 '당연한 의심'으로 정상화
✓ 목소리 톤: 느리고 따뜻함
✓ "당신이 느끼는 게 합리적이에요"라는 메시지 전달
```

**상담사 내부 모니터**: 고객의 불안감이 "이건 사기다"에서 "이게 뭐하는 건지 잘 모르겠다"로 변함

---

#### Step 2: 프레임 전환 (핵심)
```
상담사: "근데 우리는 상조가 아니라 여행 회사예요. 
         상조는 '만약을 위해'인데, 우리는 '홍콩 신혼여행을 현실로'이거든요."

체크포인트:
✓ "상조가 아니다"가 아니라 "우리는 여행회사"로 선언
✓ 명확한 차이 제시 ("만약" vs "현실로")
✓ 상조를 언급하되, 우리 정체성으로 완전히 덮음
```

**상담사 내부 모니터**: 고객의 뇌가 "상조 불안" → "여행 회사"로 프레임 전환됨

---

#### Step 3: 욕망 재점화 (가장 중요)
```
상담사: "당신이 신청했던 그 홍콩 여행, 기억하시죠? 
         신혼 떨어져 있다가 처음 함께 떠나는 여행... 
         그걸 지금 당장 준비하는 거예요. 월 33K로."

구체적 장면 복원:
- "홍콩 빅토리아 항구 보는 데크"
- "배우자 손을 잡고 일몰 보기"
- "빅토리아 호텔 뷔페에서의 첫 낭만"

체크포인트:
✓ "그걸 현실로 만들어주는 게 우리의 일"이라는 명시
✓ 고객이 실제로 가고 싶었던 장면을 복원
✓ 상조는 언급하지 않음 (완전히 다른 공간에서 대화)

상담사 톤: 부드럽지만 확신 있음
```

**상담사 내부 모니터**: 고객의 눈빛이 변함 ("불안" → "기대")

---

#### Step 4: 신탁 구조 설명 (신뢰 회복)
```
상담사: "할부금도 신한은행 신탁으로 관리되니까, 
         상조처럼 회사 돈과 섞이지 않아요. 
         은행이 안전하게 보관하고 있어요."

추가 설명 (고객이 더 묻는다면):
- "상조는 회사가 당신의 돈을 가져가는 거지만"
- "우리는 신한은행 신탁계좌에서 보호받고 있다는 뜻이에요"
- "은행의 신용도가 우리보다 훨씬 높으니까요"

체크포인트:
✓ "안전하다"가 아니라 "은행이 관리"로 (제3자 신용)
✓ 상조와의 근본적 차이 명시
✓ 고객이 느끼는 불안감 = 정보 부족에서 오는 것임을 인식

상담사 톤: 전문가스럽고 담담함
```

**상담사 내부 모니터**: 고객이 "아, 그럼 다르네"라고 느낌

---

#### Step 5: 라이프스타일 투자
```
상담사: "월 33,000원으로 더 좋은 여행 경험을 누리는 거, 
         이게 투자 아닌가요? 
         인생의 첫 신혼여행에 투자하는 거."

비교:
- "상조는 '만약을 위한' 불안감 상품"
- "우리는 '지금 현재를 풍부하게' 하는 선택"

체크포인트:
✓ 고객이 자신의 생각으로 "이건 투자다"라고 결론내리도록 유도
✓ 불안감 → 희망감 전환
✓ "월 33K"를 큰 지출로 보지 않게 함 (신혼여행 가치에 비해)
```

**상담사 내부 모니터**: 고객이 긍정적 감정으로 의사결정하기 시작

---

#### Step 6: 클로징
```
상담사: "그래서 안심하셔도 괜찮아요. 준비할래요?"

톤: 자신감 있으면서도 강압적이지 않음
상담사의 내재적 신뢰가 고객에게 전달되는 순간
```

**상담사 내부 모니터**: 고객이 신뢰하고 의사결정 준비

---

### 실제 고객 반응 시뮬레이션

#### Q1: 고객 추가 질문 (Step 2 직후)
```
고객: "그런데 정말 상조 아니에요?"

상담사 답변 (○ 올바름):
"네, 완전 다른 거예요. 상조는 어떻게 되세요? 
 당신이 죽고 난 뒤를 대비하는 거죠. 
 우리는 당신이 지금 살아있을 때, 
 홍콩에서 배우자와 일몰을 보는 경험을 만들어드리는 거예요."

상담사 답변 (✗ 잘못됨):
"맞아요, 상조가 아니에요. 우리는 건강 검진도 제공하고..."
← 이렇게 하면 상조처럼 들림
```

#### Q2: 가격에 대한 추가 질문 (Step 5 직후)
```
고객: "그래도 월 33만원은 좀..."

상담사 답변 (○ 올바름):
"네, 그런 생각도 정상이에요. 
 그런데 당신이 홍콩 신혼여행 가려면 얼마가 필요해요? 
 호텔 4박에 400만원, 항공료 300만원... 
 월 33K로 그 모든 게 준비된다고 생각하면 
 실제로는 월 3,300원 수준의 투자인 거죠."

상담사 답변 (✗ 잘못됨):
"330만원은 비싼 게 아니에요."
← 반박처럼 들려서 고객 저항 유발
```

#### Q3: 배우자와 상의하고 싶다 (Step 6 직후)
```
고객: "좀 더 생각해보고 남편이랑 얘기해봐야 할 것 같아요."

상담사 답변 (○ 올바름):
"당연하죠. 근데 한 가지만 말씀드리고 싶은데, 
 당신이 남편분한테 '이거 봐, 우리 홍콩 여행 현실화할 수 있어'라고 말씀하면, 
 남편분도 당신을 따라갈 거예요. 
 당신의 설득력이 가장 강하거든요."

상담사 답변 (✗ 잘못됨):
"그럼 나중에 연락 드리겠습니다."
← 고객이 절대 응할 가능성 5% 이하
```

## 시나리오 2: 자녀 동반 부모 (B 세그먼트)

### 상황 설정
- **고객**: 박현수, 42세 (딸 13세, 아들 10세)
- **배경**: 아이들한테 크루즈 경험을 선물하고 싶음
- **Opening 후**: "이거 정말 상조 맞는 거 아니에요?"
- **신뢰도**: 15% (매우 낮음, 자녀 보호 본능)

### 대화 스크립트

#### Step 1: 자아 인정
```
상담사: "네, 그런 생각 많으시죠. 특히 아이들을 생각하는 부모님이시니까 
         조심스럽게 생각하시는 거 정말 좋아요."

포인트: 자녀 보호 본능을 긍정하면서 신뢰도 상승
```

#### Step 2: 프레임 전환
```
상담사: "근데 우리는 상조가 아니라 여행 회사예요. 
         상조는 '만약을 위해'인데, 
         우리는 '지금 아이들과의 추억을 만드는' 회사거든요."

아이들과의 차이점 강조:
- 상조: 부모가 죽고 난 뒤
- 우리: 아이들이 살아있는 지금, 함께 만드는 기억
```

#### Step 3: 욕망 재점화
```
상담사: "당신이 아이들한테 가고 싶다고 했던 그 크루즈, 기억하시죠? 
         아이들이 학교 친구들한테 자랑할 수 있는 그 경험...
         
         월 33K로 그 꿈을 현실로 만드는 거예요. 
         아이들이 평생 기억할 추억을."

구체적 장면:
- "아이들이 갑판에서 바다 보며 뛰노는 모습"
- "밤하늘의 별을 보며 '우와!'라고 외치는 아이들"
- "나중에 어른이 되어도 '엄마, 아빠 그때 기억나?'라고 묻는 아이들"
```

#### Step 4: 신탁 구조
```
상담사: "할부금은 신한은행 신탁으로 안전하게 보호받고 있어요. 
         상조처럼 회사가 당신의 돈을 가져가는 게 아니라, 
         은행이 관리하고 있다는 뜻이에요.
         
         당신이 아이들 교육비를 맡기듯이, 
         은행의 신용에 당신의 돈이 보호받고 있다고 보시면 돼요."
```

#### Step 5: 라이프스타일 투자
```
상담사: "월 33,000원으로 아이들의 영원한 기억을 만드는 거, 
         이게 투자 아닌가요? 
         
         사교육비로 생각하면 한 달 학원비에도 못 미치는데, 
         아이들이 평생 기억할 추억을 만드는 거죠."
```

## 실전 트레이닝 체크리스트

### 상담사 자가 진단

#### Step 1-2 연결 (전환 매끄러움)
- [ ] Step 1에서 불안감을 충분히 인정했는가
- [ ] Step 2로 자연스럽게 넘어갔는가 (위화감 없음)
- [ ] "상조가 아니라"가 아닌 "우리는 여행회사"로 선언했는가

#### Step 2-3 연결 (욕망 재점화)
- [ ] 상조를 계속 언급하지 않았는가
- [ ] 고객의 실제 여행 꿈을 명시했는가
- [ ] 구체적 장면을 3개 이상 복원했는가
- [ ] 고객의 눈빛이 변했는가 (불안 → 기대)

#### Step 3-4 연결 (신뢰 형성)
- [ ] Step 3의 긍정 감정을 유지하면서 Step 4로 전환했는가
- [ ] "안전하다"가 아닌 "은행이 관리"로 설명했는가
- [ ] 상조와의 근본적 차이를 명시했는가

#### Step 4-5 연결 (투자 관점)
- [ ] 신탁 구조에 대한 이해가 고객에게 전달되었는가
- [ ] 투자 관점으로 재설정했는가 (불안 → 희망)
- [ ] 고객이 자신의 생각으로 "투자다"라고 결론내렸는가

#### Step 5-6 연결 (클로징)
- [ ] 긍정 감정을 유지하며 클로징했는가
- [ ] 강압적이지 않았는가
- [ ] 고객의 준비 상태를 감지했는가

### 금지 표현 체크

**절대 금지 3가지**
- [ ] "의료진도 있으니까" ← 상조 프레임 강화 금지
- [ ] "할부금이 안전해요" ← "은행이 관리"로 대체
- [ ] "상조가 아니라고요" ← "우리는 여행회사"로 대체

**필수 표현**
- [ ] "우리는 여행 회사예요" (최소 1회)
- [ ] "당신이 신청했던 [여행 목표]" (구체적 장면)
- [ ] "신한은행 신탁으로" (은행의 신용)
- [ ] "투자" vs "보험" (가치 비교)

## Role-Play 실전 훈련

### 훈련 1: 신혼 부부 (15분)
```
역할 배치:
- 상담사: 훈련자
- 고객: 담당자 (또는 동료)

시나리오: 김민영, 33세, "이거 상조 아니에요?"

흐름:
1분: Opening (완료한 상태에서 시작)
1분: Step 1 (자아인정)
1분: Step 2 (프레임전환)
2분: Step 3 (욕망재점화 — 가장 중요, 구체적 장면 최소 3개)
2분: Step 4 (신탁구조 설명)
2분: Step 5 (투자 관점)
1분: Step 6 (클로징)
5분: 피드백 (상담사의 강점/약점 분석)
```

### 훈련 2: 중년 부부 (15분)
```
역할 배치: 동일
시나리오: 이준호, 52세, "음... 이게 상조 같긴 한데..."

특별 주의점:
- Step 2에서 "사후" vs "지금" 강조
- Step 3에서 유럽 크루즈 구체적 장면 (알프스, 피요르드, 베니스)
- Step 4에서 "인생의 마지막 시간"을 은연중에
```

### 훈련 3: 추가 질문 처리 (10분)
```
상황 1: "정말 상조 맞는 거 아니에요?" 재질문
상황 2: "가격이..."라는 가격 저항
상황 3: "배우자와 상의하고..."

각 상황에 대해 올바른/잘못된 답변 2가지 비교
```

## 성공 신호 & 실패 신호

### ✅ 성공 신호
- [ ] 고객의 눈빛이 변함 (불안 → 기대)
- [ ] 고객이 자신의 여행 꿈을 이야기하기 시작함
- [ ] "그건 투자네요"라고 고객이 말함
- [ ] Step 6에서 고객이 준비를 동의함
- [ ] 후속 질문이 상조 관련이 아니라 여행 관련 (언제 가나, 어디 먼저)

### ❌ 실패 신호
- [ ] 고객이 상조를 계속 언급함
- [ ] 고객의 불안감이 사라지지 않음
- [ ] "의료진"을 언급하자 고객이 "상조 맞네"라고 반응
- [ ] 고객이 "생각해보겠습니다"로 종료하려 함
- [ ] 상담사가 상조를 반복적으로 부정함 (부정의 악순환)
