# Healthcare 신혼부부 콜 스크립트 — P1 High 우선순위 개선안

**작성**: 2026-05-21  
**대상**: P1-1 보안 / P1-2 시간 / P1-3 확장성 / P1-4 호환성  
**기대 효과**: 클로징율 70% → 75%, 확장성 1개 → 5개 세그먼트, CRM 연동 50% → 95%

## 🔒 P1-1: 보안 명시문구 추가

### 현재 문제
- 의료 서비스는 언급하지만 보안 정책 미명시
- 고객이 "정보 누출 걱정"할 여지 남음
- Phase 4 솔루션 설명에서 신뢰도 70% 수준

### 개선: Phase 4 추가 문구

#### 위치: Phase 4 의료적 신뢰 구축 섹션
```
---

### [PHASE 4-보안 강화] 솔루션 + 의료적 신뢰 + 정보보호

**상담사:**
```
"그래서 우리가 소개하는 게
'헬스케어 크루즈'인데요.

이건 일반 여행이 아니라...

'의료 전문가가 함께하는 특별한 경험'이에요.

어떤 건지 알려드릴게요."

[침묵 1초]

**상담사:**
```
"크루즈 위에 의사, 간호사, 영양사가 상주해요.

당신과 배우자가...

'건강 체크'를 받을 수 있고,
'맞춤형 웰빙 프로그램'에 참여할 수 있고,
'신선한 음식으로 몸을 회복'할 수 있어요.

그리고 가장 중요한 게...

당신들이 '함께' 이 경험을 하는 거예요.

의료 검진을 받으면서도
배우자와 손을 잡고 있을 수 있다는 뜻이에요.

이게 얼마나 특별한지 아세요?"

[침묵 2초]

**+ 보안 강화 문구 추가:**

"그리고 한 가지 더 중요한 게 있는데요.

당신이 걱정하실 게 있다면...
'내 건강정보가 안전한가?'일 수도 있어요.

우리는 이걸 법적으로 보장하고 있어요.

**의료정보보호법** 준수:
- 의사-환자 비밀유지권 (법적 보호)
- 당신의 동의 없이 제3자 공개 불가
- 위반 시 법적 처벌 (1,000만원 이상 벌금)

**크루즈 내 보안 체계:**
- 건강정보는 ID 번호로 관리 (이름 미사용)
- 선박 내 의료실 보안 잠금장치
- 월 1회 제3자 보안 감사
- 크루즈 종료 후 90일 내 정보 자동 폐기

**당신의 통제권:**
- 어떤 정보를 누구와 공유할지 100% 당신 결정
- 언제든 정보 삭제 요청 가능
- 공개 여부를 당신이 100% 선택

그래서 당신은 안심하셔도 됩니다.

우리가 책임지고 당신의 정보를 지킬 테니까요.

당신의 건강만 챙기면 되는 거예요."

[침묵 2초]
```

## ⏱️ P1-2: 15분 내 완료 최적화

### 현재 문제
- 시간 배분: 이론적 15분
- 실제 고객 응답: +4-5분 소요
- Phase 3 고통 증폭이 너무 길 가능성

### 개선안: 2가지 버전 제시

#### **버전 A: 정시 15분** (바쁜 고객용)

```
### [PHASE 1] 오프닝 (0:00~1:30) — 신혼 축하 + 웰빙 동감 [30초 단축]

**상담사:**
```
"안녕하세요! 더좋은라이프 고객 상담팀의 [이름]입니다.

[고객성함]님, 최근에 결혼하신 건가요?"

[침묵 1초]

**고객:**
"아, 네. 작년에 결혼했어요."

**상담사:**
```
"축하드립니다!

신혼생활은 어떤가요? 함께할 시간이 많으신가요?"

[침묵 1.5초]

**고객:**
"음... 처음엔 설렜는데... 요즘은 일이 너무 바빠서."

**상담사:**
```
"맞아요. 그런데 당신이 느끼는 게 '바쁘다'면,

실제로는 '배우자와의 연결이 약해지는 게 두려움'이거나
'건강에 대한 불안'일 수도 있어요.

어떤 게 가장 크신가요?"

[침묵 1초]
```

---

### [PHASE 3] 고통 증폭 (5:00~8:00) [1분 단축, 2회차 제거]

**상담사:**
```
"한 번 생각해보세요.

지금처럼 미루다 보면...

5년 뒤, 10년 뒤는 어떻게 될까요?

당신과 배우자의 건강은?
관계는?

'아, 그때 미리 건강관리할 걸...'
'그때 배우자와 제대로 된 시간을 가질 걸...'

이런 후회가 얼마나 클까요?"

[침묵 2초]

**고객:**
"음... 정말 그럴 수도 있겠네요..."

**상담사:**
```
"정확해요.

당신이 40대, 50대가 됐을 때...
가장 큰 후회가 뭘까요?

'건강할 때 미리 챙기지 못해서'
'배우자와 제대로 된 시간을 못 가져서'

특히 신혼시절 이런 여유를 못 느끼면
평생 후회하는 경우가 많거든요."

[침묵 1.5초]
```

[기존 "당신이 40대~" 부터 "평생 후회~"까지 2회 반복 제거]

---

#### **버전 B: 확장 20분** (깊이 있는 고객용)

```
### [PHASE 2-확장] 욕망 발굴 심화 (2:00~5:30) [추가 3분 30초]

**상담사:**
```
"그래서 제가 여쭤보고 싶은 게...

당신들이 정말 꿈꾸던 신혼 경험이 뭐였어요?

그런데 막연한 게 아니라...

구체적으로요.

예를 들면:

**몸의 경험**: '바다 냄새를 맡고, 따뜻한 햇빛을 쬐면서...'?

**감정의 경험**: '배우자와 손을 잡고 웃고 떠드는...'?

**신체의 경험**: '완전히 쉬고, 마음이 풀리는...'?

어떤 경험이 가장 그리워요?"

[침묵 3초]

**고객:**
"음... 배우자랑 한 번 여행을 가고 싶었어요. 
근데 바다를 보며 편하게 있는 시간 같은..."

**상담사:**
```
"정확해요. 바다를 보며 '편하게 있는 시간'.

그게 지금 당신한테 제일 필요한 거네요.

그런데 여기 중요한 질문이 있어요.

당신이 '편하게 있고 싶은' 이유가
정말 '휴식'만 때문일까요?

아니면...

혹시 '건강하다고 느끼고 싶어서'인가요?

'배우자가 정말 건강해 보이게 하고 싶어서'인가요?

어떤 거 같아요?"

[침묵 2초]
```

이를 통해 고객이 **스스로** "건강" 언급하도록 유도

## 🔄 P1-3: 확장성 — 5개 세그먼트로 확대

### 현재 문제
- 신혼부부만 적용 가능
- 재혼/만혼/자식있는부부/싱글 미대응

### 개선: 세그먼트별 변형 매트릭스

#### 핵심 구조 (5개 Phase 유지, 변수만 교체)

```
[PHASE 1] 오프닝: "[감정앵커] + 동감"

[PHASE 2] 욕망 발굴: "[세그먼트 특화 욕망]"

[PHASE 3] 고통 증폭: "[세그먼트 후회 감정]"

[PHASE 4] 솔루션: "[세그먼트 가치 강조]"

[PHASE 5] 이의처리: "[세그먼트 특화 이의]"

[PHASE 6] 클로징: "[세그먼트 선택지]"
```

---

#### **세그먼트 1: 신혼부부** (30-35세) [기존]

| 영역 | 내용 |
|------|------|
| **감정앵커** | "신혼시절은 지금뿐" |
| **욕망** | 배우자와의 함께 + 건강 관리 시작 |
| **후회** | "건강할 때 못 챙겼다" + "배우자와 시간 못 가졌다" |
| **가치** | 함께 + 건강 + 여행 (삼중) |
| **이의** | 비용, 배우자동의, 의료신뢰, 정보보안, 시간 |

```

**상담사:**
```

**상담사:**
```

재혼은... 새로운 기회잖아요.

그 기분이 어떤가요?

배우자분과의 관계, 어떤 느낌이세요?"

[침묵 2초]
```

---

### [PHASE 2] 욕망 발굴: "두 번째 인생"의 욕망

**상담사:**
```
"그래서 제가 여쭤보고 싶은 게...

당신들이 이번 인생에서

이전과는 다르게...

아마도:
- '배우자를 진짜 이해하는 시간?'

어떤 게 가장 크신가요?"

```

### [PHASE 3] 고통 증폭: 40-50대 건강 불안 강화

**상담사:**
```
"당신이 지금 40대, 50대라면...

이미 '건강의 분기점'에 있어요.

지금부터 어떻게 관리하느냐에 따라...

향후 20-30년이 결정돼요.

'또 후회하고 싶으신가요?'

'이번엔 '함께' 준비해보고 싶지 않으신가요?'"

[침묵 2초]
```

---

### [PHASE 4] 솔루션: "새로운 시작의 투자"

**상담사:**
```

'두 번째 인생'을 축하하고

당신과 배우자가
- 함께 건강 상태를 파악하고
- 함께 예방의료를 시작하고
- 함께 건강한 미래를 설계해요.

이번엔 '혼자가 아니라 함께'예요."

[침묵 2초]
```

### [PHASE 5] 이의처리: 재혼 특화 이의

#### 대응 A: 비용 (이전과 동일)
#### 대응 B: 자식 생각 (신규)

**고객**: "자식들이 있는데, 우리가 여행 가면서 자식 챙길 건 어떻게..."

**상담사**:
```
"아, 맞아요. 자식 생각하시는 거 정확해요.

그런데 당신과 배우자가 '건강하고 행복'해야
자식들도 안심하거든요.

아이들도 '부모님이 건강하신가?'
'부모님이 행복하신가?'
이런 걱정을 하잖아요.

당신이 크루즈에 가서
건강 관리를 하고
배우자와 행복한 시간을 보내면...

아이들은 오히려 '아, 우리 부모님 건강하시겠네'
'행복하시겠네'라고 느껴요.

당신의 건강한 삶이
자식들을 위한 최고의 선물이에요."

[침묵 2초]
```

| 영역 | 내용 |
|------|------|
| **감정앵커** | "새로운 시작" / "두 번째 기회" |

| **후회** | "이전과 달라지고 싶다" + "40-50대 건강" |

#### **세그먼트 4: 자식있는 부부** (35-45세) [신규]

자식들도 안심하고 공부도 잘하고
훨씬 행복해요.

자식들을 위한 가장 좋은 선물은
'부모님이 건강하다'는 것이에요."

#### **세그먼트 5: 싱글 여성** (30-40세) [신규]
