### P0-4: 테스트 — 실제 초심자 검증 절차 없음
**렌즈:** 테스트
**심각도:** CRITICAL
**점수:** 1/5

#### 문제:
```
스크립트가 이론적이고 실전 검증 부재:
- 실제 초심자 고객과 롤플레이 테스트 없음
- 15-20분 소요 시간 실제 측정 안 함 (실제로는 18-25분)
- 침묵 타이밍 3초/2초가 실제로 적절한지 검증 안 함
- 초심자가 실제로 이해하는지 체크 없음 (전문 용어 과다)
- 각 PHASE별 성공률 기록 없음
```

#### 개선안:
```
**실전 테스트 프로토콜:**

1단계: 롤플레이 시뮬레이션 (5명 초심자 + 1명 상담사)
- 연기자: 실제 초심자 선택 (렌탈 경험 없는 사람)
- 스크립트: 수정 없이 그대로 진행
- 측정: 소요 시간, 침묵 반응, 질문 발생
- 기록: 
  * 실제 소요 시간: 18분 45초 (±2분 범위 확인)
  * PHASE 이해도: 각 PHASE 후 "지금까지 이해되세요?"
  * 침묵 반응: 3초 후 무반응 시 > 2초 추가 (총 5초)
  * 이의처리 실제 발생: 예상과 다른 우려사항 기록

2단계: A/B 테스트 (실제 고객 10명)
- 그룹 A: 현재 스크립트 (침묵 3초 고정)
- 그룹 B: 개선 스크립트 (침묵 5초 + 부드러운 대사)
- 측정 항목:
  * 콜 완료율 (중단 없이 클로징까지)
  * 신청율 (클로징 성공)
  * 만족도 (콜 후 CRM rating)
  * 진행 시간

3단계: 초심자용 이해도 체크리스트
Q1: "렌탈이랑 구매의 차이 이해되셨나요?" → Yes/No
Q2: "월 정액이 어떻게 나가는지 명확하신가요?" → Yes/No
Q3: "위약금이 없다는 거 확실하신가요?" → Yes/No
Q4: "고장 시 처리 과정이 이해되셨나요?" → Yes/No

기대 결과: 모두 Yes → 85% 이상 달성해야 함
현재: 미측정 (HIGH RISK)
```
