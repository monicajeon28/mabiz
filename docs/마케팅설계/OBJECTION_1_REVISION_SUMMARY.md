## 핵심 변경 사항

### 6단계 재설계

| 단계 | 기존 | 새로운 | 개선점 |
|------|------|--------|--------|
| Step 1 | 감정인정 (기본) | 감정인정 + 정상화 (강화) | 불안감이 정상임을 명시 |
| Step 2 | Socratic Method (질문형) | Complete Frame Reframing (선언형) | 불안 → 프레임 즉시 전환 |
| **Step 3** | **사후 vs 현재 비교** | **욕망 재점화 (구체 장면)** | **상조 최소화, 여행 꿈 복원** |
| Step 4 | Social Proof (타인) | Reassurance + Authority (은행) | 신탁구조 명시, 제3자 신용 |
| Step 5 | 없음 | 라이프스타일 투자 (새로 추가) | 불안감 → 희망감 전환 |
| Step 6 | 없음 | 명확한 클로징 신호 (새로 추가) | 행동 유도 |

## 자주 묻는 질문 (FAQ)

### Q1: 왜 욕망 재점화가 중요한가?
```
A: 상조 의혹은 고객의 "불안감"에서 나온다.
   불안감을 제거하려고 하면 "왜 불안한가?"를 강조하게 되어,
   더 불안감이 커진다.
   
   반대로 욕망을 복원하면,
   고객의 뇌가 자동으로 "상조"에서 "여행"으로 이동한다.
   
   심리학: Narrative Transportation
   고객이 '여행 장면'을 상상하는 순간,
   상조 불안감은 자동으로 사라진다.
```

### Q2: 신탁 구조가 왜 상조와 다른가?
```
A: 상조 사기의 핵심 메커니즘
   → 회사가 고객의 돈을 가져갔다가 
     회사가 망하거나 사라진다.
   
   우리의 신탁 구조
   → 고객의 돈이 신한은행 계좌에서 순수 보관
   → 회사가 손댈 수 없음
   → 은행의 신용도로 보호
   
   따라서 "은행이 관리"는 상조와의 근본적 차이를 명시하는 것.
```

### Q3: 왜 의료 서비스 언급을 금지하나?
```
A: 상조의 마케팅 포인트가 정확히 "의료 서비스"
   - 상조도 건강검진 강조
   - 상조도 의료진 네트워크 강조
   
   따라서 우리가 의료 서비스를 강조하면,
   고객의 뇌가 "아, 상조처럼 보험성 상품이네"라고 인식한다.
   
   대신 우리는 "여행 회사"로 정체성을 명확히해야 한다.
```

### Q4: 각 세그먼트별로 다르게 해야 하나?
```
A: 절대적으로 그렇다.

세그먼트별 욕망이 다르기 때문:
- 신혼: 첫 신혼여행의 낭만
- 가족: 아이들과의 추억
- 중년: 부부 둘의 시간
- 시니어: 건강할 때 함께하는 마지막

각 욕망을 정확히 복원해야 전환율이 높다.
```

### Q5: Step 3에서 어떤 장면들을 구체적으로 언급해야 하나?
```
A: 고객이 실제로 관심 있었던 여행지를 반드시 포함:
   - 이미 크루즈닷에서 검색한 기록 (CRM에 있음)
   - 고객이 처음 콜에서 언급한 여행지
   - 고객의 나이/세그먼트에 어울리는 여행지
   
   구체 장면 3개 예시:
   - "빅토리아 항구에서 일몰"
   - "배우자 손을 잡고"
   - "별 가득한 밤하늘"
   
   추상적 표현 피하기:
   × "좋은 경험"
   × "추억 만들기"
   × "여행"
   
   구체적 표현:
   ○ "홍콩 빅토리아 항구 갑판에서 일몰"
   ○ "유럽 피요르드를 배경으로 한 자정 태양"
   ○ "아이들이 바다 보며 뛰노는 모습"
```
