**목표**: 상조 의혹 95% 제거, 클로징율 85-95%
**훈련 기간**: 5일 (Day 1 교육 + Day 2-3 Role-Play + Day 4-5 실전 모니터링)

## 📋 Day 1: 교육 (30분)

### 핵심 메시지 학습

#### 1. 회사 정체성 명확화
```
❌ 틀림: "상조 회사가 아니에요"
✅ 맞음: "더좋은라이프 회사예요"

차이:
- 첫 번째는 '부정' (상조를 다시 언급)
- 두 번째는 '정체성' (회사명으로 완전 전환)
```

#### 2. 월 27,000원의 의미
```
❌ 틀림: "여행 준비금이에요"
✅ 맞음: "라이프 업그레이드 선택이에요"

구성:
- 크루즈 여행 (꿈)
- 의료 서비스 (건강)
- 함께의 시간 (관계)
= 라이프 전체의 확장
```

#### 3. 신탁 구조의 의미
```
❌ 틀림: "할부금이 안전해요"
✅ 맞음: "신한은행이 신탁으로 관리해요"

상조와의 차이:
- 상조: 회사가 직접 관리 (자금 유용 위험)
- 우리: 은행이 신탁 (상조와 근본적으로 다름)
```

## 🎭 Day 2-3: Role-Play 훈련 (90분)

### 시나리오 A: 신혼부부 (30분)

#### 상황 설정
- 고객: 32세 남성, 신혼 1년
- 대사: "이거... 상조인가요?"
- 심리: 신혼여행 꿈 vs 할부 불안감

#### Role-Play 스크립트

**상담사**: (느린 톤, 낮은 목소리)
"네, 상조 같은 느낌 들 수 있죠. 장기 할부라고 하니까요."

**고객**: "그럼 이게 뭐예요?"

**상담사**: (회사명 명시 — 절대 핵심)
"우리는 상조 회사가 아니라 '더좋은라이프' 회사예요. 
이름부터가 다르잖아요."

**고객**: "근데 상조처럼 월마다 내야 하는데..."

**상담사**: (욕망 재점화)
"당신이 신청했던 홍콩 신혼여행, 기억하시죠?
그걸 지금 당장 준비하는 거예요.
월 27K로."

**고객**: "그런데 이게 정말 안전한가요?"

**상담사**: (신탁 설명 — 신한은행 명시)
"할부금은 신한은행 신탁으로 관리돼요.
우리 회사가 만지지 못하는 거죠.
은행이 안전하게 보관하고 있다는 뜻이에요."

**고객**: "그래도 좀..."

**상담사**: (투자관점 전환)
"월 27,000원으로 더 좋은 여행 경험을 누리는 거,
이게 투자 아닌가요?
인생의 첫 신혼여행에 투자하는 거.
결국 당신들의 선택이에요."

**고객**: "그럼 준비해볼까요?"

**상담사**: (클로징)
"네, 안심하셔도 괜찮아요.
준비해볼까요?"

## 🎓 추가 교육: 심리학 원리

### Step별 심리학 기법

#### Step 1: Validation (인정 + 정상화)
```
원리: 고객의 감정을 인정하면, 저항이 줄어든다
효과: 신뢰도 +10%

고객 심리: "상담사가 내 의심을 인정해줬다"
→ 방어심 낮아짐
→ 다음 메시지에 열려 있음
```

#### Step 2: Identity Shift (정체성 변화)
```
원리: 회사명 자체가 정체성을 결정한다
효과: 프레임 완전 전환

고객 심리: "상조는 상조인데, 더좋은라이프는 다른 회사네?"
→ 상조 프레임 탈출
→ 새로운 정체성 인식
```

#### Step 3: Umbrella Concept (포괄적 개념)
```
원리: 상조를 부정하지 말고, 더 큰 개념에 포함시킨다
효과: 모든 의심 통합 해결

고객 심리: "아, 상조는 그 안의 일부고, 전체는 라이프구나"
→ 상조에 대한 경계 해제
→ 전체 라이프 개념으로 확장
```

#### Step 4: Third-Party Trust (제3자 신뢰)
```
원리: 회사 신용이 약하면, 은행 신용을 활용한다
효과: 신뢰도 +40%

고객 심리: "상조는 위험하지만, 은행은 신뢰할 수 있다"
→ 신한은행 신탁 = 상조와의 근본적 차이
→ 안전성 확보
```

#### Step 5: Self-Generated Content (고객의 결론)
```
원리: 상담사가 강요하는 게 아니라, 고객이 스스로 깨닫게 한다
효과: 저항 최소화, 긍정 감정 강화

고객 심리: "내가 스스로 선택한다"
→ 강압감 없음
→ 주도권 있음
→ 클로징 확률 높음
```

#### Step 6: Commitment (약속)
```
원리: 명확한 행동 신호가 결정을 유도한다
효과: 클로징 완성

고객 심리: "상담사가 확신한다"
→ 불안감 해소
→ 행동으로 이행
```

## ⚠️ 절대 금지 표현 체크리스트

상담 중 다음 표현을 사용하면 **즉시 재교육**:

```
□ "상조가 아니에요"
   → 부정이 아닌 정체성 전환 필요

□ "의료진도 있으니까"
   → 상조도 제공하므로 절대 금지

□ "상조와는 달라요"
   → 계속 상조 언급, 프레임 유지

□ "할부금이 안전해요"
   → 신한은행 신탁 명시 필요

□ "우리는 여행 회사예요"
   → 일부만 강조, 통합 개념 부족

□ "건강검진도 받을 수 있어요"
   → 상조처럼 들림, 절대 금지

□ "은행이 안전하니까 괜찮아요"
   → 신한은행 신탁 기관 미명시
```

### Tip 2: 구체적 장면 묘사

```
"당신이 신청했던 홍콩 여행"
→ "당신과 배우자가 홍콩 빅토리아 피크에서 일몰을 보는 장면"
→ 더 구체적일수록 욕망 재점화 강함
```

### Tip 3: 고객의 언어 반복
```
고객: "월 27,000원?"
상담사: "네, 월 27,000원으로..."
→ 고객이 이미 수치에 수긍한 상태
```

### Tip 4: 질문 아닌 선택지 제시
```
❌ "준비하시겠어요?"
✅ "준비해볼까요?"
→ 함께 결정하는 느낌
```
