## 📚 10렌즈 Framework 개요

마비즈 CRM의 "10렌즈" 고객 분류 시스템은:
- 고객이 거절하는 "진정한 이유" 10가지를 정의
- 각 이유별 심리학 기반 전환 전략 제시
- 콜/SMS/CRM 자동화를 통한 맞춤형 판매 경로 구성

**핵심 철학**: "가격도, 상품도, 일관성도 아니라, '고객의 심리'에 맞춰서 판다"

## 🔄 사용 흐름

### **Step 1: 고객 분류** (CallNotes 입력 시)
```
고객 발화 → 4개 카테고리 키워드 분석 → Bayesian 점수 계산 → L1-L10 자동 분류
```

**예시**:
- "가격이 비싼데요" → L1 (40% 확률) + L6 (20% 확률)
- "준비가 복잡할 것 같은데" → L2 (80% 확률)
- "호텔이랑 뭐가 달라요?" → L3 (75% 확률)

### **Step 2: 맞춤 콜 스크립트 실행**
```
L1 → [L1_WORK 참고] → 6단계 콜 스크립트 실행
→ PHASE 1 공감 → PHASE 2 문제발견 → ... → PHASE 6 클로징
```

### **Step 3: 자동화 SMS 트리거**
```
L1 렌즈 분류 확정 → [SMS Day 0] 자동 발송 (30분 내)
→ [SMS Day 1] 옵션 제시 (오후 2시)
→ [SMS Day 2] 재접촉 (오전 11시)
→ [SMS Day 3] 최종 클로징 (오전 11시)
```

### **Step 4: 이의 대응**
```
고객 이의 발생 → [해당 렌즈 이의대응 섹션] 참고
→ 10단계 대응 구조 따라가기
```

### **Step 5: 성과 추적**
```
ContactLensClassification 테이블
→ confidenceScore (Bayesian)
→ CRM 태그 (#L1_가격민감, #CallPhase_3 등)
→ 성과 지표 (SMS 열람율, 콜 완료율, 전환율)
```

## 💡 각 렌즈별 핵심 포인트

### **L1 (40% → 목표 수치)**
- **절대금지**: 가격 비교, 경쟁사 폄하, 뜬금없는 할인
- **핵심**: 가격 아닌 "가치" 언급, 제3자 증명 (300명 고객), PASONA 100%
- **SMS톤**: 감정 기반, 안도→욕망→신뢰

### **L2 (45% → 목표)**
- **절대금지**: "쉬워요", "복잡하지 않아요", 공포 마케팅
- **핵심**: "체계적", "10분씩", "우리가 챙길게" 반복, 자동화 구체화
- **SMS톤**: 구체화+안심, 일정표+체크리스트+알림

### **L3 (50% → 최고 수준)**
- **절대금지**: "특별해요", "완전 달라요", 호텔 폄하
- **핵심**: 호텔 인정→구조적 차이→시각화→리조트 개념, 이미지/영상 필수
- **SMS톤**: 시각화 중심, 이미지+스토리

### **L4 (35% → 목표)**
- **절대금지**: "멤버십", "가입", "약정"
- **핵심**: "선택", "자유", "언제든 취소" 반복, 통제감 회복
- **SMS톤**: 자유도+존중

### **L5 (45% → 목표)**
- **절대금지**: "모든 사람이", "대부분이"
- **핵심**: "당신 같은 사람", 구체적 사례, 자기투영
- **SMS톤**: 사례+자신감

### **L6 (40% → 목표)**
- **절대금지**: "지금 안 하면 못 가요", 과도한 긴급성
- **핵심**: "지금이 최고 시점", 자리 한정, 가격 변동, 손실회피
- **SMS톤**: 실시간 자리+손실앵커

### **L7 (42% → 목표)**
- **절대금지**: "당신만 가세요", 가족 배제
- **핵심**: "함께 설득할게요", 구성원별 맞춤 키트
- **SMS톤**: 배우자/아이별 맞춤내용

### **L8 (50% → 최고 수준, 기존 고객)**
- **절대금지**: "다시 구독해야..."
- **핵심**: "연간 절약 $3000", "경험의 누적", 프리미엄 업그레이드
- **SMS톤**: 연간계산+기존경험회상

### **L9 (38% → 목표)**
- **절대금지**: "멀미 안 합니다", "문제 없어요", 공포 마케팅
- **핵심**: 의료진 상주, 응급실, 안전 기록 (0 건강문제), 국제기준 인증
- **SMS톤**: 의료+안전기준

### **L10 (85% → 최고)**
- **절대금지**: "어떤 배 원하세요?" (무한 선택)
- **핵심**: "A, B, C 중 어디가 제일 좋아요?" (삼중 선택), 빠른 클로징
- **SMS톤**: 최종확인+행동유도

## 📝 현장 가이드

### **영업 사원용 빠른 참고**

**통화 중 렌즈 판단 3초 규칙:**
1. 고객 발화 들으면서 키워드 캐치
2. 가장 해당하는 렌즈 1개 판단
3. 해당 렌즈 파일의 콜 스크립트 실행

**예시**:
- "가격이..." → L1 파일 열기 → PHASE 1 시작
- "준비 복잡..." → L2 파일 열기 → PHASE 1 시작
- "호텔이..." → L3 파일 열기 + 이미지 준비 → PHASE 1 시작

### **콜 후 즉시 조치**

1. CallNotes에 고객 발화 입력
2. CRM이 자동으로 렌즈 분류 (수동 수정 가능)
3. SMS Day 0 자동 발송 (30분 내)
4. Day 1-3 SMS는 자동 발송 예약

## 📞 고객별 대화 흐름도

```
고객 전화 수신
  ↓
초기 발화 듣기 (키워드 캐치)
  ↓
렌즈 판단 (L1-L10 중 1개)
  ↓
해당 렌즈 파일 → 콜 스크립트 실행
  ↓
PHASE 1 (공감) → PHASE 2 (문제) → ... → PHASE 6 (클로징)
  ↓
CallNotes 입력 + 렌즈 확정
  ↓
자동화 트리거:
  - Day 0: SMS 자동 발송
  - Day 1: SMS 자동 발송
  - Day 2: SMS 자동 발송
  - Day 3: SMS 자동 발송
  ↓
고객 반응 모니터링
  ↓
구매 또는 이의 발생
  ↓
구매시: 예약 완료
이의시: [해당 렌즈] → 이의대응 10단계 실행
```
