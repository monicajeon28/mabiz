## 🔴 Phase 1: 고우선순위 (L0, L2) - 예약 전환율 극대화

### Menu #47: 부재중 고객 재활성화 (L0 렌즈)

**심리학 기법**: Loss Aversion (손실회피) + Scarcity (희소성)

**구현 사항**:
1. **CRM 자동분류**
   - reactivation_segment: "3-6개월", "6-12개월", "1년+"
   - SMS_Day0_Sent, SMS_Day1_Sent, SMS_Day2_Sent, SMS_Day3_Sent 트래킹
   - Risk Score: reactivation_likelihood (0-100)

2. **SMS Day 0-3 자동화 시퀀스**
   ```
   Day 0: "서비스 담당자입니다. ○○님 마지막 크루즈 이후 벌써 6개월이 지났네요. 
          11월 특가: 카리브해 5박 $799 (정가 $1,299) - 마지막 3석 남음"
   
   Day 1: "고민 중이신가요? 계약금 $0으로 예약 가능합니다. 
          담당자와 5분 통화로 모든 불안 해소 가능"
   
   Day 2: "같은 배(USS Liberty) 타신 분들이 올해 재예약했습니다! 
          후기 영상 보기: [링크]"
   
   Day 3: "오늘만! $100 추가 할인 + 무료 객실 업그레이드. 
          지금 예약하세요 (자리 3개 남음)"
   ```

3. **API 엔드포인트**
   - GET /api/segments/reactivation → 세그먼트별 고객 리스트
   - POST /api/sms/reactivation-campaign → Day 0-3 자동 발송
   - PATCH /api/contacts/{id}/reactivation-status → 진행상태 추적

4. **대시보드**
   - 재활성화율 추적 (목표: 62-97%)
   - SMS 클릭율, 예약 전환율 실시간 모니터링
   - A/B 테스트: 메시지 3개 변형

**예상 효과**: 부재 고객 → 62-97% 재예약 전환

### Menu #48: 준비 불안 해소 (L2 렌즈)

**심리학 기법**: SPIN Selling (상황→문제→함의→필요→보상)

**구현 사항**:
1. **"준비 복잡도" 스코어링**
   ```
   visa_required: Y/N → +40점
   passport_days_left: <180 → +30점
   first_time_cruise: Y → +20점
   family_with_kids: Y → +20점
   health_concerns: Y → +15점
   
   Total: 0-125점 (80점 이상 = high_anxiety)
   ```

2. **AI 챗봇 기반 5단계 중재 질문 (SPIN)**
   ```
   상황(S): "해외 경험은 어떻게 되세요?"
   문제(P): "비자/여권 준비가 복잡하신 느낌 있으시나요?"
   함의(I): "준비 미흡시 탑승 불가 상황도 가능한데, 그게 걱정되세요?"
   필요(N): "준비 과정을 단계별로 안내받으면 도움 될까요?"
   보상(R): "우리가 비자/여권 체크리스트 제공 + 1:1 상담사 배정하면?"
   
   → 예약 확정율 +38-45%
   ```

3. **자동화 워크플로우**
   - Day 0: "준비 불안도" 자동 진단 (챗봇 5문)
   - Day 1: 세그먼트별 가이드 발송
     - 비자 필요: 대사관 연락처 + 체크리스트
     - 여권 갱신: 접수 링크 + 기간 계산기
     - 건강: 예방접종 정보
   - Day 2: "선배 탑승자 후기" 공유 (같은 상황)
   - Day 3: 1:1 상담사 배정 + 화상 상담 예약

4. **API**
   - POST /api/anxiety-score → 준비 불안도 계산
   - GET /api/preparation-guides/{category} → 비자/여권/건강 가이드
   - POST /api/sms/anxiety-sequence → Day 0-3 자동화

**예상 효과**: 불안감 해소 → +38-45% 예약 완료율
