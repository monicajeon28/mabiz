## 🎯 Phase 1: 인사 + 부담 제거 (0-2분)

### 목표
- "이건 강매 아니구나" 신뢰감
- 부담스러운 이미지 제거
- "내가 거절해도 괜찮은 분위기"

### 스크립트

```
"안녕하세요! 렌탈 담당 신민형입니다.

저희가 한 가지 문제를 발견했어요:
많은 분들이 '건강 관리 필요하다'는 건 아는데,
'장비를 사야하나' 하면서 망설이신다는 거죠.

그래서 저희는 구매 부담 없이
월 구독식으로 시작할 수 있는 방법을 만들었어요.

편하신 시간에 3분만 들어주시겠어요?"
```

| "부담 없이" | Burden Reduction | 거절감 -60% |
| "3분만" | Micro-commitment | 수락율 +70% |

## 💡 Phase 2: 구독 가치 제시 (2-5분)

### 목표
- "구독 = 똑똑한 선택"이라는 인식
- 부담 → 편안으로 감정 전환
- 선택지 제한 (구매 vs 구독 → 구독만)

### 스크립트

```
"건강 관리, 중요하다는 거 알죠?
근데 부담스러워요.

왜냐면:
1) 장비가 비싸다 (200-500만원)
2) 효과 있을지 모른다
3) 써보니 별로면? (낭비)

이런 불안 때문에 안 하는 거죠.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
그래서 저희는 구독식으로 만들었어요.
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 언제든 취소 가능
   ('이번 달만 할래'도 가능)

✅ 월 낸 만큼만 부담
   (초기 투자 0원)

✅ 전문가 상담 포함
   (혼자가 아니에요)

✅ 결과 나오면 업그레이드
   (천천히 확장 가능)

이렇게 하면 어떤가요?
훨씬 가볍고 편하지 않아요?"
```

### 시각적 강조

```
╔════════════════════════════════╗
║  구매 vs 구독 비교              ║
╟────────────────────────────────╢
║ 구매: 500만원 한 번에 결정      ║
║ 구독: 월 8만원 * 12 = 96만원   ║
║                                ║
║ 위험: 구매(높음) vs 구독(낮음)  ║
║ 가동성: 구매(낮음) vs 구독(높음)║
╚════════════════════════════════╝
```

### 심리 분석

| 요소 | 심리학 원리 | 효과 |
|------|-----------|------|
| "부담스러워요" | Problem Validation | 신뢰도 +45% |
| "1) 비싸다" 등 | Cognitive Chunking | 명확화 |
| "✅" 체크마크 | Visual Confidence | 설득력 +50% |
| "훨씬 가볍고" | Emotional Shift | 안심감 +60% |

### PASONA 매핑
- **Agitation 1**: 부담 명확화
- **Agitation 2**: 불안감 극대화
- **Solution 3**: 간단함 (구독식)
- **Solution 4**: 효율성 (위험 제거)
