## 📋 5박 크루즈 필수 짐 (간편 패킹)

### Category 1: 여행 증명 (여권 포함)
```
□ 여권 (유효기간 6개월 이상)
  └ 스캔본도 폰에 저장해두기

□ 항공권 (PDF로 폰에)
  └ 체크인 시 필요

□ 여행자보험증권 (선택)
  └ 의료 사용 시 필요

□ 신용카드 2장 (VISA/MASTERCARD)
  └ 배 위에서 스파/쇼핑 결제용
  └ AMEX는 배에서 먹히지 않을 수 있음

□ 현금 (₩100,000 정도)
  └ 배에서의 팁 (각 직원별 $1-2)
  └ 항구 도시의 소액 쇼핑

□ 신분증 (국내용)
  └ 비행기 탑승 시 필요할 수 있음
```

---

### Category 2: 기본 옷 (배 안은 쾌적하므로 가볍게)

**원칙**: "5박 5일 + 세탁" → 옷 4-5개만 있어도 된다

#### 낮 시간 (배 위에서)
```
□ 반팔 T셔츠 × 3개 (면 100%)
  └ 배의 에어컨은 약간 찬 편, 입으면 편함
  └ 색깔: 밝은 색 (배에서 햇빛 반사되므로)

□ 반바지 또는 짧은 바지 × 2개
  └ 수영장 + 갑판 다닐 때
  └ 긴 바지는 준비할 필요 없음 (배에서 안 입음)

□ 편한 신발 × 1쌍
  └ 캔버스 슈즈나 스니커즈 추천
  └ 샌들 × 1쌍 (객실/라운지용)
  └ 슬리퍼는 배에서 제공 (준비 불필요)

□ 가벼운 얇은 셔츠 × 1개
  └ 에어컨 찬바람용
```

#### 저녁 시간 (레스토랑/극장)
```
□ 여유 있는 드레스 또는 치마 × 1개
  └ "우아한 저녁" dress code (jean, sneaker 금지)
  └ 남자는 슬랙스 + 폴로셔츠 정도면 OK

□ 정장 구두 또는 로퍼 × 1신
  └ 저녁 식사 시 필수
  └ 매우 불편하다면 편한 구두도 가능
```

#### 특수 상황
```
□ 수영복 × 1개 (선택)
  └ 풀장/스파 갈 때
  └ 배에서 판매도 함 (₩30-50)

□ 내복/속옷 × 5개
  └ 배의 세탁 서비스가 있으므로 필요최소량

□ 양말 × 3-4켤레
  └ 어느 정도만 있어도 배에서 빨래 가능

□ 잠옷 × 1벌
  └ 배에서 제공하는 침구가 편하지 않으면 사용
```

---

### Category 3: 세면도구 (최소한만)

```
□ 칫솔 × 1개 + 치약 × 1개
  └ 배의 객실에 제공되지만 익숙한 걸 쓰려면 준비

□ 샴푸 & 린스
  └ 여행용 작은 포장 (100ml 이하)
  └ 배의 객실에도 제공됨

□ 로션/보디 크림 × 1개
  └ 배의 에어컨으로 피부가 건조해질 수 있음
  └ 여행용 작은 사이즈

□ 스킨 & 토너 (얼굴용)
  └ 여행용 사이즈 필수

□ 화장품
  └ 기본 화장품만 (더 가져갈 필요 X)
  └ BB크림 정도로도 충분

□ 선크림
  └ SPF 50 이상 (배 위에서 햇빛 노출 심함)

□ 생리용품 (필요 시)
  └ 충분히 준비 (현지 구매 어려울 수 있음)

□ 상처약 & 소화제
  └ 밴드, 파스, 감기약 정도만
  └ 배에 의료실이 있으므로 과다 준비 불필요

□ 칫솔 & 치약
  └ 객실에 제공되지만 익숙한 거 가져가기

□ 면도기 (남자용)
  └ 여행용 일회용 면도기 3-4개
```

---

### Category 4: 전자기기

```
□ 휴대폰 × 1개 + 충전기
  └ 배 위에서 와이파이 접속 가능 (유료)
  └ 해외 데이터는 비활성화 추천

□ 멀티탭 / USB 멀티 충전기
  └ 객실에 콘센트 2-3개 정도만 있음
  └ 여러 기기 충전할 때 필수

□ 이어폰 (선택)
  └ 라운지에서 영화/뮤직 감상용

□ 카메라 (선택)
  └ 휴대폰 카메라만 해도 충분
```

---

### Category 5: 기타 (사실 불필요한 것들)

```
❌ 드라이기: 배의 객실에 이미 제공됨
❌ 목욕용품 (비누/바디워시): 배에서 제공
❌ 수건: 배에서 충분히 제공 (스파 수건도 포함)
❌ 슬리퍼: 배에서 제공
❌ 우산: 배 안에서 필요 없음, 기항지에서는 주로 맑은 날씨
❌ 두꺼운 겨울옷: 배의 에어컨은 적당히 찬 정도
❌ 무거운 신발 여러 켤레: 한두 켤레만 있어도 충분
❌ 책이나 잡지: 배의 라운지에 이미 많음
```

## 💼 짐 용량 가이드

### 항공사 수하물 규정
```
대부분의 크루즈 항공사:
├─ 기본: 23kg 1개 + 수하물 1개 (7kg)
├─ 프리미엄: 23kg 2개
└─ 크루즈 함께 예약: 무료 추가 (25kg 1개)

💡 팁: 크루즈와 항공을 같이 예약했다면 
   25kg 추가 무료가 대부분! 확인해보세요
```

### 짐의 80/20 규칙
```
총 짐의 20%:
├─ 여행 증명 (여권, 항공권)
├─ 전자기기 (휴대폰, 충전기)
└─ 약/화장품 (꼭 필요한 것만)

총 짐의 80%:
└─ 옷 (그런데 5박이면 T셔츠 4개, 바지 2개로도 충분)

"배에서 자주 입는 것":
├─ 수영복 (풀장 시간: 2시간)
├─ 편한 반바지 (낮 시간: 4시간)
├─ 저녁 드레스 (저녁 식사: 1시간)
└─ 잠옷 (밤: 8시간)

→ 옷이 대부분인데도 T셔츠 4개, 바지 2개로 가능한 이유:
   배에는 빨래 서비스가 있어서!
```

## 🚨 흔한 실수 (체크리스트에서 제거)

| 실수 | 이유 | 대신 |
|------|------|------|
| 짐 너무 많이 | 배에서 쓸 일이 적어서 + 선실이 좁음 | 옷 4-5개, 신발 2켤레만 |
| 긴 바지 여러 개 | 배에서 거의 입지 않음 (배 위는 따뜻함) | 반바지 2개 + 저녁 드레스 1개 |
| 정장 신발 여러 켤레 | 저녁에만 입음, 무거움 | 정장 구두 1신 + 샌들 1신 + 운동화 1신 |
| 두껍고 무거운 외투 | 배의 에어컨은 약간 찬 정도 | 얇은 얇은 셔츠 1개 면충분 |
| 화장품 가득 | 5박인데 그렇게 많이 쓰지 않음 | 여행용 사이즈만 |
| 책/영화/게임 많이 | 배의 엔터테인먼트가 풍부함 + 휴식이 목표 | 없어도 됨 |
