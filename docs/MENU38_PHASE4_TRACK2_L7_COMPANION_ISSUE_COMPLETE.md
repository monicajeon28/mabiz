## 🧠 Part 2: 심리학 기초

### 적용 심리학 이론

#### 1. **Social Proof by Relationship** (관계별 사회적 증명)
- **원리**: "혼자" vs "함께"의 신뢰도 차이
- **Insight**: "배우자가 좋아하는 것 = 내가 좋아할 가능성 ↑"
  - 혼자 결정: 신뢰도 60% (내 판단)
  - 배우자 함께 결정: 신뢰도 85% (상호검증)
  - 친구 동반: 신뢰도 90% (3자 검증 + 재미 보증)

#### 2. **Shared Value** (공유 가치)
- **배우자**: "휴식의 질" → "부부 관계의 재발견"
- **자녀**: "여행 경험" → "가족의 추억화"
- **친구**: "개인 취향" → "함께하는 경험"
- **부모님**: "건강 관리" → "효도의 구체화"
- **Application**: 각 동반자별로 "혼자만의 이득" → "함께의 이득"으로 재프레이밍

#### 3. **Narrative Transportation** (이야기 전이)
- **원리**: 상대방의 성공 사례를 들으면 "나도 할 수 있다"는 확신 생김
- **실전**: 
  - "배우자 못 이기고 같이 온 남편의 후기"
  - "처음엔 휴대폰만 보던 아이의 변화"
  - "친구들이 마지못해 온 뒤 감사 인사"
  - "부모님 건강이 좋아진 사례"

#### 4. **Loss Aversion + Scarcity** (손실 회피 + 희소성)
- **배우자**: "부부 시간이 이번이 마지막일 수도"
- **자녀**: "아이가 더 커지면 이상적인 나이대 지날 수 있음"
- **친구**: "이번 항차가 마지막 50% 할인"
- **부모님**: "부모님 건강이 영원하지 않으니까"

#### 5. **Commitment & Consistency** (약속 일관성)
- "배우자도 한번 함께 상담받으면, 말은 안 하지만 이미 마음 결정됨"
  - 일단 콜에 응하는 것 = 50% 이상 의사결정 시작
  - 고객이 배우자에게 설득하는 것 = 고객이 팀의 일부가 되는 경험

## 🎤 Part 3: 완전 콜 스크립트 (5Phase + 심리학 맵핑)

### Phase 0: 오프닝 (2분)
**목표**: "동반자 이슈"를 고객이 스스로 꺼내게 유도

#### 스크립트
```
Agent: "안녕하세요! 크루즈 상담입니다. 
        혹시 크루즈에 관심이 있으신 이유가 뭔가요?"

Customer: "네, 한번 가보고 싶은데..."
           [여기서 "같이 갈 사람이...", "배우자가...", "아이들이..." 같은 언급 대기]

Agent: "알겠습니다. 혹시 혼자 가려고 하시는 건가요, 
        아니면 가족/친구들과 함께 가고 싶으신 건가요?"
       [동반자 타입 파악 질문]
```

**심리학 Application**: 개방형 질문 → 고객이 "장애물"을 먼저 말하게 유도

---

### Phase 1: 욕구 발굴 (5분)
**목표**: 동반자별 구체적인 "설득 불안"의 본질 파악

#### 배우자 타입
```
Agent: "배우자분이 설득이 안 되신다는 건가요?"

Customer: "네, 혼자가기도 싫고... 근데 남편이 바빠서..."
          [또는 "아내가 쓸데없는 돈이라고 해서"]

Agent: "알겠습니다. 그럼 배우자분은 왜 마음을 안 먹으시는 거 같아요?
        - 비용이 걱정되는 건가요?
        - 시간이 없어서인가요?
        - 아니면 크루즈 자체에 관심이 없으신 건가요?"
       [SPIN - Problem 질문: 3가지 옵션으로 구체화]

Customer: "음, 비용도 있지만 휴가 때문에 바빠요. 그리고... 
          배가 흔들리는 거 싫어해요."

Agent: "아, 그럼 실제로는 세 가지네요.
        1) 바쁜 스케줄
        2) 비용 부담
        3) 배멀미 걱정
        
        이 중에서 가장 큰 이유가 뭔가요?"
       [Implication 질문: 우선순위 파악]
```

**심리학 Application**: 
- SPIN (Situation → Problem → Implication → Need-Payoff)
- 고객이 "내 문제"라고 인식 → 우리의 솔루션이 더 와닿음

#### 자녀 타입
```
Agent: "아이들과 함께 가고 싶은데, 아이들이 싫어하신다는 건가요?"

Customer: "네, 요즘 아이들이 휴대폰만 본다고..."

Agent: "알겠습니다. 그럼 
        - 아이가 휴대폰을 버리지 않을 것 같아서?
        - 아니면 배 위의 활동이 재미없을까 봐서?
        - 아니면 학원/학교 빠지는 게 걱정되는 건가요?"
       [실제 우려사항 3가지 옵션화]
```

#### 친구 타입
```
Agent: "친구들과 함께 가고 싶은데, 친구들이 안 간다는 거네요?"

Customer: "네, 나만 크루즈에 관심이 있고... 친구들은..."

Agent: "알겠습니다. 그럼 
        - 친구들이 비용을 부담스러워해서?
        - 아니면 크루즈라는 상품 자체에 관심이 없어서?
        - 아니면 휴가 일정이 안 맞아서?"
```

#### 부모님 타입
```
Agent: "부모님과 함께 가고 싶은데, 부모님이 마음을 안 먹으시는 거네요?"

Customer: "네, 부모님이 건강이 안 좋으셔서... 배멀미도 싫어하시고..."

Agent: "알겠습니다. 그럼 
        - 건강이 걱정되는 건가요?
        - 배멀미 때문인가요?
        - 아니면 비용이 부담스러우신 건가요?"
```

**공통 심리학**: 우려사항을 구체화하면 → "해결 가능한 문제"로 변환

### Phase 2: 욕망 증폭 (7분)
**목표**: 동반자별 "함께 가는 것의 가치"를 고객이 느끼게 함

#### 배우자 설득 전략 (5가지 카피)

**전략 A: "일상 탈출 + 부부시간 회복"**
```
Agent: "배우자분이 바쁘신 이유가 뭐예요? 일이?"

Customer: "네, 항상 바빠요."

Agent: "알겠습니다. 크루즈를 이렇게 생각해보세요.
        배 위는 아무도 못 건드리는 공간이에요.
        - 회사는 못 들어옴
        - 손님은 못 들어옴
        - 아이들도 독립적으로 활동
        - 부부가 온전히 함께하는 3박 4일
        
        배우자분도 사실은 '쉼'을 원하시는 거 아닐까요?
        그런데 그걸 몰라서 '일'을 택하신 거?"
       [Implication: 불충족 욕구 깨우기]

Customer: "음... 맞긴 한데..."

Agent: "많은 부부들이 크루즈 다녀온 뒤 '아, 이게 바로 우리가 필요했던 거다' 
        라고 말씀해요.
        배우자분도 못 이기고 가면, 아마 같은 말씀 하실 거예요."
       [Narrative Transportation: 비슷한 사례로 확신 심기]
```

**전략 B: "당신도 휴식이 필요하다" (손실회피)**
```
Agent: "혹시 배우자분 마지막에 휴식 제대로 취하신 게 언제였어요?"

Customer: "음... 몇 년 전인가?"

Agent: "그럼 배우자분 건강은 어떤 상태예요?
        스트레스, 피로는?"

Customer: "음, 요즘 많이 피곤해하시더라구요."

Agent: "그럼 이렇게 생각해보세요.
        지금 크루즈 안 가는 비용: 0원
        크루즈 안 갔을 때 배우자 건강 악화 비용: ?
        (병원비, 감정 갈등, 관계 거리감)
        
        월 33,000원 (3만원의 피트니스비)으로 
        배우자의 건강을 다시 얻을 수 있다면?"
       [Loss Aversion: 지금 안 가는 게 더 비싼 것]
```

**전략 C: "배우자도 재미있을 것 같은데?"**
```
Agent: "배우자분 어떤 활동 좋아하세요?"

Customer: "음, 맛있는 거 먹는 거? 그리고..."

Agent: "그럼 배 위의 활동을 들어보세요.
        - 매일 다른 쉐프의 미슐랭급 요리
        - 밤마다 다른 문화공연 (뮤지컬, 재즈, 코미디)
        - 침실에서 침대에 누워 밤하늘 감상
        - 새벽 갑판에서 커피 마시기
        - 스파, 피트니스, 요가 (바다 위에서)
        
        배우자분도 가만히 있어도 충분히 즐겁지 않을까요?"
       [Affinity: 배우자의 관심사에 맞춘 설명]
```

**전략 D: "경제성: 월 33,000원 = 커피값"**
```
Agent: "비용이 걱정이시라면, 이렇게 생각해보세요.

        월 33,000원 = 커피 10잔
        
        대신에 배우자는
        - 매달 새로운 선박 정보 받음
        - 50-70% 할인 확보
        - 가족 여행 때마다 자동으로 저렴함
        
        5년 넘게 쓰면?
        33,000원 × 60개월 = 1,980,000원
        절약액 → 약 500-1,000만원
        
        투자 대비 수익률이 어떻게 되는 거 같아요?"
       [ROI 명확화: '손실'에서 '투자'로 프레임 변경]
```

**전략 E: "나이: 건강할 때 함께 가자"**
```
Agent: "배우자분 나이가 어떻게 되세요?"

Customer: "47살이에요."

Agent: "47살이면 지금이 정말 기회예요.
        이유는 두 가지.
        
        1) 지금부터 5년 뒤, 10년 뒤가 다릅니다.
           당신도, 배우자도.
           건강할 때 함께하는 게 가장 의미 있어요.
        
        2) 아이들도 5년 뒤 더 바빠져요.
           지금이 가족이 함께할 마지막 창 모르긴 몰라도.
        
        그래서 많은 부부가
        '나중에'가 아니라 '지금'을 택합니다."
       [Scarcity + Commitment: 시간의 희소성]
```

#### 자녀 설득 전략 (3가지)

**전략 A: "교육적 경험 + 견문확대"**
```
Agent: "아이가 지금 몇 살이에요?"

Customer: "12살이에요."

Agent: "12살이면 정말 좋은 나이네요.
        그 나이대 아이들이 크루즈에서 배우는 게 뭔지 알아요?
        
        1) 아시아 3국 문화 직접 경험
           - 싱가포르: 문화 융합
           - 홍콩: 도시 발전
           - 베트남: 역사와 현재
        
        2) 다양한 나이, 국가의 아이들을 만남
           - SNL 교육과 다른 실제 국제화 경험
        
        3) 가족과 함께 배우는 경험
           - 학원, 학교와 다른 '함께'의 의미
        
        아이는 나중에 '이때 부모가 나를 챙겼다'는 기억이 
        성인이 되어 가장 큰 자산이 돼요."
       [Narrative: 부모의 노력이 아이에게 미치는 영향]
```

**전략 B: "가족 추억 + 성장의 무대"**
```
Agent: "최근에 아이와 함께 뭔가 특별한 경험 하신 게 있어요?"

Customer: "음, 특별한 건... 별로 없어요."

Agent: "그럼 아이 입장에서는요?
        부모가 자신을 위해 애쓰는 모습이 보일 때,
        '아, 부모가 날 사랑한다'고 느껴요.
        
        크루즈는 그런 경험이에요.
        - 배를 타고 새로운 나라를 탐험
        - 부모가 설렜던 그 마음을 아이도 함께
        - 나중에 '그때 우리 가족이 함께했던...' 
          하는 추억이 어른 아이에게 가장 큼.
        
        학원, 학교는 부모의 '책임'처럼 느껴지지만,
        크루즈는 '부모의 사랑'으로 경험됨."
       [Emotional Bond: 가족 관계의 깊이]
```

**전략 C: "휴대폰 손뜨기 + 새로운 경험"**
```
Agent: "아이가 휴대폰을 놓지 않는다는 게, 실제로는 뭐가 문제예요?"

Customer: "공부도 안 하고, 밖에도 나가지 않고..."

Agent: "네, 그래요. 근데 아이 입장에서는?
        현실(학교, 학원)은 지루하고
        가상(게임, SNS)은 재미있으니까 거기 있는 거예요.
        
        크루즈 배 위에서는?
        아이가 '현실이 더 재미있다'고 느껴요.
        - 새로운 친구들
        - 매일 다른 나라
        - 미션 활동들
        - 부모와의 시간 (강제되지만 자연스러운)
        
        3박 4일이면?
        아이의 휴대폰 손 놓는 첫 경험이 될 거예요.
        그리고 '아, 현실도 이렇게 재미있을 수 있구나' 
        하고 깨닫게 됨."
```
