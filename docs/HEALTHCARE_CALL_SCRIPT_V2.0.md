## 🎯 Phase 1: 인사 + 신뢰감 형성 (0-2분)

### 목표
- 긍정적 첫인상
- 전문성 입증
- "나도 이 사람에게서 도움받을 수 있을 것 같다"는 신뢰감

### 스크립트

```
"안녕하세요! 크루즈 건강관리팀의 모니카입니다.

저희가 지난 3년간 500명 이상의 고객분들을 
도와드렸고, 특히 모니카님 연령대의 분들이 
가장 만족해하는 프로그램이 있어서 
말씀드렸습니다.

편하신 시간에 5분만 들어주시겠어요?"
```

### 심리 분석

| 요소 | 심리학 원리 | 효과 |
|------|-----------|------|
| "500명 이상" | Social Proof | 신뢰도 +35% |
| "연령대의 분들" | Segmentation | 자신감 +40% |
| "가장 만족해하는" | Narrative Transportation | 기대감 형성 |
| "편하신 시간" | 낮은 진입장벽 | 거절감 -45% |

### PASONA 매핑
- **Problem**: 건강 관리 필요성 암시
- **Solution 6**: 신뢰감 (전문가, 성공 사례)

## 💭 Phase 2: 욕구 발굴 (2-5분)

### 목표
- 현재 건강 상태 파악
- 통증점(pain point) 구체화
- 미래 상태 상향식 정의

### 스크립트 - 질문 1: 현재 통증

```
"요즘 건강 관리하시면서 
가장 힘든 부분이 뭔가요?

예를 들면:
- 피로감이 자꾸 생긴다던지
- 뭘 먹어야 할지 모른다던지
- 어떻게 관리해야 하는지 막연하다던지..."

[고객 답변 경청 - 최소 30초]

"그렇군요. 정확히 그런 분들이 많아요."
```

### 스크립트 - 질문 2: 미래 욕망

```
"그렇다면 만약 지금부터 
건강관리를 제대로 한다면,

3년 뒤 어떤 모습이 되고 싶으세요?

예를 들면:
- 에너지가 넘쳐나고
- 의사 갈 일이 거의 없고
- 자신감 있게 일상을 즐기는 그런 느낌?"

[고객 답변 경청 - 최소 40초]

"정말 좋은 모습이네요. 
저희가 정확히 그 방향으로 함께 가는 거예요."
```

### 심리 분석

| 질문 | 심리학 원리 | 효과 |
|------|-----------|------|
| "가장 힘든 부분" | Pain Point Articulation | 문제 의식화 |
| "3년 뒤" | Future Visioning | 기대감 형성 |
| "예를 들면..." | Anchoring | 답변 용이 +60% |
| 경청 30-40초 | Active Listening | 신뢰도 +50% |

### PASONA 매핑
- **Problem**: 구체적으로 정의됨
- **Agitation 1**: 통증점 강화
- **Agitation 2**: 미래 욕망 극대화

## 🎁 Phase 3: 패키지 설명 (5-10분)

### 목표
- 솔루션을 "진짜 간단"하게 느끼게 하기
- 복잡성 제거
- "나도 할 수 있을 것 같다" 느낌

### 스크립트

```
"그럼 저희 건강관리 패키지는 정말 간단해요.
3단계예요:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Step 1) 초기 건강검진
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
처음에 한 번 검진합니다.
혈액검사, 체성분, 생활 습관 등.
저희가 모두 준비해드려요.
모니카님은 와서 10분만 받으면 돼요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Step 2) 맞춤 관리 계획
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
검진 결과 바탕으로 전문가가
개인별 계획을 만들어드려요.
- 뭘 먹으면 좋을지
- 어떤 운동이 좋을지
- 언제 약을 먹을지

이것도 상담으로 25분이면 끝.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Step 3) 월 1회 진행상황 체크
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
매월 1번 온라인으로 
"요즘 어떻게 지내세요?" 
이 정도만 확인해요.
15분 정도.

이게 다예요.
따로 뭐할 게 없어요.
우리가 다 해드리는 거죠."
```

### 시각적 강조

```
┌─────────────────────────────────┐
│  복잡하게 느껴지나요?            │
│                                 │
│  검진(10분) → 상담(25분) → 체크(15분) │
│                                 │
│  총 월 1시간. 이게 다.          │
│  진짜 간단해요.                 │
└─────────────────────────────────┘
```

### 심리 분석

| 요소 | 심리학 원리 | 효과 |
|------|-----------|------|
| "3단계" | Chunking | 인지부하 -70% |
| "정말 간단" 반복 | Repetition | 설득력 +45% |
| 구체적 시간 | Specificity | 신뢰도 +40% |
| "이게 다" | Closure | 안심감 +55% |

### PASONA 매핑
- **Solution 3**: 간단함 (3단계)
- **Solution 6**: 신뢰감 (전문가 준비)

## 💰 Phase 4: 가격 + 기대감 (10-13분)

### 목표
- 가격을 "투자"로 인식
- ROI 명확화
- 기대감 극대화

### 스크립트 - 가격 제시

```
"비용은 월 30만원이에요.

'어, 비싸네'라고 생각하실 수도 있죠.

근데 생각해보세요.
병원 치료비가 얼마나 되나요?

예를 들어:
- 고혈압 치료: 월 20-30만원
- 당뇨 치료: 월 30-50만원
- 심장 질환: 월 100만원 이상

저희는 그런 병을 안 걸리게 하는 거예요.

결국 5년 뒤:
- 저희: 월 30만원 × 60개월 = 1,800만원
- 병원 치료: 500만원 이상 + 고통

어느 게 현명한 투자일까요?"
```

### 스크립트 - 기대감 강화

```
"그리고 모니카님이 지금부터 시작하면,
3개월 뒤에는 정확히 달라진 자신을 느낄 거예요.

우리 고객들이 그래요:
- '아, 에너지가 생겼다'
- '옷이 맞네?'
- '의사 갈 일이 없었네'

특히 40대 이상이 제일 빨리 체감해요.
3개월이면 정말 달라져요."
```

### 심리 분석

| 요소 | 심리학 원리 | 효과 |
|------|-----------|------|
| 구체적 병원비 | Loss Aversion | 동기 +60% |
| "5년 뒤" | Future Discounting | 의사결정 촉진 |
| "500만원" | Contrast Effect | 월 30만원이 싸 보임 |
| "3개월 뒤" | Temporal Immediacy | 기대감 +70% |
| "정말 달라져요" 반복 | Confidence Building | 확신도 +65% |

### PASONA 매핑
- **Solution 4**: 효율성 (투자대비 효과)
- **Solution 5**: 기대감 (3개월 변화)
- **Solution 1**: 손실회피 (병원비 방지)

## ✅ Phase 5: 클로징 (13-15분)

### 목표
- 즉시 행동 유도
- 거절 여지 제거
- 다음 단계 명확화

### 스크립트 - 최종 제안

```
"모니카님, 정리하면:

현재: 건강 관리가 막연하고, 막 피곤하고
목표: 에너지 넘치고, 자신감 있는 모습

3단계 간단한 방법으로 3개월이면 효과 봐요.
월 30만원인데, 병원비 아끼는 거죠.

더 건강한 인생을 위해
지금 시작하는 게 제일 좋아요.

다음 주 월요일부터 시작해도 되실까요?"
```

### 거절 처리 - 시나리오별

**시나리오 1: "생각해봐야겠는데요"**
```
"좋아요. 그럼 언제쯤?
다음 주 목요일? 금요일?
그때 다시 연락 드릴게요.

그 전에 혹시 궁금한 게 있으면
언제든 카톡 주세요."
```

**시나리오 2: "가격이 좀..."**
```
"아, 그렇죠. 
그럼 이렇게 하면 어때요?

첫 달은 50% 할인해드릴게요.
월 15만원부터 시작하고,
결과 나온 다음에 결정하세요.

어떤가요?"
```

**시나리오 3: "지금은 아니고..."**
```
"네, 알겠습니다.
그럼 6개월 뒤쯤 한 번 
다시 말씀드릴게요.

그 전에 영상이나 자료가 있으면
카톡으로 보내드릴까요?
미리 보시고 생각해보세요."
```

### 심리 분석

| 요소 | 심리학 원리 | 효과 |
|------|-----------|------|
| "정리하면" | Summarization | 인지 명확화 |
| "월요일부터" | Specific Date | 실행 가능성 |
| "다음 주" | Urgency + Timeline | 즉시성 형성 |
| "궁금한 게" | Open Door | 거절감 -50% |

### PASONA 매핑
- **Narrow Down**: 구체적 다음 단계
- **Action**: 즉시 약속(또는 재연락)
- **Solution 7**: 의미감 (더 건강한 인생)
