## 🧠 핵심 심리학 3가지

### 1️⃣ Mortality Salience (죽음 인식)
```
"부모님이 지금처럼 건강할까?" 
→ 고객의 뇌에서 자동으로 
   "시간이 제한적이구나" 인식
→ 즉시 행동 욕구 발생
```

### 2️⃣ Narrative Transportation (이야기를 통한 감정 전달)
```
"당신은 지금 몇 살이세요? 부모님은?"
→ 고객이 자신의 부모님을 자동으로 상상
→ 나이/건강을 계산
→ 미래의 후회를 현재로 가져옴
```

### 3️⃣ 침묵의 위력
```
상담사 100줄 말 ≠ 침묵 3초 + 고객 1줄 깨달음

침묵이 길수록:
- 고객이 스스로 문제 발견
- 자신의 깨달음이 가장 강력한 설득
- 설득력 극대화
```

## 📞 Q&A

### Q1: 어떤 Opening부터 시작하나요?
**A:** 모든 상담사는 Opening A' (기본값)부터 100% 마스터하세요. 이후 신호에 따라 B-E를 선택합니다.

### Q2: 침묵이 불편한데요?
**A:** 처음엔 불편합니다. 하지만 침묵 3초의 효과 = 상담사 말 100줄입니다. 시계 옆에 놓고 매 콜마다 초를 세세요.

### Q3: 신호 포착이 어려우면?
**A:** Week 3 훈련을 반복하세요. YouTube 신호 5가지를 노트에 써두고, 실제 콜에서 체크하면서 정확도 80%을 목표로 합니다.

### Q4: 3개월 만에 정말 +50% 효과가 나올까요?
**A:** 네. 효도 신호만 포착해도 57-60% 전환율입니다. 일반 Opening은 32%이므로 +25p 향상입니다.

### Q5: 기존 스크립트와 뭐가 다른가요?
**A:** 
- 기존: 상담사가 말 많음 → 고객이 거절
- 개선: 침묵 + 고객 자기발화 → 고객이 스스로 "해야겠다" 깨달음
