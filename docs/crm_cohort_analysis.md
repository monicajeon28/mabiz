## 📊 Cohort Analysis 개요

### 목표
**동시기 고객군(코호트)의 행동 패턴을 추적**하여 심리학 렌즈, SMS 자동화, 가격정책의 효과를 장기 측정

### 마비즈의 필요성

```
현재 문제:
- 월간 매출만 추적 (스냅샷)
- 어느 고객군이 재구매하는지 모름
- L6 타이밍 손실회피 효과 검증 불가
- SMS Day0-3 자동화가 정말 효과 있는지 모호

해결책:
→ 5월 신입 고객 100명이
  6월/7월/8월에 얼마나 재구매했는지 추적
  
→ SMS 자동화 그룹 vs 기존 방식 비교
  → 월별 재구매율 추세 파악
```

### Type 2: Revenue Cohort (매출 기준)

**정의:** 같은 월에 첫 구매한 고객들의 LTV 추적

```

├─ Month 3 (8월): $520 × 65명 = $33,800
├─ Month 4 (9월): $520 × 58명 = $30,160
├─ Total (12개월): $1,522.50 × 135명 = $205,537.50
```

## 📈 코호트 분석 메트릭

### 1. Retention Rate (유지율)

```
Retention(t) = 시간 t에 활성 고객 / 코호트 초기 고객

계산:
Week 2 Retention = 102명 (재구매) / 120명 (초기) = 85%
Week 4 Retention = 84명 / 120명 = 70%
Month 2 Retention = 62명 / 120명 = 52%

→ 각 기간의 "살아있는" 고객 비율
```

### 2. Churn Rate (이탈율)

```
Churn(t) = 1 - Retention(t)

Week 2 Churn = 1 - 85% = 15%
Month 2 Churn = 1 - 52% = 48%

→ 고객이 떠나는 비율
```

### 3. Revenue Per User (RPU)

```
RPU(t) = 누적 매출 / 초기 고객수

5월 코호트:
├─ Month 1 RPU: $57,000 / 120 = $475/고객
├─ Month 2 RPU: ($57,000 + $29,450) / 120 = $722.08
├─ Month 3 RPU: ($57,000 + $29,450 + $25,650) / 120 = $936.25
└─ Month 12 RPU: $125,700 / 120 = $1,047.50

6월 코호트 (L6 적용):
├─ Month 1 RPU: $64,125 / 135 = $475
├─ Month 2 RPU: ($64,125 + $39,520) / 135 = $767.00 ↑ (+6%)
├─ Month 3 RPU: ($64,125 + $39,520 + $33,800) / 135 = $1,026.67 ↑ (+10%)
└─ Month 12 RPU: $205,537.50 / 135 = $1,522.50 ↑ (+45%)
```

### 4. Cumulative Revenue (누적 매출)

```
그래프로 표현:

       │ 6월 코호트 (L6 적용) ╱╱     │

       │╱                ╱          │

       M1  M2  M3  M4  M5  M6  ... M12

6월 코호트는 6개월 만에 5월보다
```

## 🔄 Cohort 분석 자동화

### 월간 배치 작업 (매월 1일)

```typescript
// src/lib/analytics/cohort-analysis.ts

async function generateMonthlyCohortReport() {
  const months = ['2026-01', '2026-02', ..., '2026-09'];
  
  for (const month of months) {
    const cohort = await getCohortMetrics(month);
    
    // 각 코호트별 추적
    cohort.retentionCurve = calculateRetention(cohort);
    cohort.revenueCurve = calculateRPU(cohort);
    cohort.churnAnalysis = detectChurnPatterns(cohort);
    
    // 대시보드 저장
    await saveCohortReport(month, cohort);
    
    // 렌즈 효과 분석
    if (month >= '2026-06') { // L6 적용 시점
      const improvement = compareWithPreviousCohort(month);
      console.log(`${month} 코호트 개선율: +${improvement}%`);
    }
  }
}

// 실행
generateMonthlyCohortReport();
```

### Cohort 메트릭 계산 쿼리

```sql
-- 월별 신입 고객의 재구매율 (n개월 후)
WITH acquisition_month AS (
  SELECT 
    DATE_TRUNC('month', c."createdAt") AS cohort_month,
    COUNT(DISTINCT c.id) AS cohort_size,
    COUNT(DISTINCT CASE WHEN c.conversionStatus = 'SOLD' THEN c.id END) AS converted
  FROM "Contact" c
  GROUP BY DATE_TRUNC('month', c."createdAt")
)
SELECT 
  cohort_month,
  cohort_size,
  ROUND(100.0 * converted / cohort_size, 2) AS conversion_rate,
  ROUND(((100.0 * converted / cohort_size) - LAG(100.0 * converted / cohort_size) 
    OVER (ORDER BY cohort_month)) / LAG(100.0 * converted / cohort_size) 
    OVER (ORDER BY cohort_month) * 100, 2) AS yoy_improvement
FROM acquisition_month
ORDER BY cohort_month DESC;
```

## 🎯 Cohort 분석 의사결정

### 1. SMS 자동화 효과 검증

```
가설: SMS Day0-3이 재구매율 +33% 개선

검증:
Cohort (Pre-SMS): 52% retention @ M+1
Cohort (Post-SMS): 69% retention @ M+1
차이: +17%p (+33%)

결론: ✅ 가설 입증 → 계속 투자

예상 수익: 월 150명 × 17%p = 25명 추가 재구매
           25명 × $475 = $11,875/월 추가 매출
```

### 2. 가격 인상 영향 분석

```
가설: 가격 인상이 retention을 감소시킬까?

Before (5월-6월): $475 고정
After (7월+): $495 (+4.2%)

비교:
6월 cohort (M+1): 56% retention @ $475
7월 cohort (M+1): 57% retention @ $495

결론: ✅ 가격 인상 영향 미미
      → +4.2% 가격인상해도 retention 유지
```

### 3. L6 타이밍 렌즈 효과 측정

```
L6 도입: 2026-06-01부터

Before cohorts (5월 이전): 평균 M+2 LTV $722
After cohorts (6월 이후): 평균 M+2 LTV $767

개선: +$45 (+6.2%)
12개월 누적: +$45 × 12 = +$540 LTV 증가

월 150명 신입 × $540 = $81,000/월 추가 LTV
```
