## 💬 Day 0 메시지: 기대감 + 그루밍
**발송시점**: 콜 직후 (5분)  
**심리학**: Narrative Transportation + Expectation Building  
**PASONA**: Agitation → Solution (전환점)

```
안녕하세요 모니카님!

아까 얘기 나눴던 건강관리 프로그램 소개해드릴게요 👇

┌─────────────────────────────────────────┐
│ ✅ 초기 검진: 10분 (저희가 다 준비)      │
│ ✅ 맞춤 계획: 25분 (개인별 설계)         │
│ ✅ 월 1회 체크: 온라인으로 가능          │
└─────────────────────────────────────────┘

"정말 간단하네요" 라고 생각하셨죠? 😊

지금 우리 고객들 대부분이 
💪 3개월 후: 에너지 회복
💪 6개월 후: 의사 갈 일 거의 없음
💪 1년 후: 완전히 달라진 체질

이런 변화를 경험하고 있어요.

자세한 내용은 아래 링크를 눌러보세요:
[건강관리 프로그램 설명] 
https://cruise.health/programs/intro

혜선
```

**심리학 분석**:
- **Narrative**: "3개월 후 → 6개월 → 1년" = 시간 경과에 따른 변화 스토리
- **Social Proof**: "우리 고객들 대부분이" = 다수성 원칙
- **Expectation**: "완전히 달라진 체질" = 미래 상태 구체화
- **Low Friction**: 링크 1개 = 진입장벽 최소화

## 💬 Day 1 메시지: 솔루션 간단성 + 불안감 제거
**발송시점**: 콜 후 24시간  
**심리학**: Simplicity + Fear Reduction  
**PASONA**: Solution (복잡성 제거)

```
모니카님, 안녕하세요! ☀️

건강검진을 앞두고 
"복잡하지 않을까?" 걱정하시나요?

전혀 복잡하지 않아요! 👇

┌─────────────────────────────────────┐
│ 🏥 검진 준비물:                      │
│   - 신분증 1개                       │
│   - 그게 끝! (식사 금지 주의)        │

- **Social Proof**: 실제 고객 후기 "박현정님"
- **Benefit Emphasis**: "맞춤 계획이 효과의 핵심"
- **Scarcity**: "첫 달 상담료 50% 할인" (제한된 혜택)

## 💬 Day 2 메시지: 신뢰감 + 나이별 성공사례
**발송시점**: 콜 후 48시간  
**심리학**: Social Proof (Age-Matched) + Authority  
**PASONA**: Narrative (신뢰감 강화)

```
모니카님! 👋

혹시 어제 메시지 봤어요? 
아직 예약 안 하셨다면, 
나이 비슷한 분들의 성공사례 소개할게요 📈

┌──────────────────────────────────────┐
│ 👩 신혼부부 (30-35세) - 이지현님      │
│   • 목표: 임신 전 체질 개선          │
│   • 결과: 3개월 후 정상체중 달성     │
│   • 만족도: ⭐⭐⭐⭐⭐             │
│   → "다른 곳과 달라요"               │
├──────────────────────────────────────┤
│ 👨 4050 자녀있는가정 (45-50세)       │
│   • 목표: 만성피로 극복              │
│   • 결과: 6개월 후 에너지 회복       │
│   • 만족도: ⭐⭐⭐⭐⭐             │
│   → "회사 동료들에게 자랑했어요"     │

│   • 결과: 1년 후 혈당 정상화         │
│   • 만족도: ⭐⭐⭐⭐⭐             │

이게 바로 "효과"를 만드는 세 가지예요.

- **Authority**: "병원급 장비", "전문의"
- **Relatability**: "여름 전에 변화" (고객 관점의 기대)
- **Scarcity**: "이번 주까지" (마감 압박)

## 💬 Day 3 메시지: 배움의 가치 + 최종 클로징
**발송시점**: 콜 후 72시간  
**심리학**: Learning Value + Loss Aversion  
**PASONA**: Narrowing (최종 결정 촉구)

```
모니카님, 마지막 메시지입니다! 📬

지난 3일 동안 
"건강관리 진짜 필요하긴 한데..."
이렇게 생각했을 것 같아요. 맞나요? 😊

그렇다면 꼭 알아야 할 것! 💡

─────────────────────────────────────
❌ 건강검진만으로는 부족한 이유
─────────────────────────────────────

일반 검진 = 
"혈당이 높습니다" ← 여기까지만 말함

우리 건강관리 = 
"혈당이 높다 → 원인은 야식 → 
 야식을 줄여야 함 → 대신 이렇게 
 대체하세요 → 1주일 후 체크" ← 해결까지

이 차이! 여기가 바로 
"결과"가 나오는 지점이에요.

─────────────────────────────────────
🎯 모니카님을 위한 특별 오퍼
─────────────────────────────────────

✅ 첫 검진: 무료 (일반가 12만원)
✅ 맞춤 계획: 50% 할인 (일반가 8만원 → 4만원)
✅ 월 관리비: 5만원 (평생 유지)

이 가격은 오늘까지만 가능합니다.
내일부터는 일반가 적용됩니다.

왜냐하면, 우리가 이 프로그램을 통해
모니카님이 건강해지는 것을 봤으니까,
"꼭" 시작해주길 바라기 때문이에요. 🙏

[지금 예약하기 - 오늘까지만]
https://cruise.health/reserve/step1?today

아니면,
톡으로 예약 날짜만 말씀해주셔도 돼요!
시간을 우리가 맞춰드릴게요. ✨

화이팅! 💪

P.S. 
혹시 이 프로그램이 맞지 않는다면,
언제든 톡으로 말씀해주세요.
그럼 다른 옵션을 찾아드릴게요. 😊
```

**심리학 분석**:
- **Learning Value** (PASONA Solution 7): "검진" vs "건강관리" 차이 = 교육적 가치
- **Loss Aversion**: "오늘까지만" = 기회 상실 두려움
- **Scarcity + Urgency**: "내일부터 일반가"
- **Personal Reason**: "모니카님이 건강해지는 것을 봤으니까" = 진정성
- **Low Pressure Alternative**: "톡으로 예약" = 선택권 제공
- **Objection Handling**: "맞지 않으면 다른 옵션" = 신뢰감
