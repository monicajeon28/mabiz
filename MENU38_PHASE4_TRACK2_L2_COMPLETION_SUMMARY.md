### 3. CRM 자동분류 규칙 ✅

**분류 기준**:
```
IF (Q1 ∈ {처음} OR Q2 = {복잡할까봐}) AND Q4 ∈ {매우바쁜, 중간}
  THEN L2렌즈 할당 (Score: 0.65 이상)
```

**자동 식별 질문 4개**:
1. 해외 여행 경험? (처음 = 0.9점)
2. 여행 준비에 대한 느낌? (복잡할까봐 = 0.95점)
3. 여행 준비 주도권? (여행사 도움 받는 = 0.85점)
4. 시간 여유? (매우 바쁜 = 0.9점)

**페르소나 매핑**:
- P2 (업무 바쁜 신혼): L2 = 0.9
- P6 (50대 직장인): L2 = 0.85
- P9 (60대 은퇴): L2 = 0.95 ← 최고 우선순위
- P15 (글로벌 기업): L2 = 0.75

**산출물**:
