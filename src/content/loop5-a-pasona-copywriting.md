**산출물**: 세그먼트별 Day 0-3 메시지 20개 + A/B 변형 40개 (총 60개 메시지)

## 🎯 프레임워크 개요

### PASONA 6단계 매핑 (Day 0-3)

| 일자 | PASONA 단계 | 심리학 렌즈 | 주요 메시지 역할 |
|------|-----------|----------|-----------------|
| **Day 0** | P(Problem) + A(Agitate) | L6(손실회피) + L10(즉시구매) | 고통점 자극 + 긴급성 인식 |
| **Day 1** | S(Solution) + A(Agitate 강화) | L7(동반자) + L6(손실회피) | 해결책 제시 + 감정 강화 |
| **Day 2** | O(Offer) + N(Narrowing) | L10(즉시구매) + L8(재구매) | 가격/혜택 명시 + 한정성 |
| **Day 3** | N(Narrowing) + A(Action) | L6(손실회피) + L10(즉시구매) | 마지막 기회 + 즉시 행동 |

### 심리학 렌즈별 강조도

- **L6 (타이밍 손실회피)**: "지금이 아니면 영원히" → Day 2-3 강조
- **L7 (동반자 설득)**: "함께하는 특별함" → Day 1 강조
- **L8 (재구매/습관)**: "다시 돌아올 고객" → Day 0-1 강조
- **L9 (의료신뢰)**: "안전하고 건강한" → Segment별 리스크 플래그 맞춤
- **L10 (즉시구매)**: "오늘만 혜택" → Day 0, Day 2-3 강조

### **Message A-1: Day 0 (P + A 단계)**

#### 기본 메시지
```
김민준님, 결혼 3년차가 마지막 자유로운 시간이에요.

자녀 계획 전, 부부 둘만의 특별한 추억을 남길 기회.
지금 신청하면 신혼부부 특가 50만원 할인 + 커플 스파 무료.

[신청하기] (5분이면 완료)
```

**메시지 분석**
- **PASONA 단계**: P(결혼 후 자유상실) + A(마지막 기회) + O(50만원) + A(액션)
- **심리학 렌즈**: L6(지금이 아니면 영원히) + L10(오늘만 혜택)
- **문자 길이**: 85자 (1건)
- **톤**: 감정적 + 긴박감
- **CTA**: 즉시 신청 (5분 예상)

#### A/B 변형 - 공격형
```
"부부만의 마지막 여름, 이번이 아니면 못 해요.
아이 전 딱 3개월 남았습니다. 카리브 럭셔리 크루즈로 평생 추억을
당신의 손으로 만들어보세요.

[지금 신청]"
```

#### A/B 변형 - 감정형
```
"남편님, 아내가 가장 보고 싶은 경험이 뭔지 아세요?
당신과 함께 바다 위에서 별 보기.
신혼부부 특가 + 무료 로맨틱 디너.

[우리의 추억 만들기]"
```
