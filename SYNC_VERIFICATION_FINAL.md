## 🔍 확인 가능한 위치

### 1️⃣ 고객 관리 페이지 (`/고객 관리`)
- 194명의 고객이 모두 표시됨
- 출처별로 필터링 가능 (상품문의, 어필리에이트 등)
- 각 고객의 상세 정보 확인 가능

### 2️⃣ 고객 상세 정보 보기
```

상태: PROSPECT (관심), CUSTOMER (구매완료)
```

## 🛠️ 기술 사항

### 어떻게 연결됨?

1. **Supabase** (크루즈닷 백업 DB)
   - `User` 테이블 → 67명 구매 고객
   - `CruiseProductInquiry` → 7건 상품 문의
   - `AffiliateLead` → 100명 어필리에이트 리드

2. **Neon** (마비즈 CRM 운영 DB)
   - `Contact` 테이블에 194명 모두 저장됨
   - 각 고객의 출처 기록됨 (`channel` 필드)
   - 상품 정보 링크 가능함

3. **동기화 스크립트**
   ```bash
   npx tsx sync-cruisedot-to-crm.ts
   ```
   → 자동으로 모든 데이터 동기화
