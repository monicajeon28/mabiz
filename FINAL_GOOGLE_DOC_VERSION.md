# 부재중 고객 크루즈 120만원 지원 콜 스크립트 (클로징율 80% 최적화 버전)
**작성일**: 2026-05-20  
**대상**: OO월에 신청했으나 구매 결정을 못 한 부재중 고객  
**목표**: 클로징율 80% 이상 달성  
**기반**: 10렌즈 무한루프 검토 (신뢰도, 심리학, 클로징, 감정, 이의처리, 톤, 시간, 실행, 세분화, 안전)  
**전체 시간**: 10분 (충분한 고객 응답 시간 확보)

## 📞 OPENING (30초) - 신뢰있는 첫인상

```
"안녕하세요, 김OO님. 저는 크루즈닷 매니저 ___입니다.

OO월에 신청하셨던 거, 기억나세요?

실은요, 저희가 당신을 위해 
'120만원 할인 혜택'을 준비했어요.

지금 딱 10분만 시간 괜찮으실까요?
당신이 이게 어떻게 가능한지 알면
분명 관심 가질 거예요."

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 상담사 주의사항:
- 톤: 친근하지만 "침착하고 신뢰있게" (가볍지 않게)
- "좋은 소식"이 아니라 "준비했어요" (진정성)
- "딱 10분" (충분한 시간 신호)
```

**심리학 적용**: 희소성(10분) + 호기심(어떻게 가능?) + 신뢰(매니저 자기소개)

## 💭 욕망 찾기 (2분 30초) - 고객의 진정한 필요 발굴

### 【Step 1: 기초 정보 파악】

```
상담사: "먼저 한 가지 여쭤봐도 될까요?

OO월에 신청하셨을 때는, 
어떤 여행지를 생각하고 계셨어요?"

(고객 답변 경청, 반복)

상담사: "아, [여행지명]이군요. 
정말 좋은 선택이네요. 왜 그 여행지를 생각하셨어요?"

(고객이 깊이 있는 답변을 하도록 포즈 제공)
```

**상담사 팁**: 고객이 "음... 그냥"이라고 하면, "가족이랑? 부부가? 친구들이랑?" 유도

---

### 【Step 2: 세그먼트 파악】

```
상담사: "누구랑 함께 가고 싶으셨어요?"

(고객 답변)

상담사: "아, [가족/부부/친구]이군요. 
당신이 그 분들과 [여행지]에서 
가장 하고 싶은 게 뭐예요?"

(고객의 감정 깊이 파악)
```

**상담사 주의**: 답변에 따라 세그먼트 분류
- "아이들이랑" → 가족 세그먼트 (아래 ⓐ 참조)
- "남편이랑/와이프랑" → 부부 세그먼트 (아래 ⓑ 참조)
- "혼자" → 싱글 세그먼트 (아래 ⓒ 참조)

---

### 【Step 3: 미래상 구체화】

```

당신이 [여행지]의 [구체적 장소] 있고,
[고객이 말한 행동]을 하고 있어요.

그 순간 어떤 기분이 들 것 같아요?"

(침묵 5초 - 고객이 감정 표현하도록)
```

**상담사 팁**: 고객의 감정을 단순 반복 아니라 강화하기
- 고객: "신나요"
- 상담사 (❌ 나쁜 예): "신나시는군요"
- 상담사 (✅ 좋은 예): "그런 신남을 지금 경험할 수 있다면, 정말 좋지 않을까요?"

## 🎓 상담사 체크리스트

```
【콜 전 준비】
□ 고객 정보 확인 (이름, 신청월, 이전 상담 기록)
□ 현재 남은 객실 수 확인 (거짓 금지)
□ 7월까지 남은 기간 계산
□ 카톡 신청 링크 준비
□ 라이브방송 시간 확인 (금요일 오후 8시)

【콜 도중】
□ Opening: 침착하고 신뢰있는 톤
□ 욕망 찾기: 세그먼트 파악 (자녀? 부부? 싱글?)
□ 손실회피: 3개 함축 질문 + 침묵 유지
□ 해결책: 10만원 강조 (69만원 아니라)
□ 삼중선택: 양자택일로 강제 (①번 or ②번만)
□ 이의처리: 고객별 재프레이밍
□ 액션: 링크 클릭 확인

【콜 후】
□ 신청 확인 (Day 0, 2시간 후 SMS)
□ Follow-up 콜 일정 입력 (Day 1, 3, 7)
□ CRM에 고객 상태 기록
□ 성공/실패 이유 분석
```

## 💡 핵심 포인트 (반드시 기억)

1. **침착하고 신뢰있게** (가볍지 않게)
2. **"10만원"을 강조** ("69만원" 아니라)
3. **고객 말을 70% 경청** (상담사 말 30%)
4. **SPIN Implication 깊이** ("맞다"고 고객이 인정하게)
5. **양자택일 강제** (①번 or ②번, 거부 옵션 없음)
6. **침묵의 힘** (5초 이상 포즈 → 고객 발화)
7. **세그먼트별 맞춤** (자녀/부부/싱글 다른 라인)
8. **Day 1, 3, 7 Follow-up** (기억 70% 유지)
9. **이의 처리** (재프레이밍으로 거부 옵션 제거)
10. **감정 + 논리** (감정 70% + 논리 30%)
