# 부재중 고객 크루즈 멤버십 콜 스크립트 V5 (큰 뼈대)
## 전략: 신뢰 → 욕망 → 불안 → 해결책
## 목표: 2만 7천원 골드멤버십 판매 (50%+ 클로징율)
## 전체 시간: 8분

---

## 📐 V5 큰 뼈대 구조

```
【신뢰 Opening】(2분) - 고객의 욕망 파악 + 신뢰 구축
           ↓
【욕망 각성】(1분 30초) - 여행 꿈 감각화
           ↓
【불안 증폭】(2분) - 손실 이미지화 (구체 숫자)
           ↓
【해결책 Reveal】(1분) - 월 2만 7천원
           ↓
【Closing】(1분 30초) - 양자택일 강제
```

---

## STEP 0: 신뢰 Opening (2분)
### 목표: 신뢰 구축 + 고객의 욕망/문제 파악

```
【Phase 0-1: 신뢰 + 상황 확인】

상담사: "안녕하세요, 김OO님!

저는 크루즈닷 상담원 000입니다.

OO월에 저희 크루즈 신청해주셨었죠?

지금 2분만 괜찮으세요?"

(고객의 "네" 확인)

【Phase 0-2: 욕망 파악】

상담사: "그때 신청하셨을 때는
어떤 마음이 있으셨어요?

여행을 가고 싶으셨던 거죠?

어디를 생각하고 계셨어요?"

(고객 답변 청취)

상담사: "[여행지]이군요. 
왜 그곳이 좋으신가요?"

(감정 파악)

【Phase 0-3: 문제 인정】

상담사: "그럼 그때 신청 후에
왜 아직 못 가셨어요?

현실적으로 뭐가 제일 막혔어요?"

(고객이 문제를 스스로 언급)

상담사: "아, 그런 이유였군요.

사실 우리 고객들도 대부분 
그 이유 때문에 미루고 있어요.

당신만 그런 게 아니란 거죠."

(정상화 + 공감 = 신뢰)

【핵심 포인트】
✅ 이름 부르기 (친근감)
✅ 신청 사실 확인 (신뢰 신호)
✅ 여행 꿈 청취 (욕망 공감)
✅ 문제 인정 (자기검증)
✅ 정상화 (혼자가 아님)

→ 이 단계가 끝나면 고객은 
   "이 사람은 나를 이해하는군" 
   이라고 느낌
```

---

## STEP 1: 욕망 각성 (1분 30초)
### 목표: 여행 꿈을 감각적으로 이미지화

```
【Phase 1-1: 감각 이미지】

상담사: "근데 생각해보니까,

[여행지]에서 [고객이 말한 활동]을 하고 있으면

(침묵 2초)

그 시간이 얼마나 좋을까요?

[구체 장면 그리기]:
- 날씨는 어떨까?
- 누가 옆에 있을까?
- 뭘 하고 있을까?

상상해보세요."

(침묵 3초 - 감정이 살아나도록)

상담사: "그런 시간들이 있으면
정말 좋겠지 않을까요?"

(침묵 2초)

【핵심 포인트】
✅ 구체 장면 (뇌가 이미지화)
✅ 감각 자극 (시각, 감정)
✅ 침묵 (고객의 감정 침투)

→ 이 단계가 끝나면 고객은
   "아, 나 이 여행 정말 가고 싶네"
   라는 감정 상태
```

---

## STEP 2: 불안 증폭 (2분)
### 목표: 미루는 것의 대가를 구체적으로 이미지화

```
【Phase 2-1: 현재 상태의 악화】

상담사: "그런데 말이에요.

지금처럼 미루다가
3년이 지나면 어떻게 될까요?

당신은 어떻게 될까요?

더 건강해질까요? 아니면 더 피곤해질까요?"

(침묵 3초)

상담사: "그리고 함께하는 사람들은?

당신의 배우자, 
혹은 당신의 부모님은?

3년 뒤에도 지금처럼 건강하실까요?"

(침묵 4초 - 깊은 불안)

【Phase 2-2: 구체 손실 계산】

상담사: "지금 월 2만 7천원으로 가입하면
5년 동안 162만원.

5년 뒤에 가입하면
가격이 올라서 210만원.

48만원을 그냥 잃는 거랑 같아요.

(침묵 3초)

그 외에도:
- 가족과의 시간 손실
- 건강 나빠지는 시간
- 아이 성장 못 본 시간

이게 다 돈으로 환산할 수 없는 손실이죠."

(침묵 5초 - 절망)

【Phase 2-3: 반복 패턴 자각】

상담사: "그리고 생각해보니까,

작년에도 '올해는 가자'고 했을 거예요.

근데 지금도 못 가있죠?

3년 전에도 '언젠가 가야지'라고 했을 거고요.

같은 일이 또 반복될 가능성 없을까요?"

(침묵 5초 - 무기력감)

【핵심 포인트】
✅ 현재 악화 (건강, 시간)
✅ 구체 숫자 (48만원, 162만원)
✅ 가족 (배우자, 부모님, 아이)
✅ 반복 패턴 (매년 같은 말)
✅ 침묵 충분 (불안 깊어지기)

→ 이 단계가 끝나면 고객은
   "맞아, 이대로는 안 되겠는데?"
   라는 불안 상태
```

## STEP 3: 해결책 Reveal (1분)
### 목표: 불안을 해결책으로 변환

```
【Phase 3-1: 상황 반전】

상담사: "그런데요.

사실 이 문제를 해결하는 방법이 있어요.

(침묵 1초 - 호기심)

당신이 지금 '이 순간'에 결정하면
이 모든 게 해결돼요."

(침묵 2초)

【Phase 3-2: 해결책 제시】

상담사: "2가지 방법이 있어요.

① 지금 바로
   - 120만원 지원받고
   - 69만원만 내고
   - 7월에 떠나기

② 계속 즐기기
   - 월 2만 7천원 (커피값)
   - 3~5년 동안
   - 언제든 가능

당신 같은 사람들이 이미 12명이 했어요.

남은 자리가 18개.

3주 안에 다 찼을 거예요."

(침묵 3초)

【핵심 포인트】
✅ 희소성 (12명, 18자리, 3주)
✅ 2가지만 (양자택일)
✅ 구체 가격 (월 2.7만, 69만원)
✅ 사회증명 (12명)
✅ 긴급성 (3주)

→ 이 단계가 끝나면 고객은
   "어? 이게 가능한 거네?"
   라는 호기심 + 확신 상태
```

## STEP 4: Closing (1분 30초)
### 목표: 자발적 선택 유도 + 신청

```
【Phase 4-1: 선택 유도】

상담사: "그럼 어디가 더 맞아요?

당신 입장에서 어느 쪽이?"

(침묵 5초 - 고객의 답변 기다림)

(고객이 답함)

상담사: "좋은 결정이에요.

당신은 이제 가족과 
[여행지] 여행을 다닐 수 있어요."

(감정적 마무리)

【Phase 4-2: 신청】

상담사: "카톡으로 신청 링크 보내드릴게요.

(링크 전송)

5분이면 끝나요."

(고객 클릭 확인)

상담사: "완료되셨어요?

축하합니다!

금요일 라이브방송에서 
더 자세히 설명해드릴 거예요.

정말 감사합니다!"

【핵심 포인트】
✅ 강제 선택 아님 (고객이 말하게)
✅ 감정적 마무리 (여행 이미지)
✅ 빠른 행동 (5분)
✅ 후속 (금요일 라이브)

→ 이 단계가 끝나면
   자연스러운 신청
```

```

따뜻 → 뜨거움
심리: "아, 나 그 여행 정말 가고 싶다"

【불안 증폭】
뜨거움 → 불안
심리: "맞아, 이대로는 안 돼"

불안 → 확신

【Closing】
확신 → 행동
심리: "그럼 해볼까"
```

## 💡 V5 핵심 원리

### 1. 신뢰 Opening
- 고객 이름 + 신청 확인
- 욕망/문제 청취
- 공감 + 정상화
→ 콜드가 따뜻해짐

### 2. 욕망 → 불안 연쇄
- 욕망 (여행 꿈)
- 불안 (못 가면 어떻게?)
- 거리감 (지금 vs 3년 뒤)
→ 해결책이 필요해 보임

### 3. 월 2만 7천원의 위력
- 불안 상태에서 제시
- 선택지 2가지만 (양자택일)
- 희소성 (12명, 18자리)
→ "해봐야겠다" 심리

### 4. 감정적 마무리
- 선택은 고객이 함
- 여행 이미지로 마감
- 불안 해소 + 욕망 만족
→ 자발적 신청
