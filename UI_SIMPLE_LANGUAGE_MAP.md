## 📌 기본 원칙

❌ **피해야 할 표현**
- 영어 단어 (HOT, WARM, COLD, LOST 등)
- 기술 용어 (API, Webhook, Sync 등)
- 복잡한 설명 (5단어 이상)

✅ **사용할 표현**
- 순한글 (한자 최소화)
- 일상용어 (아이들이 쓰는 말)
- 단순한 문장 (3-4단어)
- 이모지 활용 (시각적 이해)

## 📋 변환 테이블

### 고객 상태 (Customer Status)

| 현재 (English/Complex) | 초등학생용 | 이모지 | 예시 |
|----------------------|---------|-------|------|
| HOT | 뜨거움 | 🔥 | "매우 관심 많음" |
| WARM | 따뜻함 | ☀️ | "조금 관심 있음" |
| COLD | 차가움 | ❄️ | "아직 관심 없음" |
| LOST | 못 찾음 | 💤 | "연락 안 함" |

### 고객 유형 (Contact Type)

| 현재 | 초등학생용 | 이모지 | 설명 |
|------|----------|-------|------|
| PROSPECT | 관심있는 사람 | 👁️ | "크루즈에 관심 있어 하는 사람" |
| CUSTOMER | 이미 샀어요 | ✅ | "이미 크루즈를 샀어 하는 사람" |
| LEAD | 물어본 사람 | ❓ | "크루즈에 대해 물어본 사람" |
| VIP | 특별한 고객 | 👑 | "우리를 많이 도와주는 사람" |

### 액션 버튼 (Action Buttons)

| 현재 | 초등학생용 | 이모지 | 위치 |
|------|----------|-------|------|
| 태그 SMS | 메시지 보내기 | 💬 | 메인 버튼 |
| 그룹 SMS | 모두에게 메시지 | 📢 | 메인 버튼 |
| 팀 공유 | 팀에 알려주기 | 🤝 | 상단 우측 |
| 담당자 할당 | 담당하는 사람 정하기 | 👤 | 각 고객 옆 |
| 전체 백업 | 모든 고객 저장하기 | 💾 | 우측 상단 |
| 엑셀 가져오기 | 엑셀에서 가져오기 | 📊 | 우측 상단 |
| 고객 추가 | 새로운 고객 추가 | ➕ | 우측 상단 |

### 필터/검색 (Filters)

| 현재 | 초등학생용 | 기능 |
|------|----------|------|
| 유형별 필터 | 어떤 사람인지 골라보기 | 관심있는 사람, 샀어 하는 사람 등 |
| 담당자별 필터 | 누가 담당하는지 골라보기 | 담당자 이름으로 찾기 |
| 그룹별 필터 | 어느 그룹인지 골라보기 | 팀, 부서별로 보기 |
| 태그 필터 | 라벨별로 골라보기 | 같은 표시가 있는 사람들 |

### 시간 표현 (Time Expressions)

| 현재 | 초등학생용 | 예 |
|------|----------|-----|
| D-14 | "14일 남았어요" | D-5 → "5일 남았어요" |
| D-DAY | "오늘이 마지막이에요" | 출발하는 날 |
| D+3 | "3일 지났어요" | 출발 후 3일 |
| 처음 연락 | "아직 연락 안 했어요" | 처음 보는 사람 |
| 오늘 연락 | "오늘 연락했어요" | 오늘 통화/메시지 |
| 3일 전 연락 | "3일 전에 연락했어요" | 최근 통화 기록 |

### 모달/팝업 텍스트 (Modals)

| 현재 | 초등학생용 |
|------|----------|
| "엑셀 파일 형식 안내" | "엑셀에 이렇게 써주세요" |
| "필수: 이름, 전화번호" | "반드시 써야 할 것: 이름, 전화번호" |
| "선택: 이메일..." | "있으면 좋은 것: 이메일..." |
| "파일을 클릭하거나 드래그" | "여기다가 파일을 끌어오거나 클릭" |
| "OOO명 등록 완료" | "OOO명을 추가했어요 ✅" |
| "성공" | "완료했어요" / "잘 됐어요" |
| "실패" | "안 됐어요" / "다시 해 봐요" |
