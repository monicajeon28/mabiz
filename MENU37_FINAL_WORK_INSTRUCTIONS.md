## 절대법칙: 3가지 관점 균형

모든 결정이 A(상담사효율) + B(전환율) + C(전사일관성) **3가지 모두**에 기여하는가 확인

```
합계 점수 = (A점수 × 가중치40%) + (B점수 × 가중치35%) + (C점수 × 가중치25%)

예시:
렌즈 1: A5 + B4 + C3 = (5×0.4) + (4×0.35) + (3×0.25) = 2.0 + 1.4 + 0.75 = 4.15점
렌즈 2: A4 + B5 + C4 = (4×0.4) + (5×0.35) + (4×0.25) = 1.6 + 1.75 + 1.0 = 4.35점 ⭐더 우선
```
