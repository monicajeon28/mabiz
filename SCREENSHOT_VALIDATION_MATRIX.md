# 스크린샷 검증 Matrix (P1-1)

## 📊 개요

이 문서는 4개 해상도별 스크린샷 검증을 **체계적**으로 수행하기 위한 체크리스트입니다.

각 스크린샷은 다음 5가지 검증 항목에 대해 **예/아니오** 로 판단합니다:
1. **텍스트 가독성** (font-size, line-height, 대비도)
2. **그리드 레이아웃** (컬럼 정렬, 마진, gap)
3. **이미지 비율** (aspect-ratio, lazy loading)
4. **터치 타겟** (최소 44×44px, 간격 ≥ 8px)
5. **성능** (Lighthouse 점수, CLS/INP)
