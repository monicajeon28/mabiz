## 3. Track D: A/B 테스트 할당 알고리즘

### 개요
200콜 A/B 테스트(12주)를 Block Randomization + Stratification + Crossover로 설계하여 선택편향 제거 및 통계파워 95% 확보

### 알고리즘 3단계

#### Step 1: Block Randomization (균등 배치)

```
원리: 각 상담사가 A/B를 50:50으로 균등 경험

구현:
상담사 5명 × Block size 4 × 12주
= 각 상담사 48콜 (A 24, B 24)

예시 (상담사A):
┌─────┬──────────────────────────┐
│Week │ Block (Random shuffle)   │
├─────┼──────────────────────────┤
│ 1   │ [A, B, B, A]            │
│ 2   │ [B, A, A, B]            │
│ 3   │ [A, B, A, B]            │
│ 4   │ [B, A, B, A]            │
│ ... │ ...                      │
│ 12  │ [A, B, B, A]            │
├─────┼──────────────────────────┤
│합계 │ A: 24, B: 24 (50:50)     │
└─────┴──────────────────────────┘

결과: 선택편향 제거 (각 상담사가 무작위로 A/B 경험)
```

#### Step 2: Stratification (능력별 제어)

```
원리: 상담사의 능력 차이가 A/B 효과 추정을 오염하지 않게 층화

데이터:
상담사A: 55% 전환율 (HIGH)
상담사B: 52% 전환율 (HIGH)
상담사C: 48% 전환율 (MIDDLE)
상담사D: 45% 전환율 (LOW)
상담사E: 42% 전환율 (LOW)

층화:
HIGH:   상담사A, B (평균 53.5%)
MIDDLE: 상담사C   (48%)
LOW:    상담사D, E (평균 43.5%)

할당 규칙:
- HIGH 레이어: A/B 균등 분배 (각 상담사 A 12, B 12)
- MIDDLE 레이어: A/B 균등 분배
- LOW 레이어: A/B 균등 분배

결과: 상담사 능력 차이가 A/B 효과에 영향을 주지 않음
      (능력이 높은 상담사가 A만 담당하는 선택편향 제거)
```

#### Step 3: Crossover (시간 효과 제어)

```
원리: 같은 상담사가 여러 시간대에서 A/B를 교대로 경험
     → 시간에 따른 시장 변화(Week 1 vs Week 12) 제어

설계 (예: 상담사A, B):
┌──────────────────┬────────┬────────┬────────┬────────┐
│ Period           │ 1      │ 2      │ 3      │ 4      │
│ (Week)           │ (1-3)  │ (4-6)  │ (7-9)  │(10-12) │
├──────────────────┼────────┼────────┼────────┼────────┤
│ 상담사A          │ A      │ B      │ A      │ B      │
│ 상담사B          │ B      │ A      │ B      │ A      │
├──────────────────┼────────┼────────┼────────┼────────┤
│ 각 상담사 합계   │ A:6/   │ B:6/   │ A:6/   │ B:6/   │
│                  │ B:6    │ A:6    │ B:6    │ A:6    │
└──────────────────┴────────┴────────┴────────┴────────┘

Period 2와 4에서 A/B가 스왑됨 (Crossover)

결과: 같은 상담사가 A와 B를 모두 경험
     → "상담사A는 능력이 높아서 A안 효과가 좋다"는 오류 제거
     → "A안이 정말 좋다"는 결론 확보
```

### Monday.com 자동화

```
주간 태스크 자동 생성:

[A/B Test - Week 1]
├─ 상담사A: A안 목표 2콜 (✓ 2/2)
├─ 상담사A: B안 목표 2콜 (✓ 2/2)
├─ 상담사B: B안 목표 2콜 (◐ 1/2)
├─ 상담사B: A안 목표 2콜 (○ 0/2)
├─ 상담사C: A안 목표 4콜 (◐ 3/4)
├─ 상담사D: B안 목표 2콜 (✓ 2/2)
├─ 상담사D: A안 목표 2콜 (◐ 1/2)
├─ 상담사E: A안 목표 2콜 (○ 0/2)
└─ 상담사E: B안 목표 2콜 (◐ 1/2)

주간 요약: 13/20 (65%)
기한: 2026-05-25 (금요일)
```

### 성공 기준
- ✅ Block Randomization: 각 상담사 A 24-26회, B 22-24회
- ✅ Stratification: HIGH/MIDDLE/LOW 레이어별 균등분배
- ✅ Crossover: Period 2, 4에서 A/B 스왑 확인
- ✅ Monday.com API: Week 1 태스크 10개 생성 성공
- ✅ 12주 일정: 모든 상담사 A/B 50:50 달성
