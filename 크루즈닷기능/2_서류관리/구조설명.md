# 📄 서류관리 상세 설명

## 폴더 구성

### `documents/` (관리자 UI + API)
- **page.tsx**: 서류 관리 페이지
- **route.ts**: CRUD API

## 주요 API 엔드포인트
- `GET /api/admin/documents` - 서류 목록
- `POST /api/admin/documents` - 서류 업로드
- `DELETE /api/admin/documents/[id]` - 서류 삭제

## 파일 타입 지원
PDF, Word, Excel 등 일반 문서 형식

## 보안 기능
- 파일 타입 검증 (MIME type)
- 파일 크기 제한
- 접근 권한 확인 (IDOR 방지)
