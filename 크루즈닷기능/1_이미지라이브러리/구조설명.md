# 📸 이미지 라이브러리 상세 설명

## 폴더 구성

### `images/` (관리자 UI)
- **page.tsx**: 이미지 관리 대시보드
- 기능: 이미지 검색, 필터, 페이지네이션, 업로드

### `batch-1-images-sync/`, `batch-2-images-sync/` (배치 처리)
- Google Drive에서 이미지를 주기적으로 동기화
- WebP 변환 및 메타데이터 저장

### `batch-sync-images/` (통합 배치)
- 모든 이미지 동기화 로직 통합

## 주요 API 엔드포인트
- `GET /api/admin/images` - 이미지 목록 조회
- `POST /api/admin/images` - 이미지 업로드
- `DELETE /api/admin/images/[id]` - 이미지 삭제
- `PATCH /api/admin/images/[id]` - 이미지 수정

## 데이터베이스
`ProductImage` 테이블에 저장됨
- id, productId, url, altText, createdAt, updatedAt 등
