# 🌐 B2B 랜딩페이지 상세 설명

## 폴더 구성

### `b2b-landing/` (B2B 랜딩페이지 관리)
- 관리자가 B2B 고객용 랜딩페이지를 만드는 페이지
- 페이지 CRUD, 미리보기, SEO 설정

### `landing-pages/` (일반 랜딩페이지)
- 공개 랜딩페이지 관리

### `b2b/` (B2B 공개 페이지)
- 파트너/B2B 고객이 보는 실제 페이지

### `landing/` (공개 랜딩페이지)
- 일반 사용자가 보는 페이지

## API 엔드포인트
- `GET /api/b2b` - B2B 페이지 조회
- `POST /api/b2b` - B2B 페이지 생성
- `PATCH /api/b2b/[id]` - B2B 페이지 수정
- `DELETE /api/b2b/[id]` - B2B 페이지 삭제

## 주요 기능
- 동적 페이지 생성
- CTA (Call To Action) 버튼 관리
- SEO 메타데이터 설정
- A/B 테스트
