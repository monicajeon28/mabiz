# 👨‍💼 관리자 공용 상세 설명

## 폴더 구성

### `admin/` (API 엔드포인트 - lib)
- `auth/` - 로그인, 로그아웃
- `settings/` - 시스템 설정
- `marketing/` - 마케팅 설정
- `customers/` - 고객 관리
- `customer-groups/` - 고객 그룹
- `products/` - 상품 관리
- `orders/` - 주문 관리
- `reports/` - 리포트 & 분석

### `admin/` (Component)
- 관리자 UI 컴포넌트
- 대시보드, 차트, 테이블, 폼 등

### `admin/` (Lib)
- 관리자 유틸리티 함수
- 권한 검증, 감사 로깅 등

## 주요 API
- `GET /api/admin/dashboard` - 대시보드
- `GET /api/admin/settings` - 설정 조회
- `POST /api/admin/settings` - 설정 변경
- `GET /api/admin/customers` - 고객 목록
- `POST /api/admin/customers` - 고객 추가

## 보안
- JWT 인증 (관리자만)
- 역할 기반 접근 제어 (RBAC)
- 감사 로깅 (모든 변경사항 기록)
- CSRF 토큰 검증
