# 🔄 퍼널 상세 설명

## 폴더 구성

### `funnel/` (퍼널 관리 대시보드)
- 퍼널 생성, 편집, 삭제
- 각 단계별 고객 수 추적
- 전환율 분석

### `funnel-messages/` (퍼널 메시지)
- 각 단계에서 전송할 메시지 설정
- 자동 발송 규칙 정의

### `funnel-stage-transition/` (단계 전환)
- 고객이 한 단계에서 다른 단계로 이동
- 이벤트 기반 자동 전환

### `marketing/` (마케팅 퍼널)
- 마케팅 캠페인 기반 퍼널

### `funnel-visualizer/` (시각화)
- 퍼널 구조를 시각적으로 표시

### `partner/` (파트너 퍼널)
- 파트너가 자신의 퍼널을 관리

## API 엔드포인트
- `GET /api/admin/funnels` - 퍼널 목록
- `POST /api/admin/funnels` - 퍼널 생성
- `PATCH /api/admin/funnels/[id]` - 퍼널 수정
- `DELETE /api/admin/funnels/[id]` - 퍼널 삭제
- `POST /api/admin/funnel-stage-transition` - 고객 단계 이동

## 데이터베이스
- `Funnel`: 퍼널 정의
- `FunnelStage`: 각 단계 (1단계, 2단계, ...)
- `FunnelCustomer`: 고객과 퍼널의 관계
- `FunnelMessage`: 단계별 발송 메시지
