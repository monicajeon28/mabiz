# 💬 문자 퍼널 (SMS) 상세 설명

## 폴더 구성

### `sms/` (SMS 기본 기능)
- SMS 발송 API
- 메시지 큐 관리

### `messages/` (메시지 발송)
- 메시지 발송 로직
- 템플릿 관리
- 스케줄 발송

### `settings/` (설정)
- SMS 공급자 설정 (Twilio 등)
- API 키 관리
- 발송 제한 설정

### `customers/` (고객별 발송)
- 특정 고객에게 SMS 발송
- 고객 세분화 (필터링)

### `message-center/` (메시지 센터)
- 모든 메시지 관리
- 통계 & 분석

## API 엔드포인트
- `POST /api/sms/send` - SMS 발송
- `GET /api/sms/status/[id]` - 발송 상태 확인
- `POST /api/admin/messages/send-sms` - 관리자용 발송
- `GET /api/admin/settings/sms` - SMS 설정
- `PATCH /api/admin/settings/sms` - SMS 설정 변경

## 주요 기능
- 즉시 발송
- 예약 발송
- 템플릿 기반 발송
- 대량 발송 (세분화)
- 발송 결과 추적
- 불량 번호 관리

## 보안
- API 키 암호화
- 발송 로그 감사
- 고객 동의 확인 (옵트인/옵트아웃)
