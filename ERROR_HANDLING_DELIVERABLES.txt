🎯 핵심 기능

【400 - 클라이언트 검증 오류】
❌ 입력값 오류
전화번호 형식이 올바르지 않습니다.
올바른 형식: 010-1234-5678
→ 인라인 에러 (빨강) / field 속성으로 입력란 강조

【413 - 파일 크기 초과】
⚠️ 데이터 크기 초과
현재: 250 MB | 최대: 100 MB
👉 파일을 분할하여 여러 번에 나누어 업로드하세요
→ 모달 UI (황색) / 가이드 제시

【500 - 서버 일시적 오류】
🔄 일시적 오류가 발생했습니다
작업 ID: op_6f3a9b2d
자동으로 다시 시도 중... (2/3)
→ 토스트 UI (주황색) / 자동 재시도 + 진행률
