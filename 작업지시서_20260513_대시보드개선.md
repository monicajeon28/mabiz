## 작업 1️⃣ | assign-stats 500 에러 수정 (P0)

**파일**: `src/app/api/contacts/assign-stats/route.ts`

**문제**:
- line 20: `requireOrgId(ctx)` 실패 → "ORGANIZATION_REQUIRED" 에러
- line 68: "UNAUTHORIZED"만 처리 → 나머지 에러는 모두 500

**해결**:
```typescript
// 현재 (라인 66-71):
catch (err) {
  const msg = err instanceof Error ? err.message : String(err);
  if (msg === 'UNAUTHORIZED') return NextResponse.json({ ok: false }, { status: 401 });
  logger.error('[GET /api/contacts/assign-stats]', { err });
  return NextResponse.json({ ok: false }, { status: 500 });
}

// 수정안:
catch (err) {
  const msg = err instanceof Error ? err.message : String(err);
  if (msg === 'UNAUTHORIZED') return NextResponse.json({ ok: false }, { status: 401 });
  if (msg === 'ORGANIZATION_REQUIRED') return NextResponse.json({ ok: false, message: '조직 정보 필수' }, { status: 400 });
  logger.error('[GET /api/contacts/assign-stats]', { err });
  return NextResponse.json({ ok: false }, { status: 500 });
}
```

**테스트**: 브라우저 개발자 도구 → Network → `/api/contacts/assign-stats` → 상태코드 확인
