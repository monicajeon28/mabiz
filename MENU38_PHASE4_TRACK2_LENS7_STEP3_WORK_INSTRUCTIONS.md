### 목표

### 주간 커리큘럼

#### Week 1 Day 1-2: PASONA 정의
- Module 1.1: PASONA란? (영상 30분)
  - Problem 단계 정의 (고객 불만 파악)
  - Agitation 단계 정의 (문제 증폭)
  - Solution 단계 정의 (해결방안)
  - Offer 단계 정의 (가격+조건)
  
- Module 1.2: 렌탈 고객의 4가지 기본 불만
  - 비용 부담 (카드할부 피로)
  - 자유도 부족 (패키지 투어 지겨움)
  - 안전 우려 (렌탈차 품질 불신)
  - 편의성 부족 (차량 픽업/반납 번거로움)
  
- 실습: 자신이 상담한 콜 5건 선택 → 각 콜에서 P/A/S/O 찾기

#### Week 1 Day 3-4: 렌탈 콜 분석 (실제 음성)
- Module 1.3: 성공 콜 vs 실패 콜 비교 (각 2건)
  - 성공 사례: "어떻게 하면 렌탈하나요?" 유도
  - 실패 사례: 가격만 설명하다가 종료
  
- Module 1.4: 침묵의 힘 (심리학)
  - I(시사) 질문 후 3~5초 침묵의 중요성
  - 침묵 중 고객 심리 변화 (불안→성찰→결심)
  - 실습: 침묵 5초 견디기 (녹음 연습)

#### Week 1 Day 5: 피드백 + 자신의 콜 분석
- 강사 피드백: 제출한 5건 콜 분석
- 개인별 1:1 회의 (15분)

### Week 2: PASONA 렌탈 특화 (온라인)

#### Week 2 Day 1-2: 렌탈 고객 세그먼트 (Problem/Agitation)
- Module 2.1: 3가지 고객 타입별 불만
  
  **① 자유 추구형 (연령 30-50, 커플/가족)**
  - Problem: "패키지 투어는 시간이 정해져 있어"
  - Agitation: "시간낭비도 많고, 가고 싶은 데 못 간다"
  - 상담사 멘트: "렌탈이면 당신 시간이 당신 것"
  
  **② 가성비 추구형 (연령 40-60, 직장인)**
  - Problem: "항공+호텔+렌탈 = 너무 비싸"
  - Agitation: "2박 일정으로 100만원 초과... 자주 못 간다"
  - 상담사 멘트: "하루 렌탈비 2만원으로 자유 여행"
  
  **③ 안전 추구형 (연령 45-70, 부모님 동반)**
  - Problem: "낯선 땅, 낯선 차... 불안해"
  - Agitation: "혹시 사고 나면? 차량 상태는 좋은가?"
  - 상담사 멘트: "신차만 제공, 24시간 긴급서비스"

- 실습: 각 타입별 고객 1명씩 콜 분석

#### Week 2 Day 3-4: 렌탈 Solution (가성비 + 자유)
- Module 2.2: 가성비 앵커링
  - "하루 렌탈비" vs "비행기+패키지" 수치 비교
  - 예시: 동남아 4일
    - 항공편: 80만원
    - 패키지투어: 120만원 (3박 4일)
    - 가이드+식사+호텔 포함
    - **VS 렌탈: 차량 10만원×4일=40만원 + 자유 선택 이동**
  
  - 심리학: "같은 비용으로 2배 자유도"

- Module 2.3: Offer 설계 (가격 제시 타이밍)
  - 절대 1순위: 하루 단가 (가장 작은 수)
  - 2순위: 월 총액 (구체성)
  - 3순위: 발생할 이득 (자유도 수치화)
  
  - 금지 표현:
    - ❌ "월 할부" → ✅ "하루 단가 얼마예요"
    - ❌ "약정" → ✅ "렌탈 계획"
    - ❌ "위약금" → ✅ "조기 해지 수수료"

#### Week 2 Day 5: 라이브 롤플레이 (강사 + 학습자)
- Live Session (1시간)
  - 강사가 "까다로운 고객" 역할
  - 학습자가 Week 1-2 배운 내용 적용
  - 동료들 피드백 (3명 이상)

### Week 3-4: 실전 롤플레이 (오프라인 + 온라인)

#### 목표: PASONA 4단계 완전 숙달

#### Week 3: P→A→S 단계 마스터

**Day 1-2: Problem 질문 5가지**
```
렌탈 고객의 불만 유도 질문 (SPIN 구조)

S(상황): "요즘 여행은 어떻게 다니세요?"
응답 예: "패키지 투어로 주로 가요"

P(문제): "패키지 투어에서 불편했던 게?"
응답 예: "일정이 정해져 있어서..."

강사의 피드백:
- P 질문이 충분히 구체적인가?
- 고객이 불만을 "자신의 말로" 표현했는가?
```

**Day 3-4: Agitation 기법 (침묵 5초 필수)**

```
강사 데모:
"패키지 투어로 계속 다니면...
 시간도 낭비되고, 돈도 많이 들고,
 정작 가고 싶은 데 못 가는 거 반복되겠네요.
 
 [침묵 5초]
 
 5년 후에 당신의 여행은 어떨 것 같아요?"

학습자 실습:
- 침묵 동안 고객 반응 관찰 (녹음)
- "고객이 뭔가 말하는 순간" = 성공 신호
```

**Day 5: 주간 평가 + 개인 코칭**

#### Week 4: S→O 단계 (Solution + Offer)

**Day 1-2: Solution 제시 (신뢰 구축)**

```
멘트 구조:
"제가 지난 3년간 렌탈 상담을 해봤는데요.
 정확히 당신 같은 고객분들이 
 렌탈로 변경하신 후에
 '이거 왜 진작 안 했냐'고 하세요."

+ 구체적 사례 1개 투입 (사회적 증거)
```

**Day 3-4: Offer 제시 (가격 앵커링)**

```
절대 공식:

"저희 렌탈 요금이
 
 스탠다드 가격: 하루 2만원
 프리미엄 가격: 하루 3만원
 
 입니다.
 
 [침묵 3초]
 
 어떻게 생각하세요?
 (절대 추가 설명 금지)"

학습자 평가 항목:
- 하루 단가를 먼저 말했는가? ✓
- 침묵 견딜 수 있었는가? ✓
- 총액은 언급하지 않았는가? ✓
```

**Day 5: 종합 롤플레이 (30분 콜 재현)**

```
시나리오:
"당신은 렌탈 상담사, 고객은 동료(배우)"

흐름:
1. 오프닝 (2분): 경계 해제 + 신뢰 구축
2. 현황 파악 (3분): S 질문 + P 질문
3. 문제 증폭 (3분): I 질문 + 침묵
4. 솔루션 (4분): 사례 + 경험담
5. Offer (4분): 가격 제시 + 침묵
6. 클로징 (4분): 플랜 선택 유도
7. 이의처리 (6분): 거절 대응

평가:
- 침묵 타이밍 정확도
- 고객 발화 비율 (60% 이상?)
- Offer 후 침묵 견디기
```

### Week 5-6: 멀티 터치포인트 + SMS 자동화

#### 목표: 렌탈 고객 3일 시퀀스 설계

#### Week 5: SMS 자동화 기초

**Module 5.1: 렌탈 고객 Life Cycle (3일 자동화)**

```
Day 0 (콜 직후): 긴급 신청
- SMS 1번: "지금 신청하시면 가입 완료" (1시간 내)
- 심리: 긴박감 (희소성)

Day 1 (24시간 후): Follow-up
- SMS 2번: "고민중이신가요? 1분에 완료됩니다" (오전 9시)
- 심리: 결정 유도 (사회적 증거)

Day 3 (72시간 후): 마지막 제안
- SMS 3번: "오늘까지만 특별가... 내일부터 일반가" (오후 5시)
- 심리: 마감 (희소성 + 손실 회피)
```

**Module 5.2: PASONA SMS 카피 작성**

```
Step 1: 각 Message의 목적 명확히

Message 1 (Day 0 1시간):
- 역할: "이미 가입할 준비가 됐다"는 신호
- 심리: 손실 회피 (지금 안 하면 더 비싸짐)
- 카피: "[고객명]님, 지금 신청하면 가입비 50% 할인이 끝납니다!"

Message 2 (Day 1 아침):
- 역할: "남들도 다 했어"
- 심리: 사회적 증거
- 카피: "어제 신청하신 분들 중 89%가 이미 첫 렌탈 예약했어요"

Message 3 (Day 3 저녁):
- 역할: 최후의 기회
- 심리: 희소성 + 손실 회피
- 카피: "마감 임박! 오늘까지 특별가... 내일부터는 정상가 적용됩니다"
```

#### Week 6: SMS + 콜센터 통합

**Module 6.1: SMS 응답 시나리오**

```
고객 SMS 응답 3가지:
1. "좋아, 언제부터 가능?" → 즉시 콜 (hot lead)
2. "생각해 봐야겠어" → 자동응답 + 48시간 후 재콜
3. 응답 없음 (침묵) → 자동 카톡 연결 (채팅)
```

**실습: 주간 SMS 발송 + 응답 처리 시뮬레이션**

### Week 7-8: 성과 측정 + 피드백 루프

#### 목표: 실제 콜 데이터 적용

#### Week 7: 주간 콜 분석

```
학습자 제출: "이번 주 녹음 5건"

강사 피드백 항목:
□ PASONA 4단계 모두 포함?
□ 침묵 5초 이상 사용?
□ 고객 발화 60% 이상?

□ 클로징 성공?

점수화:
- 5/5점: "어떻게 하면 렌탈해요?" 고객이 먼저 물음
- 4/5점: 가격 제시 후 고객 긍정 반응
- 3/5점: "한 번 더 생각해 봐야겠어" 다음 콜 약속
- 2/5점: 자료 카톡 발송 허락
- 1/5점: 그냥 종료
```

#### Week 8: 최종 인증 + 수료식

```

합격 기준:
□ PASONA 모든 단계 포함
□ 침묵 타이밍 정확 (±1초 범위)
□ 고객 발화 60% 이상
□ 클로징 성공 또는 다음 콜 약속

수료:
- 수료증 발급
- 급여 인상 +5% (렌탈부문)
- 월 콜 목표 증가 (50→70건)

### 각 상품별 PASONA 4단계 카피 (예시 5개)

**1. 신라호텔 - 스탠다드룸**

| 단계 | PASONA 카피 |
|-----|-----------|
| **P (Problem)** | "서울에서 편하게 쉬고 싶은데, 대부분의 호텔은 너무 삭막하지 않나요?" |
| **A (Agitation)** | "특히 회사 출장이 많으시면, 밤에 돌아와서 푹 쉬고 싶은데... 차가운 객실에서는 마음이 안 편합니다." |
| **S (Solution)** | "신라호텔은 Forbes 5-Star 호텔로 50년 이상 축적된 서비스 노하우가 있습니다. 객실은 따뜻한 톤의 인테리어로, 한발을 들이는 순간 '여기는 다르다'고 느끼실 거예요." |
| **O (Offer)** | "1박에 474,320원부터. 하루의 스트레스를 완벽히 풀 수 있는 가치라고 생각합니다. 조기 예약 시 30% 할인도 있으니 지금 예약하시면 가장 저렴하게 경험 가능합니다." |

**2. 파라다이스호텔 - 오션테라스룸**

| 단계 | PASONA 카피 |

### Module 1: 신민형 5단계 (60분)

**강의자:** 모니카

**학습 목표:**
- 신민형 5단계의 심리학 원리 이해
- 각 단계에서의 상담사 역할 파악
- 렌탈/호텔 고객에 맞춘 변형 적용

**스크립트 구조:**

```
[오프닝 3분]
"안녕하세요, 저는 지난 5년간 렌탈·호텔 고객 
 2,000명 이상을 상담한 모니카입니다.

 오늘 배울 '신민형 5단계'는 제가 실제로 
 적용해서 전환율을 42%에서 68%까지 올린 
 비결입니다. 이건 이론이 아니라 
 '실제로 작동하는 심리학'입니다."

[Phase 1: 욕망 깨우기 - 12분]
"고객은 렌탈을 몰라요. 왜냐면 지금까지
 패키지 투어나 자동차 렌탈 정도만 알고 있기 때문입니다.

 우리 첫 번째 단계는 '욕망 깨우기'입니다.
 즉, 고객이 지금까지 모르던 필요성을 깨우는 것.

 예: '자유 여행을 하고 싶으신 적 있으세요?'
     '아, 그런데 혼자는 좀 불안하고?'
     
 이 순간, 고객의 '숨겨진 욕망'이 깨어나는 거예요."

[Phase 2: 프리미엄 경험 제시 - 12분]
"깨어난 욕망에 '어떤 경험이 있다'는 걸 보여줍니다.

 패키지 투어 vs 렌탈:
 패키지: '정해진 일정을 따라가는 경험'
 렌탈: '나만의 속도로 만드는 경험'
 
 이게 '프리미엄'인 이유는 
 '시간'이 내 것이 되기 때문입니다."

[Phase 3: 비용 문제 해결 - 12분]
"이제 '비용이 든다'는 걸 인정하고 풉니다.

 상담사: '여행 1회에 보통 얼마 드세요?'
 고객: '200만원 정도'
 
 상담사: '그 200만원이 카드할부로 남으세요?'
 고객: '네, 아직 갚고 있어요'
 
 상담사: '그럼 우리 렌탈은 그 비용을 
          카드값 대신 여행 적금으로 바꾸는 거예요.
          같은 비용인데, 빚 대신 추억이 쌓이는 거죠.'
 
 = 비용 해결"

[Phase 4: 클로징 심화 - 12분]
"고객이 '원하는 마음'이 생겼어요.
 이제는 '행동 유도'입니다.

 절대 '지금 가입하세요'라고 하지 마세요.
 대신: '어떻게 하면 가입해요?'라는 질문을 
       고객이 스스로 하게 만드세요.

 방법: '더 궁금하신 거 있으세요?' + 침묵 5초
      
      [침묵....... 고객이 먼저 입을 열 때까지]"

[Phase 5: 관계 유지 + 재방문 - 12분]
"렌탈이 끝나고 6개월 뒤...

### Module 2: PASONA 기초 워크숍 (90분)

**강의자:** 신민형  
**형식:** 강의 40분 + 실습 50분

**[강의 40분]**

```
Part 1: PASONA vs SPIN의 차이 (15분)

SPIN 질문형:
S → P → I → NP (Question-based selling)
"당신의 불만이 뭐예요?" → "앞으로도 그럴까요?" 
→ "그걸 풀 수 있다면?"

PASONA 제안형:
P → A → S → O (Proposal-based selling)
"당신의 불만을 알아요" → "그게 얼마나 심한지 아세요?"
→ "우리가 풀 수 있어요" → "이 가격에 이 가치"

= 차이점: SPIN은 고객이 답하게 유도
         PASONA는 상담사가 답을 제시

Part 2: Problem 정의 (10분)

렌탈의 Problem:
- 가격 문제: "패키지 투어보다 비싸다" (거짓)
- 시간 문제: "일정이 정해져 있어" (진짜 문제)
- 심리 문제: "혼자면 무섭다" (진짜 문제)

호텔의 Problem:
- 신뢰 문제: "사진과 다를까봐" (온라인 불안)
- 가치 문제: "그냥 자는 곳 아닌가?" (경험 의심)
- 특별함 문제: "어디나 비슷한데?" (차별성)

Part 3: Agitation 기법 (10분)

Agitation = "문제를 증폭"하되, 
         "공격적이지 않게" 진행

상담사 톤:
"맞아요, 그거 진짜 힘들어요.
 그런데 생각해보니까...
 (침묵 3초)
 이게 계속되면 5년 후에는 어떨 것 같아요?"

= 동의 + 질문 + 침묵
  (고객이 스스로 답하게 함)

Part 4: Solution이 약한 이유 (5분)

절대 금지:
"우리 렌탈이 최고예요!" ← 뻔함
"가격이 가장 싸요!" ← 거짓
"서비스가 최상이에요!" ← 자뻑

올바른 Solution:
"제 고객 중에 [구체적 상황]인 분이 계셨어요.
 그분이 렌탈을 사용한 후에는...
 [구체적 변화]라고 하셨거든요.
 [이름]님도 그런 경험을 하실 수 있습니다."

= 개인화 + 사회적 증거
```

**[실습 50분]**

```
실습 1: 자신의 고객 Problem 정의하기 (15분)

학습자가 제시한 고객 프로필:
- 나이: 45세 회사원
- 직업: IT 부장
- 현재: "요즘 스트레스 좀 풀고 싶어"

질문: 이 고객의 Problem은?

답안 작성:
□ 스트레스 (표면적)
□ 카드할부 피로 (심층적)
□ 반복적인 패키지 여행 지겨움 (진짜 문제)

정답: 세 가지 모두!
하지만 우리가 공략할 건 "자유로움의 갈망"

실습 2: Agitation 멘트 작성 (15분)

학습자 작성:
"당신 같은 고객분들이 계신데요.
 월급쟁이로 살다 보니까...
 시간만 되면 자동으로 
 '어디 가봐야지' 생각하는 거죠?
 
 근데 갈 때마다 일정에 맞춰야 하고,
 카드 값 남고...
 
 (침묵 3초)
 
 5년 후엔 어떤 여행을 하고 계실 것 같아요?"

평가:
✓ 공감 언어 사용 O
✓ 침묵 타이밍 O
✓ 고객 자체 답변 유도 O

실습 3: Solution + Offer 통합 (20분)

케이스: "비용 질문" 들어올 때

상담사 답안:
"하루에 커피 한 잔 값이에요.
 약 2,500원인데요.
 
 [침묵 3초]
 
 그 돈으로 연간 12번의 자유 여행을...
 카드 값으로 남는 대신,
 추억으로 쌓을 수 있다는 거예요."

검증:
□ 하루 단가부터 시작? O
□ 총액은 안 말함? O
□ 가치 제시? O
□ 침묵? O
```

③ 경험 불안
  - "호텔에만 있으면 뭐해?"
  - 해결책: "주변 정보 + 투어 제공"

Part 2: 호텔 예약 심리의 3가지 단계 (30분)

Stage 1: 꿈꾸기 (Dream)
- 마음: "어디 가면 좋을까?"
- 상담사 역할: 가능성 제시 ("우리 호텔은...")

Stage 2: 비교하기 (Compare)
- 마음: "다른 호텔과 뭐가 다른가?"
- 상담사 역할: 차별성 강조 ("우리만 가능한...")

Stage 3: 결정하기 (Decide)
- 마음: "정말 이 가격이 맞나?"
- 상담사 역할: 신뢰 보증 ("최저가 보장...")

Part 3: 호텔 카피의 심리학 적용 (30분)

약한 카피:
"신라호텔은 럭셔리합니다"
→ 고객 반응: "그래서? (차별성 없음)"

강한 카피:
"Forbes 5-Star 호텔에서 
 당신은 매번 다른 VIP 대우를 받습니다"
→ 고객 반응: "어떻게?" (관심 유발)

카피 작성 원칙:
1. 호텔이 아닌 "당신의 경험" 중심
2. 숫자 (객실 크기) 아닌 감정 (편안함)
3. 기능 (침대, 욕실) 아닌 변화 (휴식 후 일어나는 변화)
```
