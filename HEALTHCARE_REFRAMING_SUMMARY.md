# Healthcare 스크립트 리프레이밍 완료 (SPIN 기반 크루즈닷골드)

## 목표
라인 348 healthcare 스크립트를 크루즈닷골드 회원 기반 SPIN 구조로 리프레이밍
- 기존: "월 2만7천원 건강검진" 단순 상품
- 변경: "여행 + 건강 통합 패키지" (크루즈닷골드 + 헬스케어)

---

## 완성된 3개 세그먼트

### 1. 신혼부부 (30-35세)
**핵심 심리**: 여행 꿈 + 새로운 시작 + 아기 계획

#### Phase 별 구조 (총 5단계, 15분 콜)

| Phase | 제목 | 시간 | 목표 | 핵심 기법 |
|-------|------|------|------|----------|
| 1 | 감정 연결 + 여행 꿈 | 0-3분 | 신뢰감 | Emotional Connection, Aspiration |
| 2 | SPIN 질문 | 3-10분 | 필요성 인식 | S(상황)→P(문제)→I(함의)→N(필요) |
| 3 | 패키지 설명 | 10-20분 | 솔루션 제시 | 3단계 분명화 + 새미패키지 강조 |
| 4 | 강점 강조 + 이의 | 20-30분 | 신뢰성 | Authority, Loss Aversion, Narrative |
| 5 | 클로징 + 첫 여행 | 30분 | 행동 촉구 | 감성 마무리 + 미래 비전 공유 |

#### 주요 컨텐츠
```
Phase 1 도입: "여행 꿈꿀 때가 있으세요? 건강 때문에 못 가본 분들 많아요"
Phase 2 SPIN: "비용? 시간? 건강? 무엇이 가장 많이 막아요?"
Phase 3 패키지: "크루즈 반값 + 배에서 24/7 건강관리 전담"
Phase 4 강점: "배 위에서 건강 관리 못 하면 여행 못 해요"
Phase 5 클로징: "당신의 첫 여행이 가장 기억에 남는 여행이 될 거예요"
```

**새미패키지 강조 포인트**:
- 배에서 아침 6시부터 저녁까지 24/7 건강관리
- 고혈압/당뇨 맞춤 식단 + 운동 처방
- 응급상황 대응 + 약물 관리
- "개인 의사 + 영양사 + 트레이너" 역할

**이의 대응 예시** (5가지):
1. "가입비 비싼데?" → "연 4회 여행 반값(3,000만원)으로 1년이면 회수"
2. "건강 검진은 병원에서?" → "우리는 여행하면서 건강 챙기는 거"
3. "아직 결정 못했어요" → "5년 뒤도 바빠요. 내 인생에 투자할 때가 지금"
4. "배우자 상의해야" → "반값 여행 + 평생 특권이라 대부분 OK"
5. "기회 놓칠까" → "300명 신청 중 아직 M자리 남음"

### 2. 자녀있는가정 (40-50세)
**핵심 심리**: 부모의 책임감 + 아이와의 추억 + 황금기 의식

#### Phase 별 구조 (총 5단계, 30분 콜)

| Phase | 제목 | 시간 | 핵심 멘트 |
|-------|------|------|----------|
| 1 | 감정 연결 + 아이 | 0-3분 | "아이들과 여행 많이 다니세요?" |
| 2 | SPIN 질문 | 3-10분 | "아이 성장 시기의 추억을 놓치면 영원히 못 가요" |
| 3 | 아이-부모 통합 | 10-20분 | "초등+중등+고등 황금기는 10년뿐이에요" |
| 4 | 책임감 강조 | 20-30분 | "부모가 건강하면 아이도 자신감 있어요" |
| 5 | 가족 약속 | 30분 | "첫 여행이 가족의 가장 큰 추억이 될 거예요" |

**아이-부모 통합 패키지 특징**:
- 부모 체력 검사 + 아이 체력 평가
- 가족 공동 준비 (매주 산책 + 함께 요리)
- 여행 중 부모는 스파, 아이는 키즈 프로그램
- "아이들이 부모 건강 챙기는 모습 배움"

**강점 포인트**:
1. 지금 건강하면 아이들이 안심 → 학교생활 자신감
2. 황금기 놓치면 돌아오지 않음 (초/중/고 총 10년)
3. 부모의 역할 모델 → 자녀의 노년까지 영향
4. 약물 악순환 방지 (운동 = 약 대체)
5. 부부의 마지막 로맨스 (함께 배 위에서)

---

### 3. 시니어 (55세+)
**핵심 심리**: 인생 2막 완성 + 나이의 유한성 + 가족 유산

#### Phase 별 구조 (총 5단계, 30분 콜)

| Phase | 제목 | 시간 | 핵심 톤 |
|-------|------|------|--------|
| 1 | 은퇴 후 꿈 | 0-3분 | "배우자분과 함께 세계 여행 꿈꾸신 적 있으세요?" |
| 2 | SPIN 질문 | 3-10분 | "지금이 가장 체력이 남아있을 때예요" |
| 3 | 시니어 특화 | 10-20분 | "약물 최적화 + 배의 의료진 24/7" |
| 4 | 인생 완성 | 20-30분 | "당신의 인생에 이 경험은 필수예요" |
| 5 | 첫 드림 여행 | 30분 | "배우자분과 함께 알래스카? 지중해?" |

**시니어 특화 요소**:
- 약물 최적화 상담 (여행용 가이드)
- 배 위의 응급 대응 (의료실 24/7)
- 하루 2회 바이탈 체크 (혈압, 맥박, 혈당)
- 맞춤 식단 (저염식/저당식/저포화지방)

**강점 강조**:
1. 55-70세 체력 추이 (매년 1-2% 저하) → "지금이 최고"
2. 배 위의 저녁 식사 = 인생 최고의 추억
3. 가족에게 물려줄 유산 ("할머니는 60대에 세계 여행을 다녔어")
4. "움직이는 것 자체가 약" (운동 → 혈압 저하 → 약물 감소)
5. 50년 인생 계산 (월 200만원 절약 × 600개월 = 12억 효과)

**시니어 시나리오별 답변**:
- "부모님 모시고 가도 돼요?" → "70-80대도 많이 와요. 특별 코스 있어요"
- "건강이 이미 안 좋아요" → "더 좋죠. 관리하면 약을 줄일 수 있어요"
- "평생 여행하는 게 현실적?" → "배 위에 모든 게 있어요. 거동 불편해도 괜찮아요"

## 핵심 심리학 기법 (SPIN + PASONA + 10렌즈)

### SPIN 구조 (모든 세그먼트)
```
S(Situation): 현재 여행 경험 + 건강 상태 파악
   └─ "요즘 여행 자주 다니세요? 어디 정도?"

P(Problem): 막혔던 경험 = 건강 때문
   └─ "건강 때문에 여행을 못 가본 적 있어요?"

I(Implication): 그것이 미치는 영향 = 인생 손실
   └─ "건강 없으면 20년 30년 여행을 못 간다면?"

N(Need): 필요성 = 여행 + 건강 통합
   └─ "건강 유지하면서 여행도 자유롭게 가려면?"
```

### 주요 렌즈 활용
| 렌즈 | 세그먼트 | 활용법 |
|------|--------|--------|
| L6: 타이밍 손실회피 | 신혼+자녀 | "지금이 유일한 기회, 나이 들면 못 해요" |
| L3: 차별성 미인식 | 모두 | "배 위에서 24/7 건강관리는 우리만" |
| L7: 가족/애정 설득 | 자녀 | "아이와의 추억을 위해 부모가 준비" |
| L9: 건강/신뢰 | 모두 | "건강하지 않으면 여행 못 한다" |
| L10: 즉시구매 | 시니어 | "배워자분과의 마지막 로맨스" |

### PASONA 구조
```
P(Problem): "건강이 없으면 여행도 못 해요"
A(Agitate): "다음 10년을 건강하지 않으면서 여행 못 간다면?"
S(Solution): "우리는 배 위에서 당신의 건강을 24/7 책임져요"
O(Offer): "월 59,000원 (A) ~ 199,000원 (C) 패키지"
N(Narrow): "분기 1회? 월 1회? 무제한?"
A(Action): "지금 가입하면 첫 달 50% 할인 + 항공료 할인권"
```

## 새미패키지 강조 포인트

### 가치 제시
```
따로 구매 가격 분석:
- 크루즈 여행: 월 50만원
- 건강검진: 월 15만원
- 새미패키지: 월 25만원
- 24/7 헬스라인: 월 30만원
━━━━━━━━━━━━━━━━━━
합계: 월 120만원
→ C 패키지: 월 19만9천원 (84% 절감)
```

### 구체적 역할 (배 위에서)
```
아침 6시: "혈압 재측정합니다" + 아침 운동 제안
점심: "오늘은 저염식 정식으로 드릴게요"
오후: "산책 30분 함께 할까요?"
저녁: "내일 기항지 운동 계획 짜요"

결과: 6개월 후 약물 용량 감소 가능
```

## 이의 대응 체계 (5-7가지/세그먼트)

### 공통 이의
1. "가입비 / 월 비용 비싸요"
   → "이미 월 여행비 + 렌탈비 + 검진비로 80만원 나가고 있어요"

2. "건강하니까 필요 없어요"
   → "지금은 괜찮지만 10년 뒤 혈압/당뇨 약 월 80만원 들어요"

3. "일정이 안 맞아요"
   → "A(분기 1회) / B(월 1회) / C(무제한) 중 선택해요"

4. "생각해 보고 연락할게요"
   → "오늘 가입만 첫 달 50% + 항공료 할인 (내일부터 없음)"

5. "배우자 상의해야 해요"
   → "지금 스피커폰 5분이면 충분해요"
