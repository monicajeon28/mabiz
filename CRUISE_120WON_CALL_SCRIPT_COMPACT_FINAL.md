## 📞 OPENING (30초)

```
"김OO님, 좋은 소식 하나 있어요.

저는 크루즈닷 매니저 ___입니다.

OO월에 신청하셨던 거, 기억나세요?
(잠깐 대기)

지금 딱 2분만 괜찮으실까요?
120만원을 드릴 수 있는 방법이 있거든요."

↳ PASONA 패턴: "좋은 소식" (긍정 프레임)
↳ SPIN S: 상황 인식 + 긍정 신호 (예 스타트)
↳ 심리학: 희소성("딱 2분"), 호기심
```

## ⚠️ 문제 확인 (1분 30초)

### 【패턴 1: "A만이 B는 아니다"】

```
"그런데 혹시...

패키지 여행이 '비싼' 게 아니라,
'준비 스트레스'가 비싼 거 아닐까요?"

---

여행을 하려면:
- 비행기표? 직접 끊고
- 호텔? 직접 예약하고
- 짐? 계속 옮기고
- 가이드? 따로 비용 내고

이런 식으로 하면
피곤하지 않을까요?"
```

↳ PASONA P 패턴: "A만이 B는 아니다" (고정관념 깨뜨리기)
↳ 심리학: 손실회피(구체적 피로 이미지)

---

### 【패턴 2: "A하면 B"】

```
"여행이라고 하면,
설렘이 먼저예요?
아니면 '어, 얼마나 들까?'가 먼저예요?"

(침묵 → 고객이 깨닫도록)
```

↳ PASONA A 패턴: "A하면 B" (상황의식)
↳ 심리학: 현재 고통의 언어화 유도

---

### 【패턴 3: 구체적 장면 (언더독 효과)】

```
"사실 저도 처음엔 똑같았어요.

'이게 이렇게 복잡하나?' 했거든요.

근데 고객분들 얘기 들어보니,
다들 그 불편함 때문에
여행을 자꾸 미루더라고요."
```

↳ PASONA A 패턴: 자신의 경험 공개 (신뢰도 ↑)
↳ 심리학: 사회적 증명 (다른 사람들도 같음)

```

내년? 
그 다음해?

(침묵)

그런데 자녀분 있으신가요?
(혹은 배우자)

아이들은 정말 빨리 자라요.
지금 가려던 그 여행,
몇 년 후엔 아이가 '엄마/아빠와 못 가겠다'고
할 수도 있거든요."
```

↳ PASONA P 패턴: "미래 예측" (손실회피 극대화)
↳ 심리학: 시간의 창문(Ikigai), 인생 소중한 순간 상실 불안

---

### 【함축 질문 2: 체력 창문】

```
"그리고요,

지금처럼 일만 하다가,
5년 후에 갑자기 여유 시간이 생길까요?

아니면 더 바빠질까요?

(침묵 → 고객이 깨닫도록)

'언젠가 여행 가야지' 하다 보면,
그 '언젠가'는 안 와요.
지금이 정말 마지막일 수 있거든요."
```

↳ PASONA P 패턴: "현재편향" (지금이 유일한 기회)
↳ 심리학: 기회비용 (미루면 사라짐)

---

### 【함축 질문 3: 금전적 손실】

```
"그리고 한 가지 더 있어요.

'선착순 30인'이라고 했잖아요?

이건 일주일이면 다 차거든요.

그럼 당신이 2주일 미루면?

선사 할인이 없어져요.
70~80만원을 더 내야 하는 거죠.

지금이 '지금 아니면 없다'는 거예요."
```

↳ PASONA N 패턴: "희소성" (시간 + 수량 한정)
↳ 심리학: 손실회피 (구체적 금액으로 강화)

## 💎 PASONA 카피 패턴 사용 요약

| PASONA 단계 | 사용한 패턴 | 문장 |
|-----------|----------|------|
| **P(문제)** | "A만이 B는 아니다" | "패키지가 비싼 게 아니라 준비 스트레스가 비싼 거" |
| **P** | "A하면 B" | "여행이라 하면 설렘? 비용이 먼저?" |
| **P** | "미래 예측" | "지금 안 가면 언제 가세요?" |
| **A(공감)** | "자신의 경험" | "저도 처음엔 똑같았어요" |
| **A** | "사회적 증명" | "고객분들 후기가..." |
| **A** | "상상해보세요" | "침대에서 깨어나면 이미 다음 나라" |
| **A** | "~처럼" | "마치 친구들과 여행 다니는 것처럼" |
| **S(해결)** | "구체적 그림" | 호텔 이동 vs 크루즈 비교 |
| **O(제안)** | "전체 비용 명확화" | "10만원이면..." |
| **N(한정)** | "희소성" | "선착순 30인", "일주일이면 다 참" |
| **A(행동)** | "즉시성" | "5분이면 끝나요" |
