2️⃣ 로직 검증
   ✅ 시간 계산:
      - now = Date.now() (현재 시간, ms 단위)
      - diff = targetDate.getTime() - now (남은 시간)
      - days = Math.floor(diff / 86400000)
      - hours = Math.floor((diff % 86400000) / 3600000)
      - minutes = Math.floor((diff % 3600000) / 60000)
      - seconds = Math.floor((diff % 60000) / 1000)

   ✅ 긴박감 레벨 판정:
      - totalHours >= 24 → "safe" (초록, 안전)
      - 6 <= totalHours < 24 → "warning" (황색, 경고)
      - 1 <= totalHours < 6 → "alert" (주황, 주의)
      - totalHours < 1 → "critical" (빨강, 긴급)

   ✅ setInterval 스마트 관리:
      - 1시간 미만 && hours == 0 → 1초 간격 (1000ms)
      - 그 외 → 60초 간격 (60000ms, 배터리 절감)
      - 간격 변경 시 clearInterval + 새로운 setInterval

   ✅ 메모리 정리:
      - 마감 시 (diff <= 0): clearInterval(timerId)
      - 언마운트: return () => clearInterval(timerId)

   결과: PASS ✅ (버그 0개, 로직 100% 정확)
