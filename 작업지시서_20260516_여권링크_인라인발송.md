# 여권 링크 발송 UX 개선 - 인라인 메시지 미리보기
**작성일**: 2026-05-16  
**우선순위**: P1 (사용성 개선)  
**범위**: `src/app/(dashboard)/passport/page.tsx` (1개 파일)  
**영향**: 사용자 경험, SMS 발송 흐름

---

## 🎯 목표

**현재 문제:**
- "링크 생성하기" → 모달 열기 → 복사 (과정이 복잡함)
- 일괄 발송 시 링크 미리보기 불가능

**개선 목표:**
- 링크 생성 결과를 **"🔗 링크로 직접 보내기" 섹션 내에서 인라인으로 표시**
- 메시지 미리보기 + 링크 모두 복사 가능
- "링크만 전송" vs "메시지로 발송" 옵션에 따라 다른 UI 표시
- 기존 기능은 모두 유지 (호환성)

**현재 로직:**

**변경 로직:**
```typescript
const handleSend = async () => {
  try {
    setIsBulkGenerating(true);
    
    // 1. 먼저 링크 생성 (기존 로직)
    const linkResult = await fetch('/api/passport/admin/lead-link', {
      method: 'POST',
      body: JSON.stringify({
        userIds: selectedIds,
        expiresInHours: expiresInHours,
        // ... 기존 파라미터
      })
    });
    
    const linkData = await linkResult.json();
    
    if (!linkData.ok) {
      throw new Error(linkData.message);
    }
    
    // 2. 메시지 렌더링 (template + 변수 치환)
    let message = '';
    if (sendMode === 'message') {
      // 선택한 템플릿 또는 기본 템플릿 로드
      const template = templates.find(t => t.id === selectedTemplateId) || 
                      templates.find(t => t.isDefault);
      
      message = renderMessage(
        template?.body,
        { 
          link: linkData.data[0]?.link, 
          customerName: '고객명',
          productName: '상품명',
          departureDate: '출발일'
        }
      );
    } else {
      // "링크만 전송" 모드
      message = linkData.data[0]?.link;
    }
    
    // 3. 결과를 inlineLink 상태에 저장 (모달 대신 인라인 표시)
    setBulkLinkResult({
      link: linkData.data[0]?.link,
      message: message,
      expiresAt: linkData.data[0]?.expiresAt,
      selectedSendMode: sendMode,
      templateId: sendMode === 'message' ? selectedTemplateId : undefined
    });
    
    showSuccess(`${selectedIds.length}명에게 링크가 생성되었습니다.`);
    
  } catch (error) {
    showError(error.message || '링크 생성에 실패했습니다.');
  } finally {
    setIsBulkGenerating(false);
  }
}
```

### 4️⃣ 새로운 함수 추가

```typescript
// SMS 발송 (링크 생성 후)
const handleBulkSendAfterLinkGeneration = async () => {
  try {
    setIsSending(true);
    
    const result = await fetch('/api/passport/admin/send', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        userIds: selectedIds,
        templateId: bulkLinkResult?.templateId || selectedTemplateId,
        expiresInHours: expiresInHours,
        channel: 'sms',
        messageBody: bulkLinkResult?.message,
      })
    });
    
    const data = await result.json();
    
    if (!data.ok) {
      throw new Error(data.message);
    }
    
    showSuccess(`${selectedIds.length}명에게 SMS가 발송되었습니다.`);
    setBulkLinkResult(null);
    setSelectedIds([]);
    
  } catch (error) {
    showError(error.message);
  } finally {
    setIsSending(false);
  }
}
```

## 💡 주의사항

1. **기존 modals 유지**: 개별 고객의 "링크 생성" 버튼은 모달로 유지
2. **SMS 비용**: 발송 시 잔액 확인 필수
3. **링크 유효성**: 생성된 링크의 만료 시간 표시
4. **메시지 변수**: `{고객명}`, `{링크}`, `{상품명}`, `{출발일}` 모두 치환되어야 함
