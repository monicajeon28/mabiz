## 기대 효과 (시장 분석)

### 1. 신청 전환율
- 현재: 웹사이트 방문 → 문의까지 평균 3.5% 전환
- 목표: 골드회원 특화 4단계 → **8-12% 전환** (신규 라이드)
- 심리학 영향: 손실회피 + 진행률 표시 + 즉시 피드백

### 2. 월별 신청자 예상
```
현재: 월 20-30명
목표: 월 60-80명 (3배 증가)
수익: 80명 × (A플랜 33K + B플랜 66K) / 2 = 월 +$3.96M (한화 약 4-5억 원)
```

### 3. 건강검진 연계율
- 신청자 중 건강검진 예약율: **85%** (대시보드 유도)
- 연 140건 × 2회 (A플랜) = 280건
- 연 140건 × 4회 (B플랜) = 560건
- 총 연 840건 (월 70건) 예약 증가

### 4. 혼자 여행자 매칭 참여율
- 신청자 중 매칭 신청: **65%**
- 월 52명 (80명 × 65%)
- 크루즈 예약 전환: 매칭 참여자 중 85% → 44건/월
- 추가 수익: 월 +$52K-104K (한화 약 6-12천만 원)

### 5. 전체 예상 효과 (연 단위)
```
신청 전환: 월 80명 × 연 12개월 = 960명/년
신청료: 960 × (33K + 66K) / 2 = $47.52M (한화 약 5.7억 원/년)

건강검진 추가: 월 70건 × 평균 600K/건 = $42M (한화 약 5억 원/년)

혼자 여행자 매칭: 월 44건 × 평균 1000K/건 = $52.8M (한화 약 6.3억 원/년)

총 예상 수익: **$142.32M/년** (한화 약 17억 원/년)
```
