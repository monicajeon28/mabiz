# 부재중 고객 크루즈 콜 스크립트 V3 (최종 실전)
## 목표: 콜드 고객이 자발적으로 "어떻게 해요?"라고 묻게 만들기
## 예상 클로징율: 75~85% | 전체 시간: 10분

---

## 🎯 STEP 0: 심리 쿠션 Opening (30초) | 0°C → 15°C

```
"안녕하세요, 김OO님! 

저는 크루즈닷 상담원 000이라고 합니다.

OO월에 저희 신청해주셨던 거 기억나실까요?

오늘은 좋은 소식이 생겨서 전화를 드렸어요.

(침묵 1초 - 고객의 호기심 기다림)

근데 잠깐, 먼저 여쭤봐도 될까요?"
```

**심리학 적용:**
- ✅ "좋은 소식" (세일즈콜 프레이밍 제거)
- ✅ 시간 부담 없음 ("근데 잠깐")
- ✅ 질문 시작 (고객의 감정 참여 시작)

**절대 금지:**
- ❌ "이벤트 안내", "멤버십 추천", "특가 상품"

## 👀 STEP 1: Situation - 여행 꿈 파악 (1분 30초) | 15°C → 35°C

### 【Phase 1-1: 여행지 파악】

```
상담사: "OO월에 신청하셨을 때, 
어디를 가고 싶으셨어요?

부산에서 나가는 일본? 
아니면 다른 지역?

구체적으로 말씀해볼래요?"

(고객 답변 청취 - 완전히 들음)

상담사: "아, [여행지]이군요. 

그게 왜 가고 싶으셨어요?
특별한 이유가 있으셨나요?"

(고객의 감정 파악)
```

**SPIN-Situation 적용:**
- ✅ 개방형 질문 (선택지 주지 않음)
- ✅ 고객이 충분히 말하도록 경청
- ✅ 상담자의 첫 발언 최소화

---

### 【Phase 1-2: 감정 이미지 + 동반자】

```
상담사: "알았어요. 그럼 한번 상상해봐요.

[여행지 구체 명소]에서 
[고객이 말한 활동]을 하고 있는데,

(침묵 2초)

그 순간이 어떻게 느껴질 것 같아요?

그리고 이거... 
혼자 가고 싶으신 건가요?
아니면 가족이랑? 
부부끼리? 
친구들이랑?"

(고객 답변)

상담사: "아, [동반자]와 함께군요. 

그 느낌 정말 좋죠.
[구체 이미지]... 
그런 시간들이 대부분 그림 같거든요."

(침묵 2초 - 감정이 박히도록)
```

**PASONA-Affinity 적용 (공감):**
- ✅ 감각 이미지 구체화 (지중해 선셋, 와인, 음악)
- ✅ 동반자 파악 (가족=추억, 부부=로맨스, 친구=우정)
- ✅ 고객의 감정을 상담사가 반영하며 강화

**Bridge → STEP 2:**
```
상담사: "그런데 말이에요...

그 여행을 왜 아직 못 가셨어요?

구체적으로 뭐가 제일 막혔는지 
말씀해볼래요?"
```

## 🚫 STEP 2: Problem - 왜 못 갔는가 (1분) | 35°C → 50°C

```
상담사: "최근 1~2년 사이에 
여행을 못 가신 이유가 뭐예요?

정말 구체적으로요."

(고객이 스스로 문제를 언급하도록 대기)

(고객 답변: 예를 들어 "준비가 복잡해서요")

상담사: "맞아요, 그게 진짜 제일 문제예요.

비행기표는 직접 끊어야 하고,
호텔도 직접 비교하고,
현지 가이드도 알아봐야 하고,
혹시 문제 생기면 혼자 대처해야 하는 거 
정말 복잡하죠.

그래서 대부분 분들이
'올해는 힘들겠지', '내년에 해야지'
이렇게 자꾸 미루게 되는 거거든요.

당신도 그래서 이렇게 되신 거 아닐까요?"

(침묵 3초 - 고객이 공감하고 인정하도록)
```

**SPIN-Problem 적용:**
- ✅ 고객이 자기 언어로 문제 표현
- ✅ 상담사가 문제를 확대하며 공감
- ✅ "당신만이 아니라 다들 그래요" 정상화

**Bridge → STEP 3 (가장 중요):**
```
상담사: "근데 더 무서운 게 있어요.

계속 미루다 보면 어떻게 될까요?

한번 생각해봐요."
```

## 💥 STEP 3: Implication - 미루면 어떻게 될지 (2분) | 50°C → 75°C
### **가장 강력한 단계 - 파충류 뇌 자극 극대화**

### 【차원 1: 시간 이미지 (개인화)】

```
상담사: "자녀분이 지금 몇 살이에요?"

(고객: "11살이요" 라고 답함)

상담사: "아, 11살이면 
3년 뒤에는 14살 고2가 되겠네요?

그때쯤이면 여전히 
부모님이랑 여행 가고 싶어할까요?

아니면 친구들이랑만 가고 싶을까요?

지금 이 시간, 
가족끼리 함께할 수 있는 순간이 
언제까지 있을 것 같아요?"

(침묵 5초 - 깊은 생각하도록)

또는 노부모 있으신 경우:
"부모님 건강은 어떠신가요?

70대 중반이신가요?

5년 뒤에도 
활동적으로 여행 다니실 수 있을까요?"

(침묵 5초)
```

### 【차원 2: 감정적 후회 이미지】

```
상담사: "작년에 
'올해는 꼭 가자'고 했던 여행이 있어요?"

(고객이 "네" 라고 답함)

상담사: "그 여행이... 
올해 할 것 같아요?

(침묵 3초)

아마도 같은 이유로 또 미루게 되는 거 아닐까요?

3년 전에도 
'언젠가 가야지'라고 했을 것 같은데...

지금도 같은 자리에 있으신 거 아닌가요?"

(침묵 5초 - 깊은 절망과 후회)
```

**SPIN-Implication 심리학 (가장 강력):**
- ✅ **시간 구체화**: 아이 나이 계산 = 미룰 수 없는 느낌
- ✅ **부모님 건강**: 지금이 아니면 영원히 못 갈 수 있다
- ✅ **반복 패턴 인식**: "매년 같은 말 반복" = 무기력감
- ✅ **감정 극대화**: 논리 (숫자)보다 감정 (후회)이 강함

---

### 【차원 3: 기회비용 손실】

```
상담사: "그런데 돈으로 계산해보면...

지금 가입: 월 2만 7천원 × 36개월 = 97.2만원

3년 뒤 가입: 가격이 올라가 있을 텐데,
보통 연 5% 상승하니까
월 3만 5천원 × 36개월 = 126만원

(침묵 2초)

28.8만원을 그냥 버리는 거나 마찬가지인데...

그게 맞아요?

(침묵 5초 - 숫자가 박히도록)

여행 기회, 가족 추억, 
부모님 건강, 아이 성장...

다 합치면 
'3년 뒤가 낫다'는 생각이 들어요?"

(침묵 5초)

근데 솔직히 여쭤볼게요.

지금 이대로도 괜찮으세요?"
```

**손실회피 심리학:**
- ✅ 구체 숫자 (28.8만원) > 추상적 표현
- ✅ 여러 차원의 손실 (시간, 돈, 추억, 건강)
- ✅ 선택의 여지 없음 (이대로는 불가능함을 깨달음)

**Bridge → STEP 4 (호기심 극대):**
```
상담사: "사실 이게 전부 해결되는 
방법이 있어요.

근데 좀 특이해요.

들어보시면 
'이게 왜 이렇게 싼데?'라고 
하실 거예요."

(침묵 1초 - 궁금증)
```

## 🔥 STEP 4: Hook Reveal - 월 2만7천원 (1분) | 75°C → 90°C

```
상담사: "월 2만 7천원이에요.

(침묵 3초 - 숫자가 박히도록)

커피 한 잔에 5천원 하니까,
커피 5잔 값이에요.

그 정도로:

- 50만원 여행을 
  35만원에 갈 수 있고,

- 또는 3년에 3번을 다닐 수 있어요.

어떻게 이게 가능한지 
설명드릴게요."
```

**후킹 기법 (Reveal):**
- ✅ 미스터리 + 해소 (빨려들어감)
- ✅ 가격 재설정 (커피값 = 일상적 부담 없음)
- ✅ "어떻게 가능한가" 궁금증 극대

## ✨ STEP 5: Solution - 시스템 설명 (2분) | 90°C → 98°C

### 【Phase 5-1: 선사의 강점】

```
상담사: "우리가 선사인데요,

그게 무슨 의미냐면...

크루즈 가격 정보가 가장 먼저 들어와요.

'부산-오사카 3박' 같은 
인기 일정은 
3주 전에 이미 다 끝나버리거든요.

근데 골드멤버분들은 
그 정보를 먼저 받고,
가장 저렴한 가격으로 예약할 수 있어요.

그게 핵심 가치예요."

(침묵 2초)
```

### 【Phase 5-2: 사회적 증명 (신뢰도 극대)】

```
상담사: "지금까지 가입하신 분들을 보면:

- 50대 부부: 89%
- 40대 직장인 가족: 78%
- 만족도: 98%

당신 같은 분들이 이미 12명 가입했어요.

(침묵 2초)

남은 자리가 18개예요.

왜냐하면 선발로 모집하기 때문에
인원이 차면 닫혀요.

'부산-오사카 일정도 
3주 전에 끝난다'고 했잖아요?

남은 18자리도 
앞으로 3주 안에 차버릴 거예요.

그때 후회하지 말고..."

(침묵 3초)
```

**사회적 증명 + 희소성:**
- ✅ 구체 통계 (89%, 98%)
- ✅ "이미 12명" (나만 못하는 느낌 = FOMO)
- ✅ "남은 18자리" (시간 제한)

### 【Phase 5-3: 비교 (감각적 선택)】

```
상담사: "일반 여행사는 
하루 8시간 자고 
나머지 16시간은 
관광지 따라다니는 거잖아요.

우리는 배 위에서 24시간 
편하게 머물면서
기항지를 경험하는 거예요.

아침에 일어나면 새로운 나라,
저녁에는 배 위에서 와인,
밤에는 쇼 관람,
새벽에는 선상 조깅...

이게 진짜 럭셔리인 거예요."

(침묵 2초)
```

**감각 이미지:**
- ✅ 구체적 활동 (와인, 쇼, 조깅)
- ✅ 시간대별 체험
- ✅ 호텔 vs 크루즈 비교 (실제 경험의 질)

**Bridge → STEP 6:**
```
상담사: "이것까지 듣고 보니,
어때요?

정말 솔직하게 말해봐요."

**SPIN-Need-Payoff 적용:**

- ✅ 상담사는 경청만 함 (주장 X)
- ✅ 자발적 "네, 해보고 싶어요" 유도

## 🚨 예상 객각 응답 대응 (추가)

### 【"그래도 비싼데요" - 가격민감형】

```
상담사: "네, 1번만 가면 비싸죠.

하지만 생각해보세요.

당신이 여행을 좋아하는 사람이라면,
5년에 3~4번은 다니겠죠?

그럼:
① 일반: 50만원 × 4 = 200만원
② 우리: 월 2.7만 × 60개월 + 40만 × 4
        = 162만 + 160만 = 322만...

아, 계산 실수.

다시 하면, 월 2만 7천원은
이미 할인을 포함한 가격이거든요.

더 간단히 말하면:
'50만원 여행을 자주 다니신다면,
우리 멤버는 그걸 35만원에 가거나
같은 비용으로 여러 번 다닐 수 있다'

커피값으로 전환하면
월 5천 × 5.4잔 = 월급의 0.3%

이 정도면 비싼 게 아니라
투자라고 봐요."
```

### 【"배우자랑 상담해야 해요" - 결정 회피】

```
상담사: "당신의 결정이 제일 중요해요.

보통 한 분이 먼저

## ✅ 최종 체크리스트 (콜 전 필수 확인)

### 【Tone & Delivery】
- ☐ 세일즈맨이 아닌 "상담원" 톤 유지
- ☐ 침묵은 수행 (5초 이상)
- ☐ 고객의 말을 완전히 들음 (끝말 거느리지 않음)
- ☐ "나는 당신을 이해한다" 공감 표현

### 【온도 상승 곡선】
- ☐ Opening: 경계 → 호기심 (15°C)
- ☐ Situation: 호기심 → 공감 (35°C)
- ☐ Problem: 공감 → 인정 (50°C)
- ☐ Implication: 인정 → 불안/욕망 (75°C)
- ☐ Hook: 불안 → 호기심 폭발 (90°C)
- ☐ Solution: 호기심 → 확신 (98°C)
- ☐ Payoff: 확신 → 자발적 신청 (100°C)

### 【SPIN 요소】
- ☐ S: 여행 꿈 깊게 파악
- ☐ P: 고객이 문제를 스스로 언급
- ☐ I: 미래 손실을 구체적으로 이미지화
- ☐ N: 고객이 스스로 이익을 말함

### 【PASONA 요소】
- ☐ P: 문제 명확 (못 간 이유)
- ☐ A: 공감 깊게 (감정 이미지)
- ☐ S: 해결책은 호기심으로 (숨김)
- ☐ O: 가격 재설정 (커피값 = 일상)
- ☐ N: 희소성 강조 (남은 18자리)
- ☐ A: 즉시 클로징 (양자택일)

### 【파충류 뇌 자극】
- ☐ 감각 이미지 사용 (와인, 선셋, 쇼)
- ☐ 구체 인물 언급 (아이 나이, 부모님)
- ☐ 후회 이미지 (작년에 못 간 여행)
- ☐ 사회 증명 (이미 12명)
- ☐ FOMO 강화 (남은 18자리, 3주 안 끝남)
