# 부재중 고객 크루즈 120만원 지원 콜 스크립트 (최종 - 클로징율 70~85% 목표)

## 📞 OPENING (2분) - 거부심 제거 + 신뢰도 구축

```
"안녕하세요, 김OO님. 

저는 크루즈닷 매니저 ___입니다.

최근에 저희 부산출도착, 일본크루즈 같은 
'크루즈 여행 월 2만 7천원 멤버쉽' 이벤트가 나왔는데, 

김OO님이 선착순 30인 추천 대상이 되셔서 
안내 전화를 드렸어요.

【미리 말씀드릴 3가지】

아마도 이 3가지 질문이 나올 것 같은데, 
미리 답변해드릴게요.

1️⃣ '5년 약정이 부담스러운데 빠져나올 수 있나요?'
→ 네, 1년차부터 탈퇴 가능합니다. 
   위약금도 없어요. 
   다만 그때 남은 할인 혜택은 못 받으시는 거죠.

2️⃣ '월 2만 7천원 말고 다른 비용이 있나요?'
→ 없습니다. 
   그게 저희가 약속하는 모든 비용이에요.
   여행할 때 추가로 드리는 비용도 없어요.

3️⃣ '지금 당장 여행할 생각이 없는데요?'
→ 알겠어요. 
   하지만 3년 뒤를 생각해보세요.
   지금 가입 vs 3년 뒤 가입의 비용 차이가 
   얼마나 클지를 아시면 후회할 수도 있어요.

(잠깐 침묵)

그래서 길지 않구요, 
지금 딱 2분만 시간 괜찮으실까요?"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**P0-1 적용**: 거부 이유 3가지를 **미리 제거**
- "5년이 부담" → "탈퇴 가능"으로 해소
- "숨겨진 비용" → "없다"로 명확화
- "지금은 안 됨" → "3년 뒤 손실" 힌트

## ✨ 해결책 (3분) - P0-2 적용 (신뢰도 × 가격 재설정)

### 【구체적 비교: 정가 vs 우리】

```
상담사: "일반 여행과 우리를 비교해볼게요.

【일반인 50만원 여행 1번】
항공 + 호텔 + 식사 + 가이드 = 50만원

거기에 항만세, 팁, 기항지 비용
10~15만원이 더 들어요.

실제: 50~65만원

【우리 회원 같은 여행】
월 2만 7천원 가입
여행당 35만원 (30% 할인 적용)

총 비용: 27,000 + 350,000 = 377,000

(침묵 2초)

'어? 비싸네요?'
네, 1번만 가면 비싸요.

그럼 2번 가면?

【2번 여행 비교】
① 일반인: 50만원 × 2 = 100만원
② 우리: 27,000 + 35만원 × 2 = 102.4만원

거의 같은데, 우리는 추가 혜택까지 있어요.
왜냐면 우리는 선사이거든요.

【3번 여행 비교】
① 일반인: 50만원 × 3 = 150만원
② 우리: 27,000 + 35만원 × 3 = 137.4만원

12.6만원 더 싸죠?

5년에 5번 다니면?
① 일반인: 250만원
② 우리: 27,000 × 60개월 + 35만원 × 5 
       = 162만원 + 175만원 = 337만원

아, 계산 실수.
다시: 월 2만 7천원 × 60개월 = 162만원
     여행비 35만원 × 5번 = 175만원
     총 337만원

아니다, 월 2만 7천원은 멤버비고,
거기서 이미 할인이 계산된 가격이에요.

더 간단하게:
"50만원 여행을 자주 다니신다면,
우리 회원은 그 비용으로 '여러 번' 다닐 수 있다"

이게 핵심이에요."

(침묵 3초)
```
